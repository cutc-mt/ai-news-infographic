Archon OS - AIコーディング用OS 完全ガイド

2025年9月15日

Archon OSとは？

  • 史上初のAIコーディング用OS。AI開発者Cole Medan氏によるオープンソースプロジェクト。
  • AIコーディングアシスタントの知識とタスク管理を担うバックボーン。
  • Claude CodeやKiraなどのAIエージェントを強化する司令塔。

主要コンセプト

  • 統一知識ベース: 全エージェントがリアルタイムでアクセスできる共通の知識基盤。ドキュメントやURLを登録可能。
  • リアルタイムタスク管理: エージェントのタスクを視覚化。進捗確認やリアルタイムでのタスク編集が可能。
  • MCPサーバー: 各AIエージェントとArchon OSを接続し、機能を拡張する。

インストール要件

Archon OSを動かすために必要なツール群です。

  • Docker Desktop
  • Supabase (データベース)
  • Git
  • VS Code
  • モデルプロバイダー (Ollama, OpenAI, Gemini等)

インストール手順

  1. リポジトリをクローンする。
  2. 環境変数ファイル(.env)を設定する (Supabase URL, APIキー等)。
  3. Supabaseで提供されるSQLスクリプトを実行し、DBをセットアップ。
  4. Docker Composeコマンドでサービスを起動。
これであなただけのAIコーディングOSが起動します！

UIの力

洗練されたUIでエージェントの動きを完全に把握。

  • プロジェクト管理: カンバンボード形式でタスクの状況（未着手、進行中、完了）を一覧。
  • リアルタイム編集: 進行中のタスク内容をその場で編集し、エージェントの挙動を即座に変更。
  • 各種設定: APIキー、RAG設定、UIテーマ（ダークモード等）をGUIで簡単に設定。

エージェントを接続

お気に入りのAIコーディングツールをArchon OSに接続します。

  • Archon OS内のMCPサーバー情報をコピー。
  • Claude CodeやKiraなど、各ツールの設定ファイルにMCP情報をペースト。
  • これだけで、エージェントがArchon OSの知識ベースとタスク管理機能を利用開始します。
AIエージェントの真価を最大限に引き出そう！
