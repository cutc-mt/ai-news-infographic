Local AI Autocomplete: Your Private Copilot

2025年9月17日

The Goal: Offline Copilot

  • インターネット接続なしでコード補完を実現。
  • プライベートでセキュア。コードが外部に送信されない。
  • GitHub Copilotなどの有料サービスからの完全な代替。
  • すべてが自分のマシン上で完結する安心感。
クラウド不要！
完全オフラインの
AIコーディング環境！

The Toolkit

  • エディタ: VS Code
  • 拡張機能: Continue
  • バックエンド: Ollama
  • モデル: Code Llama (FIM対応)
この4つを揃えるだけ！
競合する拡張機能は
無効化しておこう！

The Autocomplete Secret

  • Next-Token Prediction: 通常のチャットモデルは「次の単語」を予測する。
  • Fill-in-the-Middle (FIM): コード補完には「中間を埋める」能力が必要。
  • FIM用に特別に訓練されたモデルを選択することが成功の鍵。
  • ContinueのドキュメントやOllamaでFIM対応モデルを探そう。
普通のチャットモデルじゃダメ！
FIM対応が必須！
これが一番大事なポイント！

The Configuration

  • Continue拡張機能の config.yaml ファイルを編集。
  • チャット用モデル: 好きなコーディングモデルを設定 (例: `codellama:7b`)。
  • 補完用モデル: FIM対応の軽量モデルに role: 'autocomplete' を指定。
models:
  - name: "My Chat Model"
    model: "codellama:7b"
    provider: ollama
  - name: "My Autocomplete Model"
    model: "codellama:1.5b"
    provider: ollama
    role: "autocomplete"
チャット用と補完用、
2つのモデルを使い分けるのが
プロのやり方！
