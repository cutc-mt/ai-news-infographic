Gemini CLI UI: GoogleのAIをブラウザで使う

2025年9月20日

1. Gemini CLI UIとは？

  • GoogleのAI開発ツールGemini CLIに、WebベースのUIを提供するオープンソースプロジェクト。
  • どのデバイスからでもAI支援コーディングを可能にすることを目指す。
  • React製のフロントエンドとExpress製のバックエンドで構成。

2. 主な機能

  • インタラクティブチャット: 使い慣れたUIでGeminiと対話。
  • 統合ターミナル: 必要に応じて従来のCLIインターフェースも利用可能。
  • ファイルエクスプローラー: プロジェクトファイルの閲覧とライブ編集。
  • Git連携: 変更履歴の確認やコミット操作。
  • モデル選択: Gemini 2.5 ProとFlashを簡単に切り替え。
  • YOLOモード: Geminiが許可なしでアクションを実行。

3. インストール方法

  • 1. `git clone`でリポジトリをクローン。
  • 2. `cd`でプロジェクトディレクトリに移動。
  • 3. `npm install`で依存関係をインストール。
  • 4. `.env.example`を`.env`にコピー。
  • 5. `npm run dev`でサーバーを起動。

4. ライブデモ：マインスイーパ作成

  • 「ネオンカラーのマインスイーパをHTML/CSS/JSで作って」と指示。
  • 初期失敗: デフォルトでツールが無効なため、ファイルが生成されない。
  • 解決策: 設定でYOLOモードを有効化。
  • 成功: AIがHTML、CSS、JSファイルを生成し、動作するゲームが完成。
ファイルがUIに表示されない時は、ブラウザのリロードが必要な場合があります。

5. ユースケースと評価

  • 最適な用途: サーバーにインストールし、チームや個人がリモートで開発環境にアクセスする。
  • ローカル開発: VS CodeのターミナルやGemini Code Assist拡張機能でも同様の体験は可能。
  • 結論: リモート開発環境をWeb UIで構築したい場合に非常に価値のあるツール。
