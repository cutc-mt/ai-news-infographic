Claude CodeでGLMコーディングプランを使う方法

2025年9月21日

目標

  • 高価なClaude Codeの代わりに、激安のGLMコーディングプランを利用する
  • Windows環境でClaude CodeのバックエンドをGLM 4.5に切り替える
  • コマンド一つでGLM版Claude Codeを起動できるようにする

Step 1: 環境変数の設定

  • Win + Rで「sysdm.cpl」を実行
  • [詳細設定] > [環境変数] をクリック
  • ユーザー環境変数に新規で変数を追加
  • 変数名: ZHIPUAI_API_KEY
  • 変数値: ZhipuAIから取得したAPIキー

Step 2: バッチファイルの作成

  • メモ帳などのテキストエディタで新規ファイルを作成
  • 以下の4行をコピーして貼り付け
    • @echo off
set ANTHROPIC_BASE_URL=https://open.bigmodel.cn/api/paas/v4
set ANTHROPIC_API_KEY=%ZHIPUAI_API_KEY%
claude-code
  • zc.bat という名前で保存

Step 3: パスの設定

  • Cドライブ直下などに Tools フォルダを作成
  • 作成した zc.bat を C:\Tools に移動
  • 再度、環境変数の設定画面を開く
  • ユーザー環境変数の Path を選択して編集
  • 新規で C:\Tools を追加
