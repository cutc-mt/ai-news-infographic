Top 10 Trending Open Source GitHub Projects - Part 1

September 21, 2025

1. Tongyi Deep Research

  • Agentic model for deep information seeking
  • Massive 128k context length
  • Dual inference modes: ReAct and Iter-Research

2. Convex Chef

  • Builds full-stack apps from a prompt
  • Includes backend, database, auth, and file storage
  • Real-time UI updates

3. Agent Payments Protocol (AP2)

  • Trusted and auditable payments for AI agents
  • Uses cryptographically signed mandates
  • Backed by Mastercard, PayPal, Coinbase, Adobe, etc.

4. Serverless Stack (SST)

  • Build full-stack apps on your own infrastructure
  • Define your entire app in one place with code
  • Live development workflow with tight feedback loops

5. WebGoat

  • Learn web security by breaking and fixing real vulnerabilities
  • Purposefully insecure web app for safe experimentation
  • Lessons mapped to OWASP Top 10

6. Times FM

  • Google's pre-trained time series foundation model
  • Delivers strong zero-shot predictions
  • Efficient with 200M parameters

7. LazyVim

  • Powerful yet lightweight Neovim setup
  • Works out of the box with sane defaults
  • Easily extendable with a lazy-loading plugin manager

8. Redoc

  • Generates beautiful and interactive API documentation
  • 3-panel layout for easy navigation
  • Supports OpenAPI, Swagger, and more

9. MCP-GraphQL

  • Lets LLMs tap into any GraphQL API
  • Safe by default, with mutations disabled
  • Dynamic discovery of schema

10. Ripple

  • Elegant TypeScript UI framework blending React, Solid, and Svelte
  • Fine-grained reactivity for better performance
  • TypeScript-first at its core
