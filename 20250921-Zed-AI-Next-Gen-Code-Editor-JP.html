Zed AI: 次世代コードエディタ

2025年9月21日

Zed AIとは？

  • Rust製の高性能オープンソースコードエディタ
  • 人間とAIとの協調コーディング
  • 思考の速さでコーディング
  • エージェント指向IDE

主な機能

  • Claude Code と Gemini CLI を内蔵
  • ネイティブの多言語対応デバッガ
  • Gitリポジトリをエディタ内で管理
  • リアルタイム共同編集
  • 編集予測、リモートデプロイなど

はじめに

  • macOS と Linux はワンクリックインストーラ
  • Windows はGitHubからベータ版をインストール
  • 無料アカウントを作成して開始
Windows正式版は10月リリース予定！

AI機能の使い方

  • エージェントパネルでZedエージェントとチャット
  • Gemini CLIとClaude Codeを統合
  • フォローモードでエージェントの編集を追跡
  • AIが生成したコードの変更を確認・承認
  • 拡張機能のインストールやカスタムルールの追加
