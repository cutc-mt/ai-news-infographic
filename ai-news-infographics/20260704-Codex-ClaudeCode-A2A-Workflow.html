①Traycerとは: AIコーディングの指揮センター

Traycerは、複数のAIコーディングエージェントを一つの共有ワークスペースで統合管理できるフリーデスクトップアプリ。計画から実行までを構造化してサポートする。

  • ウェブサイト: traycer.ai
  • オープンソースで公開中
  • デスクトップアプリとして動作
「オーケストラの指揮者になれる感覚!」とユーザーから絶賛の声

②オープンソース化: 完全無料で使える

Traycer Desktopアプリがフリー＆オープンソースとしてリリース。GitHubから誰でもダウンロード可能で、コードも公開されている。

  • GitHub ReleasesからDL可能
  • macOS (ARM64) に対応
  • ソースコードも完全公開
「Bring Your Own Agent」で自分が使いたいエージェントを自由に接続OK!

③A2Aプロトコル: エージェント同士が直接対話

Googleが提唱するAgent-to-Agentプロトコルを活用。異なるAIエージェント同士がウォーキートーキーのようにコミュニケーション可能に。

  • 質問のやり取りやレビューの依頼が可能
  • 別々のチャット間で作業を引き継げる
  • Linux Foundationに寄贈済み
エージェント同士がトランシーバーで話すイメージ!

④対応エージェント: 主要ツールを全て網羅

Traycerは多様なコーディングエージェントに対応。既存のサブスクリプションをそのまま活用できる「BYOA」モデルを採用している。

  • Claude Code AnthropicのCLIエージェント
  • Codex OpenAIのコーディングエージェント
  • OpenCode オープンソースのコーディングAI
  • Cursor AI搭載コードエディタ
さらにGeminiやローカルランタイム、カスタム設定にも対応

⑤EPIC Mode: 構造化された計画レイヤー

Traycerのコア機能であるEPIC Mode。単なるコーディングではなく、スペックからチケット、ストーリー、レビューまで一貫した計画を生成する。

  • Spec → Ticket → Story → Review の流れ
  • 意思決定と作業項目を保存・管理
  • コードを書く前に計画を確認・修正可能
5ヶ月かかった仕事をEPIC Modeでたった6日で完了した実績あり!

⑥Regular Mode vs EPIC Mode: 2つの使い分け

Traycerには2つのモードが用意されている。タスクの複雑さに応じて最適なアプローチを選べる。

  • Regular Mode 直接的なコーディング作業
  • EPIC Mode 構造化された計画レイヤー付き
  • チャット内でモードを柔軟に切り替え可能
単純な修正はRegular、大規模開発はEPICと使い分けるのが正解!

⑦タスクシステム: ワークを構造化して管理

すべての作業は「タスク」というトップレベルのコンテナにまとめられる。チャット、ターミナル、ファイル、git diffを一箇所で管理。

  • チャットセッションを含む
  • ターミナルエージェントを統合
  • 履歴の検索・再開・リネームが可能
過去のタスクをフィルタリングして再利用できるのが便利!

⑧共有コンテキスト: 1タスク1共有空間

各タスクには独自の共有ファイルシステムがある。アーティファクト、履歴、作業コンテキストを全エージェントで共有できる。

  • 意思決定の記録が自動保存
  • チャットの途中でモデルを切り替え可能
  • スレッドを失わずに作業を継続
Claudeで計画してCodexで実行するような横断的な使い方ができる!

⑨アーティファクト: 4種類の耐久性のある成果物

EPIC Modeで生成される4種類のアーティファクトが、開発の意思決定と品質を担保する。

  • Specs 要件定義書・仕様書
  • Tickets 個別作業項目
  • Stories ユーザーストーリー
  • Reviews レビュー記録
「動くだけ」から「堅牢に動く」への品質向上がポイント!

⑩Worktree対応: Git作業の分離

エージェントの作業をworktreeで安全に分離。ワークスペースフォルダ、新規worktree、既存worktreeの3つの選択肢がある。

  • ワークスペースフォルダで直接作業
  • 新しいworktreeを作成して分離
  • 既存のworktreeに接続
複数エージェントが同時に作業しても競合しない!

⑪BYOAモデル: 自分のエージェントを持参

「Bring Your Own Agent」はTraycerの核となる考え方。既に払っているサブスクリプションを無駄にしない設計。

  • 推論コストは$0
  • 無制限のエージェントセッション
  • トークンメータリングなし
  • 推論クレジット不要
BYOAプランは月額$0! 自分のAPIキーを使えば完全無料!

⑫内蔵スキル: 再利用可能なワークフロー

Traycerには組み込みのスキルが標準搭載。計画・デバッグ・レビュー・議論・ドキュメント生成などを効率的に実行。

  • Plan: 作業計画の自動生成
  • Debug: デバッグ支援
  • Review: コードレビュー
  • Document: ドキュメント自動生成
作成したワークフローは何度でも再利用できる!
