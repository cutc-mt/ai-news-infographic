②今回のテーマ: 手書き風ホワイトボードアニメ

  • ペンが実際に描いているような手書き風アニメーションをAIで制作
  • Claude Sonnet 5の性能を動画制作の全工程で検証
  • 台本作成から音声合成・アニメーション・編集まで一貫してAIに任せる
  • 完成した1本の解説動画ができるまでの舞台裏を紹介
Claude Codeに全部任せて
1本の動画ができるまで！

③使用したAIモデル: Claude Sonnet 5

  • Claude Sonnet 5はAnthropicの最新コーディング特化モデル
  • ベンチマークでOpus 4.8に迫る性能を発揮
  • Claude Codeとしてターミナル上のコーディングエージェントとして動作
  • コストパフォーマンスに優れ、実務での利用が急増中
Sonnet 5がOpusに迫る性能！
しかもコストも安い

④スキル1: handwriting-video-maker

  • SVGの図解をペンを持った手が実際に描くアニメーションに変換
  • ホワイトボード解説動画でよく見られる手書き風演出を実現
  • SVGパスを解析して、ペンの軌跡を自動生成する仕組み
  • 静止画の定着時間も調整可能で、テンポの良い映像に
SVGが動いて手書き風に！
これが見せ場だよ

⑤スキル2: aivis-speech（音声合成）

  • ローカル音声合成エンジンAivisSpeechを使用
  • 今回は「まお」ボイスでナレーションを生成
  • クラウドAPI不要でローカル環境で完結
  • シーンごとに音声を個別生成し、正確な尺を計測
まおボイスかわいい〜
ローカル完結だから安心

⑥制作工程1: 台本作成とシーン分割

  • まず台本を書き、内容ごとに3〜5個の短いシーンに分割
  • シーン単位で独立させることで、制作と修正が効率的に
  • 各シーンの構成を事前に決めておくことが品質の鍵
  • Claude Codeが台本の構成案から執筆までを担当
シーン分割が大事！
3〜5個に分けるのがちょうどいい

⑦制作工程2: ナレーション音声の生成

  • シーンごとにナレーション音声を個別に生成
  • 生成した音声から正確な尺（秒数）を計測
  • この尺情報が次のアニメーションのタイミング調整に必須
  • 音声のトーンや速度も微調整可能
尺の計測が超大事！
ここが映像のテンポを決める

⑧制作工程3: SVG図解の作成

  • シーンごとに手描きアニメーション用のSVG図解を作成
  • テキスト・図形・矢印などをSVG形式で設計
  • Claude Codeがコードを直接書いてSVGを生成
  • 複雑な図解もパスデータとしてきれいに描画
SVGを直接コードで書く！
Claude Codeの得意分野

⑨キーポイント: 尺合わせのテクニック

  • 手の動きは常に速くリズミカルにする
  • 尺合わせは「テキストが定着している時間」で行う
  • これによりテンポの良い仕上がりを実現
  • 描画速度と静止時間のバランスが映像の質を左右
手は速く動かして
止まっている時間で尺を合わせる
← ここがポイント！

⑩制作工程4: 映像と音声の合成

  • 各シーンの映像と音声を合成して1つのクリップに
  • 全シーンを最後に1本の動画として結合
  • ffmpegなどのツールでコマンドラインで処理
  • 自動化で人間が手作業する部分を最小化
シーンごとに合成して
最後に全部くっつけるだけ！

⑪制作工程5: 完成後の通し確認

  • 完成後は必ず映像を通しで確認する
  • 不自然な箇所がないか細部までチェック
  • 問題があれば修正して再生成のループを回す
  • 品質確認をAI自身にさせることで効率化
通し確認は必須！
不自然なところを見つけて直す

⑫遭遇した不具合: 手が消えない問題

  • ペンを持った手が消えずに画面に残ってしまう不具合が発生
  • 原因を特定して修正するところまでをAIと一緒に実施
  • SVGパスの終点処理にバグがあったことが判明
  • 試行錯誤を経て無事に解決
手が消えない！？笑
原因特定して修正完了 ✨

⑬Claude Codeの実行環境とCLI

  • Claude Codeはターミナル上で動くコーディングエージェント
  • 独自のCLI（コマンドライン）を開発して活用
  • 要はマウスを動かしたりコマンドを叩いたりする操作をAIが実行
  • 外出先からDispatch経由でスマホから指示も可能
Claude Code = AIエージェント
ターミナル上で自律動作

⑭Loop Engineering: 改善ループの構築

  • Loop Engineeringの概念を動画制作に応用
  • タスクをループで回してAIに自律的に改善させる
  • AI自身にチェックさせて完了と判断させる
  • 4つの層が積み重なって段階的に進化する仕組み
AIにループ回して
自律的に改善させる！
