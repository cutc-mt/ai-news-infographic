Bitebot: The True Open-Source AI Desktop OS

2025年9月17日

The Problem with "AI OS"

  • 多くは高価な有料サービス。
  • 実際はクラウドベースで、デスクトップを操作しない。
  • 機能がブラウザ自動化に限定されていることが多い。
  • 真の「AIオペレーティングシステム」とは言えない。
名ばかりのAI OSは
もう終わり！

Meet Bitebot

  • 完全無料でオープンソースの真のAIデスクトップOS。
  • AIエージェントに仮想デスクトップ環境を丸ごと提供。
  • 人間のように画面を見て、マウスとキーボードを操作。
  • まるで自分専用のPCを持つ「仮想従業員」。
AIに自分専用のPCを
まるごと与える！

What Can It Do?

  • メール、Office、IDEなどデスクトップアプリを操作。
  • 独自のファイルシステム内でファイルをダウンロード・整理。
  • PDFやスプレッドシートを読み込み、複数アプリをまたいで処理。
  • 数百のAIエージェントを並列実行し、タスクをスケール。
ブラウザの操作だけじゃない！
デスクトップ作業を
完全自動化！

How to Get Started

  • Docker Composeを使用したセルフホストを推奨。
  • 商用APIの他、Ollama経由でローカルモデルも利用可能。
  • リポジトリをクローンし、APIを設定してコマンドを実行するだけ。
  • 最低4GBのRAMとDocker環境が必要。
Dockerで、誰でも
無料で始められる！
