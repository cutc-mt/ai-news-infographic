GPT-5 CodexをClaude Codeの外部脳として使う

2025年9月17日

GPT-5 Codexとは？

  • 高度な自立性: 自己リファクタリング、コードレビュー、修正
  • 高精度なコードレビュー
  • 多様な開発環境に対応
  • スクリーンショットなど非テキスト情報も活用

MCPサーバーとしてのCodex

  • CodexをMCPサーバーとして利用可能
  • Claude CodeやCursorなど他のAIエージェントから呼び出し
  • ツールの使い分けで開発効率を最大化
Claudeの弱点をCodexで補う新しい開発スタイル！

Claude Codeとの連携

  • Codex CLIをv0.36.0以上にアップデート
  • Claude CodeにCodexをMCPとして追加
# MCP追加コマンド
claude mcp add codex codex mcp
`/mcp`コマンドで接続を確認！

Web検索を有効にする

  • `config.toml`ファイルを編集
  • `tools = ["web_search"]`を追加
  • Codexが最新情報にアクセス可能に
MCPとして使うならWeb検索は必須！
