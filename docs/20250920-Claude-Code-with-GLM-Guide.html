Claude CodeでGLM-4.5を激安で使う方法

Windows初心者向けチュートリアル - 2025年9月20日

なぜGLM-4.5なのか？

  • Claude Code公式プランは高価 ($17~$100/月)
  • GLM-4.5は公式に匹敵する高性能モデル
  • GLMコーディングプランは月額$6 (初月$3)と激安
コストを気にせずAI駆動開発を学べる！

設定のキモ：環境変数

  • Claude Codeは起動時に設定を読み込む
  • APIキーとエンドポイントURLを上書きする
  • Windowsの環境変数設定は初心者には難しい…
でも大丈夫！簡単な方法があります！

Step 1: APIキーをPCに記憶

  • システムのプロパティから環境変数画面へ
  • ユーザー環境変数に新規で変数を追加
  • 変数名: ZHIPU_API_KEY
  • 変数値: Zhipu AIから取得したAPIキー
これでAPIキーの準備はOK！

Step 2: 魔法のバッチファイル作成

  • メモ帳に以下の4行をコピペ
    • @echo off
set ANTHROPIC_BASE_URL=https://api.zhipuai.cn/v4/anthropic
set ANTHROPIC_AUTH_TOKEN=%ZHIPU_API_KEY%
claude-code %*
  • zc.bat という名前で保存
このファイルがモデルを切り替える鍵！

Step 3: パスを通す

  • Cドライブ直下などにToolsフォルダを作成
  • 作成した zc.bat をToolsフォルダに移動
  • 環境変数の「Path」に C:\Tools を追加
これでどこからでもコマンドが使える！

使い方

  • claude-code と入力:
    通常のClaudeモデルで起動
  • zc と入力:
    GLM-4.5モデルで起動！
コマンド1つでモデルを簡単切り替え！
