謎のAIコーディングモデル「Code Supernova」レビュー

2025年9月20日

1. 謎のステルスモデル登場

  • Code Supernovaという名の、高性能なコーディングモデルが突如出現。
  • 現在アルファ版として、Windsurf, Kilo Code, CursorなどのAIエージェントを通じて無料で利用可能。
  • その正体は Grok 4 Coder か、はたまた Claude 4.5 の派生版か、様々な憶測を呼んでいる。

2. Code Supernovaの主な特徴

  • マルチモーダル: 画像入力をサポート。
  • 巨大コンテキスト: 200kという広大なコンテキストウィンドウを持つ。
  • 高速＆高性能: 非常に高速でありながら、推論能力も兼ね備えた堅実なコードを生成。

3. テスト① ワイヤーフレームからWebサイト構築

  • タスク: Webサイトのワイヤーフレーム画像を読み込ませ、それに基づいたWebページを生成させる。
  • 結果: ワイヤーフレームの構造をほぼ忠実に再現した、アニメーション付きの機能的なランディングページを生成。マルチモーダル能力の高さを示した。

4. テスト② SVGとアニメーション

  • タスク: ゲームコントローラーのSVGを生成し、それをアニメーションさせる。
  • 結果: SVGの品質はまずまずだったが、アニメーションは不自然で、モデルの限界が露呈。最高峰モデルではない可能性が示唆された。

5. テスト③ 暗号資産アプリのプロトタイピング

  • タスク: 暗号資産取引アプリのダッシュボードを生成。
  • 結果: 機能的なダッシュボードは生成されたものの、デザインが「安っぽいAI生成物」の域を出ず、UI品質には課題が残った。

6. 結論と評価

  • 長所: 圧倒的な速度、巨大なコンテキスト、まともなマルチモーダル能力。ラピッドプロトタイピングに最適。
  • 短所: 生成物の品質は最高峰には及ばず、複雑なタスクやデザインの洗練さには欠ける。
無料で使えるモデルとしては非常に優秀。アルファテスト期間中にぜひ試してみる価値あり！
