MCPレジストリ登場：AIツールハブの紹介

2025年9月20日

MCPレジストリとは？

  • MCPサーバーを発見・共有するためのコミュニティ主導のサービス。
  • Model Context Protocolチームによるオープンソースプロジェクト。
  • 増え続けるMCPサーバーの中から必要なものを見つけやすくする。

MCPレジストリUI

  • レジストリを簡単に検索・探索できるオープンソースのUI。
  • サーバーの詳細（公開者、更新日、リポジトリ情報）を一覧表示。
  • インストールや利用に必要なコマンドも簡単にコピー可能。

使い方とアーキテクチャ

  • レジストリ本体はGitHubリポジトリで管理。
  • UIはFastAPI製のバックエンドを介してリポジトリの情報を取得。
  • 開発者はガイドに従って自作のMCPサーバーをレジストリに公開可能。

関連ツール「Rove」

  • composio上に構築された、500以上のアプリ連携を提供するツール。
  • Claude DesktopなどのMCPホストに簡単に統合可能。
  • 「Roveを使ってGitHubの最新リポジトリ名を取得して」のように自然言語で様々なツールを操作できる。
Roveを導入すれば、Claudeの中からGmailやSlack、Notionなど、数百のツールを呼び出せるようになります！
