AIの新しい活用法：MCPとAPIコネクト

2025年9月21日

MCP/APIとは？

  • MCP: AIと外部アプリを接続する仕組み
  • API: 外部アプリの機能を利用する仕組み
  • AIエージェント「Manus」でこれらの機能を活用

Gmail & Googleカレンダー連携

  • Gmailから過去のサブスクリプションを自動でリストアップ
  • Googleカレンダーの空き時間から打ち合わせ日程を自動調整

ElevenLabs連携

  • API経由でリアルな効果音を生成
  • Manusがプロンプトを自動生成し、効果音を作成

Google Gemini (NanoBananan)連携

  • API経由で画像を線画に変換
  • 漫画制作など、さらなる応用の可能性

Tripo AI連携

  • API経由で3Dアセットを生成
  • 画像から3Dモデルを作成し、3Dプリンターで出力可能

組み合わせ活用事例

  • YouTube動画 → ブログ記事: ElevenLabsで文字起こし、Manusで記事作成とスクショ挿入
  • 画像 → 3Dフィギュア: NanoBanananでフィギュア風画像を生成し、Tripo AIで3Dデータ化
