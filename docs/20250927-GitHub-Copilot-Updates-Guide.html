GitHub Copilot アップデートまとめ

2025年9月27日

Copilot CLIの登場

  • ターミナルで動作する新しいAIエージェント。
  • Claude CodeやCodex CLIの競合となるツール。
  • npm install -g @github/copilot-cli で簡単にインストール可能。
  • 現在パブリックプレビュー中で、無料で試せる可能性あり。

CLIの主な機能と特徴

  • GitHubアカウントと連携し、リポジトリの文脈を理解。
  • コードのビルド、編集、デバッグ、リファクタリングが可能。
  • MCPサーバーをサポートし、機能を拡張できる。
  • 全ての操作にユーザーの承認が必要で、安全性が高い。
  • デフォルトで Claude Sonnet 4 を使用し、GPT-5 に切り替えも可能。

コーディングエージェント

  • GitHubのIssueにCopilotエージェントをアサイン可能に。
  • エージェントがIssueを元に開発を行い、プルリクエストを作成。
  • 人間はレビューとマージに集中できる。
まるでGitHub上にDevinがいるみたい！開発がさらに加速しそう。

モバイル対応

  • GitHubのモバイルアプリからCopilotのタスクを開始可能に。
  • クラウドベースの開発環境と組み合わせることで、外出先でも開発が進められる。
  • Issueの作成からエージェントへのタスク割り当てまで、スマホで完結。

料金とプラン

  • Copilot Pro: 月額$10程度。
  • Business / Enterprise: より高機能なプラン。
  • 現在はパブリックプレビュー期間中で、無料で利用できる可能性が高い。
会社でCopilotが支給されている人は、すぐにでも試す価値あり！
