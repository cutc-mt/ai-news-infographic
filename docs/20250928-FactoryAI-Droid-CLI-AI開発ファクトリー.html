Factory.AI & Droid CLI: 新世代のAI開発ファクトリー

2025年9月28日

Factory.AIとは？

単一のAIエージェントではなく、特化した「ドロイド」たちが「工場(Factory)」のように連携して開発を進めるプラットフォーム。

  • 専門家集団: 各タスク（コーディング、リサーチ等）に最適化されたドロイドが存在。
  • 分業体制: 複雑なタスクをドロイドの分業によって効率的に解決する。
プロンプトやコンテキストをタスク毎に分けるという考え方を具現化！

Droid CLI: ターミナルの新戦力

Factory.AIの能力をコマンドラインで実現するツール。ベンチマークで高い性能を証明。

  • 高性能: Terminal-Benchで他のAIエージェントを10%以上上回るスコアを記録。
  • 権限レベル: 誤操作を防ぐため、コマンドの自動実行レベルを細かく設定可能。
  • IDE連携: VS CodeやCursorと統合し、エディタ内でシームレスに利用できる。

ドロイドの種類

タスクに応じて専門のドロイドを選択・組み合わせます。

  • Code Droid: コーディングやリファクタリングを担当。
  • Knowledge Droid: 技術ドキュメントのリサーチや要約を担当。
  • Reliability Droid: PRのレビュー、デバッグ、品質分析を担当。

Web UI: 開発の司令塔

ブラウザからドロイドを管理し、プロジェクトを推進するGUI。

  • インテグレーション: GitHub, Slack, Linearなど多くの開発ツールと連携。
  • Factory Bridge: 専用アプリで、ローカルのファイルにも安全にアクセス可能。
  • 豊富なコンテキスト: ファイルだけでなく、PRやチケット、Notionページも参照させられる。

非同期ワークフロー

数時間に及ぶ大規模なタスクをドロイドに任せ、人間は他の作業に集中できます。

  • 進捗報告: 長時間タスクの実行中も、ドロイドは定期的に状況を報告。
  • 人間による介入: 次の作業の優先順位など、要所で人間が指示を与えることが可能。
夜間にリファクタリングを実行させ、朝に結果を確認する、といった働き方が可能に！

柔軟な料金体系

個人でも試しやすい魅力的な料金プランが用意されています。

  • BYOK (Bring Your Own Key): 自身のAPIキーを使えば、Factory.AIのプラットフォーム機能は無料で利用可能。
  • Proプラン ($20/月): チーム機能や各種インテグレーションに加え、毎月2000万トークン分のクレジットが付属。
