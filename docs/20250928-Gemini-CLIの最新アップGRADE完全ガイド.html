Gemini CLI 最新アップグレード完全ガイド

2025年9月28日

Gemini CLIの進化

  • 単なるチャットツールから、スクリプト可能な開発アシスタントへ。
  • ファイル編集、クラウドデータ連携、アプリデプロイ、セキュリティ分析など多機能化。
  • AIエージェントがエディタ、ターミナル、CIなど、いつもの作業場所に統合。
「開発者のための外部脳」として、コーディング作業を強力にサポート！

初期の進化 (Early Wave)

  • Zedエディタにサードパーティエージェントとして統合。
  • 設定ダイアログの追加で、JSON編集不要に。
  • メッセージキューイングで、応答中でも次の質問を予約可能。
  • セッション統計で、コードの追加・削除行を追跡。
  • カスタムコマンドが引数対応＆高速なNo-Promptモードを搭載。
  • ヘッドレス編集フローでCIでの自動変更が可能に。

自動化と洗練 (Automation & Polish)

  • GitHub ActionsでPR時のコードレビューやIssue整理を自動化。
  • カスタムテーマで、編集モードなどを視覚的に区別。
  • 信頼できる操作は確認をスキップするAllowed Tools機能。
  • ツールコールの一括承認でワークフローを高速化。
  • 組織単位でのベース設定を定義可能に。

デプロイとセキュリティ

  • CLIから直接Google Cloud Runへアプリをプッシュ。
  • 脆弱な依存関係を検出し、修正を提案するセキュリティスキャン機能。
  • 入力中に提案を行うプロンプト補完機能。
  • 出典を自動表示する引用機能でトレーサビリティ向上。
  • ローカルファイルやディレクトリをプロンプトに直接埋め込み可能。
高速な修正サイクルを実現！セキュリティスキャンは超クールな機能。

ツール連携の強化

  • MCPサーバーのインストールと管理を効率化。
  • ワンライナープロンプトで即座に結果を得られる非対話モード。
  • 長いログ出力を要約し、フル出力は別途保存。
  • 複数ファイルにまたがる変更と、よりクリーンな差分表示。
  • 組織全体で特定の認証タイプを強制可能に。

最新機能：データ連携と上限解放

  • AI Pro/Ultra登録者向けにGemini 2.5のクオータ上限を大幅に引き上げ。
  • AlloyDB, BigQuery, CloudSQLなど多数のDBや分析ツールと連携。
  • JSON出力モードで、結果をクリーンにパース可能。
  • チャット履歴をMarkdownやJSONファイルにエクスポート。
  • 過去のアイデアを素早く見つけるプロンプト検索機能。
  • 入力の取り消し（Undo/Redo）機能。
CLIがデータコンシェルジュに！これはすごい。

主な機能の場所 (UI)

  • Zed Editor: エージェントパネルからGeminiスレッドを開始。
  • Terminal: CLIから直接設定パネルやセッション統計にアクセス。
  • Cloud Tools: ツールメニューからCloud Runへのデプロイやセキュリティスキャンを実行。
  • Data Tools: ツールとしてDBやBigQuery拡張機能が表示され、データのクエリや探索が可能。
  • Chat Export: メニューから会話をファイルに保存。

長所と注意点

  • 生産性向上: ヘッドレス編集や一括承認は驚くほど高速。
  • 強力な連携: クラウドやデータソースとの連携でワークフローが劇的に改善。
  • 要ガードレール: 本番環境ではサンドボックス化やポリシー適用が重要。
  • 初期設定: Google Cloud連携は初期設定が必要だが、価値は高い。
  • クオータ: 無料利用枠の場合、使用量を意識し軽量モデルを選択することが賢明。
