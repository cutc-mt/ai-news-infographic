OpenRouter無料LLMを無制限に！OpenInference裏設定ガイド

2025年9月28日

OpenRouter無料モデルの課題

OpenRouterで提供される無料LLMには、厳しいレート制限が存在します。

  • 1分あたり: 20リクエスト
  • 1日あたり: 50リクエスト
  • この上限は、チャットを少し続けるだけですぐに到達してしまいます。
残高が$10以上あれば1日1000リクエストに緩和されますが、完全無料ではありません。

解決策: OpenInference

OpenInferenceは、OpenRouterと協力関係にある組織で、コーディング関連の学習データ収集を目的としています。

  • データ提供: あなたのプロンプトと生成結果を公開データセットとして提供します。
  • 制限解除: その見返りとして、無料モデルのレート制限が解除されます。

簡単！設定方法

設定は非常に簡単で、スイッチを1つ切り替えるだけです。

  1. OpenRouterのSettings > Privacyにアクセス。
  2. 「Enable free endpoints which may publish prompts」のトグルをオンにする。
たったこれだけで、あとは通常通りAPIキーを使ってモデルを利用できます！

【超重要】注意点！

この設定を有効にすると、あなたのプロンプトは公開データとして扱われます。

  • 機密情報: APIキーやパスワードなどを絶対に送信しない。
  • 個人情報: 個人を特定できる情報を送信しない。
  • 非公開コード: 会社のプロジェクトなど、公開されていないコードを送信しない。
あくまで公開されても問題ない、テストや学習目的での利用に限定しましょう。

相性の良いツール

この設定と相性が良く、簡単にプロバイダーを切り替えられるツールがおすすめです。

  • エディタ (VS Code): Kilo Code, Rue Code
  • CLI: Grok CLI
  • GUIアプリ: Bolt-DIY-Local, Quests

まとめ

OpenInferenceの設定は、メリットとデメリットを理解して利用することが重要です。

  • メリット: 様々な無料LLMをレート制限なしで存分にテストできる。
  • デメリット: 送信したデータが公開され、学習データとして利用される。
