Claude 4.5 Sonnet特集：AIエージェントと開発者向け新機能

驚異的な性能向上

  • SWE-Bench: 77.2%を記録し、GPT-5 Codex (74.5%)を上回る

開発者向け新機能

  • Claude Agent SDK: 独自のAIエージェントシステムを構築可能に

ツールキットで開発体験が劇的に向上！
