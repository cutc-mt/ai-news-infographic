魔人式プロンプトV3 - 無料で高品質なGoogleスライドを自動生成

2025年9月30日

1. 魔人式プロンプトとは？

  • 無料で提供されている高品質なスライド自動生成ツール
  • 開発者の魔人さんが100時間以上かけて開発
  • Gemini GemとGoogle Apps Script(GAS)を連携
プロンプトだけでなく、アプリとの連携を含めた「仕組み」がすごい！

2. スライド生成の仕組み

  • Gemini Gemに資料(テキスト、PDF等)を読み込ませる
  • Gemがスライド構成のJSONコードを生成
  • GAS製ジェネレーターにJSONを貼り付け
  • Googleスライドが自動生成される
コピペだけで、誰でも簡単に高品質なスライドが作れる！

3. 主な特徴

  • 高品質なデザインでポン出し可能
  • ロゴや配色をプリセットとして保存可能
  • 生成が高速（約60秒/16枚）
  • エラー発生時もGemに聞けば自己修復
使いやすさを追求した、ユーザー目線の設計が素晴らしい！

4. 誰でも使える

  • 無料版Gemini (Flash 1.5)でも動作
  • 設定はノート記事を見ながら15分程度で完了
  • AIに詳しくない人でも使えるように設計
AIの力を誰もが享受できるようにという思想が素敵！
