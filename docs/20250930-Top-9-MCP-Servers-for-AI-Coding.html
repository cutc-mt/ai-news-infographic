開発効率を5倍にする！AIコーディングで役立つMCPサーバーTOP9

2025年9月30日

1. Supabase MCP

  • DBスキーマやデータをAIが直接参照
  • RLS設定の確認・作成も可能
  • スキーマ不整合によるエラーを防止
バックエンド開発の必須ツール！

2. Stripe MCP

  • サブスク商品や価格をAIが作成・検索
  • 取引履歴の確認
  • 最新ドキュメントを検索しWebフック等を実装
決済機能の実装が驚くほど楽に！

3. Serenade MCP

  • 巨大なコードベースから高速・高精度に検索
  • サブエージェントと連携してAIを賢くする
  • 特定の修正箇所をスマートに特定
コード探索の時間を大幅に短縮！

4. Context7 MCP

  • 最新の公式ドキュメントをAIに提供
  • Next.js, Supabaseなど人気ライブラリに対応
  • GitHubリポジトリから独自ドキュメントを追加可能
古い情報によるエラーを防ぐ！

5. Playwright MCP

  • ブラウザを自動操作してE2Eテストを実行
  • スクリーンショットを撮影してUIの状態を確認
  • UIデザイナーのサブエージェントと連携してUIを改善
リリース前の動作確認が自動でできる！

6. Notion API MCP

  • ブログ記事やメモをAIが自動執筆
  • Notionからデータを取得して活用
  • 学習内容の記録や整理にも
情報整理とコンテンツ作成を自動化！

7. Figma MCP

  • Figmaのデザインを忠実にコードへ変換
  • HTML/CSSやReactコンポーネントを生成
  • 既存サイトのUIを模倣することも可能
AI臭さのないオリジナルデザインを実現！

8. GitHub MCP / gh CLI

  • リポジトリの自動作成
  • プルリクエストの作成やIssueの確認
  • GitHub Pagesへの自動デプロイ
面倒なGit操作をAIにおまかせ！

9. Chrome DevTools MCP

  • コンソールエラーや警告をAIが検出
  • LCPなどのパフォーマンス指標を計測
  • パフォーマンス改善のための具体的な提案を取得
フロントエンドのデバッグと最適化に！
