話題のオープンソースGitHubプロジェクト10選 Part2

2025年9月30日

1. Chrome DevTools MCP

  • AIエージェントにブラウザを「見る」力を与える
  • ライブデバッグ、パフォーマンス分析、自動操作が可能
AIがブラウザを直接操作！開発の常識が変わる。

2. RAG-Anything

  • 画像、表、数式を含むマルチモーダル文書を処理
  • 単一の統合システムで複雑な文書を理解
テキスト以外の情報もAIが理解する時代へ。

3. Everyone Can Use English

  • AIが発音や流暢さを評価し、個別フィードバック
  • 実践的な練習ループでスピーキングを改善
AIがあなたの英語学習をパーソナライズ。

4. Human Layer

  • 重要な操作の前に人間の承認を要求
  • Slackやメール経由で承認依頼を通知
AIの自律性と安全性を両立させる必須ツール。

5. Onyx

  • セルフホスト可能なチーム専用AIナレッジハブ
  • 40以上の情報源と接続し、社内情報を横断検索
自社のデータを安全に活用するAIアシスタント。

6. Trendfinder

  • AIがSNSやWebを監視し、トレンドを自動検出
  • Slack/Discordにリアルタイムで通知
流行の兆しを誰よりも早く掴む！

7. Gemini CLI

  • Google Geminiをターミナルから直接操作
  • 100万トークンの巨大なコンテキストウィンドウ
開発者のための強力な相棒がターミナルに。

8. Dolphin

  • 複雑な文書のレイアウトを理解して解析
  • 要素を並列処理することで高速に動作
PDFやスキャン画像の構造化に革命を。

9. Jellyfin

  • 完全無料のセルフホスト型メディアシステム
  • データ追跡や隠れ課金なしでメディアを管理
「あなたのメディアは、あなたのもの」を実現。

10. Ultralytics (YOLO)

  • 最先端の画像認識AI (YOLO)を簡単に利用可能
  • リアルタイム性能に最適化され、エッジデバイスでも高速
専門家でなくても高度な画像AIが使える！
