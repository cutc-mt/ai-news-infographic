AGI UI Protocol解説

2025年10月5日

AGI UI Protocolとは？

  • AIエージェントとフロントエンドアプリを接続するための標準規格。
  • オープンソースで軽量なイベントベースのプロトコル。
  • ユーザー向けのUIを持つエージェントに焦点を当てている。
これからのエージェント駆動なWebの世界で重要になる！

主な特徴

  • フロントエンドツール呼び出し: UI上のボタンの色を変えたり、カードを表示したりできる。
  • 様々なバックエンドエージェント（MCP, A2A, CrewAIなど）と連携可能。
  • Next.jsベースのフロントエンドとFastAPIベースのバックエンドで構成。

ADKとの連携

  • GoogleのAgent Development Kit (ADK)と組み合わせて使用。
  • ADKが提供するマルチステッププランニング、ツール、状態管理、メモリ管理機能を利用。
  • LLMにはGemini 2.5 Flashを使用。

デモの概要

  • 「テーマを紫に変えて」と指示すると、UIのテーマカラーが変更される。
  • 「AIについての俳句を書いて」と指示すると、俳句を生成して表示。
  • 「SFの天気を教えて」と指示すると、天気を取得して表示。
エージェントがUIを直接操作する未来がすぐそこに！
