CEXCLI v0.44.0 アップデート解説

2025年10月5日

カスタムプロンプトの強化

  • 名前付き引数 ($VAR) と位置引数 ($1) をサポート。
  • プロンプトをテンプレート化し、再利用性が向上。
  • プロンプトは .codex/prompts フォルダに保存。
これでプロンプトの管理がもっと楽になる！

新コマンド `codex cloud`

  • CLIから直接Codexのクラウド環境を操作可能に。
  • クラウド上のタスクの確認や新規タスクの実行が容易に。
  • Web UIを開かずに、出先などからでも手軽に開発作業が可能。

Claude Codeとの比較

  • 知能: GPT-5 Codexへの信頼が厚い。複雑な実装はCodexが優勢。
  • 自律性: Claude Codeの自律性やスピード感も魅力的。
  • 完成度: アプリの完成度はClaude Codeが高いが、Codexも急速に進化中。
用途に応じてツールを使い分けるのが重要！

Claude Code v2.0.5

  • Rewind機能: 会話の履歴を巻き戻し、特定の時点からやり直せる。
  • サブエージェントを動的に追加可能に。
  • VS Code拡張機能はまだ使いづらさが残る部分も。
