Jules AI Agent: 最新アップデート解説

2025年10月5日

Pull Request連携

  • Pull Request内のコメントでJulesに直接修正を依頼可能。
  • @Jules とメンションすることで、Julesがコメントに反応。
  • 修正が完了すると、自動でPull Requestを更新。

ファイルセレクター

  • Julesに作業させたいファイルを正確に指定可能に。
  • 曖昧さをなくし、より細かい制御でタスクのコンテキストを限定。
  • アタッチオプションからファイルやフォルダを簡単に選択。

メモリ機能

  • 対話からユーザーの好みや修正内容を自動で学習。
  • 次回以降のタスクで記憶を呼び出し、より少ない指示で高精度な結果を出す。
  • リポジトリごとにメモリのON/OFFを切り替え可能。

Jules CLI

  • コマンドラインからJulesを直接操作。
  • Julesが作業中のコードをローカルに即座に適用し、テスト可能。
  • 他のツールと連携させ、スクリプト化や自動化を実現。
  • インタラクティブなダッシュボード（TUI）も提供。
