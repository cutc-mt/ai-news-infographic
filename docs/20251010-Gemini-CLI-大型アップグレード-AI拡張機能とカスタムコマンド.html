Gemini CLI 大型アップグレード

2025年10月10日

Gemini CLIとは？

  • Google製の強力なコマンドラインインターフェース(CLI)ツール。
  • 開発者がGemini 2.5 Proのような最新AIモデルと直接対話できる。
  • コーディング、自動化、デバッグ、AI支援ワークフローを効率化。
  • 完全無料で1日1000リクエストまで利用可能。
最もアクセスしやすく、高性能なAI CLIの一つ！

新機能：Gemini CLI拡張機能

  • Gemini CLIを完全にカスタマイズできる新フレームワーク。
  • お気に入りのツールやサービスをコマンドラインから直接接続。
  • アプリ間のコンテキストスイッチングが不要に。
  • MCP、コンテキストファイル、カスタムコマンドを軽量パッケージにバンドル。
ローカルスクリプトから外部APIまで、あらゆるツールの使い方をGeminiに教え込める！

拡張機能の導入方法

  • 公開されている拡張機能ストアから探せる。
  • GitHub URLやローカルパスから直接インストール可能。
  • 簡単なコマンドでインストール:
gemini extension install <GitHub URL or Local Path>
例えば「NanoBanana」拡張機能で、ターミナルから直接画像を生成できるように！

カスタムコマンドの強化

  • 反復的な操作を/コマンドで再利用可能に。
  • 複雑な指示を一行のシンプルなコマンドに変換。
  • Geminiエージェントにツールの適切な使い方をより良く指示できる。
  • 開発の速度、一貫性、生産性を劇的に向上させる。
ツール呼び出しが苦手なGeminiの能力を補う強力な機能！

インストールと始め方

  • 前提条件: Node.js 18以上が必要。
  • npmを使ってグローバルにインストール:
npm install -g @google/gemini-cli
  • Googleアカウントで認証後、すぐに利用開始。
gemini auth

実際の使用例

  • Chrome DevTools MCP拡張機能をインストール。
  • /extensionsコマンドで有効な拡張機能を確認。
  • AIエージェントにブラウザ操作を指示。
  • 例: 「このウェブページに移動して」と指示すると、新しいChromeでページを開く。
ターミナルからブラウザを直接操作し、開発ワークフローを加速！
