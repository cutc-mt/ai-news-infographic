週刊AIニュース

2025年10月10日

Gemini 3.0は来なかった

  • Googleのイベント「Gemini at Work」で期待されたが、発表されず。
  • 多くの開発者やAIウォッチャーが落胆。
  • 次期リリースに関するリーク情報も乏しく、登場時期は不明。
モデル性能をさらに向上させ、競合と圧倒的な差をつけてからリリースする戦略かも？

Gemini Enterprise発表

  • 企業向けの新しいAIエージェント構築プラットフォーム。
  • 全社データを活用し、大規模なワークフローエージェントを作成可能。
  • 特定のタスクに特化した「Opal」とは規模感が異なる。
  • 超巨大企業向けのソリューションという位置づけ。

Google Veo 3.1の噂

  • Googleの動画生成AI「Veo」のv3.1に関するリーク情報。
  • 最長30秒の動画生成が可能になるとの噂。
  • 解像度が向上し、より映画的な映像が作れる可能性。
あくまで噂レベルの情報であり、公式発表ではない点に注意！

Claude Code プラグイン

  • Claude Codeにプラグイン機能が新登場。
  • GPTsストアのように、自作プラグインを共有・利用可能。
  • 複数のプラグインを組み合わせて利用することも可能らしい。
  • 開発ワークフローの可能性を広げる注目の新機能。

GensparkでGPT-5 Proが無制限

  • AIプラットフォーム「Genspark」でGPT-5 Proが利用可能に。
  • 有料プランユーザーは無制限でアクセス可能。
  • API料金が非常に高価なモデルを無制限で提供できる理由は不明。
  • 昨日から既に利用可能になっており、話題を呼んでいる。

  • Fellow Slides 3.0: デザインを完全自動化する新バージョンがリリース。
  • Dia Browser: macOSで誰でもダウンロード可能に。
