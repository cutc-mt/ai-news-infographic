Gemini 2.5 Computer Useモデル完全解説

2025年10月11日

Computer Useとは？

  • ブラウザ操作を超えた広範な操作
  • アプリ操作の自動化を実現
  • コンピュータ自体を直接操作する概念
  • スクリーンショットで画面を認識
  • マウスやキーボードの操作を自動化
ブラウザユースよりもっと広く、PC操作全般を自動化する技術だよ！

仕組み

  • タスクが与えられる
  • スクリーンショットを撮影
  • モデルが次のアクションを判断 (例: クリック)
  • このプロセスをループしてタスクを遂行
  • 画面のリアクションを次のスクショで認識
見て、考えて、実行する。これを繰り返すんだね！

Gemini 2.5 Computer Useモデル

  • ベンチマークで高パフォーマンス
  • 画像認識能力が非常に高い
  • 現在OSレイヤーの操作は未サポート
  • Web操作で高いスコアを記録
  • 高速かつ高精度な動作
Geminiは元々画像認識が得意だから、この分野でも強い！

デモと実践

  • GoogleがPlaywright使用のデモを公開
  • Google AI StudioでAPIキーが必要
  • レートリミットと課金に注意
  • ブラウザ自動化のデモが試せる
  • BrowserBase上でGemini Browserをテスト可能
# Google公式デモの実行イメージ
pip install google-generativeai playwright
python computer_use_demo.py

Computer Useの未来

モデル詳細とアクション

  • 入力: 128k トークン (画像+テキスト)
  • 出力: 64k トークン (テキスト)
  • 利用可能なアクション
  • ブラウザ操作、クリック、タイピング、スクロール、ドラッグ＆ドロップなど
ドラッグ＆ドロップまでできるのは面白い！
