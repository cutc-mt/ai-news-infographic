Gemini CLI v0.7.0 & v0.8.0 アップグレード解説

2025年10月11日

v0.7.0の主な新機能

  • IDEプラグイン仕様: エディタ内での差分表示対応
  • Flutter拡張機能: アプリ開発を支援
  • Nano Banana拡張機能: 画像生成
  • 環境変数による柔軟なテレメトリ設定
  • 実験的なToDoリスト機能
  • share chatがツールコールを含むように
  • 全ユーザーで引用が有効に
  • カスタムコマンドがヘッドレスで動作可能に

v0.8.0: 拡張機能の正式発表

  • ギャラリーとシンプルな管理機能を備えて正式発表
  • GitHub URLやローカルフォルダからインストール可能
  • enable/disableで有効/無効を切り替え
  • listでインストール済み拡張機能の一覧表示
  • updateで拡張機能を更新
  • newで新しい拡張機能の雛形を作成
CLIがツールハブになる！

ドキュメントとホームページ

  • 新しいホームページが開設
  • ドキュメントが改善され、より分かりやすく
  • geminacle.comでアクセス可能
情報が整理されて、始めやすくなった！

その他の改善点

  • 非インタラクティブな許可済みツール
  • ターミナルのタイトルにステータスを表示
  • より速く、信頼性の高いインタラクション
  • クリーンになったUI
