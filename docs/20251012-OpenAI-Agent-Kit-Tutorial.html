OpenAI Agent Kit 使い方解説

2025年10月12日

Agent Kitとは？

  • ワークフローツール: OpenAIが提供する、AIエージェントを作成するためのビジュアルなワークフローツール。
  • ブロックベース: 「エージェント」や「ガードレール」などのブロックを繋げて、複雑な処理フローを構築。
  • APIとして利用可能: 作成したワークフローはAPIとして呼び出し、自分のアプリケーションに組み込むことができる。
N8nやMake.comのように、複数のAIエージェントを連携させた高度な自動化処理を、ビジュアルで簡単に作れるのが魅力です！

基本的な使い方

  • エージェントの作成: エージェントブロックを配置し、Instruction（システムプロンプト）や使用するモデル（GPT-5 Miniなど）を設定。
  • MCPの利用: Reasoning Levelを「Low」以上に設定すると、MCPサーバーをツールとして追加可能に。
  • RAG (File Search): ファイルをアップロードしてVector Storeを作成し、エージェントにその内容を検索・参照させることができる。

ガードレール機能

  • 入力チェック: ユーザーの入力に個人情報や不適切な言葉が含まれていないかなどをチェック。
  • ハルシネーション検出: エージェントの出力が、参照したファイルの内容に基づいているか（幻覚を見ていないか）をチェック。
  • if/elseロジック: ガードレールのチェック結果に応じて、処理を分岐させることが可能。
入力と出力の両方でガードレールを挟むことで、より安全で信頼性の高いAIエージェントを構築できます。

実践：複数エージェントの連携

  • File Researcher: RAGを使い、アップロードされたPDFの内容について回答するエージェント。
  • Summarizer: File Researcherの長い出力を受け取り、指定された形式（3〜5文、子供にも分かるように）で要約するエージェント。
  • コンテキストの連携: 前のブロックのアウトプットを、次のブロックのインプットとして簡単に渡すことができる。
