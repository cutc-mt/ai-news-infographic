Qwen Code v12-v14 アップデート解説

2025年10月12日

より安全な計画 (Safer Planning)

  • Plan Mode: 変更を適用する前に、完全な実装計画を提案。
  • 事前承認: ユーザーが計画を承認するまで、コードの変更は行われない。
  • Gemini CLIのフォーク: Gemini CLIをベースにしており、その影響も受ける。
CIの承認ゲートのように、ローカルでの変更も意図を先に確認できるので、チーム開発での安全性が向上しますね！

より賢い視覚 (Smarter Vision)

  • Vision Intelligence: 画像がペーストされると、自動でQwen 3VL+モデルに切り替え。
  • 巨大なコンテキストウィンドウ: 256kの入力と32kの出力トークンウィンドウで、長いコンテキストを維持しつつ、長い応答を生成可能。
  • 高解像度画像対応: Qwen 3VL+で高解像度画像がサポートされ、詳細を損なわない。
UIのスクリーンショットや図を貼り付けるだけで、モデルを手動で切り替える必要がないのは便利！

信頼性の向上 (Cleaner Reliability)

  • Zed連携: OpenAIとQwenの両方のOAuth認証をサポート。
  • 多数のバグ修正: Windowsでの複数行ペースト、Markdownリストのレンダリング、不正なツールコール、エスケープ文字の挿入など、多くの問題が解決。
  • パフォーマンス向上: サブエージェントのパフォーマンスとUIが改善され、よりスムーズな対話が可能に。

制御とガードレール

  • ループ検出: タスクが繰り返し始めた場合に検出する機能。
  • 上書き確認: 設定ファイルが既に存在する場合に、上書き前に確認プロンプトを表示。
重いポリシーファイルで全てを縛るのではなく、小さなコントロールで安全性を高めているのが良いですね。
