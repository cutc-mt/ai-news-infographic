OpenSpec: 軽量スペック駆動開発ツールキット

2025年10月15日

OpenSpecとは？

  • AI駆動開発のための軽量なスペック駆動開発フレームワーク。
  • 既存プロジェクトへの途中導入が容易。
  • 仕様の変更履歴管理に強く、進化し続けるプロジェクトに最適。
  • npmで簡単にインストール可能。
これまでのツールと違い、既存のコードベースを壊さずに導入できるのが嬉しい！

開発フロー

  1. Propose: AIに実装したい機能を提案。
  2. Review: AIが生成した仕様とタスクリストを人間がレビュー＆修正。
  3. Apply: 確定した仕様をAIに渡し、実装を開始。
  4. Verify & Archive: 実装を人間が確認し、問題なければアーカイブ。仕様書が自動更新される。
人間とAIが対話しながら、仕様を固めていくのがポイントですね。

使い方とコマンド

  • インストール: 
    npm install -g openspec-cli
  • 初期化: 
    openspec init
    プロジェクトのルートで実行。
  • 提案: 
    openspec propose "Add a new feature"
  • 適用: 
    openspec apply <spec_file>
  • アーカイブ: 
    openspec archive <spec_file>
コマンドはシンプル！すぐにでも始められそう。
