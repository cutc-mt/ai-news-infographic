週刊AIニュース 2025年10月前半

2025年10月15日

1. Google Opal 日本上陸

  • ワークフロー型のAIサービス。
  • ジェムをコード化したような、より高度な多段操作が可能。
  • 期待値を上げすぎるとがっかりするが、ジェムの高度化版として捉えると非常に有用。
  • Geminiのロングコンテキストやワンショット性能を活かせる。
まずは全員が抑えておくべき最重要ニュース！

2. Sora 2 & GPT-image-mini

  • Sora 2: 高品質なビデオを生成。ただしAPIコストが高いのがネック。
  • GPT-image-mini: 日本語が崩れず、高品質な画像を生成。
  • GPT-image-miniは低コストで、特にミディアムモデルがおすすめ。
  • サムネイル作成など、コンテンツの大量生産に革命を起こす可能性。
個人的にはGPT-image-miniがお気に入り！日本語に強いのが嬉しい。

3. Codex アップデート

  • Slack連携で、メンションするだけでDevinのように動作。
  • Codex SDKが登場し、アプリへの組み込みが容易に。
  • 現在のコーディングモデルではCodexが最もホットでオススメ。
開発の現場が大きく変わりそうなアップデート！

4. Claude Code プラグイン

  • マーケットプレイスから他人が作ったプラグインを導入可能に。
  • プラグインのON/OFFで開発環境を柔軟に切り替え。
  • サブエージェントやスラッシュコマンドなどをパッケージ化して共有できる。
  • ただし、Sonnet 4.5の性能評価が微妙なため、今すぐ乗り換える必要は薄いかも。

5. OpenAI Agent Kit

  • 開発者向けのツールで、Difyやn8nとは方向性が異なる。
  • AIエージェントをアプリに組み込みやすくするためのパーツを提供。
  • Agent Builder, Chat Kit, Registryの3つで構成。
業務自動化ツールではなく、エージェント開発を加速させるためのキット！

その他の注目ニュース

  • n8n: AIエージェントと協調してワークフローを構築する機能をクラウド版で提供開始。
  • AppsSDK: チャットUIにHTMLカードなどを埋め込みやすくするSDK。
  • Gemini Enterprise: チャットUIから社内データと連携する機能が登場。
  • Computer-use: ブラウザ操作やPC操作を行うモデルの性能が向上中。
