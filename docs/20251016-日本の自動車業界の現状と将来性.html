日本の自動車業界の現状と将来性

2025年10月16日

業界全体の大きな流れ

  • EV化への変革期。ただし、現状はハイブリッドが主流。
  • 車のスマホ化：ソフトウェア搭載で機能がアップデートされるのが当たり前に。
  • 地産地消：関税対策として、現地生産体制へのシフトが加速。
  • 安全と省エネの競争：自動運転技術とエネルギー効率化が重要テーマ。
IT化が進み、単なる製造業から変化している点がポイント！

トヨタ自動車

  • 追い風：世界的なハイブリッド需要の強さ。最強の総合力。
  • 向かい風：将来のEV専業メーカーとの競争激化への懸念。
  • 株価：75日移動平均線がサポートライン。3000円台定着が焦点。
「トヨタ生産方式」は製造業の教科書的存在！

本田技研工業

  • 追い風：二輪事業（バイク）が世界首位で安定収益源。北米での現地生産も強み。
  • 向かい風：量産販売体制強化のための先行投資コスト。経営判断が重要に。
  • 株価：高値圏にあり、方向感に乏しい。配当利回りの高さが下支えになるか。
万年割安株（バリュートラップ）の傾向に注意が必要。

日産自動車

  • 追い風：他社に先駆けたEV事業の実績と技術力。「技術の日産」復活なるか。
  • 向かい風：リストラ等のネガティブなブランドイメージの払拭が課題。
  • 株価：300円台から400円台への回復局面。400円台の売り圧力をこなせるかが鍵。
経営再建計画「Nissan NEXT」を着々と進行中。悪い情報を先出しする戦略も。

マツダ

  • 追い風：デザイン性の高さと「走り」への評価。選択と集中が上手い。
  • 向かい風：事業規模の小ささから、為替や部品コストなど外部環境の影響を受けやすい。
  • 株価：1000円の節目が重要。ここを固めて上昇できるか。
車好きから「職人肌メーカー」として支持される玄人好みのブランド。

関税と為替の影響

  • 関税：当初はメーカーが負担していたが、今後は価格転嫁や生産地の変更で対応。
  • 為替：各社140円〜145円を想定レートに設定。円高が進むと業績にマイナス影響。
レクサスのような高級車は日本から輸出し、価格競争力のある車種は米国で生産するなどの戦略が進行中。

総括と将来展望

  • 短期的：ハイブリッド車と現地生産で粘り強く対応。
  • 中長期的：EVシフトとソフトウェア化への対応が不可欠。
投資の際は、目先の動向に加え、次世代で戦うためのビジョンと実行能力があるかを見極めることが重要！

関連注目市場

  • サイバーセキュリティ：「車のスマホ化」に伴い、セキュリティの重要性が増大。
