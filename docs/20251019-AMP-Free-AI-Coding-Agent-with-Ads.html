AMP Free: 広告付きAIコーディングエージェントの登場？！

2025年10月19日

AMP Freeの衝撃

  • Sourcegraphが提供するAIコーディングエージェント「AMP」に無料ティア「AMP Free」が登場。
  • 広告表示と学習データ共有を条件に無料で利用可能。
  • AIコーディングエージェントとしては初の広告モデル導入で、業界で話題に。
  • 学習データ共有はQwen CodeやGemini CLIなど、他のツールでも一般的。
ついにAIツールにも広告の波が！無料ユーザーのマネタイズの新たな形か？

モデルとパフォーマンス

  • 無料ティアのモデルは非公開だが、Groq Code Fastが使用されている可能性が高い。
  • インターフェースは大幅に改善され、応答速度は非常に高速。
  • パフォーマンスは「悪くはないが、特筆すべきほどではない」。Groq Code FastはGodoには優れるが、他のフレームワークには不向き。
  • Kilo CodeではGroq Code Fastが広告なしで無料で利用できるため、AMP Freeの価値が問われる。
広告モデルは良いが、提供されるモデルがすでに無料で使えるものだと、ユーザーは魅力を感じにくいかも。

AIマネタイズの未来

  • AI企業にとって広告は無料ユーザーをマネタイズする上で避けられない流れとなる可能性。
  • モデルインフラを持つOpenAI, Anthropic, Googleなどの企業が広告モデルを導入する方が有利。
  • 多くのAI企業は、広告よりも学習データ収集とエンタープライズ向け有料サービスで収益化を目指す。
  • モデルの運用コストが高いため、広告収入だけでカバーするのは困難。
AIの無料サービスは、今後どのように収益化されていくのか？業界の動向に注目！
