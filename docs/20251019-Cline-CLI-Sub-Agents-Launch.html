Klein CLI & サブエージェント機能がついに登場！

2025年10月19日

Kleinの現状と進化の背景

  • これまでアップデートが遅く、Kiloなどの競合に遅れをとっていた。
  • 拡張機能のバックエンドとフロントエンドを分離するアーキテクチャの再構築を実施。
  • よりエンタープライズ向けのツールへとシフト。
  • 今回のアップデートで、より高速かつメモリ消費の少ないツールに進化。
遅れを取り戻すべく、内部構造から大きく変革！今後の進化に期待！

Klein CLIの登場

  • CI/CDワークフローへの統合を主な目的としたCLIツール。
  • ローカルでの利用も可能で、VS Code拡張機能と同等のエージェント機能を提供。
  • ターミナルとIDE間で状態を同期し、シームレスな作業が可能。
  • リモートコーディングエージェントを構築し、Slack, Discord, GitHub Actionsなどと連携。
開発ワークフローの自動化と連携を強化！エージェントの活躍の場が広がる！

サブエージェント機能で複雑なタスクを自動化

  • VS Code拡張機能から複数のサブエージェントを起動し、複雑なタスクを自動で実行。
  • Claude Codeのサブエージェント機能と同様に、タスクのオーケストレーションが可能。
  • サブエージェント機能の利用にはKlein CLIのインストールが必須。
  • ローカルモデル向けにコンパクトなシステムプロンプトオプションを提供。
複数のAIが連携して一つの目標を達成！より高度なAIアシスタンスが期待できる！
