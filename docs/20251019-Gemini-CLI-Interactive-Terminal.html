Gemini CLIが神アップデート！対話型ターミナルで開発が激変

2025年10月19日

v0.9リリース：対話性の革命

  • Vim, top, git rebase などの対話型コマンドをGemini CLI内で直接実行可能に。
  • ターミナルを切り替える必要がなくなり、シームレスなワークフローを実現。
  • 全ての操作がGeminiのコンテキスト内に留まるため、AIが文脈を完全に把握。
  • 無料で利用でき、Claude Codeに対する強力な選択肢に。
もうフリーズを恐れなくていい！AIとの対話が途切れないのは最高！

新技術：疑似ターミナルとシリアライザ

  • node-ptyライブラリを使用し、バックグラウンドで疑似ターミナルを生成。
  • 新しいGeminiシリアライザがターミナルの状態をリアルタイムでスナップショット化。
  • スナップショットをストリーミングすることで、静的なテキストではなく、ライブ感のある対話的なフィードを提供。
  • 完全な双方向対話が実現し、キー入力やウィンドウのリサイズにも即座に対応。
まるでターミナルの画面をビデオストリーミングしてるみたい！技術的にも面白い！

その他のアップデートと未来

  • 拡張機能の作成がより簡単に。
  • OpenTelemetryによるGenkitメトリクスで、相互運用性が向上。
  • /memory listコマンドで長期記憶ファイルの一覧表示が可能に。
  • バージョン1.0のリリースが間近に迫り、Gemini 3.0も12月に登場するとの噂。
Canva拡張でターミナルからデザインも！？Gemini 3.0との連携が楽しみすぎる！
