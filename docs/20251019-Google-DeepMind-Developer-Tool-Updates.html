Gemini CLI の進化

  • エクステンション（拡張機能）が追加され、さらに拡張性が向上。
  • MCP、エクステンション、カスタムツールを組み合わせて、高度な環境構築が可能。
  • デフォルトで検索機能が組み込まれているのが特徴。
  • 初心者から上級者まで満足できる、奥の深いツール。
Claude Codeに負けない人気！やりこみ要素満載で、自分だけの最強CLIを構築できる！

Chrome DevTools との連携

  • DevTools内でGeminiと直接チャットできるように。
  • MCPサーバー機能を搭載し、他のAIツールとの連携が容易に。
  • AIアシスタントと一緒に、ペアプログラミングやデバッグが可能。
  • 現在、利用するには言語設定を英語にする必要あり。
ブラウザ上で直接AIと会話しながらデバッグできるなんて、開発効率が爆上がり！

Gemini Code Assist エージェントモード

  • 単一の指示でコードベース全体を理解し、自律的に計画・実行。
  • 複数ファイルにまたがる新機能実装や大規模リファクタリングに対応。
  • 開発者はAIが提示した計画をレビューし、承認するだけ。
  • VS CodeとIntelliJで利用可能。

NotebookLM の超パーソナライズ機能

  • ソース資料からポッドキャスト風のAI音声概要を生成。
  • 動画解説機能も搭載し、内容の理解を深める。
  • AIに役割（ペルソナ）を設定し、振る舞いをカスタマイズ可能。
  • コンテンツ生成や自己学習など、幅広い用途で活用できる。
