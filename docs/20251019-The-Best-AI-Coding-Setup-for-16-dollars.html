最強の組み合わせ：2つのサービス

  • GLM Coding Plan：$3〜$6の低コスト。コーディングとデザインに優れたGLM 4.6モデルを利用可能。
  • GitHub Copilot：$10で多様なモデル（GPT-5 Mini, Sonnet, GPT-5 Codecなど）にアクセス。プランニングやデバッグに強力。
合計たったの$13〜$16で、Cursorよりも安価で強力な開発環境が手に入る！

Kilo Codeでの設定方法

  • VS CodeマーケットプレイスからKilo Codeをインストール。
  • 「Z AI」プロファイルを作成し、GLMのAPIキーを設定。
  • 「Copilot」プロファイルを作成し、プロバイダーとしてVS Code LM APIを選択。
  • これにより、Kilo Codeのインターフェース内でCopilotのモデルをシームレスに利用可能に。
使い慣れたインターフェースを離れることなく、複数の強力なAIモデルを切り替えられるのが最高！

モード別おすすめモデル

  • Architect (設計)：GPT-5 Codec または Sonnet。高度なプランニングに。
  • Editor (編集)：常にGLM 4.6。優れた編集、ツールコール、デザイン能力。
  • Ask (質問)：GPT-5 Mini。クレジットを節約しつつ、気軽に質問。
  • Debug (デバッグ)：GPT-5 Codec。Sonnetよりも優れたデバッグ能力。
各タスクの得意なモデルを使い分けることで、開発ワークフロー全体が最適化される！
