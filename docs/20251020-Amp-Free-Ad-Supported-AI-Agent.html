AIコーディングの新時代？
広告付き無料AIエージェント「Amp Free」登場

2025年10月20日

「Amp Free」とは？

  • Sourcegraph社のAIコーディングエージェント「Amp」に新しく登場した無料プラン。
  • 無料でAIコーディング支援が受けられる。
  • 代わりに、エディタ内に広告が表示される。
  • 入力したコード等のデータはモデルの学習に利用される。
ついにコーディングAIも広告モデルへ！無料で使える新時代の到来か？

「Amp Free」の仕組み

  • ユーザーが得るもの:
    無料のAIコーディングエージェント機能。
  • ユーザーが提供するもの:
    ① 開発ツール関連の広告表示の受諾
    ② モデル学習のためのデータ共有
広告とデータを提供する代わりに、無料でAIの力を手に入れるトレードオフの関係です。

広告とプライバシー

  • 広告は開発ツール関連のテキストベースで、作業の邪魔になりにくい。
  • ユーザーの利用状況に基づかない非パーソナライズ広告がランダムに表示される。
  • データ利用はモデル改善目的で、多くの無料AIツールと同様の仕組み。
広告は控えめ。でも、業務コードなど機密情報の共有には注意が必要かもしれません。

パフォーマンスと使用モデル

  • 応答速度は非常に高速。
  • 使用モデルは非公開だが、速度から「Grok Code Fast」と推測される。
  • 「Grok Code Fast」はGodotエンジンには強いが、他のWebフレームワークでは平凡との評価。

価値はある？厳しい評価

  • 問題点: Amp Freeが使うモデルは、他で広告なし・無料で既に利用可能。
  • もし…: GPT-5やClaudeのような高級モデルが無料で使えるなら、広告があっても価値は高い。
  • 疑問: なぜ広告を見てまで、既に無料のモデルを使う必要があるのか？
コンセプトは面白いけど、中身のモデルが残念…？今後の改善に期待！

AIの未来のビジネスモデル

  • 広告モデルは、AI業界の今後の主流になる可能性がある。
  • OpenAIやGoogleのようなモデル開発元自身が採用すれば、より成功するかもしれない。
  • モデルの運用コストが高いため、広告収入だけで賄うのはまだ難しいのが現状。
