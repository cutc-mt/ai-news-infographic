Claude Code on the Web

  • クラウドベースのコーディング環境。
  • GitHubリポジトリに接続し、ブラウザ上から直接コーディングタスクを実行。
  • Webブラウザだけでなく、スマホアプリからも利用可能。
  • 現在、ProとMaxユーザー向けに提供中。
ChatGPTのCodexのように、Claudeの画面から直接コードを扱えるようになりました！

デモ：戦国ゲームの改良

  • 題材はシンプルな戦国シミュレーションゲーム「群雄将伝」。
  • 変更前: 12人の大名のうち、3人（信長、秀吉、家康）しかプレイできない。
  • 目標: 残りの9人も含め、全12人の大名でプレイできるようにする。
  • UI改善: 3つのボタンを「ゲーム開始」ボタン1つにまとめ、モーダルで大名を選択する形式に変更。

実行プロセス

  • 1. コード読解: Claudeがゲームのindex.htmlを読み込み、全体の構造を把握。
  • 2. 現状確認: READMEを作成させ、現状（3人しかプレイ不可）を正しく理解しているか確認。
  • 3. 指示: 「残り9人もプレイ可能にし、UIをすっきりさせて」と自然言語で指示。
  • 4. 自動実装: Claudeが自律的に計画を立て、HTML、CSS、JavaScriptを修正。

驚きの結果

  • 実装成功: 指示通りにゲームが完全に修正された。
  • タイトル画面が「ゲーム開始」ボタン1つになり、クリックすると12人の大名が並ぶ選択モーダルが表示されるように。
  • 選択したどの武将でも正しくゲームが開始できることを確認。
自然言語の指示だけで、UIの改善からゲームロジックの変更まで自律的に完遂！
