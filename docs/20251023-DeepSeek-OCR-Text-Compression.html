DeepSeek-OCR: テキスト圧縮でLLMの記憶限界を突破

2025年10月23日

DeepSeek-OCRとは？

  • DeepMindがリリースした新しいOCRモデル。
  • 画像や写真から文字を読み取るVLM（Vision Language Model）の一種。
  • 最大の特徴はOCR性能より、革新的なテキスト圧縮メソッドにある。
この技術の真価は、LLMの記憶容量を飛躍的に増大させる可能性にあります！

Optical 2D Mappingによる圧縮

  • スクリーンショットのようにテキストを画像として保存。
  • 驚異的な圧縮率: 1/10のサイズに圧縮しても、97%の精度でテキストを復元。
  • 1/20のサイズでも60%の成功率を維持。
  • これにより、LLMのコンテキストウィンドウを10倍以上に拡大できる可能性。

モデルの仕組み

  • 1. パッチ分割: 入力画像を16x16の小さなパッチに分割。
  • 2. ローカルアテンション: 80Mパラメータのモデルで各パッチを処理。
  • 3. エンベディング: 300MのCLIPモデルでビジョントークンを埋め込み。
  • 4. テキスト生成: 3BのDeepSeekモデルが最終的な出力を生成。
複数の専門モデルを組み合わせた、合計約4Bパラメータの強力なアーキテクチャです。

未来へのインパクト

  • トークンからピクセルへ: コンテキストの保存方法がテキストから画像に変わる可能性。
  • LLMの記憶限界という大きな課題を克服する一歩に。
  • 今後のさらなる研究開発が期待される分野。
