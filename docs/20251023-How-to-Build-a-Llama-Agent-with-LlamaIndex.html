LlamaIndexでLlamaエージェントを構築する方法

2025年10月23日

Llamaエージェントとは？

  • Llama Indexの新しいクラウドプラットフォームLlama Cloud上の新製品。
  • バックエンドやフルスタックサービスとしてワークフローを構築・デプロイ可能。
  • 現在はプライベートアルファ版として提供。
アイデアを素早く形にし、クラウド上で実行できる強力なツールです！

プロジェクト：害虫駆除の異常検知

  • 農作物を食べる害虫の個体数を監視。
  • 異常な個体数増加を検知し、生産性の低下を防ぐ。
  • 過去の正常な個体数データと現在のデータを比較して異常を判断。

ワークフローの3ステップ

  • 1. データ処理: CSV文字列をPolarsデータフレームに変換。
  • 2. 可視化と保存: ヒートマップを生成し、Supabase（S3互換）にアップロード。
  • 3. 異常検知: データをベクトル化し、Quadrantで類似データを検索。距離が遠ければ「異常」と判断。

実装とデプロイ

  • uvでPythonプロジェクトをセットアップ。
  • llama-index-workflowsでワークフローのロジックを定義。
  • lamactl CLIツールでローカルプレビューとデプロイを実行。
  • GitHubリポジトリと連携し、Llama Cloudに直接デプロイ。
コードを書いてGitHubにプッシュするだけで、簡単にワークフローをクラウドに展開できます！
