KiloCode CLI - AIコーディングエージェント

2025年10月25日

KiloCode CLIとは？

  • ターミナルで動作するAIコーディングエージェント
  • VS Code拡張機能のCLI版
  • 日本語に完全対応
  • CursorやCode Llamaの良いとこ取り
計画から実装まで、開発を高速化する相棒！

強力な機能

  • モード切替：Architect, Code, Debugなど
  • モデル切替：数百のLLMを自由に選択
  • 自律モード：CI/CDでの自動実行
  • BYOK：OpenRouter等のキーを持ち込み可能
タスクに応じて最適なモードとモデルを使い分け！

推奨LLMプロバイダー

  • 無料：OpenInference (OpenRouter)
  • 速度重視：Groq (超高速・低コスト)
  • 量重視：GLM Coding Plan (大容量)
  • 品質重視：KiloCodeクレジット (GPT-5/Claude 4.5)

実演：ゲーム開発

  • ふわっとした自然言語の指示からスタート
  • Architectモードでゲームの計画を立案
  • 計画承認後、自律的にコーディングを完了
  • 簡単な修正指示でパドルのサイズを調整
  • ほぼワンショットで高品質なゲームが完成！
GLM-4.6でも驚きの成果！ツールの優秀さが光る！
