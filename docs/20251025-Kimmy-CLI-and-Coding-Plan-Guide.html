Kimmy CLI & Coding Plan

2025年10月25日

Kimmy CLIとは？

  • PythonベースのオープンソースAIコーディングエージェント
  • Kimmyモデルに最適化されており、モデルの能力を最大限に引き出す
  • ACPプロトコル対応でZedやJetBrainsに統合可能
Kimmyモデル開発元が作った、最強の相棒！

Coding Plan

  • 月額$20から利用可能なサブスクリプション
  • 週間クオータ制で柔軟な利用が可能（例：$20で週248クエリ）
  • APIキーはKilo, Rue, Kleinなど他のツールでも利用可能
Kimmyファンならずとも注目の高コスパプラン！

注目機能

  • シェル/エージェントモード：Ctrl+Kで一発切替！メインシェルとしても使える
  • スラッシュコマンド：/compact, /debugなど便利なコマンドが豊富
  • ツール呼び出し：Kimmyモデルの強力なツール呼び出し能力をフル活用
  • /release-notes：アップデート内容をCLI内で即座に確認

他プランとの比較

  • GLM：ほとんどのタスクでより安価な選択肢
  • Synthetics：$20でより多くのモデルと高い使用上限を提供
  • Kimmy公式：Web UIや独自ツール、CLIとの最適化が魅力
あなたの使い方に合ったプランを選ぼう！
