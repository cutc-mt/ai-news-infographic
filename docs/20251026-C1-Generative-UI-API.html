C1 by Thesis: AIアプリのUXを革命する生成的UI API

2025年10月26日

現在のAIアプリの課題

  • ほとんどのAIアプリはテキスト入力、テキスト出力が中心。
  • 直感的でなく、ユーザーの離脱を引き起こしやすい。
  • ハードコードされたUIは、動的なモデル出力に対応できない。
  • 新しいユースケースごとに手動でのUI更新が必要。
モデルは進化しているのに、UXが追いついていないのが現状です。

解決策: C1 API

  • LLMとフロントエンドの間に位置するAPIミドルウェア。
  • テキストの代わりに、対話的なUIをリアルタイムで生成。
  • 「生成的UI API」として機能し、UI開発のボトルネックを解消します。
  • 10倍速くリッチなAIアプリを開発可能に。

C1の仕組み

  1. バックエンドがモデルの応答をC1 APIに送信。
  2. C1がデータを解析し、システムとUIルールを適用。
  3. JSONベースのUIスキーマを生成し、フロントエンドにストリーミング。
  4. C1 React SDKがリアルタイムでUIをレンダリング。
ストリーミング、状態管理、エラー処理もSDKが担当してくれるので楽々です！

主な特徴

  • データを視覚化する動的なインターフェースを生成。
  • あらゆるフォームファクタに適応し、ブランドのデザインに一致。
  • ツールコールをサポートし、外部APIやデータベースと連携可能。
  • テンプレートやロックインなしで、ショッピングエージェントなども構築可能。

高い互換性

  • GPT, Claude, Geminiなど、あらゆるLLMに対応。
  • MCPサーバーとも互換性あり。
  • Vercel AI SDK, Copilot Kit, AGUIなどのUIコンポーネントと連携。

はじめ方

  • テストプロジェクトを一行で開始:
npx create-c1-app
  • 既存アプリへの統合:
  • 1. バックエンドのエンドポイントをThesisに向ける。
  • 2. ThesisのAPIキーを追加。
  • 3. C1 React SDKをインストールし、ロジックを更新。
サインアップで500万無料トークンがもらえるチャンスも！
