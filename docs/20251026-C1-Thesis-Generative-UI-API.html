C1 by Thesis: AIアプリのUXを革命する生成的UI API

2025年10月26日

C1 (Thesis) とは？

  • AIアプリ開発を簡素化する新しいフレームワーク。
  • LLMからのテキスト応答を、リッチでインタラクティブなUIに自動で変換。
  • 開発者はバックエンドのAPIをC1に向けるだけで、複雑なUI実装から解放されます。
「AIの応答をいい感じのUIで表示したい」という悩みを解決！

仕組み

  • アプリのバックエンドがC1 APIにリクエストを送信。
  • C1がLLM（GPT-5など）と通信し、結果を統一フォーマットで受信。
  • C1が提供するUIキット（ChatUI）が、そのフォーマットを元にUIを自動生成。
// OpenAI APIのベースURLをC1に変更するだけ！
const openai = new OpenAI({
  apiKey: process.env.C1_API_KEY,
  baseURL: 'https://api.c1.com/v1',
});

主な特徴

  • GPT-5, Claude Sonnetなど複数のモデルに対応。
  • Vercel AI SDKなどの主要フレームワークと簡単に統合可能。
  • Web検索、画像検索、天気情報などのツールを標準搭載。
  • プレイグラウンドでプロンプトやUIをインタラクティブに試せる。

注意点

  • LLMのAPIを直接利用するより、コストが若干高くなる。
  • OpenRouterには非対応。
  • 現時点ではGeminiやGroqはサポート外。
MVP開発でUI実装コストを抑えたい場合に特に有効です！
