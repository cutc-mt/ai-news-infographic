Microsoft Copilot大型アップデート速報

2025年10月28日 | Windows 11 & Edge AI機能強化

第1章: Windows 11のAI化

  • 全てのWindows 11 PCがAI PCへ。対話・共同作業の実現を目指す。
  • 新しいCopilot: AIチャットがリニューアル。直近のファイルやアプリ、会話を表示。
  • Copilot Vision: メガネボタンでアプリ画面を共有し、使い方などを質問可能。
  • 音声会話機能も搭載。マイクボタンでCopilotと話せる。
  • Copilot Actions: ファイル操作やタスク自動化をCopilot経由で実行（Insiderで試験中）。
  • Copilot Connectors: OneDriveやGoogle Calendar等をCopilotから直接操作（Insiderで試験中）。
「ヘイ、コパイロット」と話しかけるだけで、小型Copilotが起動！いつでもどこでもAIアシスタントを呼び出せるように！

第2章: Copilotの新機能

  • 音声モード: 新キャラクターが登場し、より自然で魅力的な音声対話を実現。
  • メモリ機能: ユーザーの発言を記憶し、以降の会話に活用。「豚骨ラーメンが好き」と伝えると、ラーメンの話題で豚骨を勧めてくれる。
  • アプリ接続: Gmail、Google Calendar、Google Driveと接続可能に。Copilot内でメール要約や予定確認ができる。
  • Copilot 4 Health: 健康に関する質問に、ハーバードヘルスなどの信頼できる情報源を基に回答。
メモリ機能は設定画面からいつでも削除可能。プライバシーも安心！ 
// メモリ機能の例
User: 「私が豚骨ラーメンが好きだと覚えておいて」
Copilot: 記憶しました。
...後日の会話...
User: 「おすすめのラーメンは？」
Copilot: 「豚骨ラーメンがお好きでしたね。〇〇はいかがでしょう？」

第3章: Edgeブラウザの新機能

  • Copilot Actions for Edge: 複雑なタスク（ホテルの予約、フォーム入力）をAIが自動実行。ユーザーは別の作業に集中できる。
  • Copilot Journeys: 閲覧履歴を記憶し、中断した作業にすぐ戻れるようにサポート。
  • これらの新機能は米国で先行提供中。日本のユーザーはウェイトリストに登録可能。
旅行の計画が超簡単に！「東京のホテルを予約して」と頼むだけで、Copilotが全部やってくれる未来がすぐそこに。

まとめ: MicrosoftのAI戦略

  • オープン戦略: これまでのOpenAI依存から脱却し、自社開発モデルも含め「最高のAIモデル」を取り入れる方針へ。
  • Windows 11、Copilot、EdgeというMicrosoftエコシステム全体で、一貫した強力なAI体験を提供。
  • 紹介された多くの機能が無料で利用可能。AIの民主化を加速。
PCの操作が「命令」から「対話」へ。AIがもっと身近なパートナーになる、大きな一歩ですね！
