Minimax M2 AIモデル 詳細レビュー

2025年10月28日 | エージェント機能に優れた高コスパモデル

モデル概要とスペック

  • Minimax M2: コーディングとエージェント機能に特化した、Mixture of Expertsモデル。
  • 総パラメータ数230B、アクティブパラメータは10B。
  • コンテキスト長は200K。最近のモデルとしてはやや物足りない印象。
  • オープンソースとしてHugging Faceでリリース予定。ローカル（M4 Max/Ultra等）での動作も期待される。
M1モデルは456Bだったので、約半分に小型化！より手軽に試せるようになりそうですね。

価格と利用方法

  • OpenRouterで11月7日まで無料で利用可能！
  • 価格: 入力$0.3/Mトークン、出力$1.2/Mトークン。Sonnetの約1/10という驚異的な安さ。
  • Web UI (agent.minimax.io) も提供されており、Googleアカウントでログインしてすぐに試せる。
  • API経由で、CursorやVS CodeのAIエージェントとしての利用が期待される。
無料期間は限定的！気になる方は今すぐOpenRouterで試してみるのが吉です！

ベンチマークと性能評価

  • Artificial Analysisのベンチマークでは、Claude 4.5 Sonnetに次ぐ高スコアを記録。
  • エージェントコーディングのベンチマークTerminal-Benchでは、GLM 4.6を僅かに上回る性能。
  • 一般的なコーディングベンチマークCode-Benchでは、Grok-1.5 Fastとほぼ同等のスコア。
  • 知識を問うQAタスクでは、85%〜89%の正答率を発揮。
ベンチマークはあくまで参考値。オーバーフィットの可能性もあるため、実際のタスクで試すことが重要です。

実践テストと結論

  • 独自のバグ修正ベンチマークでは、42%の成功率。これはSonnet 4.5に次ぐ結果。
  • 面白いことに、思考（Thinking）レベルを上げると逆にスコアが下がる現象が発生。Baseモデルが最も性能が良かった。
  • Web UIでのスペースインベーダーゲーム作成テストでは、計画を立て、スクリーンショットで自己テストしながら完璧に動作するゲームを生成。
  • 結論: Claude 4.5 Sonnetに匹敵する品質を、オープンモデル（予定）かつ低価格で実現した非常に優秀なモデル。特にエージェント機能の完成度が高い。
中国ベースのモデルである点を考慮する必要はあるものの、この性能と価格は魅力的。試す価値大です！
