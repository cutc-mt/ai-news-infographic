GitHub Copilot大型アップデート！エージェント機能が神進化！

2025年10月29日

エージェントHQ構想

  • すべてのエージェントを単一プラットフォームに統合
  • VS Code InsidersからOpenAI Codexや他のエージェントにタスクを割り当て
  • Devin、Gemini CLI、Claude Codeなどとの連携の可能性
複数のAIエージェントをVS Codeから一元管理！

GitHub Agents

  • クラウド上でタスクを実行する「プチDevin」
  • 専門のエージェント機能が登場
  • カスタムエージェントを作成し、特定のタスクに集中させることが可能
クラウドの力で、より複雑なタスクを自動化！

Slack & Linear連携

  • SlackでGitHubボットにメンションし、タスク実行とプルリクエスト作成
  • タスク管理ツールLinearからCopilotにタスクをアサイン

CLIの神進化

  • CLIからクラウド上のCopilot Coding Agentにタスクを渡せる
  • パフォーマンスが向上し、並列呼び出しも可能に

その他の注目機能

  • Copilotがコードレビューエージェントとして利用可能に
  • `agents.md`でエージェントの振る舞いを制御
  • 新しい料金プラン「Copilot Pro Plus」が登場（1500リクエスト/月）
