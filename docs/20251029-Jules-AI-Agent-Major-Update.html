Jules AI Agent Major Update

2025年10月29日

Jules API

  • プログラムからJulesの機能にアクセス
  • 開発ワークフロー全体で強力な自動化と統合を実現
  • Slackと連携してチャットOpsを構築
  • JiraやLinearなどのプロジェクト管理ツールと連携
# Simple curl command to start a new session
curl -X POST https://api.jules.ai/v1/sessions \
-H "Authorization: Bearer YOUR_API_KEY" \
-d '{ "prompt": "Fix the bug in the login page" }'
API経由で、独自のシステムやCI/CDパイプラインにJulesを組み込める！

Jules Tools (CLI)

  • タスクとセッションの作成・管理
  • コミット前にパッチをローカルでテスト
  • インタラクティブなターミナルダッシュボード
  • npmまたはmpxで簡単にインストール可能
CLIを通じて、より速く、より柔軟にJulesを操作できる！

File Selector

  • AIエージェントが作業するファイルを正確に制御
  • コードベース全体をスキャンする代わりに、特定のファイルを手動で選択
  • デバッグ、ドキュメント作成、編集対象を限定

Environment Variables

  • リポジトリレベルの環境変数を提供
  • ビルドの実行、テストの実行、外部サービスとの連携に必要なプロジェクト固有の構成にアクセス
  • リポジトリ設定で変数を追加し、タスク開始時に有効化

Jules's Memory

  • 対話から学習し、好み、修正、ワークフローの習慣を保存
  • ニーズを予測し、確立されたコーディングパターンに従う
  • リポジトリ設定のナレッジセクションで機能のオン/オフを切り替え

Image Upload

  • JPEGまたはPNG画像をアップロードして視覚的なコンテキストを提供
  • フロントエンドのバグ、UIモックアップ、デザインのインスピレーションに最適

Stack Diff

  • 複数のファイルの変更を単一画面に垂直に表示
  • コードベース全体の変更を一目で確認
  • 従来のタブ付きdiffビューに切り替えるオプションも

Jules Critique

  • UIにリアルタイムの思考プロセスを表示
  • コードを評価する方法を段階的に表示
  • 潜在的なバグやロジックの欠陥を提示

Jules & Stitch Integration

  • StitchでUIをデザインし、Julesに渡してバックエンドを自動構築
  • フロントエンド設計から完全な機能コード生成までのシームレスなワークフロー
