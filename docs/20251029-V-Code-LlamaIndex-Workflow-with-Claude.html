V-Code a LlamaIndex Workflow with Claude

2025年10月29日

プロジェクト設定

  • `uv`プロジェクトを初期化
  • `vblama`をインストールしてエージェント用ドキュメントをダウンロード
  • `vblama starter`で`cloud-code`、`llama-cloud`、`llama-index-workflows`、`llama-deploy`のドキュメントを取得
  • `text-classification`と`llama-usage`スキルを選択

Claudeによるワークフロー生成

  • ClaudeとチャットしてLlamaIndexワークフローを生成
  • 財務書類（キャッシュフロー計算書、損益計算書、貸借対照表）を分類するタスクを指示
  • Claudeが分類ロジックのみを含むPythonスクリプトを生成
Claudeがドキュメントとスキルを理解し、自律的にコードを生成！

パッケージ化とデプロイ

  • ClaudeにワークフローをLlamaエージェントとしてパッケージ化するように依頼
  • `pyproject.toml`に`llama-deploy`設定を追加
  • プロジェクトをGitHubリポジトリにプッシュ
  • `lamactl serve`でワークフローをローカルで提供
  • `lamactl deployments create`でLlama Cloud Platformにデプロイ
