Geminiの新機能：プロンプトでGoogleスライドを作成

2025年10月31日

Geminiキャンバスでスライド作成

  • Geminiの「キャンバス」ツールをオンにして利用
  • 作成したいスライドの内容を指示するだけで、プレビューが自動生成
  • ワンクリックでGoogleスライドにエクスポート、またはPDFでダウンロード可能
面倒なスライド作成が、プロンプト一つで完了する時代に！

高品質なスライドを作るコツ

  • Googleドキュメントで詳細な構成案を作成し、添付する
  • コーポレートカラーやデザインのイメージをプロンプトで具体的に指示
  • 繰り返し使う指示は「ジェム」として保存し、作業を効率化
「山口式スライド」のように、プロンプトを工夫すればクオリティはさらに向上！

利用可能状況

  • 10月28日に発表され、有料プランユーザーから段階的に展開中
  • 11月12日までに全有料ユーザーが利用可能になる予定
  • 無料アカウントや教育機関向けプランでも将来的に展開予定

活用のポイント

  • 公式機能は画像やアイコンが正常にエクスポートされるのが大きな利点
  • 細かい修正は、Googleスライドにエクスポートしてから行うのがおすすめ
  • 簡単なスライドなら公式機能、凝ったデザインなら従来の「山口式」と使い分けも
Googleスライドで管理したいなら、もう公式機能一択！
