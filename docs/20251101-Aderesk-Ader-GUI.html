Aderesk

伝説のAIコーダー「Ader」に強力なGUIを搭載した、新しいオープンソースツール

Adereskの主な特徴

  • グラフィカルインターフェース: CUIが基本だったAderを、直感的に操作できるGUIで利用可能に。
  • ワンクリックインストール: PythonやRustなどの複雑な環境構築は不要。クリック一つで簡単に導入できます。
  • Aderコアの継承: Gitを意識したファイル編集エンジンなど、Aderの強力な機能はそのまま利用可能です。
  • マルチプロジェクト管理: ブラウザのタブのように、複数のプロジェクトを同時に開いて作業できます。
  • タスク管理: プロジェクト内で複数のエージェントを起動し、異なるタスクを並行して実行させることが可能です。

独自のAIエージェント

Adereskは単なるAderのGUI版ではありません。Vercel AI SDKをベースにしたカスタムAIエージェントを搭載しています。

  • 計画と実行の分離: 先にカスタムエージェントがタスクの計画を立て、その計画をAderが得意なコード生成・編集に引き渡すという分業体制をとっています。
  • 柔軟なエージェント設定: エージェントごとに使用するツールやルール、コンテキストを細かく設定できます。
  • モデルの使い分け: メインの強力なモデル（コーディング用）と、軽量なモデル（コミットメッセージ生成など）を使い分けることで、コストと効率を最適化します。

多彩な操作モード

Adereskには、タスクに応じて切り替え可能な複数の操作モードが用意されています。

Agent Mode Code Mode Ask Mode Architect Mode Context Mode
Agent Mode: ファイルの読み書きを自律的に行い、プロンプトに基づいて自動でコンテキストを追加する、最も強力なモード。

Architect Mode: 計画モデルが先に変更計画を立て、それを編集モデルが実装するという、より精度の高い開発が可能なモード。
