Nemotron-4 12B VLMとは？

  • 120億パラメータを持つ強力なビジョン言語モデル。
  • TransformerとMambaのハイブリッドアーキテクチャを採用し、高速処理を実現。
  • 画像、動画、PDFなど多様な入力形式に対応。
ローカルでもAPI経由でも使える柔軟性が魅力！

利用方法

  • Hugging Faceから8bit/16bitモデルをダウンロード可能。
  • ローカル実行にはNVIDIA GPUが必要。
  • GPUがない場合でも、NVIDIAのクラウドAPIを通じて利用できる。
  • APIキーを生成し、PythonやNode.jsから簡単に呼び出し可能。
# PythonでのAPIコール例

高度な機能と応用例

  • リーズニング機能を有効にすると、より深く、文脈を理解した回答を生成。
  • OCR機能が非常に優秀で、手書き風文字やデザイン文字も高精度で抽出。
  • 動画内のオブジェクトや場所（例: トルコ）を特定し、内容を要約。
  • PDFの決算資料から26%といった具体的な数値を抽出し、テーブル形式で出力可能。
