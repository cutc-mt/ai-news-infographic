Sora 2をn8nで自動化！AIエージェントで動画生成を効率化

2025年11月1日

Sora 2の課題

  • 高コスト: 10秒の動画生成に約150円、高画質だと450円〜750円かかる。
  • ウォーターマーク: 生成された動画にSoraのロゴが入ってしまう。
  • 手間: 解像度を上げる作業や、生成を待ってやり直す手間がかかる。
Sora 2はすごいけど、手軽に使うにはまだハードルがあるんだね。

AIエージェントで解決

  • n8nで動画生成ワークフローを自動化。
  • ロゴを自動で除去し、視認性を向上。
  • 高解像度化も自動で実行。
  • 生成→確認→再生成のループを効率化。
AIエージェントを作って、面倒な作業は全部自動化しちゃおう！

KAI APIでコスト削減

  • OpenAIの公式APIより約1/6のコストでSora 2を利用可能。
  • 価格比較表でもKAI APIの安さは圧倒的。
  • n8nからKAI APIを呼び出して、安価に動画を生成する。
同じSora 2を使うなら、安いほうがいいに決まってる！

n8nワークフロー構築

  • HTTP RequestノードにcURLをインポートしてSora 2 APIを叩く。
  • APIキーを正しく設定し、認証を通す。
  • 必須パラメータとオプショナルパラメータを確認。
    • // cURLのインポートで設定を簡略化
curl 'https://api.example.com/...' -H ...
  • remove_watermarkをtrueに設定。

テキストから動画生成

  • プロンプトに生成したい動画の内容を記述。
  • 例："Cute dachshund teaching English"
  • HTTP POSTリクエストで動画生成タスクを開始。
  • 生成された動画のURLを取得してダウンロード。
アイデアを言葉にするだけで、動画が作れるなんてすごい！

画像から動画生成

  • 商品画像などを基にCM動画を生成。
  • キャラクター（例：ダックスフンドの友）が商品を宣伝する動画を作成。
  • 静的な画像を自然な動きのある動画に変換。
商品の魅力を伝えるCM動画も、AIエージェントが作ってくれる時代に。

カメオ機能

  • 特定の人物（例：サム・アルトマン）を動画に登場させる機能。
  • プロフィール名を指定して、その人物の動画を生成。
  • 自分自身を登録すれば、自分の分身AIに動画を作らせることも可能。
有名人や自分自身を動画の主役にできるなんて、夢が広がる！

ワークフローのポイント

  • Waitノード: 動画生成には時間がかかるため、一定時間待機する。
  • Ifノード: 動画生成のステータスを確認し、処理を分岐させる。
  • ステータスが "success" になるまで、10秒ごとに確認を繰り返すループを構築。
  • この構造により、非同期の動画生成処理を確実にハンドリングできる。
