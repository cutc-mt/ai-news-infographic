Sora 2とn8nで動画生成を自動化するAIエージェント

Sora 2の動画生成をより効率的かつ低コストで実現する方法

Sora 2の課題

  • 高コスト: 10秒の動画生成に約150円、高画質版では450円〜750円かかる。
  • ロゴの表示: 生成された動画にSoraのロゴ（ウォーターマーク）が入ってしまう。
  • 手作業での高解像度化: 映像の解像度を上げるのに手間がかかる。
  • 待ち時間: 生成を待って、気に入らなければ再度生成し直す必要があり、時間がかかる。

AIエージェントによる解決策

Sora 2とノーコードツールn8nを組み合わせることで、これらの課題を解決するAIエージェントを構築できます。

  • ロゴの自動除去: ウォーターマークを自動で消去し、視認性を向上。
  • 自動高解像度化: 動画のアップスケールを自動で行う。
  • ワークフローの自動化: 生成から加工までの一連の流れを自動化し、時間と手間を削減。
  • コスト削減: KAI API経由でSora 2を利用することで、公式APIより大幅にコストを削減。

自動化のキーポイント: KAI API

このAIエージェントの構築には、Sora 2を安価に利用できるKAI APIを使用します。OpenAIの公式APIと比較して、約1/6〜1/10のコストで動画を生成できる場合があります。

n8nでのエージェント構築ステップ

  1. HTTPリクエスト(POST): KAI APIに動画生成のタスクを依頼します。プロンプトや、ウォーターマーク除去(`remove_watermark: true`)などの設定をここで行います。
  2. Waitノード: 動画生成には時間がかかるため、一定時間（例: 10秒）待機するノードを挟みます。
  3. HTTPリクエスト(GET): 生成タスクのステータス（処理中、成功など）を問い合わせます。
  4. IFノード: ステータスが「success」になったかどうかを判定します。
  5. ループ処理: 成功するまで、Wait → GET → IFの処理を繰り返します。
  6. 成功後の処理: 生成された動画のURLを取得し、ダウンロードや次の処理（高解像度化など）へ繋げます。
