課金AIサービスまとめ

2025年11月2日

ChatGPT Pro

  • $20/月 (Plus), $200/月 (Pro)
  • 思考モデルで高精度な回答
  • 20万文字のコンテキストウィンドウ
  • Web検索が非常に優秀
  • Google検索の代わりに利用
情報収集のメインツール！思考モデルで精度も抜群！

Gemini AI Pro

  • ¥2900/月 (Pro)
  • 難しい文章の解説が得意
  • 会話形式での説明が分かりやすい
  • 100万トークンのコンテキスト
  • Googleドライブ2TB特典
論文や専門記事の読解に最適！

Claude Max

  • $100/月 (Max)
  • コーディング能力が非常に高い
  • 思考モデルが高速
  • 筋トレメニュー管理など日常でも活用
  • Notion連携で外部ドキュメント参照
日常のチャットから開発まで、メインで活躍！

Cursor

  • $20/月
  • コーディング時の予測変換が便利
  • 最近は利用場面が減少気味
  • v2.0のアップデートに期待
予測変換は手放せないが、他のツールも強力に！

Codex CLI & Claude Code

  • 最も使っている開発ツール
  • Codex CLI: 複雑な実装も正確
  • Claude Code: 新しいことの学習やドキュメント整理に
  • TMAXで複数起動して並列開発
開発効率を最大化する最強の組み合わせ！

AquaVoice

  • $9.99/月
  • 高精度な音声文字起こし
  • 画面情報を理解して精度を向上させるDeep Context機能
  • プログラミング時の英語混じりの会話も正確
開発者にもおすすめ！音声入力がストレスフリーに！

Arc Browser

  • 無料（招待制）
  • ウィンドウ分割機能が便利
  • 複数AIモデルの出力を比較検討
  • AI搭載ブラウザだが、分割機能がメイン
AIの比較検証に欠かせないブラウザ！
