Atis: Agentic AI Code Editor ガイド

2025年11月3日

Atisとは？

  • 超軽量で生産性の高いエージェント型AIコードエディタ。
  • オープンソースで、Vimモード、Git統合、React、TypeScript、Tauriで構築。
  • 高速、カスタマイズ可能、インテリジェントな開発環境を求める開発者向け。
  • VS Codeクローンではなく、ゼロから構築されたユニークなエディタ。
肥大化した機能を排除し、速度と柔軟性を追求！

簡単なインストール

  • Mac OS, Windows, Linuxに対応。
  • GitHubリポジトリから簡単にインストール可能。
  • 最新機能を使いたい場合は、Change Logから各OS用のインストーラーをダウンロード。
  • Windowsの場合は.exeインストーラーを実行するだけ。
ベータ版で問題が発生した場合は、公式のインストール方法を試してみて！

AI機能のセットアップ

  • 右上のAIチャット機能で、最先端のモデルを利用可能。
  • 設定からAPIプロバイダーを選択（OpenAI, Open Routerなど）。
  • APIキーを設定し、使用したいモデルを選択。
  • Google Geminiの無料APIティアを利用するのがおすすめ！
  • AIチャットやAI補完などの機能を有効にすることを忘れずに。
APIキーの入力場所は、設定をスクロールダウンしてドロップダウンメニューを開くと見つかるよ！

主な機能

  • Vimモード: 高速な編集のための完全なキーバインディング。
  • Git統合: ブランチ、コミット、差分管理をエディタ内で完結。
  • シンタックスハイライト: 主要なプログラミング言語を美しく表示。
  • 拡張機能ストア: コーディング、データベース、テーマなど様々な拡張機能。
  • 画像編集: エディタ内で直接画像を編集可能。
  • グローバル検索: コードベース全体を高速に検索。
  • SQLightビューア: 組み込みのビューアでデータベースを直接確認。

開発とロードマップ

  • 一人の開発者によって開発されている。
  • 「生産準備完了」とされているが、まだ新しいプロジェクト。
  • 今後のバージョンではエージェント型AI編集にさらに注力。
  • 拡張機能ストアにさらなる拡張機能を実装予定。
ゼロからエディタを構築するのは非常に困難。Zedエディタも完成までに数年かかった。
