Code Supernovaサービス終了と代替モデル

2025年11月3日

Code Supernovaの終了

  • Kilo Codeプラットフォームで2番目に多く使われたモデル。
  • コーディングタスクの主要な選択肢だった。
  • 高価なモデルを使いたくない場合の代替案として人気。
多くの開発者が利用していたため、大きな変化だよ！

代替モデルの提案

  • 無料: Grok Code Fast One
  • 安価: GPT-5 Mini
  • ハイブリッド: Grok単体、またはGrokとGPT-5 Miniの組み合わせ。
ベンチマークテストでは、TypeScript、Bun、Esculiteを使ったジョブキューシステムで比較されたんだ。

Grok Code Fast Oneの評価

  • 性能はCode Supernovaと同等。
  • よりクリーンで短いコードを生成。
  • API設計が優れており、パラメータが分離されていて拡張性が高い。
  • スケジューリングがミリ秒で直接指定でき、シンプル。
  • JSONを自動的にパースしてくれるため、手動での処理が不要。
小さな改善だけど、開発体験が向上するね！

ハイブリッドセットアップ

  • GPT-5 Mini (Architect Mode): システム全体の計画を担当。
  • Grok Code Fast One: 計画の実行を担当。
  • 結果として、単体モデルよりもはるかに構造化されたコードが生成された。
  • リトライ機構、インデックス、エラーハンドリングなど、本番向けの機能も計画に含まれる。

コストとワークフロー

  • GPT-5 Miniは安価で、キャッシュを利用するとさらにコスト削減。
  • Grokが無料であるため、中小規模のタスクでは実質ゼロコストも可能。
  • 計画駆動型ワークフロー: 最初に計画し、次に実装する、という実際の人間の開発者に近い流れ。
  • 小さな修正はGrok単体、大規模な開発はGPT-5 Miniとのハイブリッドが推奨される。
$10のチャージで$30分のクレジットがもらえるプロモーションも実施中！
