FlowisOS フローモード完全解説

2025年11月4日

Flowモードとは？

  • 無限のキャンバスを持つ「作業スペース」です。
  • 画像生成、動画生成、チャット、リサーチなど、様々なAIタスクを実行できます。
  • アイデアを自由に広げながら、AIと共にクリエイティブな作業を進めるための空間です。
まさにAI時代のホワイトボード！アイデア出しから実行まで、すべてがこの中で完結します。

全自動 vs AIとの協業

  • 全自動にこだわる必要はありません。現状では複雑なタスクには限界も。
  • Flowモードは、人間の作業を補助し、助けてくれる存在として活用するのが鍵です。
  • AIを「部下」や「アシスタント」のように使い、面倒な作業を任せることができます。
「これやっといて！」とAIに頼む感覚。人間はより創造的な部分に集中できます。

Flowモードの使い方

  • サイドメニューの「フロー」からアクセスします。
  • 一般、検索、画像/ビデオ生成、比較など、様々なモードを切り替え可能。
  • GPT-5, Gemini 2.5, Claude 4.5など、最新のAIモデルを選択して利用できます。
利用可能なモデル例：
- GPT-5 / Pro
- Gemini 2.5 Pro / Flash
- Claude 4.5 Opus / Sonnet
- Midjourney, Sora 2, Veo 3.1

実践例：動画をTikTokへ

  • Flowモード内でVeo 3を使い、BGM付きの動画を生成。
  • 生成した動画をPCにダウンロードします。
  • 新しいタスクを起動し、動画ファイルを添付。
  • 「この動画をTikTokにアップロードして」と指示を出すだけで、説明文やハッシュタグ付きで投稿が完了します。
クリエイティブな作業（動画生成）と、単純作業（アップロード）をAIと分担する好例です！
