Goose AI Agent: オープンソースAIエージェント徹底解説

2025年11月4日

Goose AI Agentとは？

  • オープンソースでローカルで動作するAIエージェント
  • Rust製で高いパフォーマンスを誇る
  • 特定のLLMに依存せず、好きなモデルを利用可能
  • CLI、デスクトップ、Webの3つのインターフェースを提供
開発者にとって自由度が高く、カスタマイズしやすいのが魅力！

主な特徴

  • 複雑なタスクを自動で分割し、効率的に処理
  • 豊富な拡張機能で、機能を自由に追加可能
  • スケジュール機能で、タスクを定期的に自動実行
  • Goose Hintsファイルで、エージェントの振る舞いを事前設定
  • レシピ機能で、タスクをテンプレート化し再利用

拡張機能 (Extensions)

  • Developer Tools: シェルやGitなど、開発に必要なツールへのアクセスを提供
  • YouTube Transcript: YouTube動画の文字起こしを自動化
  • Cogni: エージェントに永続的な記憶層を追加し、過去の対話を記憶
  • その他: Chrome Dev Tools, Blender連携など、コミュニティによる多くの拡張機能

レシピとサブレシピ

  • プロンプト、モデル、オプションをYAMLファイルに定義し、タスクを再利用可能に
  • サブレシピを組み合わせることで、エージェントのチームを構築し、協調作業を実現
  • タスクの並列実行を指示することで、処理を高速化
旅行プランナーのレシピに、天気予報と観光情報を取得するサブレシピを組み合わせるなど、高度な連携が可能です！

導入と使い方

  • curlコマンド一発で簡単にインストール
  • goose configureでLLMプロバイダーとモデルを設定
  • gooseで新しいセッションを開始
  • goose sessionで過去のセッションに再接続
# インストール
$ curl -sSL https://goose.run/install | sh

# 設定
$ goose configure

# 新規セッション
$ goose
