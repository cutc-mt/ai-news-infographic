2025年11月8日

AIニュースまとめ

1. Codex Miniリリース

  • コストが安くなり、従来のCodexの4倍利用可能に。
  • 性能は3%低下するが、許容範囲内。
  • 処理速度が向上。
  • ほとんどのユーザーが利用可能。
  • 複雑なタスクには従来のCodex、簡単なタスクにはCodex Miniと使い分けを推奨。

2. GPT-5.1リリース日

  • 11月24日リリースとの噂。
  • あくまで噂レベルであり、確定ではない。
  • Gemini 3のリリース時期との関連が注目される。

3. NotebookLMアップデート

  • GoogleドライブからPDFやスプレッドシートを直接インポート可能に。
  • 以前はファイルフォルダからのみインポート可能だった。
  • 「近日公開」の機能。

4. Coder (QwenのCLI版)

  • OpenSpecをサポート。
  • OpenSpecの詳細は不明だが、AI開発における標準化された形式。
  • 人気があり、バージョン管理や人間によるレビューが可能。

5. JetBrains IDE向けCoderプラグイン

  • JetBrains IDE (WebStormなど) でCoderプラグインが利用可能に。
  • 期間限定でComposer 1が無料で利用可能。

6. マインドキャプショニングAI

  • 脳の活動を文章に変換するAIを研究者が開発。
  • 発話能力を失った人々の意思疎通を可能にする。
  • 倫理的な問題や一般普及までの道のりは長いが、非常に興味深い技術。
