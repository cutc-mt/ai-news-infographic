マルチエージェント＆マルチモデル

  • 最大8体: 1つの指示に対し、最大8体のエージェントが並列実行。
  • 異なるLLM: 各エージェントにGPT-5, Gemini, Claudeなど異なるモデルを割り当て可能。
  • 自動最適化: タスク内容に応じて、最適な強みを持つモデルを自動で選択・利用。
まるで優秀なAI専門家チーム！タスクに最適なエージェントが自動で働いてくれる感覚。

3つの新モード

  • エージェントモード: AIが自律的に作業を進める万能モード。
  • プランモード: 実装前に、AIと対話しながら開発計画を作成するモード。
  • アスクモード: コードの解説やエラーの意味など、シンプルな質問・相談用のモード。
目的に応じてAIとの対話方法を切り替え、よりスムーズな開発体験を実現！

デモ：シューティングゲーム開発

  • 1. 計画 (プランモード): AIと対話し、ゲームの仕様や実装手順をマークダウンで計画。
  • 2. 実装 (エージェントモード): 計画に沿ってHTML, CSS, JSのコードを自動生成。
  • 3. 修正 (コンポーザー): 高速なコンポーザーモデルを使い、敵への命中エフェクトを爆速で追加。
計画→実装→修正の全工程をAIとシームレスに連携して進められる！
