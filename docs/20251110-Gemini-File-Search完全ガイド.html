RAGシステムを簡単・安価・高速に構築

Gemini File Searchとは

2024年11月7日にリリースされた、GoogleのRAG（Retrieval-Augmented Generation）システム構築ツール

特徴: PDFやMarkdownなどのファイルをアップロードし、その内容を参照しながらGeminiが回答するRAGシステムを簡単に構築できる
  • 簡単: わずか数行のコードで実装可能
  • 安価: エンベディングが100万トークンあたり$0.15のみ
  • 高速: 即座にファイルをアップロードして利用開始
  • 便利: 複数ファイルに対応

対応ファイル形式

PDF

最大100MBまでのPDFファイルをアップロード可能

Markdown

Markdown形式の文書を直接アップロード

テキストファイル

あらゆるテキストファイルに対応

複数ファイル

複数ファイルを同時に参照可能

セットアップ方法

ステップ1: APIキーの取得

Google AI Studioから無料でAPIキーを取得

ステップ2: APIキーの設定

Google Colabのシークレット機能を使用して安全に保管:

  • 名前: GOOGLE_API_KEY
  • 値: 取得したAPIキー
ステップ3: ファイルのアップロード
  • PDFファイルをColabにアップロード
  • ファイルパスをコピー
  • コードに貼り付け
ステップ4: コードの実行

提供されたコードを実行してファイルをアップロード・エンベディング

ステップ5: Geminiで利用

Gemini 2.5 FlashまたはProでプロンプトを送信

実装例

コードは非常にシンプルで、概要欄のリンクから完全な実装コードが入手可能

注意: 公式ドキュメントのコードには一部誤りがあるため、動画で紹介されている修正済みコードを使用することを推奨

基本的な流れ

# 1. APIキーの設定
# 2. ファイルのアップロード
# 3. エンベディングの実行
# 4. Geminiモデルの選択 (Flash/Pro)
# 5. プロンプトの送信
# 6. レスポンスの取得

レスポンス内容

  • テキスト: Geminiの回答
  • 引用: 参照したPDFの該当部分
  • ソース: どのファイルから情報を取得したか
