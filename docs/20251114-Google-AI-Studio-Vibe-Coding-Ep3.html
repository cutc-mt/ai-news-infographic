  • 連動Google検索API活用

アプリコンセプト

ジャンルと視聴者レベルを選ぶだけで、Google検索結果を分析して、YouTubeの動画企画ネタを10個提案してくれるアプリだよ！
ユーザー入力は2つだけ
  • ジャンル（例：Blender）
  • 視聴者レベル（初心者/中級者/上級者）

デモ実演

  • 例「Blender」×「初心者」で検索
  • 「AIが企画を考えています...」
  • Google検索結果を参考中
  • 10個のネタが即座に生成される
処理時間 検索→分析→生成まで数十秒で完了

生成企画例

Blender初心者向けネタ
① インストールから日本語設定まで
② 最初に覚えるべき基本操作3選
③ サイコロで3D基本を体験
  • 企画意図まで詳しく説明
  • そのまま使えるタイトル
  • 視聴者の悩みに寄り添う内容

1ウェブ検索

Google検索ワード例
  • 「ジャンル 初心者 よくある悩み」
  • 「ジャンル 始め方」
  • 「ジャンル つまずきやすいポイント」
リアルタイムのトレンドや悩みを検索エンジンから収集するのがポイント！

2検索結果分析

  • 抽出よく出てくる悩み
  • 発見よく語られるテーマ
  • 理解視聴者が知りたいポイント
  • 整理3〜7個のポイントに要約
AIの役割 検索結果から視聴者ニーズを的確に抽出

発見された悩み例

  • 機能が多くて難しそう
  • 基本操作でつまずく
  • 初期設定で戸惑う
  • チュートリアルが多すぎて選べない
  • 完璧主義で挫折しやすい
  • ショートカットキーが覚えられない

3企画ネタ生成

ネタ生成ルール
  • YouTubeでそのまま使えるタイトル
  • 誰に何を届けるか明確に
  • 視聴者レベルに合わせて難易度調整
  • パターンをバランスよく混ぜる

企画タイプ

  • 解説系基本知識を伝える
  • 体験系実際にやってみる
  • 比較系複数の選択肢を提示
  • 失敗談注意喚起
  • まとめ系ランキング形式
同じパターンばかりにならないよう、バランスよく企画を混ぜるのがコツ！

プロンプト構造

  • コンセプトアプリの目的を明確に
  • ユーザー入力2つだけ入力させる
  • 処理フロー3ステップを詳細に
  • 生成ルールタイトルの作り方指示
  • 企画タイプバリエーション指定
  • 出力形式フォーマットを定義

Google Search設定

ビルドモードでの設定手順
  1. 「Use Google Search data」を選択
  2. チェックマークを確認
  3. プロンプトウィンドウに添付される
  4. 作成したプロンプトを貼り付け
ポイント 第2回ではImagenを使用。今回はGoogle Searchに変更するだけ！

プロンプト作成法

AIとブレインストーミング
優秀なAIモデルと対話しながらアイデアを具体化していく方法がおすすめ！
  • アプリアイデアを伝える
  • AIと対話して詳細を詰める
  • 「プロンプトを作って」と依頼
  • 生成されたプロンプトをコピペ

推奨AIモデル

ハイエンドモデルを使う
  • OpenAIGPT-5 / GPT-5.1
  • AnthropicClaude Sonnet 4.5
  • 推奨Claude Opus 4もOK
プロンプト生成には優秀なモデルを使うことで、バッチリのプロンプトが作れる！

ワンショットのコツ

重要な一文を追加
「どんなアプリが出来上がるかテストしたいので、1回のプロンプト入力（ワンショット）でアプリが完成するような指示が必要です」
効果 この一文を最後に加えることで、AIが包括的で詳細な指示を生成してくれる

プロンプト詳細度

  • 長文詳細な指示が必須
  • 処理ステップごとに明確に
  • ルールタイトル作成方法まで指定
  • 多様性企画タイプの指示も重要
今回のプロンプトは長いけど、それだけ詳細な指示を出しているから、ワンショットで完璧なアプリができる！

ビルド実行

実行手順
  1. Google Search dataを添付
  2. プロンプトを貼り付け
  3. 「ビルド」ボタンをクリック
  4. アプリ開発が自動スタート
開発時間 プロンプト1本でアプリが完成！

アプリの実用性

  • YouTuberネタ切れ解消
  • リサーチトレンド把握が簡単
  • 視聴者ニーズを的確に捉える
  • 効率化企画作成時間を大幅短縮
ジャンルと視聴者レベルを選ぶだけで、リサーチから企画まで全自動！YouTubeクリエイターの強い味方になるアプリだね
