動画解説新機能

  • 自由なテキスト指示が可能

動画解説の使い方

設定手順
  1. ノート右側の「動画解説」ボタンをクリック
  2. ペンアイコンを押す
  3. 形式：説明動画/概要を選択
  4. 言語：日本語に設定
  5. 「カスタム」を選択
  6. スタイルをテキストで入力

焦点設定機能

「AIホストが焦点を当てるべきこと」で、解説動画のどこに焦点を当てるかを指定できる！
  • 重要ポイントを強調
  • 特定トピックに絞る
  • 学習者向けに最適化
出力 生成した動画はダウンロード可能

Deep Research概要

自動リサーチ機能
面倒で複雑なネット検索を自動化！専属リサーチ担当のように働いてくれる
  • 範囲数百のウェブサイトを閲覧
  • 出力整理されたレポート生成
  • 時間数分で調査完了

Deep Researchの仕組み

  • 1リサーチ計画を立てる
  • 2数百のウェブサイトを自動巡回
  • 3必要に応じて検索内容を調整
  • 4情報を整理してレポート作成
特徴 検索範囲を指定サイトに限定することも可能

Deep Researchの強み

Gemini Deep Researchと似ているけど、NotebookLMは「レポート作成→ノート追加」がシームレス！
  • 統合レポートをノートに即追加
  • ソース元情報もまとめて保存
  • 並行調査中も他作業が可能

Deep Researchの使い方

操作手順
  1. 「ノートブックを新規作成」をクリック
  2. 「ソースを探す」ボタンを押す
  3. 「Deep Research」タブを選択
  4. 調査テーマを入力して送信
  5. 数分後に調査完了
  6. 「インポート」でノートに追加

調査例

例：お米の価格調査
「2025年における日本の米の価格が高騰している原因を調べてください」
