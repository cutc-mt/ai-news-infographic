Vertex AI Agent Engine 入門

ローカルのAIエージェントを、本番環境のスケーラブルなアプリケーションへ

開発者の課題: 「本番の壁」

  • ローカルで完璧に動くAIエージェント。でも、どうやって本番提供する？
  • コンテナ化、セキュリティ、認証・認可、スケーラビリティ、監視... 専門知識がなければ難しい。
  • この「本番環境へのギャップ」が、多くのプロジェクトのボトルネックになっている。

解決策: Agent Engine

  • AIエージェントのデプロイ・管理・スケールを担うマネージドサービス。
  • ADKやLangGraphなど、既存のフレームワークと連携して動作。
  • 開発者はインフラを気にせず、エージェントのロジック構築に集中できる。

Agent Engineの主要コンポーネント

ランタイム

エージェントをデプロイ・スケールする中核。コンテナ化やセキュリティ、監視機能を提供。

品質・評価

エージェントの品質を測定し、継続的にパフォーマンスを最適化するためのサービス。

コンテキスト管理

Session（短期記憶）とMemory Bank（長期記憶）で、対話の文脈を管理し、パーソナライズを実現。

Example Store

Few-shot példákat adhatsz meg, hogy finomhangolás nélkül irányítsd az ügynök viselkedését.

コード実行

安全なサンドボックス環境で、エージェントが生成したコードを実行。データ分析や複雑な計算に利用。
