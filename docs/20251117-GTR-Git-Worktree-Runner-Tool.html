覚えるコマンド3つ

gtr new [branch]、gtr editor [name]、gtr ai [name]の3つだけ覚えればOK！
3 コマンドだけ

gtr new [branch]

新しいWorktree作成！自動でファイルコピー・npm install・npm run build実行
  • 自動.envファイルコピー
  • 自動パッケージインストール

gtr editor [name]

指定したWorktreeでエディタ起動！Cursor・VS Code・Zedなどに対応
  • 対応Cursor・VS Code
  • 対応Zed

gtr ai [name]

指定したWorktreeでAIエージェント起動！Claude Code・Codexに対応
  • 対応Claude Code
  • 対応Codex

Hooks機能

post-createフックでWorktree作成後に自動コマンド実行！npm install・buildを自動化
  • タイミング作成後自動実行
  • 削除時pre-deleteフック

設定ファイル

.gtr-configで初回設定！エディタ・AIエージェント・コピーファイルを指定
  • 設定gtr config実行
  • ファイル.gtr-config生成

環境ファイルコピー

.envなど環境変数ファイルを自動コピー！AIエージェントが即座に開発開始可能
  • .env環境変数ファイル
  • 設定copyFiles指定

見るだけでOK

パッケージインストールも自動！Worktree作成後は見るだけ・レビューするだけ
  • 体験待ち時間削減
