ADKでMCPサーバーを構築する完全ガイド

MCPサーバーとは

ADKツールをMCPサーバーでラッピングして、あらゆるMCP対応クライアントから使えるようにする！

ADK以外のフレームワークのエージェントもあなたのツールにアクセス可能に

MCPライブラリ

Pythonライブラリがサーバー構築のフレームワークを提供
  • サーバーフレームワーク
  • ADKツールとの統合

ADKツールの準備

Function ToolでPython関数をラップするか、既存のADKツールを使用

どんなADKツールでもMCPサーバー経由で公開可能

list_toolsハンドラ

クライアントに利用可能なツールを通知する最初のハンドラ

ツールの発見と一覧表示を担当

call_toolハンドラ

クライアントがツールを呼び出した時に実行する2番目のハンドラ

実際のツール実行を処理

接続メカニズム

  • STDIO: ローカル開発用
  • Streamable HTTP: 本番環境用

スケーラブルなネットワーク通信が可能

Async IO

ADKとMCPライブラリの両方がPythonのAsync IOを使用

サーバーコードは非同期ファーストで実装

ユースケース1: API MCPサーバー

load_web_pageツールをAPIとして公開

エージェントがWebページを取得できるようになる

サーバースクリプト作成

  • my_adk_mcp_server.py
  • list_toolsでツール公開
  • call_toolで実行

テスト方法

2つ目のADKエージェントをクライアントとして起動

ADK Webでもテスト可能。init.pyファイルを追加して親ディレクトリから実行

ユースケース2: データベースサーバー

エンタープライズデータへのアクセスが必要なエージェント向け

カスタムDB統合をゼロから書く必要なし！

MCP Toolbox for Database

データベースをMCPツールとして公開するオープンソースサーバー

サービスとしてデプロイし、ADKエージェントをMCP2Setで接続

安全なクエリと分析

MCP抽象化レイヤーを通じて安全にデータをクエリ

セキュアなデータアクセスを実現

ユースケース3: カスタムサーバー

標準カテゴリに当てはまらないツールを公開
  • Imagen（画像生成）
  • Veo（動画生成）

社内API統合

プライベートREST APIをラップしてMCPクライアントから利用可能に

内部APIをエージェントに公開

カスタムロジック

数学計算などの独自ロジックをツールとして定義

ビジネスロジックをMCPツール化

Fast MCP

MCPサーバー構築を簡素化するヘルパーライブラリ

デプロイを簡単にし、開発を加速
