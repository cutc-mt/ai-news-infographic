FlowithOS: AIエージェントOS完全ガイド

2025年11月20日公開

FlowithOSとは？

  • AI OSの新しい形: AIブラウザの概念を超え、AIエージェントが思考し行動できる新しい環境を提供。
  • 多機能ツール統合: ブラウザやターミナル操作など、高性能ツールを多数搭載。
  • 汎用AIエージェント能力: ウェブ操作の問題解決において、Google GeminiやOpenAI Operatorを凌駕するベンチマークスコアを記録。
AIエージェントが賢く、より自律的に動くためのOSだよ！

主な特徴

  • ターミナル操作: ローカルのターミナルをAIが使用し、コストパフォーマンスに優れる。
  • スキル機能拡張: 様々なツールを組み合わせて、より複雑なタスクを解決。
  • プリセット機能: 頻繁に行うタスクをテンプレートとして登録し、ワンクリックで実行可能。
  • タスク完了通知: OSスクリプトと連携し、タスク完了をデスクトップ通知。
ブラウザだけでなく、ターミナルもAIが直接操作できるのがすごい！

自動化と効率化

  • 航空券検索の自動化: 複雑な検索ロジックをAIが実行し、最安航空券を自動発見。
  • ショッピング支援: Amazonでのギフト探しなど、AIが最適な商品を検討し購入プロセスを支援。
  • 情報収集レポート作成: Product HuntやHacker Newsなどから情報を収集し、自動でレポートを生成。
  • 日常業務の自動化: 定型的な作業をFlowithOSに任せ、自分の時間を創出。
人間がポチポチしていた作業をAIが肩代わりしてくれるから、超効率的！

使い方と実践例

  • チャットUI: AIエージェントとの対話を通じてタスクを依頼（プロ・ファストモード）。
  • GitHubリポジトリのクローン: 指定URLからリポジトリをクローンし、ブラウザで表示。
  • 生成ファイルの確認: 作成されたマークダウンファイルやログをアプリ内で閲覧可能。
コード例: 2048ゲームのクローンと表示 
git clone https://github.com/gabrielecirulli/2048.git
cd 2048
# HTMLファイルをブラウザで開くコマンドをAIが実行
start index.html
具体的な指示でAIにPC操作をさせられるよ！プロンプトが重要！

プロンプトの最適化

  • 場所 (WHERE): どのサイトやツールを使用するか明確に指定。
  • やること (WHAT): 具体的なタスク内容を記述。
  • 条件 (HOW): 注意点やフィルタリング条件を詳細に設定。
  • 成果物 (OUTPUT): レポート形式や通知方法など、最終的な出力を指定。
良いプロンプトは「場所・やること・条件・成果物」を明確にすること！

セキュリティと注意点

  • 機密情報保護: 銀行口座など、AIに操作させたくないサイトにはログインしないよう注意。
  • 利用環境: AIブラウザを使用する際は、セキュリティリスクを理解し、慎重に利用する。
  • ヒューマン・イン・ザ・ループ: 複雑な処理の途中で人間が介入できる仕組みも重要。
便利だけど、大切な情報にはアクセスさせないように気をつけようね！
