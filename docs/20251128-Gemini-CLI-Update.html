Gemini CLI大型アップデート

Gemini 3.0 Pro搭載で革命的進化

2025年11月28日

Gemini 3.0 Pro統合

Gemini CLIが最新Gemini 3.0 Proモデルを搭載。コーディング能力が飛躍的に向上し、完全無料でフルプロジェクト生成が可能に。

3分でゲーム開発

通常数週間かかるゲームを3分で生成。フルプロジェクト生成、エージェントコーディング、マルチファイル推論の組み合わせで実現。

フルプロジェクト生成

高レベルな説明から完全な実行可能ビルドを生成。アーキテクチャ、アセット、最適化を自動処理。3D Three.jsデモやフルスタックアプリも。

ゴールデンゲートブリッジ

3Dビジュアル、シェーダー、UIコントロール、最適化されたシーンロジック。視覚的に美しいシミュレーションをHTML一つで実現。

ビジュアルからアプリへ

スケッチを分析してコンポーネント、レイアウト、スタイリングを認識。完全なHTML、CSS、JavaScriptを即座に生成して機能的なアプリに変換。

自然言語でシェルコマンド

複雑なマルチステップのターミナルタスクを自然言語で実行。git bisectの自動実行、ターミナル出力を明確な英語に翻訳。

自動ドキュメント生成

コードから完全に正確なユーザー向けドキュメントを自動生成。関数、パラメータ、アプリロジックを分析して整理された検索可能なドキュメントを作成。

クラウドサービスデバッグ

Cloud RunやSyncなど複数ツールを連携してエンドツーエンドでデバッグ。パフォーマンス問題の根本原因を特定、修正提案、デプロイまで自動化。

改善されたインタラクティブシェル

カスタムローディングフェーズでユーザー入力待ちが明確に。Control+F/Command+Fでコマンドにフォーカス。混乱を防ぎワークフローがスムーズに。

コピー&ペーストモード改善

テキスト選択時のビジュアルヒントが明確に。/clearコマンドが画面を完全にクリア。複数行プロンプト処理の基盤も整備。

UIオーバーホール

レンダリングを刷新してちらつきと跳ねるプロンプトを排除。より滑らかで直感的、視覚的に安定したターミナル体験を提供。

マウスナビゲーション

矢印キーに頼らず、マウスで直接プロンプトをナビゲート可能。ターミナルインタラクションがより高速で直感的に。

リサイズ対応

ターミナルリサイズ時の視覚的なグリッチを排除。常にクリーンで洗練された表示を維持。外観を自由に調整可能。

スティッキーヘッダー

ツール確認とアクションのためのスティッキーヘッダーセクション。複雑なインタラクション中もコンテキストが常に表示され、何が起きているか把握可能。

エクステンション機能

異なるツールを接続してコマンドラインをパーソナライズ。データベースツール、MCPなど多数の新ツールをエクステンションストアから簡単インストール。

Jules拡張機能

自律的なサイドキックが非同期でコーディングタスクを処理。コードクローン、依存関係インストール、バグ修正、新ブランチへの変更提出を自動化。

Code Wiki CLI統合

コードベース理解を加速する新ツール。常に最新のウィキとして、コードベース全体をスキャンしてドキュメントを生成。インテリジェントでコンテキスト認識可能。

完全無料でパワフル

Gemini 3.0搭載で完全無料。最高のターミナルベースツールの一つに進化。詳細な変更ログで全ての改善点を確認可能。
