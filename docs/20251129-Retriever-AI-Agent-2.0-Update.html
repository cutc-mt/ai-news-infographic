Retriever AI Agent 2.0

ブラウザ拡張からフルインフラツールへ進化

2025年11月29日

大型アップデート2.0

便利なブラウザ拡張から、AI自動化のための本格的インフラツールへ完全進化。ブラウザをリモートMCPサーバーに変換可能。

MCP サーバー対応

ローカルChromeブラウザをMCPサーバーに変換。Claude Code、Cursor、SlackbotなどがWeb閲覧可能に。接続性の問題を解決。

Webナビゲーション橋渡し

強力なLLMと実際のライブWebを接続。ログイン済みのブラウザが全てのAIの「手」として機能。MCP URLをコピー&ペーストするだけ。

クラウド機能

拡張機能インストール不要のクラウドインターフェース。タスクを渡せばクラウド上でRetrieverエージェントが実行。どこからでもアクセス可能。

3つのモデル選択

Flash、Flashlight、Proから選択可能。Flashlightが最速、Flashが中間、Proが最高品質。用途に応じて使い分け。

クラウドまたはローカル

クラウド上で実行、または接続されたブラウザで実行を選択可能。柔軟な実行環境で様々なワークフローに対応。

Teach a Trick機能

タスクを一度録画してRetrieverに学習させる。ポータルログイン、ダッシュボードナビゲート、PDFダウンロードなどを記録して保存。

コマンドで再実行

/download reportと入力するだけで、学習した一連の動作を正確に再現。セレクタを学習して次回から自動実行。日常タスクを効率化。

エンタープライズ級スクレイピング

シンプルな自然言語プロンプトでエンタープライズレベルのスクレイピング。複雑な設定不要で大規模データ収集。

WhatsApp統合

WhatsApp番号をRetrieverアカウントにリンク。外出先からWhatsAppでタスクをテキスト送信するだけで実行。

モバイルからの自動化

ランチ中でもスマホからWhatsApp bot にメッセージ。Amazonの価格チェックや配送日確認などをリクエスト。クラウドが処理して返信。

デスクトップから分離

エージェントをデスクトップから切り離し。どこからでもクラウド自動化をトリガー可能。デバイスに依存しない柔軟性。

Chrome拡張基本機能

Y Combinatorから創業者情報取得、Crunchbaseで企業詳細検索、Google Sheetにデータ整理。エンドツーエンドで完全自動化。

Geminiモデル活用

Geminiモデルを使用してタスクを処理。これらのタスクに最適なモデルで、精度の高い結果を実現。長時間ワークフローにも対応。

コンテキスト添付

ファイル、シート、その他のソースからコンテキストを添付可能。タスクに必要な情報を事前に提供して精度向上。

タブグループ化

エージェントが作業中のタブを自動グループ化。どのタブで何をしているか一目瞭然。複数タスクの並列処理も管理しやすい。

無料ティア充実

すべての機能を試せる充実の無料ティア。大規模ジョブ用のクラウドコンピューティングパワーが必要なら有料プランも用意。

時間節約ツール

実際に時間を節約できる稀有なツール。日常的なWeb作業を自動化して生産性を大幅向上。シンプルなプロンプトで複雑なタスクを実行。
