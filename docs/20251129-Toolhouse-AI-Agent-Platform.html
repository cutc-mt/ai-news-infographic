数分でAIエージェント構築

サーバー設定不要、複雑なフレームワーク不要、膨大なドキュメント不要。Toolhouseは数分で完全なAIエージェントを構築・デプロイ可能。

Backend as a Service

AI開発者専用に構築されたバックエンドサービス。複雑なバックエンドを全て処理し、開発者は構築に集中できる。

バイブコーディング対応

摩擦なく素早く優れたものを作成。開発者フレンドリーで、バイブコーディングを好む人向けに設計された高速開発環境。

デモ：メール送信エージェント

通常の人間のように振る舞うAIエージェント。ミーティングリクエストメールを自動生成・送信。数分で構築完了。

スケジューラーエージェント

受信トレイをスキャンしてミーティングリクエストを検出。自動でカレンダー招待を作成し、Discordで通知。完全自動化。

MCP統合

リモートMCPサーバーのURLを貼り付けるだけ。ZapierでGoogle Calendar、Gmail、Discordに接続。複雑な設定や手動統合は不要。

ワンクリックデプロイ

ワンクリックでエージェントをデプロイ。Toolhouseが即座にホスト型UIを起動。すぐにチャットを開始可能。

バックエンドコード不要

バックエンドコードを一行も書かずに完全自動化を実現。Toolhouseとmcpが全て処理。開発時間を大幅短縮。

統合プラットフォーム

API、データベース、ベクターストアを扱う煩雑さを解消。AIアプリケーション用の完全なバックエンドを箱から出してすぐ提供。

あらゆるLLMに対応

任意の大規模言語モデルでAIエージェントを迅速に構築。様々なツールに接続して自動化を実現。柔軟性が高い。

豊富な外部ツール

Notion、Gmail、SlackをMCP経由で接続。RAGでドキュメント取り込みも数秒。全て統一プラットフォームからスムーズに実行。

自然言語でエージェント構築

Toolhouseを開いて自然言語でエージェントを構築。数分以内にデプロイ可能。プログラミング知識不要でAI開発。

2つの構築方法

Agent Studio（自然言語プロンプト）とCLIの2つの方法。開発者の好みに応じて選択可能。どちらも強力で使いやすい。

プリビルトテンプレート

様々なドメインの事前設定済みエージェント。カテゴリ別に整理されたテンプレートから選択して即座にスタート可能。

バンドル機能

複数のMCPをグループ化。Tavily検索、メール送信など、外部ツールをバンドルしてエージェントの能力を拡張。

フロントエンド連携

Lovable、Bolt、v0などのフロントエンドビルダーと統合。バックエンドはToolhouse、フロントエンドは他ツールでバイブコーディング。

CLI完全対応

npm install、th new、th deploy、th vibeコマンド。CLI経由で完全な開発フローを実現。認証、デプロイ、UI生成まで対応。

Firecrawl MCP統合

より高度なスクレイピング能力を追加。APIキーを設定するだけで、FirecrawlのMCPサーバーを統合。スクレイピング品質が大幅向上。
