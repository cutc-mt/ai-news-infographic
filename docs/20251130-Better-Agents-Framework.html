Better Agents - Langchainフレームワーク

AIエージェント構築のベストプラクティスを標準化するCLIツール

2025年11月30日

LangchainによるCLIツール

AIエージェントを正しく構築するための標準を設定。コーディングエージェント・汎用エージェント両方に対応。エキスパートエージェント化を実現。

エージェント能力向上

Kilo Code・Cursor・Claude Codeなどコーディングエージェントを強化。任意の分野のエキスパートエージェントに調整可能。

自律タスク実行

銀行カスタマーサポートエージェントを例に完全推論で実行。カード解除・取引表示・人間へのエスカレーションを自動化。

適切な構造を自動生成

ゼロからエージェント用の適切な構造を構築。ツール生成・セマンティックコードインデックス設定・自動テスト実行まで完全自動化。

反復改善ループ

Kilo Codeと連携し更新を通信。問題を修正しコードを書き換え。プロンプトを更新し新機能を追加。完全に機能するまで反復。

Langsmith内テスト実行

生成されたエージェントをMastra Studioフレームワークでテスト。最新クレジットカード取引表示などのタスク実行を検証。テスト結果を可視化。

Better Agent構造

選択したフレームワーク(Agno・Mastra等)に基づく本番環境対応エージェントプロジェクトを足場構築。標準化されたレイアウトを提供。

テストとシナリオ生成

ノートブック・データセットを使用してRAG・プロンプトバージョン・分類などパフォーマンスを測定。詳細な評価環境を自動構築。

prompts.json生成

プロンプトを同期させるレジストリを作成。バージョン管理とプロンプトの一元管理を実現。チーム開発での一貫性を確保。

mcp.json自動設定

適切なMCPサーバーを全て設定。コーディングアシスタントが利用可能なツールを自動認識。将来の世代で自動的にツール利用可能。

agents.mdベストプラクティス

構築・テスト・保守の正しい方法を記載したガイド生成。エージェント開発の品質を担保。チーム全体で標準を共有。

評価と可視化

シナリオテストでエージェントの動作を確認。Langsmithで評価を可視化。エージェントが期待通り動作することを検証。

コーディングアシスタント連携

問題修正のためコーディングアシスタントと調整。コード書き換え・プロンプト更新・新機能追加を自動化。完全な協調動作。

本番環境対応エージェント

完全な構造・完全なテスト・完全な評価を備えたエージェントを自動生成。実世界のユースケースに対応可能な品質を実現。

利用推奨理由

アーキテクチャを推測せず新規エージェントプロジェクトを開始可能。最初から本番グレード標準を強制。構造がシンプルで保守性が高い。

組み込み観測性

プロンプトバージョン・評価を自動テスト。多くの人がスキップする評価を標準装備。エージェントの信頼性・保守性が向上。

インストール方法

npm installでパッケージをインストール。Node.js 22以上・コーディングアシスタント・APIキーが必要。新規プロジェクトで初期化。

実例: 個人財務エージェント

支出追跡・予算管理・財務インサイト自動提供エージェントを作成。Langwatch・MCP選択。Claude 4.5でフロントエンド含め完全開発。
