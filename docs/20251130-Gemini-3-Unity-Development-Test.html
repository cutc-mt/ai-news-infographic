Gemini 3.0 Proリリース

Googleの最新AIモデル。性能が飛躍的向上。これまでGemini 2.5 Proは力不足だったが、3.0で実用レベルに到達。

Anti-Gravity登場

Googleの新AIエージェントコードエディター。VSCodeベースでプレビュー1段階。無料でGemini 3.0 Pro・Claude Sonnet 4.5使用可能。

Unity開発でのAI活用

AIにゲーム開発を丸投げ検証。プログラムだけでなくUnityツール操作も必要。最新エージェントならほぼ自動化可能に。

コスト問題の解決

Claude Codeは有料必須。Cursorは無料版が制限多い。Kilは無料で高性能Sonnet 4.5使用可能だが回数制限あり。

無料でガンガン使える期待

Gemini CLIは無料でかなりの回数使用可能。Gemini 3.0 Proと Anti-Gravityで無料高性能エージェントが現実に。

検証内容: 弾幕シューティング

派手な2D縦スクロールシューティング制作。機能・演出モリモリで自由に作成。エディタースクリプトでUnity設定自動化。

AIエージェント3種比較

Gemini 3.0 Pro vs Claude Code vs Codex CLI。同じ要件で弾幕シューティング制作。性能・速度・修正回数を徹底検証。

Gemini 3.0 Pro結果

弾幕シューティングらしきものを作成成功。修正箇所が3つで最少。手堅くきっちり作る印象。以前の2.5 Proから大幅進化。

Claude Code結果

モリモリの仕様書作成。パーティクル演出がいい感じ。ただし敵・弾が出ず遊べない状態。コーディング能力は高くテクニカル。

Codex CLI結果

BGM・効果音対応。背景ぐにゃぐにゃ動く。シェーダー使用。ただし敵キャラ表示不完全で遊べず。最も修正箇所多い。

所要時間比較

Claude Codeがプランニングで最長。想定より差は小さい。Gemini 3.0 Proは性能の割に高速。効率的な開発フロー実現。

修正回数比較

Gemini: 3回。Claude: 中程度。Codex: 最多。Gemini 3.0 Proが最も安定。AIは毎回結果が異なるが傾向は明確。

Anti-Gravityの評価

かなり良いツール。無料で使えるのが最大の魅力。Gemini CLIより使いやすい。正式版の無料枠次第で最有力候補に。

画像素材自動生成

Gemini 3.0 Proが画像素材を自己生成。完成には至らずも可能性を提示。ゲーム制作の画像素材作成手間が削減期待。

Claudeのプログラム画像生成

画像生成機能なしでプログラムで画像作成。パーティクルテクスチャなら実用的。技術力の高さを示す。

タスク完了まとめ機能

Anti-Gravityが作成内容を自動まとめ。AIエージェント開発に最適化された便利機能。進捗確認が容易に。

正式版への期待

途中経過・ドキュメントが英語。日本語対応で使いやすさ向上期待。Unity MCP対応必須。有料プランでも使う価値あり。

総合評価

Gemini 3.0 Pro×Anti-Gravityは実用レベル。無料で高性能開発環境実現。Unity開発がAI単独でほぼ可能に。未来のゲーム開発形態。
