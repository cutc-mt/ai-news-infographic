Antigravity ワークフローエージェント作成

チャット対話で記事量産・翻訳を完全自動化

2025年12月2日

2つのワークフローエージェント

記事自動生成エージェントと記事翻訳エージェントを作成。チャットで指示するだけで100本でも1000本でも記事を量産可能。

記事自動生成エージェント

「集中力というテーマで10本書いて」と指示するだけ。AIが自動で記事をライティングし、指定フォルダにプレーンテキストで保存。

翻訳エージェント

英語記事をフォルダに入れて「翻訳して」と指示。Pythonスクリプトで自動実行し、翻訳完了後にファイルを完了済みフォルダへ移動。

Pythonコード自動生成

Wordファイル読み込み用のPythonスクリプトをエージェントが自動作成。環境確認からコード生成まで全て対応。

ルールブック自動作成

.agentsディレクトリのworkflowsフォルダにMDファイルで手順書を生成。指示を受けると参照して一貫した処理を実行。

対話型ワークフロー構築

「こういうことしたいんだけど」と質問すれば「こうすればできます」と回答。マニュアルを読まずに自然な会話で構築可能。

Gemini 3 Think Slow使用

記事生成に最新のGemini 3モデルを活用。高品質な記事コンテンツを安定的に生成する基盤。

フォルダ自動管理

articlesフォルダに入力、articles_jpに日本語出力、completedフォルダに処理済みファイルを移動する3フォルダ構成。

Wordファイル対応

DOCXファイルはそのまま読めないが、Pythonで自動抽出。わざわざテキスト変換する手間を完全排除。

重複記事チェック

記事生成前にトップページやアーカイブを参照。同じテーマの記事がないか確認し、被らないよう自動調整。

テスト駆動開発

ワークフロー構築後は必ずテスト実施。3本の記事で動作確認し、成功したらルールブック化する安全な開発フロー。

マイクロブログコース連動

Antigravity マスタークラス シーズン1「マイクロブログ作成」の特別編。超初心者向けコースと並行して高度な実装を紹介。

ステップバイステップ手順書

「こういう指示が来たらまずこうして、次にこうして」と明確な手順をMDファイルに記録。エージェントが忠実に実行。

10記事を数分で生成

デモでは10記事を数分で作成。「脳に効く食事」「ポモドーロテクニック」「デジタルデトックス」など多様なテーマを自動生成。

ファイル移動自動化

翻訳完了後、元の英語ファイルを自動的にcompletedフォルダへ移動。手動管理不要で整理整頓された状態を維持。

初心者でも高度な自動化

Agent Managerを一切触らず、設定もデフォルトのままでこのレベルを実現。技術的障壁を大幅に下げた設計。

質問攻めで解決

マニュアルやドキュメントより、とにかく質問しまくる方が早い。やりたいことを明確にして聞けば即座に解決策を提示。

コンテンツ量産革命

SEOやキーワード戦略は別途必要だが、コンテンツ生成の物理的作業は完全自動化。ブログ運営の生産性が劇的に向上。
