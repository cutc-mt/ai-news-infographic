1位：Google Gemini 3 Pro

おすすめ度100点。Google Workspace経由で月1600円。2TBストレージ、NotebookLM Pro、Gemini AI Pro付き。最高知能のモデル。

思考モード必須

Geminiは高速モードではなく思考モードを常に使用すべき。ファストモードはまだアップデート待ちの状態。

2位：GLM 4.6

おすすめ度90点。中国系モデルでコスパが異常に高い。月3ドルから。Claude Proの15倍使える驚異のコスパ。

API活用に最適

GLM 4.6はAPIキー発行可能。OpenAI互換APIでローカルAIエージェントや開発に最適。日本語性能も良好。

MCP サーバー提供

GLMはウェブサーチ、ウェブリーダー、画像分析などのMCPサーバーを提供。プランにより使用量が変動。

3位：ChatGPT Pro

おすすめ度90点。月200ドル。GPT-5.1 Proのマクロ思考が圧倒的。本質を語れる抽象思考の強さが魅力。

GPT-5.1 Pro の実力

事業の壁打ち、戦略、メタ認知、抽象的分析で頼りになる存在。情報要約の的確さもGeminiより優秀。

Codex エージェント

Plus以上でターミナルAIエージェント「Codex」が利用可能。VSCode拡張機能も使える開発支援ツール。

4位：Claude Opus 4.5

おすすめ度85点。月100ドルのMaxプラン。Claude Codeでの開発に最適化。知能レベルは1000点超え。

Claude Code 専用

Claude課金の主目的はClaude Codeでの開発やタスク処理。CLI経由のAIエージェントに最適化された選択。

6位：Windsurf
