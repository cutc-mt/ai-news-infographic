Conduit・ターミナル中心IDE・エージェント並列化

GitHub開発者によるローカル・クラウド統合コックピット環境

2025年12月5日

Conduitとは

GitHub開発者が作成したターミナル中心IDE(非公式プロジェクト)。エージェントによるコーディングタスクをローカル・クラウド間で並列化。統合コックピット環境を提供。

柔軟なタブシステム

複数のCLIエージェント(Claude Code・Copilot CLI・Open Code)をタブで管理。各タブにターミナル・エディターペイン・埋め込みブラウザを配置可能。

独立エージェントセッション

タブを複製し独立サンドボックスとして起動。複数ターミナルを同期してコマンド一斉実行。ローカル・クラウドターミナルを混在配置。CLI経由で全自動化可能。

従来の痛点を解消

TMux分割・VS Codeタブ・複数ターミナルのバックグラウンドジョブの煩雑さを解消。統合コックピットで好みの形状にカスタマイズ可能。オープンソースではない点に注意。

情報密度の高いタブ

ペインのリサイズ・折りたたみ・クローン・レイアウトロック機能。最適な配置を作成後に固定。作業に合わせて柔軟に環境構築。

自動Worktree管理

新ターミナル分割・タブ複製時に独立サンドボックス作成。Git Worktree自動管理でリポジトリクローンに変更を加え、良ければメインにマージ。

クラウドターミナル統合

リモート計算リソースを起動しローカルシェルと混在配置。並列タスク数を増やし高速処理。ローカル・クラウドをシームレス操作。

エージェントカスタマイズ

使用するコーディングエージェントを事前定義。任意のターミナルでスラッシュコマンドで呼び出し。一貫性のある開発体験。エージェント引数も設定可能。

デザインモード

ドラッグ&ドロップでペイン管理。ペイン作成・削除・タブ間移動・分割・ラジアルクイックアクション設定。視覚的に理想レイアウト構築。

フォーカスモード

ペインのズームイン・アウトとミニマップ表示。集中作業と状況把握を切替。コンテキスト喪失なく作業効率化。UI最小化でクリーンな環境。

タブプレビュー・概要

タブホバー時にライブプレビュー表示。複数タブの鳥瞰図(View All)で全体把握。バックグラウンドタスクの監視が容易。

エディターペイン

ファイルエクスプローラー・テキストエディター内蔵。リアルタイムGitステータス・Diff・ステージング・コミット・プッシュ・プルワークフロー。コンテキスト切替不要で編集。

クイックアクション

レイアウトを乱さず単発タスク実行。ハイパーターミナル(Cmd+L)でタブを消費しない一時ターミナル起動。迅速なコマンド実行。

Conduit CLI自動化

環境のあらゆる部分(ターミナル・ブラウザ・エディター)をCLI経由で設計・操作・自動化。エージェントがワークフロー全体をスクリプト化可能。

ペイン同期

複数ターミナルへのキーストロークミラーリング。マルチサービスコマンドを一括実行。並列デプロイ・テスト実行などを効率化。

ワークスペース保存

タブセットを保存・復元しプロジェクト間を簡単移動。カスタマイズしたレイアウトをエクスポート・インポート。プロジェクト別環境を即座に再現。

カスタムキーバインド

ショートカットキー設定でフロー最適化。Cmd+Shift+Pでコマンドパレット。Cmd+Tで新タブ。Cmd+Lでブラウザ起動。MRUリストで素早く操作。

埋め込みブラウザ

ツール内でブラウザペイン起動(Cmd+L→URL入力)。ドキュメント参照・プレビュー確認をIDE内で完結。外部アプリ切替不要。開発効率向上。

総評・今後

非常に洗練されたUI。スナッピーで高速動作。これまで見たツールで最高のUI。TMux・VS Code・複数エージェントを統合コックピットで管理。クラウドVM機能は要検証。
