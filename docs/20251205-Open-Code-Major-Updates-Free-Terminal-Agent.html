Open Code大型アップデート・無料ターミナルエージェント

Claude Code・Gemini CLI代替のオープンソースコーディングエージェント進化

2025年12月5日

Open Code概要

完全無料のターミナル型AIコーディングエージェント。Claude CodeやGemini CLIの代替としてオープンソース。LSP対応・美しいTUI・豊富なコマンド搭載。

MJrep統合・4倍高速化

セマンティックgrep機能MJrep統合でスピード4倍・トークン使用量1/3に。コード・画像・PDFを意味で検索。npm install mgrep open-codeで即導入可能。

MJrep効果

トークン無駄遣いを削減し高速・効率的。品質を犠牲にせず結果を即座に提供。デモでは明確にMJrep有無のワークフロー差が確認できる。

新デスクトップアプリ

ウェイトリスト受付中の新デスクトップアプリ。ターミナルと複数セッション管理をWarp風に統合。オープンソースで無料・Warpより機能的。

デスクトップアプリ機能

ターミナルエージェントを直接アプリ内実行。新TUI搭載。計画・ビルドを簡単に管理。複数プロジェクトの同時並行作業が可能。

新TUIデザイン

テキストベースインターフェースが大幅改善。クリア・大きく・明確なアニメーション。エージェント進捗を視覚化。ノイズ削減でフォーカス性向上。

右側チェックインパネル

コンテキスト使用量・コスト・LSP(Language Server Protocol)をリアルタイム表示。透明性の高い開発体験を提供。

Zed拡張機能

Zedエディターで1クリックインストール。ログイン不要。エディターとターミナルエージェントを並行使用。拡張ストアから即座に追加可能。

Webサンドボックス環境

Open Code SDKをホストするオープンサンドボックス環境がWeb上で利用可能。Open Code自身が他ツールを実行。主要ノードPRによる強力アップグレード。

TUI出力可視化

任意LLMがTUI出力をリアルタイムレンダリング可能。ターミナル内でインタラクティブに動作確認。実際の開発者と同じ操作感を実現。

Open Code SDK

型安全なJavaScript・TypeScriptクライアント。プログラム的にOpen Codeを組み込み・制御可能。内部的にTUIが使用する同一SDK。

SDKでできること

タスクを並列実行・カスタム統合構築・アプリへ直接組み込み。TUIが実行可能な全アクションを自動化。AIコーディングエンジンを完全プログラム可能に。

Zenモデルプロバイダー

コーディング最適化済みAIモデルのキュレーションリスト。プラグアンドプレイで高品質モデル利用。自分で設定不要。ただし推奨は独自設定。

最適ワークフロー

Opus 4.5を計画エージェント、Gemini 3.0をコード実行エージェントに設定。ベストプラクティスとして推奨。アーキテクトとコーダーの役割分担。

簡単インストール

npm・bun・brew・peru・curlで1コマンドインストール。ターミナルでopen-codeと入力すれば起動。最新版がクリーンなUIで即利用開始。

モード切替・モデル選択

PlanモードとBuildモード切替可能。マウス操作でモデルプロバイダーとモデルを選択。エディターをターミナル内で直接起動可能。

複数セッション・新コマンド

複数セッション同時実行機能追加。model cycle・reverse switching agents・エディター起動など新コマンド多数。変更ログで詳細確認推奨。

無料Grok提供・新UI

Grock Code Fast 1が無料で利用可能。新TUIは左側にアニメーション、右側にコンテキストパネル表示。完全無料でプロ品質の開発環境。
