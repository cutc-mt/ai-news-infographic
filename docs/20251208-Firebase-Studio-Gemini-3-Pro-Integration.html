Firebase StudioとGemini 3 Pro統合：AI搭載開発環境の進化と活用

公開日: 2025年12月8日

Firebase Studioの変遷

  • 旧名Project IDX: 元々はウェブ上のVS CodeインターフェースとVMを提供。
  • AI機能の追加: その後、AI機能が統合され価値が向上。
  • Vibeコーディング: コーディングを視覚的に楽しむVibeコーディングインターフェースも追加。
初期はバグが多く、GoogleのAIモデルの品質も低かったため、
評価は今ひとつでした…

Gemini 3 Proで「驚異的な結果」

  • モデルの改善: Gemini 3 Proの統合により、Firebase Studioの機能が劇的に向上。
  • シンプルツールとMVP: 特にシンプルなツールやMVP（Minimum Viable Product）作成に最適。
  • 優れたフロントエンド生成: 魅力的で洗練されたフロントエンドを効率的に作成。
複雑なプロジェクトには向かないものの、
無料で利用でき「驚異的な結果」を生み出します！

Gemini AIアシスタンス

  • チャットモード: コードベースについてモデルと対話（読み取り専用権限）。
  • エージェントモード: ファイルの編集や作成が可能。各ステップはユーザーの承認が必要。
  • エージェント自動実行: 全てのアクションを自動承認し、タスク完了まで自動で進行。
  • MCPサーバーオプション: MCP (Multi-Cloud Platform)サーバーの設定が可能。
プロジェクト要件ドキュメントをアタッチしたり、
ファイルを指定してコンテキストを集中させることが可能！

開発支援と今後の展望

  • コード補完とアクション: コードの自動補完、選択コードの説明、コメント追加、ユニットテスト生成。
  • スラッシュコマンド: /ask, /clear, /generateなどの便利なコマンド。
  • インラインチャット: コード内の特定箇所について変更を依頼し、その場で修正。
  • VM環境の提供: VMを利用できる点がGoogle AI Studioよりも優れていると評価。
GoogleはFirebase Studioにもっと注力すべき！
将来的にはAnti-gravity cloudのような存在になる可能性も。
