Antigravity概要

Googleの新しいエージェント型開発環境。単なる自動補完ではなく自律エージェントがソフトウェアの計画・コード・テストを実行。コードの煉瓦職人から設計者へ昇格。

70年の進化・次のステップ

1950年代マシンコード→コンパイラ登場でロジック重視に。低レベル作業停止。AIツールは現代のコンパイラ。煉瓦職人から設計者へ。設計図作成→AIが煉瓦を積む。

GuestPassアプリ構築

カフェ・ホテル・図書館向けWi-Fiパスワード問題解決。Wi-Fi詳細入力→QRコード生成→印刷・表示。ゲストがスキャン→即座に接続。ルーター確認不要。

仕様書作成・アーキテクト思考

初心者「Wi-Fiアプリ作って」。アーキテクトは仕様書作成。高品質コード→高品質要件必要。Tailwind via CDN指定・アーキテクチャをリーン化・具体的指示が鍵。

セキュリティ設計

生成QRコードをサーバー保存しない指示。画像に実際のWi-Fiパスワード含む。保存するとセキュリティリスク。アーキテクトがユーザー保護。設計段階での判断重要。

Node.js 23・TypeScript最適化

最新版Node.js使用を強制。TypeScript通常はコンパイル必要→最新Node版はネイティブ実行可能。煉瓦職人は古いDockerfileコピペ。アーキテクトはビルドステップ削除。

実装計画自動生成

他AIツールでは手動で実装計画作成。Antigravityは自動生成。編集可能。アーティファクトとして右側に表示。ベストプラクティス組み込み済み。

自動ブラウザテスト

画面上の青い点が移動。マウスポインタではない。ツールが独立してナビゲート・クリック・アプリテスト。他AIツールではブラウザ手動テスト→エラーコピペ必要。不要に。

スクリーンショット自動記録

テスト成功後スクリーンショットをアーティファクトに追加。他ツールでは手動作業。Antigravityは自動化。半手動プロセスをベストプラクティスとして組み込み。

追加テストケース指示

ハッピーパスのみテスト→ネットワーク名未入力ケース追加指示。メッセージ表示・不正QRコード生成防止。実装計画に追加→テスト実行→エラーメッセージ確認。優秀！

アーティファクトへのコメント機能

タスクリスト等のアーティファクトにGoogleドキュメント風コメント追加可能。Dockerローカルビルド不要コメント追加→青いSubmitボタン→タスクリストから削除。便利！

コードレビュー・編集機能

開発者はコード確認したい。自分の名前で提出→最後のセミコロンまでレビュー。アイコンクリック→通常のコードエディタビュー。ツールの編集を承認・拒否可能。

Cloud Runデプロイ・チートコード

サーバーレス・スケーリング心配不要・アイドル容量課金なし。VM準備不要・nginx設定不要。コンテナ渡し→安全でスケーラブルなURL取得。インフラがAI記述コードと同速。

gcloudコマンド不要デプロイ

Cloud Runへのgcloudコマンド記憶不要。Antigravityにデプロイ依頼。Node 23 Dockerfileの設計図をCloud Runが使用。推測せず設計者が設計した通り実行。

エラー自動修正・ポート不一致

デプロイエラー発生。過去:ログからエラー検索→変更内容判断→アップデート。Antigravity:自動判断。「ポート不一致。Cloud Run 8080期待・アプリ3000使用」→介入なしで修正。時間節約。

再デプロイ待機・紅茶タイム

クラウドは新デプロイ処理に1分必要。高速化不可。違いは:待機中かもしれず紅茶飲む。作業なし。StackOverflow検索なし。エージェントにリトライ任せ。デプロイ完了→アプリ稼働。

クラウドでも動作確認

パブリックURLクリック→アプリ表示。偽のネットワーク名・パスワード入力→QRコード生成確認！ローカルマシンだけでなくクラウドでも動作。グローバルURL取得成功。

3つの重要ポイント

①シフトは現実:煉瓦職人→設計者へ。構文入力ではなく設計図作成に貴重スキル使用。②エージェント型:実装計画・エラー補足・ターミナル駆動。③Cloud Run:AIコード記述と同速インフラ。
