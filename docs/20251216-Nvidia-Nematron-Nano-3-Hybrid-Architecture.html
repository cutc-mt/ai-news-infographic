Nvidia Nematron Nano 3

ハイブリッドアーキテクチャ・Mamba×Transformer×MoE・30B→3B推論革命

2025年12月16日

小型モデルの常識を覆す
従来、ローカルで動く小型モデルは高速だが複雑な論理思考や多段階指示に弱かった。本格的推論にはH100クラスタで動く大型モデルが必要。Nvidiaがこの常識を完全に覆します。
ハイブリッドアーキテクチャ革命
Mamba層(線形スケーリングの状態空間モデル)と従来のTransformer層を組み合わせた全く新しい設計。単なる蒸留モデルではなく、根本的に異なるアーキテクチャです。
Mixture of Experts設計
MoE(専門家混合)デザインを採用。専門家ライブラリを持ちながら、特定の単語生成に必要な専門家だけを呼び出す効率的なルーティング機構を実現しています。
30B→3B魔法のパラメータ
総パラメータ数は30億だが、任意のトークン生成に使用するアクティブパラメータは約3億のみ。大型モデルの推論能力と知識ベースを、小型モデルの推論速度と低レイテンシで実現。
100万トークンコンテキスト
最初から100万トークンのコンテキストウィンドウをサポート。長大な文書やログファイルを一度に処理可能。Mamba層により線形スケーリングで効率的に処理します。
OpenAI互換API即座利用
複雑なカスタムローダー不要。OpenAI互換APIで動作。PythonのOpenAIクライアントでNvidiaのベースURLとAPIキーを指定するだけで即座に利用開始できます。
結婚式席次パズル完全解決
7人の複雑な席次制約問題を推論モード(thinking tokens)で解決。両親と義理の両親を隣に座らせない、牧師の配置など複数の競合変数を同時にメモリ保持し論理導出します。
推論モードthinkingトークン
単なる推測ではなく段階的に論理をマッピング。推論モデルと同様にthinkingトークンを出力ストリームで確認可能。通常70Bパラメータモデルが必要な問題を数秒で解決します。
乱雑データJSON構造化
映画レビューの長い段落から監督、感情、プロットポイントを抽出し完璧なJSONオブジェクトを生成。ハイブリッドMambaアーキテクチャがコンテキストを効率的に処理し混沌から構造を創出。
巨大サーバーログ解析
数千行のエラーコード、タイムスタンプ、システム警告を含む巨大ログファイルを楽々処理。推論トークンでノイズをフィルタリングし、データベースタイムアウトによるカスケード障害を特定。
Mamba線形vs Transformer2乗
標準Transformerは2次複雑度で長文になると指数的に遅くなる。Mamba層は線形スケーリングで長シーケンスを処理。メモリフットプリントを爆発させずに巨大コンテキストを推論可能。
Nemo Gym強化学習FW
Nemo Gymフレームワークで強化学習を使用したエージェント訓練が可能。環境をシミュレートし、特定ワークフローに特化したnanoモデルを微調整できます。
専門エージェント微調整
Shopify API特化のショッピングアシスタントや銀行取引監視ボットなど専門化エージェント構築が可能。Nemo Gymで環境シミュレートし特定ワークフローに優れたスペシャリストを育成。
3Bアクティブ超高スループット
3億アクティブパラメータ数により驚異的な高スループットを実現。命令フォローとツール呼び出しに優れ、素早く関数選択が必要なエージェント型ワークフローに最適です。
エージェント型WF向け設計
データ構造化や意思決定を行う中間エージェントとして優れた性能。コンテンツ作成する創造的エージェントではなく、論理、ルーティング、データ処理タスクに特化した設計です。
制限創造タスクには不向き
サイズに対し驚異的だが依然nanoモデル。複雑な新規ソフトウェアシステムのゼロからの記述や創造的小説生成には不向き。GPT-4クラスの膨大な世界知識は持ちません。
推論・論理・ルーティング特化
推論能力、論理処理、ルーティング判断、データ処理タスクに最適化。構造化されたデータを扱い決定を下すミドルマンエージェントとして真価を発揮します。
上位モデルも間もなく登場
より大規模な兄弟モデルも近日公開予定。オープンモデルとして高いOpenness Indexスコアを獲得。自分で実行、ウェイトを検査、またはAPIキーで即座テスト可能です。
