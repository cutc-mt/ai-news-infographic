Toolhouse 初のエージェントUI

最速最簡単AIエージェント作成・MCP統合・CLI完備・分単位デプロイ

2025年12月22日

数分でAIエージェント構築
サーバー設定不要、複雑なフレームワーク不要、膨大なドキュメント不要。Toolhouseは完全なバックエンドをそのまま提供。開発者は構築に集中可能です。
AI開発者専用BaaS
Backend as a ServiceとしてAI開発者向けに特化。日常タスク処理アシスタント、コードレビューボット、カスタム自動化エージェントを高速かつ摩擦なく構築できます。
メール送信エージェントデモ
通常人間のように振る舞うAIエージェントを数分で構築。会議リクエストメールを自動生成・送信。裏側はToolhouseで作成されたAIエージェントが全て処理します。
スケジューラーエージェント自動化
受信トレイをスキャンし会議リクエストを検出、カレンダー招待を自動作成、Discordで通知。ZapierホストのMCPサーバーURLを貼り付けるだけで統合完了です。
MCP統合ノーコード接続
GoogleカレンダーGmail Discord接続が複雑なセットアップなしで完了。手動統合不要でワンクリックデプロイ。ホストUIが即座に起動しチャット開始可能です。
バックエンドコードゼロ実装
メール読み取り、カレンダーイベント作成、Discord通知が全て自動実行。ToolhouseとMCPがパワーを提供。バックエンドコードを一行も書かずに実現できます。
エージェントスタジオ自然言語構築
プレーンな自然言語でプロンプトを入力するだけでエージェント構築。またはCLIでビルド可能。どちらの方法でもAIエージェントを迅速に作成できます。
事前構築テンプレート豊富
異なるドメインの複数エージェントテンプレートを提供。カテゴリー別に様々なエージェントを発見可能。テンプレートで簡単にスタートできます。
エージェント実行トラッキング
作成した全エージェントを管理。各実行の個別記録を追跡可能。全ての結果とログを確認でき生成プロセスを完全に可視化できます。
バンドル機能MCP一括管理
複数のMCPをグループ化してバンドル作成。Tavily Web検索やメール送信など外部ツールをエージェントが利用可能。エージェント機能を大幅に拡張します。
シードスタートアップファインダー
CrunchbaseとAngelListからシードステージSaaSスタートアップを検索。Web検索とクローリング機能を搭載。ウェブサイト、説明、場所、資金調達段階を抽出します。
LovableBolt フロントエンド統合
バックエンド機能作成後、LovableやBoltと接続可能。フロントエンドをVibe Codeでさらに構築。エージェントと連携した完全なアプリを作成できます。
CLIコマンド完全対応
npm installでToolhouse CLIをインストール。th newで新エージェント作成、th deployでデプロイ、th vibeでフロントエンド生成。全てコマンドラインから実行可能です。
YAMLファイル設定編集
デプロイ前にプロンプトを設定したい場合はYAMLファイルで設定。IDEやエディターでシステムプロンプトを直接編集。エージェントスタジオと同等の機能を実現します。
Firecrawl MCP強化スクレイピング
FirecrawlMCPサーバーでスクレイピング品質向上。APIキーを設定するだけで統合完了。AngelListとCrunchbaseから以前より多くの企業を抽出できます。
APIキー管理シンプル化
MCPサーバーにボタンから直接アクセス。APIキーを管理画面から直接管理可能。エージェントスタジオのコード内でバンドル実行を指定するだけで動作します。
米国シード企業高精度抽出
Firecrawl MCPを使用して再実行。以前のスクレイパーよりも大幅に改善。米国ベースでシードステージ資金調達の企業を高精度で発見できます。
分単位デプロイ本番環境準備
数分以内にAIエージェントを構築し異なるMCPサーバーを接続して即座にデプロイ。AI展開の力を指先に。本番環境対応エージェントを驚くほど迅速に作成できます。
