2025年AIツールティア表

非エンジニア整体師が使い続けた＆滞在時間の長かったツールランキング - AI整体師

S-Tier: Cursor（トップ）

開かない日がなかった

個人開発にどっぷりはまった1年。サブディスプレイはずっとCursorが開いている状態。カーソルを開かない日がなかった。土台の上にClaudeやCodexを使う構図。滞在時間1位。

S-Tier: Claude

Web版 + Claude Code拡張機能

Cursor内で1番動かしていたのがClaude。Claude Code拡張機能を多用。Web版でもClaude Codeでも触らない日はほとんどなかった。Cursor上でのClaude Codeという使い方が主流。

S-Tier: ChatGPT

Web + スマホアプリ + Codex拡張機能

なんだかんだ1年間Web版とスマホアプリを使い続けた。Cursor上でCodex拡張機能を使う頻度も多い。音声モード、検索機能、Cursor統合で総合的に高評価。Claude、ChatGPT、Cursorの順番は決められない。

A-Tier: Genspark

インダストリーアンバサダー

使わない日はほとんどなかった。画像作成、AI Developer実行で滞在時間長い。クレジットが3000余ることはほとんどなかった。個人開発、AIシートリサーチ、画像作成で使用。使うことが当たり前になった。

A-Tier: Perplexity

改めて課金して使用頻度アップ

後半にかけて急上昇。検索体験の速さ、まとまり具合、検索精度が優秀。Perplexity Cometをブラウザで開いているので毎日開いている。AI生活の基盤になっている。

A-Tier: Junie

JetBrains WebStorm専用拡張機能

WebStormでしか使えない拡張機能。大規模リファクタリング、複数エラー一気解消、不要な行削除、ファイル構成整理。最後の切り札。ClaudeやCodexでどうにもならない時に投げる。時間かかるがちゃんとやってくれる。

A-Tier: Suno

唯一の「意味を度返しにした使い方」

GPTラジオで毎週1曲作成。経済的効果を度返しにして単純に楽しい、QOL向上のための使い方。クレジット余るが無駄とは思っていない。生産性から降りたAI使用に価値がある。

A-Tier: NotebookLM

2025年の進化がやばかった

そもそものコンセプトが強いツールがさらに進化。スライド生成、データテーブル作成、スプレッドシート化。難しく考えなくても使える。会社・仕事での使用にフィットしやすい。存在感と唯一無二のポジション。

B-Tier: Grok

X（Twitter）リサーチの唯一無二

Xの最新情報、リーク情報にダイレクトアクセス。情報の速さと鮮度がXの価値。そこから直接情報を引っ張れる唯一のツール。出力の速さ、要約の仕方が使いやすい。来年はもっと上がる可能性。

B-Tier: Gemini

Google Workspaceで使用

Googleドキュメント、スプレッドシートに滞在時間はそこそこあるが、Gemini搭載が理由ではなかった。サイドパネルGeminiの機能増加・精度向上で今後上がる可能性。NotebookLMとの連携も期待。来年末にティア上昇の可能性。

B-Tier: Google AI Studio

毎日使用：チャプター分け＆文字起こし

YouTube動画のチャプター分けと文字起こしを全てGoogle AI Studioで実施。開かない日はなかった。タスクとしてはそれぐらい。たまに研修や画像作成で使用。

C-Tier: Manus

課金せず無料分のみ使用

無料クレジット分だけの使用。Manus 1.5・1.6でバージョンアップ後評価上昇。ユーザー増加で情報増加。来年は課金して両方（GensパークとManus）使う可能性。論文リサーチで保険として走らせる使い方。

C-Tier: Phellow

年間契約だから使っていた

年間契約の同期でしか使っていなかった。Phellow Dogで機能統合後に評価上昇。リサーチだけじゃなくなった。年間契約切れ後の再課金は迷う。日本企業、月額が安い。初心者には良い選択肢。

C-Tier: Mappify

年間契約更新せず

ほとんど使わなかった。夏ごろに年間契約切れ。触ってみたが今後使わないと判断。毎月課金にも切り替えなかった。マインドマップはNotebookLMでできる。課金をお勧めしない。来年以降も候補に入っていない。

Junieの強み

GPT-4o + Claude Opus選択可能

裏側でGPT-4oやClaude Opus選択可能。挙動、プラン立案、正確な実行能力が強い。時間かかるがちゃんとやってくれる印象。リリース前の最終整理、セキュリティチェックで頼る。Discord yuさんに教えてもらった。

滞在時間の定義

使い続けた = 滞在時間が長かった

課金して使ったツールがメイン。Manusは課金していないがちょこちょこ使用。「使い続けた」は「滞在時間が長かった」と同義。AIモデルランキングとは別の観点。

2026年の期待

GeminiとManusが上昇可能性

Geminiの期待値高い。Google Workspaceエコシステムでの統合力。Manusも課金検証予定。Genspark進化継続。Cursorの牙城を崩す何かが出てくると面白い。来年末の比較が楽しみ。

非エンジニアの個人開発

整体師が没頭した1年

非エンジニアの整体師が個人開発に没頭。サブディスプレイ導入後はずっとCursor表示。AIツールで経済的効果を度返しにした使い方も重視。生産性だけでなくQOL向上の観点も大切。
