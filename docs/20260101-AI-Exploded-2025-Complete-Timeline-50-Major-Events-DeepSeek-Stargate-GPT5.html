2025年通年 - $37B企業AI投資

2023年$1.7B → 2025年$37B

企業のエンタープライズAI投資が爆発的増加。2023年の$1.7Bから2025年は$37Bへ。84%の開発者がAIツールを使用または使用予定。約半数が毎日AIコーディングアシスタント使用。

AIスタートアップが全世界資金の50%

2021年から劇的な進化

2025年、AIスタートアップがほぼ50%の全世界資金を獲得。49社の米国AIスタートアップが最低$100M調達。メガファンディングラウンド（$100M+）が常態化。

1月 - DeepSeek-R1リリース

Nvidia時価総額$600B喪失の引き金

革命的オープンソースLLM。議論はあるが、これによりNvidia時価総額が$600B減少（米国株式市場史上単一企業の最大1日損失）。中国スタートアップが世界に衝撃。

1月 - Stargateプロジェクト発表

今後4年間で$500B投資

OpenAIの新AIインフラ構築のために設立された企業。4年間で$500B投資予定。OpenAIのOperatorエージェント（自分のブラウザで数百万タスク実行）もリリース。

2月 - Vibe Codingが公式化

Andrej Karpathyが命名、Wikipedia掲載

AIエンジニアAndrej Karpathyが「Vibe Coding」という用語を作成。非常に人気を博しWikipediaページも作成。公式化。EU AI Act（世界初の包括的AI法的枠組み）も施行。

2月 - Harvey $300Mシリーズ D

法律AI業界は金鉱

リーガルAIスタートアップHarveyが$300M調達。多くの人が認識していないが、法律業界はGenerative AIの金鉱的機会。専門知識と大量文書処理の需要が完璧にマッチ。

3月 - Manus汎用AIエージェント

コンテキストエンジニアリングの先駆者

初の汎用AIエージェント。現在のコンテキストエンジニアリング戦略と長時間実行タスク処理エージェント構築の先駆者。Anthropic Model Context Protocol（MCP）も本格化。

3月 - Google Gemma 3 & Gemini 2.5

Gemini 2.0技術ベースのオープンソース

GoogleがGemma 3（オープンソースLLM、Gemini 2.0技術ベース）リリース。Gemini 2.5も発表（大量コンテキスト処理で革命的、数ヶ月間ベスト汎用LLMとして君臨）。

3月末 - OpenAI $40B調達

ポストマネー評価額$300B

OpenAIが$40B調達、$300Bポストマネー評価額に。Meta Llama 4 "Herd"（Behemoth、Maverick、Scout）を4月にリリース。Metaはその後動き少なく2026年にLlama 5期待。

4月 - OpenAI o3 & o4-mini

推論能力の限界を押し広げる

OpenAIがo3とo4-mini推論モデルリリース。LLMに組み込まれた推論能力の限界を押し広げ続ける。Alibaba Qwen 3オープンソースモデルも4月末にリリース（今も最高のオープンソースLLM）。

5月 - OpenAI Codex Agent

GitHub直接統合、リモート作業可能

OpenAIがCodex導入（クラウドベースソフトウェアエンジニアリングエージェント、GitHub直接統合、リポジトリをリモート作業）。Codex CLI後にリリース。AIコーディングアシスタントの最高峰の1つ。

5月 - Google Veo 3 & Windsurf買収失敗

AI動画生成が「本当に良い」と感じる転換点

Google Veo 3リリース（AI動画生成がついに「本当に良い」と感じる）。OpenAIがWindsurfを$3Bで買収合意も7月に破談（Windsurf CEOがGoogleへ）。GoogleがWindsurf実質買収でAnti-Gravity IDEに機能統合。

5月 - Claude 4ファミリー

コーディングキングの誕生

AnthropicがClaude 4ファミリーリリース。Claudeが真のコーディングキングに。Claude Codeに夢中。現在も最高のAgentic Codingアシスタント。Cursor $500M ARR到達（6月、SaaS史上最速成長）。

6月 - Meta超知能ラボ & Kiro

9桁給与でトップ研究者引き抜き

Mark ZuckerbergがMeta Super Intelligence Labs設立発表。9桁給与で他社からトップ研究者引き抜きドラマ。Amazon Kiroコーディングエージェントも7月リリース（12月Re:Invent後に企業関心急増）。

7月 - Moonshot Kimi K2

ChatGPTとClaudeをコーディングで上回ると主張

MoonshotがKimi K2リリース（ベンチマークでChatGPTとClaudeをコーディングで上回ると主張、議論はあるが）。オープンソースLLMが限界を押し広げ続ける。

8月 - OpenAI GPT-5導入

比較的静かな8月の最大ニュース

8月はGenerative AIにとって比較的静かな月。最大のニュースはOpenAI GPT-5導入。9月にAnthropic $13B調達（$183B評価額）。著作権侵害訴訟で$1.5B和解（50万冊の本、1冊約$3,000）。

9月 - Perplexity $20B評価額

AI検索スタートアップの競争力維持

PerplexityがAI検索スタートアップとして$200M調達、$20B評価額。LLMの組み込み検索がPerplexityを置き換えると何度も予想されたが、最先端を維持し競争力を保つ。Mistral €1.7B調達。

9月 - OpenAI Sora 2

動画＋音声生成で世界を席巻

OpenAI Sora 2リリース（動画と音声生成で世界を席巻）。10月にChatGPT 800M週間アクティブユーザー到達。IBM × Anthropic提携（エンタープライズ対応AI開発加速、Anthropicがエンタープライズ LLMプロバイダーとして推進）。

11月 - Microsoft MAI超知能チーム

ヒューマニスト超知能を目指す

MicrosoftがMAI超知能チーム設立発表（人間とより広く人類に常に奉仕する高度なAI能力）。曖昧だが2026年詳細判明予定。Anthropic Claude Code for Web（OpenAI Codexの競合、リモートリポジトリ作業）。

11月 - Google Nano Banana Pro

AI画像生成の次レベル

Google Nano Banana Pro（特にProバージョン、11月末リリース）がAI画像生成を次レベルへ。生成画像の品質と現実性が狂気レベル。Microsoft・NVIDIA・Anthropic戦略的提携（循環投資でAIバブル可能性示唆）。

11月 - Google Gemini 3

現在最強の汎用LLM

Google Gemini 3リリース（現在最強汎用LLMと広く認識）。ベンチマーク圧倒、長コンテキストタスク処理優秀。MS 365 Copilot Fortune 500の90%採用（11月到達、1製品で驚異的エンタープライズ採用）。

12月 - Claude Opus 4.5

AIコーディングキング、競争なし

Claude Code $1B収益到達。Anthropic Bun（JavaScriptランタイム）買収でClaude Codeスケール強化（需要が驚異的）。Claude Opus 4.5リリース（競争なし、AIコーディングキング、Sonnet 4.5も素晴らしかったが Opusが圧倒）。

12月 - DeepSeek 3.2 & GLM-4.7

オープンソースがビッグガイと競合

DeepSeekバージョン3.2（オープンソースLLM、Gemini・Claude・GPTと競合、大きすぎて自ハードウェアで実行困難）。GLM-4.7（AIコーディング特化、DeepSeek 3.2・Claude Sonnet 4.5・GPT 5.1と同等、速く安い）。

12月 - Agentic AI Foundation

Anthropic MCPをLinux Foundationへ寄贈

Anthropic Model Context Protocol（MCP）をAgentic AI Foundation（Linux Foundation配下の専用基金、Anthropic・Block・OpenAI共同設立、多数の大企業サポート）へ寄贈。企業が共通善のためAI構築で協力開始。

12月 - Claude Agent Skills

Progressive Disclosure、MCP置き換え

Anthropic Claude Skills（10月リリース、12月トラクション獲得、MCPの置き換え的存在、コンテキスト最適化）。Progressive Disclosure（エージェントに必要時のみ能力提供、コンテキスト節約）。2026年のエージェントシステム構築で人気パラダイム。

12月 - OpenAI GPT-5.2 & Disney $1B

Gemini 3と競合、Sora注目

OpenAI GPT-5.2リリース（GPT-5.1から大幅改善、Gemini 3と競合、Claude はAIコーディング特化）。Disney $1B投資（Sora注目、AI生成動画＋音声の需要と使用事例示す）。Databricks $4B Series L（$134B評価額、Neon買収）。

12月 - NVIDIA Groq資産$20B買収

買収か資産取得か、2026年1月に詳細判明

NVIDIAがGroq資産を約$20B買収発表。タイトルが「買収」ではなく「資産買収」で奇妙だが、詳細見ると買収的。2026年1月に多くの興味深い展開が判明予定。

12月30日 - Meta Manus買収

2026年Q1-Q2で統合、Meta AI強化

Meta Manus買収発表（12月30日記事）。両社とも既に言及済み。Manusは2025年初めに革命的、Metaは久しくGenerative AIで目立った動き なし。水面下で何か進行中、2026年Q1-Q2で詳細判明予定。ManusをMeta AIを含む製品に統合。
