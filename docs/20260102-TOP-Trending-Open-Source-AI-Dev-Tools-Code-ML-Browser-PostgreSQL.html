TOP Trending Open Source & AI Dev Tools

Code, ML in Browser, PostgreSQL, File Sharing - 7 Powerful Projects

OpenDrop - AirDrop for All

Open-source AirDrop-style file sharing written in Python. Speaks Apple's protocol to share files over Wi-Fi to macOS/iOS from Linux/macOS.

AWDL Protocol Implementation

Uses Open Wireless Link (OWL) implementation of Apple Wireless Direct Link (AWDL) for low-level wireless discovery and file transfer.

OpenDrop Installation

Install via pip, run simple commands to find, send, or receive files. Requires Python 3.6+, AWDL support, and specific libraries.

code-chunk - Semantic Code Splitting

Aware code splitting for AI. Parses source code as structured trees, not text, providing semantically meaningful pieces for embeddings and RAG systems.

Tree-sitter Integration

Uses Tree-sitter to break projects into chunks at real semantic boundaries like classes, methods, imports with rich context (scope chains, siblings).

Multi-Language Support

Supports TypeScript, JavaScript, Python, Rust, Go, Java. Critical for teams building universal search or code assistance tools.

sohez - Personal Portfolio Site

Simple portfolio website built with Next.js and Tailwind CSS. Uses static site generation for fast loading on all devices.

Mysti - Multi-Agent AI Coding

VS Code extension bringing multiple AI assistants together. Pick from Claude Code, Codex, Gemini and combine two in brainstorm mode.

Brainstorm Mode

Agents tackle your prompt and synthesize a single answer. Refactoring, debugging, and architecture suggestions come with broader perspectives.

GitStory 2025 - GitHub Wrapped

Your GitHub year in review. Aggregates contributions, stats, languages used into a cinematic experience similar to music streaming wrapped features.

Visual Analytics

Built with Next.js, TypeScript, Tailwind CSS. Gives developers a sharable format with animated contributions breakdown for impact.

jax-js - JAX-Style ML in Browser

Brings JAX and NumPy ideas to JavaScript. Run numerical and ML code in browser on WebAssembly and WebGPU without server dependencies.

GPU Acceleration

Translates array operations into compiler representation, synthesizes high-performance kernels. NumPy-like APIs for grad, vmap, JIT transformations.

Client-Side ML

Runs entirely client-side with no external dependencies. Makes ML experimentation and interactive demos accessible anywhere a browser does.

pg-aiguide - PostgreSQL AI Guidance

Helps AI coding tools generate better PostgreSQL code by providing semantic search over official Postgres manual for version-aware guidance.

Semantic Documentation Search

When you ask "how do I write this query", tool pulls exact authoritative info from PG docs and guides AI outputs toward correct SQL.

MCP Server Integration

Built with support for MCP servers and plugins. Reduces syntax errors, ensures version correctness for PostgreSQL code generation.

Developer Workflow Transformation

These 7 tools show how fast new ideas are shaping workflows. From file sharing to ML in browser, open-source is driving innovation forward.
