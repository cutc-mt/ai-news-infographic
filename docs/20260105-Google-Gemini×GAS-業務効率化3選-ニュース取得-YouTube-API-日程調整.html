ChatGPT・Gemini何でもOK。「GASコード書いて」指示だけで一瞬生成。Google NewsのRSS情報元にコード作成。拡張機能からApps Script実行。

初心者つまづきポイント①: 名前

シート名とタブ名の混同NG。シート名・タブ名を正確に伝達必須。逆にして伝達・未伝達でエラー頻発。明確区別が重要。

初心者つまづきポイント②: ID

スプレッドシートID伝達必須。URLのD～editの間の英数字列。どのシート対象か判別に必要。未伝達でエラー発生率上昇。

プロンプト: 4-5行で充分

簡潔質問文でOK。「GASコード作成して」＋シート名・タブ名・ID。後はテスト実行で調整。タイトル以外に出展・URL追加も簡単指示。
