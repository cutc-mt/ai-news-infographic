TweakCN完全ガイド

ShadCN UI・Opus 4.5・Gemini 3 Pro・テーマカスタマイズ徹底解説

TweakCNとは: ShadCN視覚エディター

ShadCN/Tailwind専用オープンソースツール。コード書かずに視覚的テーマカスタマイズ。無料、インストール不要。Webサイト訪問のみで利用開始。

フルダッシュボードでリアルタイム確認

tweakcn.com訪問で即利用。チャート・カード・入力欄・カレンダー等全UIコンポーネント表示。データビジュアライゼーションでの色影響確認可能。

組み込みテーマプリセット豊富

Presetsから選択。Doom 64（レトロゲーミング）・Amethyst・Haze・Neon・Sapphire。クリックで全ダッシュボード即座更新。ゼロから不要。

Colorsタブで細密調整

プライマリカラー自由変更。スライダードラッグで全ボタン・アクティブ状態リアルタイム更新。Muted・Card背景・Popover調整。HSL値推測不要。

Typographyでフォント・間隔制御

デフォルトInterから変更可能。JetBrains Mono等選択、Letter Spacing調整。負側でタイトモダン、正側でエディトリアル雰囲気。

Radius: 角丸で印象変化

Border Radius調整が雰囲気決定。高値でピル型（親しみやすい）、ゼロでシャープ角（ブルータリスト・金融系）。チャート・サイドバーで即座反映。

AI Theme Generation: スクショから複製

革命的機能。Linearなど好きサイトのスクショアップロード→カラーパレット自動抽出。Hex一個一個抽出不要。秒単位で適用。

AI生成の精度: 背景・コントラスト分析

単純カラー抽出以上。支配色・背景コントラスト・微妙な光彩まで分析。Linear暗紫ルック完璧再現。インスピレーション→実装の架け橋。

実装: Copy Codeでワンクリック

複雑init不要。Copy Codeボタン→生CSS変数・root定義コピー。globals.cssへ貼付けのみ。既存セットアップ破壊なし。

AIエージェント連携ワークフロー

Cursor・Windinsurf・Verdant対応。globals.cssへ貼付け→AIに「theme設定済み」指示。標準コンポーネント使用で自動継承。

実例: 映画トラッカーアプリ

Next.js空プロジェクトでデモ。TweakCNテーマ貼付け→Verdantに指示→深紅ブランド色自動反映。黒白デフォルト脱却。カスタムフォント・影も適用。

デザイン・コーディング分離戦略

ワークフロー効率化。TweakCNで視覚的デザイン処理、AIでロジック処理。ボタン色・角丸20回指示不要。root levelで処理完結。

汎用テンプレート脱却

AI生成アプリの悩み解決。黒白ShadCNデフォルトから脱却、ユニーク感獲得。あらゆるAIラッパーが同じ見た目問題解消。

Shad Create vs TweakCN比較

役割分担。Shad Create: フレッシュスタート最適、TweakCN: 精密化・完璧化最適。Createはベース色・半径のみ。TweakCNは全デザイントークン制御。

CSS変数でTailwind互換

ShadCNのTailwind/CSS変数活用。AIは変更を知らずともコンポーネントインポートで自動適用。レンダー時にカスタムスタイル反映。

Clerk/Supabase認証コンポーネント対応

ボーナス機能。認証プロバイダーのプリビルトコンポーネントがCSS変数マッピング。ログインフォームも自動スタイル継承。統一感完璧。

オープンソース: GitHub公開

仕組み理解可能。GitHubでビルド方法確認可能。JSON configエクスポートオプションあり。コピペCSS方式が最も確実で全フレームワーク対応。

デザイン苦手な開発者必須ツール

救世主ツール。デザイン苦戦者・AI生成アプリ汎用ルック脱却者必携。手書きCSS不要でユニーク感獲得。Next.js・Vite・Remix全対応。
