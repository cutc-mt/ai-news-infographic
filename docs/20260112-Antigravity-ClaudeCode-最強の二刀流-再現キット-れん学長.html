①X(旧Twitter)で1万ブックマーク超えの「最強の二刀流」

ResDegenさんがXで公開した「Antigravity × Claude Code」の開発術が1万ブックマークを超えて大反響を呼んだ。「自分でもやってみたい」「どこから手を付ければ?」という声が多数寄せられたため、コピペだけで再現できる完全キットが作成された。プロンプト全文、チェックリスト、詰まりポイントTOP5と解決策まで全部入りで、今日からAIでアプリ開発を始められる。

②再現キットの中身: プロンプト・チェックリスト・解決策

再現キットには実践に必要なすべてが含まれる。①Geminiへの設計依頼プロンプト全文、②Claude Codeへの実装依頼プロンプト全文、③環境構築チェックリスト、④詰まりポイントTOP5と回避策、⑤用途別追加プロンプトテンプレート(爆速モード・高品質モード・README生成)。Noteメンバーシップ(月980円、初月無料)で全て読め、単品購入だと2,980円。今後の再現キットも全て読めるようになる。

③実演アプリ: YouTubeサムネイルをNote用にリサイズ

動画内で実演されたのは、YouTubeサムネイル(16:9)をNote記事用(横長)に最適化するアプリ。2MB以下に圧縮し、上下が切れないように余白を追加、余白部分は元画像をぼかして埋める処理を自動実行。ドラッグ&ドロップで画像をアップロードし、プレビュー表示後にダウンロードできる。パステル調の明るく楽しいデザインで、数分で完成した。

④「脳と手の分業」: Gemini × Claude Codeの最強連携

Antigravity × Claude Codeの核心は「脳と手の分業」にある。Gemini(Antigravity内、無料)で設計・計画を練り、Claude Code(月20ドル)で精度の高い実装を行う。相談は税理士にじっくりと、作業は職人でサクッと。Geminiに設計を任せることで、Claude Codeの使用量を大幅に削減でき、数時間で上限に達する問題を回避。お財布にも心にも優しい開発フロー。

⑤なぜ今Claude Codeが熱いのか: 精度の高さと初心者フレンドリー

Claude Codeが注目される理由は3つ。①かなり精度が高く、1発でアプリが作れて修正が少ない。②Antigravityを使えばCLI版(ターミナル版)の黒い画面でのコマンド操作が不要で、ポチッとインストールできる。③初心者でもハードルが低く、最初のステップを踏むのに最適。以前はCLI版のインストールがハードルだったが、Antigravityで劇的に簡単になった。

⑥4つのステップで進める開発フロー

開発フローは4ステップ。①環境構築: AntigravityインストールとClaude Code拡張機能導入。②Geminiへの指示: コピペでプロンプト入力し、仕様書を作成。③Claude Codeへの実装指示: 仕様書を渡してアプリ作成。④調整: 自分の欲しいアプリに仕上げる。ゴールはAIに丸投げでアプリが完成すること。簡単にAIでアプリ開発が体験できる。

⑦環境構築チェックリスト: 3つの準備

環境構築は3つの準備で完了。①Antigravityをインストール: Google公式サイトからダウンロード。②拡張機能インストール: 「Claude Code for VS Code」を検索してインストール。ウニマーク(スパークマーク)が表示される。③Claude Codeログイン: Claude Code ProまたはTeamプラン(月20ドル以上)が必要。Noteに詳細なチェックリストがあるため、初心者でも迷わず進められる。

⑧Geminiへの指示: 「コードは書かないで」が重要

Gemini(Antigravity内)への指示で最も重要なポイントは「コードは書かないでください」。Geminiは設計者・軍師として、タスクと実装計画を作成するだけに徹してもらう。アプリの説明を記載し、「詳細な実行計画を含む新しいフォルダーを作成してください。あなたはコードを書かないでください」と明示。GeminiをArchitectとして使い、実際の大工仕事はClaude Codeに任せる分業が成功の鍵。

⑨実装計画書とタスク定義書の自動生成

Geminiにプロンプトを投げると、自動的にフォルダを作成し、2つのMarkdownファイルを生成する。①implementation_plan.md: 概要、ロジック、デザイン方針、実装手順を記載した実装計画書。②task.md: 具体的なHTML構造、CSSスタイリング、JavaScriptロジックを記載したタスク定義書。これらを日本語で出力するよう依頼でき、壁打ちで内容を修正することも可能。設計が気に入らなければ、この段階で何度でも調整できる。

⑩Claude Codeへの実装指示: フォルダパスを指定するだけ

Claude Codeへの指示は驚くほどシンプル。「作成されたフォルダを見てください。そこに実装計画があります。読んで理解し実行してください。フォルダ名は[フルパス]です」。フルパスはフォルダを右クリックして「パスをコピー」で取得。これだけでClaude Codeが実装計画書とタスク定義書を読み込み、index.html、style.css、script.jsを自動生成する。数分で完成。

⑪Claude Codeの実装プロセス: タスクリスト自動作成と実行

Claude Codeは実装計画を読み込むと、タスクリストを自動作成して実装を開始する。①index.html作成 → ②style.css作成 → ③script.js作成 → ④テスト実行、の順で進行。各ファイル作成時に「Yes」をクリックして承認。ターミナル版より出力が分かりやすく、どのファイルを作成中かが視覚的に把握できる。途中で中断されても「続きをお願いします」で再開可能。

⑫アプリ完成と動作確認: ブラウザで即座に開ける

Claude Codeが実装を完了すると、数分でアプリが完成する。index.htmlを右クリックして「デフォルトブラウザで開く」を選択するだけで、ローカル環境でアプリが起動。以前はnpmコマンドを打つ必要があったが、今は不要。サムネイル画像をドラッグ&ドロップすると、プレビュー表示され、ダウンロードボタンでNote用サイズの画像を保存できる。パステル調デザインで余白にぼかし処理も完璧に実装されていた。

⑬詰まりポイントTOP5: 重要な3つの対策

詰まりやすいポイントとその対策。①Claude Codeが重くなる/遅くなる → 不要なタブを閉じ、CPUとメモリ使用量を減らす。②Geminiがコードを書こうとする → 「計画だけ作ってください。コードは書かないでください」と強調する。③ファイルが見つからない → フォルダのフルパスを右クリック「パスをコピー」で正確に指定。Noteに詳細な解決策とチェックリストが用意されている。

⑭バイブコーディング: ノリを伝えて会話しながら開発

バイブコーディングとは「Vibe(ノリ)」を伝えて会話しながら開発すること。最初のプロンプトで「YouTubeサムネイルをNote用にリサイズするアプリ」と大まかな方向性を伝え、Geminiとの壁打ちで詳細を詰めていく。「もっと明るい感じの楽しいデザインにしてください。パステル調で」のように、自分のイメージを自然言語で伝えると、実装計画書が即座に修正される。会話を通じて自分の欲しいものに近づけていく。

⑮なぜClaude Codeなのか: Antigravity内のClaudeとの違い

Antigravity内でもClaude Opus 4.5 Thinkingが無料で使えるが、Claude Codeを使う理由がある。賢さの差は「熟練エンジニア vs 見習いエンジニア」くらいある。Claude Codeには「ハーネス(馬の手綱)」があり、AIが暴走しないようにしっかり守られている。エージェント的な振る舞い(ファイル作成、プログラム実行など)が最適化されており、コーディング専用に調整されたClaudeだから精度が高い。

⑯用途別追加プロンプトテンプレート: 爆速・高品質・README

Noteの再現キットには用途別の追加プロンプトテンプレートも含まれる。①爆速モード: 開発速度を最優先し、最低限の機能で素早く動くものを作る。②高品質モード: コードの可読性、保守性、エラーハンドリングを重視し、本番環境レベルの品質を目指す。③README生成モード: プロジェクトの手順書(README.md)も自動生成。状況に応じてプロンプトを使い分けることで、開発効率がさらに向上する。

⑰コスト最適化: Gemini無料 + Claude Code月20ドル

この開発フローはコスト最適化されている。Gemini(Antigravity内)は無料から使え、設計・計画という思考重視の作業を担当。Claude Code(月20ドル)は実装という作業重視の部分だけを担当。全てをClaude Codeに任せると、設計から実装まで全て使用量にカウントされ、数時間で月間上限に達してしまう。分業することで、Claude Codeの使用量を最小限に抑え、長期間使い続けられる。お財布にも心にも優しい。
