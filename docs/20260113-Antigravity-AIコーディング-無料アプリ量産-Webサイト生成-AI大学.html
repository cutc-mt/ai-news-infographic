日本語指示でWebサイト・アプリ・ゲームを自動生成するAIコーディングアプリ

①Antigravityとは: 複数AIモデルを横断活用

動画では、2025年11月にリリースされたAIコーディングアプリとして紹介。Gemini 3 Pro、Nova 7 Pro、Claude Opus 4.5などの複数の最先端モデルを横断して活用できる。日本語テキストで指示するだけで、AIエージェントが自動でWebサイト、アプリ、ゲームを作成。

②無料で利用可能: コードが書けなくてもOK

Antigravityの大きな特徴は、コードを書けないユーザーも手軽に利用可能で、なおかつ無料で使える(2026年1月時点)。そのため、現在AI市場において最も勢いのあるアプリの1つとなっている。

③対応プラットフォーム: Windows・macOS・Linux

Antigravityはクロスプラットフォーム対応。Windows、macOS、Linuxに対応したPCにインストールして使うAIを搭載したコーディングアプリ。VS Codeベースで作られており、外観はCursorやWindsurfに似ている。

④AIエージェントの自動処理: 計画から検証まで

チャットフォームに「こういうWebサイトを作って欲しい」とイメージを入力して送信するだけで、AIエージェントが計画を立て、関連ファイルを生成し、コーディングを行い、自らブラウザを立ち上げて検証し、最終的に成果物を提出。これら全工程をAIが自動で実行。

⑤複雑なプロジェクト対応: ユーザーチェックモード

より複雑なWebサイトを作る際には、一定のタイミングでユーザーにチェックを求めるモードも実装。安全性とスピードのバランスを取ることができる。Review-Driven Developmentモードが推奨されている。

⑥CursorやWindsurfとの違い: 最速出力に特化

CursorやWindsurfと同じくMicrosoft VS Codeをベースに作られているため外観はそっくり。しかし、CursorやWindsurfはAIが出したコードを人間が細かくレビューして仕上げる前提だが、Antigravityは中身のコードよりも動く成果物を最速で出力することに特化。

⑦搭載AIモデル①: Gemini 3 Pro・Flash

動画では、2026年1月時点で利用できるAIモデルとして紹介。GoogleのGemini 3 Pro、Gemini 3 Flashを搭載。LM Arenaランキングで総合1位を獲得した最高性能のAIとされている。高速モデルのGemini 3 FlashもGemini 3 Proに劣らない性能。

⑧搭載AIモデル②: Claude Sonnet 4.5・Opus 4.5

コーディング性能の高いAnthropicの推論AIモデルも搭載。Claude Sonnet 4.5とClaude Opus 4.5を使用可能。複数のモデルを切り替えながら、タスクに応じて最適なモデルを選択できる。

⑨搭載AIモデル③: GPT OS 120B

OpenAIのオープンなAIモデルも利用可能。GPT OS 120Bを使うことができる。GoogleやAnthropicに加えて、OpenAIのモデルも選択肢に入る点が特徴。

⑩インストール方法: 公式ページからダウンロード

Windows環境では、公式ページの「Download for Windows」ボタンからアプリをダウンロード。ダウンロードしたアプリデータをダブルクリックし、利用規約に同意して「次へ」ボタンを押していくとインストール完了。macOSやLinuxも同様の手順。

⑪初期設定: 3つの動作モード

インストール後のウェルカム画面で、AIエージェントの動作モードを選択。①Review-Driven Development(推奨・バランス型)、②Agent-Driven Development(高速・確認なし)、③Secure Mode(安全・毎回確認)。後から変更可能なので、迷ったらReview-Drivenを選択。

⑫Googleアカウントでログイン

初期設定の最後に、Googleアカウントへのログイン画面が表示される。「Sign in with Google」ボタンを押すとWebブラウザが立ち上がり、Googleアカウントでログインすればアプリを使える。無料ユーザーも全てのAIモデルを使用可能。

⑬有料プラン: 使用量が増加

無料ユーザーでも全てのAIモデルを使えるが、Google AI ProやGoogle AI Ultraプランに加入すると、無料ユーザーよりも各AIモデルを使える量が増える。個人ユーザーでヘビーに使う場合は有料プランも検討。

⑭画面構成: 左・中央・右の3段構成

Antigravityの画面は3段構成。左側はPC内のフォルダと連動したファイルや拡張機能を検索、中央はコーディングや拡張機能の詳細確認、右側はAIチャット。右側のチャット画面でAIエージェントに仕事を依頼する。

⑮日本語化: Japanese Language Pack

初期画面は英語だが、拡張機能で日本語化できる。拡張機能の検索で「日本語」と入力し、Japanese Language Packをインストール後、青い再起動ボタンを押すと画面が日本語化。チャットの回答も日本語にするには、カスタマイゼーション設定が必要。

⑯便利な拡張機能: Quota・Browser Extension

2つの便利な拡張機能を紹介。①Antigravity Quota: 各AIモデルの残存使用量を確認(右下にAGQアイコン)、②Antigravity Browser Extension: Chrome拡張機能で、AIエージェントが自動でWebサイトを見たり操作できる。

⑰実践例: コーヒー豆販売サイトを作成

実際にWebサイトを作成するデモを実施。PC内にWebフォルダーを作成してAntigravityにドラッグ&ドロップ。「コーヒー豆を販売するWebサイトを作ってください。画像はNova Banana Proで作って仮で入れてください」と指示するだけで、AIエージェントが自動で計画・コーディング・ブラウザ確認・調整を実行。

⑱活用範囲: アプリ・ゲーム・ライティングまで

Antigravityを使えば、Webサイトの他にもアプリやゲームを自動で作ることができる。画像生成AIを実装しているのでUIも優れたものを作れるし、ライティング性能も高いモデルを実装しているので文章執筆や資料要約にも使える。カスタマイズ性が高く、できることが多い。
