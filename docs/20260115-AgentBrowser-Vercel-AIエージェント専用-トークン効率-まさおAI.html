①Agent Browser登場: VercelからAI専用ブラウザ操作ツール

VercelからAgent BrowserというAIエージェント専用のブラウザ操作CLIツールがリリースされた。従来のPlaywright MCPやDevTools MCPと比べて、トークン消費効率が圧倒的に良いのが特徴。なかなか良いツールで、実際に使ってみて使用感が非常に良かった。

②従来の問題点: MCP経由はトークンを大量消費

今までAIエージェントがブラウザを使おうとすると、MCP経由でPlaywright MCPやDevTools MCP、Clean Chromeなどを使っていた。これらを使っていた人は体感したことがあると思うが、トークンを非常に使い、効率が悪いと感じることもあった。

③トークン効率の改善: AIエージェントフレンドリーに進化

Agent Browserでは、トークン消費の問題が結構改善される。元々AIエージェントフレンドリーに設計されており、シンプルに色々な要素を取ってきてくれて、トークンの消費効率も良い。個人的にも非常に良いと感じている。

④CLIから直感的操作: シンプルで使いやすい

Agent Browser自体は、CLIから直感的にブラウザ操作することができ、すごくシンプルに様々な要素を取ってくる。本当にトークンの消費効率も良く、使い勝手が非常に良い。

⑤MCPからの移行: スキル化して運用

動画投稿者は、MCPを一旦剥がして、このAgent Browserで運用していこうと考えている。スキルを作成し、メンバーシップの共有チャンネルで共有している。公式もスキル定義を提供しているので、それを参考にすることも可能。

⑥基本機能: ページ操作・スナップショット・クリック・フォーム入力

Agent Browserでできることは、普通のブラウザ操作と同じ。ページを開く、ページのスナップショットを取る、要素のクリック、フォーム入力など、基本的に普段やっているブラウザ操作をCLIコマンドで実行できる。帰ってくる要素の中身もすごくシンプル化されている。

⑦クッキー操作: MCPにはない機能を搭載

Agent Browserの良い点として、クッキーを直接入れられる機能がある。確かPlaywright MCPなどではクッキーを直接いじるツールはなかったはず。AIエージェントがクッキー操作できるのは非常に面白く、使いたくなる要素が多い。

⑧ネットワーク・デバッグ機能: 充実したCLIコマンド

ネットワーク系のコマンドももちろんあるし、デバッグ機能もある。自分が普段やっているブラウザ操作を、CLIコマンドを組み合わせながら実現できないかAIに聞いて、スキルにしていくのが良い。公式のスキル例を参考にしつつ、カスタマイズすることも可能。

⑨インストール方法: npm installだけ

始め方はめちゃくちゃ簡単。npm install -g @anthropic-ai/agent-browserを実行するだけでインストール完了。これだけで始められるので、拡張性がすごく高い。

⑩基本的な使い方: agent-browser open example.com

AIエージェント経由でなくても直接使える。agent-browser open example.comのようにコマンドを実行すると、特定のサイトにアクセスし、タイトルとURLを返してくれる。非常にシンプルな操作性。

⑪スナップショット機能: サイト構成を簡潔に取得

snapshotコマンドを実行すると、そのサイトの構成がバーッと出てくる。テキストの中身も見えるし、要素もすごくシンプルに指定してくれている。AIエージェントが使う場合はすごく助かる形式になっている。

⑫REFシステム: セレクター不要でID指定のみ

Agent BrowserにはREFシステムが導入されている。セレクターも不要で、IDで要素を指定するだけ。E61やE4063などのIDが表示され、それをクリックコマンドで指定するだけで操作できる。本当に良いシステム。

⑬クリック操作: agent-browser click E69

スナップショットで表示された要素IDを使って、agent-browser click E69のようにコマンドを実行するだけでクリック操作が完了。めちゃくちゃシンプルで、これだけで色々なブラウザ操作ができる。

⑭スクリーンショット撮影: screenshot page.png

クリック操作後に移動したページのスナップショットを取ることも可能。screenshot page.pngのようにコマンドを実行すると、すごくシンプル化した画面情報として取得できる。クリック操作も正常に反応し、スクリーンショットも簡単に撮影できる。

⑮実際の使用感: 無駄なトークンが入らない

実際に使ってみて、使用感がすごく良かった。無駄なトークンがあまり入ってないのが非常に良い。スキルを作ったばかりで、画面をいくつか作らせてみて、スクリーンショットを取らせながら画面作成させたが、非常に快適だった。

⑯スキル化がおすすめ: シンプルで扱いやすい

Agent Browserは、めちゃくちゃシンプルで操作もしやすいので、やっぱりスキル化して使っていくのが良い。これは便利だと思う。多分みんなにおすすめできるツールなので、使ってみた方が良い。使うとしたら、このスキルを入れるだけで済む。

⑰Webアプリ開発に最適: MCPツールより詳細

設定できる部分が、MCPのツールの中にも入っていないくらい詳細な感じで色々提供してくれている。Webアプリを開発していると、すごく便利そうだと感じる。AIエージェントフレンドリーな設計が光る。

⑱まとめ: 全員におすすめできる神ツール

agent-browser openでURL開き、ページタイトルとURLが返ってくる。コマンドをクリック、フィル、URL取得しながら操作を進める。めちゃくちゃシンプルだが、本当にAIエージェントフレンドリー。みんなにおすすめなので、使ってみて感想をコメントで教えて欲しい。
