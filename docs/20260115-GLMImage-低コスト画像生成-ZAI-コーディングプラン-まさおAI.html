GLM Image

低コスト画像生成AIの新星 - Z.AIの注目モデル
まさおAIじっくり解説ch 13分 2026年1月15日

①GLM Imageとは: Z.AIの新画像生成モデル

Z.AIから登場した最新の画像生成モデル。Nano Bananを超え、GPT Imageより優れた画像モデルとして公式ブログで紹介。日本語ラベルにも対応するとのこと。中国系モデルだが日本語も一応対応している。

②ベンチマーク: 高スコアを記録

GLM Imageはベンチマークスコアが結構高いレベルとして共有されている。C.Dream 4.5も良いが、GLM Imageは旧式のGPT Image 1よりは確実に優れている。現状ではNano Banana以外の選択肢として注目。

③FAL AIでの価格: 1枚約0.15ドル

FAL AI経由だと1枚あたり約0.15ドル（約22円）で使える。これはちょっと高め。Nano Banana Proだと1枚30円程度。トップクラスモデルは0.2ドル程度なので、FAL AIは割高感がある。

④Z.AI直接: 驚異の1枚約0.015ドル

Z.AIのAPI経由だと1枚あたり約0.015ドル（約2-3円）で生成可能。FAL AIの10分の1の価格。この価格なら最低限の使い道として十分。GPT Image 1のLowモデル（約0.01ドル）と比較しても競争力がある。

⑤レート制限: 同時生成は1枚まで

API経由での画像生成は同時に1枚までという制限がある。現在GLMが非常に人気のため、同時リクエスト数が制限されている可能性。マックスプランでも以前は8個同時だったのが1個に落ちている。

⑥出力画像の品質: リアル系は良好

宝石などのリアル系画像は結構いい感じに作れる。ただし、どことなく中国感がある印象。寿司の画像などに顕著。ピクセルアートも昔のモデルと比べると全然しっかりしている。

⑦日本語ラベル: やや微妙

アルファベットは結構いい感じだが、日本語はちょっと微妙なところがある。存在しない文字が混じったりする。中国語が強いモデルなので、日本語の使いにくさは感じる。

⑧推奨用途: 差し絵や素材作成

しっかりとした画像を作るなら結局Nano Bananaを使った方が映える。GLM Imageは差し絵や素材を大量に作りたい時に重宝。GPT Image 1.5のLow/Mediumと組み合わせて使うのがおすすめ。

⑨GLM人気の理由: コーディングプラン

GLMが今日本でめちゃくちゃ話題になっている理由はコーディングプランの存在。ライトプランでも月3ドルでClaude Proプランの3倍動かせる。GLM 4.7を使いまくれるのが魅力。

⑩サブスクプラン: ライト3ドルから

ライトプラン3ドル、プロプラン15ドル、マックスプラン30ドルの3種類。おすすめはプロプラン。マックスは使い切れないほどの量だが同時リクエスト制限があるため、現時点ではおすすめしにくい。

⑪OpenAI APIプロトコル互換

エンドポイントをZ.AIに変更するだけでOpenAI SDKからGLMモデルを呼び出せる。CursorやOpen Codeなど様々なツールに組み込み可能。コーディング用途を想定しているので利用規約の確認は必要。

⑫Open Codeとの連携: おすすめの使い方

Open CodeにZ.AIを入れて常にGLMを使い続けるのがおすすめ。Open Codeのハーネスは無料で使えるので、それに合わせてコスパの良いGLMモデルを動かし続けられる。Claude Codeでも可能。

⑬MCP機能: 各種拡張ツール

Vision MCP（画像分析）、ビデオ理解、検索、Web読み込み、ドキュメントサーチなどのMCPが利用可能。プランによって制限があり、ライトプランは100回などの上限あり。Claude Codeの使用感に近づける拡張機能群。

⑭API設定方法: シンプルな手順

API Keysページでキーを生成し、curlやPythonで画像生成可能。SDKも用意されている。コーディングプラン使用時はコーディング用途を想定している点に注意し、利用規約を確認して使用することが重要。

⑮GPT Image 1.5 Lowとの比較

GPT Image 1.5のLowモデルは約0.01ドルで、GLM Image（Z.AI経由0.015ドル）と近い価格帯。乱発するならGPT Image 1.5 Lowも選択肢だが、同時1枚制限がGLMのネック。低コスト系モデルとして両方を検討する価値がある。

⑯GLM 4.7の評価: Gemini 3 Flash級

GLM 4.7モデル自体の評価はGemini 3 Flashみたいな感じで使えるモデルという印象。図を作るのが結構うまく、そこそこ賢い。日本語の文章も全然悪くない。中国系モデルの中では1番おすすめ。

⑰実用例: SVGやガントチャート

GLM 4.7はSVGをしっかり作ってくるし、ガントチャートのアプリも作れる。図を使ったりショットでのプログラムを書かせたりするのが強力。GB 5.2と比べると微妙だが、コスパを考えると十分な性能。

⑱まとめ: 低コスト系の大穴

GLM Imageは低コスト系画像生成モデルとして見るなら全然アリで、大穴かもしれない。Z.AI経由の約3円という価格は魅力的。日本でも話題沸騰中なので、この機会にGLMを試してみる価値がある。
