Thesys C1 - Generative UI API

LLMのテキストレスポンスがグラフィカルUIに変換される革命
Generative AI and InfoBusiness Institute 43分 2026年1月19日

①Thesys C1とは: 世界初のGenerative UI API

LLMのテキストレスポンスをインタラクティブなUIに変換する世界初のAPI。従来のチャットボットは「テキストでしか返せない」という制約があったが、C1はそれを根本から覆す。

②従来との違い: テキストからグラフィカルへ

通常のLLMは「Text to Text」でテキストを返すだけ。C1を使うと、グラフ・チャート・テーブル・カードなどのUIがダイレクトに生成される。ストリーミングでリアルタイム表示も可能。

③ユーザー調査: 83%がインタラクティブを好む

調査によると、83%のユーザーは平坦なテキストよりもインタラクティブな可視化を好む。テキストだけの回答はエンゲージメントを下げてしまう。

④動的UI生成: ハードコーディング不要

事前にUIをハードコーディングする必要がない。AIの回答内容に応じて最適なグラフ・カード・フォーム・画像をその場で生成。コンテキストに適応するUI。

⑤MCP対応: AIエージェントとの連携

CursorやClaudeCode、AntigravityなどのIDEにMCPで統合可能。C1のドキュメントをAIエージェントが参照しながらアプリを自動構築してくれる。

⑥セットアップ: 数時間でアプリ完成

MCPをセットアップし、AIエージェントに指示するだけで数時間でチャットボットアプリが完成。ソネット4.5やOpus4.5と組み合わせて開発が可能。

⑦金融分析デモ: S&P500・日経・ビットコイン比較

「過去5年間のS&P500、日経平均、ビットコインの成長率を比較して」と入力すると、インタラクティブなグラフがストリーミングで表示される。マウスオーバーで詳細も確認可能。

⑧資産シミュレーション: 複利計算の可視化

「年収800万の世帯が毎月10万円を3%・5%で複利運用した資産推移をシミュレート」と入力すると、福利計算グラフがリアルタイムで生成される。

⑨旅行計画デモ: 京都3泊4日の旅程

「京都隠れ名所グルメツアー3泊4日、予算10万」と入力すると、12箇所の観光地がインタラクティブな旅程表で表示。宿泊施設の詳細もボタンで展開可能。

⑩プロジェクト管理: ガントチャート生成

「AIエージェント開発プロジェクト2ヶ月間のロードマップを作成」と入力し、追加で「ガントチャート風のタイムライン」を要求すると、タスク管理UIが自動生成。

⑪技術比較デモ: TypeScript vs Go

「TypeScriptとGoの並行処理の仕組みの違いを図とコード例で説明」と入力すると、コード例付きの比較UIが生成。各言語の特徴が視覚的に整理される。

⑫チャット履歴: コンテキスト維持

チャットボットとして会話履歴を保持。前の質問を踏まえた追加要求が可能。「さっきの内容にグラフを追加して」といった要求にも対応。

⑬Artifacts機能: レポート・スライド生成

新機能のArtifactsでレポートやプレゼンテーションスライドをアプリ内で即座に作成可能。プレゼンテーションモードやスクロールモードに対応。

⑭データ連携: CSV・Excel・Postgres対応

CSVやExcelファイルのアップロード、PostgresなどのDB接続で独自データを可視化可能。自社データをAIで分析・表示できる。

⑮カスタマイズ: ブランド対応

カラー、フォント、ダークモード切り替えなど高度なカスタマイズが可能。ブランドに合わせたデザインでUIを生成できる。

⑯競合比較: ManusやGensparkを超える

「ManusもGensparkもぶっ壊せる」というタイトル通り、レスポンスのリッチさで既存ツールを凌駕。テキストベースのAIアシスタントに革命をもたらす。

⑰高速レスポンス: ストリーミング対応

レスポンスが驚くほど高速でストリーミング表示。グラフィカルなUIがリアルタイムで描画されていく様子は圧巻。

⑱未来の展望: Text to UIの新時代

Text to Text、Text to Image、Text to Videoに続く「Text to UI」の新時代。AIのレスポンスがグラフィカルになることで、ユーザー体験が根本から変わる。
