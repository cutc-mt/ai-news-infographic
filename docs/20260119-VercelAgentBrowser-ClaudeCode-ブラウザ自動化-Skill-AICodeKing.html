①Agent Browserとは: AI専用のブラウザ自動化CLI

Vercel Labsが開発したAIエージェント専用のヘッドレスブラウザ自動化CLI。高速なRust CLIとNode.jsフォールバックを搭載。PlaywrightやPuppeteerの複雑さを解消。

②インストール方法: npm一発でセットアップ

npm install -g agent-browserでグローバルインストール。その後agent-browser installでChromiumをダウンロード。Linuxでは--with-depsフラグでシステム依存も自動処理。

③3ステップワークフロー: Open→Snapshot→Interact

シンプルな3ステップ:①Open（ページ移動）②Snapshot（要素取得）③Interact（操作）。CSSセレクターやXPathの複雑さから解放される。

④参照システム: @E1, @E2で要素を特定

Snapshotでインタラクティブ要素に@E1, @E2, @E3などの参照を自動付与。決定論的で、ページが変わらない限り同じ要素を指す。セレクター不要。

⑤インタラクションコマンド: 豊富な操作セット

click, doubleclick, fill, type, press, hover, check, select, scroll, drag, upload。Webページで必要なあらゆる操作をカバー。agent-browser click @E1のようにシンプルに実行。

⑥セマンティックロケーター: 自然言語で要素検索

find button click --name "submit"のように平易な英語で要素を指定可能。find label "email" fillでラベル付き入力を探して値を入力。可読性と保守性が向上。

⑦セッション機能: 複数の分離環境を並列実行

--sessionフラグで複数の分離ブラウザインスタンスを並列実行可能。各セッションは独自のCookie、ストレージ、履歴を持つ。ユーザーAとBで同時テストも可能。

⑧認証とネットワーク制御

--headersフラグでHTTPヘッダーを注入し認証テストが可能。リクエストインターセプション、ブロッキング、モッキングにも対応。--headedで実際のブラウザ表示も。

⑨Claude Code/Verdantスキル統合

skill.mdファイルでClaude CodeやVerdantに完全なコンテキストを提供。AIがAgent Browserの使い方を完全に理解し、/agent-browserコマンドで即座に活用可能。

⑩スキルセットアップ: 簡単な導入手順

npm root -gでグローバルパッケージを探し、skillsフォルダを.claude/skillsにコピー。またはGitHubから直接curlでダウンロード。数分で統合完了。

⑪Verdantとの連携: 並列エージェント実行

Verdantで複数エージェントを並列実行し、各々がAgent Browserでテスト。一方がログインテスト、もう一方が検索テストを同時に実行。作業時間を大幅短縮。

⑫カスタムコマンド: /webtestの作成例

commandsフォルダにwebtest.mdを作成し、Web自動化専用のスラッシュコマンドを定義可能。テストワークフローを標準化し、/webtestで即座に実行。

⑬ユースケース: 多様な自動化シナリオ

Webスクレイピング・データ抽出、自動テスト、スクリーンショット撮影、フォーム自動入力、価格モニタリングなど。複雑なマルチステップタスクの自動化に最適。

⑭クロスプラットフォーム対応

Mac ARM64/x64、Linux ARM64/x64、Windows x64にネイティブRustバイナリを提供。未対応プラットフォームはNode.jsフォールバック。どこでも動作。

⑮Apache 2.0ライセンス: 自由に利用可能

Apache 2.0ライセンスで完全無料・オープンソース。商用プロジェクトでも自由に使用可能。Vercel Labsによる継続的なメンテナンスも期待できる。

⑯注意点: まだ新しいツール

ドキュメントがまだ発展途上。デフォルトはChromiumのみ対応（Firefox/WebKitは設定で対応可能）。Windowsではやや不安定な報告も。

⑰LLMとの親和性: 同じ言語を話す

Agent BrowserはAIコーディングアシスタントと同じ言語を話すよう設計。単なるブラウザ自動化ツールではなく、AI時代のブラウザインターフェース。

⑱結論: 最高のブラウザ自動化スキル

「これまで使った中で最高のツール＆スキル」とAICodeKingが絶賛。Playwright/Puppeteerのセットアップ不要で、AIエージェントとの統合が格段に簡単に。
