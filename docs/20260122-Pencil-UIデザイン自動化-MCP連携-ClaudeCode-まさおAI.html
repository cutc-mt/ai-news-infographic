Pencil - UIデザイン革命

Claude CodeとMCP連携でUIデザインを自動化する神ツール
まさおAIじっくり解説ch 11分 2026年1月22日

①Pencilとは: AIでUIデザインを作成

AIエージェントと対話しながらUIデザインを作成できるツール。Figmaのような操作感でデザインを構築可能。

②MCP連携: AIエージェントにコンテキスト提供

作成したUIデザインをMCP経由でAIエージェントに渡せる。Claude CodeやCursorと連携してバイブコーディングが可能。

③Figmaライクな操作感

Figmaのような直感的なインターフェース。テキスト、パーツ、レイアウトを組み合わせてUIを構築できる。

④.pen拡張子: Git管理対応

デザインファイルは.pen拡張子で保存。コードと一緒にGit管理できるため、バージョン管理が容易。

⑤VSCode/Cursor拡張

VSCodeやCursorの拡張機能としても利用可能。エディタ内でUIデザインの編集ができる。

⑥Claude Code自動連携

アプリをインストールするとClaude Codeとの連携が自動設定される。グローバル領域にMCPが設定される。

⑦デザインシステム対応

ShadCNなどのデザインシステムがプリセットとして用意されている。これを元にUIを構築できる。

⑧リアルタイム生成アニメーション

UIを生成する際に読み込み中のアニメーションが表示され、コンポーネントが徐々に組み上がる様子が見える。

⑨デモ: 習慣化アプリのUI生成

「行けてる習慣化アプリ」と入力するだけで、デザインシステムに沿ったUIが自動生成される。

⑩ダッシュボード再現デモ

チャット欄から「このダッシュボードのUIを再現して」と指示すると、MCPを介してデザインが再現される。

⑪生成後の編集機能

生成されたUIは自由に編集可能。レイアウト調整、アイコン変更、オブジェクト選択など細かい修正ができる。

⑫スタイルガイド選択

スタイルガイドを選択してデザインのベースを変更可能。画像を渡すこともできる。

⑬メモ・カラー機能

Excalidrawのようなメモ機能も搭載。カラーを入れたり、コンテキストとしてAIに渡せる情報を追加できる。

⑭Figmaインポート対応

メニューからFigmaのデータをインポート可能。既存のデザイン資産を移行しやすい。

⑮AIエージェントベースの設計

ツール全体がAIエージェントフレンドリーに設計されている。コンテキストを効率的にAIに渡せる構造。

⑯個人開発者に最適

UIデザインも開発もする個人開発者には絶対に欲しいツール。デザインシステムをコードと一緒に管理できる。

⑰導入方法: 簡単インストール

pencil.devからダウンロードしてスタート。30分程度で慣れることができる。

⑱まとめ: サブスク出たら課金レベル

まさおAI曰く「サブスク出たら間違いなく課金するレベルで刺さっている」。革命的なUIデザインツール。
