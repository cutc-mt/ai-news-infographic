Craft Agents

オープンソースのClaude Code GUI - 最高のデスクトップアプリ
AI Code King 5分 2026年1月25日

①Craft Agentsとは

AIエージェント用のオープンソースデスクトップアプリ。craft.doチームが開発。ターミナルではなく美しいGUIでClaude Codeを操作できる。

②Claude Agent SDKベース

Claude Codeと同じClaude Agent SDKを基盤に構築。Read、Write、Edit、Bash、Glob、Grepなど全てのコアツールが使える。

③インボックス風インターフェース

メールとタスクマネージャーのハイブリッドのようなUI。ステータス付きの永続的な会話が可能で、履歴も保持される。

④マルチセッション対応

複数のセッションを同時に実行して切り替え可能。別の作業をしてから戻ってもコンテキストが維持される。

⑤3つの権限モード

Explore（読み取り専用）、Ask to Edit（承認必要）、Auto（自律実行）の3モード。用途に応じて切り替え可能。

⑥Exploreモードの活用

ExploreモードでAIに調査・計画させてからAutoで実行するワークフローが便利。ファイル変更の心配なく探索できる。

⑦MCPサーバー対応

GitHub、Linear、Slack、Craftドキュメントなどに接続可能。REST APIやローカルファイルシステムにも対応。

⑧複数ソースからデータ取得

1つの会話でLinearのチケット、GitHubのコード、Craftドキュメントを同時に扱える。手動のコンテキスト切り替えが不要に。

⑨YMLとMarkdownで設定

スキル、テーマ、ステータス、権限はすべてファイルで設定。複雑なUI不要。claude.mdやスキルフォルダに慣れた人なら即座に使える。

⑩32以上のCraftツール

MCP統合により32種類以上のCraftドキュメントツールが利用可能。Craftユーザーならシームレスに統合できる。

⑪インストール方法

GitHubからクローンしてbun install → bun run electron startで起動。Anthropic APIキーまたはClaude Maxサブスクで動作。

⑫モダンな技術スタック

Bun、Electron、React、Shadcn+Tailwindで構築。AES-256-GCM暗号化で認証情報を安全に保存。

⑬マルチファイルDiffビューア

コード変更を美しいUIで確認してから承認可能。ターミナルよりはるかに見やすく、個別に承認・拒否できる。

⑭ドラッグ&ドロップ対応

画像、PDF、Officeドキュメントを会話に直接ドロップ可能。ファイルパス指定やコピー不要でAIが分析・活用できる。

⑮セッションドキュメント化

会話ごとに決定事項と実装理由を記録。チームと共有したりGitHub IssueやPRに添付可能。

⑯ワークスペース分離

プロジェクトごとに完全に独立した環境を構築可能。異なるプロジェクトを混同せずに管理できる。

⑰Apache 2.0ライセンス

オープンソースでフォーク・カスタマイズ可能。アクティブにメンテナンスされており、コントリビュートも歓迎。

⑱ターミナル不慣れな人にも

ターミナルのパワーユーザーでなくてもGUIで直感的に操作可能。チームコラボレーションやマルチタスクにも強い。
