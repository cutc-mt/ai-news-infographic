①Goose: ローカルAIエージェント

開発者向けのオープンソースAIエージェントフレームワーク。自然言語でビルド、デバッグ、ワークフロー実行。MCPサーバーとLLMに接続可能。

②VibeVoice: 音声合成フレームワーク

長尺の会話音声をテキストから生成。ポッドキャストのような複数話者の対話も可能。ローカル動作でストリーミング対応。

③NVIDIA Dynamo: 分散推論

大規模AI推論を複数GPU/ノードで実行。TensorRT、vLLM、SGLangと連携。低レイテンシでスケール可能。

④Browser Use: AI Webオートメーション

自然言語でブラウザを操作するPythonライブラリ。クリック、フォーム入力、スクレイピングをAIが自動実行。

⑤Agentation: AIフィードバックツール

Webページ上で要素をアノテーションしてAIエージェントに伝達。UIの問題箇所を正確に指示可能。

⑥Anti-gravity Kit: エージェントテンプレート

AIエージェントのテンプレートとワークフロー集。ブレインストーミング、デバッグ、機能強化などのスキルを提供。

⑦Qwen3-TTS: 音声合成モデル

Apache 2.0ライセンスのTTSモデル。ボイスクローン、多言語対応、ストリーミング生成が可能。

⑧OpenR Video: ブラウザ動画編集

ブラウザ上で動作する動画エディタ。マルチトラック、トランジション、WebGPUアクセラレーション対応。インストール不要。

⑨Claude Config: 設定リポジトリ

Claude Codeの設定、スキル、スクリプトをまとめたリポジトリ。設定の同期と再利用が簡単に。

⑩Craft Agents: デスクトップAI

GUIでAIエージェントを操作できるデスクトップアプリ。複数セッション、MCP連携、ファイルDiff表示対応。

⑪Marketing Skills: マーケティングAI

Claude Code向けのマーケティングスキル集。CRO、コピーライティング、SEO、アナリティクスなどのワークフロー。

⑫Websockets Live: リアルタイムデモ

WebSocketのリアルタイム通信デモバックエンド。ハートビート、バックプレッシャー制御を学べる。

⑬Compound Product: 自己改善システム

レポートを分析して自動で修正PRを作成する自動化フレームワーク。AIドリブンな継続改善を実現。

⑭Copilot SDK: エージェント統合

GitHub Copilotエージェントを自分のアプリに組み込めるSDK。JS、Python、Go、.NET対応。

⑮Crate Indexer: Rustクレート検索

RustクレートをCLIとMCPで検索・インデックス。セマンティック検索、関数/構造体の一覧表示が可能。

⑯GitHub Copilot CLI

ターミナルで自然言語でコード操作が可能に。ビルド、デバッグ、コード理解をCLIから実行。

⑰Learn Claude Code: チュートリアル

AIエージェントをゼロから構築する学習リポジトリ。Bash、ツールループ、サブエージェント、スキルを段階的に学べる。

⑱Shimmer: 構造化→自然文変換

XML、JSON、YAMLをLLMで自然な文章に変換するPythonライブラリ。バッチ処理対応。
