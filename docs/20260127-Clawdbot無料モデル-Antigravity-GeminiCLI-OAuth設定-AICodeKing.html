Clawdbotを無料で使う方法

Antigravity / Gemini CLIでClaude Opus 4.5やGemini 3 Proに無料アクセス
AI Code King 8分 2026年1月27日

①無料モデルで使えるの？

ClawdbotはClaude Code基盤なのでモデルが必要。しかしAntigravityやGemini CLIで無料モデルが使える。

②Antigravityの無料枠

GoogleのIDEであるAntigravityは寛大な無料枠を提供。Claude Opus 4.5、Sonnet 4.5、Gemini 3 Pro/Flashにアクセス可能。

③組み込みプラグイン

ClawdbotにはAntigravity OAuth用のプラグインが組み込み済み。有効化するだけでモデルにアクセス可能。

④インストール手順

まずnpm install -g clawdbot@latestを実行。その後clawdbot onboard --install-daemonでセットアップ開始。

⑤Antigravity OAuthの有効化

clawdbot plugins enable google-antigravity-oauthでプラグイン有効化。認証フローを自動処理してくれる。

⑥ログイン設定

clawdbot models oauth login --provider google-antigravity --set-defaultを実行。画面の指示に従うだけで完了。

⑦Gemini CLIも同様

Gemini CLIもほぼ同じ手順。google-gemini-cli-oauthプラグインを有効化してログイン。

⑧複数アカウントで制限回避

レート制限に達したら次のアカウントに自動ローテーション。2-3アカウントで制限を実質的に倍増。

⑨モデル一覧の確認

clawdbot models listで利用可能なモデルを確認。正しく設定されているか検証できる。

⑩デフォルトモデルの設定

clawdbot model set provider/model形式で設定。軽いタスクはFlash、複雑なコードはOpusなど使い分け。

⑪Qwen Oauthオプション

さらに無料オプションとしてQwen Portalも利用可能。コーダーモデルとビジョンモデルにアクセス。

⑫Qwenビジョンモデル

スクリーンショットや画像をAIアシスタントに送って分析させることが可能。視覚的なタスクに便利。

⑬メッセージング設定

clawdbot channels loginでDiscord、WhatsApp、Telegramなどを接続。ブラウザボードで監視も可能。

⑭マルチエージェントルーティング

リクエストの種類に応じて自動的に適切なモデルへルーティング。簡単な質問はFlash、複雑なコードはOpusへ。

⑮ローカルモデル対応

Ollamaが自動検出される。クラウドとローカルモデルを混在させて使用可能。

⑯設定ファイル

~/.clawdbot/clawdbot.jsonで詳細設定。許可リストやグループメンションルールをカスタマイズ。

⑰実際の使用感

Antigravityの無料枠は通常使用では十分。極端なセッションでなければ制限に達しない。

⑱セットアップ時間

5〜10分で完了。Discord/WhatsAppから話しかけられる完全なAIアシスタントが手に入る。
