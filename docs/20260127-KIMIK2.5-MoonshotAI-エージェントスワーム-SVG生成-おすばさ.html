KIMI K2.5 レビュー

MoonshotAIの新モデルがOpus 4.5超え？エージェントスワーム機能が特徴
おすばさ 18分 2026年1月27日

①ベンチマークで話題

Humanity's Last Examで50.2%を記録。GPT-5.2、Claude Opus、Gemini 3と比較しても高スコアで話題に。

②エージェンティックコーディング

SWE-Benchで76.8%を達成。コーディング性能も前モデルより向上している。

③API価格

Input $0.6、Output $3でGemini 2/3 Flashと同程度のミドルレンジ価格帯。

④トークン出力速度

1秒あたり約30トークンでやや遅め。KIMIのモデルは以前から出力速度に課題がある。

⑤Agent Swarm機能

K2.5の最大の特徴。複数エージェントによる協調動作機能だが、使うにはサブスク課金が必要。

⑥KIMI Code

オープンソースのCLIツールが登場。ターミナルで動作しIDEとの統合も可能。

⑦サブスク価格

サブスクは高め。この価格ならClaude ProやMaxに課金したくなる水準との評価。

⑧OpenRouterで利用可能

すでにOpenRouterで公開済み。すぐに試すことができる状態。

⑨SVG生成が優秀

ドラえもんのSVG生成でトップクラスの品質。GPT-5.2やFlashよりも良い結果。

⑩UI作成が得意

ウェブデザイン、LP作成、スライド生成が特に強い。表現力のあるUIを生成できる。

⑪スライド生成の可能性

ワンショットでのスライド生成は中国系モデルの中でトップレベル。バナナのイラスト自動挿入も可能。

⑫ライティングが強い

日本語の文章が読みやすい。EQが高いと評されたK2の特徴を継承。文字数もしっかり。

⑬ロジック実装に課題

パスファインディングが動かない、物理シミュレーションが微妙など複雑なロジックは苦手。

⑭賢さはやや物足りない

ベンチマーク上は高いが、実際に使うとGemini・Claude・GPTには劣る印象。

⑮GLMとの比較

GLM 4.7は月$3からで安い。コスパではGLMが優位だが、ライティングはKIMIが好み。

⑯中国系モデルの注目株

GLM、Qwenなどと並ぶ中国系LLMの注目株。特にUI生成とライティングで存在感。

⑰Claude Codeの代替には？

エージェンティックな動きはするが、Claude Codeの代わりには使わないというのが正直な評価。

⑱まとめ

UI生成・SVG・スライド・ライティングは優秀。サブ利用としては良いモデル。Swarm機能の正式公開に期待。
