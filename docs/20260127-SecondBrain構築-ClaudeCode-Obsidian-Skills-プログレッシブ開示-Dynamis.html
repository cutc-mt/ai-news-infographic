Claude Code + Obsidian + Skills

第二の脳（Second Brain）を構築して毎週数十時間を節約する方法
Cole Medin / Dynamis 20分 2026年1月27日

①Claude Codeはコーディングだけじゃない

Claude Codeを「コーディング専用」と思うのは間違い。第二の脳として使うことで全く新しい可能性が広がる。

②第二の脳の3つの目的

自動化やAIスロップ生成ではない。目的はアイデア発想・整理・リサーチの3つ。人間を置き換えるのではなく拡張する。

③コーディングエージェントの5つの能力

ファイル操作、検索、ターミナル実行、Web検索、コードインテリジェンス。コードインテリジェンス以外は汎用アシスタントとして使える。

④なぜObsidianか

すべてがローカルのMarkdownファイルとして保存される。LLMが理解しやすく、Claude Codeがファイルシステムで直接操作可能。

⑤Obsidianはキャンバス

第二の脳が生成するPowerPoint、YouTubeスクリプト、投稿アイデアをすべてObsidianで管理。人間のタッチを加える場所。

⑥Notionではダメな理由

NotionはMCPサーバー経由でしかアクセスできない。ファイルシステムで直接操作できるObsidianが優位。

⑦Skillsの重要性

自分のプロセスやガイドラインをClaude Codeに与える方法。柔軟でコンテキスト効率的な機能拡張を実現。

⑧プログレッシブ開示とは

すべてのツール情報を最初から渡さない。短い説明だけ渡して必要な時にロード。数十〜数百の機能でも混乱しない。

⑨3層の開示構造

1層目：スキル説明、2層目：skill.mdの全体ロード、3層目：cookbookフォルダの個別スクリプト参照。

⑩Brand & Voice Generator

自分のブランドとトーンを定義するスキル。質問に答えるだけでパーソナライズされた設定が生成される。

⑪PowerPoint Generator

Python実行でスライドを生成・編集。ブランドに合わせたPowerPointがプロンプト一つで完成。GammaやAnthropic公式より強力。

⑫MCP to Skillスキル

MCPサーバーをスキルに変換するツール。コンテキストを肥大化させずにZapierなどを接続可能。

⑬Zapier連携の例

Gmail、Googleカレンダー、Slack、Asanaに接続。読み取り専用に制限してセキュリティを確保。

⑭その他のスキル例

Excalidraw図生成、YouTubeスクリプト、Remotionで動画Bロール生成、LinkedInポスト作成など多数。

⑮リサーチエンジン

最新AIニュースを調査し、キュレートしたソースからコンテンツアイデアを提案。毎日のアイデア出しを支援。

⑯Skill Creatorスキル

Anthropic公式のスキル作成ツール。自分専用のカスタムスキルを効率的に構築できる。

⑰Global Rulesでの調整

MCP使用時のゴッチャ（落とし穴）をCLAUDE.mdに文書化して信頼性を向上。システムを徐々に改善。

⑱5〜10分で始められる

テンプレートリポジトリをクローンして.claude/skillsに配置。すぐに全スキルが利用可能。毎日が楽しくなるシステム。
