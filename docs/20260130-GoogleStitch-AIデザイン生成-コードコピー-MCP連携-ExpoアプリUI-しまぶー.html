Google Stitch

AIでプロ級アプリデザインを生成し、コードとしてそのまま開発に活用
しまぶー 15分 2026年1月30日

①Stitch: Google製デザイン生成サービス

Googleが提供するデザイン作成用Webサービス。AIにざっくりした指示を出すだけでプロ級のアプリUIデザインを生成。スマホアプリとWebサービスの両方に対応。現在ベータ版で無料枠あり。

②ふわっとした指示で高品質な出力

「AIでスマホの背景画像が作れるアプリ」程度のざっくりした指示の方が力を発揮する。約1分で複数画面のデザインを生成。日本語への変換も「全部日本語で書いてください」の一言で対応。

③コードとしてコピー可能

生成したデザインを右クリックで「Copy as Code」してコードをコピーできる。そのコードをClaude CodeやCursorに貼り付ければ、デザインそっくりのアプリを実装可能。

④追加ページの提案機能

「他に追加した方がいいページはありますか？」と聞くと、オンボーディングフロー、課金画面、プロンプトアシスタントなど必要なページを提案してくれる。画面遷移図もテキストでまとめてくれる。

⑤Expoで実際にアプリを再現

StitchのデザインをコピペしてClaude Codeに渡し、Expoプロジェクトで実際にスマホアプリとして再現。オンボーディングフロー、ホーム画面、課金画面など、約3分でUIが完成した。

⑥MCPサーバーも提供

StitchはMCPインテグレーションも用意。認証設定後にMCPを起動すればコピペ不要でデザインデータからそのまま忠実に再現可能。Figma MCPと同じ使い方でリンクを貼り付けるだけ。

⑦Webサービスでも活用可能

スマホアプリだけでなくWebサービスのデザインにも対応。「ミニマルな感じで」とテーマ指定すればミニマリスト的なランディングページも生成。デプロイまで一気通貫で開発できる。

⑧テーマ・スタイル指定のコツ

デフォルトでも高品質だが、「ミニマルな感じで」「ゴージャスな感じで」などスタイルのテーマを指定するとより自分の求めるデザインが得られる。モデルはGemini 3 Flashでも十分な品質。

⑨まとめ: デザインの弱点が克服された

数ヶ月前は「AIのデザインは微妙」「紫グラデーションばかり」と言われていたが、Stitchの登場でデザインの課題がほぼ克服。AIの弱点を探すのではなく、共存してビジネスを加速させる方向が重要。
