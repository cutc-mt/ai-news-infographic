PersonaPlex, Git-AI, OpenTUI, Vault など注目のオープンソースツール
ManuAGI 15分 2026年1月30日

①Persona: フルデュプレックス音声AI

リアルタイム音声対話を実現するオープンソースのSpeech-to-Speechモデル。同時に聞きながら話せるフルデュプレックス対応で、割り込みや高速なターンテイキングをサポート。Moshiアーキテクチャベースでローカル実行可能。

②Superpowers: エージェント向けスキルフレームワーク

AIコーディングエージェント向けの再利用可能なワークフローを提供するスキルフレームワーク。OpenAI CodexやOpenCodeなどのツールと連携し、計画・コードレビュー・タスク実行の一貫性を向上させる。

③Awesome LLM Apps: LLMアプリ実例集

RAG、AIエージェント、マルチエージェント、音声ワークフローを使った実践的なLLMアプリケーションのリファレンスコレクション。OpenAI、Anthropic、Google、XAI、オープンソースモデルに対応。

④Godoxy: セルフホスト型リバースプロキシ

Docker連携のリバースプロキシとコンテナオーケストレーションツール。Dockerラベルまたは Web UIで設定可能。TLSサポート、サービスディスカバリ、アクセス制御を備え、ホームラボの複雑さを軽減する。

⑤Hive: LLMオブザーバビリティプラットフォーム

アプリ内のLLM使用状況をAPI使用量・レイテンシ・コストをリアルタイム監視するプラットフォーム。複数のAIプロバイダーを統合するチーム向けで、SDKを通じてアプリコードに直接接続する。

⑥Tree Pilot: ツリーベースのエージェント実行

エージェントの意思決定とワークフローを線形プロンプトではなく分岐パスとして構成するプロジェクト。代替案の探索と状態管理をより明確にし、高度な計画・意思決定エージェントに有用。

⑦Launchpad: Reactベースの動画制作ツールキット

Next.jsとTailwind CSSを使ったプログラマティックな動画生成プラットフォーム。動画プロジェクトをコードで構成し、共有アニメーションコンポーネントを再利用できる。

⑧Solid: 仮想DOMなしのリアクティブUIライブラリ

テンプレートを直接DOMノードにコンパイルし、状態変更があった箇所だけを更新するJavaScriptライブラリ。仮想DOMを使わず高速で宣言的なWebアプリを構築できる。

⑨Git-AI: AI生成コードの追跡ツール

AIが生成したコード変更をGitワークフロー内で追跡するCLIツール。コーディングエージェントがAI生成行をマークし、コミット時にGit Notesにメタデータを保存。rebase、merge、squash、cherry-pickにも対応。

⑩Critique: ターミナルでのGit Diffレビュー

ターミナルでGit Diffをシンタックスハイライト・分割表示・単語レベル差分で確認できるツール。ブラウザやIDEを開かずにコードレビューを高速化する。

⑪OpenTUI: TypeScriptのターミナルUIライブラリ

フロントエンドUIのようなコンポーネント手法でリッチなTUIをTypeScriptで構築できるライブラリ。再利用可能なUIパターンでCLIアプリのUXを向上させる。

⑫Claude Code: ターミナル型エージェントコーディング

ターミナルで直接動作し、プロジェクトコンテキストを理解してルーチンタスクを実行するエージェント型コーディングツール。ローカル開発とリポジトリ操作に密接に統合されている。

⑬Learn Claude Code: AIコーディングエージェント学習ガイド

「モデルをエージェントとして設計する」「ツール使用」「タスク追跡」「サブエージェント構造」などのコアメカニクスを段階的な例で解説する教育リポジトリ。

⑭Vault: シークレット管理プラットフォーム

APIキー、パスワード、証明書を暗号化と厳格なアクセスポリシーで管理するセキュリティプラットフォーム。監査ログと複数の認証方式をサポートし、クラウドやDevOps環境に統合可能。

⑮PageIndex: AI検索用ドキュメントインデクサー

ページレベルのコンテンツをRAGスタイルのワークフローに対応する検索可能な単位にインデックス化するパイプライン。大量のドキュメントセットと検索システムを扱う開発者向け。

⑯Clip.js Copilot: CLIPベースのマルチモーダルコパイロット

CLIPベースの埋め込みを使ってテキストと視覚的理解を接続するAIコパイロット体験を構築するプロジェクト。マルチモーダルAIツールのプロトタイピングに最適。

⑰Pi Mono: 開発者向けモノスペースフォント

コードとターミナル用に設計されたクリーンで読みやすいモノスペースフォント。エディタとターミナル環境のカスタマイズに最適なミニマルなタイプシステム。

⑱SuperMemory: AIアプリ用メモリレイヤー

知識の保存・検索・呼び出しを行うパーソナルAIメモリレイヤー。単一プロンプトを超えた長期記憶が必要なAIアシスタントやエージェントシステム向けのプラットフォーム。
