Antigravity + Stitch

完全無料でフルスタックAIアプリを構築 — BM25検索エンジンをノーコードで実装

①Antigravityの真価: 無料で最先端AI IDE

GoogleのAntigravityは完全無料でトップティアモデルにアクセス可能なAI IDE。推論、マルチモーダル理解、エージェンティックワークフローを搭載。フロントエンド設計からバックエンド構築、DB管理、アプリ自動化までカバーする。

②完全無料のフルスタック構成

今回のテックスタック構成: Stitch（UI設計）→ Antigravity（コーディング）→ TigerData（バックエンド）→ Vercel（デプロイ）。認証はO.js、決済はStripe。全て無料ティアで構築可能。一行もコードを書かずにフルスタックアプリが完成する。

③Google Stitch: AIによるUI/UXデザイン

Stitchは無料のAI UIデザインエージェント。説明文を入力するだけでプロダクションレディなフロントエンドコンポーネントを生成。Gemini 3 FlashやThinkingモデルを選択可能。ワイヤーフレームやスクリーンショットを参考画像として提供することもできる。

④Stitch MCP連携でAntigravityと接続

StitchがMCPサーバーをリリースしてAntigravityと直接接続可能に。Stitch設定でAPIキーを作成し、AntigravityのMCP設定にペーストするだけ。IDE上からデザインの反復と修正ができるようになった。

⑤TigerData: エージェント対応のPostgress

TigerDataはAIエージェントが安全にデータを管理できるAIネイティブなPostgresインスタンスを提供。無料ティアあり。MCPサーバー経由でAntigravityと接続すれば、テーブル作成、データ保存、クエリ実行、DB分岐テストまで自律的に実行できる。

⑥BM25検索エンジンをPostgres内で構築

オープンソースのPostgres拡張機能を使い、Elasticsearchや外部検索システム不要でBM25ランキング検索を実装。Postgresから離れずにリアルタイムの関連性ランキング検索が可能。Antigravityが実装計画に基づいて全て自動でセットアップした。

⑦Planning Mode + Opus 4.5 Thinkingで設計

AntigravityのPlanning Modeを使い、Opus 4.5 Thinkingモデルで詳細な実装計画を作成。プロジェクトゴール、テックスタック、各機能の実装手順を記述。この計画があることでAIエージェントが決済やDB、検索の統合まで一貫して実行できる。

⑧自律的なフルスタック実装

実装計画を実行すると、AntigravityがBM25拡張のインストール、テーブル作成、サンプルデータ挿入、インデックス作成まで全て自律的にコードアウト。ファイル編集、ターミナルコマンド実行、ブラウザ操作を含む完全なエンジニアリングタスクをエンドツーエンドで処理。

⑨完成したAI検索アプリの機能

完成した「Second Brain」アプリの機能: ナレッジベースへのファイルアップロード、AIによる文書要約、フォローアップ質問、ライト/ダークモード。最近のアクティビティ表示やプロフィール管理も完備。個人用AI検索アシスタントとして実用的な完成度。

⑩BM25ランキングの実際の動作

SQLエディタでBM25検索スクリプトを実行すると、サンプルデータに対して関連性スコアで順位付けされた結果が返される。低いスコアがより良いマッチを示す、本格的な検索エンジンと同じ仕組み。Postgres内で完結するのが最大の利点。

⑪認証とStripe決済の統合

O.jsによる認証ではGitHubやGoogleアカウントでのサインインが可能。Stripeの決済ゲートウェイも統合され、サブスクリプション管理ができる。APIキーとOAuthキーをAntigravityに提供するだけで、エンドポイント接続まで自動処理。

⑫ゴーストインプット: 自動タイプ検索デモ

TigerDataスクリプトによるゴーストインプット機能でリアルなユーザークエリを自動シミュレーション。入力された検索クエリがPostgresのBM25検索エンジンに直接送られ、関連性の高い結果がリアルタイムで返される。

⑬Vercelへの無料デプロイ

フロントエンドとバックエンドのサーバーレスファンクションをVercelで無料ホスティング。スケーラブルでインフラ管理不要のセットアップが完成する。開発からデプロイまで全工程が無料ツールで完結するのがこの構成の強み。

⑭Stitch MCPの実践的な活用法

Stitch単体でもUI生成は可能だが、MCPでAntigravityと接続するとIDEから直接デザインの反復が可能に。モバイル/Webアプリのどちらにも対応。コンポーネントの注釈編集、バリエーション追加、ヒートマップ予測なども使える。

⑮TigerData MCPでDB操作を自動化

TigerDataのMCPサーバーをAntigravityに接続すると、AIエージェントがDB操作を直接実行。サービス情報の取得、テーブル操作、データ確認が自然言語で可能に。モニタリングタブでセキュリティやログの追跡も行える。

⑯オープンソース検索拡張の統合手法

テキスト検索リポジトリはオープンソースで公開されている。Antigravityの実装計画に組み込むだけで、手動ビルドやインストール不要。拡張の有効化、テーブル作成、サンプルデータ挿入、BM25インデックス作成まで自動で実行される。

⑰実装計画の重要性

詳細な実装計画を最初に作成することで、AIエージェントが全機能を一貫して実装できる。決済、データベース、検索、認証、デプロイの各工程をテックスタックに沿って正確に処理。プロンプトは可能な限り詳細に書くのがコツ。

⑱まとめ: 無料ツールで本格的なAIアプリ構築

Stitch、Antigravity、TigerData、Vercelの組み合わせでフルスタックAI検索エンジンが完全無料で構築可能。BM25ランキング、OAuth認証、Stripe決済、サーバーレスデプロイまで全てノーコード。Antigravityの自律コーディング能力がこの無料エコシステムの核になっている。
