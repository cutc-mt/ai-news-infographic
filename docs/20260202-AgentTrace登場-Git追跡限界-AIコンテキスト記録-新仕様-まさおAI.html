Agent Trace登場: Gitで追えないAIのコンテキストを記録する新仕様

Devin・Cursor・Cloudflare・Vercelが支持する、AIエージェント時代の追跡標準

2026年2月2日 まさおAIじっくり解説ch 約12分

①Agent Traceとは

Agent Traceは、AIエージェントのコンテキストを記録するための新しい仕様です。Cognition（Devinの開発元）が公開し、Cursor、Cloudflare、Vercel、Julesなどが支持しています。バージョン0.1.0が2026年1月に公開されました。AIがコードを書く時代に、「なぜその変更をしたのか」という推論プロセスを追跡可能にすることを目的としています。

②Gitの限界: 1995年の設計思想

Gitは1995年の設計思想に基づいており、「誰が」「いつ」「何を」変更したかは追跡できるが、「なぜ」その変更をしたのかは記録できません。コミットメッセージで理由を書くことはできますが、AIエージェントの推論プロセスや意思決定のコンテキストまでは保存できないのが現状です。

③AI時代に必要な情報

AI時代には従来のGitでは追えない情報が必要になります。具体的には、推論プロセス（AIがどのような判断でコードを書いたか）、人間とAIの帰属（どの部分を誰が書いたか）、使用されたAIモデルの情報、そして意思決定のコンテキストなどが求められています。

④JSONフォーマットの仕組み

Agent TraceはJSON形式でデータを記録します。記録される内容には、会話履歴（プロンプトとレスポンス）、使用されたモデル情報、人間とAIの帰属情報、コード変更の詳細が含まれます。構造化されたフォーマットにより、さまざまなツール間でデータの共有が可能になります。

⑤設計目標: 相互運用性と拡張性

Agent Traceの設計目標は大きく2つあります。1つ目は相互運用性（インターオペラビリティ）で、異なるAIエージェント間でデータを共有できること。2つ目は拡張性で、ファイルレベルだけでなく行レベルの帰属追跡まで対応できる柔軟な設計になっています。

⑥コントリビュータータイプ: 人間かAIか

Agent Traceでは、コードの作成者を3つのタイプに分類します。「human」（人間）、「AI」、「unknown」（不明）の3種類です。これにより、コードベースのどの部分が人間によって書かれたのか、どの部分がAIエージェントによって生成されたのかを明確に区別できるようになります。

⑦モデルIDの記録

Agent Traceは使用されたAIモデルのIDを記録します。たとえばAnthropic Claude、OpenAI GPT-4、GPT-5.2など、具体的なモデル名とバージョンが保存されます。これにより、特定のコード変更がどのモデルによって生成されたのかを後から確認でき、モデルのパフォーマンス比較にも活用できます。

⑧行レベルの帰属追跡

Agent Traceはファイル単位だけでなく行単位での帰属追跡をサポートしています。コンテンツハッシュを使用することで、コードがファイル間で移動されたり、ファイル内で位置が変わったりしても帰属情報が維持されます。VCSサポートとしてGitやその他のバージョン管理システムに対応しています。

⑨所有権クエリの方法

所有権を調べる流れは次の通りです。まずVCS blame（git blame）でリビジョンを取得し、次にそのリビジョンに対応するAgent Traceを検索して、最後に該当する範囲の帰属情報を取得します。この仕組みにより、「このコードは誰が（人間かAIか）、どのような理由で書いたのか」を簡単に追跡できます。

⑩SWE-Benchで3ポイント改善

Devinのブログによると、Agent Traceを活用することでSWE-Benchのスコアが約3ポイント改善したと報告されています。コンテキストの記録と共有が、AIエージェントのコーディング能力の向上に直接つながることを示す重要な成果です。追跡可能性がパフォーマンス改善にも貢献しています。

⑪マルチエージェント時代への展望

Agent Traceは複数のAIエージェントが協調して作業する時代を見据えた仕様です。Cursor、Windsurf、Devin、OpenCode、V0など、さまざまなAIエージェントが同じプロジェクトで作業する際に、誰がどの部分を担当したのかを統一的なフォーマットで記録できるようになります。

⑫まとめ: コードからコンテキストへ

Agent Traceは、ソフトウェア開発の追跡対象を「コード」から「コンテキスト」へと拡張する画期的な仕様です。マルチエージェント時代の標準規格となる可能性を秘めています。Cognitionを中心に、業界の主要プレイヤーが支持しており、AIとの協業がより透明で管理しやすいものになることが期待されます。
