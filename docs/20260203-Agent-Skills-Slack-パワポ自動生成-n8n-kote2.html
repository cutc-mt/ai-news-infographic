Agent Skills: Slackで「パワポ作って」→自動完成

2026年2月3日

①Agent Skillsとは

  • AIエージェントに仕事のやり方を教えるマニュアル。
  • SKILL.mdというマークダウンファイル形式。
  • AI界隈で急速に注目され始めた技術。
AIエージェントに「こうやって仕事してね」と教えるマニュアル！

②なぜAgent Skillsが必要か

  • AIの出力はブレやすい。
  • 毎回同じ品質を出すのが難しい。
  • Skillsで正確で一貫したアウトプットを実現。
「毎回同じクオリティで」を実現するための仕組み！

③SKILL.mdの構造

  • タイトル: スキルの名前。
  • 説明: 何をするスキルか。
  • 手順: 具体的なステップ。
  • テンプレート: 出力形式の例。
マークダウンで書くから人間にも読みやすい！

④Claudeで作ってみる

  • Claudeにやりたいことを説明。
  • SKILL.mdを自動生成してもらう。
  • 必要に応じて微調整。
Skills自体もAIに作ってもらえる！便利！

⑤Claude Codeで作ってみる

  • Claude Codeでより実践的に。
  • プロジェクトに直接Skillsファイルを作成。
  • そのまま動作確認が可能。
開発環境で直接Skillsを作って試せる！

⑥デモ: Slackでパワポ作成

  • Slackで「パワポ作って」と投稿。
  • AIエージェントが自動で処理。
  • 社名ロゴ入り資料が完成。
Slackに一言書くだけでプレゼン資料ができる！

⑦n8nとの連携

  • n8nでワークフローを構築。
  • SlackとAgent SkillsをAPI連携。
  • 自動化パイプラインを実現。
n8nでSlack→AIエージェント→出力を全自動化！

⑧Agent SkillsをAPI化

  • SkillsをAPIとして公開。
  • 外部ツールから呼び出し可能に。
  • 様々なシステムと連携できる。
SkillsをAPIにすれば、どこからでも呼び出せる！

⑨pptxスキルの例

  • Anthropic公式のpptxスキルが公開中。
  • 社名ロゴやブランドカラーを設定。
  • テンプレートに沿った資料生成。
GitHubで公開されてるSkillsを参考にできる！

⑩社名ロゴの自動挿入

  • Skillsにロゴのパスを設定。
  • 生成される資料に自動でロゴが入る。
  • ブランド統一が簡単に実現。
毎回ロゴを入れる手間から解放！

⑪ワークフローの流れ

  • Slackでメッセージを送信。
  • n8nがWebhookで受信。
  • Agent Skills APIを呼び出し。
  • パワポファイルを生成・返却。
全自動でSlackに資料が届く！

⑫正確でブレないアウトプット

  • Skillsに従って一貫した出力。
  • AIの「気まぐれ」を防止。
  • 品質管理が容易に。
誰が使っても同じクオリティの資料ができる！

⑬agentskills.ioサイト

  • agentskills.ioで公式情報。
  • ドキュメントやサンプルが充実。
  • コミュニティでSkillsを共有。
公式サイトでSkillsの作り方を学べる！

⑭Anthropic公式Skills

  • GitHubで公式Skillsを公開。
  • pptx生成スキルが利用可能。
  • フォークして自分用にカスタマイズ。
Anthropic公式のお手本Skillsを参考にしよう！

⑮注意点: プロンプト設計

  • Skillsは明確に書く必要がある。
  • 曖昧な指示は避ける。
  • 試行錯誤で改善していく。
Skillsの書き方次第でアウトプットの質が変わる！

⑯注意点: テストが重要

  • 作成したSkillsは必ずテスト。
  • エッジケースを確認。
  • 複数回実行して安定性を確認。
本番投入前にしっかりテストしよう！

⑰活用シーン

  • 定型レポートの自動生成。
  • メールテンプレートの作成。
  • データ分析レポートの生成。
  • 契約書・提案書のドラフト。
繰り返し作業は全部Skillsで自動化できる！

⑱まとめ: AIに仕事を教える時代

  • Agent SkillsでAIに仕事のやり方を教える。
  • 正確でブレないアウトプットを実現。
  • n8nなど外部ツールと連携して業務自動化。
Slackで一言→社名ロゴ入り資料が完成する未来！
