n8nとUnstractでPDFデータ抽出を完全自動化する方法

2026年2月9日 ｜ World of AI

①概要: PDFデータ抽出を完全自動化

  • AIエージェントでデータ抽出を自動化できることは知られているが、構築が難しくコストも高い。
  • 完全無料かつローカルで動作するソリューションを紹介。
  • PDFなどの非構造化ドキュメントから構造化データを自動的に抽出するワークフローを構築する。
手作業でPDFからデータを転記する時代は終わり！オープンソースツールで自動化しよう。

②使用ツール: 2つのオープンソースの組み合わせ

  • Unstract: 非構造化ドキュメントを構造化データに変換するオープンソースツール。
  • n8n: オープンソースのAIワークフロー自動化ツール。マルチステップのエージェント自動化に最適。
  • この2つを組み合わせることで、PDFを受け取り → データを抽出 → Google Sheetsに出力、まで完全に自動化できる。

③Unstractとは: LLM搭載の文書処理エンジン

  • LLMWhispererがUnstractの中核エンジンとして動作。大規模言語モデルを活用して文書を解析する。
  • プロダクショングレードのドキュメント処理を実現。正確性、スケール、コンプライアンスに対応。
  • 任意のLLMと組み合わせて使用可能。柔軟なカスタマイズが可能。
手書き文字や複雑なテーブルも正確に読み取れるのがUnstractの強み！

④Unstractプレイグラウンド: 無料で即座に試せる

  • サインアップ不要で、ドキュメントをアップロードして即座に試すことができる。
  • レシートの例では、名前、電話番号、メールアドレス、金額テーブルなど細かい情報を正確に抽出。
  • 数値テーブルの読み取りも高精度。手書き文字が混在するドキュメントにも対応。

⑤n8nのローカルセットアップ: 超簡単に始められる

  • npxコマンド一発で起動可能。Node.jsさえあればすぐに使える。
  • Dockerでのデプロイにも対応。コンテナ化された環境で安全に運用できる。
  • 完全にローカルで動作するため、データがクラウドに送信されない。プライバシーも安全。
npx n8n

⑥n8nアカウント設定: 無料ライセンスの取得

  • ローカルホストで起動後、まずアカウント作成が必要（無料）。
  • 無料ライセンスキーをメールで受け取ることで、高度な機能にアクセスできるようになる。
  • 有料機能へのアクセスが無料で提供されるため、コスト負担なしで始められる。
ライセンスキーの取得を忘れずに！高度な機能が使えるようになるよ。

⑦n8nテンプレート: 7,888種類のワークフロー

  • ダッシュボードにはすぐに使えるテンプレートが豊富に用意されている。
  • データ入力、マーケティング、ドキュメント処理など、幅広い分野をカバー。
  • 今回はゼロからワークフローを構築するが、テンプレートを活用すればさらに素早く始められる。

⑧Unstractカスタムノード: n8nへの統合

  • npm installコマンドでUnstractカスタムノードをn8nにインストール。
  • 設定 → コミュニティノード → インストールの手順で簡単に追加できる。
  • Unstract APIとの接続設定を行い、n8nからUnstractを直接呼び出せるようにする。
npm install n8n-nodes-unstract

⑨ワークフローエディタ: ノードベースの直感的設計

  • n8nのエディタはノードベースで、ドラッグ＆ドロップで自動化ワークフローを構築できる。
  • 手動トリガー、アプリイベントトリガーなど複数の起動条件に対応。
  • 各ノードを線でつないでデータフローを視覚的に設計可能。
プログラミング不要で、ビジュアルにワークフローを組み立てられるのが嬉しい！

⑩活用例: Gmailインボイス自動処理

  • Gmailノードを設定し、メール受信時にワークフローを自動起動。
  • メールに添付されたインボイスをUnstractが自動で処理し、構造化データに変換。
  • 抽出されたデータをGoogle Sheetsに自動で出力し、スプレッドシートとして整理。
請求書処理を完全自動化できれば、経理作業が劇的に楽になる！

⑪フォーム送信トリガー: チャットボット風の入力

  • フォーム送信をトリガーにしたチャットボット型ワークフローを構築。
  • ファイルアップロード要素を追加し、複数のPDFを同時に送信できるようにする。
  • 内部チームや個人的なユースケースにそのまま活用可能。

⑫条件分岐: バイナリデータの処理フロー

  • IFトリガーを使い、PDFのバイナリデータを正常に処理できるかを判定。
  • 処理可能な場合はUnstractノードへ転送し、非構造化データを構造化。
  • エラーハンドリングにより、不正なファイルや破損データを事前に除外。

⑬Google Sheets連携: 抽出データの自動出力

  • Googleアカウントを認証し、Google Sheetsと直接連携。
  • インボイスの日付、番号、送信元、送信先、合計金額などのフィールドを事前に定義。
  • Append Row機能で、処理されたデータをシートに自動追記。
Google Sheetsだけでなく、任意のデータベースやCRMにも出力先を変更できる！

⑭実演: 手書きインボイスの処理テスト

  • テストでは手書きインボイスという非常に難易度の高いドキュメントを使用。
  • ワークフローを実行し、フォームからファイルをアップロードして送信するだけ。
  • 数秒でUnstractがドキュメントを処理し、結果を返してくれる。

⑮抽出結果: 高精度なデータ取得を実現

  • インボイス番号を正確に抽出。手書きの複雑な番号も問題なく認識。
  • 送信元（From）と送信先（To）の情報も正しく構造化。
  • $3,000、$4,000、$5,000などの各項目の金額を正確に読み取り、合計も算出。
手書きの乱雑なインボイスでもここまで正確に読み取れるのはすごい！

⑯カスタマイズ: 抽出フィールドの自由設定

  • 抽出するデータのフィールドは自由にカスタマイズ可能。
  • インボイスに限らず、レポート、契約書、申請書など任意のドキュメント形式に対応。
  • 自動処理のスケジューリングも設定可能で、定期実行にも対応。

⑰オープンソースの利点: 無料で始められる

  • n8nとUnstractの両方がオープンソースで、無料で利用可能。
  • ローカル環境で完結するため、機密文書のデータ漏洩リスクがない。
  • クイックスタートガイドが用意されており、要件とインストール手順が明確に文書化されている。
機密性の高い書類もローカルで安全に処理できるのがオープンソースの最大の魅力！

⑱まとめ: 時間とコストを大幅に節約

  • 手作業で数時間かかっていたPDFデータ抽出が、数分で完全自動化される。
  • フリーランス、中小企業、経理担当者など、誰でもAI自動化の恩恵を受けられる。
  • n8n + Unstractの組み合わせにより、ドキュメント処理の自動化が誰にでも手の届くものに。
両方ともオープンソースだから、今すぐ無料で始められる。まずはUnstractのプレイグラウンドで試してみよう！
