①今回の概要: 公式ツールが大量登場

  • MCP・CLIツール周りで公式ツールが大量に登場している。
  • その中から話題性の高い厳選4つのツールを実際に触ってレビューする。

②MCPとスキルの比較: どちらが優れている？

  • スキルの方がコンテキストを最適化できるという利点がある。
  • 一方でMCPの方がより広く普及しており、エコシステムが充実している。
  • 用途に応じて使い分けるのがベストな選択肢だ。
スキルは精度重視、MCPは汎用性重視
と覚えておくとよい！

③Draw.io MCPの導入方法: NPXで簡単セットアップ

  • NPXコマンドで簡単にインストールできる。
  • Claude Codeの.mcp.jsonファイルから設定を行う。
  • 導入のハードルが非常に低く、すぐに使い始められる。
NPXで一発インストール！
設定もJSONを書くだけで簡単！

④Draw.io MCPの機能: 3つのフォーマットに対応

  • XMLフォーマットでDraw.ioの図を作成できる。
  • CSVフォーマットからも図の生成が可能。
  • Mermaid記法にも対応しており、3つのフォーマットから自由に選べる。
XML・CSV・Mermaidの3種類！
用途に合わせて選択できる！

⑤Draw.io MCPの実演: 自動で図を生成

  • 「SNSの詳細図を作成して」と指示するだけで詳細な図が自動生成される。
  • 生成される図の品質が高く、実務でも十分に使えるレベル。
  • 非常に実用的なツールとして強くおすすめできる。
自然言語で指示するだけで
本格的な図が完成する！

⑥Draw.io MCPの応用: Mermaidからの変換

  • Mermaid記法からDraw.ioの図を作成することも可能。
  • 人間が操作しやすい形式で出力されるので、後から手動で編集もできる。
  • 既存のMermaidファイルをDraw.ioに変換するワークフローが便利。
MermaidからDraw.ioへの変換で
図の見た目が一気にレベルアップ！

⑦Excalidraw MCPの導入方法: やや手間がかかる

  • リポジトリをローカルにクローンして導入する方法がある。
  • もしくはダブルクリックでClaude Desktopにインストールする方法もある。
  • Draw.io MCPと比べると導入の手順がやや面倒。
NPX一発のDraw.ioと比べると

⑧Excalidraw MCPの機能: スライド作成が可能

  • Excalidrawフォーマット（JSON形式）でスライドなどを作成できる。
  • 手書き風のスタイルでプレゼンテーション資料を自動生成する。
  • 作成した図はリンクで簡単に共有することができる。
手書き風のスタイルが特徴的！
JSON形式で出力される！

⑨Excalidraw MCPの評価: あまりおすすめしない

  • 導入が面倒な割に、出力の品質がそこまで高くない。
  • 自作のスキルの方が品質が良いという結論に至った。
  • 手間と品質のバランスを考えると、あまりおすすめできないツール。
導入が面倒なのに品質が微妙…
自作スキルの方がよさそう！

⑩Excalidraw MCPの実演: スライド3枚を生成

  • 「アートとデザインの解説スライド3枚」と指示して生成を実行。
  • 生成された図はリンクから共有できるが、品質はあまり高くない。
  • 見た目が期待したほどではなく、実用レベルには届かない印象。
リンクで共有はできるけど

  • Catalystライセンスを持つユーザー限定のアーリーアクセスCLIツール。
  • コマンドラインからObsidianを直接操作できるようになる。
  • まだ実験的な段階だが、今後の発展が期待されるツール。
Catalyst課金ユーザー限定！
公式CLIツールがついに登場！

  • デイリーノートの読み書きがコマンドラインから可能。
  • タスク管理機能もCLIから操作できる。
  • リンク管理もコマンドラインから一括操作が可能になった。

⑭Happyモバイルアプリの概要: スマホからAI開発

  • スマホからClaude CodeやCodexを操作できるモバイル&Webクライアント。
  • 外出先やソファからでもAIコーディングができるようになる。
  • PC上で動作するCLIツールにスマホからリモート接続する仕組み。

⑮Happyの導入方法: NPMでグローバルインストール

  • NPMでグローバルインストールしてPC側のサーバーを起動する。
  • スマホ側では専用のHappyアプリをダウンロードする。
  • PC側のサーバーとスマホアプリを接続すれば準備完了。
npm install -g でPC側を準備して
スマホにアプリを入れるだけ！

⑯Happyの実演: スマホからClaude Codeを操作

  • スマホから「こんにちは」と送信すると、PC上のClaude Codeが動作を開始する。
  • PC側のターミナルでClaude Codeがリアルタイムに応答する様子を確認できる。
  • リモートからの操作でも遅延なくスムーズに動作する。
スマホでメッセージを送ると
