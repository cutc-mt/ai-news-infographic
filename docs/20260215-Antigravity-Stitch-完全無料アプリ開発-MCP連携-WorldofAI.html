Antigravity + Stitch: 完全無料アプリ開発

Googleのデザインエージェントとコードエディタの最強コンボ

2026年2月15日 | WorldofAI

①Stitch: 完全無料のAIデザインツール

Googleの完全無料AI搭載UIデザインツール。1つのプロンプトから本番品質のインターフェースを生成。Google AI Studio・Figmaとも連携可能。

完全無料で使える！

②Antigravity: Googleのエージェント型エディタ

自律的に推論・記述・リファクタ・実行できるGoogleのエージェント型コードエディタ。単なるコーディングツールではなく、AIエージェントとして動作。

自律実行エージェント

③Stitch MCP: 2つを繋ぐ接続レイヤー

Googleが静かにローンチしたStitch MCPが2つのツールを接続。UIデザイン・コンポーネント・レイアウト構造をエクスポーズ。

接続レイヤーが鍵！

④Stitch Agent Skills: スマートな振る舞い

Agent Skills for Stitchが再利用可能なスマートな振る舞いを追加。デザイン・フルスタックコード生成・バリデーションなどが可能。

Agent Skills再利用可能

⑤なぜStitch単体では不十分か

Stitch単体では素早いUIモックアップしか作れない。MCP + Antigravityを使えば、エージェントが実際の動くコードを構築し、手動コピペ不要で継続的に反復可能。

実際に動くコードが必要！

⑥セットアップ: Googleアカウントだけ

Googleアカウント1つで完全無料。APIキー不要。StitchとAntigravityをインストールしてGoogleアカウントでログインするだけ。

無料API不要

⑦MCPサーバーのインストール手順

Antigravityを開き、エージェントタブの「...」→MCP Servers→Stitchを検索→インストール。Stitch APIキーを取得して貼り付け。

MCP Servers → Stitch → Install

⑧Stitch Agent Skillsのインストール

ターミナルでコマンドを実行すると全スキルがインストール。デザインMD（設計システム文書化）・React変換・Stitch Loop・プロンプト強化など。

スキルを入れると出力が向上！

⑨デモ: マルチページウェブサイトを自動生成

プロンプト1つで完全なマルチページウェブサイトを生成。Stitchがスクリーンを生成し、Antigravityがコーディングとドキュメント作成。

マルチページ完全自動

⑩Stitch Canvasでライブプレビュー

Stitch APIに接続しているため、Stitch Canvasでライブプレビューが可能。エージェントがキャンバス内で直接デザインを表示。

リアルタイムで確認！

⑪バックエンドも同時にコーディング

UI生成だけでなく、フロントエンドのバックエンド部分もコーディング。設計システム実装計画・タイポグラフィ・パレット・コンポーネントのドキュメントも作成。

フルスタックドキュメント

⑫npm run devで即座に起動

生成されたアプリはnpm run devで即座に起動。美しいコンポーネントが完全に機能し、複数ページ・ボタン・アニメーションも動作。

すぐに動く！

⑬Canvasで視覚的に反復

Stitch Canvasを使えば視覚的にデザインを確認・反復可能。右クリックで注釈追加、エリアを選択してAIに反復指示。5つのバリエーションを同時生成可能。

Canvas視覚的

⑭高品質ランディングページの例

Apple 2025デザインにインスパイアされた高級感あるページを生成。典型的なAI生成に見えない高品質。開発者が数週間かけるレベル。

AI生成に見えない品質！

⑮5つのバリエーションを同時生成

最大5つのバリエーションを同時生成。カスタムプロンプトでレイアウトを変更したり複数ページを作成。ハンドでパンして他のデザインに移動。

5バリエーション同時生成

⑯Figma・Antigravity・AI Studioへエクスポート

生成したデザインはFigmaにエクスポート、ZIPでダウンロード、またはMCP経由でAntigravityに接続。AI Studioでバイブコーディングも可能。

複数のエクスポート先！

⑰エージェントに変更を指示

Antigravityを使えばエージェントに直接変更を指示。「もっと良くして」と言うだけで自律的に改善。デザイン変更もコード修正も自動。

自律改善指示ベース

⑱まとめ: 無料でプロ品質のアプリを作る

Stitchはデザイン、Antigravityは実行。この組み合わせで、ダッシュボード・アプリ・ウェブサイト・UIを完全無料で構築可能。

無料でプロ品質を実現！
