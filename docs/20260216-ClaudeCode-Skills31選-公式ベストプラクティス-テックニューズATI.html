③cache-components (Vercel)

Next.jsの最新キャッシュ戦略を学べるスキル。

  • 使わないと時代遅れの設定を使い続ける
  • useCache + cacheLifeで細やか制御
  • コンポーネント単位で戦略変更可能
従来のall-or-nothingから脱却！

④zustand (LobeHub)

状態管理のベストプラクティスを学べる。

  • ストア設定パターン
  • アクション実装
  • セレクター最適化（重要！）
無駄な再レンダリングを防いでパフォーマンス改善！

⑤i18n (LobeHub)

React i18nの正しい使い方を教えてくれる。

  • 使わないと翻訳管理が破綻
  • フラットキー構造を推奨
  • キーの検索性と管理性が向上
多言語で苦しまないための必須スキル！

⑥update-docs (Vercel)

ドキュメント更新を自動化するスキル。

  • Git diffで変更ファイルを分析
  • 影響するドキュメントを特定
  • 更新・検証・コミットまで自動化
ドキュメント負債を貯めない必須スキル！

⑦release-note-generation (Microsoft)

リリースノートを自動生成するエンタープライズ品質。

  • GitHubマイルストーンのPRから抽出
  • Features/Bugs/Breakingに自動分類
  • 高品質なリリースノート生成
手作業のチェンジログから解放！

⑧coding-agent (OpenClaw)

AIエージェントを統一制御する技術的に重要なスキル。

  • PTY通信が必須（バッシュ経由はNG）
  • 通常のバッシュ呼び出しは出力破損
  • execl + async + ptyで正しく呼び出し
エージェント連携でつまづいている方は必見！

⑨pr-creator / code-reviewer

Google公式のPR作成・レビュースキル。

  • PRテンプレートを自動検出
  • プリフライトチェックで品質担保
  • ローカル・既存PR両方をレビュー
テンプレートに沿わないPR失敗を防止！

⑩testing (LobeHub)

壊れにくいテストを書くためのVitestスキル。

  • vi.mockの乱用はNG（リファクタで壊れる）
  • vi.spyOnで直接依存をスパイ
  • 実際の振る舞いをテスト
メンテナンス不能なテストから脱却！

⑪webapp-testing (Anthropic)

静的サイトと動的Webアプリの判断基準。

  • 静的HTMLならソースを直接読む
  • 動的アプリはPlaywrightで操作
  • MCPサーバー経由で正しく起動
判断を間違えると時間を無駄にする！

⑫backend-patterns

Node.js/Express/Next.jsのアーキテクチャパターン。

  • リポジトリーパターン
  • サービスレイヤー
  • エラーハンドリングの正解
動くコードと保守しやすいコードの違いを理解！

⑬SpringBoot系スキル

Javaエンタープライズ開発のベストプラクティス。

  • springboot-patterns: REST API・キャッシング
  • springboot-security: 認証・認可実装
  • jpa-patterns: N+1問題対策
Springを使うなら必須のスキルセット！

⑭golang-patterns

イディオマティックなGoパターン集。

  • Javaのクラス設定を持ち込まない
  • if err != nil のエラーハンドリング作法
  • ゴールチンとチャネルの正しい使い方
Goを始めたばかりの方に特におすすめ！

⑮FastAPI系スキル

Python API開発のベストプラクティス。

  • fastapi-templates: 本番対応テンプレート
  • Pydantic V2 + SQLAlchemy 2.0対応
  • fastapi-htmx: HTMX組み合わせ時の注意点
PythonでAPIを作るなら必須！

⑯prisma-expert

Prisma ORMの完全ガイドで地雷を回避。

  • 本番では必ずmigrate deployを使う
  • migrate devを本番で実行はデータ消失危険
  • N+1問題・マイグレーション・スキーマ設定ミス対策
TypeScriptでORMを使うなら必須！

⑰セキュリティスキル

認証・セキュリティは「なんとなく動く」では危険。

  • auth-implementation-patterns: JWT/OAuth/Session/RBAC
  • security-review: XSS/CSRF/インジェクション対策
  • トークン保存場所・権限チェックの正しい実装
認証機能を実装する前に必ず確認！

⑱ツール・アーキテクチャースキル

開発効率を上げるユーティリティ系スキル。

  • slidev: Markdownでスライド作成
  • session-logs: jqでログ分析
  • command-development: カスタムコマンド作成
  • parallel-agents: 独立タスクの並列処理
自分だけのワークフローを自動化！
