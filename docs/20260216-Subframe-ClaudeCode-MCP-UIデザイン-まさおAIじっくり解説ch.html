Subframe × Claude Code

AIでUIデザインを高速構築！MCP連携でデザインシステム同期

2026年2月16日 | まさおAIじっくり解説ch

①Subframeの概要

SubframeはAIを使ってUIを高速で構築する次世代のUI設計プラットフォームです。

  • ドラッグ&ドロップのビジュアルエディタ
  • 本番対応のReact + Tailwind CSSコード生成
  • デザインとコードの境界線をなくす
Figmaのようなデザインと実コードを統合した画期的なツール！

②Pencilとの比較

PencilとSubframeは競合関係にあります。それぞれ特徴が異なります。

Subframeの強み
• MCP連携
• コンポーネント同期
• システム設計が整理済
Pencilの強み
• シンプルな仕組み
• フローが外部に出ない
• 管理しやすい
どちらも一長一短あり。開発フローに合わせて選ぼう！

③MCP連携の特徴

Claude Codeと直接連携してデザインを生成できます。

  • テールウィンドのテーマをローカルと同期
  • コンポーネントをSubframeからコードへ同期
  • スキル（Setup / Design / Develop）で操作
MCP Model Context ProtocolでAIエージェントと連携！

④セットアップ・アカウント登録

始め方はとても簡単です。

  • メールまたはGoogleで認証スタート
  • クイックスタイルでデザインオンボーディング
  • 人気サービスのテンプレートから選択可能
ChatGPTなど有名サービスのスタイルをベースにカスタマイズできるよ！

⑤デザインオンボーディング

初期設定でブランドカラーを選択してテーマを決めます。

  • プライマリーカラーを選択
  • グレイ系（Zinc, Slate等）を選択
  • フォントスタイル（モダン等）を選択
  • コーナーラディウスを調整

⑥Claude Codeプラグインの導入

ターミナルからコマンドを入力してClaude Codeと連携します。

  • 公式サイトからコマンドをコピー
  • ターミナルでコマンドを実行
  • 認証を実施して接続完了
Cursor、Replit、Codexにも対応！

⑦スキル（Setup / Design / Develop）

プラグインを入れると3つのスキルが追加されます。

  • Setup プロジェクト設定
  • Design UIデザイン生成
  • Develop コード化
公式ガイドに従ってセットアップすれば、難しい設定不要！

⑧プロジェクト構成・コンポーネント一覧

プロジェクトにはコンポーネントがデフォルトで用意されています。

  • Avatar（アバター）
  • Accordion（アコーディオン）
  • Toast（トースト通知）
  • shadCN風のコンポーネント群
shadCNに似たコンポーネントが最初から使える！

⑨コンポーネント同期機能

Subframeからコードベースへ同期できますが、一方通行なので注意。

  • Subframe → コードベースの同期
  • 逆方向の同期は非対応
  • コンポーネントはプレゼンテーション用
一方通行という点には要注意！

⑩新規コンポーネント作成

ビジュアルエディタでコンポーネントを作成できます。

  • レイアウトを調整
  • テキストやボタンを配置
  • シャドウや角丸を設定
  • コードレベルでのプレビューも可能
ビジュアルエディタのUXには改善の余地あり...

⑪テンプレート・カタログ機能

既製テンプレートを利用してデザインを始められます。

  • 好みのテンプレートを探す
  • Forkしてエディタで開く
  • コードも共有・プレビュー可能
ポートフォリオやボイラープレートとして活用できる！

⑫テーマ設定

細かいデザインシステムの設定が可能です。

  • ブランドのプライマリーカラー
  • デフォルトフォント
  • エラー色の設定
  • タイポグラフィ設定
  • シャドウの設定
tailwind.configとしてコピーできる！

⑬AIデザインページ生成

Claude Codeからスラッシュコマンドでページ生成を指示できます。

  • /subframe-design コマンドを実行
  • Subframe内部のAIがデザイン生成
  • 複数のパターンを提示
「ネオンでエモいポートフォリオ」のような指示で生成可能！

⑮AI生成UIのプレビュー

生成されたUIはプレビューURLで確認できます。

  • URLが自動発行される
  • 複数パターンから選択可能
  • 選択したパターンを編集可能
4つ程度のパターンを提示してくれる！

⑯プロトタイプ・アノテーション機能

デザインにコメントや注釈を追加できます。

  • サイト上で共有されるコメント
  • アノテーション（注釈）機能
  • Developモードで読み取り可能
アノテーションを元にAI編集で修正できる！

⑰チャットベースのAI編集

Subframe内部のAIとチャットでデザインを修正できます。

  • 反応速度が速い
  • 特定箇所を指定して修正
  • パーソナライズも可能
  • オープンサンドボックス機能も
Gem系のツールと似た感覚で使える！
