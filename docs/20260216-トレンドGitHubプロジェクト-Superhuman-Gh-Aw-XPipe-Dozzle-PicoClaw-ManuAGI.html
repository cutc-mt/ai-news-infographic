週刊 #232 - Superhuman, Gh-Aw, XPipe, Dozzle & PicoClaw

2026年2月16日 | ManuAGI - AutoGPT Tutorials

①Superhuman Reasoning

Google DeepMindの研究リポジトリでAI推論評価用のデータセットとツールを提供します。

  • 推論評価用データセット
  • 数学・論理問題解決の研究
  • 再現可能な実験環境
AI研究者向けの推論ベンチマーク理解に必須！

②Gh-Aw (GitHub Agentic Workflows)

GitHub CLI拡張機能でAIエージェントがリポジトリ内で自動化を実行します。

  • 自然言語Markdownでワークフロー定義
  • Issue分析・PRレビュー自動化
  • サンドボックス環境で安全に実行
複雑なCI設定不要でAI自動化！

③Grok2API

GrokモデルをOpenAIスタイルAPIで公開するFastAPIサーバーです。

  • チャット・画像生成対応
  • ストリーミングレスポンス対応
  • MySQL/Redis/PostgreSQL対応
ローカルまたはDockerで動作可能！

④QMD

ローカルCLIセマンティック検索で個人ナレッジベースを検索します。

  • ドキュメント・ノートをインデックス
  • 完全ローカルでプライバシー重視
  • ターミナルから直接クエリ
クラウド依存なしで知識を検索！

⑤Awesome LLM Apps

LLMアプリケーションの実例集でアーキテクチャパターンを学べます。

  • RAGパイプラインの実装例
  • エージェントシステムの例
  • OpenAI/Anthropic/Gemini対応
次のAIプロジェクトのアイデア探しに！

⑥Eigent

AIエージェント搭載デスクトップ環境でタスク管理を一元化します。

  • エージェントとワークスペース統合
  • セルフホスト型アプリ
  • 開発者ワークフロー向け
AIアシスト付きデスクトップコラボレーション！

⑦World Monitor

グローバルインテリジェンス監視ダッシュボードです。

  • グローバルニュース集約
  • 地政学的シグナル分析
  • リアルタイム状況把握
セルフホストで独自監視ダッシュボードを構築！

⑧GB Studio

ドラッグ&ドロップでレトロゲームを作成できるビジュアルエディタです。

  • コーディング不要でゲーム作成
  • ROMファイルでエクスポート
  • イベント駆動型ワークフロー
レトロハードウェア風ゲームを簡単制作！

⑨Add-MCP

MCP統合を素早く追加するCLIツールです。

  • NPXコマンドで簡単セットアップ
  • 設定ファイル自動生成
  • エージェントとツールの相互運用性
数秒でMCPツールをプロジェクトに接続！

⑩Gondolin

AIエージェント用セキュアサンドボックスでコードを安全に実行します。

  • 軽量MicroVMで隔離実行
  • ファイルシステム/ネットワーク制御
  • 認証情報漏洩防止
エージェント生成コードを安全に実行！

⑪XPipe

サーバーとシェルアクセスの統合ハブです。

  • SSH/Dockerを統合インターフェース
  • パスワードマネージャー連携
  • リモートマシン設定不要
複雑なインフラをラップトップから一元管理！

⑫Dozzle

Dockerコンテナログをリアルタイムで表示する軽量ウェブビューアです。

  • ブラウザでライブログ表示
  • ログ保存なしで軽量動作
  • デバッグ・運用時に便利
重いログスタック不要でコンテナ監視！

⑬TRIP

セルフホスト型旅行計画プラットフォームです。

  • インタラクティブマップで計画
  • テレメトリなしでプライベート
  • 旅行仲間とコラボレーション
商用ツールの代わりに自分でホスト！

⑭Happy

コーディングエージェントをリモート操作できるモバイル/Webクライアントです。

  • 暗号化接続で安全
  • 入力/承認待ちを通知
  • デスクトップから携帯へシームレス
デスクトップを超えてエージェントを管理！

⑮PicoClaw

超軽量個人AIアシスタントでGo言語製です。

  • 高速起動・低メモリ使用
  • 最小ハードウェアで動作
  • 自己完結型エージェントシステム
組み込み環境や軽量デバイスで動作！

⑯Claw Compactor

AIエージェントのコンテキストを圧縮するPythonツールです。

  • メモリファイル・トランスクリプト圧縮
  • LLM呼び出しなしで決定論的処理
  • 段階的サマリーでロード
複雑なマルチエージェント履歴を効率化！

⑰AntFarm

専門AIエージェントチームのオーケストレーションツールです。

  • プランナー・開発者・テスター役割
  • 構造化ワークフローで協調
  • 単一コマンドでチーム起動
手動オーケストレーション不要でエージェントチーム構築！

⑱Mdvi & Clawra

ターミナルMarkdownビューアとビジュアルエージェントの2つのツール。

  • Mdvi: Vim風ナビゲーションでMD表示
  • Clawra: エージェントに画像生成機能
  • Discord/Telegramでビジュアル応答
ターミナル体験とエージェントの視覚的プレゼンスを向上！
