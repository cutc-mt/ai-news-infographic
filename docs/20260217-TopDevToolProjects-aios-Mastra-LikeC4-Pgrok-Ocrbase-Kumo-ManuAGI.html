①新チャンネル: AI Agent Studio

AI Agent Studioという新しいYouTubeチャンネルを開設。AIエージェントプロジェクト、チュートリアル、ツールに特化したチャンネルで、最新のオープンソースAIエージェントプロジェクトを週次で紹介。

②aios-core: フルスタック開発ワークフロー

JavaScriptベースのCLIフレームワーク。AIエージェントと構造化ワークフローでソフトウェア作成を組織化。エージェント、タスク、ワークフローを統合プロジェクト構造で調整。

③Pimono: ラズベリーパイ用シングルバイナリ

Raspberry Piシステム用の軽量ランタイム環境。単一の実行可能ファイルとしてアプリをパッケージ化し、依存関係をコンパクトなバイナリにバンドルして展開。

④Awesome Native macOS: ネイティブMacアプリ

Swift、SwiftUI、AppKitなどのネイティブ技術で構築された軽量macOSアプリのキュレートされたディレクトリ。Electronや重いランタイムなしで高パフォーマンスなデスクトップツール。

⑤LikeC4: アーキテクチャ図をコードで記述

ドメイン特化言語でソフトウェアシステムを記述し、ダイアグラムを即座に生成。アーキテクチャ定義をソースファイルに保存し、Gitで追跡。モデルの変更に応じてダイアグラムを自動更新。

⑥Mastra: TypeScript AIエージェントフレームワーク

AIアプリケーションと自律エージェント構築用TypeScriptフレームワーク。モデルルーティング、エージェント推論、ワークフロー Orchestratration、評価ツールを1つのスタックで提供。

⑦Oat: ゼロ依存UIコンポーネントライブラリ

超軽量HTML＆CSS UIライブラリ。フレームワークやビルドツールなしで共通インターフェースコンポーネントを提供。最小限のスタイリングとアクセシビリティをネイティブHTMLパターンで実現。

⑧TinyFish Cookbook: Web自動化レシピ

TinyFish Web Agent APIを使用した自動化レシピのコレクション。Webサイトの自動化、フォーム入力、動的サイトからの構造化JSONデータ抽出などをデモ。

⑨SkillAudit: AIエージェントスキルスキャナー

AIエージェントスキル定義を分析し、セキュリティリスクとドキュメント問題を検出するCLIツール。危険なコマンド、不足しているドキュメント、互換性問題をスキャン。

⑩RLM: 長コンテキスト用再帰推論エンジン

超長コンテキストを処理するための言語モデル推論フレームワーク。全てをプロンプトに読み込む代わりに、データを再帰的に処理。コンテキスト制限を削減し、大きなプロンプトによる劣化を回避。

⑪Ocrbase: セルフホストOCR文書抽出

PDFを構造化データに変換するドキュメント処理プラットフォーム。オープンウェイトOCRモデルとAPI駆動ワークフローを使用。WebSocketでジョブ進行状況をリアルタイム報告。

⑫Kubecost-web: Kubernetesコスト可視化

OpenCostデータベースのWebインターフェース。Kubernetesインフラ支出をリアルタイムで把握。クラスタ使用状況と配分をシンプルなブラウザダッシュボードで表示。

⑬Ponping: 軽量アップタイム監視

分散エージェント「Peons」を使用した監視プラットフォーム。複数の場所から定期的なPingを送信し、結果を一元管理。Webインターフェースでサービス健全性とレイテンシを表示。

⑭Kaku: macOS音楽プレイヤー

オンラインストリーミングをサポートするmacOSデスクトップ音楽プレイヤー。Web音楽再生を軽量スタンドアロンアプリに統合。検索、キュー、再生を直接実行。

⑮TotalRecall: 個人データ用セマンティックメモリ

個人ファイルとノート over検索可能なセマンティックメモリ層を作成する開発者ツール。埋め込みを使用してローカルコンテンツをインデックスし、自然言語クエリを有効化。

⑯Pgrok: PostgreSQL自然言語インターフェース

自然言語プロンプトでPostgreSQLデータベースをクエリするCLIツール。平易な英語リクエストを実行可能SQLに変換し、生成されたSQLと共に構造化結果を返す。

⑰AgentViewer: エージェント可視化ツール

AIエージェントの思考と行動を表示するWebベース可視化ツール。メッセージ、ツール呼び出し、推論ステップを構造化タイムラインで表示。実行中のインタラクションをリアルタイムレンダリング。

⑱BeautifulMermaid: SwiftネイティブMermaid

SwiftライブラリがMermaid図を直接ネイティブAppleアプリケーション内にレンダリング。WebViewへの依存を解消し、ネイティブSwift UIコンポーネントを使用して図を変換。

⑲CodeToWorld: コードを3D環境に変換

ソースコードを対話型3Dシーンに変換する研究フレームワーク。関数、モジュール、関係を空間表現にマッピング。リポジトリを分析し、ナビゲート可能なワールドを生成。

⑳Step 3.5 Flash: 高速推論LLM

高速推論と対話タスク向けのオープン言語モデル。計算要件を削減しながら一般的言語能力を維持。モデルウェイト、推論例、統合ガイダンスを提供。

⑱Kumo: Rust製メール転送エージェント

Rustで実装されたメール転送エージェント。現代的な設定と可観測性機能を備え、プログラマブルルーティング、ポリシーコントロール、キューマネジメントを構造化設定で公開。
