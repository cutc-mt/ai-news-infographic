Agent Skills 入門

どこから始める？参考サイト3選・覚えておきたい解説

出典: おすま - YouTube

①スキルズとは: 専門書・辞書・手順書をパッケージ化した概念

スキルズは「専門書」「辞書」「手順書」を組み合わせてパッケージ化した概念。AIエージェントが特定のタスクに収束する際、ベストプラクティスに近づけるように専門知識や手順を辞書として与えるもの。クロードに「こういう時はこうした方がいい」という形で指示を提供する。

②メタデータYAMLフロントマター: スキルの頭に常に入れる情報

スキルの仕組みとしてメタデータYAMLフロントマターを頭に入れておく。その下の手順書や辞書の中身は常に読み込まれているわけではなく、スキルを読み込みに行く時に参照する「段階的解除」という仕組みになっている。

③スキルを使う理由: コンテキスト汚染を防ぐ・MCPより適している場合も

Claude.mdに直接書くと無駄なことが書かれているとコンテキスト汚染に繋がる。また、手順書として使う場合、MCP内のツールでできることをコマンドの組み合わせで実現できるなら、MCPよりもスキルで実現した方が良い場合もある。

④コマンズとの違い: スキルズの大きな概念に統合される

コマンズは現在非推奨になっているが、スキルズという大きな概念の中にコマンドが取り込まれるイメージで考えておくと良い。コマンドとして定義しても同じように使えるため、スキルとして定義しておくのが推奨されている。

⑤参考サイト1: Basel Skills - React Best Practices・Remotion実装系

Baselのサイト「Skills SH」には実装系の情報が載っている。Reactのベストプラクティス、Remotionのベストプラクティス、ブラウザエージェント、UIUXProMaxなど。UIデザインを作っていく上で簡単に型抜きができる技が公開されている。

⑥Agent Browser: AIエージェント向けの軽量ブラウザ・動作確認に最適

Agent BrowserはAIエージェント向けのブラウザ。動作確認やスクリーンショット取得などに使用。軽量でおすすめ。慣れるまで違和感があったが、使いこなすと非常に便利。

⑦参考サイト2: Skills MSMP - ネット上のスキルを収集・検索機能付き

Skills MSMPはネット上のスキルを集めてきたサイト。Tailwind CSSに関するスキルなどがピックアップされている。会員登録するとAIサーチなどが解放され便利。ただし、最高品質のスキルは表に出ていないことが多い。

⑧スキル探しのコツ: エージェントスキル掘ってそうな人の近くで参考にする

最高品質のスキルは表に出ていないことが多い。おすすめは「エージェントスキル掘ってそうな人の近くで参考にさせる」こと。スキルサイトで参考にできそうな資料や、そのまま採用できそうなものを探すのも良い。

⑨参考サイト3: Open Crew - 汎用エージェントとしてのスキルを多数提供

Open Crewは汎用エージェントとしてのスキルを色々提供している。作者が頑張って作っており、Discord、Apple Note、Obsidian、NanoBANProなど多数のスキルがある。開発系の重要な知識が得られる。

⑩Open Whisper: CLI Whisperで文字起こし・ローカルリソースでノーAPI

Open WhisperはCLIでWhisperを使って文字起こしをするスキル。スペックが良いモデルを使いたい時に重要。お金をかけずにローカルリソースでノーAPIで実行できる。入れておいて損はないスキル。

⑪Goog: Google WorkspaceをAIエージェントが操作・Gmail・カレンダー・ドライブ

GoogはGoogle WorkspaceをAIエージェントが操作できるスキル。Gmailの最新一覧取得、カレンダー、ドライブなどシェルツールを使える。初期設定が必要だが、汎用的に使える。

⑫Open Crewの高品質: ソースコードを見るのも楽しい・ブラウザ部分の作り込みが凄い

Open Crewのソースコードは見るのが楽しい。ブラウザ部分の作り込みなどが非常に凝っている。オープンソースで多くの人が開発しているため、プロダクトで戦うのが厳しいレベル。スキルクリエイターなども自分で入れている。

⑬他のスキル: Slack・Picaboo・MacOS操作・BlogWatcherなど

Open CrewにはSlack、Picaboo、MacOS操作など様々なスキルがある。MacOSを自動化するCLIスキルなどが、Open Crew自体のクオリティラインを実現している。セッション管理も優秀。

⑭自作スキルのメリット: 破壊力抜群・思考フローや専門知識が埋め込まれる

自作スキルには思考フローや専門知識が埋め込まれているため、破壊力が抜群。メンバーシップなどで共有されていないスキルも多い。OCRスキルなど面白いスキルをシェアしている場合もある。

⑮有料ノート: Agent Skills × Remotionの30分以上解説動画

数量限定でエージェントスキルズとRemotionの組み合わせで30分以上の解説動画を含むノートを販売。コンテキストエンジニアリングの考え方、スキルそのもののソースコード共有、動画解説などが含まれている。

⑯スキルの組み合わせ: 複数のスキルを組み合わせて1つのスキルを作成

複数のスキル（画像生成、文章執筆など）を組み合わせて、1つのブログ執筆スキルのような形で使うこともできる。専門知識を書いておいて、Claudeに判断させる部分も残しながら、手順書として定義しておく組み合わせも可能。

⑰マニュアル作成: 手順書として定義・この動作の後にはこれをする

スキルを手順書として定義することで、「この動作をした後にはこれをした方がいい」「このタスクはこういう形で進めていく」といった手順をAIに指示できる。専門書と手順書を組み合わせてパッケージ化する。

⑱まとめ: スキルを使ってAIエージェントのベストプラクティスを実現

スキルズを使うことで、AIエージェントが特定のタスクでベストプラクティスに近づける。専門知識や手順を辞書として与え、コンテキスト汚染を防ぎ、MCPより適している場合もある。Basel Skills、Skills MSMP、Open Crewなどが参考サイトとして推奨されている。
