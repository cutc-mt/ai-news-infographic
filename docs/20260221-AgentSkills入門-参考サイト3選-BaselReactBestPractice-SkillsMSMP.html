Agent Skills 入門

どこから始める？参考サイト3選・覚えておきたい解説

2026-02-21

①スキルズとは

「専門書」「辞書」「手順書」を組み合わせてパッケージ化した概念。AIエージェントが特定のタスクに収束する際、ベストプラクティスに近づけるように専門知識や手順を辞書として与える。

②メタデータYAML

フロントマターを頭に入れておく。手順書や辞書の中身は常に読み込まれているわけではなく、「段階的解除」という仕組みになっている。

③スキルを使う理由

Claude.mdに直接書くとコンテキスト汚染に繋がる。MCP内のツールでできることをコマンドの組み合わせで実現できるなら、スキルで実現した方が良い場合もある。

④コマンズとの違い

スキルズという大きな概念の中にコマンドが取り込まれるイメージ。コマンドとして定義しても同じように使えるため、スキルとして定義しておくのが推奨されている。

⑤参考サイト1 Basel

Baselのサイト「Skills SH」には実装系の情報が載っている。Reactのベストプラクティス、Remotionのベストプラクティス、ブラウザエージェント、UIUXProMaxなど。

⑥Agent Browser

AIエージェント向けのブラウザ。動作確認やスクリーンショット取得などに使用。軽量でおすすめ。使いこなすと非常に便利。

⑦参考サイト2 MSMP

Skills MSMPはネット上のスキルを集めてきたサイト。Tailwind CSSに関するスキルなどがピックアップされている。会員登録するとAIサーチなどが解放される。

⑧スキル探しのコツ

最高品質のスキルは表に出ていないことが多い。「エージェントスキル掘ってそうな人の近くで参考にさせる」のがおすすめ。

⑨参考サイト3 Open Crew

汎用エージェントとしてのスキルを多数提供。Discord、Apple Note、Obsidian、NanoBANProなど。開発系の重要な知識が得られる。

⑩Open Whisper

CLIでWhisperを使って文字起こし。スペックが良いモデルを使いたい時に重要。ローカルリソースでノーAPIで実行できる。

⑪Goog

Google WorkspaceをAIエージェントが操作。Gmailの最新一覧取得、カレンダー、ドライブなどシェルツールを使える。初期設定が必要だが、汎用的に使える。

⑫Open Crewの高品質

ソースコードは見るのが楽しい。ブラウザ部分の作り込みなどが非常に凝っている。オープンソースで多くの人が開発している。

⑬他のスキル

Slack、Picaboo、MacOS操作など様々なスキルがある。MacOSを自動化するCLIスキルなどが、Open Crew自体のクオリティラインを実現している。

⑭自作スキルのメリット

思考フローや専門知識が埋め込まれているため、破壊力が抜群。メンバーシップなどで共有されていないスキルも多い。

⑮有料ノート

エージェントスキルズとRemotionの組み合わせで30分以上の解説動画を含むノートを販売。コンテキストエンジニアリングの考え方、スキルのソースコード共有など。

⑯スキルの組み合わせ

複数のスキルを組み合わせて、1つのブログ執筆スキルのような形で使うこともできる。専門知識を書いておいて、Claudeに判断させる部分も残しながら定義。

⑰マニュアル作成

手順書として定義することで、「この動作をした後にはこれをした方がいい」「このタスクはこういう形で進めていく」といった手順をAIに指示できる。

⑱まとめ

スキルズを使うことで、AIエージェントが特定のタスクでベストプラクティスに近づける。Basel Skills、Skills MSMP、Open Crewなどが参考サイトとして推奨されている。
