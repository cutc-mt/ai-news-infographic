Claude Code × Remotion × Qwen3 TTS

YouTube用ショートアニメーション作成・ボイスクローン活用

2026-02-21

①完成報告

10秒のショートアニメーション動画が完成。Qwen3 TTSのボイスクローンと組み合わせてナレーション入り動画を作成。ローカルサーバーのキャッシュ削除でプレビュー表示されるようになった。

②動画内容

10秒という尺の制約から、簡潔なチャンネル紹介。「AI生態士チャンネルでは非エンジニアの生成AI活用や最新AIニュースを発信しています。チャンネル登録よろしくお願いします」というテキスト。

③Qwen3 TTS

ローカルモデルを使ったボイスクローン。WebのGUIで音声を作成できるため、自分の声を3秒ほど録音するだけでボイスクローンが可能。

④動画編集アプリとの比較

「これぐらいだったら自分で作った方が早い」という意見も。NovaBananaで素材を持ってきて繋ぎ合わせただけなので、CapCutでも同様のことが可能。

⑤Claude Codeを使う理由

指示出しだけで完成まで自動化。素材をある程度揃えておいて、Claudeと一緒に構成を作成し、Remotionのスキルを使って動画を作ってポンっと投げる。

⑥ボイスクローンの応用

妻の声や子供の声を使ってみるのも面白い。子供3人がいるので3人で少し小説を読ませて5秒ぐらい音声を取れば全然使える。

⑦思い出ムービー

個人利用で考えれば、子供たちの思い出の写真を素材に、思い出ムービーを作ることもできる。結婚式で流れるような動画も手軽に作成可能。

⑧Suno AIとの組み合わせ

Sunoで音楽を制作しているが、その裏側のMV（ミュージックビデオ）もRemotionで作成できる。後ろ背景画像で文字だけがスラスラ流れるようなシンプルなものでも雰囲気が出る。

⑨YouTube Shorts

Remotionは縦長の動画にも対応可能。尺15秒や30秒にしてみたり、実験を兼ねた動画作りも面白そう。

⑩コスト効率

Claude Sonnet 4.6ではクレジットをほとんど消費しない。1発出しでOKな出し方をすれば、10秒の動画はサブスクプランの15%程度で作成可能。

⑪GLM-5契約者

課金して契約している場合はいくらでも作成できるレベル感。遊びとして十分に使えるレベル。

⑫素材ストック

NovaBananaで画像を作らせて素材をストックしておき、それを組み合わせるだけで動画が作成可能。

⑬マーケティング実験

動画最後に貼り付け、反応率やチャンネル登録者の伸びがどう変わるかをマーケティング的な実験として行う。

⑭新しい手札

今までになかった新しい手札として、ショート動画・アニメーション動画を作ることと、ボイスクローンでナレーションを入れた自分っぽい作品が作れる。

⑮作ってみないとわからない

「こういうのもやってみなきゃわからない」という感覚。一度手を出してみるのは結構いい。YouTubeで音楽系のチャンネルを見て、こういうぐらいだったらRemotionでいけちゃうかもという発想が広がる。

⑯楽しんで実験

実際に作ってみると試してみたいという欲が出てくる。ライブ配信でも遊びで作ってみてもいいし、自分の中で色々なバリエーションを作ってチャレンジ。

⑰今後の展望

ボイスクローンで子供の声にしたらどうなるか、細かく色々遊びながら実験していく。RemotionとQwen3 TTSの組み合わせでアニメーションを作ってみて、面白かったので今後も色々試していく。

⑱まとめ

10秒の動画だったら手作業で作った方が早いという意見もあるが、自分が楽しみつつ、ある程度のクオリティの動画が手軽に作れる。
