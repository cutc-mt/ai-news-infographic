Claude Code NEW Update

Security機能・Autofixes・PRモニタリング・ローカルコードレビュー

出典: Matthew Berman - YouTube

①Claude Opus 4.6の性能: 50%タイムホライズンで14.5時間を達成

Meterの推定によると、Claude Opus 4.6は人間の開発者が1労働日かかるタスクを約半分の時間で完了できる。ソフトウェアエンジニアリングが根本的に変化し始めている。

②デスクトップ版プレビュー機能: アプリケーションのプレビュー・スクリーンショット・自動反復

Claude Codeデスクトップでアプリケーションをプレビューできる。開発サーバーを立ち上げ、実行中のアプリをデスクトップインターフェース内でプレビュー。コンソールログを読み、エラーをキャッチし、リアルタイムで反復して自律的に修正。

③フロントエンド機能: モダンなコーヒーストアウェブサイトを簡単に作成

サンプル画像とシンプルなプロンプトからモダンなコーヒーストアウェブサイトを作成。美しいプレースホルダー、ダイナミックな動き、複数の機能を持つランディングページを開発者なら数日かかるものが短時間で作成可能。

④ローカルコードレビュー: 複数のサブエージェントを展開して自律的にエラー解決

ローカルコードレビューが可能。プレビューと注釈をClaude Code内で直接行い、複数のサブエージェントを展開してタスクに取り組み、ログ内のエラーを自律的に解決。

⑤PRの自動作成: GitHubリポジトリに自律的にPRを公開・サマリーも作成

Pull Requestを開いてClaudeにCIを追跡させ、autofixesを有効にして変更をコミットし、全て完璧に動作するようにする。自律的にGitHubリポジトリにPRを公開し、達成した内容のサマリーを追加。

⑥コードレビュー機能: 全てのバグと問題にインラインコメントを追加

コードをプッシュする準備ができたら「Review Code」機能を押すと、Claudeが全てのバグと問題にインラインコメントを追加。コードが完全なレビューに入る前に自動的に全てを解決。

⑦PRモニタリング: CIを追跡・自動修正・Automergeでチェックが通ると即座にマージ

PRモニタリングでClaudeがバックグラウンドでCIを追跡。autofixesで失敗を自動的に解決し、Automergeを有効にするとチェックが通るとPRが即座にマージされる。次のタスクに取り組んでいる間にClaudeが最後のタスクを処理。

⑧セッションモビリティ: デスクトップ・CLI・モバイルアプリ間でセッションを移動

セッションモビリティでデスクトップで作業中のセッションをCLIセッションに移動し、デスクトップアプリやモバイルアプリにも移動可能。異なる経路から全てを制御できる。

⑨Claude Code Security: コードベース全体をスキャンして脆弱性を検出・修正案を提示

Claude Code Securityは現在プレビュー中で、従来のツールが見落とす問題を見つけて修正できる。コードベース全体を自律的に読み取り、人間のセキュリティ研究者のように推論。コンポーネントの相互作用を理解し、複雑な問題を通してデータフローを追跡。

⑩自動コードスキャン: マルチステージ検証・人間のループ・ダッシュボード統合

自動コードスキャン、マルチステージ検証、人間のループ、ダッシュボード統合。Claude Code内で発見を検査し、修正を承認し、直接反復可能。

⑪Gitワークツリーサポート: エージェントが並列で実行・干渉なし

Gitワークツリーサポートでエージェントが互いに干渉せずに並列で実行可能。各エージェントは独自のワークツリーを取得し、異なるブランチとタスクで独立して安全に作業。

⑫フル開発ワークフローの自動化: コードベースやエディタを使う必要がなくなる

Claude Codeはフル開発ワークフローを自動化し始めており、コードベースやVS Codeスタイルのエディタを使って直接変更を加える必要がなくなる。これらの incredibleなモデルに仕事をさせることができる。

⑬セキュリティ機能の利用: 限定リサーチプレビュー・今後数週間でロールアウト

セキュリティ機能は全員が利用できるわけではない。今後数週間でロールアウトされるが、ウェイティングリストに参加して早期アクセス可能。

⑭脆弱性検出: 従来のセキュリティチームが時間をかけて見つける問題を迅速に検出

従来のセキュリティチームがコードベース全体をレビューするのに長時間かけていた問題を、Claude Code Securityが迅速に検出して修正。

⑮Anthropicのスピード: 新しいモデルとClaude Codeのメジャーアップグレードを継続的にリリース

Anthropicは新しいモデルとClaude Codeへのメジャーアップグレードを継続的にリリース。AIコーディングアシスタンスを全く新しいカテゴリーに押し上げている。

⑯並列ワークフロー: デスクトップアプリとCLIでワークツリーモードをサポート

デスクトップアプリとCLIの両方でワークツリーモードをサポートし、並列ワークフローをはるかに簡単に。異なるエージェントを分離して使用できる柔軟性。

⑰ゲームチェンジャー: 次のタスクに取り組んでいる間にClaudeが最後のタスクを処理

Automergeを有効にすると、チェックが通るとPRが即座にマージされる。次のタスクに取り組んでいる間にClaudeが最後のタスクを処理するというのはゲームチェンジャー。

⑱まとめ: Claude Codeは開発ワークフローを完全に自動化・ソフトウェアエンジニアリングが根本的に変化

Claude Codeはプレビュー機能、ローカルコードレビュー、PRモニタリング、セキュリティ機能などを追加し、開発ワークフローを完全に自動化し始めている。ソフトウェアエンジニアリングが根本的に変化している。
