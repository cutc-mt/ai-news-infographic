Gemini 3.1 Pro 解説

推論モンスター級のAIモデル・ARC AGI2倍増・エージェンティックコーディング

出典: Wonder Sato - YouTube

①公式ベンチマーク: ARC AGI2が31%から77.1%に倍増・Claude Opus 4.6を上回る

Gemini 3.1 Proの最大の特徴はARC AGI2（抽象推論）が31%から77.1%に倍増したこと。先日出たClaude Opus 4.6よりも約10ポイント高いスコアを達成。現状最強の推論AIと言っても過言ではない。

②抽象推論能力: 未知のパズルを解く能力が大幅向上・人類最後のクイズでもトップ

ARC AGI2だけでなく、Human Last Quiz（人類最後の問題）でもトップスコアを達成。抽象推論が化け物的に賢くなり、未知の問題を解く能力が大幅向上した。人間が生きている中で直面する「分からない分岐点」でどちらを選択すべきかを考える力が強化されている。

③Gemini 3.5への布石: コードネーム「Snow Bunny」・プロンプト入れるだけで3000行コード生成

Gemini 3.5はまだリリースされていないが、コードネーム「Snow Bunny」で可愛らしい名前ながら化け物のような強さを持つ。プロンプトを少し入れるだけで3000行のコードを一発生成する能力を持つ。3.1は3.5への布石となっている。

④エージェンティック化: 3.0より長く思考してコーディング・3.5への道筋

Gemini 3.1はエージェンティック（自律的）なコーディングができるようになっている。前回の3.0よりも長く思考してコーディングを実行。推論能力の強化はエージェンティックなコーディングモデルへのシフトを示唆している。

⑤マルチモーダル強さ: ローテータカフ図をSVGで正確に作成・4筋肉の構造を理解

「ローテーターカフの図をSVGで書いて」というプロンプトで、肩甲骨と上腕骨をつなぐ4つの筋肉（インナーマッスルの前面と背面）を正確に描画。Claude Sonnet 4.6では1枚で書こうとしてぐちゃぐちゃになるタスクも、Gemini 3.1 Proは綺麗に描画できる。

⑥視覚的推論: 言語トークンをビジュアルトークンに変換・立体的イメージを正確に表現

Geminiのマルチモーダル性は非常に強く、言語トークンをビジュアルトークンに変換できる。立体的なイメージや複雑な構造を正確に表現できる能力は、他のモデルにはない強み。知識と空間認識能力を組み合わせて高品質な図形を作成する。

⑦思考プロセスの長さ: 体感的に長い思考・深い推論を実行

Gemini 3.1 Proの思考プロセスは非常に長い。今までの挙動とは違い、体感的にも長く思考していることがわかる。DeepSeekを使えばさらに深くなる。この長い思考こそが、エージェンティックなコーディングを可能にしている。

⑧高品質コード生成: ホームページや図鑑を一発で作成・深い思考が高品質を生む

思考が深く長く考えられる力があるため、高品質なホームページや図鑑を一発で生成できる。1発でやるということはそれだけ思考が深くないといけないし、長く考える力が必要。AIエージェントの力でもあり、AIエージェントに本当に近づいていると言える。

⑨失敗例: サムネイル作成で「画像生成不可」と誤回答・画像入力で処理が重くなる

Gemini 3.1 Proでサムネイル作成を依頼すると「画像生成不可」と誤回答するケースがある。これは嘘で実際には画像生成が可能。画像を入れると処理が重くなり、プロンプトも長いと複雑になるため、オーバーフローを起こしている可能性がある。

⑩バランスプロンプティング: タスクに応じて思考の長さを調整・短く思考させる指示も有効

Gemini 3.1 Proは強くなるからこそ問題が出てくる。簡単なタスクでもガンガン思考してしまう傾向があるため、タスクに応じて「短く思考して」「深く考えないで」といったプロンプトで思考の長さを調整するバランスプロンプティングが必要。

⑪簡単なタスク: Claude Sonnet 4.6はほとんど思考せず・Geminiは全て思考する

Claude Sonnet 4.6の場合、簡単なタスクではほとんど思考しない。しかしGemini 3.1 Proは簡単なタスクでも全てしっかり思考して書いてくれる。この違いは適応性の問題であり、今後の進化ポイントになるかもしれない。

⑫パズルゲーム実験: 精錬のエレメンタルハーツで抽象推論をテスト

「精錬のエレメンタルハーツ」というパズルゲームで抽象推論能力をテスト。Cometアシスタントを使ってGemini 3.1 Proにゲームをクリアさせ、コツやハイスコアの出し方を教えさせる実験を実施。

⑬実験結果1: Claude Opus 4.6とSonnet 4.6は回答不可・ChatGPT 5.2は9分37秒思考

Claude Opus 4.6とSonnet 4.6は「分からない」と回答。Opusは「x=7, y=2を下に動かす」と回答したが、5個しか揃わないため不正解。ChatGPT 5.2は9分37秒思考したが、「x=1, y=10を右に動かす」と何も変わらない回答をした。

⑭実験結果2: Geminiは最長の思考ログ・回答は「x=4, y=5を左に動かす」で変化なし

Gemini 3.1 Proは最長の思考ログを記録したが、回答は「x=4, y=5を左に動かす」で何も変わらない結果に。パズルゲームを解かせるのはまだAIにとって難しい課題であることが判明。

⑮今後の実験計画: 盤面を固定して丸の数を減らしたパズルを作成

今後は盤面を固定して丸の数を減らしたパズルを作成し、平等に実験できる環境を整える予定。AIにパズルを解かせる実験を継続して、抽象推論能力の限界を探っていく。

⑯まとめ: 圧倒的な推論能力・思考の深さ・エージェンティックなコーディング

Gemini 3.1 Proの3つの特徴は圧倒的な推論能力、思考の深さ、エージェンティックなコーディング。Gemini 3.5「Snow Bunny」への布石となるモデルであり、抽象推論がないとずれてしまうため、3.5への布石としての3.1が見える。

⑰使い分けの重要性: Claudeは適応的に思考・Geminiは深く思考・タスクで選択

Claudeはタスクの難易度に応じて適応的に思考時間を調整し、Geminiは常に深く思考する傾向がある。タスクに応じて使い分けることが重要。コーディングではGeminiの強さが発揮される一方、簡単なタスクではClaudeの適応性が便利。

⑱今後の展望: 3.5「Snow Bunny」への期待・エージェンティックモデルの進化

Gemini 3.5「Snow Bunny」はプロンプトを少し入れるだけで3000行のコードを一発生成する化け物的な強さを持つ。3.1で既にエージェンティックな思考が動いており、3.5ではさらに進化したエージェンティックコーディングが期待される。
