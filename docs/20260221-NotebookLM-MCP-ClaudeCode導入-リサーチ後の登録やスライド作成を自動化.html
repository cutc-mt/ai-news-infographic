NotebookLM MCP導入

Claude Codeでリサーチ後の登録やスライド作成を自動化

出典: 世界のをか - YouTube

①NotebookLM MCP: ノートブック作成・ソース追加・ディープリサーチ・コンテンツ作成が可能

NotebookLM MCPを使うとノートブックの作成、ソースの追加、ディープリサーチ、コンテンツ作成ができる。Claude Codeをかなり使ってエージェントチームで調べることが多いので、CursorにCLIを追加する。

②WSL環境への導入: UVツールインストール・ノートブックLM MCPインストール

WSL環境にUVとツールをインストールしてNotebookLM MCPをインストール。インストール後、バージョン確認で正しくインストールされたことを確認。

③ログイン認証: クッキー取得・Chrome拡張機能「Get cookies.txt LOCALLY」を使用

NotebookLMへのログイン認証にはChrome拡張機能「Get cookies.txt LOCALLY」を使用。クッキーを取得してNotebookLM.google.comに保存。認証情報を手動で指定してログイン。

④ファイルアップロード: クロードコードがファイルを読んでしまう問題

NotebookLM MCPを使ってファイルを登録する際、Claude Codeが1回ファイルを読んでしまい、ファイルをそのままアップロードできないことが課題。ファイルをそのままアップロードする機能が欲しい。

⑤MCPの利便性: 定常的なアクションをMCPに追加すると便利

毎回違うことをやっていて定常的に同じことをするケースが少ないが、定常的なアクションはMCPに追加できると便利。NotebookLM系は定常的なアクションなので追加しておくと良い。

⑥リサーチ機能: ノートブックLMでディープリサーチが可能

NotebookLM MCPを使うとディープリサーチが可能。リサーチ後の登録やスライド作成を自動化できる。

⑦スライド作成: ノートブックLMでスライドを自動作成

NotebookLM MCPを使うとスライドの自動作成が可能。スライド作成は結構時間がかかるが、自動化することで効率化が図れる。

⑧設定ファイル: settings.jsonにコマンドを登録・確認せずに実行可能

今後は確認せずに実行したいので、settings.jsonに使用するコマンドを登録。毎回確認がめんどくさいので、確認しなくてもいいように設定しておくと便利。

⑨一括処理: 複数のファイルをまとめてアップロード可能

NotebookLM MCPを使うと複数のファイルをまとめてアップロードできる。1個1個入れる必要はなく、まとめファイルを作成すれば一括処理が可能。

⑩エージェントチーム: Claude Codeでエージェントチームを使用

Claude Codeでエージェントチームを使用してリサーチやコンテンツ作成を行う。NotebookLM MCPと組み合わせることで、より高度な自動化が可能。

⑪導入の苦戦: クッキー認証やファイルアップロードで少し苦戦

導入には少し苦戦した。クッキー認証やファイルアップロードで少し手間取ったが、最終的には無事に導入できた。

⑫導入完了: NotebookLM MCPが追加され便利に使える

無事にNotebookLM MCPが追加され、便利に使えるようになった。時間がかかりそうだったが、めんどくさかったが無事に導入完了。

⑬自動化の効果: リサーチ後の登録やスライド作成が自動化され効率化

リサーチ後の登録やスライド作成が自動化されることで、作業効率が大幅に向上。手作業でやるよりもはるかに効率的。

⑭CLI対応: WSLのClaude CodeやCursorで利用可能

WSLのClaude CodeやCursorで利用可能。CLIベースで操作できるため、柔軟な使い方ができる。

⑮MCPの追加: 他のMCPも追加していく予定

あんまりMCPを追加していないので、これを追加した上で他のMCPもどんどん追加していく予定。定常的なアクションはMCPに追加していく。

⑯Bedrockエージェントチーム: MCPで使えるランキングを作成

BedrockエージェントチームでMCPで使えるランキングを作っているので、NotebookLM MCPを追加していく。

⑰Cursorへの追加: CursorでMCPを追加して使う

CursorにMCPを追加して使うことも可能。設定に追加するだけで簡単に使えるようになる。

⑱まとめ: NotebookLM MCP導入でリサーチとコンテンツ作成が自動化・効率化

NotebookLM MCPを導入することで、リサーチ後の登録やスライド作成が自動化され、作業効率が大幅に向上。導入には少し苦戦したが、一度導入すると非常に便利。
