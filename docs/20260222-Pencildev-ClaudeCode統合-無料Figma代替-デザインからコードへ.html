Claude Code × Pencil.dev

無料のインフィニットデザインキャンバス・Figmaライク・コード生成統合

2026-02-22

①Pencil.dev

完全無料のインフィニットデザインキャンバス。Figmaの代替として機能し、Claude Codeと統合してAI支援のフロントエンド開発を可能にする。

②動的システムコンポーネント

再利用可能なスタイルと動的システムコンポーネントを導入。デザインシステムをAIエージェントに適用し、トークン浪費を削減し一貫性を向上。

③Figmaインポート

FigmaからPencilにコピーするだけでレイアウト、間隔、スタイルがピクセルパーフェクトに保持。コンポーネントの再構築なし、再デザインなしで完璧に維持。

④コード生成

インポートされたデザインを理解し、リポジトリのコーディング標準に合ったフロントエンドコードを生成。AIが決定を独断せず、ユーザーがコントロールしながら実行を担当。

⑤デザインシステム選択

Shadcn、Lunarisなど複数のデザインシステムから選択可能。スタイルガイドをAIエージェントに適用し、製品のビジュアルアイデンティティに一致するコンポーネントを生成。

⑥MCP統合

PencilのMCPを有効化し、Claude Codeから直接デザインコンテキストを操作。チャットインターフェースでフレームを選択し、スタイルガイドを指定して生成。

⑦ライブプレビュー

生成されたコードをライブプレビューで確認。Pencil MCPがキャンバスを分析し、選択したフレームのコードを生成。2つのツールが手を取り合って連携。

⑧プロンプトギャラリー

様々なスタイルやテンプレートから開始できるプロンプトギャラリー。アプリ作成のビルディングブロックとして活用し、より良い生成を実現。

⑨個別編集

各ブロックを個別に調整可能。配置、デザイン構造、詳細な編集をキャンバスコンポーネントで実行。レイヤーを選択してアノテーションを追加。

⑩配色変更

「全てのレイヤーの配色を青アクセントに変更」といった指示で瞬時に全ブロックコンポーネントに変更を適用。

⑪ワンショット生成

単一ショットで美しいフロントエンドを開発。技術ダッシュボードをLunarisデザインシステムで生成し、見事にコンポーネントをブロック化。

⑫マルチエージェント対応

Claude Code、CodeEx、Gemini CLIなど様々なエージェントと連携可能。完全無料のワークフロー構築も実現。

⑬クロスプラットフォーム

macOS、Windows、Linuxに対応。Cursor、Antigravityなどの複数のIDEで動作。

⑭デザイナー向け

Pencil内のチャットはデザイン属性に特化。Claude Codeは全アクションをチャットで実行可能。ニーズに合わせて使い分け。

⑮ベクター対応

ベクター画像生成モデルを含み、拡大しても画質が劣化しない。画像内のテキスト表示にも対応（日本語も可能）。

⑯デザイン忠実性

デザイン忠実性を損なうことなくコード化。Figmaデザインをインポートし、レイアウトやスタイルを完全に保持したままコード生成。

⑰まとめ

Pencil.devは完全無料でFigmaライクの機能を提供。Claude Codeと統合し、デザインからコードへのシームレスなワークフローを実現。今のうちに活用すべき。
