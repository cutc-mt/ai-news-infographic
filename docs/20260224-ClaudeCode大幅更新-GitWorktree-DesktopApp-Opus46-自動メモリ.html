Claude Code 大幅更新

Git Worktree・Desktop App・Opus 4.6・自動メモリ・セキュリティ

2026-02-24

①Git Worktree分離

-wフラグでワークツリー分離が可能に。同じリポジトリで複数のClaude Codeセッションを動かしても、コード編集が競合しない。

②サブエージェント対応

エージェント定義でisolation:worktreeを指定可能。サブエージェントも独自のワークツリーで動作。

③バックグラウンド制御

Ctrl+Fで2回押し確認で全バックグラウンドエージェントを停止。3秒以内に2回押す必要がある。

④設定変更フック

設定ファイルが変更された時に発火するconfig_changeフックイベントを追加。セキュリティ監査に活用可能。

⑤Simple Mode強化

環境変数CLAUDE_CODE_SIMPLEでfile editツールが追加。MCPやフックを無効化した軽量モードがより実用的に。

⑥LSPタイムアウト設定

Java言語サーバーなど遅いサーバーのために起動タイムアウトを設定可能に。ミリ秒単位で指定。

⑦Desktop App Preview

Claude Code Desktopにアプリプレビュー機能が追加。開発サーバーを起動してリアルタイムでプレビュー可能。

⑧エラー自動検出

コンソールログからエラーを自動検出して修正。コピー&ペースト不要でエラーを反復修正可能。

⑨コードレビューパネル

インラインコメント付きの変更レビューが可能に。プッシュ前にバグを特定できるミニPR体験。

⑩GitHub連携

PR監視・CI監視・自動修正が可能。チェックが通ったら自動でマージする設定も追加された。

⑪セッション移動

CLIで/desktopコマンドでCLI→Desktop→Webへセッションを移動可能。コンテキストが完全に維持される。

⑫Opus 4.6登場

20万コンテキスト（ベータで100万）、最大出力128Kトークン。適応的思考で動的に思考量を調整。

⑬14時間タスク継続

Anthropicのデータで50%の成功率で14時間タスクを継続可能。大規模マルチファイルプロジェクトに最適。

⑭自動メモリ機能

学習・パターン・洞察を自動記録。~/.claude/projects/memoryにプロジェクトごとに保存。

⑮Claude Code Security

セキュリティ分析の自動スキャン。意味的推論で従来のツールが見逃す脆弱性を検出。研究プレビュー版。

⑯MCP Lazy Loading

MCPサーバーの遅延読み込みで95%コンテキスト削減。ツール検索機能で全サーバーを効率的に検索可能。

⑰JetBrains公式プラグイン

JetBrains Marketplace公式プラグインが登場。差分ビューアーとLSP統合でリアルタイムコード理解。

⑱まとめ

Claude Codeは最も機能豊富なAIコーディングエージェントに成長。Desktop・マルチエージェント・ツール統合の組み合わせは唯一無二。
