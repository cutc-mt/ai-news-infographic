AIでUI作るとダサくなる問題を解決

脱テンプレートの7つのコツ・Inter+Lucide+青グラデから脱却

2026-02-26

①AI製UIの三種の神器

Inter + Lucideアイコン + 青いグラデーション。これがAI生成UIのデフォルト。

②学習データの偏り

Tailwind UIやshadcn UIなどが学習データの大部分を占め「平均的なデザイン」が出力される。

③安全な選択の優先

AIは批判されないデザインを選びたがる。無難で誰の心にも刺さらないデザインに。

④曖昧な指示が危険

「モダンでクリーンに」はNG。人によってイメージが違うのにAIは1つのモダンしか知らない。

⑤コツ1：スケッチファースト

Excalidrawでラフなワイヤーフレームを描く。5分でOK。画像としてAIに渡す。

⑥コツ2：ビジュアル参考を見せる

DribbbleやPinterestで参考画像をスクショして添付。「このスタイルで」と指示。

⑦コツ3：カラーパレットを指定

Realtime ColorsやCoolorsでパレットを作成。16進数のカラーコードを直接渡す。

⑧コツ4：デザイントークンを定義

Claude.mdに色や余白のルールを事前定義。primary: #FF5733のように書く。

⑨コツ5：アンチパターンを明示

「グラデーション禁止」「Interフォント使わない」など禁止事項を明記。

⑩コツ6：フォントとアイコンを変える

Interの代わりにClash DisplayやOutfit。Lucideの代わりにPhosphorやHeroicons。

⑪コツ7：反復的な改善

スクショに赤ペンで修正指示を書き込んで画像で渡す。2-3周で大幅改善。

⑫AIは想像より模倣が得意

具体的なビジュアルを見せれば見せるほどAIは良いデザインを出してくれる。

⑬「温かみ」も解釈が違う

「温かみのある色」でもAIは青になることがある。具体的な色を指定すべき。

⑭パーツ単位で参考にする

ナビはこれ、フッターはあれとパーツごとに参考画像を指定すると統一感が出る。

⑮NuPicksで画像から色抽出

好きな写真からカラーパレットを作成。カフェの写真からおしゃれな色を抽出。

⑯余白もトークン化

8px、16px、24px、32pxの倍数で統一。バラバラな余白はAI生感が爆上がり。

⑰引き算のデザイン

何を入れるかより何を入れないかで差が出る。禁止リストを作ることで差別化。

⑱人間がディレクター

AIはアシスタントとして使い倒す。指示→修正→指示のループで品質向上。
