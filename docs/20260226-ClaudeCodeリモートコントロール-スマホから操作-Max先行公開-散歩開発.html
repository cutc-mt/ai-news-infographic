Claude Code Remote Control登場

スマホから操作・QRコードで接続・散歩しながら開発・Max先行公開

2026-02-26

①Remote Control機能登場

`claude remote-control`コマンドで自宅PCのClaude Codeをスマホから操作可能に。

②散歩しながら開発

外出先から自宅PCで動いているセッションにアクセス。散歩中も開発継続。

③QRコードで接続

コマンドを入れるとQRコードが表示。スマホで読み取るだけで接続完了。

④Maxユーザー先行公開

現在はMaxユーザー向けに先行公開中。Proユーザーは少し待つ必要がある。

⑤APIキー不要

OpenCrewなどと違ってAPIキーの工夫が不要。純正機能なので手軽に使える。

⑥従来の選択肢

これまではHapy・Termius・Mosh・VibeTunnelなどサードパーティに頼っていた。

⑦claude rcコマンド

短縮形の`claude rc`でもOK。connectと入力すると接続開始。

⑧インタラクティブセッション

スマホからメッセージを送信して対話可能。Opusなどで贅沢な対話も。

⑨Amphetamineでスリープ防止

MacBookを常時起動するためにAmphetamineで無制限セッションを設定。

⑩画面共有との併用

必要なら画面共有で放置中のPCにアクセスすることも可能。

⑪スラッシュコマンドの制限

現状スラッシュコマンドが弱い。clearコマンドなどが動かない場合がある。

⑫自動発火スキルで対応

自動発火するスキルを入れておけばコマンドの制限を回避可能。

⑬セッション切り替え

複数セッションの切り替えが絶妙に使いづらい点は改善待ち。

⑭AI疲れへの対策

タスク走ってる間はリフレッシュする。オンオフの切り替えが重要。

⑮スマホを持たない選択も

あえてスマホを持たないことでオフの時間を確保する人もいる。

⑯ハーネスを作り込む

リモートコントロールに頼るよりハーネスを長時間運用できるよう作り込むのもあり。

⑰Claude Code Webアプリ

Claude Code Webアプリから自宅PCのセッションにアクセス。

⑱まとめ

純正機能で手軽に外出先から開発可能。APIキー不要でQRコード読み取りだけで接続。
