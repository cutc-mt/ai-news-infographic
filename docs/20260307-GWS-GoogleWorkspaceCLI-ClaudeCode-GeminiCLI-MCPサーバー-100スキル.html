GWS - Google Workspace CLI

Claude Code & Gemini CLI連携・MCPサーバー・100以上のエージェントスキル

2026-03-07

①GWSとは

Google Workspace CLI。Drive・Gmail・Calendar・Sheets・Docs・Chatなどをコマンドラインで操作。

②久々のお宝発見

GitHubで1万2000スター獲得。アンチグラビティ連動と同レベルのインパクト。

③Discovery Service活用

Google Discovery Serviceからコマンドツリーを動的生成。ツール自体のアップデート不要。

④自動的に新API対応

Google側でAPIが追加されると即座に利用可能。開発者のボイラープレートをゼロへ。

⑤人間向け・AI向け両対応

デュアルパーパスアーキテクチャ。人間は直感的操作、AIは構造化データ。

⑥人間向け機能

自動ページネーション・ドライラン・プレビュー・タブ保管・全コマンドヘルプ。

⑦AIエージェント向け機能

構造化JSON出力・LLMが直接解釈可能・テキストパース不要。

⑧100以上のエージェントスキル

100以上のAPI対応スキルと50以上の厳選レシピを標準搭載。

⑨スキル階層構造

ペルソナ&レシピ→ワークフロー→ヘルパー→サービス→API。基礎から高度な自立ワークフローへ。

⑩MCPサーバー機能

gws mcp startでMCPサーバー起動。Claude Desktop・VS Codeから直接呼び出し可能。

⑪Gemini CLI拡張機能

Gemini CLIの拡張機能として統合。認証情報を自動継承してダイレクトに操作可能。

⑫エンタープライズセキュリティ

Google Cloud Model Armorとネイティブ統合。プロンプトインジェクション攻撃対策。

⑬セットアップ手順

1. Gemini CLIアップデート→2. GWSインストール→3. gws setupでウィザード形式。

⑭Google Cloud CLI必要

Google Cloud SDKのインストールが前提。gcloud CLIで認証してからGWS設定。

⑮GCPプロジェクト作成

GWS専用のGoogle Cloudプロジェクトを作成。API有効化は選択式で必要なものだけ。

⑯OAuth同意画面設定

外部ユーザー設定・テストユーザー追加・クライアントIDとシークレット作成が必要。

⑰ Gemini CLI拡張インストール

gemini extension install gwsで完了。100以上のスキルが即座に利用可能。

⑱まとめ

人間とAI双方の生産性を極限まで高める次世代CLI。非公式だが設計が本格的。
