Pi Coding Agent

オープンソース・最小構成・拡張システム・200トークンプロンプト

2026-03-07

①Piとは

オープンソースのターミナルベースコーディングエージェント。Claude CodeやCodexをよりカスタマイズ可能。

②開発者Mario Zechner

LibGDXの開発者として知られる。Javaゲームフレームワークで有名。

③哲学：ワークフロー適応

「ツールをワークフローに適応させる」。その逆ではない。カスタマイズ重視。

④4つのツールのみ

Read・Write・Edit・Bashの4つだけ。Claude Codeは8-9個のツールを持つ。

⑤200トークンプロンプト

Claude Codeは1万トークンのシステムプロンプト。Piはわずか200トークン。

⑥25以上のフックポイント

拡張システムでカスタムツール・サブエージェント・オーケストレーションを構築可能。

⑦15以上のモデル対応

Claude・GPT・Gemini・Groq・DeepSeekなど15以上のプロバイダーに対応。

⑧スキル＆パッケージシステム

必要な時だけ機能を読み込む。コンテキストウィンドウをリーンに保つ。

⑨ツリーベースセッション

Gitのようなツリー構造で会話を管理。フォーク・ナビゲーションが可能。

⑩セッションアーキテクチャ

会話の分岐・合流が自然にできる構造。実験的アプローチを保存可能。

⑪最小限で柔軟

不要なものを排除。本当に必要なものだけを積み上げる設計思想。

⑫学習ツールとして最適

コーディングエージェントの仕組みを理解するのに最適な教材的側面。

⑬ブラックボックス解除

Claude Codeのブラックボックスを自分で中身を見て学習できる。

⑭Claude Code vs Pi

Claude Codeは洗練された体験。Piは自由度の高い構成要素。

⑮エンジニア向け

ツールを完全にコントロールしたいエンジニアに最適な設計。

⑯セキュリティと透明性

オープンソースで中身を全て確認可能。セキュリティリスクを把握できる。

⑰カスタム構築

自分専用のコーディングエージェントをゼロから構築する基盤として利用可能。

⑱まとめ

RIP Claude Code & OpenCodeと言われるほどの自由度。学習・カスタム構築に最適。
