AIエージェント、オーケストレーション、CLIツール

①Agency Agents: 特化型AIロールプロンプト集

専門家チームをシミュレートするAIエージェント定義のコレクション。各エージェントに役割、責任、ワークフローを定義。

②SEO Machine: 自動SEOコンテンツワークフロー

Claude Code用のSEOコンテンツ自動化ワークスペース。研究、執筆、分析、最適化を一元管理。

③MoneyPrinter V2: オンライン収益自動化

Twitterボット、YouTubeショート、アフィリエイト、コールドアウトリーチなど収益化ワークフローを自動化。

④RuView: Wi-Fiで姿勢推定

カメラを使わずWi-Fi信号で人間の姿勢と動きを推定する実験的センシングシステム。プライバシーを重視。

⑤RuFlo: マルチエージェントオーケストレーション

Claude用のマルチエージェントオーケストレーションプラットフォーム。スウォームスタイルの実行をサポート。

⑥Superpowers: AI能力レイヤー

AIアシスタンスを特化したケーパビリティで拡張するツール。再利用可能なモジュールとして定義。

⑦OpenClaw: Clawシステムのオープン実装

ソフトウェアと対話するアジェンティックAI環境を構築するオープンソース実装。自律ワークフローの実験に。

⑧Hugging Face Skills: スキルライブラリ

AIアシスタントやエージェント向けの再利用可能なスキル集。ツール連携や情報検索などのケーパビリティを提供。

⑨Heretic: ターミナル用Git管理ツール

ターミナルから直接Gitリポジトリを管理・調査するCLIツール。高速ナビゲーションと簡素化されたワークフロー。

⑩Goggly: GOGプラットフォームCLI

GOGゲームプラットフォームをターミナルから操作。ゲーム一覧、インストーラー管理をGUIなしで実行。

⑪PM Skills: プロダクトマネージャー向け

プロダクト管理に必要なコアスキルを体系化した知識ベース。戦略、発見、デリバリー、リーダーシップを網羅。

⑫GWS: Google Workspace CLI

Google Workspace公式のコマンドラインインターフェース。Gmail、Driveなどをターミナルから管理。

⑬YT Browse: ターミナルでYouTube閲覧

コマンドラインからYouTubeコンテンツを閲覧。検索、チャンネル、プレイリストをキーボード操作で探索。

⑭LiveKit Agents: リアルタイム音声AI

リアルタイム音声・動画AIエージェントを構築するフレームワーク。STT、TTS、LLMを統合パイプラインで接続。

⑮Shuo: ターミナルでAIと音声会話

ターミナルからLLMと音声で会話するCLIツール。音声認識、テキスト変換、音声合成をループで処理。

⑯Spank: タスクランナー

繰り返しタスクを定義・実行する軽量コマンド自動化ツール。ビルド、テスト、デプロイを設定ファイルで管理。

⑰BullshitBench: AIハルシネーション検出

誤った前提をどう処理するかを測定するベンチマーク。作り話をするのではなく拒否するかを評価。

⑱Lysium: メモリを持つAIアプリ

構造化されたメモリを維持するAIアプリケーションを構築するフレームワーク。過去の会話を記憶。
