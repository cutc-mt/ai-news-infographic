Goose v1.26.0

ローカル推論、Telegramゲートウェイ、Peekaboo Vision

①ローカル推論: 完全にオフラインで動作

llama.cppバックエンドを内蔵してモデルをローカルで実行できるようになった。HuggingFaceをブラウズしてモデルをダウンロードし、推論を開始できる。

②クラウド不要のAI環境

クラウドもAPIキーもインターネット接続も不要。自分のハードウェア上で完全に完結するAIエージェント環境を実現。

③Telegramゲートウェイ

新しいゲートウェイでTelegramや任意のメッセージングプラットフォームからエージェントとチャット可能。どこからでもアクセスできる。

④エージェントへの常時アクセス

メッセージ一つでエージェントにアクセス。Webhookを通じてあらゆるメッセージングプラットフォームと連携できる柔軟な設計。

⑤Peekaboo Computer Vision

コンピュータコントローラーが完全に刷新。画面を見て要素を識別し、ネイティブにアクションを実行できるようになった。

⑥画面認識のネイティブ実装

外部ツールなしで画面を理解して操作できる。Gooseが自律的にコンピュータを操作する能力が大幅に向上。

⑦Cerebras: 超高速推論

新しいプロバイダーとしてCerebrasが登場。 blazing fastな推論速度を提供し、待ち時間を大幅に削減。

⑧Moonshot with Kimi Code

MoonshotがKimi Codeと共に追加。コーディングに特化したモデルを選択肢として利用可能に。

⑨LM Studio連携

LM Studioがローカルモデルホスティングのために追加。14以上のプロバイダーがラインナップに揃う。

⑩MCP App Sampling: 拡張機能が推論

MCPアプリがLLMコンプリーションを直接リクエスト可能に。拡張機能は単にデータを表示するだけでなく、推論して応答できる。

⑪拡張機能の高度化

拡張機能が推論し、応答し、チャット内でより豊かな体験を構築できるようになった。静的な拡張から動的なエージェントへ。

⑫統一されたストリーミング

ストリーミング処理が統一され、よりスムーズなリアルタイムレスポンスを実現。ユーザー体験が向上。

⑬モデル設定の簡素化

モデル設定が簡素化され、より直感的にモデルを切り替え可能。複雑な設定なしで様々なモデルを試せる。

⑭30以上のバグ修正

v1.26.0では30以上のバグ修正が含まれている。安定性と信頼性が大幅に向上。

⑮プライバシー重視の設計

ローカル推論によりデータが外部に送信されない。機密情報を扱う環境でも安心してAIエージェントを利用可能。

⑯Gooseが住む場所

Goose now lives where you doというコンセプト通り、ユーザーの環境に合わせて柔軟に動作するAIエージェント。

⑰シームレスな体験

プロバイダーが増えても同じシームレスな体験を維持。どのモデルを選んでも一貫した操作性を提供。

⑱今すぐアップグレード

block.github.io/gooseから今日アップグレード可能。オープンソースで無料で利用できるAIエージェント環境。
