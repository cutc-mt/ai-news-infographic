2026年3月 AIサブスク選び方ガイド

Claude CodeとCodex二強時代の最適解

2026年3月18日 | まさおAIじっくり解説ch

①

①二強時代の到来

  • Claude CodeとCodexが頭一つ抜けている
  • エージェンティックへの進化が加速
  • ハーネス部分の差が大きく
  • Aランク以下の評価が困難に
モデル性能が良くなりすぎて、体験の差が重要に
②

②$20だけならChatGPT Plus

  • GPT-5.4が利用可能
  • ハーネスなしでも課題解決しやすい
  • Codexのクレジットが緩い
  • Codexアプリも使い放題
$100以下の課金ならChatGPT Plusが正解
③

③Claudeは$200以上推奨

  • Opus 4.6をフル活用するのが前提
  • $20プランでは制限が厳しい
  • Claudeはすぐに枯れやすい
  • Codexより使用量が少ない傾向
$20でClaude課金はあまりおすすめじゃない
④

④GPT-5.4の強さ

  • 知能のアッパーが非常に高い
  • 今までできなかったことが可能に
  • 5.2→5.4でスピードも向上
  • OpenAIが大きく巻き返し
今はGPT-5.4の時代と実感
⑤

⑤Geminiの立ち位置

  • スタンダードプランで十分
  • API経由でCodex/Claudeから呼び出し
  • NotebookLMスライド機能
  • Gemini 3.1 Flashで画像生成
使い方: CLIツールとして定義 → デザインパターン作成など
⑥

⑥GLM: コスパ重視の選択肢

  • 使い放題で雑に使える
  • 性能はSonnet/5.2と同等程度
  • ライトプランがコスパ最強
  • Codexプラン契約するならGLMも
GLM5は初印象いいが、長期コンテキストでずれる傾向
⑦

⑦Cursor / Windsurfの現状

  • Composer 1.5など高速モデル
  • CLI経由でGemini Flash呼び出し
  • 使用量が少ないのが課題
  • セミマニュアル作業向け
CodexやClaude Code契約済みなら使いにくい
⑧

⑧Codex vs Claude Code比較

  • Codex: 使用量多く、枯れにくい
  • Claude: フロントエンド得意、ハーネス充実
  • Codex: フロントエンド苦手
  • Claude: すぐ枯れる傾向
著者の構成: Claude 2台 + GPT 1台
⑨

⑨Devin Reviewが便利

  • PRレビューで自動チェック
  • ガードレールとして活用
  • 今は期間中無料
  • コードレビューに最適
Devin Reviewはめちゃくちゃいい、おすすめ
⑩

⑩GitHub Copilotの立ち位置

  • Claude + OpenAI + GitHubで良好
  • GitHub統合が使いやすい
  • ローカルCLIでのロングランに課題
  • 絶妙なポジション
会社で割り当てられてたら一応使う程度
⑪

⑪音声入力: Amical / Aqua Voice

  • Amical: 無料オープンソース
  • Whisperでローカル動作
  • Aqua Voice: 品質が良い
  • スタバ作業で思考入力
キーボード叩いている間に思考を入力
⑫

⑫Ghostty / Raycast

  • Ghostty: 軽くて使いやすいターミナル
  • Raycast: ショートカットでAI呼び出し
  • OpenAI/Gemini APIキーで無料
  • メッセージ作成などを瞬時に
Raycastは入れておいて損なし
⑬

⑬Manus

  • Webリサーチ機能
  • SimilarWebと連携
  • AIエージェントの先駆者
  • 作りが良いと評価
リファラルクレジットでウォッチ中
⑭

⑭API課金: 画像生成

  • Gemini 3.1 Flash Imageが良い
  • GPT Image 1.5も併用
  • CreAIでまとめて管理
  • CLIからルーティング呼び出し
画像生成はAPI課金で十分
⑮

⑮OpenRouter / Kimi

  • OpenRouter: いろんなモデルを試す
  • 結局GeminiをAPIで叩くことが多い
  • Kimiは別途検証が必要
  • 新モデル確認時のみ使用
Geminiが強すぎて他が選ばれない
⑯

⑯Alibaba Cloud コーディングプラン

  • クレジットカード入力に抵抗
  • GLM、Qwen、Q-3.5、MiniMax
  • Qwenがバランス良くおすすめ
  • 今後検証予定
MiniMaxは日本語がもう少し上手ければ...
⑰

⑰予算別の最適解

  • $20 → ChatGPT Plus
  • $100以下 → ChatGPT Plus
  • $200 → Claude Code or Codex
  • $400 → Claude 2台 + GPT 1台
推奨: $20 Codex + $200 Claude Code + GLM coding plan
⑱

⑱まとめ: 選び方の結論

  • Claude Codeが最もおすすめ
  • $20 Codex + GLMを補助に
  • GPT-5.4はAPI (Kilo)経由で
  • ハーネスエンジニアリングが差に
Claude Code + GLMの組み合わせが最強
