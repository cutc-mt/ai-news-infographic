Agent Skills - Claude Codeで使える自律型ワークフロー

にゃんたのAIチャンネル｜プロンプトとプログラムをパッケージ化

Agent Skillsはプロンプトを自動で差し込む仕組み。必要な時に必要なスキルを読み込んで、業務を効率化します。

①Agent Skillsとは: プロンプトを自動で差し込む仕組み

ユーザーが入力すると、AIが事前に設定したスキルの中からどのスキルを読み込めば良いかを判定して選択します。

②スキルの中身: 手順が書かれた文章

各スキルにはコードレビューや翻訳などどういう手順で何をやればいいのかが書かれています。

③プロンプトとプログラムのパッケージ化

スキルはプロンプトだけでなく特定のプログラムファイルも一緒にパッケージ化して渡すことができます。

④再現性の確保: 同じ入力には同じ出力

プログラムを実行するため同じ入力に対しては同じ出力になり、再現性を持った状態で自動化しやすいです。

⑤SKILL.mdの構成: 名前と説明と手順

スキルはskill.mdファイルで定義。name、description、そして手順を記述します。

⑥段階的な読み込み: 上部だけまずは判定に使用

最初はスキルの上部（名前と説明）だけが読み込まれ、使用が決まったら中身を読み込む効率的な設計です。

⑦精度とコストの最適化: 必要な情報だけを入力

言語モデルは入力が長いと精度が落ちるため、必要な時に必要な情報だけを入れることで品質とコストを最適化します。

⑧2種類のスキル: AIの能力向上と外部知識付与

AIが知っていることをより上手にやらせるスキルと、AIが知らないことを外部から教え込むスキルの2種類があります。

⑨業務特化の価値: 自分専用のスキルを作成

多くのスキルを学ぶより、自分の業務に特化したスキルを自分で作っていくのが長期的には良いアプローチです。

⑩ワークフロー型タスク: 定型的な業務を自動化

議事録作成、競合調査レポート、提案ドラフトなど定型的にやっているデジタルタスクは効率化できます。

⑪Dify・n8nとの違い: 自立型ワークフロー

従来のワークフローは事前に定義した処理を実行しますが、Agent SkillsはAIが各ステップで自律的に判断します。

⑫柔軟な実行: 結果を評価して再実行可能

Web検索の結果が不十分ならAIが判断してもう一度検索するなど、自立した動きができます。

⑬スキル作成の基本: .claude/skillsフォルダー

.claudeフォルダーの下にskillsフォルダーを作成し、その下に各スキルをフォルダーで定義します。

⑭翻訳スキルの例: シンプルな構造

「ユーザーの入力を自然な英語に翻訳する」など自然言語だけでワークフローを記述できます。

⑮競合調査スキル: 複数ステップのワークフロー

Web検索→情報整理→構成提案→承認→レポート作成と複数ステップを自然言語で定義できます。

⑯HTMLレポート生成: 高品質な出力

AIはパワポよりもHTMLやウェブサイトを作る方が得意で、見やすいレポートを生成できます。

⑰skill-creator: スキルを自動作成・評価

Claude Codeの公式プラグインで自然言語で指示するだけでスキルを自動作成できます。

⑱評価と改善: テストケースで品質確認

スキルあり・なしで比較検証し、評価基準に基づいて自動的にスキルを改善するサイクルを構築できます。
