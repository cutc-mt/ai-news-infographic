提案資料はストーリー構成で8割決まる

2026年3月18日 | しごおもTV

よくある勘違い

①営業資料と提案資料は別物

  • サービス説明資料（汎用）と混同している人が多い
  • サービス説明資料: どんな顧客にも同じものを説明
  • 個別提案資料: その顧客用のストーリーが描かれた資料
全く似て非なるもの！混同すると行けてない提案に...

②行けてない提案のパターン

  • 共感がないままサービス説明に入る
  • 「弊社こういうサービスあります。どうですか？」
  • 「それ、うちにとってどういうものか全然わからない」
  • 単なる自己紹介になってしまう
「確かにうちこんなことできます、こんな機能持ってます、ふーん」で終わる...

③解釈が抜けるパターン

  • 前回の相談内容をただ整理しただけ
  • 「予算感こう、スケジュール感こう、このサービスがいいです」
  • あなたの解釈がない = 相手の気持ちになれない
  • 疑似録とサービス説明だけでは共感が生まれない
× 前回の相談を整理 + サービス紹介
○ 状況 → 解釈 → 解決策のセット
正しい提案資料の構成

④王道パターンの構成

  • まず共感を作る
  • 問題解決ストーリーを納得してもらう
  • 「確かにそれなら良さげ」と思ってもらう
  • その後にサービスの具体を出す
共感 → 納得 → サービス紹介
この順番が大前提として重要！

⑤3点セット: 状況・解釈・解決策

  • 状況: 今こういう状態ですよね
  • これまでのやり取り、問題意識、中継計画、方針
  • 解釈: 要するにこういうことじゃないか
  • ボトルネックを特定、解消後の世界を提示
  • 解決策: うちがこういうご支援できますよ
状況: 「今こういう状態ですよね」
解釈: 「要するにこういうことでは？」
解決策: 「だからこれが良いんです」

⑥解釈こそが肝

  • 状況と解釈で共感を作るのが大事
  • 「あ、そうそう、まさにそうなんですよね！」
  • 「なるほど、そういう解釈してくれるんだ」
  • 確かにこれに頼んだらそれ作れそう → ゴーと言える
解釈が抜ける = 相手の気持ちになれない！
あなたの考えを示すことが重要
共感を作る「4つの壁」
1

⑦やばいよの壁

  • そもそも何が問題だと認識されているか
  • 理想と現状のギャップを認識してもらう
  • 抽象的・先進的なサービスだと超えられないことが多い
「従業員エンゲージメントを高めるサービスです」
→ 「別にエンゲージメント低くてもやばいと思わないけど」で話が始まらない！
2

⑧なんとかしないとの壁

  • 理想と現状のギャップは死ぬほどある
  • その中でも「こいつは確かに重要だぞ」と認識してもらう
  • 優先度を高くしてもらう必要がある
  • 「そのままでいいじゃん」「他にも色々あるから優先度低い」を超える
コピー機が古い、Aさんの営業がダメ、組織フードを変えなきゃ...
その中でこれが重要だと思ってもらう！
3

⑨これがベストの壁

  • 解決策は色々ある
  • お金かけずに内製？A社？B社？
  • 「これが一番いいね」と思ってもらう
  • コスト面でも期間面でも良さそうと納得
なんとかするぞ！と思った上で
「この会社が一番良さそうだ」と
判断してもらう
4

⑩早速やろうの壁

  • 問題は認識、なんとかしたい、これがベスト
  • でも「今じゃない」と止まってしまう
  • スケジュールとして今やる必要があると認識
  • 初めて話が動く
4つの壁を全部超えないと話が進まない！
どれか1つでも止まると契約に至らない
実践ポイント

⑪順番に超える必要がある

  • 4つの壁は順番に超えていく必要がある
  • いきなり全部は飛び越えられない
  • 今どの壁まで超えられているか認識しながら
  • 商談プロセスとして段階を踏む
1 → 2 → 3 → 4
1個1個潰していくことが大事！

⑫同時多発的に起きることも

  • ルート営業では同時多発的に進む
  • 「早速やろう」だけ決まって「なんとかしたい」が曖昧
  • 新商品入れたいが「やばいよ」が取れない
  • 4つ全部をチェックリスト的に潰す意識を持つ

フレームワークとして頭に入れておく

⑬提案資料で「一人歩き」させる

  • 決裁権者が常に同席するとは限らない
  • 担当者が決裁者を説得できる資料が必要
  • 資料を見るだけで4つの壁を超えられる
  • いい意味で一人歩きする資料を作る
担当者が後で説明しなくても分かる資料こそが
本当の提案資料！
まとめ

⑭今日のまとめ

  • サービス説明資料≠提案資料
  • 状況・解釈・解決策のセットが大事
  • 特に解釈が抜けがち（要注意！）
  • 4つの壁を超えるストーリーを組み立てる
  • 決裁者に届く資料を作る

⑮これからの時代に必要な力

  • 課題を特定する力
  • 問題解決ストーリーを描く力
  • 人を納得させて動かす力
  • 営業に限らず普遍的に強いスキル
提案ストーリーを組み立てて
人を動かす力を身につけよう！

⑯4つの壁フレームワーク

1. やばいよの壁
   → 問題だと認識させる

2. なんとかしないとの壁
   → 優先度を高くする

3. これがベストの壁
   → 自社を選んでもらう

4. 早速やろうの壁
   → 今やる理由を作る

重要 営業必須知識

⑰提案資料の効果的な使い方

  • 目の前の担当者だけでなく決裁者に訴える
  • 商談プロセスの進捗を可視化
  • どの壁まで超えられたか把握
  • 資料単体でストーリーが完結する
担当者が社内で説明しやすい資料かどうかが鍵！

⑱実践へのアクション

  • 今の提案資料に「解釈」があるか確認
  • 4つの壁を全て超えられているかチェック
  • 資料だけで決裁者が納得できるか検証
  • ストーリー構成を見直してみる
