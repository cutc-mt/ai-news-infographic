第1章: スキルの概要

①スキルとは

  • Claude Codeにマニュアルを渡す仕組み
  • 正式名称: Agent Skills（エージェントスキルズ）
  • 2025年12月に正式リリース
  • オープン企画として公開（他社AIエージェントにも対応）
高品質なデザインやアプリを作るために必須の機能！

②業界標準化の動き

  • Anthropicが開発したオープン企画
  • OpenAIのCodex、GoogleのGemini CLIなども採用
  • AIエージェントに能力を付与する標準規格になりつつある
  • 共通ルールとして誰でも使える

オープン規格 標準化

③スキルの仕組み

  • マニュアル + プログラムのパッケージ
  • Claude Codeが必要な時だけ読み込まれる
  • 記憶容量を節約しながら高品質なコンテンツを作成
  • GitHubに公開されている
# スキルの構成
skill.md - スキルの本体（マニュアル）
scripts/ - プログラムデータ
reference/ - 参考資料

④公式スキルの活用推奨

  • Anthropic、Canva、Figmaなどが公式スキルを公開
  • 一般ユーザーのスキルはセキュリティ面で注意
  • よく分からない人は公式スキルだけ使うのが安全
  • Vercelのリーダーボードで人気スキルを探せる
公式スキルから始めよう！
第2章: スキルを使う

⑤スキルの配置場所

  • .claude/skills/: 全プロジェクトで利用可能
  • プロジェクト内/skills/: そのプロジェクト限定
  • ルールに従って配置すると自動認識
# 配置例（全プロジェクト共通）
.claude/
  └── skills/
      └── web-artifact-builder/
          ├── skill.md
          └── scripts/

# 配置例（プロジェクト限定）
my-project/
  └── skills/
      └── remotion/

⑥CLIでスキルを使う方法

  • Anthropic公式スキルをGitHubからダウンロード
  • ZIPを解凍してskillsフォルダに配置
  • プロジェクトフォルダでClaude Codeを起動
  • 通常通りプロンプトを入力すると自動でスキルが読み込まれる
ダウンロード → 配置だけ！

⑦Web Artifact Builderスキル

  • Anthropic公式スキル
  • AIっぽい雑なデザインにならない高品質UI
  • 複数要素を組み合わせたリッチなWebアプリに最適
  • タスク管理アプリのような複雑なUIも作成可能

Anthropic公式 高品質UI

⑧デスクトップアプリでスキルを使う

  • 左側のカスタマイズ → スキル
  • スキルを選択してトグルをON
  • プラスボタンでサードパーティスキル追加
  • skill.mdを直接アップロード可能
# アップロード方法
1. skill.mdを直接アップロード
2. 複数ファイルの場合
   → skill.mdを最上層に配置したZIPでアップロード

⑨Remotionスキル（動画生成）

  • 日本語の指示でアニメーション動画を生成
  • Reactベースの動画生成ツール
  • 横長動画（解説動画など）に最適
  • MP4でダウンロード可能
チャットで動画生成！
第3章: スキルを作る

⑩Skill Creatorスキル

  • Anthropicが公式で提供するスキル生成用スキル
  • 自分で作るよりも簡単に高品質なスキルが作れる
  • デスクトップアプリに標準実装
  • CLI用はGitHubからダウンロード可能

公式 スキル生成

⑪スキルの作り方

  • カスタマイズ → スキルでSkill CreatorをON
  • 納品フォルダーを選択
  • 「〜なスキルを作って」と指示
  • スキルとプログラムを自動生成
# プロンプト例
「スキルクリエイターを使って
90年代のレトロデザイン風の
ウェブサイトを作るためのスキルを
作って納品フォルダーに保存」

⑫作成したスキルを使う

  • 生成されたスキルの構成を確認
  • フォルダ内にスキルを配置
  • 「〇〇スキルを使って〜」と指示
  • 独自のデザインでコンテンツ生成
オリジナルスキルで独自のワークフロー！
