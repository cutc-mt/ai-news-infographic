③自動化の基本アプローチ

  • 普段やっているワークフローを言語化
  • それをClaude Codeにスキル化してと頼む
  • 一度作れば何度でも使い回し可能
  • 部下を育てる感覚でスキルを改良
ワークフローさえ言語化できれば自動化できる！
自動化①: 会議議事録 → Notion

④ワークフロー

会議を録音・文字起こし
↓
Claude Codeにファイルを投げる
↓
Claude Codeが内容を分析
↓
決定事項・タスクを抽出
↓
Notionデータベースに自動保存

⑤効果: 時間短縮

  • 人間がやると30分〜1時間
  • Claude Codeなら約3分
  • Notionにタスクが自動保存される
  • 担当・期限・ステータスも自動設定
大幅な時間短縮を実現！

⑥Notion MCP連携の手順

  • Claudeブラウザでコネクター設定
  • Notionコネクターを追加
  • ワークスペースを選択して連携
  • ブラウザのコネクターはCodeと連動
# 手順
1. claude.ai → Settings → Connectors
2. Notionを追加
3. ワークスペースを選択
4. Claude Codeで使用可能に

⑦生成されるNotionデータベース

  • タスク名: 抽出されたタスク
  • 担当者: 自動割り当て
  • 期限: 自動設定
  • ステータス: 未着手/進行中
  • 優先度: 高/中/低

自己責任 高速執筆

セットアップ方法

⑭CCカンパニーのセットアップ

  • Cursorエディターをインストール
  • 新しいフォルダを作成
  • ターミナルでclaudeコマンド
  • 公式ドキュメントとスキルを導入
# セットアップ手順
1. フォルダ作成: company-youtube-test
2. ターミナルで claude と入力
3. 公式ドキュメントをコピー
4. プラグインをリロード

⑮スキルの作り方

  • ワークフローを画像やテキストで説明
  • 「このワークフローのスキルを作って」と指示
  • Skillsディレクトリに配置
  • リファレンスディレクトリにテンプレートを追加
画像を貼るだけでスキルを作成！

⑯秘書（シークレタリー）を育てる

  • 最初は失敗しても仕方ない
  • Claude MDファイルを更新して改良
  • 次回のスキル作成時に反映
  • 部下を育てる感覚で運用
