③事例1: X投稿の自動化

  • Claude Code上で投稿を作成
  • Typefullyと連携して下書き保存
  • XのAPIだけでは下書き保存不可
  • Typefully APIとの組み合わせがポイント
スレッド投稿も自動作成！

④事例2: 議事録→お礼メール自動化

  • 会議終了後即座に議事録作成
  • 参加者へのお礼メールを自動作成
  • 下書きを確認して送信ボタンだけ
  • Nottaで文字起こし→Notionに格納
# ワークフロー
会議 → Notta文字起こし → Notion格納
    → Claude Code検知 → 議事録作成
    → お礼メール下書き → 人間が確認→送信

⑤事例3: MA案件精査の自動化

  • Notion・Slackに通知
  • AIが案件を分析して評価
不動産など他業種でも応用可能！
