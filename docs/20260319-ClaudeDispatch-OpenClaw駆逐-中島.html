Claude Dispatch: OpenClawを駆逐？

Claude Dispatchとは

①Claude Dispatchの概要

  • Anthropicが発表した新機能
  • 外出先から自宅のPCをリモート操作
  • スマホからClaudeに指示を出す
  • PC上のClaudeが処理を実行
OpenClawと同じことが公式で可能に！

②できること

  • スマホからPC上のClaudeにメッセージ送信
  • ローカルファイルにアクセス
  • ブラウザ操作を許可
  • 処理結果をメッセージで受信

リモートアクセス 公式サポート

③提供開始時期

  • 本日からMAXプランに段階的ロールアウト
  • 明日からProプランにロールアウト
  • 利用にはデスクトップ＆モバイルアプリが必要
Proプランは明日から利用可能！

④接続の仕組み

  • PCのClaudeでQRコードを表示
  • スマホのClaudeでスキャン
  • 簡単に接続完了
  • 永続的な単一スレッドで管理
# 接続手順
1. PC: Claude Desktopを開く
2. Dispatchタブをクリック
3. QRコードが表示される
4. スマホでスキャン → 接続完了
OpenClawとの比較

⑤OpenClawの問題点

  • セキュリティの脆弱性
  • 自宅サーバーをネットに公開する必要
  • 専門知識がないとリスクが高い
  • メインPC公開でハッキングリスク
注意: 素人が触るのは危険！

⑥Dispatchの利点

  • 公式サポートで安心
  • 専門知識がなくても安全に使える
  • 簡単セットアップ
  • Anthropicが管理する安全な接続
エンジニアでなくても安全に利用可能！

⑦Mac mini需要の背景

  • OpenClaw用にMac miniが品薄
  • 専用PCを用意することでリスク軽減
  • 個人情報のないPCなら流出リスクなし
  • Dispatchでも同じ考え方が推奨される

専用PC推奨 リスク管理

機能と設定

⑧主な設定項目

  • ブラウザアクション許可: 外部からブラウザ操作
  • スリープしない: PCを起動状態に維持
  • PCがスリープ状態だと操作不可
# 設定時の注意
✓ ブラウザアクションを許可
✓ スリープしない設定をON
⚠ バッテリー消費に注意（ノートPC）

⑨永続スレッドの仕組み

  • 単一の永続的なスレッドのみ
  • 新しいセッション開始は不可
  • 中断したところから再開可能
  • 以前のコンテキストを保持
コンテキストが維持される！

⑩できることの例

  • ローカルスプレッドシートからレポート作成
  • Slackメッセージ・メールからブリーフィング作成
  • Googleドライブファイルからプレゼン作成
  • 特定フォルダのファイル整理・処理

ファイル生成 ローカルアクセス

⑪できないこと（現在）

  • タスク完了通知はなし
  • スケジュール済みタスクはなし
  • 新しいスレッドの開始は不可
  • 詳細な進捗表示は出ない
研究プレビュー版なので制限あり
セキュリティと注意点

⑫セキュリティリスク

  • ネット経由でアクセスできる点は変わらない
  • Claudeに付与したアクセス権に注意
  • プロンプトインジェクション攻撃のリスク
  • 悪意あるコードでトークンが抜かれる可能性
重要: メインPCではなく専用PCの使用を推奨！

⑬安全な使い方

  • 専用PC（Mac mini等）を用意
  • 個人情報を入れない
  • クラウドデスクトップの利用も選択肢
  • 月額5000円程度でクラウドPC利用可能
# クラウドデスクトップの例
- Windows/macOSをクラウドで利用
- 月額5000円程度
- データはクラウド側に残る
- リスクを最小限に抑えられる

⑭研究プレビュー版であること

  • 正式版ではない
  • トラブルがあっても自己責任
  • 人柱になる覚悟を持って利用
  • 早く使いたい場合はリスクを理解する
自己責任で利用しよう！
