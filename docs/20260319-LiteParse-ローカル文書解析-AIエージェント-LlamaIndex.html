LiteParse: ローカル文書解析ツール

2026年3月19日 | LlamaIndex

LiteParseとは

①LiteParseの概要

  • オープンソースの文書解析ツール
  • LLM不要・クラウド依存なしで動作
  • 空間的なテキスト抽出が可能
  • 高速で完全ローカルなテキスト抽出
AIエージェント向けのローカル解析ツール！

②対応プラットフォーム

  • Linux対応
  • macOS対応
  • Windows対応
  • スタンドアロンバイナリでインストール可能

クロスプラットフォーム 単一バイナリ

③主な機能

  • 柔軟なOCRシステム（デフォルト内蔵）
  • HTTPサーバーで拡張可能
  • スクリーンショット生成
  • 複数出力フォーマット（JSON/Text）
  • バウンディングボックス検出

④対応ファイル形式

PDF
Word (.docx)
PowerPoint
Excel
画像
テキスト

※Office文書と画像には追加依存関係が必要

インストールと使い方

⑤インストール方法

  • npm経由でインストール
  • グローバルインストールでCLI使用可能
  • TypeScriptライブラリとしても利用可能
  • Pythonパッケージ（uvツール）としても利用可能
# npm経由でインストール
npm install -g llamaindex/liteparse

# インストール確認
lite-parse --help

⑥基本的な使い方: テキスト出力

  • parseコマンドで文書を解析
  • PDFのテキストを抽出
  • 空間的なテキスト分布を保持
  • 22ページを1秒未満で処理
# PDFをテキストに変換
lite-parse parse dips-ocr.pdf > output.txt

# PyPDFと違い、空間分布を保持！

⑦JSON出力フォーマット

  • format: jsonで構造化データを出力
  • ページごとの幅・高さ・テキスト情報
  • バウンディングボックス付き
  • フォント検出・フォントサイズ検出
# JSON形式で出力
lite-parse parse doc.pdf --format json > output.json

# 出力: pages配列、textItems、boundingBoxes

⑧バッチ処理

  • batch-parseで複数ファイルを一括処理
  • 入力ディレクトリと出力ディレクトリを指定
  • 複数のPDFを一気にテキスト化
  • JSON形式での一括出力も可能
# バッチ処理
lite-parse batch-parse \
  --input papers \
  --output papers-parsed

# 4つのPDFがテキストファイルに変換される

⑨スクリーンショット機能

  • screenshotコマンドでページ画像を抽出
  • ページ範囲を指定可能
  • PNG形式で出力
  • 文書のプレビューに最適
# スクリーンショットを抽出
lite-parse screenshot doc.pdf \
  --pages 1-10 \
  --output screenshots/

# page1.png, page2.png... が生成される
拡張機能と統合

⑩カスタムOCRサーバー

  • デフォルトはTesseract.jsが内蔵
  • HTTPサーバーでOCRを拡張可能
  • EasyOCRなどの外部OCRを使用可能
  • 言語パラメータの指定も対応
# カスタムOCRサーバーを使用
lite-parse parse doc.pdf \
  --ocr-server-url http://localhost:8000

⑪Office文書・画像の追加依存関係

  • LibreOffice: Office文書の解析に必要
  • ImageMagick: 画像処理に必要
  • macOS: brew経由でインストール
  • Ubuntu/Debian: apt-getでインストール
# macOSの場合
brew install libreoffice
brew install imagemagick

⑫Claude Code Skillとしての使用

  • Vercel Skills CLIでスキルとして追加
  • Claude Codeから直接呼び出し可能
  • 文書解析→要約のワークフローを自動化
  • スキル対応の全エージェントで利用可能
Claude Codeで「このPDFを解析して要約して」！

⑬Skillのインストール手順

  • llama-parse-agent-skillsリポジトリから追加
  • Claude Code用にセットアップ
  • シムリンクでプロジェクトにインストール
# Skillを追加
bash add-skill \
  --agent cloud-code \
  --skill-repo llama-parse-agent-skills

# lite-parse skillを選択
