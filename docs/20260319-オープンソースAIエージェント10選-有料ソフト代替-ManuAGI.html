🤖 オープンソースAIエージェント10選

有料ソフトを代替する最新プロジェクト

Open Source AI Agents GitHub Projects ManuAGI

①Goose: ローカル拡張可能なAIエージェント

Block社が開発したローカル実行のオープンソースAIエージェント。単なるコード提案ではなく、コードの記述・実行、デバッグ、ワークフローのオーケストレーション、外部APIとの自律的な連携が可能。任意のLLMで動作し、MCPサーバーとも統合できる。

②Page Agent: ウェブインターフェース制御

自然言語でウェブインターフェースを制御できるJavaScriptインページGUIエージェント。Operatorスタイルのツールで、人間と同じようにウェブサイトやインターフェースを使用できる。有料のオートメーションプラットフォームのオープンソース代替。

③Agent Orchestrator: 並列コーディングエージェント

複数のAIエージェントを協調させるアジェンティックオーケストレーター。タスクの計画、エージェントの起動、CI修正、マージコンフリクト解決、コードレビューを自律的に処理。高度なオートメーションプラットフォームのオープンソース版。

④Hermes Agent: 成長するAIエージェント

経験から学習し成長し続ける完全オープンソースのAIエージェント。学習ループを内蔵し、経験からスキルを作成、使用中に改善、セッションをまたいで記憶を構築。低コストVPSから大規模インフラまで動作可能。

⑤Claude Code Plugins: 公式プラグインディレクトリ

Anthropicが提供するClaude Code用の高品質プラグインのキュレートされたディレクトリ。カスタムコマンド、エージェント、ワークフローでClaude Codeを拡張。公式マーケットプレイスはClaude Code内から直接アクセス可能。

⑥GitHub Copilot SDK: プログラマブルなエージェントランタイム

Copilot CLIと同じエンジンにプログラム的にアクセスできるSDK。計画、ツール呼び出し、ファイル編集などを処理する本番テスト済みのエージェントランタイム。現在テクニカルプレビュー中で、独自のツールやワークフローに組み込み可能。

⑦Symphony: チーム向け自律実行システム

プロジェクトワークを独立した自律的な実行単位に変換。チームはコーディングエージェントを絶えず監督する代わりに、作業を管理できる。アシスタントスタイルのAIから本格的なアジェンティックワークフローへの移行を実現。

⑧HiClaw: マルチエージェント協調OS

チャットルーム内のAIチームを実現するオープンソースの協調マルチエージェントOS。マネージャー・ワーカーアーキテクチャで、すべての会話が可視化され、人間がいつでも介入可能。Matrixルーム経由で透明性のあるタスク調整。

⑨NotebookLM-py: Notebook LMのPython API

Google Notebook LMの非公式Python APIとエージェントスキル。Python、CLI、Claude CodeやOpenClawなどのAIエージェントからNotebook LMの機能にアクセス可能。ウェブUIでは公開されていない機能も利用できる。

⑩OpenSandbox: AIアプリ用サンドボックス

AIアプリケーション向けの汎用サンドボックスプラットフォーム。多言語SDK、統一サンドボックスAPI、Docker/Kubernetesランタイムを提供。コーディングエージェント、GUIエージェント、エージェント評価、AIコード実行、RL学習などに対応。

⑪なぜオープンソースが重要か

多くの開発者が制限的でクローズドな有料AIコーディングツールに慣れつつある。オープンソースエージェントは、カスタマイズ可能で、透明性があり、コミュニティ主導の代替手段を提供。アプリケーションだけでなくインフラ層も置き換え可能。

⑫コーディングエージェントの進化

オートコンプリートスタイルの提案から、実際のエンジニアリング作業を支援するエージェントへ進化。コードの記述・実行・デバッグ・オーケストレーションまで自律的に処理できる能力が、オープンソースでも実現可能に。

⑬ブラウザオートメーションの民主化

Operatorスタイルのツールはプレミアム製品や有料プラットフォームにロックインされがち。Page Agentのようなオープンソースプロジェクトが、自然言語でのウェブ制御を誰でも利用可能にする。

⑭マルチエージェントシステムの台頭

単一のアシスタントから複数のエージェントが協調するシステムへ。Agent OrchestratorやHiClawは、企業が高額なオートメーションプラットフォームに支払うような機能を、オープンソースで提供。

⑮永続的な記憶と学習

多くの有料AIアシスタントはセッションごとにリセットされる印象。Hermes Agentは反対方向に進み、記憶し、学習し、改善し、時間とともにより役立つ永続的なAIチームメイトを実現。

⑯モジュラーなエージェントエコシステム

有料ソフトの価値はモデルだけでなくエコシステムにもある。ワークフロー、統合、コマンド、専門的な拡張機能。Claude Code Pluginsのようなプロジェクトが、そのエコシステムをオープンで柔軟なものに変える。

⑰インフラとしてのエージェントソフトウェア

エージェントランタイムがプログラマブルになると、開発者は固定されたAI製品体験に制限されない。独自のツール、ワークフロー、アプリケーションにエージェントの振る舞いを埋め込める。Copilot SDKがその動きを示唆。

⑱安全な実行環境の重要性

AIエージェントがデモを超えて本格的な作業を行うには、安全で構造化された実行場所が必要。OpenSandboxは、コードを書き、テストし、インターフェースを実行するための制御されたランタイムインフラを提供。
