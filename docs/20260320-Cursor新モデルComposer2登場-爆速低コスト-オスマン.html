🚀 Cursor新モデル Composer 2登場

爆速モデル&低コスト化で鍵になる可能性

Cursor Composer 2 独自モデル オスマン

①Composer 2の登場

Cursorから新しい独自モデルComposer 2が登場。1.5では速度だけの価値だったが、2になってトップクラスのモデルに近づいた、あるいは超えているというアピールがある。

②モデルベンダー依存からの脱却

CursorはこれまでOpenAIやClaudeのOpusなどに依存し、コスパの悪い戦いをさせられていた。Composer 2の登場でその可能性から脱却できる期待が高まっている。

③ベンチマークでの位置づけ

Cursor独自ベンチマークではGPT 5.4 Mediumより下、Opusより上とアピール。ただしOpusの代わりに使うかと言うと、Claude CodeとOpus 4.6は別格との評価。

④VS miniやGPT 5.4 Miniと比較すると優秀

トップモデルと比較するとがっかりだが、VS miniやGPT 5.4 Mini、GLMと比較すると非常にいいモデル。GLMよりもComposerの方が好きという評価もある。

⑤フロンティアレベルのコーディング性能

コーディング系の性能は高い印象。ただしSVG作成などはあまり得意ではなさそう。開発に特化したモデルという感じがある。

⑥ターミナルベンチマークでの向上

ターミナル上の操作がうまくこなせるようになった。CLIから使って検証した結果、ターミナルベンチが上がっていて非常に嬉しい改善点。

⑦標準モデルの価格設定

Input 0.5、Output 2.5ドル。Gemini Flashと同じ価格帯。Opusや5.4よりも遅いが、価格は大幅に安く設定されている。

⑧ファストモデルの価格と速度

1秒あたり200トークンの高速モデルはInput 1.5、Output 7.5ドル。DeepSeek Proよりは安く、Gemini 2.0より少し高い価格設定。

⑨コスパ問題の解消

以前はClaude Opusを使うとAPI課金が高くなり、Claude Codeの方がコスパが良いという状態だった。独自モデルで攻めることでこの問題を解消。

⑩専用プールと利用枠

オートモードやオートプラスComposerモードにするとより多くの利用枠が使える。独自モデルを持つことで、こうした柔軟な提供が可能に。

⑪月額利用目安

毎日エージェントを使うユーザーは月6〜100ドル、パワーユーザーは200ドル以上。料金の目安が明示されている。

⑫テトリス作成デモの速度

実際にテトリスを1枚のHTMLで作成させるととにかく速い。さらにそこそこ賢いので、ベンチマークの見せ方を勘違いしなければ良いモデル。

⑬マルチエージェントベンチ結果

ノイズの中からロールプレイ、マルチステップでのスキル呼び出しテストでMiniMaxやKimiより良好。指示追従も問題なく対応。

⑭3000文字記事作成テスト

総合して80点ぐらいで結構良い出力。MiniMaxは全然ダメだったが、5.4と同等レベルで記事作成ができている。

⑮LLM論文の罠トラップ問題

よくあるトラップにあまり引っかからず、指示追従の能力が高いことを確認。制約付きのポエム作成などもうまく対応。

⑯SVG描写は苦手

サムネイル作成や複雑なSVG描写はGPT 5.4 MiniやKimiの方が得意。画像描写関連はあまり得意ではなさそう。

⑰画像分析は得意

画像の詳細な描写に関してはMiniMaxやGPTより高得点。細かく画像を分析できる能力がある。

⑱マーメイドチャート作成が優秀

マーメイドチャート作成では非常に高得点。見やすさや課題への対応も良好で、コーディングエージェントとしての実力を発揮。
