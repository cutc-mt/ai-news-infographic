🟢 Nemotron 3 Super + NemoClaw

NVIDIA無料APIで使えるエージェントAI

NVIDIA Nemotron GTC 2026 WorldofAI

①GTC 2026の最大発表: Vera Rubin

NVIDIAが次世代エージェントAIのフロンティアを開放。Vera CPU、Rubin GPU、NVLink 6、ConnectX9、BlueField4、Spectrum6、Groq LPUを統合した巨大AIスーパーコンピュータープラットフォーム。

②Dynamo 1.0: 推論用OS

AIファクトリー向けのオープンソース推論ソフトウェア。ルーティング、メモリ移動、スケジューリング、推論経済性を最適化。Blackwell推論パフォーマンスを最大7倍向上と主張。

③NemoClaw: OpenClawコミュニティへ

単一コマンドでNemotronモデルとOpenShellランタイムをインストール可能。プライバシーとセキュリティコントロールを強化した常時稼働自律エージェント向け。クラウドからRTX PC、DGX Station、DGX Sparkまで対応。

④拡張モデルファミリー

NVIDIAがモデルファミリーを拡張。NemotronはエージェントAI、Cosmosは物理AI、Isaac Grootはロボティクス、Alpamoは自動運転、BioMoは科学・ヘルスケア向けに位置づけ。

⑤Nemotron連合の発足

Cursor、LangChain、Mistral、Perplexity、Reflection、Sarvam、Thinking Machines Labなどと協力しフロンティアオープンモデルを共同構築。最初の連合モデルはNemotron 4ファミリーの基盤に。

⑥Nemotron 3 Superとは

Mixture of Experts + Mamba Transformerハイブリッドのオープンエージェント推論モデル。コーディング、ツール使用、ターミナル使用、長文脈推論、エージェントワークフローに特化。

⑦パラメータ構成

総パラメータ約1,200億、トークンあたりのアクティブパラメータ約120億。能力と効率のバランスを実現した設計。

⑧圧倒的な推論スループット

GPT OSS 120Bより約2.2倍高いスループット、Qwen 3 235Bより約7.5倍高いスループットを達成（NVIDIAテスト環境）。オープンモデルながら高速推論を実現。

⑨オープンな提供形態

ウェイトはオープン、トレーニングレシピを共有、ポストトレーニングデータの一部も公開。クローズドエコシステムへの依存を回避。

⑩コーディングエージェントに最適

NVIDIAはNemotronをコーディング支援、検索、複雑なワークフロー自動化に位置づけ。Kilo CLI、OpenCode、Roo、ClineなどのOpenAI互換ツールで利用可能。

⑪ターミナル＆ツール使用

コードベースの検査、シェル出力の推論、長時間稼働エージェント向けに設計。NVIDIAはOpenClawとNemoClawのエコシステムで長時間エージェントを強化。

⑫無料APIアクセス

build.nvidia.comからAPIキーを取得し、無料で試用可能。セルフホスティングのセットアップなしでモデルを試せる。

⑬OpenAI互換API

ベースURLはintegrate.nvidia.com/v1で通常のChat Completionスタイル。OpenAI互換エンドポイントをサポートするツールなら即座に統合可能。

⑭推論モードの切り替え

推論はデフォルトで有効。高速なレスポンスが必要な場合はオフに可能。難しい計画立案やデバッグにはオン、単純な編集にはオフが推奨。

⑮Kilo CLI / OpenCodeでの設定

/connectコマンドでNVIDIAプロバイダーを選択し、APIキーを入力。/modelsからモデルを選択。OpenCodeではAPIキーなしで無料試用も可能。

⑯実践的なユースケース: リポ計画

大きなタスクを与え、コードベースを検査させ、実装計画を生成させる。アーキテクトモード、プランニングモード、リポ理解に最適。

⑰実践的なユースケース: コードレビュー＆デバッグ

エージェント推論とターミナル使用のトレーニングにより、失敗のトレース、ログ読み込み、問題特定に強い。

⑱NVIDIAのフルスタック戦略

NVIDIAは単なるモデルではなくオープンエージェントAIのフルスタックを構築。インフラ（Vera Rubin）、推論オーケストレーション（Dynamo）、エージェントランタイム（NemoClaw）、オープンモデル（Nemotron）を統合。
