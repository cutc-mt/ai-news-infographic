📱 Claude Code Channels機能

Discord/TelegramからClaude Codeをリモート操作

Claude Code Channels Discord Telegram AI Jason

①Channels機能の概要

Claude Codeの新しいChannels機能で、自宅のPCで動作するClaude CodeセッションをスマホからTelegramやDiscord経由でリモート操作可能に。

②外出先からも操作可能

自宅のPCでClaude Codeセッションを起動しておけば、外出先からスマホでメッセージを送受信できる。家を出て、スマホでTelegramやDiscordを開いてセッションを管理。

③研究プレビュー版

この機能はまだ研究プレビューの段階。バグが存在する可能性があるため、使用時は注意が必要。

④Claude Code 2.1.8以降が必要

Channels機能を使うにはClaude Code 2.1.8以降にアップデートが必要。ターミナルでclaude updateを実行して更新。

⑤Claude.aiログインが必須

Claude.aiアカウントでのログインが必要。コンソールやAPIキー認証を使用している場合は機能しないため、ログアウトして再ログインが必要。

⑥プランによる違い

ProまたはMaxプランでは即座に利用可能。TeamまたはEnterpriseアカウントの場合は、設定ファイルで権限を手動で変更する必要がある。

⑦Telegram設定: Bot作成

TelegramのBotFatherにアクセスし、/newで新しいボットを作成。ボット名とユーザー名を設定してボットトークンを取得。

⑧Telegram設定: プラグインインストール

Claude CodeでTelegramプラグインをインストール。デフォルト設定（user scope）でインストールし、セッションを再起動。

⑨Telegram設定: トークン設定

/telegram-configure [ボットトークン]コマンドでトークンを設定。プラグインインストール後はセッションを再起動する必要がある。

⑩Telegram設定: チャンネルタグで起動

claude --channels telegramでチャンネルタグを有効化してClaude Codeを起動。これでTelegramからのメッセージを待機開始。

⑪Telegram設定: ペアリング

Telegramでボットにメッセージを送るとペアリングコードが返ってくる。そのコマンドをClaude Codeに入力して接続を承認。

⑫Telegram設定: アクセス制限

ペアリング後、/telegram-policy-allowlistを実行して自分だけがアクセス可能にロックダウン。他のユーザーからのアクセスをブロック。

⑬Discord設定: アプリ作成

Discord Developer Portalで新規アプリケーションを作成。Botセクションで「Message Content Intent」を有効化し、トークンを取得。

⑭Discord設定: OAuth2設定

OAuth2タブでURL Generatorを使用。Botスコープを選択し、必要な権限を設定。生成されたURLでボットをサーバーに追加。

⑮Discord設定: プラグインとトークン

Telegramと同様にDiscordプラグインをインストールし、Developer Portalで取得したトークンを設定。セッション再起動後に有効化。

⑯メッセージの分離

Telegram/Discordから送ったメッセージはセッションで確認できるが、セッション内で送ったメッセージはTelegram/Discordには送信されない。双方向の完全同期ではない点に注意。

⑰動作確認

設定完了後、TelegramやDiscordから「Hi」と送信すると、Claude Codeから「Hi, how can I help you?」とレスポンスが返ってくる。リモート操作が機能していることを確認。

⑱セキュリティのベストプラクティス

必ずallow listでアクセスを自分だけに制限。ペアリングコードを入力した直後にポリシーを設定して、第三者によるセッションへのアクセスを防止。
