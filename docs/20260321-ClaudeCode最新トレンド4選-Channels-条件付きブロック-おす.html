Claude Code Channels スキル おす

①Claude Codeのアップデート加速

Claude Codeのアップデートが猛烈に加速中。ハーネスを作っている間にもトレンドをキャッチアップできないほど。LLMハーネス全体が急速に進化している。

②CLAUDE.mdの条件付きブロック

重要な指示を条件付きブロックで囲うことで、Claudeが常に意識するようにできる。CLAUDE.mdが長くなると参考情報として流されやすい問題を解決。

③条件付きブロックの書き方

important ifコードブロックで指示を囲う。HumanLayerが推奨するこの方法で、常時重要な情報と条件付きで重要な情報を分けて記述すると守りやすい。

④HumanLayerのCLAUDE.md改善スキル

HumanLayerがCLAUDE.mdを改善するスキルも公開。叩き台として活用すれば、より効果的なCLAUDE.mdが作成可能。

⑤スキル内でのコマンド実行

スキルの中で感嘆符(!)で始めてコマンドを書くと、マークダウン読み出しタイミングで実行され、その出力結果がコンテキストに注入される。

⑥動的コンテキスト注入

この機能で動的にコンテキストを注入可能。例えば!ghコマンドでGitHubの情報を取得し、スキル読み込み時に自動的に反映。

⑦動的スキルのセキュリティ注意点

便利だが他人のスキルを使う場合は危険。コマンド実行が含まれている可能性があるため、信頼できるスキルのみ使用すること。

⑧Channels機能: OpenClaw的な使い方

Claude CodeのChannels機能でDiscordやTelegramからAIエージェントを呼び出し可能。汎用AIエージェントとしてパーソナル秘書や仮想社員のように活用。

⑨Telegram設定手順

TelegramのBotFatherで新規ボット作成→APIキー取得→プラグインインストール→トークン設定→チャンネルオンで設定完了。

⑩Telegram設定コマンド

/install-pluginでプラグインをインストールし、/telegram-configure [トークン]で設定。チャンネルをオンにしてペアリングコードを入力。

⑪スマホとターミナルを連携

設定完了後、スマホのTelegramからClaude Codeセッションにメッセージを送信可能。外出先から自宅のPCで動作するClaude Codeを操作。

⑫Dispatch機能: デスクトップアプリ

Claude CodeのデスクトップアプリにはDispatch機能が搭載。スリープ防止やブラウザーアクションの設定が可能。

⑬Dispatch: トランシーバー機能

Dispatch機能でスマホとデスクトップアプリを接続。公式は「トランシーバーのような機能」と表現。スマホからPCのClaudeを操作。

⑭ChannelsとDispatchの使い分け

Channelsはターミナルで起動したClaude Codeを操作。Dispatchはデスクトップアプリを起動しているPCを操作。用途に応じて使い分け。

⑮OpenClaw的な活用

自分でClaude Codeにスキルやハーネスを導入していれば、OpenClawのように汎用エージェントとして活用可能。Discord/Telegram連携でいつでもアクセス。

⑯チェンジログの激しさ

Claude Codeのチェンジログはめちゃくちゃ激しい。追いきれないほどのスピードでアップデートが続いている。

⑰プロンプト公開文化の価値

今回話題になった条件付きブロックなど、再現性の高いプロンプトが公開されると一気に広まる。特定用途に特化したプロンプトを作れる人の価値が上昇。

⑱コミュニティの活性化

Claude Code関連でハーネス勉強会なども開催され、コミュニティが形成されている。ノートやメンバーシップで深い情報も共有。
