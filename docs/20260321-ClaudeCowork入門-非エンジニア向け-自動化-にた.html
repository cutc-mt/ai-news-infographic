Claude Cowork 自動化 非エンジニア にた

①Coworkとは

Claude Codeをビジネス職が使いやすく設計したアプリ。データ分析、Excel処理、パワーポイント作成などを自動化できる。

②注目の背景

2026年1月にリリース後、同様の機能を販売する企業の株価が暴落して話題に。3月にはMicrosoft Copilotに統合されさらに注目。

③利用料金

Claudeのプロプラン（月額20ドル）から利用可能。クロードデスクトップアプリをインストールして使用。

④ChatGPTとの違い: ファイルアクセス

ChatGPTは毎回ファイルをアップロードする必要があるが、Coworkはパソコンのフォルダに直接アクセス可能。100個のファイルも一瞬で参照できる。

⑤作業の保存が簡単

途中経過を保存したり、アウトプットをダウンロードする手間がない。フォルダ内で直接読み書きが可能。

⑥基本的な使い方

作業フォルダを指定し、@でファイル名を入力して参照。プロンプトを入力するとAIがファイルを作成・編集してくれる。

⑦実践例: 複数ファイルから提案書作成

見積もり書、アンケート結果、業務時間分析、他社事例など複数ファイルを参照して提案書を自動作成。5枚のスライドが数分で完成。

⑧要件を整理してから作業

いきなり作らせるのではなく、AIに質問させて要件を整理してから作業すると品質が向上。「構成案を作って確認してください」と指示。

⑨コネクターで外部サービス連携

Googleドライブ、Googleカレンダー、Notionなどと連携可能。カレンダーの予定を確認したり、ノートの内容を参照できる。

⑩スケジュール機能

定期的にタスクを実行可能。毎日9時にメールチェックして下書きを作成、など自動化。パソコンがスリープでない時のみ実行。

⑪実践例: 会議議事録の自動作成

Googleドライブに文字起こしファイルを置いておくと、Coworkが定期的に確認して議事録を自動作成。アクションアイテムも抽出。

⑫スキル機能

特定の順番でタスクを実行するワークフローを設計可能。自分で作成したスキルをアップロードして使用できる。

⑬Dispatch機能

Maxプラン加入者はスマホからPCのCoworkを操作可能。外出先からパソコンのファイルを確認・編集できる。

⑭Claude Codeとの違い

できることは基本的にClaude Codeでも可能。違いはUIの使いやすさ。VS Codeの「いかつい画面」が苦手ならCoworkがおすすめ。

⑮VS Codeのメリット

VS Codeならファイルを見ながら作業できる。PDFやExcelの拡張機能を入れると、参照しながらの作業がより直感的。

⑯プランモード

Claude Codeにはプランモードがある。いきなり作成せず、まず要件を整理するステップを入れられる。簡単なこと以外はファイルを書き込まない。

⑰並列実行

1つの画面で複数のClaude Codeを並列実行可能。画面を分割して3つ同時に動かし、効率よくタスクを処理。

⑱注意点: 入力上限

ChatとCoworkがあり、Coworkは裏で設定が多いため入力上限に達しやすい。単純な質問はChatを使用するのがおすすめ。
