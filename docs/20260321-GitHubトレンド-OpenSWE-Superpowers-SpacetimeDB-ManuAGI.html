OpenSWE・Superpowers・SpacetimeDBなど20選

Open Source AI Agents Developer Tools ManuAGI

①Open-SWE: 自動コーディングエージェント

Langchainが開発したバグ修正や機能実装を自動化するオープンソースAIエージェント。コードを読んで計画し、パッチを適用してテストで検証する。

②Superpowers: ローカルAIツールランナー

ローカルファーストのAIツール実行環境。ファイル、API、システムツールと連携して、言語モデルが構造化コマンドでタスクを実行。

③OpenDataLoader PDF: PDF構造化抽出

PDFドキュメントからデータを抽出して構造化するパイプライン。テキストとレイアウトをデータパイプラインやAIシステムで使える形式に変換。

④Arnis: 実世界データでマインクラフト生成

実世界の地理データからマインクラフトのワールドを生成。地形、道路、建物を変換してゲーム内で再現。

⑤Newton: リアルタイム物理エンジン

剛体ダイナミクスのリアルタイム物理シミュレーションエンジン。3D環境での衝突、力、運動を計算。ゲームやシミュレーションに統合可能。

⑥TradingAgents: 金融取引マルチエージェント

複数のAIエージェントで金融取引をシミュレーション。市場データを分析し、戦略を生成して、意思決定プロセスを模倣。

⑦OpenRocket: ロケット設計シミュレーション

モデルロケットの設計と飛行シミュレーションができるソフトウェア。軌道、安定性、性能を物理モデルで計算。

⑧MiroFish: Miroボード管理ツール

Miroボードのコンテンツをプログラムで管理・処理するツール。Miro APIに接続して、要素の読み取りや変換を自動化。

⑨OpenViking: AIエージェントアプリフレームワーク

構造化ワークフローでAIエージェントアプリを構築・デプロイするフレームワーク。エージェント、ツール、実行パイプラインを管理。

⑩The Agency: マルチエージェントオーケストレーション

複数のAIエージェントが協力してタスクを実行するフレームワーク。エージェントの定義、役割割り当て、通信管理を提供。

⑪Deep Agents: 深層推論AIエージェント

Langchainのマルチステップ推論とタスク実行ができるエージェントフレームワーク。メモリ、ツール使用、反復的推論ループを構造化。

⑫Dimos: AIネイティブOS

AIネイティブアプリとワークフローをサポートするオペレーティングシステム概念。AIエージェント、タスク、リソースを統一環境で管理。

⑬Impeccable: ウェブパフォーマンス検証

ウェブサイトのパフォーマンス問題とベストプラクティスをチェックするツール。速度、アクセシビリティ、最適化の問題をレポート。

⑭Context Hub: 共有コンテキストシステム

AIアプリケーション間でコンテキストを共有・管理するシステム。複数のアプリやエージェントが一貫した情報にアクセス可能。

⑮RuView: コード変更レビューツール

コードの変更をレビュー・可視化するツール。コード差分を処理して、修正箇所を明確にハイライト表示。

⑯World Monitor: グローバルデータ監視

リアルタイムでグローバルデータを可視化する監視ダッシュボード。外部ソースからデータを収集して更新表示。

⑰Claude Code Best Practice

Claude Codeを効果的に使うための推奨プラクティス集。プロンプトとワークフローの構造化パターンとガイドラインを提供。

⑱SpacetimeDB: リアルタイムマルチプレイヤーDB

リアルタイムマルチプレイヤーアプリのためのデータベース。データストレージとサーバーロジックを統合し、自動で状態を同期。
