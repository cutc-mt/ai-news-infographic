⚔️ マルチエージェントシステム構築

Google ADKとMCPで学ぶ実践ハンズオン

Google ADK MCP Server Hands On AI Google Cloud Tech

①RPGゲーム形式で学ぶ

このハンズオンラボはRPGゲームをテーマにした2部構成のシリーズ。パート1では基礎を学び、最後にはボス戦も用意されている。

②MCPサーバーの構築

Model Context Protocol（MCP）サーバーを作成して、AIエージェントにツールを公開する方法を学ぶ。外部ツールとAIを繋ぐ共通言語。

③Google ADKでエージェント作成

Google Agent Development Kit（ADK）を使用してワークフローエージェントを作成。Google Cloud上でエージェントを構築する。

④マルチエージェントのオーケストレーション

複数のAIエージェントが協力して動く仕組みを構築。エージェント同士の連携と通信プロトコルを学ぶ。

⑤環境設定の準備

ラボに参加するにはノートPCまたはデスクトップが必要。スマートフォンは非対応。個人用Gmailアカウントの使用を推奨。

⑥無料クレジットの取得

ラボ用の無料GCPクレジットが提供される。リンクからアクセスしてクレジットをプロジェクトにリンクさせる手順を解説。

⑦Google Cloudプロジェクトの作成

新しいGoogle Cloudプロジェクトを作成し、必要なAPIを有効化。Cloud Shellを使って環境をセットアップする。

⑧Cloud Shellでの作業

ブラウザ上で動作するCloud Shellを使用。ローカル環境を汚さずに、クラウド上で開発環境を構築できる。

⑨エージェントアーキテクチャの設計

マルチエージェントシステムのアーキテクチャ設計の基礎概念を学ぶ。エージェント間の通信と役割分担を理解する。

⑩Pythonでエージェントを実装

Pythonコードを使って実際にエージェントを実装。ADKのサンプルコードをベースにカスタマイズする。

⑪ツールの定義と公開

エージェントが使用できるツールを定義して公開する方法。MCPサーバー経由でツールを登録する。

⑫ワークフローの構築

シーケンシャルなワークフローを構築。タスクを順番に実行するエージェントのチェーンを作成する。

⑬エージェント間通信プロトコル

エージェント同士が通信するプロトコルを学ぶ。メッセージの渡し方と状態の共有方法を理解する。

⑭デプロイと実行

構築したマルチエージェントシステムをGoogle Cloudにデプロイして実行。実際の動作を確認する。

⑮Codelabとソースコード

公式のCodelabとソースコードが提供されている。動画を見ながら手順通りに進められる教材付き。

⑯2部構成シリーズ

このラボはパート1。翌日のパート2でさらに高度な内容を学び、最終的にボス戦に挑む構成になっている。

⑰学習リソース

Google Cloud MCP servers ドキュメントやADKサンプルなど、追加の学習リソースも豊富に用意されている。

⑱実践的なAIエージェント開発

座学だけでなく実際に手を動かして学ぶハンズオン形式。終了時には自分でマルチエージェントシステムを構築できるように。
