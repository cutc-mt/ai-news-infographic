【超AI時代の人間】ご縁が大切になる

2026年3月22日 ｜ PIVOT × 落合陽一

①AI時代の働く再定義

  • ロボットが直接生産、AIが知的生産を担当
  • 人間は人間同士の関係をどう楽しくするかに集中
  • 文化的価値を創造することが主要な活動に
  • AIエージェントは全世界に配布され生産設備が手に入る状態
新しいものを作ろうとする人が極めて少ないのが現状

②ケインズの予測と現実

  • ケインズ予測：労働時間が750時間減る
  • 現実：1500時間以上も減っていない
  • 経済成長は予測以上（17.5倍）に達成
  • 再分配はうまくいっていない
予測と現実の乖離は2倍以上

③150年間の異常な労働

  • 産業革命：物質生産を支えるために働く
  • ホワイトカラー：知的システムを回すために働く
  • 江戸時代の人には「Excelを埋めることが飯に繋がる」全く理解不能
  • この150年間は異常な労働の仕方をしていた
近代の労働スタイル自体が歴史的異常事態だった

④娯楽の道具が不足

  • 働くためのツールは溢れている
  • 受動的な娯楽（映画）は100年前と変わらない
  • フィジカルの世界には楽しみがいっぱい
  • デジタル世界には能動的な楽しみが不足
AIを使って面白いことを想像するための生産設備は安価に普及

⑤消費者になってしまった人間

  • Excelを埋めるような薄い仕事に満足
  • AIを使って消費するだけで攻撃性がない
  • エントリージョブが消滅している
  • ベーシックインカムの代わりにChatGPTが配られている状態に近い
「作らなかったのは君の選択だから」納得させる独裁的強さ

⑥人口減少はチャンス

  • 日本は少子化とAI自動化を同時に食らっている
  • 人口減少局面でAIが自動化してくれるのは良いことしかない
  • 人足りないから、不要なところに人がいるだけ
  • 職業構造のリセットが可能になる
人口爆爆時にAIが出てきたら内乱になっていた！

⑦軍事×ライフの並存

  • エージェントの出現で軍事の構造が変わる
  • 自爆ドローンが大量に安価で飛ぶ時代に
  • 一方でライフ（暇つぶし）をどう楽しむかが重要に
  • 軍事とライフを同時にやる時代へ
1920年代（第2次世界大戦前）の空気に酷似

⑧キャラクターを磨く

  • 個性を磨くことが最優先事項
  • AIはコピーを作れるが、オリジナリティは持たない
  • 人間唯一の価値は独自のキャラクター
  • 自分だけの色、スタイル、世界観を持つ
誰も真似できない自分だけの価値を創り出す

⑨大恋愛時代が来る

  • AI時代には人間関係が最も価値ある資産に
  • デジタルな繋がりはAIが代行できる
  • 物理的な共感、体温を感じる関係が貴重に
  • 恋愛はAIが絶対に代替できない領域
恋愛はご縁の延長！ピリッと辛いのがちょうどいい

⑩ご縁が何より重要に

  • 人脈はビジネスっぽい、ご縁はもっと緩やか
  • 双方が引かれ合う関係性が価値を持つ
  • 同じ時間を楽しく過ごすことを共有
  • 食事を共にする、一緒に笑うことが重要
人間はサピエンス（知性）ではなくコンビアル（共に楽しむ）へ

⑪良い縁を作るには

  • 世界と対話する回数を増やす
  • 面白いものを見聞きする
  • 移動距離を増やす
  • 不思議なことをやる
  • 自分のコピーと対話して自己理解を深める
新しい世界を見せてくれる人が縁の創造主

⑫サバイブからコバイブへ

  • サバイブ：生き残るための組織（20世紀型）
  • コバイブ：共に楽しむ組織（21世紀型）
  • 管理から環境設計へ
  • スタティック・セパレーションからダイナミック・シンクロナイゼーションへ
探索も進化もAIがやるから人間は楽しむことに集中

⑬デジタル発酵の世界

  • 素材：データモデル
  • 工場：クラウドプラットフォーム
  • 工房：人間の意図
  • 超立者：人間（場をどう立ち上げるか）
  • 上の4つはすべてAIが担当
人間は微生物の動きを制御できない→環境設計が重要

⑭レコードバリがリーダーに

  • 世の中をちゃんと認識できる人
  • 味が分かる、場の空気がわかる
  • 「なんか変だな」と思える感性
  • 村のお役所さんのような存在
  • センスと直感で組織の楽しさを左右
命令じゃなく環境設計と超立ができる人がリーダー

⑮さよならホモサピエンス

  • サピエンス＝賢い人
  • 人間が文明のドライバーだった時代は終わり
  • AIが科学を続け、発明し、文明をドライブする
  • 文明は加速しているが、人間は変わっていない
  • 1970年大阪万博のテーマ：進歩も調和もしてない
論文を誰が書いたか気にならない時代に

⑯ホモコンビアルへの移行

  • コンビアル＝共に楽しむ（ラテン語）
  • 人間同士が楽しく生きる
  • 喜びを共有する
  • 時間や縁を共有する
  • 「あれ面白かったよね」と話すことが重要
サピエンスの話は人間を離れる

⑰人間には意義も意味もない

  • サピエンスとしての意味はゼロになった
  • 意味を感じると考えるとサピエンスの話に
  • 意味を考えると楽しくない
  • 祭りをすることで意味が生まれる
  • 祭り自体には意味はない、後に意味ができる
縁が生まれた後に恋愛、子供、価値が生まれる

⑱人類の存在意義は楽しむこと

  • みんなで楽しんでわいわいする
  • 子供を産む行為はAIには絶対にできない
  • 新しい面白いことをする人が増える
  • 未知と未来を見せてくれる人が重要
  • 人類社会で作られてなさそうなものを遊んで作る
みんなで面白い縁を作りながら楽しく生きよう
