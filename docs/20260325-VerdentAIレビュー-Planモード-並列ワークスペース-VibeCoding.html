Planモード・並列ワークスペース・マルチモデルレビュー

Verdent AI Plan Mode Multi-Model Vibe Coding

①レビューしやすさにこだわり

リビューを確定させる前に要約を示してくれたり、diff差分を見やすくするなど、レビュー体験に配慮した設計。

②Planモードの特徴

実装前にMermaid図でアーキテクチャを表示。データフローやコンポーネント構成を視覚的に確認可能。

③Mermaid形式のデータフロー図

各コンポーネントのデータ構造やデータの流れが図示される。ローカルストレージへの永続化などが一目で分かる。

④並列ワークスペース機能

複数タスクを同時並行で進められる。Git Worktreeを意識せずに、競合なしで並行作業が可能。

⑤最新モデル対応

Claude Sonnet 4.6、Opus 4.6 1M、GPT-5.4など最新かつ実用的なモデルに対応。

⑥agents.md対応

agents.mdに対応。拒否するコマンドのルールを入力しやすい形式で設定可能。

⑦スキル機能

クロードコードやCodexよろしくスキルを作成可能。フロントエンドデザインスキルなどが最初から用意されている。

⑧マルチモデルレビュー

Gemini 2.5 Pro、Opus 4.6、GPT-5.2の3モデルで同時レビュー。観点の偏りを防ぎいいとこ取りができる。

⑨レビュー専用モデル設定

内容に応じて最適なモデルを自動選択してレビュー。レビュー時のルールも設定可能。

⑩曖昧な指示から詳細計画

「看板ダッシュボードを作って」程度の曖昧な指示から、実装ステップ、テストケース、完了条件まで自動生成。

⑪Buildボタンで実装開始

立てたプラン通りに実装を進める時はBuildボタンを押すだけ。思考レベルとモデルを選択可能。

⑫Next Action機能

次にAIに与えそうな指示を候補として表示。ワンボタンでレビューの呼び出しなどが可能。

⑬ワークスペース作成

ボタン1つでGit Worktreeベースの独立ワークスペースを作成。ドキュメント作成など並行して進められる。

⑭MCP対応

MCPサーバーにも対応。外部ツールとの連携も可能。

⑮ターミナル操作

npm run devで開発サーバー起動などターミナルコマンドを実行し、結果を確認してくれる。

⑯フロントエンドデザインスキル

既存のUIをデザインブラッシュアップするスキルも用意。配色やスタイルの改善を自動化。

⑰料金プラン

7日間無料トライアル。スタータープランで月19ドルから。ポイント制で使用量を管理。

⑱まとめ: 初心者に優しい設計

分かりやすさ、効率化、高品質を慣れていない人でも簡単に行えるツール。無料トライアルで試せる。
