Visualizing World Models with Project Genie

世界モデルの可視化と Google AI Studio でのバイブコーディング | Google (Vibe Coding with Gemini)

① Project Genie とは: 全く新しい「世界モデル」

ゲームでも動画でもない全く新しいタイプの AI モデル。テキストや画像でプロンプトを入力すると、インタラクティブな世界を生成する。

② Google AI Studio でバイブコーディング

Project Genie のデザインツールは、Google AI Studio を使って「バイブコーディング」で構築された。アイデアをそのままコードに変える革新的なアプローチ。

③ Crystal Ballifier プロトタイプ: 3D 屈折マテリアル

AI Studio で構築したカスタムデザインツール。3D の屈折マテリアルを物理スライダーでリアルタイムに操作できるプロトタイプ。

④ デザイナーによる 3D マテリアル発案

デザイナーの Joselyn、Kiati、Jesu、Enley が世界を表現するために屈折する 3D マテリアルというコンセプトを発案し、コードで実装を検証。

⑤ フィジックスライダーで「中間の空間」を探索

各エフェクトにスライダーを追加して遊ぶことで、通常はプロンプトでは表現できない中間の空間を発見できる。ムードボードにも載っていない領域を開拓。

⑥ 「スーピーな効果」の発見: ラフネスのスイートスポット

スライダーを調整した結果、ラフネス（粗さ）がフェードアウトするスイートスポットに到達。独特のスープのような液体感のあるエフェクトを実現。

⑦ 球体からレンズへの進化

当初は球体だったが、特定の光学設定を実現するためにレンズの形状へと進化。微細な光学調整が可能な仕上がりになった。

⑧ JSON で設定を保存・共有

プロンプトでJSON ファイルへの保存と読み込み機能を追加。カスタム設定をそのまま他の開発者に送信できるようになった。

⑨ デザイナーとエンジニアの翻訳レイヤー

このアプリがデザイナーとフロントエンドエンジニア間の翻訳層として機能。開発者はこのプロトタイプを実際のウェブサイト構築の出発点として活用。

⑩ 正確な数値によるコミュニケーション

デザイナーは「IUR 値は 3.5 にして」といった正確な数値で指示できるように。「もっと太くして」のような曖昧な表現が不要になり、よりインクルーシブなプロセスに。

⑪ 自発的コラボレーションの誕生

コーダーの Kaloyan と Arden の間で自発的な共同作業が発生。アプリをコピーしてお互いの機能を追加し合う、全く新しいコラボレーションスタイル。

⑫ アイデアのバージョンコントロール

ツールをバージョン管理するのではなく、アイデアそのものをバージョンコントロールするという新しいアプローチ。お互いのアイデアをbounceしあう創造的なプロセス。

⑬ 1 プロンプトでカラーコレクション追加

Kaloyan が「カラーコレクションを追加して」と提案し、たった 1 つのプロンプトで機能を追加。バイブコーディングのスピードと柔軟性を実証。

⑭ キーフレームアニメーション: タイムラインで制作

キーフレームを配置したアニメーションタイムラインが自動生成。驚くほど簡単にアニメーションを再生・編集できるようになった。

⑮ AI Studio で動画エディターを構築

Three.js コードを使って AI Studio 内で動画編集ツールをゼロから構築。既存のデザインソフトを再構築するのではなく、必要な機能だけを超特化して作れる。

⑯ マルチプレイヤー編集が AI Studio に登場予定

AI Studio にマルチウェイ GitHub 同期とリアルタイムのマルチプレイヤー共同編集機能が近日追加予定。同じ場所にいなくても一緒に開発が可能に。

⑰ 個人プロジェクト: ASI アート映画制作ツール

Kaloyan が個人で開発中。スクリプトを書くと自動で音楽とシーンを生成する ASI（Artificial Super Intelligence）アート映画制作ツール。

⑱ 個人プロジェクト: 生成型ローグライクゲーム

Arden が個人で開発中。選んだ世界に落ち込むジェネレーティブなローグライクゲーム。Genie の lo-fi 版として構想されたインスピレーションの結晶。
