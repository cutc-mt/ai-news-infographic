④Skillsの基本と使い方

SkillsはClaude Codeの中で「/スキル名」のコマンドで呼び出せる拡張機能です。各スキルには特定のタスクを実行するための指示が書かれており、Claude Codeがその指示に従って動作します。

  • ターミナルで「/スキル名」と入力して実行
  • マークダウンファイルに指示を記述
  • 複数のスキルを組み合わせて活用可能
  • パラメータを渡してカスタマイズもできる
# Claude Codeでスキルを実行 > /commit # 自動コミット生成 > /review-pr 123 # PRのレビュー > /deploy # デプロイ実行

⑤スキルのインストール方法

Skillsのインストールは非常にシンプルです。コマンド1つでインストールが完了し、すぐに使い始めることができます。GitHubのリポジトリから直接インストールすることも可能です。

# Skillsのインストール手順 1. Claude Codeを起動 2. スキルギャラリーをブラウズ 3. 好きなスキルを選択してインストール 4. 「/スキル名」で実行！
  • ギャラリーから目的のスキルを検索
  • 数秒でインストール完了
  • アップデートも自動で反映

⑥カスタムスキルの作成

既存のスキルだけでなく、自分専用のカスタムスキルを作成することもできます。マークダウン形式で指示を書くだけで、チームに合わせた独自のワークフローを構築できます。

  • マークダウン形式で記述するだけ
  • チームメンバーと共有可能
  • 複数のスキルをチェーンで連携
