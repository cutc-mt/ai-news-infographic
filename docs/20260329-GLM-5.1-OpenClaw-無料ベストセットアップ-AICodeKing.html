⑤OpenClawのインストール

  • ターミナルで以下のコマンドを実行: 
git clone https://github.com/
  openclaw/openclaw.git
cd openclaw
npm install
npm run build
  • Node.jsがインストール済みであれば数分で完了
インストールはシンプルで初心者でも安心！

⑥APIキーの取得と設定

  • Zhipu AIの公式サイトでアカウント作成
  • 無料枠でAPIキーを取得可能
  • OpenClawの設定ファイルにキーを記述: 
# .env ファイル
GLM_API_KEY=your_api_key
MODEL=glm-5.1
無料枠でも十分な利用量が確保できる！

⑦OpenClawの基本設定

  • 設定ファイル（config.json）で詳細なカスタマイズが可能
  • 温度パラメーターの調整で創造性をコントロール
  • トークン制限やコンテキスト長の設定
  • テーマや出力フォーマットの変更も自由自在
好みに合わせて微調整できる柔軟性！

⑧GLM-5.1との統合手順

  • OpenClawをインストールしてビルド完了
  • Zhipu AIのAPIキーを取得
  • 環境変数にAPIキーを設定
  • プロバイダーをGLMに指定して起動 
openclaw --provider glm \
  --model glm-5.1
たった4ステップで最強環境が完成！
