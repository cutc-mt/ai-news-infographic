①Mission Control: Antigravityのコア機能

  • AntigravityのIDE内にはMission Controlというミッション管理システムが存在
  • バックグラウンドのタスク委譲を処理し、カスタムサブエージェントを構築できる
  • 専門スキルを持つエージェントをプログラム可能なAIモジュールとして扱える
Mission ControlがAIチームの指揮センターになる！

②サブエージェントの自動タスク分割

  • 例: 「新しいデザイナーのオンボーディング」というタスクを自動で分割
  • 1つのエージェントが会社のWebサイトをクロールしてフォントやロゴを抽出
  • 別のエージェントが社内ウォークスルーの資料をまとめる
  • 3番目のエージェントがオンボーディングメールをGmailで作成・送信
1つの指示で3つのエージェントが並行で作業！

③自動化ワークフローの威力

  • コピー＆ペーストなし、タブの切り替えなし、手動の引き継ぎなし
  • ワークフロー全体が完全に自動化される
  • サブエージェントがチームのように連携してタスクを完遂
人間が介在するステップがゼロに！

④Antigravityの限界と課題

  • Antigravityのエージェントシステムは「計画」の段階で止まってしまう
  • GitHub、Slack、Docs、Email、Webなどで実際のアクションを取る方法が不十分
  • 既存のツール連携は脆弱だったり、ハック的だったり、安全でなかったりする
計画は完璧でも実行できなければ意味がない...

⑤Arcade.dev: AI実行レイヤー

  • Arcade.devは無料のMCPランタイムで、Antigravityのエージェントに実行能力を付与
  • エージェントは計画するだけでなく、実際にツールにログインしてアクションを実行できる
  • 安全にアプリを横断してタスクを実行し、ワークフローを完了できる
「計画するAI」を「実行するAI」に変える魔法のレイヤー！

⑥Arcade.devのツールカタログ

  • 7,500以上のツールに接続可能（メール、ファイル、カレンダー、APIなど）
  • GmailやSlackなどのツールが簡単にセットアップでき、異なる機能を選択可能
  • ツールごとに個別にAPI呼び出しを選択することも可能
7,500ものツールが使えるなんて圧倒的！

⑦MCP Gateway: AIと外部システムの橋渡し

  • MCP GatewayはAIエージェントと外部システム間のブリッジとして機能
  • すべてのツールを標準化してセキュアに接続
  • シングルエンドポイントでエージェントが必要なツールをすべてバンドル
1つのGatewayで全ツールに統合アクセス！

⑧セキュリティと認証管理

  • OAuth認証が自動的に処理される（コード内にAPIキー不要）
  • スクレイピングなしで安全にデータにアクセス
  • ユーザー認証、監査ログで全操作を追跡可能
セキュリティも安心！APIキーの露出リスクゼロ。

⑨Server機能: ツールのホスティング

  • 制御された環境でツールをホストしてエージェントが利用可能
  • 各ツールやMCPごとにシークレット（秘密鍵）を管理
  • OAuthプロバイダーとの連携も管理可能
秘密鍵の管理も自動でやってくれる！

⑩Arcade.devの料金プラン

  • Hobbyプラン: 無料で十分な利用量を提供
  • $25/月プラン: より多くのMCPのAPI呼び出しと使用量上限
  • コストパフォーマンスを考えると非常に手頃な価格設定
無料プランから始められるのが嬉しい！

⑪セットアップ手順: ステップ1

  • Antigravityをコンピュータにインストール（OS問わず対応）
  • Arcade.devでアカウント作成（Hobbyプランは無料）
  • メインダッシュボードからエージェント、API、MCP、ワークフローを管理
まずはインストールとアカウント作成から！

⑫セットアップ手順: ステップ2

  • ArcadeでMCP Gatewayを作成（名前を付けてツールを選択）
  • Gmail、Slack、Google Calendar、Google Docsなどのツールを選択
  • 48のツールが一括で選択可能、個別選択も対応
必要なツールをチェックするだけ！

⑬セットアップ手順: ステップ3

  • MCP Gatewayのスニペットをコピーして設定に貼り付け
  • Antigravityの「追加設定」→「MCPサーバー」→「MCPサーバーの管理」を開く
  • Raw Configにスニペットを貼り付けてMission Controlに接続
コピペだけで接続完了！超簡単セットアップ。

⑭対応IDEとツール

  • Cloud Desktop、VS Code、Windsurf、Gemini CLIに対応
  • どの開発環境でも同じMCPスニペットで統合可能
  • エージェントはMCP Gatewayを通じてすべてのツールにアクセス
好きなIDEで使えるから移行コストなし！
