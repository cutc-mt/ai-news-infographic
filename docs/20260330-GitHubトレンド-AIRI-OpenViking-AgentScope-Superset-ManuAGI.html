Project AIRI・OpenViking・AgentScope・Superset・Nova

Open Source AI Agents Developer Tools ManuAGI

①Project AIRI: インタラクティブAIアシスタント

言語モデルとツール、メモリを組み合わせたインタラクティブAIアシスタント構築フレームワーク。会話管理、エージェント動作、外部サービス連携を提供。

②Lightpanda Browser: ヘッドレスブラウザ

自動化ワークフローに最適化されたオープンソースヘッドレスブラウザ。JavaScriptを実行し、Chrome DevToolsプロトコルで制御可能。

③OpenViking: 構造化AIエージェントフレームワーク

構造化ワークフローでAIエージェントアプリを構築。エージェント、ツール、実行パイプラインの抽象化を提供。ローカルまたはサーバーで動作。

④Claude Skills: Claude Code用スキル集

Claude Codeワークフローを拡張する再利用可能なスキル定義。異なる環境で一貫したタスク実行を標準化。

⑤Scrapling: Webスクレイピングツールキット

Webサイトから構造化データを効率的に抽出。HTMLを解析し、データパイプラインやAIシステムで使える形式に変換。

⑥AgentScope: AIエージェント構築・評価

AIエージェントを構築・実行・評価するフレームワーク。エージェント定義、対話管理、ツール連携のコンポーネントを提供。

⑦Superset: データ可視化プラットフォーム

Apache財団のデータ探索・可視化プラットフォーム。データベースに接続し、ダッシュボードとチャートを作成可能。

⑧MiniMind: 最小Transformer実装

教育・実験目的の最小Transformer言語モデル。トークン化、アテンション、訓練の仕組みをコンパクトに実装。

⑨CMUX: 接続マルチプレクサ

複数のネットワークサービスを1つのTCPポートで共有。プロトコル検出で接続を適切なサービスに転送。

⑩FlipOff: 画像変換ツール

画像に反転や向きの変更などの変換を適用。画像操作や前処理が必要なワークフローで活用。

⑪Nova: YouTube自動化AIエージェント

YouTubeコンテンツワークフローを自動化するAIエージェント。コンテンツアイデア生成やプラットフォームデータとの対話を処理。

⑫OMes: Temporalベースのワークフロー

Temporal上で長時間実行ワークフローを管理するオーケストレーションシステム。分散プロセスを調整し、障害時もタスクを完遂。

⑬HandyMKV: MKV変換ツール

MKVビデオファイルを他形式に変換・管理。ビデオ入力を読み込み、ストリームを処理して出力。

⑭TREK: ナレッジグラフ探索ツール

ナレッジグラフを探索・クエリするツール。グラフデータを処理し、ノードや関係性をナビゲートするインターフェースを提供。

⑮Temporal Features: Temporalパターン集

Temporalワークフローエンジンの機能使用例とパターンを収集。ワークフロー、アクティビティ、実行ロジックの実践例。

⑯Vit: Vision Transformer最小実装

Vision Transformerモデルを簡略化して実装。画像データをパッチに分割しTransformer層を通す仕組みを学習できる。

⑰is-antibot: ボット保護検出ツール

Webサイトがボット保護メカニズムを使用しているか検出。CAPTCHAシステムなどの保護を特定。スクレイピング戦略の調整に活用。

⑱Obsidian Excalidraw: 図描画プラグイン

Obsidianに図やスケッチを描けるキャンバスを統合。視覚的なノートをMarkdownと組み合わせて作成可能。

⑲Junior: 開発者デバッグアシスタント

Sentryが開発したデバッグ・モニタリング支援ツール。アプリケーションデータに接続し、エラーやシステム動作の洞察を提供。

⑳CheevoPresence: RetroAchievements連携

RetroAchievementsのアクティビティをDiscordと統合。ゲームプレイ進捗をDiscordステータスに表示。
