⑥Markdown × Claude Code: シンプル最強

Markdownは見出しや箇条書きをシンプルな記号で書ける形式。メモ帳で開ける普通のテキストなので、特別なアプリが不要でAIとの相性が抜群。

  • 特別なアプリ不要
  • AIが読み書きしやすい構造

システム設計

⑦5層構造: 年間目標から今日のTODOまで

タスク管理を5つの階層で構成。年間ビジョンから日次タスクまで、上から下に流れるように繋がる設計。

  • Yearly 年間のビジョン・目標
  • Quarterly 四半期のテーマ
  • Monthly 月次のタスク
  • Weekly 週次のテーマ・タスク
  • Daily 今日やるべきこと
システム設計

⑫朝: 「今日のタスク作って」だけ

朝起きたらAquaVoiceで声をかけるだけ。AIが週次・前日のタスクを読み込んで今日やるべきことを自動で切り出し。カテゴリ別分類、平日・休日のスケジュール調整、繰り越しタスクの検出まで全自動。

  • AquaVoice 音声入力でキーボード不要
  • 前日・週次・月次の情報を自動読み込み
  • カテゴリ別に自動分類
  • 繰り越しタスクを自動検出
1日の流れ

⑬日中: 「進捗どう？」で進捗確認

「進捗どう？」と聞くだけで、Claude Memory（AIの記憶機能）が今日の作業記録を検索。TODOに書いてなかった作業まで「あ、実はこれもやってたよね」と自動で拾ってくれる。

  • Claude Memory で作業履歴を記憶
  • TODOと作業記録を自動突合
  • 書いてない作業も自動検出
「コンテンツの準備します」→「スピーカーノートとサンプルファイルを作りました」→ 進捗あり、と判定。

⑭夜: 「振り返りやって」で完結

「振り返りやって」と話すだけで、/eodスキルが発動。AIが今日やったこと・やるはずだったことを自動で照合し、まとめとアドバイスまで生成。信頼できるパートナーのような存在に。

  • 完了タスクと未完了タスクを自動整理

  • 翌日のTODO準備まで自動実行
AIからのツッコミで「あ、これやり忘れた！」と気づける。レビュー会でドキッとするけど、漏れ防止になる。

正直な評価

⑰想定外: 完璧じゃなくても回る

本業が忙しくて週次ファイルを作れない週もあったが、月次と日次があれば自動で補完してくれる。完璧にやらなくてもシステムが回るという発見。

ただし弱点もあり。Claude Memoryの検索精度は完璧ではなく漏れもある。スキル作成には初期投資が必要だが、1回作れば毎日使えるので投資としては十分あった。

大事なのは完璧な管理ではなく挫折しない管理。緩く長く続けるスタンスが鍵。
