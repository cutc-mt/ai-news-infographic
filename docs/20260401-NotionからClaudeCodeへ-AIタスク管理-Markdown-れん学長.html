📋 Notion→Claude CodeでAIタスク管理

音声で話すだけでタスク管理が完結する3ヶ月の記録

動画: れん学長のAIツール実験室 | 2026-04-01

01/18
🎯 タスク管理は続けるのが難しい
Notion、Evernote、GTDなど色々試したがどれも続かない
最初は綺麗に回るが、タスクが溜まると持ち越しが増えてわちゃわちゃして結局やらなくなる
  • 多くの人がタスク管理ツールの運用継続に悩んでいる
  • 「うまく回っているのか？」という視聴者からのコメントがきっかけ
02/18
❌ Notionをやめた3つの理由①
理由1: タスク管理自体がタスクになってしまう
操作を覚え、機能を使いこなし、運用に時間がかかるのは本末転倒
  • ツールの操作方法の学習コストが高い
  • 管理作業自体に時間を取られて本質的な作業がおろそかに
03/18
❌ Notionをやめた3つの理由②③
理由2: データが人質になる — 値上げやサービス終了でデータが消えるリスク
理由3: 作業環境が分断される — タスク確認のために別ツールを開くのが面倒
10年後も確実に読める自己所有のデータが必要。全部1つの環境で完結したい
04/18
💡 発想の転換: ツールじゃなくて面倒な部分をAIに
ツールを変えるのではなく、タスク管理の面倒な部分を全部AIに任せるという発想
タスクはただのテキストファイルで管理し、Claude Codeが生成・レビュー・同期を全部担当
  • マークダウン形式のテキストファイルで管理
  • 特別なアプリが不要でメモ帳で開ける
  • AIとの相性がすごく良い
05/18
🏗️ 5層構造のタスク管理システム
年間ビジョン → 四半期テーマ → 月次タスク → 週次テーマ → 日次タスク
上から下に流れる5層構造で「何のためにやっているか」を見失わない
  • Yearly → Quarterly → Monthly → Weekly → Daily
  • たった5つのフォルダー構成でシンプルに管理
Yearly Quarterly Monthly Weekly Daily
06/18
📂 フォルダー構成とリアルデータ
Daily / Weekly / Monthly / Quarterly / Yearlyの5つのフォルダーだけ
3ヶ月で59日分のデイリーファイルが溜まった
  • 本業が忙しくて飛ばした日もあるリアルなデータ
  • マークダウン形式なので見出しや箇条書きをシンプルな記号で記述
  • テキストファイルなのでいつでも自由に編集可能
07/18
🔒 崩れない3つのルール
  • ① 上から下に流す: 年間目標が日次タスクまで繋がっていて「何のため」を見失わない
  • ② わちゃわちゃはAIに任せる: 持ち越し判断、週またぎの棚卸し、レベル間同期をAIが担当
  • ③ データはツールに依存しない: テキストファイルなのでいつでも別仕組みに乗り換え可能
自分でやると漏れがちな管理作業をAIに代行させるのが最大のポイント
08/18
🌅 朝のルーティン: タスク作成
「今日のタスク作って」と声をかけるだけ
Aqua Voiceで音声入力 → キーボードすら触らない
  • AIが週次・月次・前日のタスクを読み込んで今日やるべきことを自動で切り出し
  • カテゴリー別に分類して平日と休日のスケジュール枠も自動調整
  • 前日からの繰り越しタスクも自動で検出
音声入力 Aqua Voice
09/18
☀️ 日中: 「進捗どう？」でAIと突合
日中に「進捗どう？」と聞くだけでAIが作業記録と照らし合わせ
Claude Memの記憶とTODOを照合して、書いてなかった作業まで拾ってくれる
  • AIが今日のやり取り内容を記憶しておいてタスクと照合
  • 「実はこれもやってたよね」と未記録の作業も検出
  • 進捗のある項目を具体的に報告してくれる
10/18
🌙 夜: 振り返りをAIが自動で生成
「振り返りやって」と話すだけでAIが1日の振り返りを自動作成
AIが「これやるって言ってたけどやってないですよ」とツッコみを入れてくれる
  • 完了タスクと未完了タスクを自動で照合
  • 振り返りのまとめとアドバイスまで自動生成
  • 漏れに気づける「信頼できるパートナー」のような存在に
11/18
📅 週末: 週次と月次の同期も自動
金曜は週次の締めを自動実行
翌週に持ち越すべきタスクの確認、週の振り返り、翌週のTODO作成をAIが担当
  • 週次→月次のファイル同期も自動で処理
  • レベル間の整合性チェックをAIが担当
  • 自分でやると漏れがちな棚卸し作業を確実に実行
12/18
⚙️ Claude Codeの3つのスキル構成
AIにやる手順を教え込ませた3つのスキルを活用
スキル = AIにやらせる手順書のようなもの
  • /todo: タスク生成スキル — 今日のTODOを自動作成
  • /todo2: 進捗確認スキル — Claude Memの記憶と照合
  • /EOD: End of Day — 1日の終わりに振り返り・まとめ・翌日準備
/todo /todo2 /EOD
13/18
🧠 Claude MemでAIが記憶を管理
Claude MemでAIの中身を記憶・記録するツールを活用
1日の作業記録を全て読み込んで進捗確認や振り返りに活用
  • 日中の作業内容をClaude Memが自動で記録
  • 振り返り時に1日分全部を読み込むので時間はかかるが精度が高い
  • タスクと実際の作業のギャップを正確に検出
14/18
🎤 Anti-Gravity + Aqua Voiceの環境
Anti-Gravity（AIコーディングエディター）内でClaude Codeがターミナルで動作
Aqua Voiceの音声入力でキーボードを一切触らずに操作
  • Anti-Gravity環境で全て完結するからツールの切り替えが不要
  • 音声だけで「タスク作って」「進捗どう？」「振り返りやって」が完結
  • プログラミングの知識も不要で誰でも使える
Anti-Gravity Aqua Voice
15/18
✅ 3ヶ月使った正直な成果
タスク管理が面倒な作業からAIとの対話に変わった
声をかけるだけなので「めんどくさい」という感覚がほとんどない
  • これまでタスク管理はポチポチ操作が必要で苦痛だった
  • AIとの会話で管理が完結するので継続しやすい
  • 59日分のリアルなデータが蓄積された実績
16/18
🔍 想定外の発見と弱点
想定外の良さ: AIが信頼できるパートナーのような存在に
弱点: Claude Memの読み込みに時間がかかる / 本業が忙しいと飛ばしがち
  • AIからのツッコミで「しまった、やり忘れた」と漏れに気づける
  • 1日分の記憶を読み込む処理に少し時間がかかる
  • 忙しい期間はデイリーファイルの作成を飛ばしてしまう
17/18
👥 向いている人・向かない人
向いている人:
  • AIコーディングツール（Claude Code等）を日常的に使っている人
  • テキストベースのシンプルな管理を好む人
  • 音声入力に抵抗がない人
向かない人:
  • ガントチャート等のビジュアル管理が必要な人
  • チーム全体でのタスク共有が必要な人
