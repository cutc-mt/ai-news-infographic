🧠 AI Second Brain - 第二の脳

Claude Codeでスケジュール・プロジェクト・好みを記憶するAIを構築

Second Brain Claude Code Obsidian Cole Medin

①AI第二の脳とは

スケジュール、プロジェクト、好みを記憶し、ユーザーのニーズを先回りして予測するAIシステム。

②アーキテクチャ概要

Claude CodeとClaude Agent SDKをベースに、Markdownファイル、Pythonスクリプト、Obsidian vaultで構成。

③メモリ層の実装

メモリ層が全ての情報を管理。日々の活動、プロジェクトの進捗、個人の好みを体系的に記録。

④日次リフレクション

毎日の振り返りを自動的に記録。メモリ検索で過去の情報を瞬時に引き出せる。

⑤メモリ検索機能

数ヶ月分のコンテキストを検索。過去の会議、決定事項、アイデアを即座に発見。

⑥スキルとインテグレーション

様々な外部サービスと連携。メール、カレンダー、タスク管理を一元化。

⑦Proactive Agent Heartbeat

30分ごとにメール・カレンダー・タスクをチェック。ユーザーの代わりに情報を整理。

⑧メールの自動返信下書き

ユーザーの文体で返信の下書きを作成。自分らしい表現でメール対応を効率化。

⑨Obsidian Vaultとの統合

Obsidian vaultにメモリを保存。ノレッジベースとして直感的に管理・活用。

⑩Slackでモバイル連携

Slack経由でスマホからもコミュニケーション。外出先でもAIに指示や質問が可能。

⑪ライブPRD作成デモ

動画内でPRD（製品要求仕様書）をリアルタイムで作成するデモを実演。実践的な活用法を紹介。

⑫なぜ自分の第二の脳を作るべきか

SaaS製品ではなく自分専用の第二の脳だからこそ、完全にカスタマイズ可能でプライバシーも保護。

⑬テンプレートで簡単スタート

second-brain-starterリポジトリが公開されており、テンプレートからすぐに始められる。

⑭Pythonスクリプトで拡張

Pythonスクリプトで独自の処理や連携を追加。ニーズに合わせて自由に拡張可能。

⑮Dynamousコミュニティのワークショップ

4時間のハンズオンワークショップも開催。コミュニティで実践的に学べる。

⑯GitHubでオープンソース化

github.com/coleam00/second-brain-starterで公開。誰でも自由にフォークして使える。

⑰AIが予測するニーズ

記憶した情報から次に必要な行動をAIが予測。プロアクティブなサポートを実現。

⑱まとめ: 自分だけのAIパートナー

Claude Codeベースの第二の脳で、スケジュール管理から意思決定まで全てをAIと協働。個人史上最強の生産性ツール。
