⚡ Claude Code 必須コマンド25選

初心者が覚えるべき基本コマンドを完全ガイド

Claude Code コマンドガイド 初心者向け AI大学

①/help: ヘルプシステム

/helpで利用可能なコマンド一覧を表示。初めての操作で迷ったらまず実行するべきコマンド。

②/release-notes: リリースノート確認

/release-notesで最新のアップデート内容を確認。新機能や改善点をすぐに把握できる。

③/doctor: ステータスチェック

/doctorで現在の環境の健全性を診断。問題がある場合は解決策も提示される。

④/model: モデル切り替え

/modelでOpus、Sonnet、Haikuなどのモデルを切り替え。用途に応じた最適なモデルを選択。

⑤/init: CLAUDE.md自動生成

/initでプロジェクトのCLAUDE.mdファイルを自動生成。プロジェクト固有の指示を設定できる。

⑥/memory: メモリ確認

/memoryで保存されている記憶を確認。過去の会話や設定の記録を振り返れる。

⑦/config: 設定変更

/configでClaude Codeの各種設定を変更。自分の使い方に合わせてカスタマイズ。

⑧/permissions: 権限管理

/permissionsでファイルアクセスやコマンド実行の権限を管理。セキュリティを適切に設定。

⑨/desktop: デスクトップアプリ起動

/desktopでデスクトップアプリ版を起動。ターミナル以外でもClaude Codeが使える。

⑩/chrome: Chrome拡張実行

/chromeでブラウザ拡張機能を実行。Web上でもClaudeの機能を活用可能。

⑪/insight: 使用状況の洞察

/insightで自分の利用パターンを分析。トークン消費や頻繁に使う機能を可視化。

⑫プラグイン管理: hooks・MCP

hooks、agents、MCP、pluginsを管理。Claude Codeの機能を拡張する各種アドオンを設定。

⑬/clear: 会話リセット

/clearで会話をクリア。新しいタスクに取り組む際にコンテキストをリセット。

⑭/compact: 会話圧縮

/compactで長い会話を要約して圧縮。コンテキストウィンドウを節約できる。

⑮/copy: 会話コピー

/copyで会話内容をクリップボードにコピー。共有や記録に便利。

⑯/rewind: 過去の会話に戻る

/rewindで過去の会話状態に巻き戻し。間違えた方向に進んだ場合に便利。

⑰/resume: セッション再開

/resumeで前回のセッションを再開。作業を途中で終えても続きから始められる。

⑱/fork: 会話の分岐

/forkで会話を分岐。別のアプローチを試したい時に元の状態を保持したまま実験可能。
