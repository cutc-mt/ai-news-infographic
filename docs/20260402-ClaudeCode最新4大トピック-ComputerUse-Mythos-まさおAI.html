トピック1: Computer Use（CLIビルトインMCP）

①Computer UseがCLIに登場

  • Claude CodeのCLIにComputer UseがビルトインMCPとして搭載された。ターミナルから直接デスクトップアプリを操作できるようになった。
  • 設定は簡単で、最新のClaude Codeにアップデート後、/mcpコマンドを実行してBuilt-in MCPsのComputer Useをオンにするだけ。
  • 有効化の際にスクリーンの権限追加が求められる。案内に従って設定を進める。
CLIからデスクトップアプリまで操作できるのは革命的！エージェントの「目」が格段に向上した。

②利用できるツール一覧

  • スクリーンショットを撮って画像ベースで座標を認識し、クリック操作を実行。
  • 左クリック・ダブルクリック・右クリック・マウスムーブなど、マウス操作をフルサポート。
  • タイピング・キー入力・スクロール操作が可能。
  • Switch Displayでトリプルディスプレイ環境でもディスプレイを切り替えられる。
  • クリップボードの読み書きや、アプリのオープン・待機関数も利用可能。
コンピューター操作に必要なツールが全部揃っている！

③デモ: Crew Xデスクトップアプリを操作

  • 実際にCrew X（OpenChoreのデスクトップアプリ）をClaude Codeから操作するデモを実施。
  • Claude Codeのチャットから指示を送り、アプリを開いて画面を認識させ、チャットでコミュニケーションさせることに成功。
  • アプリの起動時には権限リクエストが毎回飛んでくる。オートにすると暴走のリスクがあるため、現状は制約が強め。
  • Switch Displayでディスプレイを認識し、LGウルトラワイドモニター上のアプリを正しく特定できた。

④Computer Useの活用シーン

  • 本命はAPIのないツールの操作。APIを提供していないアプリもComputer Useで制御できる。
  • ネイティブアプリやモバイルアプリの動作検証に活用できる可能性。
  • ブラウザ操作はPlaywrightなどのエージェントブラウザが既にあるが、デスクトップアプリの自動化はComputer Useが有利。
  • Claude Chrome（ブラウザ拡張）やClaude Inlineも便利だが、デスクトップ領域での幅広さではComputer Useが勝る。
1年前はComputer Useが先の話だと思っていたが、もう現実的になっている！

Computer Useの注意点とリスク

⑤アプリごとの権限に注意

  • アプリによっては権限NGが出る場合がある。全てのアプリで操作できるわけではない。
  • 自分のログイン済みアカウントで操作させる場合、セキュリティ上のリスクも考慮が必要。
  • 権限のオート承認は便利だが、暴走の危険性があるため慎重に扱うべき。
権限の取り扱いは自己責任で！特に本番環境では要注意。

⑥スクレイピングと利用規約のリスク

  • ウェブサイトからのデータ抽出は利用規約の範囲内で行う必要がある。
  • スクレイピングは基本的に推奨されない行為。利用規約で許可されている場合を除き、NGになることが多い。
  • バイブコーディングをしている人は、情報セキュリティのリスクを意識しておくことが重要。
自分なりの「ここまではやろう」というラインを決めておくのがおすすめ。

⑦ボット対策とモバイルへの影響

  • Computer Useの普及により、ボット対策の仕組みに影響が出る可能性。
  • モバイルアプリのシミュレーターを操作されるようになると、「モバイルだからボットがいない」という前提が崩れる。
  • モバイルでもボットが存在する世界が来るかもしれない。プラットフォーム側の対策が求められる。
モバイルのボット対策が通用しなくなる世界が近づいている！

トピック2: Claude Codeソースコードリーク

⑧Claude Codeのコードが流出

  • Claude Codeのソースコードがリークし、X（旧Twitter）界隈で大话题に。
  • プロンプトの中にシステムプロンプトの魂が詰まっており、コードだけでなくプロンプトの内容も明らかに。
  • X上でソースコードの解説を行う人が続出。GitHub上でクローンアプリを公開する人も現れた。

⑨パクリアプリとオープンソース化の動き

  • 流出したコードをもとにクローンアプリを作り、GitHubで「オープンソースだ」と公開する人も現れた。
  • コードの流出自体は悪いことだが、競争環境が生まれることには一定の意義がある。
  • GitHub上でClaude Codeに追い込むようなシェアを行う人もおり、界隈が大きく盛り上がった。
流出そのものは問題だが、競争圧力が生まれることには期待もある。

⑩競争が生む一般ユーザーへのメリット

  • ClaudeがAGIを達成して一強になった場合、価格が吊り上げられる懸念がある。
  • Cursor CLIがGLMやGeminiベースで手頃な価格の代替を提供すれば、下から圧がかかる。
  • OpenAI CodexやGemini CLIなどの存在も競争圧力として機能。一般ユーザーにとって価格競争は歓迎すべき。
  • 急に100倍の値段になったら困るので、競争による適正価格の維持が望ましい。
競争があるからこそ、一般ユーザーもClaudeを使い続けられる！

トピック3: Mythosモデルの噂

⑪Mythos: Opusの上位互換モデル

  • Mythos（神話）という名前の新モデルが来るという噂が浮上。「神話」を意味する名前から、上位モデルであることを示唆。
  • 現在のOpus 4.6でもAGIを作り込めばかなり強力だが、Mythosはそれを超える性能が期待されている。
  • 様々なリーク噂が飛び交っており、真偽は不明だが界隈の期待は大きい。
Opus 4.6でもAGIにタッチしそうな勢いなのに、Mythosはどうなるのか！

⑫$200プランと5時間制限の課題

  • $200プランでも5時間制限があり、しっかり使うとすぐに制限に達してしまう。
  • Mythosが来ると、より高性能な分タスクの幅が狭まる。質は高いが短いタスクしかこなせなくなる懸念。
  • 長時間のタスクをこなそうとすると、API利用になり10倍以上のコストがかかる可能性。
5時間制限の幅＝実質的にできるタスクの時間。ここが一番の課題！

⑬モデル使い分けとJoyプランの議論

  • Mythos登場後はモデルの使い分けがより重要に。OpusやGPT-5.4でも十分なケースが多い。
  • Joyプラン（$2,000プラン）で10倍の使用量が可能に。より長くタスクを走らせる体験が得られる。
  • $200プランで月$6,000分使えるという話もあり、API利用者との格差も話題に。
  • 制限の幅を意識しつつ、最適なモデル選択とプラン選びが鍵になる。

トピック4: Claude Code公式Tips（Borisが共有）

⑭モバイルアプリとテレポート機能

  • Claude Codeのモバイルアプリが利用可能。外出先からでもClaude Codeにアクセスできる。
  • テレポート機能で、デスクトップアプリ・モバイル・CLI間でシームレスにセッションを引き継げる。
  • デバイス間の連携体験が向上し、どこからでも開発を続けられる。
モバイルとデスクトップを行き来できるのは便利！

⑮ループ・スケジュール・UseFix機能

  • ループ機能でタスクを繰り返し実行可能。定期作業の自動化に役立つ。
  • スケジュール機能で、指定した時刻にタスクを自動実行。絶対に押さえておくべき機能の1つ。
  • UseFixで決定論的に問題解決を進めることができる。確実な修正が必要な場面で強力。
ループとスケジュールは絶対に押さえておくべき機能！

⑯Chrome拡張・デスクトップアプリ・フォークセッション

  • Claude Code for Chromeは、人間がClaude Codeと向き合いながら作業する際に便利なツール。
  • デスクトップアプリは今勢いがある。CLIと併用して使うのがおすすめ。
  • フォークセッション（/fork または /branch）で、今までの会話を枝分かれさせられる。絶対にやるべき機能。
フォークセッションは絶対使った方がいい！過去の会話から別の探索ができる。

⑰ワークツリーとバッチ機能

  • ワークツリーを活用すれば、長いタスクをやっている横で動作確認環境を並行して作れる。
  • バッチ機能で複数タスクを一括実行。効率的な開発フローが構築できる。
  • 高速化のTipsも共有されており、Claude Codeのパフォーマンスを最大限に引き出せる。
ワークツリーで並行開発は実務でめちゃくちゃ使える！

⑱Add Directory・ボイスモード・まとめ

  • Add Directoryで外部ディレクトリをClaude Codeのコンテキストに追加可能。
  • ボイスモードで音声による操作もサポート。手を使わずにClaude Codeと対話できる。
  • Claude CodeはComputer Useで「何でもできる」領域に突入。今後さらに活用の場が広がる。
  • 公式のTipsは概要欄のリンクから確認可能。知らなかった機能がきっとあるはず。
Computer Useで何でもできる時代が到来。Claude Codeの進化が止まらない！
