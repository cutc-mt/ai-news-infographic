My Cloud Search

メール・ドライブ・カレンダーを横断検索できる無料ツールを公開

Google Apps Script Gmail Googleドライブ カレンダー 横断検索 無料公開

①My Cloud Searchとは：Google全体を横断検索する無料ツール

こーすけ先生がGoogle Apps Script（GAS）で開発した横断検索ツールを無償で公開。Gmail、Googleドライブ、Googleカレンダーの3つのサービスをまとめて検索できる。個人アカウントでもGoogle Cloud Searchのような体験が可能になった。

個人アカウントでCloud Searchが使える！

②元々はGoogle Workspaceの法人向け機能

Google Cloud SearchはGoogle Workspaceの法人契約者向けに提供されている社内検索機能。ビジネススタンダード、ビジネスプラス、エンタープライズスタンダード、エンタープライズプラスのいずれかの契約が必要で、個人では利用できなかった。

  • 法人契約が必須だったCloud Search
  • Workspaceユーザーは毎日ヘビーに利用
  • 個人アカウントでは手が出せない機能だった

③なぜ横断検索が必要なのか

Google Workspaceにはメール、カレンダー、ドライブ（ドキュメント、スプレッドシート、スライド）など多数のサービスが含まれている。それぞれを個別に検索するよりも、横断的にまとめて検索できた方が圧倒的に便利。だからCloud Searchが生まれた。

バラバラに検索するよりまとめて検索が便利！

④検索の対象：ファイルの中身まで検索

My Cloud Searchは単なるファイル名検索ではない。Googleドライブ内のコンテンツ（ファイルの中身）を検索できるほか、Gmailのキーワード検索、Googleカレンダーのイベント名検索にも対応している。

  • Gmailのメール本文のキーワード検索
  • ドライブ内のコンテンツ検索（ファイル名ではない）
  • カレンダーのイベント名検索

⑤GASで実現した仕組み

Cloud Searchの機能は、Google Apps Script（GAS）でも再現できるという発想から開発された。Gmail、ドライブ、カレンダーのAPIをGASから呼び出して、キーワード検索を一つのインターフェースでまとめて実行する仕組みだ。

GASのAPIを活用してCloud Searchを再現！

⑥使い方：概要欄のURLをクリックするだけ

My Cloud Searchを使いたいGoogleアカウントでログインした状態で、概要欄にあるURLをクリックするだけでアクセスできる。特別なインストールや設定は不要。ブラウザですぐに使い始められる。

URLをクリックするだけでスタート！

⑦初回の権限付与手順

初回アクセス時には、GASがあなたのGoogleデータにアクセスするための権限付与が必要。これは開発者があなたの情報を取得するわけではなく、アプリケーションがあなたのデータにアクセスするためのGoogleの標準的な手続きだ。

  • Verifiedアプリとして表示される
  • 詳細からお問い合わせアカウントを確認可能
  • Gmail・ドライブ・カレンダーの3つの許可を付与

⑧権限付与の具体的なステップ

アクセスをクリックした後、「Review Permissions」を押して、「Googleで確認されていません」という警告の左下「詳細」を開く。My Cloud Searchのページへ移動し、Gmail・ドライブ・カレンダーへのアクセスを「全て許可」すれば完了だ。

全て許可をクリックすれば準備完了！

⑨検索結果の表示：件数と内訳が分かりやすい

キーワードを入力して検索すると、処理ダイアログに結果の件数が表示される。例えば「Google」で検索すると83件の結果が表示され、メールが50件、ドライブが3件、カレンダーは該当なしというように内訳が確認できる。

メール・ドライブ・カレンダーの内訳が一目で分かる！

⑩検索結果から直接アクセス可能

検索結果の各項目をクリックすると、実際のメールやドライブのファイルに直接ジャンプできる。例えばスライドの検索結果をクリックすれば、実際のGoogleスライドがブラウザで開いて内容を確認できる。シームレスな体験だ。

クリックすれば実際のファイルが開く！

⑪期間絞り込みで効率的に検索

検索範囲を過去1日、1週間、1ヶ月、1年のいずれかに絞り込むことができる。期間を短くすれば最新の情報だけを抽出でき、目的のデータをすばやく見つけ出すことが可能だ。

  • 過去1日で最新のメールだけを表示
  • 過去1週間で直近のイベントを確認
  • 過去1年で広範囲に検索

⑫カレンダー検索の実証テスト

カレンダーが本当に検索できているかを確認する実験も行われた。「前日テスト」「テスト」というイベントを作成して検索すると、両方のイベントが正しく検索結果に表示された。期間絞り込みでも期待通りの動作を確認している。

カレンダーも問題なく検索できた！

⑬検索速度についての正直な評価

本家のGoogle Cloud Searchと比べると、GASを使っているため検索速度は少し遅め。しかし、メール・ドライブ・カレンダーを横断的に検索できるという利便性は十分にあり、個人利用では実用的なレベルだと評価されている。

少し遅いけど十分に実用的！

⑭ソースコードをGitHubで完全公開

セキュリティが不安な方のために、GASのプロジェクトファイルをすべてGitHubで公開している。中身を確認して問題がないことを確かめてから使うこともできるし、自分でGASを構築して独自のCloud Searchを作ることも可能だ。

  • GitHubで全コード公開済み
  • コードの中身を自由に確認可能
  • 自身でカスタマイズして構築もできる

⑮セキュリティへの配慮：悪いコードは書いていない

こーすけ先生は「決して悪いコードは書いていない」と明言。公開URLをそのまま使っても良いし、GitHubのコードを見て自身で構築しても良い。どちらを選んでも構わないという透明性の高い姿勢で公開されている。

透明性100%で安心して使える！

⑯キーワード検索 vs 生成AI：検索の強み

GeminiやNotebookLMなどAIが進化しても、キーワード検索の方が正確性で勝る場面がある。AIはハルシネーション（幻覚）のリスクがあるため、確実な情報を探すときはドライブやメールの検索機能を使う方が精度が高い。

AIにはハルシネーションのリスクがあるから検索も大事！

⑰今後の展望：より多くのサービスに対応予定

現在はGmail、ドライブ、カレンダーの3サービスに対応しているが、Googleには他にも多数のサービスが存在する。今後はより多くのサービスを横断検索の対象に追加していく予定で、1つでも多くのサービスを提供していきたいと語っている。

  • Gmail - 対応済み
  • Googleドライブ - 対応済み
  • Googleカレンダー - 対応済み
  • その他のサービス - 今後追加予定

⑱DX支援の一環として無料で公開

こーすけ先生はGoogleサービスの全体がレベルアップしている中で、検索という基本機能の重要性を見直すべきと強調。生成AIに頼るだけでなく、確実な検索ツールも活用することがDX（デジタルトランスフォーメーション）の鍵だと語り、このツールを無料で公開している。

有料級のツールを無料で公開！ぜひ使ってみて！
