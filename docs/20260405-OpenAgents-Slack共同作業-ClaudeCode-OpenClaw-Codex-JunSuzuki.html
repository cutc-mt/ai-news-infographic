AIエージェントたちのSlackが遂に登場 - 共同作業の新時代

OpenAgents Claude Code OpenClaw Codex Jun Suzuki

①OpenAgentsとは: AIエージェントのためのSlack

OpenAgentsは、複数のAIエージェントが一つのワークスペースに集まって共同作業するためのオープンソースプラットフォーム。Slackのようなチャットインターフェースで、エージェント同士がメンションし合いながらタスクを進めることができる。人間ももちろん参加可能で、まさに「AIの会議室」のような存在。

②ブレイクスルー直前の注目リポジトリ

現在GitHubスターはわずか3000程度だが、海外トップインフルエンサーのJulian Goldie氏がいち早く情報発信しており、他の海外YouTuberたちも続々と取り上げ始めている。まだ日本国内では情報が少なく、まさに大ブレイク寸前のプロジェクトと言える。

③ブラウザーベースの共同環境

ワークスペースはブラウザ上で動作する共同作業環境を提供。AIと人間がスレッド、ファイル、ライブブラウザーを共有できる。エージェントがページを開き、クリックし、スクリーンショットを撮る過程を、全員がリアルタイムで視認可能なのが大きな特徴。

④@メンションでタスクをアサイン

Slackのように@クロードコードや@オープンクローとメンションすることで、特定のエージェントにタスクを直接アサインできる。エージェント同士が「さっき依頼された作業が終わったからファイルを置いておくね」といったやり取りを自律的に行うのが画期的。

⑤共有ファイルシステム

コード、ドキュメント、レポートなどのファイルをアップロードして、人間とエージェントの双方が直接編集できる。例えばClaude Codeでコードを書き、その結果をOpenClawに渡して別の処理をさせるといった連携が、コピペなしでシームレスに実現する。

⑥共有ブラウザーでリアルタイム確認

シェアードブラウザー機能により、エージェントがWebページを開いたりクリックしたりする様子を、ワークスペースの全メンバーがリアルタイムで確認できる。複数のエージェントが同じ画面を見ながら作業を進めることが可能で、まるで同じオフィスにいるかのような体験を提供する。

⑦ランチャー: ワンクリックでエージェントを起動

OpenClaw、Claude Code、Codex、Aiderなどの主要エージェントをワンクリックでインストール・起動できるランチャー機能が用意されている。Windows、Mac、Linuxの全プラットフォームに対応しており、クロスプラットフォームのバックグラウンドデーモンとして動作する。

⑧PCスリープ中もエージェントが継続稼働

ランチャーはバックグラウンドデーモンとして動作するため、PCがスリープ状態になってもエージェントは作業を継続する。長時間かかるビルドやテストの実行中でも、人間は別の作業に取り掛かることができる。

⑨Python SDKでカスタムエージェントを開発

開発者向けにPython SDKが提供されている。pip install open-agentsでインストール可能で、ネットワークに登録し、イベントに応答し、ワークスペースで共同作業するカスタムエージェントを構築できる。SDKが接続、ハートビート、ツールルーティングを自動的に処理してくれる。

⑩トンネル機能でローカルサーバーを即公開

コマンド1つでローカル開発サーバーをパブリックURLとして公開できるトンネル機能が搭載されている。エージェントが構築した成果物を、どのデバイスからでも即座にプレビュー可能になる。Claude CodeのDispatchに近い機能と言える。

⑪MODシステムで独自の連携パターンを定義

MODシステムにより、メッセージ、ファイル操作、ゲームなど、独自のコラボレーションパターンを定義できる。単なるチャットツールを超えて、エージェント同士の新しい連携の形を自由に設計できる拡張性を備えている。

⑫MCPとA2Aプロトコルを完全統合

MCP（Model Context Protocol）とA2A（Agent to Agent Protocol）を完全に統合している。これにより、異なるフレームワーク間の壁を超えて、LangChainやCrewAIのカスタムエージェントが同じワークスペースで協調できるオープンな通信レイヤーが実現する。

⑬サンドボックス化された安全な環境

OpenClawのような単一エージェントはローカル環境に直接アクセスするため高リスクなセキュリティ上の懸念があった。OpenAgentsではサンドボックス化された隔離環境で動作し、必要な時のみリソースを消費するため、予測可能で安全なコスト制御が可能。

⑭競合ではなく統合: OpenClawと共存

OpenAgentsはOpenClawやClaude Codeと競合するのではなく統合する設計思想。自分のOpenClawインスタンスをOpenAgentsのワークスペースに接続すれば、強力なソロプレイヤーがチーム内のオペレーションエキスパートとして機能し始める。

⑮対応エージェント一覧

現在統合可能なのはClaude Code、OpenClaw、Codex CLI、Casy Sol、OpenCode。今後Coming SoonとしてAider、Goose、Gemini CLI、Cline、GitHub Copilot、Ampなどが追加予定。主要なAIコーディングツールの大部分を網羅する計画が進んでいる。

⑯ホステッド版とローカル版の2つの導入方法

ホステッド版はOpenAgents.orgにアクセスするだけでインストール不要ですぐに開始可能。Googleアカウントでログインすればエージェントを接続できる。ローカル版はコマンドラインからワンライナーでインストール可能。セルフホストもApache 2.0ライセンスで自由にできる。

⑰単一エージェントからマルチエージェントへの進化

課題は「単一の強力なエージェントを作ること」ではなく、複数のエージェントをシームレスに連携させることにある。ツールは増えたが連携しておらず、コンテキストの共有は手作業に頼っていた。OpenAgentsはこの「孤立から共同への進化」を実現するプラットフォーム。

⑱まとめ: 協調するAIネットワークの時代へ

孤立したツールを管理する時代は終わり、協調するAIネットワークを指揮する時代が始まった。OpenAgentsは完全オープンソースでApache 2.0ライセンス、ベンダーロックインなし。エージェント同士が自律的にコミュニケーションし、人間も参加できる新しい働き方がここに来た。
