🔗 Claude Code × GPT-5.4 連携環境

Claude Codeの中からGPT-5.4に壁打ち・ファクトチェックできる環境を構築

①AIのセカンドオピニオンが欲しい：Claude Codeの課題

Claude Codeで作業していると、「これ本当に合っているのかな？」と疑問になることがある。Claudeに調査してもらって最もらしい回答をそのまま信じて進めたら、実は違っていたという経験は多くの人が直面する。特にClaude Opus 4.6を使っていると、話していることを鵜呑みにしたり、十分に調査しないまま回答を生成してしまうことが多々あるという。

②解決策は「別のAIに聞くこと」：GPT-5.4との壁打ち

人間の世界でもお医者さんのセカンドオピニオンがあるように、AIにも別の視点を求めるのが有効だ。れん学長が最も信頼しているGPT-5.4の思考に相談できる環境をClaude Codeの中に構築した。これにより、Claude Code内から自然言語で頼むだけで、バックグラウンドでGPT-5.4が呼び出されて回答が返ってくる仕組みが完成した。

③Codex MCPとは：OpenAI CLIをClaude Codeに接続する仕組み

今回の連携に使うのがCodex MCPだ。OpenAIが提供しているCodex CLI（キーボードで命令を打ち込むコーディングツール）を、Claude Codeの外部ツール接続の仕組みであるMCPで繋ぐことができる。作業中に自然言語で頼むだけで、バックグラウンドでGPT-5.4が呼び出され、回答が返ってくる。ChatGPTのPlus、Pro、Business、Enterprise、Educationプランのいずれかに加入していれば、追加料金なしで利用可能だ。

④注意点：公式サポートではなく有志が公開した仕組み

このCodex MCPによる連携は、AnthropicやOpenAIが公式にサポートしている機能ではない。有志が公開してくれた仕組みを利用したものであり、今後の仕様変更によって使えなくなる可能性もゼロではない。しかし、れん学長は毎日作業で使い続けるほど重宝しており、「めちゃくちゃ使える」と評価している。この点を踏まえた上で活用することが大切だ。

⑤公式プラグインとの違い：Codex Plugin vs MCP方式

OpenAIがリリースしたCodex Plugin for Claude Codeは、開発者向けのコードレビューに特化している。実際のソースコード開発を行うエンジニア向けのツールだ。一方、今回紹介するMCP方式は、自由なプロンプトでGPTに投げられるのが特徴で、企画の壁打ちから記事のファクトチェック、ビジネス判断の別視点チェックまで、何でも聞けるのが強みだ。コードレビューならプラグイン、壁打ちや汎用調査ならMCPという使い分けができる。

⑥セットアップに必要な3つの前提条件

環境構築に必要なものは以下の3つだ。まずClaude Codeの月額20ドル以上のMaxプランに加入していること。次にChatGPTの有料プラン（Plus、Pro、Business、Enterprise、Educationのいずれか）を契約していること。最後にNode.jsがインストールされていること。これら3つが揃っていれば、準備は10分程度で完了する。

⑦セットアップ手順：たった2つのコマンドで完了

セットアップは非常にシンプルだ。まずcodex loginコマンドを実行して、OpenAI側に1回ログインする。このログインは次回から不要だ。次にclaude mcp add codexのようなコマンドを打ってMCP接続を設定する。最後に確認コマンドを実行して「Connected」と表示されれば完了。ノートに詳しい手順が記載されているので、覚える必要はない。

⑧基本的な使い方：サーバー名を明示して頼むのがコツ

実際に使う際は、「Codex MCPを使って」と頼むだけで起動する。ポイントは「codex」や「MCP」といったサーバー名を明示することだ。例えば「Codex MCPを使って、この設計にGPTのセカンドオピニオンをもらって」や「Codex MCPでGPTにファクトチェックさせて」のように指示すると確実に起動する。逆に「別の意見が欲しい」のような曖昧な指示だと、Codex MCPがうまく起動しないことがある。

⑨推論の深さを4段階で指定できる：Low / Medium / High / X

GPT-5.4に相談する際、推論の深さ（reasoning effort）を4段階で指定できる。Low、Medium、High、Xの4つで、Xが最も高い推論レベルだ。デフォルトはMediumだが、れん学長は日常の壁打ちやファクトチェックではHighを常用している。Highだと回答が返ってくるまで2〜3分かかることはあるが、しっかり調査して考えてくれる。本当に深く考えたい時はXもあるが、10分以上かかることもあるので使いどころを見極めるのが良い。

⑩3者の役割分担：GPTの厳密さ × Claudeのバランス感覚 × 人間の判断

GPT-5.4は非常に厳密で裏付けのある意見を収集するのに優れているが、ネチネチと細かい重箱の隅を突くような聞き方をすることもある。計画を提案すると色々な案を練ってきて「全部やるの大変じゃない」と感じることも。そこで1回GPTに聞いた後、Claude Opusに持っていくと「それはやりすぎだからここまでにしておきましょう」とやりすぎを抑える調整力を発揮する。最終的には人間が判断する、この3者の役割分担が非常に良い組み合わせだという。

⑪全てがClaude Code上で完結する3つのメリット

MCP経由でChatGPTに相談する最大のメリットは、全てがClaude Code上で完結することだ。1つ目は「作業画面が分断されない」ことで、ブラウザのChatGPTを行き来する必要がない。2つ目は「ドキュメントがローカルに残る」ことで、Web版だとマークダウンに保存する一手間がかかるが、MCP経由なら自動保存される。3つ目は「シームレスな連携」で、GPTの回答に対して「Claude、GPTが言ってたやつどう思う？」と同じチャット内で即座に確認できる。

⑫MCP経由の注意点：テキストベースのみとAPIモードに注意

MCP経由の連携には3つの注意点がある。まずテキストベースのやり取りしかできないため、PDFや画像を見せて質問したい場合はWeb版のChatGPTを使う必要がある。2つ目は、推論の深さXはかなり時間がかかるため、日常使いではHigh程度で十分だということ。3つ目はAPIキーモードになっていないか確認することで、有料課金モードで使っていないと別途料金が発生してしまう可能性がある。通常は問題ないが、APIを使っている方は注意が必要だ。

⑬スキル化で自動化：毎回の長い指示を一言に短縮

MCP接続だけでGPTに相談できるが、実用上は毎回の指示が面倒だ。「Codex MCPを使って、GPT-5.4にHighのreasoningで内容をファクトチェックさせて、回答はマークダウンファイルで保存して、Claudeの見解も添えてください」と毎回入力するのは手間がかかる。書き忘れや品質のバラつきも発生してしまう。理想は「ファクトチェックして」と一言で終わらせること。それを実現するのがスキルの仕組みだ。

⑭GPT Brainstormスキル：4つのモードを搭載した壁打ちツール

れん学長が作成したGPT Brainstormスキルは、壁打ち、ファクトチェック、セカンドオピニオン、レビューの4つのモードを搭載している。スキル内部では、話しかけ方からのモード自動判定、推論の深さの固定、GPTの役割設定と観点指示を毎回確実にセットしてくれる。さらに回答をマークダウンファイルに自動保存してClaudeの見解も添えるため、毎回同じ高い品質でブレのない壁打ちが可能になる。

⑮スラッシュコマンドで確実に発動：/gpt-brainstorm

スキルを確実に発動させるために、/gpt-brainstormというスラッシュコマンドも用意されている。このコマンドに続けて壁打ちしたい内容を入力するだけで、スキルが自動的に起動する。例えば「/gpt-brainstorm 今回のスキルの使い方、さらによくするには？」と入力すると、モードをBrainstorm、reasoningをHighと判定してGPT-5.4に相談してくれる。毎日何回も使うほど重宝するツールだという。

⑯保存されるマークダウンファイルの内容：生の回答＋要約＋見解

GPTからの回答は自動的にマークダウンファイルに保存される。ファイルには日時、モード、モデル、質問内容の概要が記録される。ChatGPTの生のレスポンスも保存されるが長いため、Claudeがまとめた要点も合わせて表示される。さらに、その要点に対してClaudeの見解（同意する点や次の一手）も添えられる。送ったプロンプトも分かるように記録されており、後から振り返りがしやすい構造になっている。

⑰スキルの導入方法：install.mdを読み込ませるだけで完了

GPT BrainstormスキルはメンバーシップでZIPファイルとして配布されている。導入方法は非常にシンプルで、Claude Codeに「このinstall.mdを読んでセットアップして」と頼むだけで完了する。自分でプロンプト設計できる方には不要だが、品質を即座に安定させたい方には最適なツールだ。スキルを使うことで、GPTの厳密な調査とClaudeのバランス感覚をいつでも同じ品質で活用できるようになる。

⑱まとめ：1分で繋がるAIのセカンドオピニオン環境
