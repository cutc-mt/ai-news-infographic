Anthropic最上位AI「Mythos」が異次元の性能を披露

Claude Opus 4.6を全方位で上回る「化け物」モデル、27年間見つからなかったバグも発見

Claude Mythos Anthropic SWE-bench 93.9% サイバーセキュリティ 脆弱性発見

①Opusを超える最上位モデル「Mythos」の存在が明らかに

AnthropicからClaude Opus 4.6を超える最上位モデル「Mythos」の存在が明らかにされた。Mythosはギリシャ神話に登場する言葉であり、Opus、Sonnet、Haikuと続く命名規律の中でもひときわ異彩を放つ。まだ一般には公開されておらず、クローズドな環境で運用されている。

②信頼できる企業にのみ限定共有

Mythosは現時点では一般公開されていない。信頼できる企業や組織にのみクローズドで共有されており、その危険度の高さから慎重な姿勢で扱われている。破壊的な性能を持つモデルであるため、安全性が確保できるまで一般ユーザーの手には渡らない方針だ。

③サイバーセキュリティベンチマークで圧倒的

Mythosはサイバーセキュリティ系のベンチマークにおいて、Claude Opus 4.6と比べてもかなり引き離している。セキュリティ分野では攻撃者と防御者のバランスが話題になりやすいが、Mythosはそのバランス自体を覆すレベルの性能を示している。まずは信頼できる組織から固めていく方針だ。

④SWE-bench Verifiedで93.9%を達成

コーディング系のベンチマークでもMythosは圧巻の結果を残した。SWE-bench Verifiedで93.9%を記録し、半年から1年前には70%を超えれば「かなり良い」と言われていた水準からは想像もできない進化ぶりだ。笑ってしまうほどの数字だと解説されている。

⑤SWE-bench Proで77.8%の驚異的スコア

難易度の高いSWE-bench Proでも77.8%を記録した。この数字はSWE-bench Verifiedですら到達が難しかった領域であり、ほとんどの開発者やエンジニアよりも確実に開発ができるレベルに到達している。もはや人間を超えた開発能力と言っても過言ではない。

⑥Terminal Benchでも大幅に進化

話題のTerminal Bench系においても、Opus 4.6でもすでに高いスコアを叩き出していたが、Mythosはそこからさらに一回り大きく進化している。ターミナル上での実務的なタスク処理能力が飛躍的に向上しており、実用性の面でも大きな意味を持つ。

⑦Humanity's Last Examで圧倒的強さ

人類の最終試験と呼ばれるHumanity's Last Examでは、Opus 4.6がツールありで53.1%なのに対し、MythosはツールなしでもOpusを上回るスコアを記録。ツールありになるとさらに一回り引き離す結果となり、汎用的な知能の高さを証明している。

⑧GPT-5.4やGemini 3 Proをも凌駕

Humanity's Last Examの比較では、GPT-5.4 HighやGemini 3 Pro、3.1 Proが45%程度にとどまる中、Mythosはそれらすべてを大きく上回った。どんなモデルに対しても一回り強力であり、どのような推論を行っているのか非常に興味が持たれる結果だ。

⑨コーディング能力が特に突出して進化

全体的なベンチマークを見ても、コーディング系の能力がとんでもなく進化しているのがMythosの最大の特徴だ。OS Worldのような難しいベンチマークではまだ少し伸びしろがあるものの、全体的にOpus 4.6を確実に上回っている。特に開発系タスクでの進化は別格だ。

⑩27年間見つからなかったバグを発見

Mythosの最も衝撃的な能力は、27年間もの間、世界中のセキュリティ専門家が見つけられなかったバグを発見したことだ。Anthropicの公式発表でもこの成果が紹介されており、文字通り「化け物」レベルの脆弱性発見能力を実証している。

⑪FFmpegなど広く使われるソフトの脆弱性も特定

具体的にはFFmpegなど、様々な場所で広く使われているソフトウェアの不具合や脆弱性を発見した。FFmpegは動画処理の基盤として世界中のシステムで利用されているため、そこに潜むバグの発見はセキュリティ全体に多大な影響を与える。脆弱性の発見能力が凄まじい。

⑫大規模サイバー攻撃の危険性が急増

これだけ強力なLLMがあれば、世界トップクラスの専門家を除くほぼ全ての人間を上回るレベルでソフトウェアの脆弱性を発見し、悪用することも可能になる。しかもLLMは人間の注意力の限界を気にせず大量の作業ができるため、量の面でも超強力だ。今年中に大規模なサイバー攻撃が激増する懸念がある。

⑬AGIの可能性と制限付き配布の方向性

Mythosの性能を見ると「AGIはいつ実現するのか」という問いに近づいている。もしAGIが実現したとしても、一定の制限の中で配布される可能性が高いと推測されている。一般に公開されたとしても、規制や制約が伴う形になるだろうという見方だ。

⑭料金設定：入力$25・出力$125

Mythosの料金は入力100万トークンあたり$25、出力100万トークンあたり$125と共有された。これは非常に高いが、Opus 4.6の5倍程度であり、$1000プラン（Opus Max）の5倍ならOpusのように使える感覚だ。GPT-5.4 Proよりも安価に設定されている。

⑮価格以上のコストパフォーマンス

この価格について、解説者は「思ったより安い」と評価している。これだけの性能向上があってすぐ使えるなら、オーケストレーションの重要な部分で活用するには十分なコストパフォーマンスだ。重要な部分で気づきを与えてくれるなら、Opusの5倍でも全然使えるという感覚だ。

⑯自立改善ループの実現に近づく

Opus 4.6時代でもアバウトな要件から改善ループを回してハーネスを自立改善する取り組みが実践されている。Mythosはその発見力をさらに高めており、メタ認知能力が強いClaude系モデルの特性を活かした自立改善がより現実的なものになっている。

⑰ハーネスエンジニアリングの重要性が増大

Mythosのような強力なモデルが登場する中で、ハーネスエンジニアリングの知識がより重要になっている。Sonnetしか使えない、あるいはHaikuしか使えない状況でも戦うためには、人間側のハーネスエンジニアリングの知識が鍵になる。制限の中でどう戦うかが問われている。

⑱今後のLLM進化への波及効果

MythosはOpus、Sonnet、Haikuの改善にも繋がっていく。GPT-5.3 Codexの時もモデル成長に使われた前例があるように、Mythosの技術は下位モデルにも還元される。国際的な競争も激化する中、LLM全体の進化がさらに加速していくことが期待される。名前の良さも含めて注目のモデルだ。
