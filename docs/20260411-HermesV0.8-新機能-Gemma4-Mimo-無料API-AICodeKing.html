Hermes V0.8 大幅アップグレード!

新機能満載の無料API & ローカルモデルで最強のエージェント体験

Hermes Agent V0.8.0 Gemma 4 Mimo V2 Pro OpenClaw比較

①Hermes Agent V0.8.0がリリース

2026年4月8日にHermes Agentのバージョン0.8.0がリリースされた。多数の新機能と改善が含まれており、エージェント体験が劇的に向上している。

②バックグラウンドタスクの自動通知

長時間かかるタスクをバックグラウンドで実行中、完了すると自動的に通知が届くようになった。作業をブロックせずに並行タスクをこなせる。

③ライブモデル切替に対応

CLI、Telegram、Discord、Slackのすべてで、セッション中にモデルをリアルタイムで切り替えられる。用途に合わせて最適なモデルをその場で選択可能。

④GPT/Codexのツール使用が改善

GPTやCodexのツール利用時の失敗モードをベンチマークし、ガイダンスを修正。これまで上手く動かなかったケースが解決されている。

⑤ネイティブGoogle AI Studio対応

HermesがGoogle AI Studioをネイティブサポート。設定画面から直接GoogleのAPIキーを入力するだけで、最新モデルにアクセスできる。

⑥Gemma 4ラインナップ: エッジモデル

Gemma 4のE2BとE4Bはエッジデバイス向けの小型モデル。リソースが限られた環境でも高速に動作し、軽量タスクに最適。

⑦Gemma 4ラインナップ: 26B MoE

26B MoE（Mixture of Experts）モデルはローカル実行に最適化されている。少ない計算リソースで高い性能を発揮し、ローカルマシンでの利用におすすめ。

⑧Gemma 4ラインナップ: 31B dense

31B denseモデルは最高品質を提供するフラッグシップ。性能を最優先する場合に選択すべきで、リモートAPI経由での利用が想定されている。

⑨無料のMimo V2 ProがNous Portalに登場

XiaomiのMimo V2 ProがNous Portalで無料で利用可能に。圧縮、要約、ビジョンなどの補助タスクに活用でき、メインモデルの負荷を軽減。

⑩三段構成の無料スタック

ローカルOllama + AI Studio無料 + Mimo無料の3層構成で構築。メイン処理はOllama、APIタスクはAI Studio、補助はMimoと使い分けられる。

⑪非アクティブタイムアウトが改善

これまでの単純な経過時間ではなく、実際のツール活動を追跡してタイムアウトを判定。作業中なのに切断される問題が解消された。

⑫危険コマンドに承認ボタンが追加

SlackやTelegramで危険な操作には承認ボタンが表示されるようになった。誤操作によるファイル削除やシステム変更を防ぐ安全策。

⑬中央集権ログと設定バリデーション

すべてのログを一元管理できる中央集権ログ機能を追加。また設定ファイルのバリデーションも入り、誤った設定での起動を防止。

⑭MCP OAuth 2.1とマルウェアスキャン

MCPサーバーがOAuth 2.1をサポートし、安全な認証フローが可能に。さらにマルウェアスキャン機能も追加され、セキュリティが強化。

⑮OpenClawとの比較: 柔軟性で勝る

OpenClawは優れたツールだが、Hermesの方がモデル選択や設定の柔軟性が高いと主張。ユーザーは自身のニーズに合わせて自由にカスタマイズできる。

⑯柔軟性が最大の魅力

Hermesの最大の魅力は圧倒的な柔軟性。ローカル、クラウド、無料API、有料APIを組み合わせて、自分好みのエージェント環境を構築できる。

⑰開発者体験の向上

V0.8では設定ウィザードの改善やdoctorコマンドの強化により、セットアップからトラブルシューティングまで快適に。初心者でも始めやすい。

⑱まとめ: 進化し続けるHermes

V0.8.0は実用性と安全性が大幅に向上。無料APIとローカルモデルを活用すれば、コストゼロで強力なエージェント環境を構築できる。「This is JUST SO BETTER NOW!」
