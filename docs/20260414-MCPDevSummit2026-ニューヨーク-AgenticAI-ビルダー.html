③ MCP: Model Context Protocol

  • MCPはAIモデルと外部ツールを繋ぐ標準プロトコル
  • AIエージェントがコンテキストを共有し、ツールを呼び出すための共通規格
  • 様々なブースでMCPを使った多様な活用法が展示されていた
  • Agentic AIのアーキテクチャの未来を今日ここで構築している
MCPはAgentic AIの基盤となる重要な技術です！

MCPの核心コンセプト

⑤ MCPサーバーの役割

  • MCPサーバーはデータソースへの窓口として機能する
  • セキュアな方法でAIに外部データへのアクセスを提供
  • ファイルシステム、データベース、APIなど多様なリソースに接続可能
  • 標準化されたインターフェースで再利用性が高い

⑥ MCPクライアントの役割

  • Claude CodeやCursorなどのAIアプリケーションがクライアントとして動作
  • 複数のMCPサーバーに同時接続してコンテキストを収集
  • AIモデルが適切なツールを自動的に選択して実行
  • ユーザーの指示に応じて動的にツールを呼び出す

⑦ ツール呼び出しの仕組み

  • AIが自律的に必要なツールを判断して呼び出す
  • ファイル読み書き、検索、計算、API呼び出しなどを統合的に実行
  • 複数ステップのタスクを自律的に計画・実行
  • 結果を検証して自己修正するループ
ツール呼び出しがAgentic AIの核心！MCPがそれを標準化します

⑧ コンテキスト共有の重要性

  • AIモデルに正しい文脈を渡すことが精度の鍵
  • コード、ドキュメント、設定ファイルなどの関連情報を整理
  • 異なるエージェント間で情報を共有する仕組み
  • コンテキストウィンドウの制約を意識した効率的な情報設計
Agentic AIの未来と実践

⑨ マルチエージェントの連携

  • 複数のAIエージェントが協力して作業するアーキテクチャ
  • 各エージェントが専門領域を担当してタスクを分担

  • 複雑なタスクを並列処理して効率化

⑩ セキュリティとプライバシー

開発の進め方が根本的に変わる！AIが作業を担当し、人間が方向性を決める時代に
