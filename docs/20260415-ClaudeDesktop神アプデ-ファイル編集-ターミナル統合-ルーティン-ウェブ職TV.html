② UI刷新: サイドバーにボタン配置が移動

  • これまで上部にあったチャット・コワーク・コードのボタンがサイドバーに移動。
  • 並び順は「チャット → コワーク → コード」で、3つのモードが明確に整理された。
  • 各モードの役割が視覚的に分かりやすくなり、画面の切り替えも直感的に。

③ 3つのモード解説: チャット・コワーク・コード

  • チャット: ChatGPTのような基本的な対話モード。最もシンプルで使いやすい。
  • コワーク: パソコンの操作やファイル編集ができる。ある程度複雑なことも可能。
  • コード: いわゆるClaude Code。最も複雑な開発タスクに対応し、何でもできる。
チャット → コワーク → コードの順で
できることが増えるイメージ！

④ ファイル編集機能: アプリ内で直接編集が可能に

  • これまではテキストファイルを編集するために、外部のエディターを開く必要があった。
  • 今回の更新で、デスクトップアプリ内でファイルを開いて直接編集できるようになった。
  • 編集後は保存・破棄を選択可能。他のアプリを開き直す手間がなくなった。

  • Claude Desktopはプランやタスクの可視化など独自の強みもあるが、機能面ではまだ発展途上。
