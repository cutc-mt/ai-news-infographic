  • pip install markitdown で即導入可能
  • 完全ローカルで動作し、APIキー不要
  • MITライセンスで商用利用も自由

  • シート名が見出しとして保持される
  • データの型情報はテキスト表現に変換

④ZIPファイルの統合出力とURLスクレイピング

ZIPアーカイブは中身を自動展開して一つのマークダウンに統合出力する。複数ファイルがまとまったアーカイブも一発で処理可能。

  • ZIP内の各ファイルを個別に変換
  • 変換結果を一つのドキュメントに統合
  • URL指定でウェブページもスクレイピング変換
CLIでの利用例:
markitdown report.zip > output.md
markitdown https://example.com > page.md

⑦なぜPDFをそのままLLMに渡してはいけないのか

PDFをそのままLLMに渡すのは非効率で精度も下がる。理由は明確だ。

  • PDFはバイナリ形式で構造情報の解釈が困難
  • 見出し階層やテーブル構造が失われる
  • チャンキング（分割）の精度が著しく低下
PDF → LLM の直渡しは
RAGの精度を大きく下げる原因に！

⑧マークダウン変換でチャンキング精度が劇的に向上

マークダウンに変換することで見出し階層、テーブル、書式がプレーンテキストで構造を保持される。これがRAGの精度を大きく引き上げる。

  • 見出し（# ## ###）でセクション分割が正確に
  • テーブル構造がそのまま保持される
  • 太字やリストなどの書式情報が残る
  • 意味的なチャンキングが容易になる
構造を保ったままテキスト化するのが
RAG前処理の正しいアプローチ！

⑨LangChain・LlamaIndexもマークダウンをネイティブサポート

LangChainとLlamaIndexはマークダウンローダーをネイティブでサポートしている。つまり、MarkItDownで変換したマークダウンはそのままRAGフレームワークに流し込める。

  • LangChainのMarkdownLoaderで直接読み込み可能
  • LlamaIndexのMarkdownReaderも標準搭載
  • 変換→チャンキング→埋め込み→検索のパイプラインがシンプルに
# パイプラインの流れ

  → LangChain MarkdownLoader

⑩競合比較：MarkItDown vs Docling vs Unstructured

RAG前処理ツールの3強を比較すると、それぞれ特徴が明確に異なる。

MarkItDown（Microsoft）
・インストールサイズ：数MB
・完全ローカル実行、無料
・軽量で高速だが複雑なPDFにはやや弱い

Docling（IBM）
・高精度なPDF変換
・インストール：数GBと重量級
・複雑な日本語PDFに強い

Unstructured.IO
・高機能なドキュメント処理
・API従量課金が発生
・エンタープライズ向け

⑪MarkItDownの最大の強み：軽量・ローカル・無料

MarkItDownが選ばれる最大の理由は「軽量・ローカル・無料」の3拍子が揃っていること。

  • 数MBのインストールサイズで済む
  • 完全ローカル実行で外部APIへの依存なし
  • MITライセンスで完全無料
  • シンプルなAPIですぐに使い始められる
Doclingは数GBもインストールするから
手軽さではMarkItDownが圧勝！

⑭課題：日本語PDFの精度に課題あり

縦書きや複雑なレイアウトの日本語PDFでは文字化けや順序の乱れが発生することがある。これはMarkItDownの現状の大きなトレードオフだ。

  • 縦書きPDFの読み取り精度が低い
  • 複雑な表組みでセルの順序が乱れる
  • 特殊フォントの文字化けが発生
複雑な日本語PDFには
IBM Doclingの使用を検討しよう

⑮課題：バージョン間のAPI破壊的変更

バージョン0.1.xから0.2.xへの移行でAPIに破壊的変更が発生している。マイグレーションガイドも不充分で、既存コードの修正が必要な場合がある。

  • 関数名や引数の仕様が変更された
  • マイグレーションガイドが不充分
  • バージョンを固定して運用するのが安全
推奨: requirements.txtで
バージョンを明示的に指定

⑰CLI活用：パイプで直接マークダウン出力

MarkItDownはCLIツールとしても利用可能で、パイプを使って他のコマンドと連携できる。

# 複数ファイルの一括変換

MarkItDownはRAG前処理を軽量に実装したい場合に最適な選択肢だ。ただし、用途に応じて使い分けることが重要。

  • MarkItDown: 軽量・高速・無料で大半のユースケースに対応
