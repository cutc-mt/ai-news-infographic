① Recall 2.0 リリース: セカンドブレインの革命

  • 2026年4月14日に Recall 2.0 がリリースされた
  • 「人間だけが使う道具」から AIエージェントが接続できるナレッジエンジン へ進化
  • これまでで最大のアップデートと公式が宣言
  • 2022年11月のHacker News投稿から始まり、50万人以上のユーザーに成長
単なるアップデートじゃない。個人の知識をAIが直接活用できる時代が始まった！

② Recall 1.0とは: 個人のセカンドブレイン

  • 自分が興味を持ったものをワンクリックで記憶するツール
  • Chrome Brave などのブラウザ拡張機能で簡単保存
  • YouTube動画、PDF、ブログ記事、ポッドキャスト、Webページを即座に保存
  • AIが自動で分類・タグ付け・関連付けを行う
  • スマホアプリもあり、共有ボタンから素早く保存可能
気になったらポイポイ放り込むだけ。AIが勝手に整理してくれるから超便利！

③ ナレッジグラフ: 知識の繋がりを視覚化

  • 保存した情報が ネットワーク状に自動で繋がる
  • Reddit、GitHub、Discord、Twitter、Gmailなど多彩なソースが関連付け
  • タグごとに分類: AIテクノロジー、バイブコーディング、3D/ゲームなど
  • 日付ごとに整理され、見返しやすくなっている
  • 2.0でグラフ表示が大幅に進化し、驚愕のビジュアルに
情報と情報がバチバチ繋がっていく様は圧巻。自分の頭の中が可視化されてるみたい！

④ リスンモード: 知識を聞く時代の幕開け

  • 保存したコンテンツを 音声で聞けるオーディオプレイバック
  • 短い音声サンプルで声を再現する ボイスクローン 機能
  • 30言語以上 に対応: 英語コンテンツを日本語要約で聞くことも可能
  • 複数のビルトインボイスから選択可能
  • 通勤時間中にヘッドホンで知識をインプットできる
英語のYouTube動画を日本語で即座に読み上げてくれる。これは本当にすごい！

⑤ ブラウザサイドパネル: 拡張機能が進化

  • ポップアップから サイドオーバーパネル へ拡張
  • ブラウザでWebページを読みながら、その場で音声再生が可能
  • タブを移動せずに開いたまま操作できる
  • 画像表やコードブロックを含む高度なテキスト抽出に対応
  • YouTube動画のタイムスタンプ付きで内容を把握できる
英語動画を見ながら、サイドパネルで日本語要約を確認できる。作業効率が爆上がり！

⑥ Recall 1.0の限界: 知識が閉じ込められていた

  • どれだけ知識を貯めても リコールアプリの中だけ の話だった
  • 外部のAIエージェントはその知識にアクセスできなかった
  • 「自分の頭脳とAIの頭脳は切り離されたまま」だった
  • 開発者は自分でBase44で専用アプリを作る必要もあった
1.0でも不満だったのはここ。知識が外に出られないのが最大の弱点だった。

⑦ API & MCP対応: 革命の核心

  • 2026年4月14日に APIとMCPサーバーが公開 された
  • 自分のセカンドブレインと AIエージェントが同期 可能に
  • 外部のエージェントが個人のナレッジに直接アクセスできる
  • MCPはフリープランからでも利用可能
  • Recallは「メモツール」から「個人AI基盤のコンテキストレイヤー」に進化
ここが1番クレイジーなポイント！俺とAIの脳が初めて同期されるってこと！

⑧ Agentic Chat: 3つのコンテキストを同時に扱う

  • ナレッジベース × インターネット × 好きなモデル を同時使用
  • 「Recall + Web」モード: 個人の知識＋最新Web情報を組み合わせて回答
  • 「Recall」モード: 自分の蓄積した知識のみをコンテキストにチャット
  • 「Web」モード: リアルタイムの最新情報を検索しながらチャット
  • タグ分類ごとに対象を絞って質問することも可能
過去のリサーチを全部まとめて、最新情報と組み合わせて回答。これができるのはRecallだけ！

⑨ 実用的な使い方: Agentic Chatの活用例

  • 「先月のリサーチを全部まとめて」という横断的な質問が可能
  • 「1ヶ月前に聞いたポッドキャストで言及されていたXXを探して」と検索
  • 「今年私が好きだったものをベースに今夜の映画を提案して」と提案生成
  • 過去にスクラップしたブログ記事を探し出す
  • 蓄積した知識情報を元に、的確な回答を生成
「あれなんだっけ？」を即座に解決。自分の知識がAIの燃料になる！
