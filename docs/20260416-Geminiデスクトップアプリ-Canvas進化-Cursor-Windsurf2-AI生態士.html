①Geminiのパーソナルインテリジェンスが全世界展開

②GeminiのMacデスクトップアプリがリリース

  • 「Claudeもデスクトップアプリから進化した」のと同じ道を歩む可能性
  • 来月のGoogle IOに向けた布石との見方
ローカルファイルにアクセスできるようになれば、AIエージェントのハーネスとして大きく進化するかも！

③Geminiチャットで各種ファイルを生成してドライブに保存

  • Googleドキュメントをチャット上で作成可能に
  • PDFファイルもGeminiチャット内で生成
  • Markdownファイルにも対応
  • 生成したファイルをそのままGoogleドライブにアップロード可能
  • チャット内でドライブのファイルを引き出したり、保存したりできる
物理的なファイルをチャット内で動かせるようになったのは、かなり進化した機能！

④Canvas上でスプレッドシートを直接編集

  • GeminiのCanvas機能でスプレッドシートを開いて編集できる
  • Canvas上で表示させながら手動で編集も可能
  • 真っさらなスプレッドシートを生成させたり、既存のものを読み込ませたりできる
  • チャットとCanvasの連携が非常に使いやすいと話題に
この機能、ネット上で大盛り上がり！スプシのサイドバーから呼び出す以外のアプローチが登場した。

⑤GeminiにスプレッドシートのAI関数を設定可能

  • スプレッドシートを出力する際にAI関数を使わせることができる
  • Geminiが自動的にAI関数を設定してくれる
  • Canvas上でプレビューしながら関数の動作を確認可能
  • 手動編集とAI関数の組み合わせが実現
めちゃくちゃマニアックな使い方だけど、使いこなせたら超便利！
