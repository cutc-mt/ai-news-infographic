AIニュース No.337
ビッグニュースだらけの大型アップデート祭り

2026年4月17日（金）

Claude Opus 4.7 / Codex大幅アップデート / Google / Perplexity / Cursor

Anthropic / Claude 関連ニュース

① Claude Opus 4.7 がリリース

  • コーディング性能が向上し、より高品質なコードを生成できるようになった
  • 資格（視覚）機能が大幅に向上。今までの3倍以上の高解像度で画像を認識可能に
  • プロンプト追従性が向上し、スキルや指示に忠実に実行するようになった
  • API料金は変更なし。今までと同じ感覚で使える
プロンプトが曖昧だと曖昧な出力になるため、具体的なプロンプトを心がけましょう！

② Claude Max にオートモード展開

  • Maxプラン限定でオートモードが追加された
  • ほぼ全操作で許可なしで自動実行してくれるモード
  • 指示通りにバーッと実行する反面、ガードレールの整備が重要
  • X上でも意見が分かれており、割とガンガン使っている上級者向けの機能
強力な機能ゆえに、使う方は十分な注意が必要です！

③ Claude Code にエクストラハイ思考レベル追加

  • 思考レベルにエクストラハイが追加され、選択可能に
  • 多く考えさせるほど精度が上がるが、その分トークン消費も増加
  • 自分のタスクに合わせて、ハイやエクストラハイから適切なレベルを選ぶのがおすすめ
考えるほどトークンを消費するので、バランスが大事ですね

④ Claude レートリミットがリセット

  • Opus 4.7のリリースに伴い、レートリミットがリセットされた
  • 昨日までガンガン使っていた人も、1回リセットがかかっている状態
  • これからまたたくさん使えるようになった。素直にありがたいアップデート
リミット制限に悩んでいた人には嬉しいニュースです！

⑤ Cursor で Opus 4.7 が期間限定50%オフ

  • CursorでOpus 4.7がすぐに使えるようになった
  • 期間限定で50%オフキャンペーンを実施中
  • 期間はおそらく1週間もないと思われる。Cursorに課金している方は今のうちに試すのがおすすめ
  • ClaudeのAPI料金は高いため、Cursorクレジットの消費が大きいが半額ならかなりお得
期間限定なのでお早めに！
OpenAI / Codex 関連ニュース

⑥ Codex デスクトップアプリが大幅バージョンアップ

  • コンピューターユーズができるようになり、パソコンの操作を自動化できる
  • GPT Image 1.5を搭載し、画像生成が可能に（サブスク内で利用可能、API料金不要）
  • オートメーション機能が改善され、途中で止まったタスクをそのまま再実行できるように
  • ブラウザーユーズも利用可能に
サブスク料金の中で画像生成が使い放題なのは大きい！

⑦ Codex に内蔵ブラウザとコメント機能が追加

  • UIが刷新され、右側にサイドバーが表示可能に
  • サイドバーからブラウザやターミナルを開けるようになった
  • コメントモードでWebページの要素を選択し、直接修正指示が可能に
  • 「ここをこう修正して」という指示が直感的にできるようになった
これはかなり便利ですね！UIの進化が止まらない

⑧ ChatGPT が医療特化モデルをリリース

  • 生物学・翻訳・医学の研究を支援するために構築された特化型モデル
  • 研究分野においてはGPT-5.4よりも高性能とされている
  • あくまで研究用モデルなので、一般ユーザーは気にしなくてよい
医療・研究分野でのAI活用がさらに加速しそうです
Google 関連ニュース

⑨ NotebookLM がバージョンアップ

  • ノートにカバーアートを追加できるようになった
  • ノートに説明文を追加可能に
  • ホーム画面のグリッド表示が見やすくなった
地味だけど意外と便利かも？

⑩ Google 検索のAIモードがアップデート

  • ウェブページを並べて表示できるようになった
  • 現在はアメリカ限定で展開中
  • 近日中に日本を含むより多くの地域に展開予定
  • AIモードのUIが変わり、使い勝手が向上する見込み
日本での展開はもう少し待ちですね

⑪ Stitch に2つの新機能が追加

  • ネイティブ画像生成ができるようになった。Webページの素材画像をその場で作成可能
  • プロジェクトコピー＆ペースト機能が追加。過去に作ったデザインを新プロジェクトに引き継げる
  • デザインやレイアウトの再利用が容易になった
Webページ制作の効率が大幅アップしそう！

⑫ Gemini デスクトップにコンピューターユーズ実装へ

  • Googleの中の人が、Geminiデスクトップアプリへのコンピューターユーズ実装を計画中と発言
  • Claudeが先行して強く、Codexも参戦。Googleがやらないわけがない状況
  • Google I/Oまで引っ張るのか、その前に出すのかは不明
AIエージェントの競争が激化しています！

⑬ Google AI Studio が課金アカウントと連携へ

  • Google AIプラン（プラス/プロ）を契約中のユーザー向けに、AI Studioとの連携が計画されている
  • モデルの切り替えや利用上限の引き上げが可能になる見込み
  • まだ実装はされていないが、今後の展開が期待できる
詳細はまだ公式発表待ちです
Perplexity / GenSpark / Cursor / Grok / Xサーバー

⑭ Perplexity が Sonar Computer をリリース

  • Mac用デスクトップアプリをリリース。コンピューターユーズに対応
  • Macユーザーおよび待機リスト登録者が利用可能に
  • Web版はすでにコンピューターユーズがあるが、別クレジット課金が必要
デスクトップ版ならサブスク内で使えると嬉しいですね

⑮ GenSpark に C-Dance 2.0 が追加

  • GenSparkの動画生成にC-Dance 2.0がリリースされた
  • C-Dance自体のクオリティは非常に高い
  • すでに2.5や3の噂も出ており、進化が早い
  • 動画生成はクレジットを多く消費する点に注意
動画生成AIの進化スピードが止まらない！

⑯ Cursor Composer 2.5 が開発中（リーク）

  • Cursorのミートアップで明らかになったリーク情報
  • 現在のComposer 2（K2.5ベース）がすでに非常に優秀だが、さらに性能が向上する見込み
  • K2.6ベースになるのか、別の学習手法なのかは不明だが、期待大
これはマジで楽しみですね！

⑰ Grok CLI が来週リリースの噂

  • イーロン・マスクがそれっぽい発言をしており、来週CLIリリースの噂が浮上
  • 「ターミナルからウィークリー」のようなグロックビルドアプリという表現
  • Grok Codeは存在するが、完成された単体ツールはまだ登場していない
  • 料金や使い勝手など、まだ不明な点が多い
出る出る詐欺も多いですが、期待して待ちましょう

⑱ Xサーバー APIがリリース

  • XサーバーのAPIが提供開始された
  • AIエージェントが直接操作して設定してくれる対象：
  • ドメイン / SSL設定 / メールアカウント / WordPress など
  • 便利だが、中でどういう設定をしているか把握しておくことが重要
サーバー設定が苦手な人には救世主かも！でも中身の理解は必要です
