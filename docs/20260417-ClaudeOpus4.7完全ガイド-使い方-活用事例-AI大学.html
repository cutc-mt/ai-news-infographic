① Claude Opus 4.7 がリリース: アンソロピックの最上位モデルが進化

2026年4月16日、アンソロピックは自社の最上位AIモデルの最新版であるClaude Opus 4.7を公開しました。前モデルのOpus 4.6から大きく進化し、特にコーディング能力と複雑で長時間に渡る作業をこなす力が大きく向上しています。

② コーディング能力が大幅に向上: 最も難しい作業でも安心して任せられる

ソフトウェア開発の分野で前モデルのOpus 4.6から大きく進化しました。いち早く触ったユーザーからは「これまでは細かく確認しながら進める必要があったような最も難しいコーディング作業でも、安心して任せられる」という声が出ています。

③ 画像認識能力が飛躍的に向上: より高い解像度で画像を認識

Claude Opus 4.7は画像を見て理解する能力が大きく向上しています。より高い解像度で画像を認識できるようになり、高解像度の画像や複雑なグラフなどを分析する能力が高くなっています。ビジネス文書の解析にも威力を発揮します。

④ ビジネス向け作業能力が強化: スライド・ドキュメントの品質が向上

仕事向けの作業ではよりセンスが良く想像的になっており、生成したアプリのUIやスライド資料、ドキュメントなどをこれまで以上に高品質に仕上げることができます。文書の作成やスライドの作成、データ分析、動画編集などの業務系タスク処理性能も向上しました。

⑤ 自己検証機能で精度が向上: 回答前に自ら結果を確認

ユーザーの指示をしっかり守るだけでなく、回答する前に自ら結果を確認し、正しいかどうかを確かめた上で返してくれるようになりました。複雑で長時間かかる作業でも精度を保ちながら安定して進めることができます。

⑥ ベンチマークスコア: コーディングとエージェント系で高スコアを記録

アンソロピックが公式でAIベンチマークを公表しています。前モデルと比べてスコアが下がっている項目もあるものの、コーディングの性能や端末操作、OS操作系などのエージェント系の性能は上がっています。画像認識力や文書理解のベンチマークでも高スコアを記録しています。

⑦ Claude Mythos Preview: 高度なサイバー能力を持つ特別モデル

参考として、先日アンソロピックが発表したClaude Mythos Previewのベンチマークスコアも掲載されています。これはソフトウェアの脆弱性を自立的に発見して、それを悪用する攻撃手法まで構築できる高度なサイバー能力を持つAIモデルです。

セキュリティ研究向けの特殊なモデルで、一般向けには提供されていません

⑧ 公式AIチャット「Claude」で使う方法: 有料プランが必要

  • 公式サイトでGoogleアカウントなどを使ってログイン
  • Claude Opus 4.7は有料ユーザー限定で利用可能
  • ProまたはMaxプランに登録すれば使用できる
  • ログイン後のチャットフォームでOpus 4.7を選択
無料ユーザーはOpus 4.7を利用できません。Pro/Maxプランへの登録が必要です

⑨ アダプティブシンキング: 新しい思考モードが自動で最適化

Opus 4.7では従来の「拡張思考」がアダプティブシンキング（Adaptive Thinking）に変わりました。この項目を有効にすると、モデルがユーザーの質問ごとに必要な思考量を自動的に判断して調整してくれるため、性能がより高くなります。

⑩ JanSparkで無料で使う方法: 無料ユーザーもOpus 4.7にアクセス可能

  • JanSparkは様々なAIモデルを使えるプラットフォーム
  • 無料ユーザーもClaude Opus 4.7を利用可能
  • 公式サイトのサインボタンからGoogleアカウントでログイン
  • フォームで「ウルトラ」をオンにしてチャットすれば利用できる
  • クレジットがある場合には無料で試せる

⑪ クレジット消費に注意: Opus 4.7はコストが高い

Claude Opus 4.7はクレジットの消費が激しいので、JanSparkで本格的に使いたい場合には有料のサブスクリプションプランに登録する必要があります。また公式ClaudeのProプランでも、数回タスクを実行しただけで使用制限がすぐにマックスになってしまうので注意が必要です。

有料プランでも使用制限に注意！Opus 4.6よりコストが高いとされています

⑫ 各種ツールへの展開: エージェントやエディタにも実装

Claude Opus 4.7は以下のツールにも実装されています：

  • Claude Code（AIエージェント）
  • Claude Cowork（AIエージェント）
  • API経由で各種プラットフォームに展開
  • CursorやWindsurfなどのAIコードエディタ
  • LovableやV0などのコーディングプラットフォーム
Opus 4.6が実装されていたツールは徐々にOpus 4.7に置き換わっていく予定です

⑬ 活用事例①: Webアプリの生成 — iOS対応アシスタントを作成

日付や天気、ニュースフィード、チャットフォームを含めたiOS対応のパーソナルアシスタントの生成を依頼すると、モバイル向けのUIで作成してくれました。作ったアプリは「Save Artifact」でHTMLプレビューできるアーティファクトとして保存できます。

簡単なアプリやゲーム、ウェブサイトならサクっと作れます。Claude Codeを使えば複雑なアプリも！

⑭ 活用事例②: スライド資料の生成 — 動画からスライドを作成

ニュース動画の現行（文字起こし）をアップロードして、「現行を要約して10枚の豪華なスライド資料を作ってください」と依頼すると、ニュースの内容をトピックごとにまとめた全10枚のスライド資料を作成してくれました。Claude標準のスライドスキルを使って細部まで調整しています。

⑮ スライドの編集とエクスポート: Googleスライドで再利用可能

作ったスライド資料はAIチャット経由で編集可能です。また、資料をダウンロードしてGoogleスライドなどで編集することもできます。シンプルな指示でもそれなりの資料を作ってくれますが、複雑な指示を入力するとスライドをさらに作り込むことができます。

ダウンロードしてGoogleスライドで編集することもできます

⑯ 活用事例③: 画像の分析 — 複雑なグラフを高速で解析

複雑なグラフをクロードに読み込んで分析してもらったところ、高速に分析して回答を出力してくれました。英語の複雑なグラフなどで内容が分からない場合には、Claude Opus 4.7に分析してもらうと非常に便利です。画像認識力と文書推論能力が高く評価されています。

⑰ 利用上限の引き上げ: 有料ユーザーの制限が緩和

Claude Opus 4.7は前モデルのOpus 4.6よりも考えるために使うトークン量が増えるため、Claudeの有料ユーザーの利用上限が引き上げられたとアナウンスされています。ただしProプランでは数回のタスク実行で制限に達するので、使いどころを見極めることが重要です。
