①スキルとは何か: AIの作業手順書

  • スキルはClaude CodeにあらかじめインストールしておけるAIの作業手順書のこと。
  • Claudeが公式で公開しているスキルもあれば、世界中の有志が作ったスキルも存在する。
  • 自然言語でスキルを呼び出すことが可能。スラッシュコマンドでも呼び出せる。
例：「スライド資料を作成して」と伝えると、
自動的に最適なスキルが選ばれて実行される！

②基本操作: デスクトップ版の使い方

  • デスクトップ版の「カスタマイズ」→「スキル」から管理。
  • プラスマークの「スキルを閲覧」から公式スキルをダウンロード可能。
  • CLIからもインストール可能：/plugin コマンドを使用。
  • デスクトップ版の一覧はまだスキルが少ないため、CLI経由が確実。
CLIで一度インストールすれば
デスクトップ版でも使えるようになるよ！

③デフォルトスキル: 最初から入っている4つ

  • PPTX - パワーポイントのスライド生成・編集。
  • Excel - ピボットテーブルや数式の作成。
  • Word - ドキュメントを綺麗に作成。
  • PDF - PDFの作成や分割。
これら4つのスキルはデフォルトで
最初から使える！インストール不要！

④フロントエンドデザイン: 人気No.1

  • ダウンロード数最多の最人気スキル。
  • AIっぽさをなくしたおしゃれなWebサイトを簡単に作成。
  • HTML、CSS、JSだけでコーポレートサイトなどが作れる。
  • デザインルールを学んで、独自スキルに改造することも可能。
「フロントエンドデザイン」と入力するだけで
おしゃれなサイトが爆速で完成する！

⑤Playwright: ブラウザ自動操作

  • ブラウザ操作を自然言語で実行してくれるプラグイン。
  • 自動でChromeを開いて情報収集や操作を行う。
  • ログインが必要な環境にもアクセス可能（自己責任）。
  • 自作スキルと組み合わせれば毎日自動投稿も実現。
MCPがセットになったプラグイン。
インストールは/pluginコマンドから！

⑥Claude Code Setup: 環境分析と提案

  • プロジェクト内のファイルを読み込んで最適な自動化を提案。
  • MCP・スキル・フック・エージェント・コマンドの5カテゴリーで推奨を提示。
  • 読み取り専用で安全。Claude Codeの相談役のような存在。
  • 各種設定の疑問にも答えてくれる。
「Claude Codeで悩んだらこいつに聞け！」
という状況を作れる頼もしいスキル。

⑨Plugin Dev: プラグイン開発スキル

  • プラグインはチーム配布に便利。ZIPで共有可能。
  • スキルとの違い：手順書＋外部ツール(MCP等)のパッケージ。
  • 実装と検証付きで本格的なプラグインを作成。
  • ただしエラー修正は難しいため、まずはスキルで試すのがおすすめ。
スキル＝手順書のみ

⑩CLAUDE.md Management: ルール管理

  • CLAUDE.mdはClaude Codeが守るべきルールを書くファイル。
  • 場所は隠しフォルダー（.claude）内にあり、手動編集が煩雑。
  • ルールの追加・古いルールの削除・自動的な監査改善を行う。
  • 定期的にこのスキルで監査すれば、CLAUDE.mdを常に最適に保てる。
Mac：Finder→移動→コンピューター
→ユーザー名→Cmd+Shift+.で隠しフォルダー表示

⑯スキルの削除方法

  • プラグインの削除：/plugin → 方向キー → uninstall。
  • スキルの削除：隠しフォルダーの.claude/skills/内のファイルを直接削除。
  • スキルフォルダーをゴミ箱に入れれば削除完了。
  • 中身は基本的にmdファイルのみのシンプルな構成。
合わないスキルはすぐ削除して
自分に合う構成を見つけよう！
