①Claude Codeと普通のAIの違い

  • 普通のAIは教える役割で、ユーザー自身が作業を行う必要がある。
  • Claude Codeはあなたの代わりに作業をしてくれる。
  • ファイルの作成、コードの編集、コマンドの実行まで全部自動でやってくれる。
AIが教えてくれるだけじゃなく、
実際に手を動かしてくれるのが大革命！

②セットアップ：Node.jsとGitをインストール

  • まずNode.jsを公式サイトからダウンロードしてインストールする。
  • Git for Windowsもインストールしてバージョン管理を有効にする。
  • ターミナルでnpm installを実行すればClaude Codeの準備は完了。
Node.jsとGitの2つだけ。
セットアップは本当に簡単！ 
npm install -g @anthropic-ai/claude-code

③料金プラン：月額17ドルのProから利用可能

  • Claude CodeはProプラン（月額17ドル）から利用できる。
  • より高性能なモデルを使いたい場合は上位プランへのアップグレードが必要。
  • まずはProプランで試して、必要に応じてプランを変更するのがおすすめ。
月額17ドルでAIが作業してくれるなら
コスパはすごく良い！

④3つのモデル：Sonnet、Opus、Haiku

  • Sonnet：バランス型の標準モデル。普段使いに最適。
  • Opus：最高性能のモデル。複雑なタスク向け。
  • Haiku：高速で軽量なモデル。簡単な作業に。
/modelコマンドで

⑤基本操作：フォルダ作成からファイル共有まで

  • 日本語で「フォルダを作って」と伝えるだけで、自動でフォルダを作成してくれる。
  • ファイルの共有もドラッグ＆ドロップで簡単にできる。
  • 画像の添付も対応しており、画面のデザインを指定する時にも便利だ。
日本語で指示するだけで
ファイル操作が全部完了する！

⑥承認ダイアログ：安全確認の仕組み

  • Claude Codeが作業を実行する前に承認ダイアログが表示される。
  • 何を実行しようとしているかを確認してから「Yes」で承認する仕組み。
  • ユーザーが意図しない操作を防ぐための安全機能だ。
勝手に作業される心配なし！
実行前に必ず確認してくれる。

⑦実践デモ：AIニュース収集ツールを作ろう

  • 動画のデモではAIニュースを自動収集してSlackに送信するツールを作成。
  • 日本語で指示するだけで、Claude Codeがコードを書いてツールを完成させる。
  • 収集したニュースを自動的にSlackのチャンネルに通知する仕組みまで構築。
「AIニュースを集めてSlackに送って」
これだけでツールが出来上がる！

⑧制作の4ステップ：基礎→カスタム→MCP→自動化

  • Step 1：ベースとなるツールの作成。
  • Step 2：ニーズに合わせたカスタマイズ。
  • Step 3：MCPでSlackと連携する。
  • Step 4：/loopで完全自動化する。
4つのステップを踏めば
誰でも自動化ツールを作れる！

⑨MCP：Claude Codeと外部サービスを繋ぐ

  • MCP（Model Context Protocol）は、Claude Codeと外部サービスを連携する仕組み。
  • 今回のデモではSlackと連携して、ニュースを自動通知。
  • Slackだけでなく、様々な外部サービスと接続できる可能性がある。
MCPを使えばClaude Codeが
世界中のサービスと繋がる！

⑩/plan：実行前に計画を確認できる

  • /planコマンドを使うと、実行前に作業計画を確認できる。
  • 何をするのかを事前に把握できるので、意図しない変更を防げる。
  • 計画を確認してから実際に実行するかどうかを判断できる。
まず計画を見てから実行。
安心して任せられる仕組みだ！ 
/plan   ← 実行前に計画を確認

⑪/rewind：一瞬で前の状態に戻せる

  • /rewindコマンドで、一瞬で前の状態に復元できる。
  • 作業が思い通りにいかなかった場合も、簡単に巻き戻しが可能。
  • 失敗を恐れずに試行錯誤できるのが大きな安心感につながる。
失敗しても一発で元に戻せるから
どんどん試して良くなる！ 
/rewind   ← 前の状態に即座に復元

⑫/resume：ターミナルを閉じても続きから再開

  • /resumeを使えば、ターミナルを閉じても続きから再開できる。
  • 作業の途中で休憩しても、状態が保存されているので安心だ。
  • 翌日でも、別のターミナルからでも、同じ作業を継続できる。
休憩しても翌日でも
どこからでも再開できる！ 
/resume   ← 作業を続きから再開

⑬/loop 24h：24時間完全自動で収集し続ける

  • /loop 24hコマンドで、24時間自動でニュース収集を継続できる。
  • 指定した間隔で自動的にタスクを実行し続ける。
  • 収集した情報は自動的にSlackに通知されるので、見逃す心配がない。
寝ている間も自動で情報を集めて
朝起きたらSlackに届いている！ 
/loop 24h   ← 24時間自動実行

⑭プログラミング不要：日本語だけでOK

  • プログラミングの知識は一切不要で、自然な言葉で伝えるだけで動く。
  • 非エンジニアでも、自分の欲しいツールを作成できる。
