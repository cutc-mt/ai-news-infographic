Claude Mythosが公開中止

📅 2026年4月18日 | 🎬 ギリギリ開発会議 by ボイスアップラボ

①Mythosは4月7日に完成したが公開見送り

  • AnthropicがClaude Mythosを4月7日に完成させたが、一般公開を見送った。
  • 理由は極めて高いセキュリティ能力により、悪用リスクが大きすぎると判断されたため。
  • 安全性が確認されるまで、一般ユーザーには一切公開されない。
モデルの能力が高すぎるがゆえに、
公開できないという前例のない事態！

②27年前のバグを発見した前例のない能力

  • MythosはOpenBSDの27年前のバグと、FFmpegの16年前のバグを発見した。
  • 前例のないペースで脆弱性を見つけ出し、人間の専門家でも発見できなかったものを特定。
  • 何十年もメンテナンスされてきた著名なOSSに潜む脆弱性を次々と暴いた。
人間が四半世紀見逃していたバグを
AIがあっさり見つけてしまった！

③大量サイバー攻撃を可能にする脆弱性を数千発見

  • Mythosが発見した脆弱性は数千件に上り、大規模なサイバー攻撃を可能にするレベル。
  • インフラ、金融、政府システムなど、あらゆる分野に影響を及ぼす可能性がある。
  • これが一般公開されれば、世界中のサイバー攻撃が激増する恐れがある。
数千の脆弱性が悪用されたら
壊滅的な被害が出る可能性がある！

④SWEベンチが13%向上、サイバーSecurityは83.1%に

  • SWE-benchスコアが13%跳ね上がり、80%に到達した。
  • サイバーセキュリティベンチマークは66%から83.1%に大幅向上。
  • 従来モデルを圧倒的に凌駕する性能を示している。
SWE-bench 80%は前代未聞のスコア！
セキュリティ分野での飛躍も凄まじい。

⑤Project Glasswingが始動：限定40組織のみ

  • Project Glasswingとして、限定された40組織のみに提供を開始。
  • 提供先はCrowdStrikeをはじめとする金融機関、クラウド企業など。
  • 厳格な審査を通過した組織のみがアクセスを許可されている。
防御側のトップ企業に限定配布で、
セキュリティ業界を先制強化する方針！

⑥AWS、Google Cloud、Azure経由でのみ利用可能

  • 利用可能なプラットフォームはAWS Bedrock、Google Cloud、Microsoft Azureのみ。
  • 承認された組織だけがこれらのクラウド経由でアクセスできる。
  • Anthropicの公式APIやWebインターフェースからの直接アクセスは不可。
大手クラウド3社を通じてのみ提供。
アクセス経路も厳格に管理されている。

⑦価格は従来モデルの約10倍

  • Mythosの利用価格は従来の約10倍に設定されている。
  • 圧倒的な能力に見合った価格として設定されたが、非常に高額。
  • 価格設定自体も、アクセス制限の一環として機能している面がある。
高額な価格も悪用を防ぐ
一つのバリアとして機能している！

⑧米国で緊急CEO会議が招集された

  • 米国で国務長官、FRB議長、Goldman Sachs、Citiなどが緊急会合。
  • 政府と金融業界のトップが一堂に会する異例の会議が開かれた。
  • AIの安全性と国家への影響について、極秘レベルで協議が行われたとみられる。
政府と金融の最高層が集まる会議。
それだけ事態が深刻だという証拠！

⑨バグバウンティパートナーにClaudeクレジットを付与

  • バグバウンティパートナーに対して、Claudeクレジットが提供される。
  • 責任ある開示（レスポンシブルディスクロージャー）を行った研究者に報酬。
  • 脆弱性を悪用するのではなく、正しい方法で報告することを奨励する仕組み。
正直に報告した研究者には
Claudeクレジットがもらえる仕組み！

⑩安全性が確認されるまで一般公開はされない

  • 一般公開の時期は未定で、安全性が確認された段階で段階的にリリース。
  • Anthropicは社内で徹底的なリスク評価を継続中。
  • Project Glasswingの成果を踏まえて、公開範囲を徐々に広げていく方針だ。
いつ公開されるかは誰にも分からない。
安全第一の慎重なアプローチが続く。

⑪AGIの意味：この技術を持つ国が圧倒的優位に

  • MythosレベルのAIはAGIに近い能力を持ち、保有国に圧倒的な優位性をもたらす。
  • 国家間の力関係を一変させる可能性がある技術として認識されている。
  • AIの軍事利用と同時に、国家の経済的優位性にも直結する問題だ。
これを持つ国と持たない国で
決定的な格差が生まれる可能性がある！

⑫他国のシステムの脆弱性を探る軍事利用の懸念

  • 政府や軍がMythosを利用して、他国のシステムの脆弱性を探る懸念がある。
  • サイバー戦争の武器として使われる可能性が議論されている。
  • AIの軍事利用に関する国際的なルール作りが急務となっている。
サイバー兵器として使われたら
国家の安全保障に関わる大問題！

⑬インフラへの潜在的な影響は計り知れない

  • 電力、通信、交通、金融など重要インフラの脆弱性が次々と見つかる可能性がある。
  • 社会を支える基盤そのものが攻撃対象になり得る。
  • 防御側も同じAIを使えるので、攻防のスピードが劇的に加速する。
電力や通信が止まったら
社会機能が麻痺してしまう！

⑭AI企業の責任と公開のジレンマ

  • 「強力すぎるAIを公開するべきか、隠すべきか」という根本的なジレンマがある。
  • Anthropicは安全第一のアプローチを選択し、段階的な公開を決断した。
  • 一方で、競合他社が同レベルのAIを公開する可能性もあり、判断は難しい。
公開すれば悪用され、隠せば
競争に遅れる。難しい選択だ。

⑮セキュリティ業界にとっては朗報でもある

  • 防御側がMythosを使えば、未知の脆弱性を先んじて発見・修正できる。
  • 攻撃される前に脆弱性を塞ぐ「先制防御」が可能になる。
  • セキュリティ業界全体のレベルが底上げされる効果が期待できる。
攻撃される前に直せるなら
むしろセキュリティは強化される！

⑯OpenBSDとFFmpeg：長年守られてきたOSSの実態

  • OpenBSDは27年間、セキュリティに特化したOSとして高い評価を得てきた。
  • FFmpegは16年以上にわたり、世界中で使われる動画・音声処理ライブラリ。
  • トップクラスのOSSでさえ見落としがあったという事実は衝撃的だ。
最も厳格なセキュリティ監査を受けてきた
プロジェクトにさえバグが潜んでいた！

⑰AIの能力が人間の専門家を超えた瞬間

  • サイバーセキュリティベンチマーク83.1%は、プロのセキュリティ研究者を凌駕する水準。
  • 人間が何週間もかけて見つけられないバグを、数分で発見する能力がある。
  • コードの規模が大きくなるほど、AIの優位性がより顕著になる。
人間の専門家が四半世紀見逃したバグを
AIが数分で見つける時代が到来！

⑱今後の展望：AIとセキュリティの新時代

  • Mythosのようなモデルが増える中で、国際的なAI安全規制の必要性が高まっている。
  • 政府、企業、研究者が協力して安全なAIの枠組みを構築する必要がある。
  • 危険性と有用性のバランスを取りながら、人類全体の利益になる方向を目指すべきだ。
AIの能力を正しく使えば
より安全な世界を作れるはず！
