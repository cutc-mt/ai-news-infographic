Claude Opus 4.7の評判、サム・アルトマン氏宅に火炎瓶、Google出遅れの理由

OpenAIがAnthropicを名指し批判、プラットフォーム競争への転換

2026年4月18日

① チャンネル登録者数1万人を突破

  • The WAVE TVが1万人の登録者に到達
  • コメント欄のレベルが高く、詳しい視聴者が多いことが特徴
  • 数字よりも質の高い視聴者との対話を重視する姿勢
1万人は1つのマイルストーン。これで「YouTubeやってます」と言える規模に！

② アルトマン氏宅に火炎瓶が投げつけられる

  • 夜中の3時にサム・アルトマン氏の自宅の玄関に火炎瓶が投げ込まれた
  • 家族には怪我がなかったが、1歳ほどの赤ちゃんがいる家庭での事件
  • アメリカ国内でAI企業に対する風当たりが強くなっているサイン
  • 日本ではまだAIに対する怒りは少ないが、時間差で影響が来る可能性
家族を持つ人間としては恐怖。AI失業が始まると日本でも風当たりが変わるかも。

③ Claude内にアプリ開発環境の画像が流出

  • Claude内にアプリ開発環境とみられる画像がX上に流出
  • CursorのようなIDE的な機能がClaude内に統合される可能性
  • ReplitやLovablrなどのパートナー企業が最大の被害者候補に
  • モデルを売る企業からプラットフォームへの転換が加速
iOSに機能が統合されてサードパーティアプリが消えた歴史と同じことが起きるかも。

④ OpenAIのCROがAnthropicを名指し批判

  • OpenAIの社内メモでAnthropicを「恐怖と制限の上に成り立っている」と批判
  • Anthropicの年間収益が39億から300億ドルに急増した件に触れる
  • 3月のClaudeの安定性の低さを「戦略的ミス」と指摘
  • OpenAIは秘密保持が甘く、社内メモがすぐにリークされる体質
ここまで攻撃的な内容の社内メモは珍しい。OpenAIの焦りが見える。

⑤ 競争はモデルからプラットフォームへ

  • 大企業でのLLM利用率はAnthropic 40% vs OpenAI 27%
  • Claude Codeが爆発的に人気で、Anthropicの勢いが圧倒的
  • OpenAIはFrontier Allianceでコンサル会社と提携して反撃を図る
  • Microsoft専用からAWSなど他クラウドへの展開も開始
「宗教のようにみんな熱狂している」とグリーンCEO。Claudeマニアが加速中！

⑥ Googleがエージェント型コーディングで出遅れる

  • 元Googleエンジニアの暴露: 20%がCursor止まり、20%は自力で書いている
  • 採用凍結で新しいカルチャーが入ってこないのが原因
  • Claude Codeは競合製品なのでGoogle内で使いづらい
  • IntelliJなどのツールも全面廃止へ。AIネイティブ環境への移行を推進
競合のClaude Codeが使えない → エージェント型開発に出遅れる。皮肉な構造！

⑦ トークン消費量が新しい生産性の指標に

  • Anthropicは1人当たり1000万〜1500万トークン/日が普通と主張
  • Metaではエンジニアのトークン使用量を社内で競わせる仕組み
  • 使用量の多いエンジニアにはトークンレジェンドの称号が付与
  • OpenAIではトークンは無制限。業界全体が「AIに使え」の方向へ
「給料の半分は遠く（トークン）に使え」——新しいエンジニアリングの常識が誕生！

⑧ Google幹部の反論と現実のギャップ

  • Demis Hassabis氏は「アクセス数を集めたいだけ」と完全否定
  • Google Cloud AIディレクターは「週1回以上エージェントを使うエンジニアが4万人以上いる」と反論
  • しかし「週1回」はハードルが低すぎて逆に使っていない証拠という反応
「週1回以上」って何それ？毎日使って当たり前の時代なのに...

⑨ Claude Opus 4.7の好評ポイント

  • コーディング性能が全モデル比で13%向上
  • 本番環境でのタスク解決率が3倍に向上
  • 出力前に自己検証するようになり、監督業務が減少
  • 画像認識の精度が3倍に。UIのスクリーンショットや図面に強い
  • 価格は4.6と同じでコスパが良い
「企業向け・開発者向けに手放しで任せられる方向に磨きがかかった」との高評価！

⑩ Opus 4.7の懸念・批判ポイント

  • GitHub Copilot経由だとトークン消費量が7.5倍に増加
  • API料金は据え置きでも、処理あたりのトークン量が2倍以上に
  • レスポンスが長くなり、よりビジネス調に変化
  • クリエイティブ作業で人間味が薄れたという感想も
賢くなった分、トークンをたくさん消費する。結局コストが増えるという悩み。

⑪ Opus 4.7でハルシネーション率がほぼ半減

  • Artificial Analysisの評価で幻覚率がほぼ半分に削減
  • より厳密に指示を守り、自己検証する能力が向上した結果
  • 4.6と4.7で得意分野が異なるため、使い分けが推奨されている
自己検証してくれるのは本当に嬉しい。信用できないのが一番疲れるポイントだから！

⑫ OpenAIのFrontier: 企業向けの反撃策

  • Frontier: AIエージェントを管理するインテリジェンス層
  • 有力コンサルティングファームとアライアンスを組み、企業向けに強化
  • 新たにFrontier Deploy Engineerを企業に駐在させる計画
  • ロードマップを頻繁に改定し、投資家から「焦っているのでは」との指摘も

⑬ 自立AIエージェントで生産性10倍〜100倍

  • 人間が介入するのを極力抑えてAIが自律的にループを回す
  • 既存事業はAIをツールとして、新規事業はAI中心のワークフローに
  • AI新聞の編集長はAIエージェント4〜5人でメディア企業と同等の成果
  • 新しい事業では完全にAIエージェント中心のワークフローを採用すべき
既存事業は人間中心、新規事業はAI中心。この二刀流が日本企業の現実的な答えかも！

⑭ ワークフローからデータフローへの転換

  • AIエージェント中心のワークフローではデータフローの設計が重要
  • AIにとって読みやすいデータ形式は人間には読みにくいことも
  • 既存の大量のパワポ資料の変換も必要になる
  • 新規事業はスケールしやすいAI中心の仕組みに適している
データの形式・ハンディが変わる。AIが扱いやすい構造への変換が鍵！

⑮ Anthropic vs OpenAI: 熱狂の違い

  • 半年前はGPTのリリースでみんな騒いだが、今はClaudeに熱狂
  • GPT 5.4が上がってもそれほど反応がないという変化
  • 半年で主導権が入れ替わったという鮮やかな変化
  • パワーユーザーを中心にClaudeの進化に注目する人が急増
OpenAIのCROがAnthropic批判の社内メモを書いたのも、この焦りの表れかも。

⑯ 日本企業への影響: ゆっくりだが確実に

  • アメリカほど急速ではないが、時間差でじわりじわりと影響が到来
  • 変化しない会社や人は苦しくなる方向へ進む
  • 部署移動などはすでに始まっている（銀行の窓口業務→営業など）
  • 自社メディアをAIエージェントで強化すれば、広告宣伝費を削減できる可能性
何もしない会社が業績を伸ばし続けるのは難しくなる。長期的には確実に影響あり。

⑰ Reasoning Modelは第3のパラダイム

  • Reasoning Modelは第3のパラダイムシフトとして位置づけ
  • 半年前までAI業界の人からも理解されていなかった
  • GPT 4.5の話ではなく、Reasoning Modelこそが大きな変化という主張
  • この理解の広まり方こそが業界のトレンドの変化を示している

⑱ 今週の大きな潮流: 3つの転換点

  • モデル競争からプラットフォーム競争へ（Anthropic vs OpenAI）
  • 人間中心からエージェント中心のワークフローへ（トークン消費が生産性の指標）
  • ツールとしてのAIから自立エージェントへ（新規事業はAI中心で）
既存事業はAIツール活用、新規事業はAIエージェント中心。この二刀流が正解かも！
