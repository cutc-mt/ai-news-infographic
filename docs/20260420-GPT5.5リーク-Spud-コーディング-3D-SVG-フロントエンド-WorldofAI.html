GPT-5.5「Spud」リーク — コーディング・3D・SVGの飛躍的進化

内部コードネーム「Spud」として開発中のGPT-5.5。推論・フロントエンド・3D・SVG生成が別次元へ

📅 2026年4月20日 | 🎬 WorldofAI

①GPT-5.5のコードネーム「Spud」

  • OpenAIが内部コードネーム「Spud」として開発中のGPT-5.5モデル
  • 今週からChatGPT内でA/Bテストが開始されている
  • Crest Pro AlphaというチェックポイントがGPT-5.5 Proを代表する可能性が高い
  • 一般ユーザーにも一部テストが始まっており、正式リリースが近いことを示唆
「Spud」ってジャガイモの意味だけど、GPT-6の「半分焼けたバージョン」という位置付けらしい！

②2つのモデル展開

  • GPT-5.5ベースモデルとGPT-5.5 Pro（拡張思考付き）の2種類が登場予定
  • Spudはバージョン5.5で、GPT-6の「半分焼けたバージョン」と位置付けられる
  • よりトークン効率が高く、出力も高速なのが特徴
  • ベースモデルとProモデルで用途に応じた使い分けが可能に
GPT-5とGPT-6の橋渡しモデル。それでも過去最大級の飛躍になるらしい！

③3つの改善: 推論・コスト・速度

  • ①より良い推論: 複雑な問題に対する論理的思考力が向上
  • ②低コスト: トークン単価が下がり、より経済的に利用可能
  • ③高速出力: Token per secondが向上してレスポンスが格段に速い
  • これまでのモデルジャンプと比べても最大級の飛躍と評価されている
推論力アップ、コストダウン、高速化の3拍子揃った improvement！品質も速度も両立している。

④Opus 4.7との比較

  • テストではOpus 4.7より良いと感じる場面もあった
  • ただしフライトシミュレーターの生成ではOpus 4.7の方が優れていた
  • モデルによって得意不得意が分かれる印象を受ける結果に
  • 一概にどちらが上とは言えず、タスクによって使い分けるのが重要
全面的に上回るわけではないけど、特定の領域では明らかに凌駕している。モデルの個性がはっきりしてきた！

⑤フロントエンド生成の進化

  • 画像からコードを生成する際の品質が劇的に向上
  • ランディングページの動的ムーブメントが以前より遥かに美しく出力
  • タイポグラフィや各種属性の再現度がGPT-5.4とは別次元
  • デザインの細部まで忠実に再現する力が格段に上がった
画像からのコード生成が別次元に進化。GPT-5.4とは比べものにならない品質らしい！

⑥Windows OSクローンを生成

  • GPT-5.5 ProでWindows 11風のブラウザOSを完全に生成
  • 全てのSVGアイコンが正確に生成されており、本物そっくり
  • Microsoft Edge、メモ帳、ペイント、設定などが忠実に再現
  • ダブルクリックでブラウザ内OSに遷移するインタラクティブな動作も実装
ブラウザの中にWindows 11が再現されるって何事！？SVGアイコンまで正確なのは驚き。

⑦Minecraftクローンを生成

  • テクスチャ付きのMinecraftクローンを一から生成
  • ブロック破壊アニメーション、インベントリ、洞窟システムを再現
  • 水流、鉱石、無限地形と水の生成も含まれている
  • ゲーム物理は不完全だが、驚異的な再現度と評価
AIに「Minecraftを作って」と言うだけでテクスチャ付きクローンが出来る時代…物理以外はほぼ本物！

⑧3Dジェネレーションの飛躍

  • Monicaのアパート（フレンズ）を3Dで完全に生成
  • 照明、レイアウト、オブジェクトが正確に配置されて本物の3Dプロジェクトのよう
  • 以前は壊れたり基本的なものしか生成できなかったのが劇的に改善
  • 3D空間の理解力が根本的に向上したことを示している
フレンズのセットを3Dで再現！照明から家具配置まで正確なのはモデルの空間理解力の証拠。

⑨太陽系ビジュアライゼーション

  • Three.jsで太陽系を完全にコーディングして生成
  • 各惑星の月、小惑星帯、スケール感がリアルに再現
  • 太陽からの照明、惑星運動の滑らかさも本格的
  • 間隔、軌道、3Dシーンの構造を理解していることが明確に分かる
太陽系を作ってと言うだけで、軌道も照明もスケールも正確に再現！空間の理解力が凄まじい。

⑩低ポリ3D生成の創造性

  • Three.jsでの低ポリ生成も非常に優秀な結果
  • 基本形状だけでなく、構図・カラー・シーンレイアウトに創造性がある
  • 自由にプロンプトすると「意図的でクリーン」な生成結果が得られる
  • 指示しなくても美しい構図を作れる力が備わっている
低ポリなのに芸術的！自分で構図を考えて創造的なシーンを作れるのが新しい。

⑪フライトシミュレーター

  • グランドキャニオン、スイスアルプス、エベレスト、マンハッタンなど複数環境を選択可能
  • GPT-5.5は3Dにおいて「指示を超えてシーンを演出する」力がある
  • マイクロマネジメントが不要になり、大まかな指示だけで高品質な結果が得られる
  • モデル自体が創造的な判断を下す力が備わった
「グランドキャニオンを飛んで」と言うだけで、地形も照明も雰囲気も自動で演出！指示が不要になった。

⑫SVGジェネレーションの向上

  • XboxコントローラーのSVG生成で、全てのボタンと構造が正確に描写
  • 猫のSVGも足、尻尾、目、ヒゲが正確に生成
  • 構図と対称性を理解しており、SVG出力が格段に向上
  • ベクターグラフィックスの品質が実用レベルに到達
SVGでコントローラーのボタンまで正確に描写できるって凄い！対称性の理解力が上がっている。

⑬ASCIIアート生成

  • Xbox 360コントローラーのASCIIアートを生成
  • アイコニックなコントローラーの構造、間隔、対称性が正確に再現
  • 100%正確ではないが、リファレンスとして非常に優秀なレベル
  • テキストベースの表現力もモデルの推論力向上を反映している
ASCII文字だけでコントローラーの形を再現！間隔や対称性まで計算されているのがすごい。

⑭推論能力の劇的な向上

  • このモデルが優れているのは何より「推論」の部分
  • 間隔や軌道、3Dシーンの構造を理解して、全体を構造化して生成
  • ただオブジェクトを投げるのではなく、文脈を理解した配置ができる
  • ユーザーのタスクに対してよりインテリジェントに対応できる
「推論力」こそがGPT-5.5の真価。空間や構造を理解して自ら考える力が備わった！

⑮モデルの思考パターンの変化

  • 以前は3Dで全てをマイクロマネジメントする必要があった
  • 今はモデルが「一緒に考えて、アイデアを押し進める」感覚
  • 指示以上のものを生成する力が明確に感じられる
  • 人間とAIの協調関係がよりクリエイティブな方向に進化している
「一緒に考える」感覚って今までなかった。指示する側から共に創る側へ。関係性が変わった！

⑯リリース時期の予測

  • 火曜日か木曜日にリリースされる可能性が高い（OpenAIがモデルを公開する傾向のある曜日）
  • ChatGPT内で既にA/Bテストが行われているので正式リリースは間近
  • A/Bテストの段階ということは、数日以内に一般公開される可能性が高い
  • OpenAIの過去のリリースパターンからも早いタイミングが予想される
すでにA/Bテスト中なら正式リリースはもうすぐ！火曜か木曜を要チェック。

⑰注意点: グラスモーフィズムの多用

  • 一部のUIコンポーネントでグラスモーフィズム（すりガラス効果）が多用される傾向
  • これはモデル特有の「癖」で、全ての生成に適用されるわけではない
  • デザインの好みが分かれるポイントで、すりガラス風エフェクトが過剰になる場合がある
  • プロンプトで調整すれば回避可能だが、デフォルトの挙動としては留意すべき点
「すりガラス効果」をかけすぎる癖があるけど、指示で調整できるから大きな問題ではない！

  • 推論力の向上、高速出力、低コストの3本柱で競合に挑む
  • SVG / 3D / フロントエンド生成における飛躍的進化が最大の魅力
  • Opus 4.7やGemini 3.1に匹敵するかそれ以上の可能性を秘めている
OpenAIの本気の反撃！推論・速度・コスト・生成品質の全てでトップレベルに。正式リリースが待ち遠しい！
