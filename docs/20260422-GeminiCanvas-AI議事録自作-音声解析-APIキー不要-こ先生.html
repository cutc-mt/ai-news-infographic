  • コード機能を使えば、HTMLやJavaScriptのアプリをその場で作成・実行できる。
  • プレビュー領域が用意されていて、作成したコードの動作をすぐに確認できる。
チャットだけでなく、本格的なコンテンツ制作環境がGeminiの中にある！

②PDF分割ツールを1分で作成する驚きの速度

  • Canvasに「PDF分割ツールを作って」と指示するだけで、リアルタイムで1分以内に完成する。
  • PDFファイルを読み込ませて、ページ単位で分割・抽出ができる。
  • 複雑なプログラミング知識がなくても、プロンプトだけで実用的なツールが作れる。
わずか1分！手作業で何時間もかかる作業が瞬時に完了する！

APIキーの設定が不要！誰でもすぐにAI機能をアプリに組み込める！

⑧疑似議事録アプリの仕組みを解明

  • 録音開始ボタンを押すと、ブラウザのマイクから音声をキャプチャする。
  • 30秒ごとに音声データを区切ってGeminiに送信する仕組み。
  • Geminiが音声を文字起こしして、結果をリアルタイムで議事録に反映する。
録音→30秒区切り→Gemini送信→文字起こし→議事録更新のループ！

⑨30秒区切りで送信する理由とメリット

  • 音声データを小刻みに分割することで、Geminiの処理負荷を抑える。
  • 長時間の録音を一括送信するとタイムアウトやエラーが発生するリスクがある。
  • 30秒間隔であれば、リアルタイム性を保ちつつ安定して処理できる。
  • 少しずつ議事録が更新されていく様子をリアルタイムで確認できる。
リアルタイム性と安定性のバランスをとるための最適な間隔！
