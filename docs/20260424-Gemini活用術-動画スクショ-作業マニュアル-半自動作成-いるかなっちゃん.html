Gemini活用術：動画からスクショ付き作業マニュアルを半自動作成

CanvasとGemでGoogle Workspace内完結の3ステップ手法

2026年4月24日

①課題：作業マニュアル作成の手間が膨大

  • スクリーンショットを取得して、説明文を作成して、並べ替えを行うという手作業の連続が必要になる。
  • 手順が多いマニュアルほど、作成に膨大な時間がかかってしまう。
  • 業務フローが変更されるたびに、スクショの撮り直しと説明文の更新が発生する。
マニュアル作成ほど面倒でサボりたくなる作業はない！何とか自動化できないか？

②Geminiの現状：プロンプト1発で全自動はまだ困難

  • Geminiはまだ完全エージェント的な動きには弱く、プロンプト1本で全自動というわけにはいかない。
  • しかし、工程を分けて指示すれば各ステップで高い精度を発揮する。
  • 3ステップに分割することで、実用的なレベルのマニュアルが半自動で完成する。
完全自動は無理でも、工程を分ければ驚くほどラクになる！

③全体像：3ステップで半自動マニュアル完成

  • ステップ1：Geminiに動画を分析させて、文章マニュアルを作成する。
  • ステップ2：Canvasのバイブコーディングで、スクショ抽出アプリを作成する。
  • ステップ3：文章と画像をGeminiに渡して、Word形式のマニュアルを完成させる。
  • すべてGoogle Workspace内で完結するので、外部ツールが不要。
文章作成 → スクショ抽出 → Word生成 の3段構えで攻める！

④ステップ1の準備：Geminiの設定と動画アップロード

  • Geminiに録画動画をアップロードして、分析の準備をする。
  • Canvas機能をONにして、ドキュメント作成モードを有効にする。
  • モデルはGemini Proを指定して、高い動画理解力を活用する。
設定はこれだけ！あとはプロンプトで指示を出すだけ！

⑤ステップ1のプロンプト：動画分析でマニュアル作成

  • 「動画を分析して、各操作ステップの作業名・作業内容・タイムスタンプを取得したマニュアルを作成して」と指示する。
  • Geminiの動画理解力が高く、操作ポイントを正確に書き出してくれる。
  • 各ステップが整理された構造化されたドキュメントが自動で生成される。
動画を見るだけでGeminiが勝手に手順をまとめてくれるのが超便利！

⑥タイムスタンプリストの抽出：次のステップへの仕込み

  • ドキュメントの最後に、タイムスタンプだけのリストを抽出するよう指示する。
  • このリストはステップ2のスクショ抽出でそのまま使う重要なデータになる。
  • 完成した文章マニュアルは、Googleドキュメントにエクスポートして保存する。
タイムスタンプリストが次のステップの鍵！ここを丁寧に用意しておこう！

⑦Gem化の推奨：頻繁に使うなら専用設定を保存

  • マニュアル作成を頻繁に行うならGem化がおすすめ。
  • Gemには目的・ルール・出力形式・トーンを詳細に設定しておく。
  • Gem化しておけば、次回以降は動画をアップするだけで、同じ品質のマニュアルが自動生成される。
一度設定すれば繰り返し使える！Gem化は時短の最大の味方！

⑧ステップ2：バイブコーディングでスクショ抽出アプリを作成

  • Canvas機能のバイブコーディングを使って、スクショ抽出アプリを作る。
  • プロンプトは1文だけ：「動画とタイムスタンプリストをアップしたらスクショしてダウンロードできるアプリを作って」
  • Webアプリとして完成して、ブラウザ上でそのまま動作する。
たった1文のプロンプトで専用ツールが完成するのがCanvasの威力！

⑨スクショ一括抽出：タイムスタンプリストで画像を取得

  • ステップ1で抽出したタイムスタンプリストを貼り付けて実行する。
  • 指定したタイムスタンプの画像が一括でダウンロードされる。
  • 手作業で再生→停止→スクショを繰り返す手間が完全にゼロになる。
何十枚ものスクショが一瞬で揃う！もう手動でコマ送りしなくていい！

⑩バイブコーディングの使い所：個人用には最適

  • 個人用のツールであれば、バイブコーディングは非常に有効な手段。
  • ただしミッションクリティカルな用途には慎重に検討が必要。
  • 生成されたコードの品質確認は忘れずに。エッジケースの動作に注意。
自分専用なら超便利！でも業務の重要システムには注意して使おう！

⑪ステップ3：文章と画像をGeminiに渡してWord完成

  • 文章マニュアル（Googleドキュメント）とスクショ画像をGeminiにアップロードする。
  • Geminiが両方を照合して、各手順に対応する画像を正しく挿入してくれる。
  • 最終的にWord形式のマニュアルとして出力される。
文章と画像を渡すだけで、正しい位置に画像が挿入されたマニュアルが完成！

⑫画像は1回に10枚まで：11枚以上は分割して送信

  • Geminiには1回あたり10枚までという画像の送信制限がある。
  • 画像が11枚以上になる場合は、分割して複数回に分けて送信する必要がある。
  • 分割送信しても、最終的なWord出力にはすべての画像が正しく反映される。
10枚超える場合は2回に分ければOK！ちょっとした手間だけど問題なし！

⑬裏側でPythonが動く：python-docxでWordを生成

  • Geminiの裏側でPython（python-docx）が実行されて、Wordファイルが生成される。
  • 1回目の実行ではPythonコードのみが出力されることがある。
  • そこで「内部的にPythonを実行してWordを作成して」と強く指示すると、2回目でWordが完成する。
1回目でコードだけ出たら、もう一言「実行して！」と追加で指示しよう！

⑭Word作成機能は非公式：自己責任で活用する

  • Google公式アナウンスなしの機能であり、正式にサポートされているわけではない。
  • 利用は自己責任で行う必要がある。
  • 今後のアップデートで仕様が変更される可能性もあるので、過度な依存には注意。
便利だけど公式機能じゃない！将来的に変わる可能性も頭の片隅に！

⑮ステップ3もGem化推奨：python-docxの指示を保存

  • ステップ3もGem化しておくことで、毎回同じ指示を書く手間を省ける。
  • Gemにはpython-docxでWordを作成するという指示をあらかじめ設定しておく。
  • 画像をアップするだけで即座にWord生成が始まるので、作業効率が大幅に向上する。
3つのステップすべてをGem化すれば、次回から爆速でマニュアルが作れる！

⑯Google Workspace内で完結：外部ツールが不要

  • 動画のアップロードからマニュアルの完成まで、すべてGoogle Workspace内で行える。
  • 外部の有料ツールや、追加のソフトウェアのインストールが一切不要。
  • データがGoogleの eco system内に留まるので、セキュリティ面でも安心。
社内でGoogle Workspaceを使っているなら、そのまま導入できる！

⑰高い再現性：3ステップ手法は誰でも同じ結果が出せる

  • 3ステップ手法は手順が明確で、誰でも同じように再現できる。
  • Gem化しておけば、チームメンバー全員が同じ品質のマニュアルを作成可能。
  • 特別なプログラミングスキルがなくても、プロンプトを書くだけで実行できる。
チーム内で共有すれば、全員がマニュアル作成の達人に！

⑱まとめ：Gemini × Canvas × Gemでマニュアル作成を革命する

  • 録画動画をGeminiに投げるだけで、スクショ付きの作業マニュアルが半自動で完成する。
  • 「文章作成」「スクショ抽出」「Word生成」の3ステップをGem化して定型化するのがポイント。
  • 外部ツールへの課金やインストールなしで、Google Workspaceだけで完結する。
  • マニュアル作成の工数が大幅に削減され、業務フローの変更にも強くなる。
面倒なマニュアル作成から解放される！Gemini活用の最佳解の一つ！
