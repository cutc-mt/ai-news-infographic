Claude × Blender MCP × GPT Image 2.0

AIイラストを3D立体化する新しいワークフローを徹底解説

2026年4月30日

①

①概要: AI × 3Dの新しいワークフロー

  • この動画ではClaude DesktopとBlenderを連携させて、2DのAIイラストを3Dモデルに変換する方法を解説しています。
  • まずGPT Image 2.0で2Dイラストを生成し、それをBlender MCPで立体化します。
  • Claudeが自然言語でBlenderを操作できるため、3Dモデリングの知識がなくても立体化が可能です。
  • AIツール同士を組み合わせることで、圧倒的に短い時間で3Dコンテンツを制作できます。
プロンプトだけで3Dモデルが作れる時代が到来！AIツールの連携が鍵です。
②

②MCPとは: Model Context Protocol

  • MCP（Model Context Protocol）は、AIモデルが外部ツールと通信するための標準プロトコルです。
  • Claude DesktopからBlender、Figma、GitHubなどの外部アプリを直接操作できるようになります。
  • AIとアプリの間でコマンドやデータを双方向にやり取りする仕組みを提供します。
  • Anthropicが推進するオープン規格で、様々なツールが公式コネクタを提供し始めています。
MCPはAIの「手足」を拡張する仕組み。ClaudeがBlenderを操作できるのもMCPのおかげ！
③

③必要なツール一覧

  • Claude Desktop — MCPコネクタ経由でBlenderに指示を送るAIクライアント。
  • Blender 5.1以降 — MCPサーバーアドオンに対応した3Dモデリングソフト。
  • GPT Image 2.0 — OpenAIの画像生成モデル。2Dイラストの作成に使用。
  • Blender MCPアドオン — Blender側でMCPブリッジサーバーを起動するアドオン。
  • インターネット接続 — Claude DesktopとBlender間の通信に必要。
全て無料で利用可能！Blenderはオープンソース、Claude Desktopも無料版でOKです。
④

④Blender MCPアドオンの入手先

  • BlenderのMCPアドオンはblender.org/lab/mcp-server/から入手できます。
  • 公式のBlender MCP ServerはBlender 5.1以降に対応しています。
  • Blender公式が提供するため、セキュリティ面でも安心して利用できます。
  • ダウンロードしたZIPファイルをそのままBlenderのアドオンとしてインストール可能です。
古いバージョンのBlenderでは動作しない場合があるので、必ず5.1以上を使いましょう！
⑤

⑤Blender側: アドオンのインストール手順

  • Step 1 Blenderを起動し、メニューからEdit → Preferencesを開きます。
  • Step 2 Add-onsタブを選択し、右上のInstall from Diskをクリック。
  • Step 3 ダウンロードしたMCPアドオンのZIPファイルを選択してインストール。
  • Step 4 インストール後、アドオンリストのMCP Bridge Serverにチェックを入れて有効化します。
インストール後はBlenderの再起動をおすすめします。サイドバーにMCPパネルが表示されます。
⑥

⑥Blender側: MCP接続の設定

  • Blenderのサイドバー（Nキーで表示）からMCPパネルを開きます。
  • Allow Online Accessにチェックを入れて、オンライン通信を許可します。
  • Start MCP Bridge Serverボタンをクリックして、サーバーを起動します。
  • ステータスが「Running」になれば、接続準備完了です。
Allow Online Accessを忘れると通信がブロックされます。必ず有効にしてください！
⑦

⑦Claude Desktop側: コネクタ設定

  • Claude Desktopを開き、設定メニューからCustomizeを選択します。
  • 「Connectors」または「コネクタ」セクションを開きます。
  • 利用可能なコネクタ一覧からBlenderを選択して有効化します。
  • これでClaude DesktopからBlenderにコマンドを送信できるようになります。
公式コネクタなので複雑な設定は不要。選択して有効化するだけです！
⑧

⑧接続確認とトラブルシューティング

  • 両方の設定が完了したら、Claudeに「Blenderのシーン情報を教えて」と入力してテストします。
  • 接続できない場合はファイアウォールの設定を確認してください。
  • Blender側でMCPサーバーを再起動することで解決する場合もあります。
  • ClaudeのPro/Maxプランだと、より安定したMCP接続が利用可能です。
最初はシンプルな指示でテストして、徐々に複雑な操作に挑戦しましょう！
⑨

⑨GPT Image 2.0で2Dイラストを生成

  • まずGPT Image 2.0（ChatGPTの画像生成機能）を使って、ベースとなる2Dイラストを作成します。
  • キャラクターやオブジェクトの正面・側面のデザインを意識して生成すると、3D化がスムーズです。
  • プロンプトの工夫: 「シンプルな形状」「くっきりした輪郭」などを指定すると立体化しやすいです。
  • 生成された画像をPNG形式で保存し、Blenderにインポートできるようにします。
画像はクリーンな背景（白や単色）で生成すると、Blenderでの作業が楽になります！
⑩

⑩2D画像をBlenderにインポート

  • Claudeに「この画像をBlenderにインポートして」と指示します。
  • 画像は参照画像としてBlenderのビューポートに配置されます。
  • 正面図や側面図として背景に配置することで、モデリングのガイドに使えます。
  • Claudeに画像の配置位置やサイズを自然言語で微調整できます。
「画像を正面ビューの背景に配置して」みたいに話しかけるだけでOK！
⑪

⑪自然言語プロンプトでBlenderを操作

  • Claudeに対して日本語や英語で直接指示を入力するだけでBlenderを操作できます。
  • 例: 「立方体を作って」「球体を追加して半径を2にして」など。
  • 材質や色の変更も可能: 「赤いマテリアルを適用して」
  • 照明の設定、カメラの配置、レンダリングまで全て会話形式で実行できます。 
// プロンプト例
「床となる平面を作成して、
その上に画像をテクスチャとして
貼り付けた立方体を配置して。
カメラを斜め上から見下ろす
角度に設定してレンダリングして」
⑫

⑫2Dイラストから3Dモデルへの変換

  • 画像をガイドにして、基本形状（立方体、球体、円柱など）を組み合わせて3Dモデルを構築します。
  • Claudeに「この画像を参考に3Dモデルを作って」と指示すると、自動的にモデリングを開始します。
  • 複雑な形状は押し出しや回転などの操作を組み合わせて作成します。
  • テクスチャとして元の画像を貼り付けることで、イラストの雰囲気を3Dでも再現できます。
完璧な3Dモデルでなくても、画像をテクスチャとして貼ればそれっぽく見えます！
⑬

⑬マテリアルとテクスチャの設定

  • マテリアルを設定することで、オブジェクトに色や質感を与えます。
  • 元のAIイラストをテクスチャとして適用することで、2Dの雰囲気を3Dに反映できます。
  • Claudeに「光沢のあるマテリアルにして」や「金属质感にして」などと指示可能です。
  • 粗さ（Roughness）や金属度（Metallic）などのパラメータも会話で調整できます。
「もっと光らせて」「艶を消して」みたいに直感的な指示ができるのが魅力！
⑭

⑭ライティングとカメラの設定

  • 照明は3Dシーンの見た目を大きく左右します。Claudeに「光を追加して」と指示するだけ。
  • ポイントライト、スポットライト、HDRI環境光など、用途に応じて使い分けます。
  • カメラの画角や位置も自然言語で調整。「斜め上から見下ろす角度にして」など。
  • 「夕暮れっぽい雰囲気にして」「スタジオライティングにして」のような雰囲気指定も可能です。
ライティング次第で同じモデルでも全く違う印象に！プロ顔負けの演出も夢じゃない。
⑮

⑮レンダリングして完成イメージを出力

  • 全ての設定が完了したら、Claudeに「レンダリングして」と指示して最終画像を出力します。
  • EEVEE（リアルタイム）とCycles（高品質）の2つのレンダリングエンジンが選べます。
  • 解像度やサンプリング数も会話形式で指定できます。
  • 出力形式はPNG、JPEGなどに対応。保存先も自由に指定可能です。
Cyclesを使うとフォトリアルな仕上がりに！ただしレンダリング時間は長くなります。
⑯

⑯実践例: 完成までのワークフロー

  • Step 1 GPT Image 2.0でキャラクターの2Dイラストを生成する。
  • Step 2 画像をClaude Desktopにアップロードして、Blenderにインポートを指示。
  • Step 3 Claudeに「この画像を参考に3Dモデルを作って」と指示。
  • Step 4 マテリアル、ライティング、カメラを自然言語で調整。
  • Step 5 レンダリングして完成！全工程を会話形式で実行できます。
従来なら数時間〜数日の作業が、AIとの対話だけで数十分で完成するかもしれません！
