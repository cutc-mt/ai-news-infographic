第1章: 概要 — AI×3D制作の新しい世界

① この動画のテーマ: 2Dイラストを3D立体化する新手法

この動画は、Claude × Blender MCP × GPT Image 2.0の3つのツールを組み合わせて、2DのAIイラストを立体的な3Dモデルに変換する方法を徹底解説する。従来ならBlenderの専門知識が必要だった3Dモデリング作業を、AIの力で誰でも実行可能にする革新的なワークフローを紹介している。

② 全体のワークフロー: 3ステップで3D立体化

3D立体化はシンプルな3ステップで実現できる。まずGPT Image 2.0でプロンプトから2Dイラストを生成し、次にClaude DesktopとBlender MCPを連携させてBlenderをAIで操作できる状態にする。最後にClaudeに自然言語で指示を出して、2Dイラストをベースに3Dモデルを生成する。

③ 誰でも使える: プログラミングや3D知識は不要

このワークフローの最大の魅力は、プログラミングの知識やBlenderの操作経験が一切不要な点だ。Claudeとの会話（自然言語）だけで3Dモデリングができるため、クリエイターやデザイナー、初心者でも直感的に3D制作に取り組むことができる。AIが間の操作を全て担ってくれる。

第2章: GPT Image 2.0で2Dイラストを生成

④ GPT Image 2.0とは: テキストから高品質な画像を生成するAI

OpenAIがリリースした画像生成AIGPT Image 2.0は、テキストプロンプトから写真のようなリアルな画像やアニメ風のイラストを高品質で生成できる。ChatGPT内では「ChatGPT Image 2.0」として利用可能で、有料プランユーザーは思考モードを使ってより精度の高い画像を作成できる。

⑤ 3D化に適したイラストのプロンプト作成

3D立体化を成功させるためには、プロンプトの書き方にコツがある。キャラクターの全身が見える構図、シンプルな背景、はっきりとしたシルエットなどが立体化しやすい。正面を向いたポーズや、各パーツの形が明確にわかるデザインを選ぶと、Blenderでの3D変換がスムーズに進む。

⑥ 生成した画像をダウンロードして準備完了

ChatGPTで生成した画像をダウンロードしてPCに保存する。この画像が後にBlenderで3D立体化のベースとなる。PNG形式で保存しておくと、背景の透過処理などにも対応しやすく、後工程での扱いが便利になる。高解像度（2Kなど）で出力すると、テクスチャとしても活用できる。

第3章: Claude Desktop × Blender MCPの導入設定

⑦ Claude Desktopアプリのインストール

まずClaudeの公式サイト（claude.ai）からデスクトップアプリをダウンロードしてインストールする。Claude DesktopはAnthropicが提供するPC用のClaudeアプリで、MCP（Model Context Protocol）に対応しており、外部ツールとの連携が可能だ。インストール後はClaudeアカウントでログインする。

⑧ Claude DesktopでBlenderコネクタを追加

Claude Desktopの設定画面を開き、「Customize」→「コネクタ」→「+」マークをクリックする。「コネクタを参照」を選択し、検索窓に「Blender」と入力してBlenderアイコンを選択。「インストール」ボタンを押せばClaude側の設定は完了。Anthropicが公式リリースしたコネクタなので安心感がある。

⑨ Blender側のMCPアドオンをインストール

次にBlender側の設定を行う。Blender公式サイトのLabページ（blender.org/lab/mcp-server/）からMCPサーバー統合用の専用アドオンをダウンロードする。注意点として、Blender 5.1以降でのみ動作するため、事前にBlenderを最新版にアップデートしておく必要がある。

⑩ Blenderでオンラインアクセスを許可してサーバー起動

アドオンをインストールしたら、Blenderのプリファレンス → システムから「Allow Online Access」にチェックを入れる。次にMCPアドオンの設定に戻り、「Start MCP Bridge Server」をクリックすると、Claude Desktopとの通信ブリッジが起動して連携準備が完了する。

第4章: ClaudeでBlenderを操作して3D立体化

⑪ 自然言語でBlenderを操作: Claudeとの会話で3D制作

Claude Desktopのチャット画面でハンマーアイコン（Blender MCPツール）が表示されれば連携成功。あとは日本語で「この画像を参考に3Dモデルを作って」のように自然言語で指示を出すだけ。Claudeが指示を理解してPythonコードを生成し、Blenderがそれを実行して3Dモデルを作成する。

⑫ 2D画像から3D立体化: Claudeに画像を読み込ませて指示

GPT Image 2.0で生成した2DイラストをClaudeにアップロードし、「このキャラクターを3D立体化して」と指示する。Claudeは画像を解析して形状を理解し、Blender上で3Dメッシュを自動生成する。フラットな2Dイラストが立体的な3Dモデルに変換されるプロセスは非常にスムーズだ。

⑬ マテリアルとテクスチャの適用

3Dモデルが生成されたら、Claudeに「肌の質感や服のテクスチャを適用して」と指示してマテリアルを設定する。生成した2DイラストをテクスチャとしてUVマッピングして貼り付けることも可能だ。光の設定や影の表現もClaudeの指示で調整でき、リアルな質感を再現できる。

⑭ カメラとライティングの調整で完成度を向上

Claudeに「カメラを少し下から見上げる角度にして」や「スタジオライティングを設定して」などの指示で、カメラアングルと照明を自由に調整できる。3Dモデルの見栄えはライティングで劇的に変わるため、Claudeとの対話を通じて理想の画づくりを行うことができる。

⑮ 反復改善: プロンプトの微調整で理想の3Dに

一回の指示で完璧な結果が出ない場合でも、Claudeとの対話を繰り返すことで徐々に理想に近づけていく。「顔をもっと丸くして」「体のプロポーションを調整して」など、具体的な修正指示を出すことで、ClaudeがBlender上でモデルをリアルタイムに修正してくれる。この反復プロセスがAIならではの強みだ。

⑯ レンダリング出力: 高品質な3D画像をエクスポート

3Dモデルが完成したら、Claudeに「このシーンをレンダリングして」と指示するだけで、Blenderのレンダリングエンジンを使って高品質な画像が出力される。最終的な3D立体化されたイラストは、SNS投稿やポートフォリオなど、様々な用途で活用できる。
