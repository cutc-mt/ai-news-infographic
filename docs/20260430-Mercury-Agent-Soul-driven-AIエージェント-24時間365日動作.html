Soul-driven AIエージェント — あなたの「第二の脳」を24時間365日で動かすオープンソースツール

①Mercury Agentとは: Soul-driven AIエージェント

Soul-driven（魂駆動）の設計思想で作られたオープンソースのAIエージェントです。CLIとTelegramの両方で動作し、24時間365日バックグラウンドであなたのタスクをこなします。単なるチャットボットではなく、人格（Soul）を持った自律的なパートナーを目指しています。

②爆速セットアップ: たった30秒で起動

インストールはたった1行のコマンドで完了します。Node.jsがあればすぐに始められます。

npx @cosmicstack/mercury-agent

追加のインストールや複雑な設定ファイルの準備は不要です。コマンドを実行するだけで、すぐに対話型のセットアップウィザードが立ち上がります。

③Soul-driven設計: エージェントに人格を与える

Mercury Agentの最大の特徴は、エージェントの人格をMarkdownファイルで定義できることです。4つのファイルで性格を構築します。

soul.md + persona.md + taste.md + heartbeat.md

soul.mdはアイデンティティ、persona.mdは対話スタイル、taste.mdは好み、heartbeat.mdは自律行動のルールを定義します。

④Second Brain: 10種類の記憶型で「第二の脳」を実現

SQLite + FTS5全文検索をベースにした強力な記憶システムです。会話から自動的に事実を抽出し、10種類の記憶型に分類して保存します。

identity（自己認識）
preference（好み）
goal（目標）
project（プロジェクト）
habit（習慣）
decision（決定）
constraint（制約）
relationship（関係性）
episode（出来事）
reflection（振り返り）

⑤自動事実抽出: 会話から賢く記憶を蓄積

各会話の終わりに、0〜3個の事実を自動的に抽出します。抽出された事実には3つのスコアが付与されます。

📊 信頼度スコア
情報の確実性を評価
⭐ 重要度スコア
記憶の優先度を判定
⏳ 耐久性スコア — 記憶をどれくらい長く保持するか

これにより、無駄なノイズを排除して本当に重要な情報だけを長期記憶に残します。

⑥パーミッションシステム: 安全第一の設計

エージェントが暴走しないよう、多層的なセキュリティが標準搭載されています。危険なコマンドはデフォルトでブロックされます。

  • sudoコマンドの実行を自動ブロック
  • rm -rf / のような破壊的コマンドを阻止
  • フォルダスコープ制限でアクセス範囲を限定
  • 承認フローで重要操作にユーザー確認を要求
🔒 Ask Me
都度ユーザーに確認
🔓 Allow All
信頼済み操作は自動許可

⑦トークン予算管理: APIコストを智能的に制御

日次トークン予算を設定して、API利用料の無駄遣いを防ぎます。予算の70%を超過すると、自動的にコンテキストをコンパクト化して節約します。

70%
自動コンパクト化閾値
/budget
予算確認コマンド

予算の使用状況は /budget コマンドでいつでも確認できます。長時間のバックグラウンド動作でも、コストの心配がいらない設計です。

⑧デーモン化: 24時間365日のバックグラウンド常駐

Mercury AgentはOSの起動時に自動で立ち上がるデーモンとして動作します。クラッシュしても自動的にリスタートする堅牢性を備えています。

  • バックグラウンド常駐でリソースを最小限に消費
  • クラッシュ時の自動リスタート（指数バックオフ）
  • 最大10回/分まで再試行、安定したら通常動作に復帰
  • OSブート時の自動起動に対応

⑨クロスプラットフォーム: 3大OSに完全対応

管理者権限が不要で、すべての主要OSで動作します。ユーザーレベルの権限でインストールと運用が可能です。

🍎 macOS
LaunchAgentで自動起動
🐧 Linux
systemdサービスで管理
🪟 Windows — Task Schedulerで自動起動

⑩CLIストリーミング: 快適なターミナル体験

CLIでの動作も一流のストリーミング体験を提供します。生成中のテキストがリアルタイムに表示されます。

  • カーソル位置のセーブとリストアでUIを維持
  • Markdownのリアルタイム再描画に対応
  • 長い出力でも途中から追従可能
  • Ctrl+Cで安全に中断・再開ができる

ターミナルで作業しながら、AIと自然な対話ができます。

⑪Telegram統合: スマホからエージェントを操作

Telegram Botとして動作し、スマホからどこでもエージェントにアクセスできます。ペアリングコードによる認証でセキュアに接続。

  • ペアリングコード認証で安全な接続
  • admin / memberの2段階ロール管理
  • プライベートチャットのみ対応（安全設計）
  • HTMLフォーマット + 編集可能ストリーミング

出勤中や外出先でも、Telegramを開けばエージェントが応えてくれます。

⑬Cronスケジューリング: 定期タスクを自動化

cron式のスケジューリングで定期的なタスクを自動実行します。schedules.yamlに永続化されるので、再起動後も設定が維持されます。

  • 標準cron式で柔軟なスケジュール設定
  • delay_secondsでワンショット遅延実行にも対応
  • schedules.yamlに自動保存で再起動後も継続
  • エージェントの自律的なタスク管理を可能に

朝のニュース収集や定期的なリマインダーなど、繰り返し作業をエージェントに任せられます。

⑭拡張スキル: Agent Skills仕準拠で無限の可能性

Agent Skills仕様に準拠した独自スキルを追加できます。コミュニティで作られたスキルも利用可能です。

スキル = SKILL.md + スクリプト + 設定ファイルの組み合わせ。標準化されたフォーマットで作成・共有が簡単。

メールの定期チェック、Webスクレイピング、ファイル整理など、用途に合わせて自由に拡張できます。公式の3種類のスキルツールで管理も簡単です。

⑮主要コマンド一覧: 直感的な操作体系

Mercury Agentはシンプルで覚えやすいコマンド体系を採用しています。

mercury up        # エージェントを起動
mercury doctor     # 環境の診断
mercury status     # 現在のステータス確認
mercury logs       # ログの表示
mercury upgrade    # アップデート
mercury telegram approve  # Telegram認証を承認
mercury telegram reject  # Telegram認証を拒否
/budget                   # トークン予算の確認

どのコマンドも直感的な名前で、初見でも使いやすい設計です。

Mercury Agentは様々なシーンで活躍します。24時間動き続けるエージェントだからこその使い方があります。
