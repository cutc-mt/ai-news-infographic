🤖 Claude × Fusion — クリエイティブワークの未来

Anthropic × Autodesk — ClaudeがFusion 360を直接操作して設計までこなす時代が到来

📅 2026年4月28日 発表
🚨#01

歴史的提携の発表

2026年4月28日、AnthropicのClaudeとAutodesk Fusionが公式連携。AIがCADソフトを直接動かして設計を行える世界が幕を開けた。

🌐#02

複数ツール一斉コネクタリリース

AnthropicはAdobe、Blender、Autodeskなど主要クリエイティブツール向けコネクターを一斉リリース。Fusionはその中の1つ。

🧠#03

AIの哲学：人のセンスは置き換えない

Anthropicの方針は明確。AIは人のセンスや想像力を置き換えるのではなく、面倒な反復作業を肩代わりしてアイデアの確信に集中できる環境を作る。

✨#04

アイデアが止まらない世界

Autodeskは今回の連携を「アイデアが止まらずに完成まで届く世界」への第一歩と位置づけ、言葉で設計して印刷まで届ける入り口が見えてきた。

🔌#05

MCPが連携の基盤技術

コネクターの仕組みはMCP（Model Context Protocol）。ClaudeがFusionの中で動くための「共通の言葉」であり、公式対応で連携が劇的に簡単になった。

⚡#06

コネクターセットアップは超簡単

Fusion側：環境設定→一般API→MCPサーバーをオンにするだけ。Claude側：コネクターズからFusionを検索してインストール、権限を全許可。

🖥️#07

Cloud Desktopアプリ必須

連携にはAutodesk Cloud Desktopアプリが必要。ブラウザ版のCloud AIでは接続できない。事前にインストールしておくことが条件。

🔺#08

実演①：言葉でピラミッド生成

「底面10cm×10cm、高さ10cmのピラミッドを作って」と普通の日本語で指示。Claudeが即座に3Dモデルを生成。シンプルな形は一瞬で完成。

🤝#09

実演②：人間が作ったモデルをAIが改良

人が作ったくぼみのあるマス（24本の辺あり）の全辺にフィレットを入れる作業をClaudeに依頼。面倒な反復作業をAIが完璧に実行。

📐#10

プロンプトのコツ：寸法まで明示

「全部の辺を丸くして」だけでは不十分。「外側は3mm、内側は1mm」など寸法を明示することでClaudeが正確に動ける。具体性が鍵。

🐕#11

実演③：画像から3Dモデル化に挑戦

犬のバッジ画像を渡してFusionに同じ形を作らせたが、非幾何学的なものはまだ難しい。画像→3Dモデルの変換は現時点では届かない。

⚖️#12

AIの得意不得意を見極める

何をAIにやらせて何を人がやるかが重要。Claudeは万能ではなく、苦手なことを知っておく方が上手な使い方につながる。適材適所で活用する。

🎹#13

実演④：3×3キーパッド設計に挑戦

複雑なものを作る時はいきなり作らせず、まず構造と仕様をしっかり伝える。ボトムプレート・スペーサー・キーマウントプレートの三層構造を指定。

❓#14

「推測しないで質問して」が黄金ルール

「不明な点は推測しないで全て質問して確定してください」— この指示でClaudeが設計意図を読み取り、不明点を質問。対話を経て最適な設計が実現する。

🏗️#15

質疑応答の後に一気に設計開始

不明点が解消されるとClaudeがどんどん作り始める。ボトムプレート、スペーサー、キーマウントプレートのフレームが完成。対話型設計の威力を実証。

🎨#16

色付けもClaudeにお任せ

完成予想図としてキースイッチとキーキャップも別コンポーネントとして作成。プレートは赤、ベースはネイビーブルー、キーキャップは青みがかったグレーに着色。

📸#17

レンダリングはまだ課題あり

レンダリング自体はClaudeがうまく実行できず。やり方をClaudeに教えてもらい、手動で操作することでレンダリング到達。教える側にもなれるAIの側面。

🚀#18

まとめ：3Dモデリングの革命が始まった

面倒な反復作業はAIに、アイデアの確信は人が握る。言葉で設計して印刷まで届く入り口が見えてきた。Claude × Fusionは3Dモデリングにおける革命的な第一歩。
