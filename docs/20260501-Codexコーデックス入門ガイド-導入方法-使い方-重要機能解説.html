第1章：Codexの概要

①

Codexとは：OpenAIのAIエージェント

Codexは2025年5月にリリースされ、同年10月に一般提供が開始されたOpenAIのAIエージェントです。アプリやウェブサイトなどのサービス構築からバグ修正まで、開発作業の自動化を手伝ってくれます。エンジニアだけでなく、プログラミングに詳しくない非エンジニア層からも高い支持を獲得しています。

OpenAI AIエージェント 2025年リリース
②

4つのインターフェースで利用可能

Codexは4つの形式で利用できます。デスクトップアプリ（Windows / Mac対応）は最も推奨される形式で、ほぼ全ての機能を利用可能。その他にCLI（コマンドライン）、VS Code等のIDE拡張機能、Webブラウザ版が用意されています。

デスクトップアプリ CLI IDE拡張 Webブラウザ
③

最新AIモデル「GPT-5.5」を搭載

2026年5月時点で、Codexは最新AIモデルのGPT-5.5を使用できます。フォームの右下からGPT-5.5やGPT-5.4などのモデルを選択可能です。「高速」モードを選ぶと動作が高速になりますが、通常モードよりも使用量を多く消費する点に注意が必要です。

GPT-5.5 GPT-5.4 高速モード
④

プラグインでCodexを拡張

デスクトップアプリではプラグイン機能を使って、Codex内で動く様々なアプリを数クリックでインストールできます。プレゼン資料の生成、Webブラウザの操作、動画生成に特化したプラグインなどが日々公開されており、自分でプラグインを作ることも可能です。

プレゼン生成 ブラウザ操作 動画生成
⑤

GPT Image 2.0を直接利用可能

CodexではOpenAIの超高性能な画像生成AIモデル「GPT Image 2.0」を直接使えます。アプリやウェブサイトを構築する際に、別途画像生成AIモデルを用意する必要がありません。デザインの品質が向上し、視覚的なコンテンツ制作がより簡単になります。

GPT Image 2.0 画像生成 統合環境
⑥

CodexとClaude Codeの違い

Codexと比較されるのがAnthropicのClaude Codeです。日本では2026年に入ってClaude Codeが先行して人気になりましたが、OpenAIは毎週のようにCodexに機能を追加して猛追中。MCP、スキル、フック、サブエージェントなどの基本機能は、Claude Codeとほぼ同じものが使えます。

Claude Code Anthropic 猛追中

料金とプラン比較

⑦

ChatGPTの料金プラン

Codexを使うにはChatGPTのサブスクリプションに加入します。月額1,400円のプラン、月額3,000円のプラスプラン、月額16,800円と3万円のプロプランがあります。プロプランは本格的なコーディングを行う開発者向けで、非エンジニアであればプラスプランでほぼ全ての機能を利用可能です。

¥1,400/月 ¥3,000/月 ¥16,800/月 ¥30,000/月
⑧

Claude Codeの利用制限問題

Claude Codeは2026年にユーザー数が急増した影響で、モデルの利用制限が以前より厳しくなっています。特に最新の上位モデルであるClaude Opus 4.6やOpus 4.7を使う場合、月額22ドルのプロプランでも数回使っただけで上限に達してしまうケースもあります。

利用制限厳格化 Opus 4.6/4.7 $22/月
⑨

Codexの使用制限とリセット

Codexではチャットするたびに使用量を消費し、制限まで使うとクレジットを購入しないと一時的に使えなくなります。使用制限は5時間と1週間の2つがあり、設定の「使用状況」から確認可能。3〜4月にかけて定期的にリセットされ、ユーザー1,000万人突破まで100万人増えるごとにリセットを予告しています。

5時間制限 1週間制限 定期リセット

第2章：Codexの使い方

⑩

① デスクトップアプリの導入方法

Codexの公式サイトからダウンロードボタンでアプリをダウンロード＆インストール。アプリを立ち上げたらChatGPTでサインインボタンを押してログインするだけ。右側のフォームからパソコン内のフォルダーを指定して、チャット経由でアプリやウェブサイト、スライド資料を生成できます。

Windows MacOS 公式DL
⑪

② CLI（コマンドライン）の使い方

Codexはターミナル（黒い画面）からも起動できます。これがCLI（コマンドラインインターフェース）です。公式ドキュメントのクイックスタートページにあるnpmインストールコマンドをコピーして、コマンドプロンプトで実行するだけ。その後「codex」と入力するとChatGPTへのサインインが求められ、完了すれば使えるようになります。

npm install -g @openai/codex
CLI npm ターミナル
⑫

③ IDE拡張機能での使い方

VS Codeなどの統合開発環境（IDE）に拡張機能としてCodexをインストール可能。VS Codeをインストール後、拡張機能の部分でCodexを追加するだけでVS Code内で立ち上げて使えます。また、VS Codeベースで作られているCursorやWindsurfなどのAIコーディング支援アプリでも同様に利用できます。

VS Code Cursor Windsurf
⑬

④ Webブラウザ版の使い方

ChatGPTのWebブラウザー版からもCodexを立ち上げ可能です。インターネット上のクラウド環境でCodexを使って作業を行います。事前にGitHubに登録しておく必要があり、登録後にWebブラウザー版ChatGPTの画面左側にある「Codex」部分を押した先の画面で利用できます。

Webブラウザ GitHub クラウド環境
⑭

基本的な操作の流れ

Codexの操作のざっくりとした流れは以下の通りです。まずパソコンにプロジェクトフォルダーを作り、デスクトップアプリを立ち上げてフォームの下からフォルダーを設定。チャットに日本語でアプリやウェブサイトの生成をお願いすると、フォルダー内に成果物が出力されます。追加の指示で修正や追加も可能です。

① フォルダー作成 ② フォルダー設定 ③ チャットで指示
⑮

プランモードと権限設定

フォームのプランモードにチェックを入れると、コンテンツを作る前に計画を立ててから作業を進めることができます。また「カスタム」を選ぶと、どれくらいの頻度で人間による承認が必要かといった権限設定を反映可能。「フルアクセス」を選択するとCodexが自律的に実行しますが、慣れるまでは使わない方が無難です。

プランモード 権限設定 フルアクセス注意

第3章：Codexの基本機能

⑯

AGENTS.mdで好みや規則を記憶

CodexにはAGENTS.mdという指示書的なファイルが存在します。会話を始める際に自動で読み込まれ、プロジェクトの規則やユーザーの好みを覚えさせることができます。作業フォルダーの直下に置くのが一般的で、例えば「質問には必ず関西弁で返す」という指示を入れておくと、以降の回答が関西弁で返ってきます。

AGENTS.md 自動読み込み 指示書
⑰

MCP・スキル・フック・サブエージェント

MCPは外部ツールやサービス（Gmail、Googleカレンダー、Notionなど）と連携する仕組み。スキルはCodexに高品質なデザインやデータ分析ができるマニュアルを渡す仕組み。フックは作業終了時に通知を飛ばすタスクを必ず実行する機能。サブエージェントは特定タスクに特化したAIアシスタントです。

MCP スキル フック サブエージェント
⑱

ユーザー1,000万人突破と今後の展望

Codexは3月から4月にかけてユーザー数が1,000万人を突破しました。OpenAIは100万人増えるごとに使用制限をリセットすると予告しており、今から始める人にとっては良いタイミングです。デスクトップアプリの機能はClaude Codeを上回り、UIも優れている点も大きな魅力。非エンジニアにも使いやすいのが最大の強みです。
