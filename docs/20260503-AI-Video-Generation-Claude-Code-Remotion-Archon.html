AI VIDEO GENERATION WORKFLOW

AI Video Generation Workflow

Claude Code + Remotion + Archon
Cole Medin
URLを1つ渡すだけで完全な動画が自動生成されるワークフローを実況解説。Hacker Newsの記事やWebページから、シーン構成・ナレーション・音楽・効果音・TypeScriptコード生成・品質チェックまで、14ステップのプロダクショングレードAIパイプラインをライブ公開。

入力はURLだけ

ワークフローの入力はたった1つのURL。
Hacker Newsのストーリー、記事、製品ページなど任意のURLを渡すだけで、完全な動画が自動生成される。手動作業はゼロ。

14ステップのパイプライン

Archon上で動く14ステップの自律ワークフロー。
ソース取得→シーン計画→ナレーション生成→音楽/SFX→TypeScript構成→品質チェック→自動修正→MP4レンダリングまで完全自動。

Claude Codeが主役

プランニング、ビルド、QAの全工程でClaude Codeが活躍。
シーン構成の設計からTypeScriptコードの記述、品質チェックまで、Claudeが自律的に実行する次世代の開発フロー。

Remotionで動画生成

Reactベースの動画フレームワークRemotionを使用。
TypeScriptで動画コンポジションを記述。プログラマティックにトランジション、動的デュレーション、プロパーなフックを実装。

39ルールのベストプラクティス

Remotion側には39のベストプラクティスルールをスキルとして注入。
生成されたコンポジションが「AIっぽい」見た目にならないよう、リアルなトランジションと動的演出を保証する独自ルールセット。

Cartesiaで音声生成

ナレーションはCartesiaが担当。
高品質なTTS音声でナレーションを自動生成。リアルな音声品質でプロダクションレベルの仕上がりを実現。

ElevenLabsで音楽&SFX

背景音楽と効果音はElevenLabsが生成。
AI生成のBGMとサウンドエフェクトで、動画にプロフェッショナルな音響体験を追加。手動で音楽を探す必要なし。

Archon - ワークフローエンジン

全体を統括するオーケストレーションプラットフォーム。
14ステップのAIパイプラインを管理。各ノードの実行、エラーハンドリング、リトリーを自動化する本格的なワークフローエンジン。

ソースフェッチング

URLからコンテンツを自動取得する最初のステップ。
Webページの本文、画像、メタデータを自動抽出。Hacker Newsの記事、ブログ、製品ページなど様々なフォーマットに対応。

シーンプランニング

Claudeがコンテンツを分析してシーン構成を自動設計。
記事の構造を理解し、最適なシーン数、各シーンの内容、遷移順序をAIが自律的に決定。映像的なストーリーテリングを自動生成。

品質チェック & 自動修正

生成後に自動QAパスを実行。
クリティカルな問題を自動検出・修正。品質基準を満たすまで反復的に改善。人間が介入せずにプロダクション品質を保証。

MP4レンダリング

最終ステップでMP4動画を出力。
Remotionのレンダリングエンジンで高品質なMP4を生成。すべてのシーン、トランジション、音声が統合された完成品を出力。

ライブ実装配信

このワークフローを初めてライブで実行。
すべてのノードをリアルタイムでウォークスルー。プロダクショングレードのAIワークフローがどのように構築されているかをEnd-to-Endで公開。

AIスロップを回避

生成動画が「AIっぽく」ならないよう39ルールのスキル注入。
リアルなトランジション、動的デュレーション、プロパーなフックを使用。単なるAI生成物ではなく、本格的な映像作品に仕上げる工夫。
