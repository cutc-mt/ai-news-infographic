OpenAIのCodex Appに新機能「ペット機能」が登場！デスクトップにアニメーション付きのコンパニオンを表示し、作業状況をリアルタイムで確認できる。標準キャラの使い方から、hatch-petスキルで自作キャラを作る方法、そしてAI開発における癒し効果まで徹底解説。

①Codex Appの凄まじい勢い

Codexアプリは最近目覚ましい進化を遂げている。コンピューターユース、GPT Image 2の統合、内部ブラウザーなど、次々と新機能が追加されている。
そんな中に「ペット機能」という遊び心あふれる機能が登場した。機能性だけでなく、開発者体験（DX）の面からもOpenAIの注力が感じられる興味深いアップデート。

②ペット機能とは何か？

Codex Petsは、Codexアプリ内で動くアニメーション付きのコンパニオン機能。デスクトップに浮遊するオーバーレイとして表示され、好きな場所に配置できる。
他のアプリを使っている間も、Codexの作業状況を視覚的に確認できる常駐型のUI。アクティブなスレッドの状態や、エージェントが考え中かどうかを一目で把握できる。

③ペット機能の設定方法

ペット機能は設定の「外観（Appearance）→ペット」から有効にできる。設定画面の一番下にペットの項目が追加されている。
もしペットの項目が表示されていなければ、Codexアプリを最新版にアップデートする必要がある。アップデート後、設定から有効にするだけで利用可能。

⑤ペットの実用的なメリット

ペットは画面上の好きな場所に置いて、作業状況を表示させながら他のアプリを使える。
例えば「Hello」と入力すると、ペットが「考え中」と表示され、結果が出たら通知してくれる。Codexアプリを開き直さなくても、画面の上でエージェントの状態をリアルタイムに把握できる。
機能としてはシンプルだが、バックグラウンドでAIを動かす開発者にとって非常に役立つ品質向上機能。

⑥自作キャラの作り方 — hatch-petスキル

自作キャラクターを作るには、hatch-petというスキルをインストールする必要がある。

② Codexアプリを再起動

インストールはチャットから簡単に行え、指示に従うだけで完了する。

⑦画像からデフォルメキャラを自動生成

ベースにしたい画像を渡すだけで、ゲーム素材風のデフォルメキャラクターを自動生成してくれる。
内部ではGPT Image 2を活用して、元画像をゲーム風の可愛いキャラクターに変換。Codexのペット仕様に合わせて、スプライトシートも自動生成される。別画像すらCodex内のイメージツールで作成できるため、外部ツールは一切不要。

⑧動く素材作りにも応用できる

hatch-petで作成したキャラクターは、ゲーム素材や動く素材の作成にも応用できる。
画像を渡すだけでデフォルメされたアニメーション付きキャラが出来上がるため、個人開発のゲーム素材作りにも活用可能。hatch-petはペット機能にとどまらず、クリエイティブな素材作成ツールとしても使える。

⑨なぜAI開発に癒しが必要なのか？

AI駆動開発やAIエージェントの作業を長時間続けていると、頭痛や精神的な疲労を感じることがある。
そうした時に、画面上に自分の好きなキャラクターがいて、作業状況を教えてくれると意外と癒される。機能としてはシンプルだが、長時間のAI開発における心理的なサポートになる。

⑩Codex Appのオートメーション機能

Codex Appには定期的な自動実行も可能なオートメーション機能が搭載されている。
スレッド内で「自動化を追加」を選ぶと、何分ごとにセッションを引き継いで定期実行するかを設定できる。同じコンテキストを維持したまま継続的なタスクの自動化が可能。
