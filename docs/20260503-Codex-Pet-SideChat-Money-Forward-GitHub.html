Codexペット登場・サイドチャット実装・マネーフォワードGitHub不正アクセス

2025年5月2日のAIニュース10本をピックアップ
AI整体師
OpenAI Codexデスクトップアプリにペット機能とサイドチャットが追加。カスタム辞書機能も開発中。一方でマネーフォワードのGitHubに不正アクセスが発生し、クレカ情報流出の可能性も。AnthropicのMythos日本展開計画も注目。

Codexペット機能登場！

OpenAI Codexデスクトップアプリにペット機能が追加。
設定画面の「外観」→一番下の「Pet」から選択。デフォルトデザインまたはカスタムペットを自分で作成可能。アニメーション付きで画面上に常駐。

ペットの実用機能

ペットは可愛いだけじゃない：
通知送信 対応
タスク進捗表示 対応
アニメーション 常時稼働
ぬくぬこさんが実際にカスタムペットを作成し、作り方をXで公開。キャラデータも配布中。

サイドチャット機能追加

Codexにサイドチャットが実装。メインチャットの横にもう1つのチャット画面を開ける。
メインチャットのコンテキストを引き継いで質問が可能。実装中のコードについて「これどういう意味？」「これで合ってる？」をメインの文脈を汚さずに確認できる。

サイドチャットの制限

サイドチャット側からは実装（コマンド実行・行動の書き換え）はできない。質問と回答のみ対応。アクションが必要な操作はメインチャットで行う必要がある。
活用例: プロンプトガイドを読みながら分からない部分をサイドチャットで質問。メインの文脈を維持したまま補助的なQ&Aが可能。

設定プラグインのインポート

Codexへの切り替えを簡単にする新機能：
新しいプロジェクト追加時に既存のディレクトリを選択すると、設定ファイル（.MD）やスキルを自動で読み込み可能に。
Codexへの移行障壁を下げる施策。ゼロスタートから既存設定を引き継いでスムーズに開始。

カスタム辞書機能（開発中）

Codex & ChatGPT向けにカスタム辞書機能を開発中。
音声入力モードで単語登録・略語登録が可能に。これまでは音声入力アプリより精度は高いが辞書機能がなく不便だった。
実装されれば専用音声入力アプリが不要になる可能性。かなり便利なアップデート。

Wordにリーガルエージェント導入

Microsoft Wordにリーガルエージェントが登場。
資料内の法的な正確性を自動チェック。何行目にどういう記述があるかを抽出し、法的観点から検証する専用エージェント。
厳密な資料作成を行う方向け。Microsoftらしい実務的AI機能。

Code with Claude来週開催

Anthropicの開発者向けカンファレンスが来週開催予定。
同時期にGPT-5.5の誕生日パーティーも開催。両社からの何らかの発表が期待される。
対抗意識を感じるスケジュールだが、確定情報はまだなし。両方から大きな発表がある可能性。

GPT-5.5のサイバー攻撃能力

イギリス政府がGPT-5.5のセキュリティ能力を評価。
サイバー攻撃能力は一部Mythosレベルに到達。全てにおいてMythosを超えているわけではないが、非常に高い水準。
Mythosの存在自体も議論されているが、実在する可能性が高いとの見方。

マネーフォワードGitHub不正アクセス

マネーフォワードのGitHubに不正アクセスが発生。認証情報が漏洩。
影響件数 370件
流出内容 名前＋下4桁

流出の詳細と影響

370件のカード保持者のアルファベット名およびカード番号の下4桁が流出の可能性。ただしクレジットカードの完全な情報は漏洩なし。
GitHubに個人情報を置くことの是非や、セキュリティ対策の甘さが界隈で大きな議論を呼んでいる。

Anthropic Mythos: 日本展開計画

AnthropicのMythosが日本を含めて提供先拡大を計画中。ただしアメリカ政府は反対の姿勢。
日本国内のどこに提供されるかは未定。政府機関または大企業が想定される。アメリカの大企業ラインナップに匹敵する規模の企業が対象か。

Mythos日本展開の意義

日本国内でもMythosの凄さを体感できる人たちが登場する可能性。セキュリティ強化にも活用。
現在先行提供中のアメリカの大企業ラインナップは非常に豪華。それに匹敵する日本企業への提供が期待される。

GitHubセキュリティの教訓

マネーフォワードの件から学ぶべきこと：
GitHubの使い方を改めて見直す重要性。個人情報の取り扱い、アクセス権限の管理、セキュリティ対策の再点検が必要。
自分たちでも同様の被害に遭わないよう、注意すべきポイントを整理しておくべき。

カスタムペットの作り方

ぬくぬこさんが実際にカスタムペットを作成し、X（旧Twitter）で作り方とキャラデータを公開。
自分の好きなキャラクターを作って設定可能。2Dアニメーションで画面上に常駐する癒やし要素。
