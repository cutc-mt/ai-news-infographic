Browserbase Skills

AIエージェント向けに再利用可能なブラウザ自動化スキルを提供するツールキット。
エージェントがブラウザ操作を「スキル」として定義・保存・再利用できる。複雑なWeb操作の自動化を大幅に簡素化し、一度覚えさせた操作は何度でも再実行可能。フォーム入力、ナビゲーション、データ抽出などをスキル化できる。
browserbase/skills ブラウザ自動化

sim — AIワークフロービルダー

AIワークフロー構築のためのビジュアルプラットフォーム。
ドラッグ＆ドロップでAIパイプラインを構築できる。ノーコードでエージェントのワークフローを設計する次世代ツール。複雑なエージェント連携を視覚的に組み立てられ、プロトタイプから本番環境への移行もスムーズ。
simstudioai/sim ビジュアルワークフロー

Hermes Agent

NousResearchによるツール活用型自律エージェントシステム。
複数のツールを組み合わせて自律的にタスクを実行するエージェントフレームワーク。関数呼び出し（Function Calling）を活用し、外部APIやデータソースと連携しながら複雑なワークフローを完遂できる。
NousResearch/hermes-agent 自律エージェント

Symphony (OpenAI)

OpenAIがリリースしたマルチエージェントオーケストレーションフレームワーク。
複数AIエージェントの調整・タスク分散・状態管理を統合的に行う。大規模なAIワークフローを効率的に構築・運用するための公式ツールで、エージェント間のコミュニケーションと協調作業をシームレスに実現。
openai/symphony オーケストレーション

Trellis — 3Dアセット生成

Mindfold AIによる3Dアセット生成ライブラリ。
テキストや画像から高品質な3Dモデルを生成。AIを活用した次世代の3Dコンテンツ作成パイプラインを実現。ゲーム開発、建築ビジュアライゼーション、VRコンテンツ制作など幅広い用途に対応。
mindfold-ai/Trellis 3D生成AI

TradingAgents — AI取引エージェント

TauricResearchによるAI取引エージェントフレームワーク。
複数のAIエージェントが協調して市場分析・取引判断を行うシステム。金融データの分析と自動トレーディングに特化し、リスク管理やポートフォリオ最適化もエージェントベースで実装可能。
TauricResearch/TradingAgents 金融AI

Fincept Terminal

AI搭載の金融データターミナル。
コマンドラインからリアルタイムの市場データにアクセス。AIによる分析機能も統合された金融専門家向けツール。Bloomberg Terminalのオープンソース代替を目指すプロジェクト。
Fincept-Corporation/FinceptTerminal 金融ターミナル

Kreuzberg — 高速テキスト抽出

高性能バックエンドフレームワーク。
高速なWebスクレイピングとテキスト抽出に特化。大量のWebデータを効率的に処理するための最適化されたバックエンドツール。PDFやHTMLからの構造化データ抽出も高速に実行。
kreuzberg-dev/kreuzberg スクレイピング

Open Design — オープンデザインツール

nexu-ioによるオープンソースデザインツール。
デザイン作業をAIで支援するプラットフォーム。UI/UXデザインの作成からプロトタイピングまで、オープンソースで完結する。Figmaのようなクローズドツールへの代替を狙う。
nexu-io/open-design デザインツール

hindsight — ログ分析ツール

vectorize-ioによるログ分析・監視ツール。
システムログの可視化と分析を強力にサポート。過去のデータからインサイトを抽出し、開発者のデバッグ効率を向上。LLMアプリケーションのログ分析に特に適している。
vectorize-io/hindsight ログ分析

Sandcastle — TypeScript型安全

Matt PocockによるTypeScript型安全ユーティリティ。
TypeScriptプロジェクトの堅牢な型安全性を確保するためのユーティリティ集。開発者の生産性とコード品質を同時に向上。型推論の限界を補う実用的なツール。
mattpocock/sandcastle TypeScript

Helium — プロキシネットワーク

imputnetによるプロキシ・ネットワークツール。
分散型プロキシネットワークでスクレイピングやデータ収集を最適化。ロケーション制約を回避する柔軟なネットワークソリューション。IPローテーションと自動リトライを組み合わせた堅牢な設計。
imputnet/helium ネットワーク

wterm & Dirac — 開発者ツール

wterm（Vercel Labs）：モダンなWebベースのターミナルエミュレータ。開発者のローカル環境を強化する最新のCLI体験を提供。
Dirac：分散AI推論システム。複数ノードでAI推論を効率的に分散実行し、大規模モデルの推論コストを削減しつつスループットを最大化する。
vercel-labs/wterm dirac-run/dirac

GitHub Copilot CLI SDK

GitHub公式のCopilot CLI統合SDK。
コマンドライン環境にCopilotを組み込むための公式SDK。ターミナル上でAIアシスタントを活用できる開発者体験を実現。シェルコマンドの提案、スクリプトの生成、エラーの解決策提示などをCLI上で直接利用可能。
github/copilot-sdk GitHub公式
