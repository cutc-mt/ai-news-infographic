Hermes Agent V2.0 Refreshed

オープンソースAIエージェントが遂に「自律」を手に入れた — その全貌
2026年5月4日

① Hermes Agentとは — Nous Researchの野望

  • Nous Researchが開発するオープンソースのAIエージェントフレームワーク
  • 完全オープンソースで、誰でも自由に拡張・カスタマイズ可能。プロプライエタリのロックインなし
  • Telegram、Discord、Slack、WhatsApp、Signal、Email、CLIなどあらゆるプラットフォームで稼働
  • 単なるチャットボットではなく、自律的にタスクを実行するAIインフラを目指す設計
ChatGPTのような対話ツールではなく、
業務を自動化する本格エージェント。
それがHermesの哲学です。

② V2.0 Refreshed — 何が変わったのか

  • アーキテクチャ全面刷新 — V1から根本的に再設計。パフォーマンスと安定性が劇的に向上
  • 記憶システムがモジュラー化され、AIが過去のやり取りから継続的に学習・成長するように
  • コールドスタート時間が大幅に短縮。起動から稼働までが爆速に
  • 新しい推論プロバイダーを追加し、より多様なLLMバックエンドに対応
「Refreshed」の名の通り、
中身を根底から作り直したメジャーアップデート。
進化の規模が違います。

③ Curator機能 — AIが自分自身を管理する時代

  • 自律型Curatorがバックグラウンドで稼働。スキルライブラリを自動で評価・整理・最適化
  • 不要なスキルを刈り込み、有用なスキルを統合。エージェントが自己メンテナンスを遂行
  • 決められたスケジュールで自動実行。人間の介入なしにスキルが常に最適な状態に保たれる
  • より賢い自己改善ループにより、使えば使うほど精度が上がり続ける仕組み
AIが自分のスキルを自分で掃除する...
SFが現実に。
人間のメンテ負担が激減します。

④ Manim Skill — プロ品質のアニメーション動画生成

  • 3Blue1Brownスタイルの数学アニメーションを1つのプロンプトから自動生成
  • 調査→スクリプト作成→高品質ビジュアル制作まで、コンテンツパイプライン全体を自動化
  • 教育コンテンツの解説動画をノーコードで作成。数式の可視化に特化
  • 出力品質はYouTuberレベル。Manimライブラリを直接操作してプロ級の映像を生成
「この動画を作って」と言うだけで
完成品が出来上がる。
教育コンテンツ制作が革命を迎えます。

⑤ モジュラー記憶システム — 成長する脳

  • 記憶脳をアプリのように交換可能に。用途に応じて最適な記憶モジュールを選択
  • 過去の会話を記憶し、文脈を引き継いで継続的に学習。使うほど賢くなる仕組み
  • クレデンシャルプールシステムでAPI鍵を一元管理。セキュリティと利便性を両立
  • 記憶の永続化により、セッションをまたいで一貫性のある対話が可能に
記憶をモジュール化する発想が
目から鱗。用途別に脳を変える

  • セキュリティアップグレードにより、本番環境での運用も安全に

  • サブエージェントによるタスクの並列実行で、複雑なワークフローを高速処理
  • エージェント間でタスクを委任し、自律的に連携する分散処理が可能
  • ビルトインのタスクスケジューリングで、定期実行も自動化
