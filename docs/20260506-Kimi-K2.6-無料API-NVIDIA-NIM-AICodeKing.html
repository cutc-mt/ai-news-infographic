Kimi K2.6とは

  • Moonshot AIが開発した最新の大規模言語モデル
  • 総パラメータ数は1兆（1T）の超大型モデル
  • Mixture of Experts（MoE）アーキテクチャを採用
  • 推論時にアクティブなパラメータは約320億（32B）
MoE方式なので、全体は巨大でも効率的に推論できるのがポイント！

NVIDIA Buildで提供開始

  • Kimi K2.6がNVIDIA NIMエンドポイントとして追加
  • アクセスURL：build.nvidia.com/moonshotai/kimik2.6
  • ログインしてAPIキーを即座に生成可能
  • 新規利用時はアカウント作成と電話番号認証が必要
NVIDIA NIMには他にもGLM、MiniMax、DeepSeekなどのモデルが多数ラインナップ！

API設定の詳細

  • Base URL 
https://integrate.api.nvidia.com/v1
  • モデルID 
moonshotai/kimi-k2.6
重要: Base URL欄には /v1 までを指定。フルのChat Completions URLを入れないこと！
モデルIDはモデルフィールドに入力するよ！

  • NVIDIA APIキーを入力してモデルを設定
  • まずはシンプルなプロンプトで接続確認
