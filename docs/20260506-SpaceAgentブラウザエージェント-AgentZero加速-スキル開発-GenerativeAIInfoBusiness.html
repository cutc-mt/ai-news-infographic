Space Agent & Agent Zero
次世代AIエージェントの最前線

ブラウザ上で動くフロントエンドAIエージェントと加速するバックエンドエージェント

2026年5月3日 | Generative AI and InfoBusiness Institute

①Space Agentとは: ブラウザで動く新しいAIエージェント

  • Space Agentは、ウェブブラウザ上で直接動作するオープンソースのAIエージェントです。
  • JavaScriptランタイム内で実行されるため、ブラウザの中に「住んでいる」エージェントとして機能します。
  • チャットを通じて独自のUI、ウィジェット、複雑なアプリケーションを即座に生成して画面上に展開できます。
  • トークン効率を極限まで高めた設計により、低コストかつ高速で動作します。
開発者自身も「何に使えるかわからないが、すごい可能性を感じる」と語る、まさに無限の可能性を秘めたツール！

②フロントエンドに住むエージェント: JavaScriptで動作

  • ブラウザの中でエージェントが動いており、JavaScriptで様々なものを構築します。
  • スネークゲーム、世界時計、ダッシュボードなど、何でも即座に作成可能です。
  • ローカルPCのファイルへのアクセス権限は一切持たず、安全性が高い設計です。
  • ただしファイルを渡せば処理は可能で、翻訳や変換などの作業もこなします。
従来のCLIエージェントとは異なり、ブラウザというGUI環境そのものが作業スペースになるのが最大の特徴！

③Agent Zeroの開発者が新プロダクトを立ち上げ

  • Agent Zeroの開発者が、新しくSpace Agentを立ち上げました。
  • Agent Zeroはバックエンド（Linux上で動作）、Space Agentはフロントエンド（ブラウザ上で動作）という住み分けです。
  • Agent ZeroがLinuxを住みかとする強力なエージェントなら、Space Agentはブラウザを住みかとするエージェントです。
  • バックエンドとフロントエンドの両面からAIエージェントの生態系をカバーする戦略的な展開です。
Agent Zeroはバージョン1.0を超えてから急速に加速中！開発者の実力には注目です。

④自律的なスキル開発: 自分で能力を拡張できる

  • Space Agentは自分自身のスキルを開発できるのが最大の特徴です。
  • APIドキュメントのアドレスを渡すと、内容を読み込んでスキルファイル（skill.md）を自動生成します。
  • テスト実行後にスキルを開発する流れで、何百枚のカードを取得するテストも自動で行います。
  • 長期記憶を持たせることも可能で、ローカルPCのアシスタントとしても活用できます。
APIキーとエンドポイントを渡すだけで、エージェントが自らスキルを習得する。これが次世代のAIエージェントの姿！

⑤Recall APIとの統合テスト: 実際に動かしてみた

  • RecallのAPIドキュメント、エンドポイント、APIキーをSpace Agentに渡してテストを実施。
  • エージェントが自動的にAPIを呼び出し、カード一覧取得やセマンティック検索などの機能を確認。
  • テスト成功後、リコールAIのスキルファイルを自動作成。認証や使い方も自動的に理解しました。
  • 使用トークンは約62,000トークン。スキル開発からウィジェット作成まで一気に完了。
APIドキュメントを渡すだけで、エージェントがリスト取得・検索・カード管理まで理解。これはすごい！

⑥Recall 2.0がAPIとMCPをサポート: 連携が劇的に向上

  • Recall 2.0がAPIとMCP（Model Context Protocol）をサポートし、外部連携が大幅に強化されました。
  • スマホで外出中に情報をキャッチアップして、Recallにポイポイと投げ込む使い方が可能です。
  • Recallは自動的にタグ付けして分類。「AIツール」「テクノロジー」「ゲーム」など自動で整理。
  • メモのように編集でき、台本として読み上げることも可能です。
Recall 2.0は激ヤバです！外出先から情報を蓄積して、デスクトップに戻ったらエージェントに整理してもらう_workflowが実現。

⑦ウィジェットの即時生成: 最新情報をダッシュボードに表示

  • 作成したRecallスキルを使って、最新5件のカードを表示するウィジェットを即座に生成。
  • 「このスキルを使って最新のカードを表示させるウィジェットを作って」と指示するだけで完了します。
  • ダッシュボード上にAIonUI、OpenCode5、論文解説などのカード情報がリアルタイムで表示されます。
  • ウィジェットはドラッグで移動可能で、自由にレイアウトをカスタマイズできます。
指示してからすぐに完成！スピードがすごい。自分の情報ダッシュボードが一瞬で作れます。

⑧トークン効率が抜群: 低コストで高速動作

  • Space Agentはトークン効率を極限まで高めた設計になっています。
  • ウェブブラウズや外部ツールの構築を低コストかつ高速で行えるのが大きな特徴です。
  • エンジンはQwen 2.6を使用。サブスクリプションはOpenCode5を活用しています。
  • スキル開発からウィジェット作成まで一連の作業を高速で完了するため、実用性が高いです。
コストパフォーマンスが最高！少ないトークンで多くのことができるから、気軽に使い倒せます。

⑨WebOS的なデスクトップ体験: スペースで作業環境を管理

  • 「スペース」という概念で、テーマごとに作業環境を作成・管理できます。
  • 各スペースには複数のウィジェットを配置し、まるでデスクトップOSのような体験が可能です。
  • 「New Space」からいくらでも新しいスペースを作成でき、用途ごとに使い分けられます。
  • ウィジェットは自由にドラッグして配置を変更でき、カスタマイズ性が高いです。
最近流行りのWebOS的なアプローチ！ブラウザの中に自分専用の作業環境を構築できます。

⑩ブラウザ操作能力: 自らウェブを閲覧して情報収集

  • アドレスを渡すとブラウザを開いて情報を取得する、人間と同じような仕事の仕方をします。
  • ハッカーニュースやBBCニュースなどを取得して、日本語翻訳してウィジェットに表示することも可能です。
  • 自ら調べ物をしたり、必要な情報を探して収集する能力を持っています。
  • YouTubeのビデオを取得して視聴できるウィジェットも作成可能です。
エージェントが自らブラウザを操作して情報収集するから、指示するだけでOK！

⑪Gitタイムトラベル機能: 変更を自由に取り消せる

  • Gitを活用したタイムトラベル機能を備えています。
  • 変更の取り消しや過去の状態への復元が簡単に行えます。
  • 失敗を恐れずに大胆に実験できるため、試行錯誤がしやすい環境です。
  • 全ての変更履歴が残るので、いつでも好きな時点に戻ることができます。
タイムトラベル機能で安心して実験！「やっぱりあの状態に戻したい」が即座に実現できます。

⑫Agent Zeroが急加速: v1.0を超えて猛烈に進化中

  • Agent Zeroはバージョン1.0を超えてから、猛烈な勢いで進化を続けています。
  • Linux上で動作するバックエンドエージェントで、Linuxでできることは全て実行可能です。
  • Dockerコンテナの中で動作するため、安全性も高く安心して利用できます。
  • ないものは探しに行って取り込み、探してもなければ自分で作ってしまう強力なエージェントです。
「ダークヒーロー」的な存在！全ての能力を自分のものにする Agent Zeroは、まさに最強のバックエンドエージェント。

⑬フロントエンドとバックエンドの住み分け

  • Agent Zeroはバックエンド：Linux上で動作し、裏側で継続的に作業を実行します。
  • Space Agentはフロントエンド：ブラウザ上で動作し、視覚的な操作が可能です。
  • 両者を組み合わせることで、バックエンドとフロントエンドの両面からAIエージェント活用が可能に。
  • 同じ開発者が両方を手がけることで、将来的な統合や連携にも期待が高まります。
バックエンドでAgent Zero、フロントエンドでSpace Agent。表裏一体の最強コンビが誕生！

⑭音声合成との連携: Iris APIで高品質な読み上げ

  • Recall標準の読み上げ機能は「しょぼい」ため、Iris APIを使った高品質な読み上げへの置き換えが可能です。
  • IrisのAPIキーとドキュメントを読み込ませてスキルを作れば、自動的に音声コンテンツを生成できます。
  • Recallのカード情報を元に台本を作成 → 音声合成という一気通貫のワークフローが構築可能です。
  • 指示一出せば、エージェントが台本作成から音声合成まで全自動で実行します。
デスクにいなくても、エージェントに音声コンテンツの自動生成を任せられる！

⑮ワークフローの革新: ニュース収集から音声コンテンツまで自動化

  • Step 1: 外出中にスマホでニュースや論文、ブログ記事をRecallにポイポイと蓄積。
  • Step 2: デスクトップに戻ったら、Recallスキルを使って最新の注目ネタを3件ピックアップ。
  • Step 3: Space Agentに「今週のAI最新ニュース」の台本を作成させる。
  • Step 4: Iris APIスキルで台本を音声合成して、音声コンテンツを自動完成。
情報収集 → ピックアップ → 台本作成 → 音声合成まで全自動！もうデスクにいらなくていいかも？

  • エンタメスペース: スネークゲームやレトロアーケードゲームなどを配置したスペース。
クリプトダッシュボード、メモ帳、Vimエディターなども作成可能！あなたの想像力が限界です。

⑲断片化の課題とSpace Agentの解決アプローチ

  • 現在のAIエージェント環境は断片化が進んでいます。
  • Claude Codeは1つの端末ウィンドウ、OpenClawは別のウィンドウ、LSエージェントは3つ目のウィンドウで動作。
  • 各エージェントが独自の設定ファイル、MCP設定を持っており、管理が複雑になっています。
  • Space Agentはブラウザ内で統合的に動作することで、この断片化の解決を目指しています。
エージェントの数は増える一方。それを一つのブラウザ空間にまとめる発想が画期的！
