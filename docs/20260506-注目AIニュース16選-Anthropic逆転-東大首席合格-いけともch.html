② Claudeがクリエイティブツールを操作: Adobe・Blender連携

  • AnthropicがAdobe、Blenderなど主要ソフトと連携するコネクタを発表
  • Blender（3Dソフト）とMCP接続し、テキスト指示で3Dモデルを生成可能に。「インテリアのある部屋を作って」と依頼すると自動で作成
  • Claudeはスクリーンショットで制作状況を確認しながら、リアルタイムに修正を加える
  • Adobe Creative Cloud経由で画像の背景切り抜きや画像編集も可能。クラウド上で処理が完結
  • Canva Affinity、Autodeskなど多様なツールが対応。AIは「相談役」から「作業代行のアシスタント」へ進化
作られたファイルは編集可能な形式で保存できるので、プロがさらに手を加えることも可能！

③ GeminiでWordやExcelファイルを直接生成可能に

  • Geminiに「まとめてExcelとWord作って」と依頼すると、直接ファイル形式で出力されるように進化
  • GoogleドキュメントやGoogleシートへの出力に加え、Excel形式でのダウンロードにも対応
  • Gemini自身もどんどん賢くなっており、より高度な処理をして最終的なアウトプットまで作成できるように
AIからの出力が直接使えるファイル形式になることで、業務効率が大幅に向上！

④ 現場特化型AI「Avoca AI」がユニコーン企業に

  • ニューヨーク発のスタートアップ。配管・空調（HVAC）などのブルーカラー業界に特化したAIフロントオフィサー
  • 現場の悩みは「忙しくて電話に対応できない」。電話・メール対応をAIが代行し、顧客フォローを自動化
  • 強みは2つ：①業界特有の専門知識（用語・見積もり）を最初から備えている ②大手が手薄な小規模市場を地道に開拓
  • 電話の取りこぼしを防ぐことで、年間で数千〜数万ドルの売上増に直結。単なるコスト削減ではなく機会損失を防止
  • 医療、ITヘルプデスク、製造業調達など、ニッチな業界ごとに専門プレイヤーが続々登場中
日本でも業界特化型AIのチャンスは大。
特定の地方や業界で、使いやすい形で提供するビジネスモデルに注目！

⑤ AIエージェントが店舗経営に挑戦: Andon Marketの実験

  • AnthropicとAndon Labsが共同で、AIに店舗運営を任せる実験を開始（2026年4月〜）
  • AIが担当するのはブランドデザイン・商品選定・壁紙発注・採用面談まで。企画から人事まで全部AIが決定
  • 商品陳列や店舗オペレーションは雇用した人間が担当。AIは「オーナー的ポジション」として機能
  • フェーズ1（2025年3月）は自販機運営で大失敗、フェーズ2（2025年12月）は車内販売で改善もWSJにハックされる等、試行錯誤を重ねている
  • 現在は赤字。商品が適当だったり、AIであることを隠そうとする等の課題があるが、AIの弱点をあぶり出す貴重な実験
AIの弱点：長期コンテキストが苦手・言われた情報が蓄積すると前提が覆る。
AIの監督者としての人間はまだ必須！

⑥ OpenAIがユーザー・売上の目標に未達: 圧倒的王者の揺らぎ

  • ユーザー数は10億人を目標としていたが9億人にとどまる。課金ユーザーはわずか5%程度
  • 売上面でも目標未達。Stargate（5000億ドル投資のデータセンター計画）も実際の投資は見送られる状況
  • 法人向け市場ではすでにAnthropicに逆転されている。一般ユーザーではChatGPTが強いものの、総合では苦戦
  • OpenAIの年間売上見込み（ARR）は250億ドル（約4兆円）に対し、Anthropicは300億ドル（約5兆円）で上回る
かつての圧倒的AI王者・OpenAIが、もはや無敵ではない状況に。
AI業界のパワーバランスが大きく動いている！

⑦ MicrosoftとOpenAIの独占契約が終了: マルチクラウド時代へ

  • これまでOpenAIモデルはAzure独占だったが、AWSやGoogle Cloudでも利用可能に
  • Anthropicは元々マルチクラウドで展開していたのに対し、OpenAIはAzure一択だったことが法人獲得の弱点に
  • 元々「AGI達成＝年間1000億ドル規模」で独占契約破棄の条項があったが、Microsoftは2032年までのライセンスを確保
  • 企業は「AnthropicもOpenAIも」というマルチAI体制の選択肢に入りやすくなる
AWSやGoogle CloudユーザーにもOpenAIモデルの選択肢が広がったことで、AI活用の柔軟性が向上！

⑧ Anthropicの企業評価がOpenAIを逆転か: 9000億ドルの衝撃

  • Anthropicの次の資金調達での評価額が9000億ドルに達するとの報道。現在のOpenAIは8700億ドル
  • 成長の原動力はClaude Codeをはじめとするコーディング専用AIの強さ。開発者コミュニティでの支持が圧倒的
  • GoogleとAmazonが出資し、独占ではなく連携することでマルチクラウド展開を実現
  • 元々は安全性と性能で定評があったが、事業面でも絶対王者だったOpenAIを抜く勢い
Anthropicは元々OpenAIの研究者が独立したスピンオフ企業。それが本体を抜くという前代未聞の事態に！

⑨ 中国政府がMetaのManus買収を阻止: 20億ドルの壁

  • Metaが約3000億円でManusを買収。すでに支払いも完了し統合も進んでいた
  • しかし中国政府が後付けで「中国発の技術だから認められない」と買収の撤回を要求
  • Manusは本社をシンガポールに移していたが、中国は「シンガポールウォッシング（国籍を偽装する行為）」だと主張
  • 背景には米中のAI技術覇権争いがある。数週間以内に最終結論が出る見通し
AIエージェントのManusは日本でも注目度が高いツール。
国際政治が民間AIツールの行方を左右する時代に！
テクノロジー

⑩ Claude ProでOpusモデルに追加料金: こっそり変わる料金体系

  • 月額20〜25ドルのProプランで、Claude Codeから最上位モデルのOpusが使えなくなる変更がこっそり実施された
  • 理由はClaude Codeがエージェントとして大量の処理を行うため、Opusの処理コストが高すぎるから
  • Opusを使うにはMaxプラン（$100〜）以上が必要、またはProでも従量課金を設定すれば利用可能
  • ユーザーからの不満が高まっている。説明も不十分で、注目される企業のマイナス面も表面化
同じ月額でも使えることが減っていく…。
AIモデルの料金設計が各社で模索中！

⑪ GitHub Copilotが6月から全プラン従量課金制へ移行

  • これまでは基本使い放題だったGitHub Copilotが、6月1日から新たなクレジット制に移行
  • AI業界全体のトレンド：使い放題から従量課金へ。ClaudeもChatGPTも同様に移行中
  • これからは「ちょっとした処理は安いモデル」「重要なものは賢いモデル」という賢い使い分けが求められる
  • おすすめは複数のAIを使い分けること。無料枠も広がり、月20ドル程度でいくつか使いながらコツを掴むのが良い
「どのAIに・どんな指示を・どのモデルで出すか」
この使い分けのセンスがこれからの価値に！

⑫ StripeがAIエージェント向け決済「Link」をアップグレード

  • Stripeのデジタルウォレット「Link」で、AIエージェントに買い物を代行させる機能が登場
  • クレジットカード情報はAIに渡さず、Linkウォレットを経由して安全に決済。セキュリティを確保
  • 利用条件の設定、事前承認の要否、予算上限などをルール化可能。「月1万円以内で毎月これを買って」など
  • エージェンティックコマース（AIが代行する商取引）時代の幕開け。まずは米国で先行、日本にも遠からず到来
AIに買い物や支払いを任せるってどういうこと？
この新しい体験に早く慣れておくことが大事！

⑬ AI雇用の影響: 米企業で金融危機並みの解雇が進行

  • Microsoft、Amazon、Meta、Salesforceなどで数万人規模のリストラが始まっている
  • 「採用されて2週間で仕事がなくなり解雇された」という実態も。AIに置き換えるという名目で大規模な人員削減が加速
  • ビッグテックの新卒採用数はコロナ前比で50%以上減少。ジュニア層の仕事をAIが代替している
  • エクスペリエンスパラドックス：企業は経験者が欲しいが、経験を積む機会自体が減っていくという悪循環
  • Anthropic社長のダリオ・アモデイ氏は「5年以内にホワイトカラーの50%が削減されるかも」と予測
会社任せの時代は終わる！
自分でAIツールを使いこなし、経験を自ら作っていくことが不可欠に。

⑭ 米国防総省がAI導入: 7社と提携、Anthropicは除外

  • Google、SpaceX、OpenAIなど大手7社と提携し、最高機密ネットワークにAIを導入
  • Anthropicは除外：理由は「国民の監視や自律的大量殺害兵器には使わせない」という方針を拒否していないため
  • ただしAnthropicの「Claude Mythos」は高性能すぎて、米政府もサイバーセキュリティ用途で活用中。技術力は無視できない
  • Googleでは600名以上の社員がSundar Pichai CEOに公開書簡で反対を要望したが、回答がないまま契約締結
  • 7社の中にはベンチャー枠でReflection AI（西側のDeepSeekと呼ばれる）が参加。創業者の1人はAlphaGo開発者
Anthropicの姿勢を支持するエンジニアたちが
同社に流れる可能性も。組織力がさらに加速するかも！

⑮ 金融庁が新型AI「Claude Mythos」を巡り作業部会を開催

  • Claude Mythosはセキュリティにおいて極めて高い攻撃能力を持ち、金融機関を狙うと甚大なリスク
  • 片山金融担当相、日銀総裁、メガバンクなどが参加して作業部会を開催。現状では日本の政府・金融機関はClaudeを利用不可
  • Anthropicは米国系企業50社限定で公開中。同盟国を含める拡大を検討しているが、米国政権は反対（安全保障＋GPUリソース問題）
AI活用は国レベルの安全保障問題に。
日本としてどう交渉していくかが重要！

⑰ ChatGPTが東大理三・京大に首席合格: 別次元の知力
