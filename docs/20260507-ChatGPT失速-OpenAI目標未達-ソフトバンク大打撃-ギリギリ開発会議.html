ChatGPT失速？OpenAIの目標未達でソフトバンクが大打撃

AI覇権争いに異変 — ユーザー数の足踏み、競合の猛追、株価への波及

2026年5月7日 | ギリギリ開発会議

① OpenAIの野心的な目標: 10億人ユーザー

OpenAIはアクティブユーザー10億人を目標に掲げていた。世界人口の約4〜5人に1人が使う計算で、非常にアグレッシブな数値だ。

10億人って、大人だけで考えると約5人に1人！かなり大胆な目標ですね。

② 現状は9億人で足踏み状態

9億人までは到達したものの、そこからユーザー数の増加が止まってしまっている。目標の10億人には届いておらず、伸び悩みが続いている。

9億人
現在のアクティブユーザー数（そこから停滞）

③ GPT-5.5は優秀でも売上は伸びず

GPT-5.5の性能は非常に高く、ユーザーからも好評価だ。しかし意外と売上は伸びておらず、ユーザー数も足踏みが続いているという状態だ。

モデルの性能が良くても、課金ユーザーに結びついていないのが課題。

④ Google Geminiの猛追: サービス連携で優位性

Google Cloud Nextで発表されたGeminiとGoogle各サービスの連携が大きな話題に。Gmail、Docs、Sheetsなどとの深い統合で利便性が格段に向上している。

  • GeminiがGoogleサービスとシームレスに連携
  • ユーザーにとって「便利」が大きな武器に
  • Googleの既存プラットフォームが強みに

⑤ Anthropicの台頭: Claude・Mitos・コアーク

AnthropicもMitosなどの新モデルを投入し、業務特化型AIとして存在感を高めている。Claudeの進化も著しく、コーディング用途でもファンが増えている。

  • Mitos — 高性能な新モデル
  • Claude — 開発者からの支持が高い
  • 業務特化型で差別化を図る
Opus 4.7からCodexに乗り換える人も続出中！

⑥ Quit GPT運動: トランプ関連でユーザー離れ

OpenAIがトランプ陣営に資金提供していたことが判明。これを受け、反トランプ派のユーザーの間で「Quit GPT」運動が広がっている。

  • OpenAIのトランプへの資金提供が発覚
  • 移民規制団体ICEへのGPT提供にも反発
  • リベラル層を中心に不買運動が拡大
日本人には馴染みがないけど、アメリカ国内では大きな問題に。

⑦ Microsoft独占契約終了: AWSでもGPT利用可能に

4月27日にMicrosoftとの独占契約が終了。これにより、AWSなどの他のクラウドプラットフォーム経由でもGPTが利用可能になった。

  • Azure以外のクラウドでもGPTが使える
  • ユーザーにとっては利便性が大幅に向上
  • しかしOpenAIの収入源が減る懸念も
使う側には嬉しいけど、OpenAIにとってはキャッシュポイントの減少リスク！

⑧ 売上の伸び悩みと競合の台頭

売上も期待ほど伸びていない。競合他社が相当に強くなっており、Gemini、Claude、中国製AIなど各社がシェアを奪い合っている。

⑨ ソフトバンクへの大打撃: ADRが12%下落

OpenAIの業績不振が報じられると、ソフトバンクグループの米国株（ADR）が1日で12%も下落する事態に。市場のショックは大きかった。

-12%
ソフトバンクADRの1日での下落率

⑩ 孫正義の巨額投資: OpenAI株を大量保有

ソフトバンクグループが持つ株式の約30%がOpenAI関連という巨額の出資を数年前に実施。孫正義氏はかつてトランプ大統領の隣で笑顔で宣言したが、今は不安な状況だ。

孫さん、ニコニコで投資を宣言したあの日から……大丈夫かな。

⑪ CFOの弱気発言: データセンター支払いが困難

OpenAIのCFOが「今の収益性ではデータセンターの支払いが続けられない」と弱気な発言。データセンターの拡充計画の見直しも示唆されている。

  • 現状の収益ではインフラ維持が難しい
  • データセンター拡充計画を見直しへ
  • AI開発競争に必要な計算資源の確保が不安

⑫ 経営陣の意見対立: Altman vs CFO

Sam Altmanは強気な発言を続ける一方、CFOは弱気な姿勢。OpenAI内部で意見の不一致が生じているようで、経営方針を巡る亀裂が懸念されている。

⑬ 中国オープンソースAI台頭: DeepSeek・Qwen猛追

中国のオープンソースAIモデルが急速に台頭。DeepSeek、Qwen、GLMなどが次々と新モデルをリリースし、性能面でも米国モデルに肉薄している。

  • DeepSeek — 高性能かつ低コスト
  • Qwen — チャットから画像生成まで
  • GLM — 中国市場で圧倒的支持
中国系のリリースラッシュがすごい勢い！毎週新モデルが出ている。

⑭ 米国の50%超が中国製AIを採用

驚くべきことに、アメリカの中小企業の50%以上が中国のオープンソースAIを既に活用している。ローカル環境で動かせる利点とコスト優位性が大きな理由だ。

50%+
米国中小企業の中国オープンソースAI利用率

⑮ ローカルLLMの台頭: オンプレミス需要急増

自社内でAIを動かしたいというニーズから、オンプレミスでのローカルLLM利用が急増中。中国製オープンソースモデルは「賢くて安い」として人気が高い。

  • データを社内に留められるセキュリティ面
  • API利用料が不要でコストを削減
  • 十分に賢いモデルがローカルでも動く
OpenAIに税を払わなくてもいいなら、そうしたいですよね。

⑯ Codex進化: GPT-5.5で大幅改善

CodexもGPT-5.5の登場で使い勝手が大幅に向上。Opus 4.7からCodexへ乗り換えるユーザーが続出しており、コーディング用途での競争も激化している。

  • 1ヶ月前より明らかな改善
  • Opus 4.7からの乗り換え続出
  • 開発者の間で評判が良い

⑰ AI覇権のデッドヒート: 各社がしのぎを削る

OpenAI、Google、Anthropicの三つ巴の競争が激化。新モデルのリリースラッシュが続き、ユーザーも次々と乗り換えを繰り返す状態だ。

  • GPT-5.5 → Codex大幅改善
  • Claude/Mitos → Anthropicが猛追
  • Gemini → 現在沈黙中、3.2が期待される
次はGemini 3.2で逆転？それとも別の伏兵が？

⑱ 今後の展望: ビジネス向けサービス拡充が鍵

OpenAIは開発者向けだけでなく、ビジネス向けサービスの拡充を急速に進めている。AnthropicやGoogleとの本格的なビジネス領域での競争が始まろうとしている。

  • 企業向けAIソリューションの拡充
  • マルチクラウド対応で利便性向上
  • データセンター投資を維持できるかが鍵
結局、ビジネス向けサービスをどう拡充していくかが勝負の分かれ目！
