Claude Code + Ollama = 二度とClaudeにお金を払わない

ローカルで無料のAIコーディング環境を構築する完全ガイド

2026年5月7日

①問題提起: Anthropicの高額な請求

  • Claude Codeを日常的に利用していると、Anthropicへの支払いがどんどん膨らんでいく
  • 使用量に応じて課金されるため、気づけば高額な請求が届くことも
  • 利用を控えるか、支払いを続けるかの二択しかなかった
毎月のAI利用料に悩んでいる開発者は多い！

②解決策: ローカルでClaude Codeを動かす

  • Ollamaを使えば、ローカル環境で無料でClaude Codeが利用できる
  • データが外部に送信されないのでプライバシーも確保
  • 使用量を気にせず無制限に利用可能
クラウド不要、完全無料、プライバシー保護の三拍子！

③ステップ1: Claude Codeをダウンロード

  • Windows用のリンクからインストーラーをダウンロード
  • macOS用のリンクも別途用意されている
  • Linuxにも対応済み
  • ターミナルで claude と打って起動確認
デフォルトでは Sonnet 4.6 が設定されている

④ステップ2: Ollamaをインストール

  • ollama.comにアクセスしてOllamaをダウンロード
  • Windows、macOS、Linuxのマルチプラットフォーム対応
  • インストール後はバックグラウンドで常駐して動作
タスクバーにOllamaのアイコンが表示されていれば稼働中！

⑤ステップ3: モデルを選択する

  • OllamaのModelsタブで多数のモデルを確認可能
  • Qwen 3.5: ビジョン、ツール、思考をサポート
  • Skimmy K26: 数週間前にリリースされた最新モデル
  • 各モデルの機能や対応状況が一覧で確認できる
無料で使うならクラウド版ではなくローカル版を選ぼう！

⑥ハードウェアに合ったモデル選び

  • 自分のPCがNPU / GPU / CPUのどれかで最適なモデルが変わる
  • ハイエンドGPUなら大規模モデルも実用的な速度で動作
  • CPUのみの場合は小型モデルを選ぶと快適
モデルが大きすぎると動作が非常に遅くなるので注意

⑦ステップ4: モデルをダウンロード

  • CLI / curl / Python / JavaScriptのいずれかで取得可能
  • ターミナルでコマンドを実行してダウンロード 
# モデルのダウンロード例
ollama pull qwen3:1.7b

# Claude Code用にモデルを設定
claude model ollama:qwen3:1.7b
初回はダウンロードに少し時間がかかるよ！

⑧ダウンロード済みモデルの確認

  • Ollamaが起動している状態で確認コマンドを実行
  • タスクバーでOllamaの稼働状態を事前に確認 
# ダウンロード済みモデル一覧を表示
ollama list
筆者の環境では4つのモデルがダウンロード済みと表示

⑨Sonnet 4.6からローカルモデルへ切替

  • デフォルトのSonnet 4.6を自分のローカルモデルに変更
  • Claude Codeの設定コマンドをターミナルで実行
  • 既にダウンロード済みなら即座に切替完了
これでAnthropicへの支払いがゼロに！

⑩初回起動時の自動ダウンロード

  • 未ダウンロードのモデルを指定すると自動でダウンロードを促す
  • Enterキーを押すだけでマニフェストの取得が始まる
  • 2回目以降は再ダウンロード不要で即座に利用可能
初回だけ待てば、あとはずっと無料で使い放題！

⑪Qwen 3.5の特徴とバリエーション

  • ビジョン機能に対応している
  • ツール使用をサポート
  • 思考（Thinking）モードを搭載

  • 機密コードや個人情報を安全に扱える
  • オフラインでも動作可能
企業の機密プロジェクトにも安心して使える！
