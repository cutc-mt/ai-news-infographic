OpenAI Codexとは

① Codexとは: OpenAIのコーディングエージェント

  • OpenAIが提供するコーディングエージェント。ChatGPTの別アプリとして独立している。
  • プログラミングだけでなく、パソコン上のさまざまなタスクを依頼できる。
  • 使用するにはCodexアプリを別途ダウンロードする必要がある。
  • ChatGPTに課金していれば利用可能。
CodexはChatGPTとは別アプリ。Claude CodeのOpenAI版のようなイメージだよ！

② Codex vs Claude Code: どっちがいい？

  • CodexとClaude Codeの機能に大きな差はない。
  • 両者が常に競争し合っており、どちらを選んでも問題ない。
  • CodexならClaude Codeのデータをインポート可能。設定画面から簡単に移行できる。

③ データ移行: Claude CodeからCodexへ

  • 設定画面の「インポートしたエージェント設定」からClaude Codeのデータをインポート。
  • 手動移行の場合: MacならFinderの隠しファイル（Cmd+Shift+.）を表示。
  • エージェントファイルやスキルファイルをコピペで移動させることも可能。
  • これまでのClaude Codeでの成果を無駄にせずCodexへ移行できる。
手動移行は上級者向け。基本はインポート機能を使おう！

④ Codexでできること: 幅広いタスク対応

  • 資料作成や動画編集など、Claude Codeと同様のタスクが可能。
  • コーディングだけでなく、PC上のあらゆる作業を自動化できる。
  • Webアプリの作成もサクッと完了。今回の翻訳ツールもCodexで開発。
  • 自然言語で指示を出すだけで、AIが自律的に作業を進める。
プログラミング知識がなくても、Codexならアプリが作れちゃう！
CodexのChrome拡張機能

⑤ Chrome操作機能: ついにCodexにも追加

  • CodexにChrome拡張機能が新しく追加された。
  • 設定 → コンピューターの使用 → Google Chromeをオンにする。
  • ChromeまたはBraveブラウザにCodex拡張機能をインストール。
  • Claude Codeには前々からあった機能が、Codexにもついに搭載。
ChromeとBraveブラウザの両方に対応しているよ！

⑥ Chrome操作の仕組み: プラグインで呼び出し

  • Codexに自然言語で指示を入力するだけ。
  • 送信時にプラグイン「Chrome」を選択して呼び出す。
  • AI用のカーソルが表示され、フォームを自動操作。
  • 黄色い背景のタブが開けば、正常に動作している証拠。 
// 指示の例
"今開いているGoogleフォームの内容を確認して、
 入力を進められるところは進めてください。"
やりたいことを伝えるだけで、AIがブラウザを操作してくれる！

⑦ フォーム入力の自動化: 実演デモ

  • お問い合わせフォームの自動入力を実演。
  • AIカーソルが画面上のフォームを認識して、スムーズに入力を開始。
  • メールアドレスなど教えていない情報は入力をスキップしてくれる。
  • お問い合わせフォームの送信などを大量に効率よく処理可能。
個人情報は勝手に入力しない賢さがある！

⑧ 複数タブ操作: まとめて情報収集

  • 複数のタブを一気に開いて、それぞれを同時に操作できる。
  • Googleスプレッドシートの内容を編集することも可能。
  • 複数タブの情報を一カ所にまとめることもできる。
  • Computer Useより高速で、Web上の情報収集に最適。
タブを何個も開いて情報をまとめる作業が劇的に楽になる！

⑨ Computer Useとの違い: ブラウザ vs デスクトップ

  • Chrome操作: ブラウザ上の情報収集に特化。高速で効率的。
  • Computer Use: デスクトップ全体をカーソルで操作。低速だが柔軟性が高い。
  • Chrome操作で困難な場合は、自動的にComputer Useが補助してくれる。
  • 使い分けが重要: Web情報はChrome操作、それ以外はComputer Use。
Computer Useは遅いけど、Chrome操作ならサクサク動くよ！
Codexの音声入力機能

⑩ 音声入力機能: Codexで音声操作

  • 設定 → 一般設定の下部にある「音声入力」をオンにする。
  • MacならCommandキーの長押しで音声入力を開始。
  • 画面中央下に音声入力のUIが表示される。
  • TaisuやSuperWhisper、AquaVoiceなどの音声入力アプリと同様の機能。
ChatGPTに課金してれば、わざわざ別の音声アプリを買わなくていい！

⑪ システム全体で利用可能: どこでも音声入力

  • Codexを開いていれば、パソコン上のどのアプリでも音声入力が可能。
  • X（旧Twitter）などのSNS上でも利用できる。
  • Macのメモアプリや他のエディタでも音声でテキスト入力。
  • Codexだけでなくシステム全体に対応するのが最大の強み。
Codexを起動しておけば、どこでも音声入力が使える！これは便利すぎる

⑫ 辞書登録機能: カスタマイズ可能に

  • 辞書登録機能が追加され、固有名詞や略語を正しく認識させられる。
  • 例: 「K」と言ったら「KEITO」と正しく変換するよう設定可能。
  • よく使うフレーズの登録にも対応。
  • これで音声入力の実用性が格段に向上した。
2週間前はなかった機能。どんどん進化してる！

⑬ SuperWhisperとの比較: より高度な音声入力

  • 著者のKEITO氏は現在SuperWhisperを愛用中。
  • SuperWhisperはカスタムプロンプトを事前に仕込める。
  • 例: 「議事録モード」を選ぶと、自動で議事録形式に整形して出力。
  • Codexの音声入力にもシステムプロンプト機能が追加されれば、完全移行する予定。
カスタムプロンプトがCodexに入ったら、もう乗り換えるしかない！
GPT Realtime 2とリアルタイム翻訳

⑭ GPT Realtime 2とは: 革命的な音声モデル

  • OpenAIから新しくリリースされた音声モデル。
  • リアルタイム翻訳が可能。まるでドラえもんの「翻訳こんぺいとう」。
  • 喋っている内容を即座に別言語の音声に変換。
  • 外部APIとして提供されており、自分でアプリに組み込む必要がある。
まだ組み込まれたアプリはゼロ。自分で作るのが一番早い！

⑮ 翻訳ツールの作成: Codexで開発実演

  • Codexに「リアルタイム翻訳サービス」フォルダを作成して作業を指示。
  • 参考としてGitHubのサンプルコードのURLをCodexに共有。
  • APIキーをCodexに貼り付けて開発を開始（※セキュリティ上、後で削除推奨）。
  • 機能要件: リアルタイム読み上げ＋画面の音声も拾う設定。 
// Codexへの指示例
"GPT Realtime 2を使った
 リアルタイム翻訳ツールを作成してください。
 リアルタイムで読み上げ音声を入れたいです。
 ウィンドウの音声も拾う設定にしてください。"
APIキーは直接貼り付けない方がいいよ！後で必ず削除してね

⑯ リアルタイム翻訳デモ: 日本語→英語

  • 左下にリアルタイムで字幕が表示される。
  • 日本語の音声がリアルタイムで英語に翻訳される。
  • さらに英語で読み上げまで自動で実行。
  • 入力 → 翻訳 → 読み上げ の一連の流れが超高速。
音声入力 → 翻訳 → 読み上げまで全部リアルタイム。感動レベル！

⑰ YouTube動画のリアルタイム翻訳: 実用例

  • ツールの「音声」モードを選択してYouTube動画のウィンドウを共有。
  • システム音声の共有をオンにして開始。
  • 動画再生 → 日本語の入力字幕 → 英語翻訳 → 英語の読み上げが一気に実行。
  • 海外の英語動画を見ながらリアルタイムで日本語翻訳することも可能。
YouTubeだけでなく、あらゆる動画のリアルタイム翻訳に対応！
