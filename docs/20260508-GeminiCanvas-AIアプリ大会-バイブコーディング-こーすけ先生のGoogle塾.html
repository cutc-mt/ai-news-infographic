④Canvasとは何か

  • Geminiのインタラクティブなスペースという定義です。
  • プロンプトバーからCanvasを選択すると、アプリを作れる状態になります。
  • 各種AIモデルにAPIなしでアクセス可能という最大の特徴があります。
  • Reactを使ったアプリケーションを自動生成してくれます。
APIキーの取得が不要なので、すぐに始められます！

⑤Canvasの使い方

  • Geminiのプロンプト入力画面を開きます。
  • ツールボタンを押してCanvasをオンにします。
  • Canvasが青く活性化されたら準備OKです。
  • プロンプトエリアに「こういうアプリを作って」と指示します。 
// Canvasの起動手順
1. Geminiを開く
2. ツール → Canvas をON
3. プロンプトでアプリを指示
4. Reactベースで自動生成

⑥大会のルール（3カ条）

  • ルール1: 開発にはGeminiを使用すること。
  • ルール2: 何かしらのAI機能を搭載すること。
  • ルール3: 悪意のあるアプリではないこと。
この3つは絶対に守ってくださいね！

⑦開発にはGeminiを使おう

  • 開発には必ずGeminiを使用してください。
  • Geminiの3.1 ProやGemini 3 Flashモデルでも、かなりいいものが作れます。
  • ClaudeやChatGPTが開発で強いと言われますが、Geminiも決して劣るわけではありません。
Geminiで十分に高品質なアプリが作れますよ！

⑧悪意のあるアプリは厳禁

  • 悪意のあるアプリは絶対に作らないでください。
  • 例: ファイルアップロード機能で、ユーザーのGoogleドライブに勝手に保存するなど。
  • Canvas上で裏側のコードが読める仕組みなので、悪意はすぐバレます。
  • 悪意が発覚した場合は大会上から削除。以降のエントリーも不可に。
情報を抜くようなアプリは本当にやめてくださいね！

⑩Canvasでアプリを作る方法

  • プロンプトに「AI会話アプリをHTMLコードで作成して」のように指示します。
  • 機能を具体的に書くと、出力の精度が上がります。
  • プロモードで送信すると、Reactを使ったアプリが自動生成されます。
単に「アプリを作って」と言うだけでなく、流れを書くのがコツです！

⑪プロンプトの書き方のコツ

  • ユーザーのマイク音声をGeminiが受け取って返答を考える。
  • 返答はGeminiのTTSモデル（テキストtoスピーチ）で音声化する。
  • 音声をユーザーに返すという流れを書くだけで、出力が変わります。 
// プロンプトの例
AI会話アプリをHTMLコードで作成してください。

主な機能:
1. ユーザーのマイク音声を
   Geminiが受け取り返答を考える
2. 返答はGeminiのTTSモデルで
   音声化してユーザーに返す

⑫AI会話アプリの作成例

  • ブラウザのマイク機能を使ってユーザーの音声を録音。
  • Gemini 2.5 Flashモデルに送信して、文字起こしと返答を同時に取得。
  • Gemini 2.5 Flash TTSモデルで返答を音声に変換。
  • わずか数分でAI会話アプリが完成します。
これだけの時間でAI会話アプリができちゃいます！

⑬実際の動作確認

  • マイクの許可が必要です。ブラウザから許可をして開始。
  • 英語で話しかけると、AIが音声で返答してくれます。
  • 「Hello, I'm Kosuke」と話すと、AIが「はい、Kosukeさん」と応答。
  • 裏側のプロンプトが表示されることがありますが、動作自体は問題なし。
マイクに話しかけるだけでAIと会話できちゃう！

⑮モデル選びの重要なTips

  • デフォルトではGemini 2.5 Flash Preview 09-2025が使われます。
  • Gemini 3 Flash Previewに書き換えると、解釈能力が格段に向上します。
  • Gemini 3.1 Flash Lite Previewも使えますが、速度重視ならLite、精度重視なら3 Flash。 
// モデル比較
Gemini 2.5 Flash (デフォルト)
  → 自動設定されるが性能はイマイチ

Gemini 3 Flash Preview
  → 手動で書き換え推奨！
  → 解釈能力が格段に高い

Gemini 3.1 Flash Lite Preview
  → 速度は速いが性能は少し落ちる

⑯裏側のコード確認方法

  • Canvasの「行動」タブを見ると、裏側で動いているモデルが確認できます。
  • 「Gemini」と検索すると、使用中のTTSモデルと生成モデルが表示されます。
  • デフォルトのモデル名を書き換えることで、より高性能なモデルに切り替え可能。
2.5 Flashを3 Flashに書き換えるだけで、出力が劇的に改善します！
