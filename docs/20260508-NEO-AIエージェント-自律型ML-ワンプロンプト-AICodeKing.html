① NEOとは: 自律型MLエンジニア

  • NEOはVS Code上で動作する自律型の機械学習エンジニアです
  • 1つのプロンプトから、データ準備・モデル学習・デプロイ・UI構築まで全てを自動実行します
  • 単なるコード補完ではなく、タスクを計画して段階的に実行・反復するAIチームメイトとして機能します
自動補完ツールとは違うよ！
プランニング→実行→検証→改善を
自分で回せるのがNEOの強み

  • ハイパーパラメータのチューニングは時間も手間もかかる作業
  • デプロイとUI構築は別エンジニアの担当で、3人の専門家が必要に
  • データサイエンティスト、バックエンドエンジニア、DevOpsの3職種分の作業が発生
モデルを作っただけでは終わらない…

  • VS Code拡張として動作し、サイドバーからすぐに利用可能です
  • ローカルのプロジェクトファイルやデータセットに直接アクセスして作業します
  • リポジトリを外部環境にアップロードする必要はありません
  • VS Code Marketplaceから無料でインストール可能です

  • AWS S3からデータセットを直接取り込み可能です

  • エンタープライズやプライベートデータを扱う環境に最適です

実践デモ: チャットモデレーション

⑦ デモの概要: チャットモデレーション

  • 目標はテキストメッセージから暴言・ヘイトスピーチ・脅迫を検出すること
  • データセットは一切提供しない状態で挑戦 — NEOがどう対応するか検証
  • 入力プロンプト: 「Build a chat moderation pipeline to detect profanity and harmful text in messages」

⑧ ステップ1: タスク計画の自動生成

  • NEOはすぐにコードを書くのではなく、まずワークスペースをスキャンして状況を把握します
  • データセットがないことを検知して、実行計画をユーザーに提示します
  • 計画内容: 合成データセットを生成してから学習を進めるという提案
いきなりコードを吐き出さない！
先に計画を見せてくれるから
何をするか把握できる

⑨ ステップ2: 合成データセットの自動生成

  • データ生成用のPythonスクリプトを自動作成します
  • スキーマ定義とアノテーションガイドラインも生成して、ラベルの一貫性を保証
  • 数秒で数千行のバランスの取れたCSVデータを生成
  • マルチラベル対応で、検証出力も作成。構造が学習に十分な信頼性を確保
CSVもスクリプトもスキーマも
全部ワークスペースに直接配置！
エクスポート/インポート不要 📁

⑩ ステップ3: モデルの自動学習

  • データ準備が完了すると自動的に学習フェーズに移行します
  • データを分析してベースライン分類器から開始。モデルの選択も自動です
  • 訓練用と検証用にデータを自動分割してローカル環境で実行
  • モデルを評価してレポートを生成し、ステージごとにログを記録

⑪ 詳細な実行ログと透明性

  • 実行ログにはタイムスタンプ、エラー、復旧アクション、パフォーマンス情報を含む
  • ブラックボックスではなく、何をしているか確認しながら進められる
  • 学習ステップが失敗した場合は原因を調査して対処可能
  • Weights & Biasesと連携すれば、プロ級の実験トラッキングも可能
透明的にログが見えるから
盲目的に信頼する必要なし！
これ一番嬉しいポイント
