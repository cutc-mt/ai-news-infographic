⑥

バックグラウンドで並列処理

  • 従来のAIブラウザ操作は乗っ取り型が多く、ブラウザを完全に占領してしまう
  • Codexはバックグラウンドで複数タブを並行処理しながら動作
  • 自分は普通にブラウザを使い続けられる。AIが別のデスクで仕事しているイメージ
画面を乗っ取らないから、
自分の作業を止めずにAIに仕事を任せられる！
⑦

対応サービスは原則すべて

  • GmailやGoogleスプレッドシートに直接アクセス可能
  • Salesforce、Notion、社内システムにも対応
  • サービス固有の連携設定は不要。ログイン状態をそのまま活用する
  • Chromeでログインしているサービスなら、原則すべてが対象
⑧

セキュリティ設計

  • 新しいサイトにアクセスするたびに確認ダイアログが表示される
  • 「今回だけ許可」か「常に許可」かを選択可能
  • 意図しないサイトをAIが勝手に操作することはない
業務ツールでも安心して使える設計がポイント！
⑨

データ保存の範囲

  • Chromeの全行動記録が保存されるわけではない
  • Codexが実際に読み取った情報だけが保存対象
  • ページから読んだテキストやスクリーンショット、チャットの呼び出し内容のみ
業務で使う場合は、機密情報を含むページを操作させる際に注意が必要
