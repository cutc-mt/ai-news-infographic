Agent View：エージェント管理の新機能
1

① Agent Viewとは何か

  • Claude Codeに追加されたAgent Viewは、全セッションを一元管理するダッシュボード
  • 複数のClaudeセッションを同時に並列実行できるようになった
  • ターミナルのタブを占有せず、1つのインターフェースで管理可能
  • 現在はリサーチプレビューとして提供中
CLIツールからエージェントOSへ！
これがClaude Codeの新しい姿

② 一覧で全体をリアルタイム把握

  • 左矢印キーを押すか、claude agentsコマンドで一覧を表示
  • アクティブに作業中のエージェントがすぐに分かる
  • ユーザーの入力を待っているセッションも一目で確認
  • タスクが完了したかどうかもリアルタイムで表示

③ セッションの覗き見とインライン返信

  • Enterキーでフルセッションに復帰可能
  • 現在のワークフローを離れずに他セッションへ介入
