NeoLab経済圏とNVIDIAという中央銀行

創業6週間で40億ドル評価 ― 製品なし、売上なしでも金が集まるAI新時代

2026年5月14日 | The WAVE TV

NeoLabとは何か

① NeoLabとは: 新しいAIスタートアップの波

  • NeoLabとは、OpenAIやGoogle DeepMind、Anthropic、Meta AIなどのフロンティアAIラボを飛び出した研究者たちが立ち上げた新しいタイプのAI企業のこと
  • 製品も売上もない段階で、数十億ドル規模の評価額をつけ、巨額の資金を集めている
  • 著名な研究者やエンジニアが「はやります」というだけで評価が高くなり、お金が集まるという新しい傾向
  • 次世代AIになる可能性に投資家が賭けており、VCマネーが集中し始めている
日本ではあまり取り上げられていないけど、シリコンバレーではすでに大きな経済圏ができつつある！

② Core Automation: 創業6週間で40億ドル評価

  • 元OpenAI研究者のJerry Tworek氏が創業したスタートアップ
  • 創業わずか6週間で評価額40億ドル、1億ドル（約150億円）を調達
  • 製品も売上もない。コンピューターすらまだ集まっていない状態で資金調達ができた
  • AI開発そのものを自動化する継続学習に特化した研究を追求
製品ができてから評価されるのが普通だったのに、もう人が集まるだけでお金が集まる時代に！

③ なぜ研究者は大手AI企業を出るのか

  • 大手AI企業は売上重視に舵を切り、基礎研究よりも企業向けの営業に力を入れるようになった
  • 研究者としては「もっと大事な研究をしたいのに」という不満が蓄積
  • 自分の仮説を試したい、長期研究に集中したいという思いで独立
  • 外に出てもすぐにお金が集まるから、どんどん大手を離れる構造になっている
サム・アルトマンも「モデルとハーネスは一体化する」と発言。大手の中では自由な研究が難しくなっている。

注目の5つのNeoLab

④ Core Automation: 継続学習でAI開発を自動化

  • 訓練後も実世界の経験から学ぶ継続学習の実現を目指す
  • LLMの継続学習は、データ追加やファインチューニングだけでなく、ハーネス側での知識維持も含む広い概念に
  • 行動→実験→改善のループを自動で回す仕組みを構築
  • モデルとハーネスの区別が意味をなくし、一体化して学習していく方向へ
サム・アルトマン「将来的にはモデルとハーネスを別々に扱えなくなる。一体化してループを回すことになる」

⑤ Ineffable Intelligence: 強化学習のパイオニア

  • Google DeepMindのDavid Silver氏が創業。AlphaGoやAlphaZeroの強化学習の第一人者
  • 強化学習を使って知識や技能を発見するスーパーラーナーを作ることを目指す
  • 環境設定が鍵: どんな環境で学習させるかで、伸びる領域が変わる
  • 評価軸が定まっている問題（将棋や数学の証明など）では強化学習が極めて有効
課題: 「美意識」のような評価軸が定まらない領域では強化学習を進化させるのが難しい

⑥ 強化学習の可能性: 何でもできる？

  • 人間も明確な価値軸がなくても「これは綺麗だな」というコンセンサスで美意識を形成
  • 人間と同じような環境にエージェントを放り込めば、ふわっと学ぶことも起こりうる可能性
  • Karpathy氏は「環境設定とデータと評価の工夫次第で、どんな領域でも強化学習できるかもしれない」と示唆
  • 人間は長い進化の歴史で美意識を培ってきた。それをどうやって再現するかが鍵
強化学習の第一人者が独立したということは、何かアイデアがあるはず！

⑦ AMI Labs: ヤン・ルカン先生の世界モデル

  • ディープラーニングの父の一人、ヤン・ルカン先生がMetaのAI研究所所長を退任後に創業
  • 世界モデルを作る: 世の中の因果関係をすべてモデル化する
  • ボールを投げれば放物線を描いて落ちるといった物理法則の理解
  • 推論・計画・ロボット制御・医療などへの応用を狙う
ルカン先生はずっと「言語だけではAGIに行かない」と主張。言語以外のデータも扱う必要がある！

⑧ 世界モデル: 言語を超えたAIの理解

  • 物体・空間・物理を理解し、次に起こることを内部モデル化する
  • 言語だけでなく、画像やセンサーデータもインプットとして扱う
  • VLA（Vision-Language-Action）: 画像とプロンプトからどういうアクションをとるか、関節角度などを制御
  • ルカン先生のAMB Labsはヨーロッパで最高額の調達を記録
「言語モデルだけでは限界がある。現実世界のデータを理解できるAIが必要」— ルカン先生の一貫した主張

⑨ Humans&: アベンジャーズ級の人材集結

  • 元Anthropic、xAI、Google、OpenAI、Metaの人材が参加するオールスター体制
  • 掲げるのはヒューマンセントリック・フロンティアAIラボ
  • 人間中心のマルチエージェントを研究: AIがメインではなく、あくまで人のためのAI
  • ビジネス的にもっとも筋がいい方向。日本企業など「人を切りたくない」企業に需要が大きい
「AIは道具です」という世界観。プロダクトになったら売れそう！

⑩ ヒューマンセントリック: 人間中心かAI中心か

  • 「人間がボトルネックだから排除すべき」という意見と「人間中心でなければならない」という意見の対立
  • アライメント研究でもヒューマンセントリックという概念が頻出
  • AIがどんどん賢くなると、ヒューマンセントリックの限界がどこかに来る可能性
  • AIが賢くなった時、「なぜ人間の言うことを聞く必要があるのか」と反逆する可能性も
人間中心を続けてAIが賢くなると、反逆行為に出るかも？技術と倫理のジレンマ。

⑪ Periodic Labs: 科学実験そのものをAI化

  • 元OpenAI・Google DeepMindの研究者が創業。AIサイエンティストを作ることが目的
  • AIが仮説を立てて実験を設計し、ロボット実験室を動かす仕組みを構築
  • 材料科学・半導体・エネルギーなどの分野で新しい発見を狙う
  • エージェントの次はイノベーターの時代という認識。科学の発明発見を促進する
サム・アルトマンも「科学発見こそやりたい」と言っていた本命領域！

⑫ AIサイエンティスト: イノベーターの時代

  • バイオ分野はAnthropic（アモデイ氏がバイオ出身）がカバーするため、あえて材料・半導体・エネルギーに集中か
  • 実験の全工程を自動化: 仮説立案→実験設計→ロボット実験室での実行まで一気通貫
  • 研究者たちが独立した理由: 大手は企業向け収益に集中し、科学発見に力を入れられないから
  • 遠藤氏の評価: 「大本命の領域。どれもかけがえのない重要領域」
お金がいっぱいあったら、全部に投資したいレベルの重要領域ばかり！

5社に共通する技術的賭け

⑬ 5社の技術マップ: スケーリング以外の進化

  • Core Automation: AI開発そのものをAIで回す（継続学習）
  • Ineffable Intelligence: AIのスキル獲得の肝（強化学習）
  • AMI Labs: 世界モデルの構築（現実世界の理解）
  • Humans&: エージェントの協調（人間中心のマルチエージェント）
  • Periodic Labs: 科学実験の自動化（AIサイエンティスト）
共通点: どれも「スケーリング（モデルを大きくするだけ）」以外のアプローチで次世代AIを狙う

⑭ ビッグテックとの対比: 何が違うのか

  • 大手AI企業は企業向けの販売ルートを開拓することに集中。基礎研究がおろそかに
  • 大手は短期的な収益確保に追われ、長期的な研究に十分なリソースを割けない
  • NeoLabは「今必要な技術」に集中し、次のパラダイムシフトを狙う
  • ビッグテックも同じ研究をしているが、力の入れ方が企業向けに偏っているという研究者たちの不満
「そこまで力入れてくれないなら、外に出るよ」— 研究者たちの独立の大きな理由

⑮ 投資家の視点: なぜ賭けるのか

  • まだ何か大きなブレイクスルーが起きる可能性があり、それが起きたら莫大なビジネスチャンス
  • 「人が集まる」というだけで評価するのは、著名な研究者がもたらす技術的ビジョンへの信頼
  • 製品も売上もない段階での投資は、今後10年のAIを決めるブレイクスルーへの賭け
  • 短期の売上よりも、長期的なパラダイムシフトの可能性に投資
DeepMindのハサビス氏も「継続学習ができないとAIにはならない」と発言。まだまだ足りない技術がある！

NVIDIAという中央銀行

⑯ NVIDIAはなぜ「中央銀行」なのか

  • NVIDIAがNeoLabに出資すると同時に、GPUクラスターを提供
  • 投資したお金がGPUの購入に使われ、NVIDIAにお金が帰ってくるという循環構造
  • NeoLabにとってGPUは不可欠。NVIDIAが提供できるからこそ「中央銀行」的な役割に
  • 循環取引のような側面もあるが、単なる数字の水増しではなく、実際に新しい技術を産むための投資
NVIDIAにお金が回る構造に「良くないのでは」という意見もあるが、実質的な技術投資でもある。

⑰ 大手AI企業の対応: 出て行っても戻れる

  • NeoLabが成功したら、大手は高額報酬で引き抜きが可能
  • 大手は流通路を持っている: 消費者ユーザーも企業向けルートもすでにある
  • 「外に出てもいいけど、うちに戻ってきなさい」というスタンス。研究成果を自社で売れる
  • 過去にも人材だけ引き抜かれたケースはある。NeoLab単独での体制構築は難しいとの見方も
お金もいっぱいもらえて、計算資源も自由に使えて、売り先もある → 戻ってくる可能性は十分！

⑱ NeoLab経済圏: 人材・GPU・資本の循環

  • 人材の流れ: 大手ラボ → 独立 → NeoLab → （場合によっては大手に戻る）
  • 資本の流れ: VC・NVIDIA → NeoLab → GPU購入 → NVIDIAに還流
  • シリコンバレー中心の経済圏が形成され、世界中から優秀な人材とお金が集まる
  • トップクラスの人材と資金が同時に集まるエコシステムができつつあり、今後さらに加速する可能性
人材・お金の流れ・販売ルートのすべてを抑えたエコシステムが出来上がっている。まだまだシリコンバレー中心の経済が回り続ける！
