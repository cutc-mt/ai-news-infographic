① Codexにリモコン登場: スマホから遠隔操作が可能に

  • OpenAIのCodexに新しくリモートコントロール機能が追加された。
  • スマートフォンからいつでもどこでも遠隔操作ができるようになった。
  • 2026年5月15日にリリースされたばかりの新機能。
これで外出中でもパソコンのCodexを操作できる！まさに「ポケットに入ったAIアシスタント」だね

② iPhoneミラーリング活用: デモに最適なMac標準機能

  • MacにはiPhoneミラーリングの標準機能がある。
  • パソコンのカーソルでiPhoneを直接操作できる。
  • 動画撮影やデモの際に非常に便利な機能。
この機能を知ってると、Codexのスマホ画面をPCに映して操作説明できるよ！

③ プロジェクトの作成と管理: スマホ専用プロジェクトを立てよう

  • PC版Codexの設定画面から「環境」を選択する。
  • 「プロジェクトを追加」でスマホ操作専用のフォルダーを作成。
  • スマホ側にChatGPTアプリ内のCodexからアクセス可能。
  • ただし、スマホ側にすぐに反映されない場合がある（不安定）。
スマホ専用プロジェクトを作っておくと、リモート操作がスムーズ！

④ リアルタイム同期の仕組み: PCとスマホがほぼ同時に動く

  • PC上でタスクを実行すると、スマホにもリアルタイムで反映される。
  • ラグをほとんど感じないレベルで同期的に動作。
  • リモート接続されたセッションは同じ内容を共有。
例えばPCで「今日のニュースを5件取得」と頼むと、スマホでもリアルタイムで進捗が確認できる！

⑤ 承認作業をスマホで実行: 通知が届くから見逃さない

  • 承認が必要な操作が発生すると、スマホに通知が届く。
  • スマホ側から承認ボタンをタップして即座に許可。
  • PC側にも承認されたことがリアルタイムで反映される。
通知が来るのが超ポイント！外出先でも承認作業ができるから作業が止まらない

⑥ 画像生成もスマホで確認: 出来上がりをその場でチェック

  • Codexが生成した画像もスマホ側で表示される。
  • スライド資料の完成品もスマホ内でビジュアル確認が可能。
  • データはPC上に保存されるが、サーバー経由でスマホでもプレビューできる。
  • PDFに変換して送り直す手間が不要になった。
画像やスライドがスマホで確認できるのは本当にありがたい！一気にCodexが好きになった

⑦ Claude Codeとの決定的な違い①: 新しいセッションをスマホから開ける

Codex vs Claude Code
新規セッション スマホから可能 不可
接続方法 アカウント連携 QRコード読み取り
  • Codexはスマホから新規タスクを直接開始できる。
  • 「新規タスク」→プロジェクト選択→指示入力→送信で即座に開始。
  • Claude CodeはPC側でセッションを作ってからQRで読み込む必要がある。
これができるかできないかで、本当のリモートコントロールかどうかが決まる！

⑧ Claude Codeとの決定的な違い②: 再接続が圧倒的にラク

  • Codexは一度接続したデバイスならトグルONで即再接続。
  • Claude Codeは毎回QRコードを読み込み直す必要がある。
  • 接続し直すのが面倒だと感じていた人はCodexの方が快適。
「これをClaude Codeにもやってくれよ！」と思ったポイント！再接続の簡単さは正義

⑨ UXの違いまとめ: OpenAIはユーザー体験にこだわる

  • OpenAIはユーザー体験（UX）の完成度が高い。
  • 「当たり前のようにできること」を自然に実装している。
  • Claude Codeはなぜ今までできなかったのだろうという印象。

再接続 ワンタップ QR再読み

⑩ 複数デバイスの接続: MacBookとMac Miniを両方繋ぐ

  • ノートPC（MacBook Pro）とMac Miniの両方をスマホに接続。
  • スマホ側でどのデバイスを操作するか選択可能。
  • デバイスではなくアカウント単位で管理されている。
  • 2台目の接続時はQRコードなしで自動的に認識。
家のノートPCか、事務所のMac Miniかを選べるのは熱い！

⑪ コントロールアザーデバイス機能: PC同士をリンク

  • 「Control Other Device」でPC同士を接続可能。
  • MacBookとMac Miniを相互にリモート操作できる仕組み。
  • 現時点ではまだ不安定な挙動が多い。
  • 接続が突然切れることもある。
PC同士のリモート操作は未来感あるけど、まだリリース直後で不安定。今後に期待！

⑫ 接続設定の手順: わかりやすい4ステップ

  • 1 Codexアプリをアップデートして「Get Started」を押す。
  • 2 「始める」→「許可」→スリープさせないをONにする。
  • 3 コンピューターユーズ機能の有効化＋Chrome拡張機能をインストール。
  • 4 QRコードをスマホで読み取り接続完了。
スリープ設定をOFFにしないと、作業中に止まってしまうので要注意！

⑬ スリープ設定の注意点: PCが寝たら作業も止まる

  • 「このコンピューターをスリープさせない」を必ずONに。
  • スリープモードになるとリモート接続が切断される。
  • 長時間の作業を任せる場合は電源接続も推奨。
PCが寝ちゃうとスマホからの操作もおやすみ。設定を忘れずに！

⑭ Chrome拡張機能の活用: ブラウザ連携でできること

  • CodexのChrome拡張機能をインストール可能。
  • ブラウザ上の操作もComputer Useで自動化できる。
  • 設定画面から簡単にインストール。
Chrome拡張を入れておくと、ブラウザ操作もCodexに任せられる！

⑮ 現時点での不安定性: リリース直後の注意点

  • リリース直後で不安定な部分がまだ多い。
  • プロジェクトがスマホに反映されないことがある。
  • 接続が突然切れることも。
  • 解決策: ChatGPTアプリを一度ログアウトして再ログインすると改善。
ログアウト→ログインし直すだけで接続トラブルが解消することも！
