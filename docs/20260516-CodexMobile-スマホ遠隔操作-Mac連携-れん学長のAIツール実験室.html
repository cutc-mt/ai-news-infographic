Codex Mobileでスマホから自宅MacのCodexを遠隔操作！

設定が簡単すぎて作業の仕込みが変わる - ChatGPTアプリから自宅PCへ即座に指示を送る

2026年5月16日

Codex Mobileとは？

① Codex Mobileとは: スマホからMacのCodexに接続

  • ChatGPTスマホアプリから手元のMacで動いているCodexに接続できる機能です。
  • スマホから作業の続きを送ったり、Codexからの確認に答えたり、必要な操作を認証できます。
  • 単なるチャットではなく、Mac側のプロジェクト情報や実行結果を見ながら続きができるのがポイントです。
  • 現時点ではプレビュー機能として提供されています。
外出中に思いついた作業を投げる、帰宅前に文字起こしや調査を始める、といった使い方と相性が抜群です！

② できることの全体像: スマホとPCをシームレスに連携

  • スマホからCodexに指示を送信してPC側で実行。
  • PC側のプロジェクト情報や実行結果をリアルタイムで確認。
  • Codexからの確認や認証要求にスマホから回答可能。
  • 複数のPCを登録して、どこからでも自分の環境にアクセス。
いつも使っているスキルもそのまま動くから、学び直しが不要です！

③ これが変わる: PCの前にいなくても作業が進む

  • これまではPCの前に座っている時しか作業できませんでした。
  • 今は外出先からスマホで即座に指示を送れます。
  • 思いついたアイデアをその場でCodexに投げて、PC側で処理を開始。
  • 「PCの前にいない時間」を活かすという新しいワークスタイルが実現。
外出中の第2の脳として、自宅PCに届いた作業ロボットへの指示端末として使うのがベストです！

④ 普段のCodexとの違い: ローカル環境とリモートの連携

  • 通常のCodexはパソコンの前でコードを書かせたり、ファイルを調べさせたりするもの。
  • Codex Mobileは同じプロジェクトにスマホからアクセスして続きができる。
  • 実行自体はMac側で行われるので、スマホの性能に依存しないのが特徴。
  • Macが起動していることが前提条件で、リモートから安全に接続。
設定手順

⑤ スマホ側の準備: アプリ更新から接続設定まで

  • ChatGPTスマホアプリを最新版に更新します。
  • アプリを開いて「Codex」という項目をタップ。
  • 「Codex Mobileを接続する」を押すと「デスクトップアプリを待っています」と表示されます。
  • この時点でMac側の設定に移ります。所要時間は約1分。
スマホ側はこれだけで準備完了。とてもシンプルです！

⑥ Mac側の設定と認証: Codexでセットアップを実行

  • まずCodexを最新版に更新しておきます。
  • Codexに「Codex Mobile設定」の項目が表示されるので選択。
  • セットアップを開始して「このコンピューターを操作することを許可する」を選択。
  • 「モバイル端末で認証」と表示されるので、スマホ側で承認します。
「安全なセットアップを完了しています」と表示されたら接続完了です！

⑦ スマホでの認証フロー: OpenAI認証から完了まで

  • Mac側でセットアップ開始後、スマホに「ChatGPTがサインインのためにOpenAI.comを使用」と表示。
  • 「続ける」をタップすると認証プロセスが開始されます。
  • 「このデバイスを認証する（Authorize）」を実行。
  • 「安全なセットアップを完了」と表示されてMacBookProローカルがスマホに表示されます。 
認証フロー:
Mac側: セットアップ開始
  → スマホ: OpenAI認証画面
    → Authorize（認証）
      → 完了 → MacBookProローカル表示
実際に使ってみる

⑧ 最初の指示送信: 「今日のTODO教えて」が一瞬で届く

  • スマホのCodexチャットから指示を入力。
  • 「今日のTODO教えて」と送信すると、一瞬でPC側に指示が届く。
  • 普段使っているTODOスキルがそのまま発動して結果を返してくれます。
  • 日時ファイルがない場合は前日・終日の記事から今日の候補を自動で組み立ててくれます。
めっちゃ簡単に設定できました！スキルもちゃんと動いて感動です！

⑨ スキルとプランモード: スマホでもフル機能が使える

  • 通常のスキルがそのまま動作します。TODO管理やメモ機能もOK。
  • スマホ版にもプランモード（Plan Mode）があり、実行前に計画を立てられます。
  • ただしスラッシュコマンド（/）の入力が難しいので、スキルの直接呼び出しは不便。
  • プラグインの選択もスマホから可能。PCと同じように柔軟に設定できます。
スマホのキーボードだと「/」が打ちづらいのが唯一のネックですね。

⑩ 権限設定: フルアクセス・カスタムで柔軟に制御

  • PC側のデフォルトは「自動フルアクセス」。スマホからも同じ権限で操作可能。
  • カスタム権限も追加されていて、細かく制御できます。
  • スマホから操作する際は権限レベルを適切に設定しておくことが大切です。
  • スマホ版で見える設定やプラン、権限はPC版とほぼ同じ。

⑪ 複数PC登録: どこからでも自分の環境にアクセス

  • 複数のPCを登録可能で、どのPCにもリモート接続できます。
  • 自宅のMacBook、会社のMacなど、複数台のMacでCodexを使っている場合に非常に便利。
  • スマホから接続先のPCを選択して、目的の環境で作業を指示。
  • PC間で切り替えもスムーズ。プロジェクトごとに別のPCを使い分けていても問題なし。
複数PCを持っている人にとっては、これが一番嬉しい機能かもしれません！
便利な使い方

⑫ 外出先での活用: TODO追加から進捗確認まで

  • 外出中にTODOを追加・整理して、帰宅後すぐに作業開始。
  • 移動中にプロジェクトの進捗確認ができる。
  • 思いついたアイデアをその場で即保存。メモするより確実で速い。
  • コンテンツアイデアを投げて、PC側で構成案を自動生成させることも可能。
「YouTubeのこのテーマでノート記事の構成だけ作って」みたいな指示も外出先から投げられます！

⑬ コンテンツ制作での活用: 構成案からサムネまで

  • YouTube動画の構成をスマホから指示してPCで作成。
  • 「サムネ候補を3方向で出して後で見るように保存して」とリモートで画像生成。
  • ノート記事の構成案だけを作成させて、後でPCで仕上げる使い方も。
  • 長い処理（文字起こし、BGM追加など）を外出先から開始して帰宅時に完成。
帰宅したらもう作業が終わっている、という状態を作るのが理想的です！

⑭ EOD振り返りと日記: ベッドや移動中で振り返り

  • EOD（End of Day）振り返りをベッドや移動中に行える。
  • 音声入力でゴロゴロしながら入力する方がやりやすい場面も。
  • 今日の日記的な振り返りをスマホからCodexに投げて記録。
  • メールの処理やTODOの整理も寝る前にサクッと完了。
疲れた時でもスマホの音声入力ならラクに振り返りができます！

⑮ 帰宅前の仕込み: PCに戻ったらすぐ作業開始

  • 帰宅前に関連ファイルを調べておくと、帰宅後すぐに作業に取り掛かれる。
  • 必要なパッケージのインストールや環境準備をリモートで済ませる。
  • 開発作業そのものよりも、事前調査と準備に向いている。
  • プログラミングを使わない方でもタスク管理や進捗確認に活用しやすい。
「帰ったらこれやる」を事前に仕込んでおくと、時間の無駄が減ります！
注意点とまとめ

⑯ スマホからやらない方がいい操作: 取り返しのつかない作業は避ける

  • 本番公開はスマホから行わないこと。
  • 送信操作（メール送信等）も確認不足のリスクが高い。
  • 削除・大量リネームは取り返しがつかない可能性がある。
  • 設定変更・Git Pushも小さい画面では確認が不十分になりがち。
  • 基本ルール：下書きまで・候補作成・確認だけに留めること。
最終実行は必ずPCで確認してから！スマホでは「準備」までにしましょう。

⑰ Macを起動したまま使う運用: スリープ防止の設定が必須

  • Codex Mobileを使うにはMacが起動している必要があります。
  • Macのスリープ設定をオフにして、外出中も起動したままにする運用が基本。
  • 電源に接続した状態でスリープしない設定にするのが推奨。
  • セキュリティ面でもMacのロック設定と併用することをおすすめします。
Macの「システム設定 → スリープ」から「スリープしない」に変更するだけでOKです！

⑱ まとめ: 外出中の第2の脳として活用しよう

  • 設定は非常にシンプルで、スマホとMac両方で数ステップの操作だけ。
  • 指示は一瞬でPCに届き、普段のスキルもそのまま動く。
  • 外出中の第2の脳として、自宅PCへの作業指示端末として使うのがベスト。
  • 便利な場面：TODO追加、進捗確認、コンテンツ保存、仕込み、EOD振り返り。
  • 危ない操作：本番公開、送信、削除、Push。最終実行は必ずPCで。
CodexをPCの前だけで終わらせたくない方に、ぜひ試してほしい機能です！
プレビュー機能なので、これからさらに進化しそうですね。
