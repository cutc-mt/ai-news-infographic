  • Cloudflare・Ciscoといった大手セキュリティ企業と政府機関がパートナーとして参加
  • 「Daybreak（夜明け）」という名前はサイバー防衛における夜明けを意味する

Codexスマホ対応：外出先からAIに指示

⑯スマホのChatGPTアプリからCodexを操作

  • CodexはOpenAIのAIコーディングエージェント。ファイル検索・修正・テスト・レポート作成を自動化
  • 毎週400万人以上が利用する人気ツールだが、外出中はPCがないと確認できなかった
  • 今回スマホがリモコン役になり、PCは動き続けながらスマホから指示・確認・承認が可能に
  • ファイルや認証情報はPC側に残ったまま。スマホに機密情報は保存されない
  • iOS・Android両対応。無料プラン含む全プラン対象。日本を含む全サポート地域で利用可能
PCの前に縛られない新しい
開発スタイルの始まり！

⑰スマホでできる4つのことと設定手順

  • 1 新しい作業の開始 → テキスト入力欄から自然な言葉で指示
  • 2 出力や進捗のリアルタイム確認 → スクショ・コード変更・テスト結果を全て確認
  • 3 次のステップの承認や指示 → 承認待ちの通知が来たらワンタップ
  • 4 使用するAIモデルの変更 → 目的に応じてモデルを切り替え
  • 設定手順：アプリ更新 → Codexタブ → Connections画面でPC追加 → 承認 → 完了
