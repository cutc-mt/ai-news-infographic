Hermes AgentがX Premiumで利用可能に

⑦ X PremiumでHermes Agentが利用可能に

  • X Premiumサブスクリプション（青バッジ）でHermes Agentが利用可能になった。
  • X Premium（月額約1,000円）とPremium+（月額約6,000円）のどちらでも対応。
  • これまでGrokサブスクからの利用だったが、X Premiumでもアクセス可能に拡大。
月額1,000円のPremiumでもOKなら、利用のハードルが大きく下がった！

⑧ X API不要で投稿検索が可能に

  • X APIを使わずにHermes AgentでXの投稿を検索できるようになった。
  • X APIを使っていたユーザーがHermes Agentに乗り換える動きがちらほら。
  • ただしAPIと比較して精度や自由度がまだ劣るという声も。
  • 設計の難しさはあるものの、自由度は向上している。
APIからの移行組もいるが、精度の面でまだ課題あり。今後の改善に期待！

⑨ Hermes Agentの利用ハードル

  • X Premium対応で利用ハードルは低下した。
  • しかし環境構築やセットアップの難しさは変わらず。
  • 技術的な知識が必要な部分が多く、初心者にはまだ厳しい印象。
料金面では入りやすくなったが、技術的な壁はまだ高い。興味がある人はまず試してみて！
