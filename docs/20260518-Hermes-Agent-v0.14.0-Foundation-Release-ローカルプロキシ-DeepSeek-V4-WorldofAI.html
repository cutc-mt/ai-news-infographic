Hermes Agent v0.14.0 Foundation Release

Nous Research が贈るオープンソースAIエージェントの最大規模アップデート

2026年5月18日

① Foundation Release の概要

  • Nous Research がHermes Agent v0.14.0をリリース。コードネームは「Foundation Release」
  • MITライセンスのオープンソースプロジェクトとして提供
  • 24時間365日稼働可能な自律型AIエージェントシステム
  • 長期記憶、再利用可能なスキル、ユーザー理解を自律的に構築
  • 使えば使うほど性能が向上する自己進化型の仕組み
単なるAIツールではなく、オープンソースのAgentic AI OSとして進化を遂げた！

② ローカルプロキシ機能

  • 1つのサブスクでClaude、ChatGPT、Grokを横断利用可能
  • プロキシレイヤーが認証とルーティングを自動処理
  • 各アプリケーションごとにAPIキーを分ける必要がなく、セットアップが簡単に
  • マルチエージェントワークフローもシンプルに構築できる
Codex CLI、Aider、Cline、Continueなどのツールが既存のサブスクだけで動くようになる！

③ OpenAI互換 OAuth プロキシ

  • hermes proxy コマンドでローカルエンドポイントを即座に起動
  • OpenAI互換のAPIをlocalhostに立ち上げ、ログイン中のサブスクで動作
  • Codexプランなどの既存のサブスクがそのまま利用可能
  • 別途APIキーは不要、1つのサブスクでHermes内ですべて完結 
$ hermes proxy
# → localhostでOpenAI互換APIが起動
# → ログイン中のサブスクで即利用可能

④ SuperGrok 統合

  • SuperGrokサブスクがHermesに直接統合
  • APIキー不要、ブラウザログインのみで利用開始
  • セットアップは約60秒で完了、全ティアに対応
  • X（Twitter）のリアルタイム検索が自律リサーチに活用可能
Xから情報を収集し、トレンドを要約してワークフローにフィードバックする自律リサーチエージェントが構築できる！

⑤ Grok 4.3 フル機能活用

  • Grok 4.3テキストチャットで高度な対話が可能
  • Grokのテキスト読み上げ（TTS）に対応
  • Imagine による画像生成と動画生成をネイティブサポート
  • X検索でリアルタイムなTwitterリサーチがワークフロー内で完結
  • 1MコンテキストのGrokワークフローにも対応

⑥ インストールとアップデート

  • curlコマンドで簡単にインストール可能
  • hermes setupで初期設定を完了
  • hermes updateで最新版に一発アップデート
  • Windowsではpip installにも対応 
# インストール
$ curl -fsSL [URL] | bash
$ hermes setup

# アップデート
$ hermes update

# Windows (pip)
$ pip install hermes-agent
プラットフォーム & パフォーマンス

⑦ ネイティブWindows対応（ベータ）

  • ネイティブWindows対応が正式にベータリリース
  • 複雑なWSLやLinux環境の構築が不要に
  • ターミナル・プロセス管理、Python/NPM環境の問題を修正
  • ネイティブPTY処理、ファイルパス管理、Windows CLI動作を改善
  • 公式PyPIパッケージで pip install に対応
Windowsユーザーにとって自律型AIへのアクセスが大幅に改善された！

⑧ Web UI ダッシュボード

  • hermes dashboard --tuiでWeb UIを起動
  • スキルやプラグインをGUIで簡単に設定
  • cronジョブでスケジュールタスクの実行が可能
  • 設定を一元管理して自律動作を容易に構成
ブラウザ上でHermesのすべての設定を管理できる便利なダッシュボード！

⑨ パフォーマンス大幅改善

  • 起動が約19秒高速化（コールドスタート最適化）
  • 重いアダプタとプラグインは必要時のみロードに変更
  • モデルカタログをローカルでキャッシュし、ネットワークアクセスを削減
  • 並列起動チェックでセットアップと起動の体感速度が劇的に向上
遅延ローディング＋キャッシュ改善でスナッピーな起動を実現！

⑩ ブラウザ自動化の進化

  • Chrome DevTools Protocolの永続接続に対応
  • 操作のたびにブラウザセッションを再起動する必要がなくなった
  • 一部のワークフローで最大180倍高速化を達成
  • Lightpandaブラウザバックエンドの統合でさらに高速な自動化を実現
数秒かかっていたブラウザ操作がほぼ瞬時に返るように！

⑪ プロンプトキャッシュ & メモリ

  • Claudeベースのワークフローで1時間のプロンプトキャッシュに対応
  • セッションをまたいだ初回応答が高速かつ低コストに
  • メモリレビューシステムの改善とバックグラウンド性能タスクを強化
  • クロスセッションでメモリキャッシュを永続化

⑫ ネイティブAI動画生成

  • Hermesエージェントが自律的に動画を生成可能に
  • 統合された video generate システムでワークフロー内で完結
  • 自動クリップ作成、ビジュアルコンテンツの自動生成に対応
  • スケジュールベースでマルチメディアワークフローを自律実行
AIエージェントが動画制作パイプラインを全自動で構築できる時代に！

⑬ DeepSeek V4 Flash 無料提供

  • DeepSeek V4 FlashがNous Portalで期間限定無料
  • Navita AIの協力により実現した無料提供
  • 最強クラスのオープンソースエージェントモデルを無料で利用
  • 自律ワークフロー、コーディング、推論、ロングコンテキストタスクに対応
最強のオープンソースモデルが完全無料で使えるのは超お得！

⑭ /handoff ライブセッション転送

  • 高速モデルで開始して、深い推論モデルにシームレスに移行可能
