⑥HERMES Agentのインストール手順

  • HERMES Agent公式ドキュメントのQuick Startページを参照
  • Macの場合はターミナルアプリを開く
  • ダウンロード用のコマンドを入力してエンター
  • ClaudeやChatGPTに手順を聞きながら進めるのがおすすめ
$ hermes-agent install
インストール方法はAIに聞くのが一番早いです。ページURLを渡して手順を聞いてみましょう！

⑦xAI OAuth認証のセットアップ

  • xAI公式ドキュメントのhermes add xai-oauthコマンドを実行
  • ブラウザで承認ページが表示される
  • 「許可する」をクリックすれば接続完了
$ hermes add xai-oauth
# ブラウザが開く → 許可する → 完了
注意: OAuth認証でエラーが出る場合があります。その場合はしばらく待ってリトライしてください！

⑧HERMES ToolsでX Searchを有効化する

  • CLIでhermes toolsコマンドを実行
  • 設定画面が表示されるので、X Searchをチェックする
  • 方向キーで移動、スペースキーでチェックON
  • 動画生成や動画分析も同時に有効化できる
$ hermes tools
# 設定メニューが表示される
# → xai-api-key (X Search) をチェック
# → エンターで確定
この手順を忘れるとX検索が使えません！ネット上の記事では抜けがちな手順です。

⑨モデルをGrok 4.3に切り替え

  • CLIでhermesと入力してエージェントを起動
  • /modelコマンドでモデル一覧を表示
  • xAI Grok 4.3を選択して変更
  • 利用量のダッシュボードが表示されて便利
$ hermes
hermes> /model
# xai-grok-4.3 を選択
Grok 4.3がおすすめ。X Searchと相性が良いです。
