価格と利用方法

⑩API料金は前世代の約3倍に

  • 入力: $1.50/百万トークン、出力: $9.00/百万トークン。
  • キャッシュ入力は$0.15/百万トークンと格安。
  • Gemini 3 Flash比で約3倍だが、フラッグシップの半額以下。
  • 非グローバル地域は入力$1.65、出力$9.90。
3倍に上がったけどフラッグシップ比だとまだ激安です！

⑪巨大なコンテキストウィンドウ

  • 100万トークンの入力コンテキストウィンドウ。
  • 出力は最大64,536トークン。
  • 大量のドキュメントやコードベース全体を一括処理可能。
  • 動画やPDFファイルも直接入力として処理できる。

⑫無料利用と複数アクセス方法

  • Geminiアプリで完全無料で利用可能。
  • Google検索のAIモードでもデフォルトモデルに採用。
  • 開発者はGoogle AI StudioからAPIアクセス可能。
  • エンタープライズはGemini Enterprise Agent Platformで利用。
無料から使えるのは本当に嬉しい！まずは触ってみよう！

Google Antigravity 統合

⑬Antigravityとは何か

  • GoogleのAI駆動統合開発環境（IDE）。VS Codeベース。
  • 「エージェントファースト」設計でAIが自律的に計画・実行・検証。
  • エディタビューとMission Controlダッシュボードの2画面構成。
  • VS Code拡張機能も活用可能な柔軟な環境。
Cursorの対抗馬になるかも！注目のツールですね。

⑭Flash × Antigravity の相乗効果

  • エージェントハーネスがGemini 3.5 Flashと共同最適化。
  • 組み合わせでコーディングが12倍速に高速化。
  • 複数エージェントの並行稼働で大規模タスクを効率処理。
  • Antigravity 2.0でFlashが無料で利用可能。

⑮Antigravity SDKと多様なサポート

  • Python SDKでプログラムからエージェントを構築可能。
  • CLI（コマンドライン）ツールも提供。
  • デスクトップアプリとしてWin・Mac・Linux対応。
  • Claudeやオープンソースモデルなど複数プロバイダーに対応。
  • AIネイティブ開発でエージェントがSDKコードを直接読み書き可能。
