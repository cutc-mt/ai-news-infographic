①Google I/O 2026 開幕: 現地サンフランシスコから速報

Googleが年1回開催する世界最大規模のAIテックイベントがついに開幕。世界中からAIインフルエンサーが集結し、サンダー・ピチャイCEOやDeepMind社長の基調講演が行われた。

  • 会場はスタジアム形式でコンサートホールのような大規模な空間
  • 2〜3時間にわたる発表で、内容の量が膨大
  • 世界各国のインフルエンサーが集結
  • 大きく5分野：個人・開発者・企業・コマース・ハードウェア

②Gemini Omni: 全モーダル対応の世界モデル登場

テキスト・画像・音声・動画をすべて入力できて、すべて出力できる全モーダル双方向の世界モデルが登場。動画生成の精度が劇的に向上した。

  • テキスト・画像・音声・動画を全対応の入出力
  • 動画を入力して動画を出力する編集も可能
  • 画像から動画の生成にも対応
  • 会話しながら動画をどんどん変更できる
Geminiのチャットから「＋」ボタンで動画生成を選択。ベータ版ではなくOmniが搭載されている！

③Gemini Omniの衝撃: 競合を凌駕する動画生成

ByteDanceのCance 2.0を超える可能性が高いと評価。会話形式で動画を自在に変更でき、これほどの世界モデルを持つ会社はほぼ存在しない。

競合状況：
Runway Pika 動画特化モデルは多数存在
しかしGoogleが本気を出すと一気に格局が変わる
  • OpenAIは動画生成から撤退済み
  • Claude（Anthropic）は画像・動画をそもそもやっていない
  • フロンティア系モデルでこれほどの動画生成を実現したのはGoogleだけ
API解放も予定！Claude Code経由で動画がバンバン作れるようになるかも

④Gemini 3.5 Flash: 高速で高性能な衝撃の新モデル

Gemini 3.1 Proよりも性能が高いのに速度は4倍という驚異的なモデルが登場。ベンチマークも高く、価格は少しだけ高いが実用性は抜群。

  • Opus 4.7より約4倍高速
  • 3.1 Proよりも精度が高くて速い
  • 値段は少しだけ高いが、気にするレベルではない
  • API経由でサービス組み込みに最適
Claude Codeが遅いと感じている人にとって、3.5 Flashを搭載したAntigravityは魅力的な選択肢に

⑤Gemini Spark: Google版AIエージェントがついに登場

CodexやOpenArtを混ぜたようなAIエージェントがGeminiのデスクトップアプリに登場。24時間動作し、ブラウザ操作から各種アプリ連携まで何でもできる。

  • 24時間バックグラウンドで稼働
  • MCP対応でサブエージェントも利用可能
  • ブラウザ操作を自動実行
  • スマホアプリからも操作可能
出張手配、Gmail送信、GoogleシートからSlack連携などOpenArt的な使い方ができる！

⑥Sparkが日本企業に与えるインパクト

日本法人にとってClaude Codeの導入ハードルは依然として高い。しかしGoogle環境の会社は多いため、Sparkのようなコパイロット系ツールは非常に使いやすい。

  • 日本企業は慣れていないツールや申請されていないツールを使えないことが多い
  • Google環境を使っている会社ならSparkをスムーズに導入可能
  • MicrosoftはすでにCopilot CoWorksを搭載済み
  • Googleもようやくコパイロット的な機能を提供開始
ChatGPTのオペレーターやCodexに近い存在。少し遅い気もしたが、Google環境の企業には大きい

⑦Antigravity 2.0: AIコーディング環境の進化

Googleの開発ツールAntigravity 2.0が公開。複数AIの同時実行、Firebase連携、スケジュール実行など、Claude CodeやCodexに追随する機能が充実した。

  • 複数AIを同時に実行可能
  • Firebase連携でバックエンド開発もシームレス
  • スケジュール実行に対応
  • CLI（コマンドライン）も利用可能
Antigravity CLIを起動するとClaude Codeのようなターミナル環境が立ち上がる。デフォルトは3.5 Flash搭載

⑧AIコーディング三つ巴: Antigravity vs Claude Code vs Codex

3つのAIコーディングツールが激しい競争を繰り広げている。それぞれの強みと弱みを整理すると、用途によって最適な選択が変わる。

Claude Code：デフォルトはOpus 4.7。30分のタスクでも遅さを感じることが増えた

OpenAI Codex：基本GPT-5.5を使用。レビューのみCodex利用可能

Antigravity：3.5 Flashがデフォルトで高速。無料枠もある
めちゃくちゃ早い3.5 Flashを使えるのがAntigravityの最大の強み！Claude Codeの乗り換え先として有力

⑨Google Flow: Adobe並みのデザインツールに進化

GoogleのデザインツールFlowが超進化。動画・画像・音声・キャラクター・アバターを全方位で最新モデルで作成可能。コード不要で自作ツールも構築できる。

  • チャットしながら細かくデザインを調整
  • 画像編集、リサイズ、カスタムシェーダーに対応
  • キャラクターやアバターの生成も可能
  • コード不要でツール作成も実現
普通にAdobeじゃんって感じ！Polarrとか動画系AIスタートアップとバチバチやりそう

Samsungとコラボし、Gentle Monsterと共同デザインしたAIグラスが今年の秋に発売予定。音声会話やスマホ操作が可能な次世代ウェアラブル。

  • 今年の秋頃に発売予定
  • メガネをつけて音声会話ができる
  • iPhoneやAndroidと連携してスマホ操作

⑪Agentic Search: 検索にエージェント機能が追加

Google検索にエージェント機能がプラスされた。24時間バックグラウンドで監視し、指定した条件で通知を飛ばすことができる。自動リサーチツールがデフォルトで使える。

  • 24時間バックグラウンドで監視可能
  • 指定条件を満たすと通知を送信
  • 例：「AIプロダクトの資金調達・米国・シリーズB以上」を自動監視
  • 検索結果をそのままおしゃれに表示する機能も追加
検索＋実行＋他の作業の同時進行ができるのが強い！ChatGPTやClaudeにも類似機能はあるがGoogleの連携力は別格

⑫Generative UI: 検索結果をインタラクティブに表示

Generative UI機能により、検索結果を物理シミュレーションや数学のグラフのようにインタラクティブに表示可能。レバーを動かすと画面が変わるような体験が実現した。

  • 物理学や数学の図表をインタラクティブに表示
  • パラメータを動かしてリアルタイムに変化
  • 教育用途に特に有効
  • Search Agentとして検索結果の監視も可能
物理学や数学を学ぶ時にかなり使える！視覚的に理解しやすい表示が魅力

  • Google検索→そのまま購入までシームレス
  • YouTubeやGmailからも直接購入可能
  • 複数企業が同じ規格を採用するMCP的アプローチ
