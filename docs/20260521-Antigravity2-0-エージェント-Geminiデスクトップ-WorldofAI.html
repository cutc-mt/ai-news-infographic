Antigravity 2.0 完全解説

エージェントAIコーディング × Geminiデスクトップアプリの全貌
2026年5月21日

①Google Developer Conference: Antigravity 2.0発表

  • Google Developer ConferenceでGemini 3.5 FlashとGemini Omniがリリースされた
  • その中でAntigravity 2.0が発表され、多くのユーザーが混乱することになった
  • GoogleはAntigravityを複数の独立したアプリに分割する大胆な方針転換を行った
YouTubeコメント欄では「Googleは一体何をしているんだ？」と大混乱！

②Antigravityの分割: 2つの独立アプリへ

  • Antigravity IDEはVS Code風の独立したIDEアプリとして分離された
  • Antigravity 2.0はエージェントチャットインターフェースのスタンドアロンアプリに変身
  • 既存のAntigravityを開くと自動的に2.0にアップグレードされるが、IDEは別途ダウンロードが必要
エージェントアプリはCodex風、CLIはClaude Code風。Googleは既存ツールを参考にしている！

③動的サブエージェント: 並列処理の革命

  • メインのAIエージェントが自動的にヘルパーエージェントを起動して並列作業を実行できる
  • 1つの巨大なコンテキストウィンドウに詰め込むのではなく、複数の専門エージェントにタスクを分散
  • タスクリストを自動的にブランチ分けし、各エージェントが専門領域を自律的に処理する仕組み
1つの過負荷エージェントより、複数の専門エージェントで分担する方が効率的！

④非同期タスクとスケジュール実行

  • 非同期タスクにより、エージェントがバックグラウンドで継続動作し、ユーザーの作業をブロックしない
  • スケジュールタスクはcronジョブのようにタイマーベースで自動実行できる
  • ClaudeのGoalシステムやCodexワークフローに強くインスパイアされた設計
エージェントが自分で予定を管理して自動実行。まるでAI秘書！

⑤新しいプロジェクトシステム

  • 会話が単一リポジトリに紐付くのではなく、複数フォルダにまたがるプロジェクトとしてグループ化される
  • プロジェクトごとにカスタム権限とワークフローを設定可能
  • プロジェクト全体ルールを追加して、エージェントの動作をプロジェクトレベルで制御できる
プロジェクトシステムが美しく進化！複数リポをまたぐ作業が容易に！

⑥新スラッシュコマンド: /goal, /browser, /grill-me

  • /goal: タスクが完全に完了するまでAIに作業を継続させるコマンド
  • /browser: ワークフロー内でブラウザの使用を強制するコマンド
  • /grill-me: AIが開始前に確認の質問をしてくれるコマンド。曖昧さを排除
/grill-meは「本当にそれでいいの？」とAIに問い返させる画期的なコマンド！

⑦ライブ音声文字起こし

  • ライブ音声書き起こし機能が追加され、音声でプロンプトを直接入力できるようになった
  • エージェントのワークフローに音声で指示を出すことが可能に
  • カスタマイズ可能なスキルやJSONフックも追加され、エージェントの拡張性が向上
話しかけるだけでエージェントが動く。キーボード不要の時代へ！

⑧Geminiデスクトップアプリ登場

  • Antigravity 2.0はスタンドアロンのデスクトップアプリとして動作する
  • AnthropicのCo-workやOpenAIのCodexに似たローカルAIエージェントアプリをGoogleが構築
  • サブエージェントを管理し、自律的なワークフローに特化したインターフェース
Google独自のローカルAIエージェントアプリ。ブラウザ不要で直接操作！

⑨Antigravity CLI: ターミナルから直接操作

  • 旧Gemini CLIがAntigravity CLIにリブランドされた
  • ターミナルから同じモデル、同じエージェントハーネス、同じ自律ワークフローを利用可能
  • キーバインド、テーマ、ターミナル設定にフルカスタマイズ対応
Claude Codeとそっくりなアプローチ。CLI派も満足の出来栄え！

⑩Antigravity SDK: カスタムエージェント構築

  • GoogleのAntigravityを支えるエージェントハーネスに直接プログラムアクセスできるSDK
  • pipでインストール可能。カスタム自律エージェントシステムを自分のアプリに組み込みできる
  • デプロイ、インフラ、ワークフローをフルコントロールしながら同じオーケストレーターを利用 
pip install antigravity-sdk
コンシューマーアプリだけでなく、独自のエージェントアプリも構築可能！

⑪Antigravity IDE: 独立した開発環境

  • VS Code風の独立IDEとして分離され、2.0の全機能を引き続き利用可能
  • Antigravityのウェブサイトから別途ダウンロードが必要。自動更新ではIDEは開かない
  • 初回セットアップ時にGoogleプラグインのインストールを推奨
IDEは完璧だったのに分割したのは少し残念という声も多数...

⑫推奨Googleプラグイン一覧

  • Modern Web Guidance: コーディングエージェントを最新のウェブベストプラクティスに常に更新
  • Chrome DevTools: 信頼性の高いブラウザ自動化を実現
  • Antigravity SDK: プラグイン経由でもSDKをインストール可能
全プラグインのインストールを強く推奨！特にModern Web Guidanceは必須級！

⑬antigravity-preview-05-2026モデル

  • Google AI Studioにひっそりと追加された新しいエージェントモデル
  • GoogleがホストするリモートLinux環境で動作する汎用自律AIエージェント
  • 単なるチャットボットではなく、コードの実行、実際のアクション、ツールの使用が可能で、大規模なトークン予算を持つ
ほとんどの人が気づいていない注目のモデル。Antigravityにも近日デプロイ予定！

⑭AI Ultra購読者向け1,000クレジット

  • Google AI Ultraの購読者は追加の1,000 AIクレジットを無料で請求できる
  • アプリ内のRedeemボタンをクリックするだけで取得可能
  • しかし使用量の制限は以前より大幅に厳しくなったという報告が多く、ユーザーの不満が増加中
クレジットをもらっても制限が厳しい...というユーザーの声が相次いでいる！

⑮新しいアーティファクトと可視化

  • 新しいアーティファクトシステムにより、スタンドアロンアプリ内ですべてを可視化できる
  • コード、図表、ドキュメントなどをアプリ内で直接プレビューして確認可能
  • 改善されたマルチプロジェクトサポートとアーティファクトレビューフローで効率的な作業を実現
成果物をその場で確認できるのは地味に便利！

⑯Claude Code vs Codex vs Antigravity比較

  • Codex風のエージェント管理 → Antigravity 2.0のスタンドアロンアプリが参照
  • Claude Code風のCLI → Antigravity CLIがほぼ同じアプローチを採用
  • Googleのアプローチはあまりオリジナルではないとの声も。しかし「エージェントファースト」の方向性は評価される
Codexをコピーしたアプリ、Claude CodeをコピーしたCLI。Googleの独自性はどこに？
