Gemini Omni + Flow 徹底解説

思考するマルチモーダル動画エージェントが登場！テキスト・AIイラスト・AI動画からAI動画生成への実践解説

2026年5月21日 | AI共創イノベーション / ワンダー佐藤

①Gemini Omniとは: ワールドモデルの正体

  • ワールドモデルとして設計されたGoogle DeepMindの新AI
  • 従来のVeoとは根本的に異なり、Geminiアーキテクチャを基盤として開発
  • テキスト、画像、音声、ビデオを単一プロンプトで処理可能
  • 科学的・歴史的知識や物理法則の理解に基づいた映像を生成
  • 初期モデル「Gemini Omni Flash」は動画生成・編集に特化
「あらゆる入力からあらゆる出力を生成する」ことが目的のマルチモーダルAI。開発初期段階からマルチモーダルとしてトレーニングされているのが特徴です。

②VeoからOmniへの進化: 名称変更の背景

  • 旧称は「Veo」だったが、Google I/O 2026でOmniに改名
  • 旧Veoは主にテキストから動画の生成に限定
  • Omniは動画→動画、画像→動画、音声対応など範囲が大幅に拡大
  • 自分自身の出力を再入力として再帰的にフィードバックできる

マルチモーダル性が大幅に強化され、「オムニ（全方位）」という名前にふさわしい進化を遂げました。

③動画から動画への変換: ブラッシュアップ機能

  • 以前に作った動画を最新AIでブラッシュアップできる
  • 「この動画を最新バージョンで」という簡単なプロンプトだけでOK
  • イラストも同様に、古くなった作品を新バージョンで再生成可能
  • 構成はほぼ同じまま、品質が大幅に向上
「動画から動画」は非常に実用的！昔作った作品を最新AIでアップデートできるのは、クリエイターにとって大きなメリットです。

④テキストから動画生成: 自然言語で映像制作

  • 自然言語のプロンプトだけで動画が作成可能
  • アスペクト比や秒数（8秒・10秒程度）を指定できる
  • Gemini画面のプラスボタンから「動画作成」を選択して開始
  • インタラクティブに調整したい場合は「動画をまだ生成しないで」と指示 
プロンプト例:
「AI共創イノベーションのPVを作って。
 内容は抽象的に。
 8秒のものにして。
 動画はまだ生成しないで。」
生成回数には制限があるため、プロンプトを詰めてから動画を生成するのがコツです。

⑤画像から動画生成: つぎちゃん実演

  • AIマスコット「つぎちゃん」の画像を入力して動画化
  • 「このシーンにあったセリフを考えて」「女の子は日本語で」など詳細な指示が可能
  • 穏やかなBGMを入れて、少し動きをつけるよう指示
  • 英語でプロンプトを入力し、日本語で出力させる実演
生成されたつぎちゃんのセリフ：「わあ、きれい。ずっと見ていても飽きないな」— 自然な日本語で動画化されました！

⑥AI共創イノベーションPV生成デモ

  • 「AI共創イノベーション」を検索してプロンプトを自動作成
  • 内容は抽象的に指定し、8秒の動画に調整
  • 生成結果：「それは単なる道具ではなく、共に響き合うパートナーへ」
  • 最後の文字を大きくしたい、タグラインが欲しいなど改善点も確認
割と高品質なPVができたとの評価。ただし、最後の部分に改善の余地あり。プロンプトを調整すればさらに良くなります。

  • 旧Grokで作成したPVをOmniでブラッシュアップ
  • 再現性が非常に高い：構成・声ともにほぼ同じ
  • ライオンの描写が大幅にリアルに改善

⑧Google Flowの基本操作とクレジット仕様

  • 新しいプロジェクトを作成して開始
  • 秒数、モデル選択（Veo / Omni）、アスペクト比を設定
  • 生成にはクレジットが必要（1枚約30クレジット）
  • 4枚一気に生成すると120クレジット消費
  • モデルや秒数によってクレジット消費量が変動
旧Veoは1日3回程度しか生成できなかったので、クレジットを大事に使うことが重要です。

⑨Flowエージェントモード解説

  • Flow Agentは単なる編集ツールではなく、クリエイティブな共同制作者
  • リファレンス画像を設定し、手順を段階的に指示可能
  • プロンプトを複数登録して、段階的な生成ワークフローを構築
  • 確認あり/なしのモード切り替えが可能

ストーリーのプロット開発、セリフの提案、VFXの追加など、複雑な映像制作を自律的にサポートしてくれます。

  • 4枚一気にやるとガチャ要素が強くなる
  • 一部はオリジナルと全く違うものが生成されることも
  • 右上・右下は原型をとどめているが、左上・左下は顔が変化

  • 「明日もね、いい日にいるんといいね」など不自然な日本語
  • Gemini側の方がキャラクター再現性が高い
  • Flowでは日本人が好む顔立ちではなく、雰囲気が変わることがある

⑫エージェント指示によるセリフ生成デモ

  • エージェントモードに画像を入れて段階的に指示を設定
  • 「このシーンに適したセリフを言って」と指示
  • 結果はあまり反映されないことが判明
  • 生成されたセリフ例：「ねえ、見てください。あそこの景色すごくきれいだよ」 
エージェント手順例:
1. 参考画像を設定
2. 「シーンに適したセリフを言って」
3. 「穏やかなBGMを入れて」
4. 「少し動きをつけて」
セリフの指示はある程度のところまでしか反映されない。完全に思い通りのセリフを生成するには、プロンプトをより具体的にする必要があります。

⑬ChatGPT制作CMをFlowへ入力して比較

  • ChatGPTで作成したCM動画をFlowに入力して比較
  • エージェントモードでプロンプトは「ゆっくりと」のみの抽象的指定
  • 結果が英語になってしまう問題が発生
  • 再現性がうまくいかず、イメージ通りにするのは難しい
プロンプトが抽象的すぎると、思い通りの結果にならない。しっかりとしたプロンプトを入れることが重要です。

⑭Flow Music: 映像に同期する音楽生成

  • Flow Musicは単なるBGM生成ではない
  • 映像の展開や視覚的なテンポをAIが理解
  • シーンの感情的な起伏に合わせた楽曲アレンジを自律的に生成
  • 映像クリエイターはロイヤリティフリーの音楽を探し回る手間から解放
  • 映像と音楽が一体となったシネマティックな作品をプラットフォーム内で完結
動画の展開に完全に同期した複雑な楽曲アレンジを自動生成。これまで別途用意していたBGM選びの手間がゼロになります。

⑮SynthID: AI生成コンテンツの電子透かし

  • すべての生成動画にSynthIDがモデルレベルで埋め込まれる
  • 目に見えない電子透かしでAI生成かどうかを識別
  • フェイク動画や誤情報の拡散を防ぐ技術的な安全網
  • Googleは安全と責任あるAI利用を最優先している
オンライン上に拡散したコンテンツがAI生成かどうかを確実に識別できる仕組み。AI映像生成の社会実装に不可欠なセーフティネットです。

⑯キャラクター一貫性の克服: ストーリーテリングの実現

  • 生成AI動画の最大の課題であったキャラクター一貫性を完全に克服
  • 複数シーンをまたいでも同一キャラクターの顔・服装・特徴を維持
  • 本格的なストーリーテリングが現実のものに
  • AI自身の出力を再入力する再帰的メカニズムが鍵
「背景を火星の砂嵐に変えて」「アニメスタイルにして」など、自然言語で段階的に編集できるのも革新的です。

⑰GeminiとFlowの使い分けと総評

  • Gemini：高品質な単体動画生成に最適
  • Flow：複数のエージェントで協働するクリエイティブスタジオ
  • プロンプトは具体的に入力することが成功の鍵
  • 「お任せ」ではイメージ通りにいかないことが多い
