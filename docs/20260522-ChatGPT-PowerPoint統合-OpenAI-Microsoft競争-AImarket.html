ChatGPTがPowerPointに統合された

OpenAI vs Microsoftの競争の最前線を徹底解説

2026年5月22日 | チャンネル: AI market

① ChatGPTがPowerPointに登場

  • 2026年5月、OpenAIがPowerPoint内で直接スライドを作れるアドインをリリースした。
  • これまではチャット画面で内容を作ってからコピーする手間があったが、サイドパネルから直接操作できるようになった。
  • できることは大きく作る更新する分析する磨くの4つ。
パワーポイントを開いたまま、チャットGPTに話しかけるだけでスライドが完成する時代に！

② 機能①：ゼロからスライドを作る

  • 会議のメモやExcelデータを添付すれば、0からスライドを自動生成してくれる。
  • テキストや画像の素材を渡すだけで、レイアウト込みのプレゼンが数分で完成。
  • チャットGPTが構成を考えて、適切なスライド数や見出しを自動で提案。
毎週の定例資料も、メモを貼り付けるだけで完成するようになりました。

③ 機能②：既存のスライドを更新する

  • 既存のデッキに対して「A社向けにして」「経営陣向けに簡潔にして」と指示するだけで書き換えてくれる。
  • 同じ内容でも、提出先に合わせてトーンや詳細度を自動調整。
  • 手動で1枚ずつ修正していた時間が丸ごと削減される。
顧客ごとにカスタマイズした提案書が、ワンクリックで作れる！

④ 機能③：スライドを分析する

  • 「このデッキのどこが弱いか教えて」と聞けば、ストーリーの穴を自動で洗い出してくれる。
  • 発表先から想定質問まで自動でリストアップしてくれる。
  • 自分では気づかなかった構成の問題点を炙り出せるのが強み。
プレゼン前にAIにレビューしてもらう感覚で、精度を上げられる！

⑤ 機能④：スライドを磨き上げる

  • タイトルを結論先出し型（BLUF）に書き換えて、メッセージを明確にする。
  • 重要なポイントを強調したBLUFスライドを自動で追加生成。
  • 視覚的なバランスや余白も考慮して、プロ品質に仕上げてくれる。
結論から伝えるスライドに自動で変換してくれるのが便利！

⑥ スクリーンショットからスライドへ

  • スクリーンショットを貼り付けるだけで、編集可能なスライドに自動変換してくれる。
  • 会議のホワイトボード写真や手書きメモ画像からもデッキを作成可能。
  • 紙ベースで作業している人にとっても、スライド化の入り口として使いやすい。
手書きのメモを1からスライドにする手間がなくなりました！

⑦ 外部データとのリアルタイム連携

  • GmailやOutlookのデータをリアルタイムで引き込んでスライドに反映。
  • SharePointのファイルにもアクセス可能で、コピペ作業が丸ごと消える。
  • 最新のデータが常にスライドに反映されるため、情報の鮮度も保てる。
メールや社内文書の情報を、そのままスライドに組み込める！

⑧ インストール方法と料金プラン

  • Microsoftのサイトから「ChatGPT」アドインを検索してインストール。OpenAIアカウントでログインすれば完了。
  • 無料でも月に数回は試せる。本格的に使うなら月額$20のPlusプラン以上が必要。
  • 機密資料を扱うならBusiness / Enterpriseプランが推奨。データが学習に使われない設定になっている。
個人利用ならPlus（月$20）、企業ならEnterpriseで安心！

⑨ 営業・経営企画での活用例

  • 営業職：顧客名と課題を伝えるだけで、その会社向けにカスタマイズされた提案書が出力される。
  • 経営企画・財務：KPIや売上データを添付すれば、結論付きのグラフ入り報告資料が数分で完成。
  • 数字スライドの作成に一番時間がかかっていた作業が、丸ごと自動化される。
同じ構成の定型資料は、もう自動化できる！

⑩ マーケ・事業開発での活用例

  • 競合分析が劇的に楽になる。「競合3社を調べてスライドにして」の一言で完了。
  • Web検索から情報収集、スライド作成まで全部自動で走ってくれる。
  • リサーチの手間が大幅に削減され、分析により多くの時間を割ける。
リサーチからスライド作成まで、全部AIにお任せ！

⑪ 研修・プレゼン準備での活用例

  • 研修担当：議事録やノートを貼り付けて「研修資料にして」と頼めば、スピーカーノートまで自動作成。
  • 「このデッキで想定される質問を出して」と頼むと、投資家からの質問をシミュレーションしてくれる。
  • 既存スライドの見直しにも有効。チャットGPTが構成の穴を見つけて改善提案を出してくれる。
プレゼンが緊張しがちな人には、想定Q&Aが超便利！

⑫ Copilot vs ChatGPT：どちらを選ぶ？

  • M365を使っている会社なら、追加費用が少ないCopilotが現実的な選択肢。
  • 個人や小規模チームなら、M365なしでも月$20のPlusプランで全機能が使えるChatGPT版が入りやすい。
  • まずは自分の環境に合う方から試すのがおすすめ。
どちらを選ぶべきかは、あなたの環境次第！

⑬ 最強のタッグから競合へ：関係の変遷

  • MicrosoftはOpenAIの最大スポンサーで、100億ドル以上を投資してきた。
  • CopilotもOpenAIのモデルを土台に動いており、当初は最強のタッグだった。
  • 2024年、MicrosoftがOpenAIを年次報告書で初めて「競合他者」として公式に記載。業界に衝撃が走った。
ついに正式にライバル認定したというターニングポイント！

⑭ AWSと5兆円提携：Microsoftが猛反発

  • 2026年、OpenAIがAmazonのAWSと5兆円規模の提携を発表。
  • Microsoftとしては「うちのAzureを使ってくれると思っていたのに」と猛反発。
  • Azureとの既存契約の精神に反すると主張し、両社の対立がさらに深まった。
最大の出資先が、競合クラウドに頼る事態に！

⑮ 価格戦争：$20 vs $19.99の真剣勝負

  • ChatGPT Plusが月$20なのに対抗して、Microsoft 365 Premiumを月$19.99に設定。
  • 完全に1円差の真剣勝負。価格でも直接対決する構図に。
  • ユーザーにとっては選択肢が増えて、より安く便利なサービスを受けられる恩恵がある。
1円の差に込められた本気度がすごい！

⑯ OpenAIの野望：独自オフィススイート開発

  • OpenAIはWordやExcelに代わる独自のオフィススイートを開発中との報道がある。
  • 文書編集や会議の文字起こし機能を搭載し、Microsoftのオフィス事業を丸ごと奪いに行く構想。
  • PowerPoint統合は単なる機能追加ではなく、この野望の第一歩だった。
Microsoftの収益源に、OpenAIが直接踏み込む一手！

⑰ Anthropicも参戦：3社競争の激化

  • Anthropicも2026年2月にClaudeのPowerPoint対応を発表。同じようにスライドを直接編集できる。
  • Claudeは日本語に完全対応しており、価格も月$20でほぼ同じ条件。
  • OpenAI vs Microsoft vs Anthropic の3つ巴の競争に突入。
ユーザーにとっては選択肢が増えて、むしろありがたい状況に！

⑱ 使う前に知っておきたい3つの注意点

  • 注意点① プライバシー：個人プランで会社の資料を貼ると学習データに使われる可能性がある。2023年にSamsungの社員が会議メモを送って大問題になった事例も。
  • 注意点② AIの誤り：まだベータ版で、スライドを誤って削除したり内容を勝手に書き換えることも。バックアップは必須で、最終確認は自分の目で。
  • 注意点③ 地域制限：現時点では日本から接続が弾かれるケースも。米国から順次展開中なので、もう少し待つ必要がある。
機密情報はBusiness / Enterpriseプランで扱うのが安全！
