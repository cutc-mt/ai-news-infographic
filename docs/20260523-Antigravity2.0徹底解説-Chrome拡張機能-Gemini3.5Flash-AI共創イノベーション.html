①

Antigravity 2.0とは

  • Google I/O 2026で発表された、エージェントファーストな開発プラットフォームです。
  • 従来のVS CodeベースのIDEから独立し、スタンドアロンのデスクトップアプリケーションへと進化しました。
  • 最新モデルGemini 3.5 Flashがデフォルトで統合されており、圧倒的なスピードを実現しています。
CodexやClaude Codeと同じCLI系のAIツールですが、初動のざっくりとした実装に最適なツールです！
②

IDEからエージェント管制塔へ

  • 単一リポジトリの制約を排除し、複数フォルダやGitワークツリーを横断管理できるようになりました。
  • 「プロジェクト」の概念が導入され、フォルダ単位でセキュリティ設定やアクセス権限をスコープ化できます。
  • 複雑なシステム全体を俯瞰しながら、安全にエージェントを指揮することが可能です。
VS Code依存から脱却して、エージェントとの対話と成果物管理が主軸の新しいアーキテクチャになりました。
③

Gemini 3.5 Flashの圧倒的スピード

  • エージェント・ハーネスとの共同最適化により、出力速度は従来のフロンティアモデルの4倍に達しました。
  • エージェントが自律的に計画→コーディング→テスト→修正を繰り返すマルチステップワークフローで待機時間が劇的に短縮されます。
  • 実演では700行のコードを爆速で生成。Chrome拡張機能を一発で書き上げる威力を実証しました。
Gemini 3.5 Flashは「頭脳より実行力が強い」モデル。コーディング・ツール利用・エージェント系タスクに特に強いです！
④

Gemini.mdと会話メモリ

  • Gemini.mdを読み込むことで、エージェントに独自のルールやガイドラインを事前に設定できます。
  • エージェント名を確認すると「私の名前はアンチグラビティです」と応答。Geminiから名称が変更されています。
  • プロンプトでブースト設定を行うことで、エージェントの能力を底上げする仕組みも利用可能です。
事前にGemini.mdでルールを決めておけば、出力品質が安定します。プロンプトの工夫が重要です！
⑤

サブエージェントの動的生成と並列処理

  • メインエージェントが必要に応じてサブエージェントを動的に生成し、並列稼働させることができます。
  • サブエージェントは独立した環境で作業するため、メインエージェントのコンテキストウィンドウを汚染しません。
  • 非同期タスク管理により、長時間のプロセスをバックグラウンドで処理し、メインの会話ループをブロックしません。
「サブエージェントを設定して」と指示するだけで、自動的に最適な構成で並列処理を開始してくれます！
⑥

会話IDの仕組みと役割

  • 各サブエージェントには一意の会話IDが付与され、通信履歴を管理するための識別子として機能します。
  • このIDを使い回すことで、過去のサブエージェントの文脈を再利用できます。
  • 別スレッドにIDを持っていくことも可能で、柔軟な文脈の引き継ぎが実現します。
会話IDのメリット：
①専門家の特化 ②文脈の混同防止 ③並列処理の効率化 ④履歴の追跡と再利用
⑦

会話IDによる文脈引き継ぎ

  • サブエージェントの会話IDを入力するだけで、過去の調査結果を新しいスレッドで引き継げます。
  • 「これ調べたよね？」という確認から、追加情報の取得まで、IDを通じてスムーズに連携できます。
  • スレッド全体のIDも取得可能で、メインの会話も含めて一括で引き継ぎができます。
トークン消費を抑えつつ、コンテキストをクリーンに保つための必須テクニックです！
⑧

サブエージェント単位 vs 全体単位の引き継ぎ

  • サブエージェント単位：そのエージェントが調べた結果だけをピンポイントで引き継ぎます。
  • 全体単位：メインの会話とサブエージェントの内容すべてをまとめて引き継ぎます。
  • 場面に応じて使い分けることで、トークン消費の最適化とコンテキストの整理が両立できます。
CLI系AIはコンテキストが重なるとトークン消費が増えるので、スレッドを変えたい時にこの引き継ぎ機能が活躍します。
⑨

5体のサブエージェントでのリサーチ実験

  • 「Antigravity 2.0についてリサーチして、サブエージェントを5つ設定して」と指示して実験を実施。
  • 1番：タスクのブレイクダウンとサブエージェントへの指示（リーダー役）
  • 2番：システムのアーキテクチャ設計
  • 3番：リサーチ担当
  • 4番：コード変更の脆弱性・バグ検知と自動修正
  • 5番：単体テスト・総合テスト
エージェントを増やしすぎると余分なこともやってしまいます。リサーチ目的なら役割を明確に指定するのがコツです！
⑩

NotebookLM検索プラグインのデモ

  • NotebookLMには検索機能が標準で備わっていません。この問題を解決するプラグインを作成しました。
  • 右上に「ノートブック内を検索」のUIを追加。キーワード入力で該当箇所を順次表示できます。
  • このプラグインは1発のプロンプトで生成され、note記事で無料配布されています。
まだ改良の余地はありますが、1発出しでここまで作れるのはかなり優秀です！
⑪

サイドチャットボット拡張機能

  • ブラウザの右側にチャットボットが表示される拡張機能を作成。Claude in Chromeに似たAIチャット体験です。
  • Gemini 3.5 Flashの特徴や活用方法、他のモデルとの比較について解説する内容が確認できました。
  • 1540行ものコードを1発で生成。スクロールや操作も軽く動作することを確認しています。
まだ改良が必要ですが、ベースとしては十分使えるレベルです。Antigravity 2.0のポテンシャルを示す好例です！
⑫

新規プロジェクト作成とフォルダ指定

  • 「New Project」からプロジェクトを作成し、フォルダを指定して開発を始めます。
  • プロジェクトを使わない場合は、パスコードを取得して自分でパスを管理する工夫が必要です。
  • ライブ音声書き起こし機能も搭載。録音した音声をリアルタイムで文字起こしし、自然な言葉をプロンプトとして利用できます。
「キラキラブラウザ拡張機能」という名前のフォルダを音声指示だけで自動作成してくれました。
⑬

Chrome拡張機能のプランニング実演

  • 「プランニングして」と指示すると、まずMarkdownファイルで計画を立ててくれます。
  • READMEも自動生成。内容を確認した上で「じゃあ実装して」と進めるワークフローです。
  • 事前にGemini.mdでルールを読み込ませておくことで、品質の安定した出力が期待できます。
まず計画を立ててから実装に進むことで、無駄のない開発が可能です。
⑭

爆速コード生成：700行を一撃で

  • Gemini 3.5 Flashのスピードにより、約700行のコードを驚異的なスピードで生成。
  • manifest.json、ポップアップHTML、サービスワーカーなど、Chrome拡張機能に必要なファイルをすべて自動生成。
  • 1発でそこそこのものが完成し、その後の調整で仕上げていく開発スタイルが確立できます。

⑮

キラキラブラウザ拡張機能の動作確認

  • マウスクリックや移動時にキラキラエフェクトを表示する拡張機能が完成。
  • エフェクトのカスタマイズが可能：星・ドット・ハートの形状や、サイズ調整に対応。
  • カラーバリエーションも豊富：ゴールド・レインボー・パステル・アクアなど。
ショッピングサイトに行くとキラーン！ Yahooニュースでもキラーン！ 少しうざいですが（笑）、1発でここまで作れるのはすごいです！
⑯

スラッシュコマンドによる高度な制御

  • /goal：タスクが完了するまで中間入力を求めずに自律実行します。
  • /grill-me：実装前にAIから質問を投げかけてもらい、詳細な認識を合わせます。
  • /schedule：バックグラウンドでの定期実行をスケジュールします。
  • /browser：AIの自己判断による暴走を防ぎ、明示的にブラウザ操作を許可します。
JSON Hooksを使えば、ツール呼び出し前後にローカルのカスタムシェルスクリプトを割り込ませることも可能です。
⑰

トークン節約の開発戦略

  • まずAntigravity 2.0でざっくりとした初動を実装し、その後に他のAIで調整する戦略が有効です。
  • イメージ：Claudeで全体をプランニング → Antigravityで最初の実装 → 難しい部分はCodexやClaudeで仕上げ。
  • この組み合わせにより、トークン消費を大幅に節約しながら効率的な開発が可能になります。
Google製品なのでChrome拡張機能はネイティブに作れます。プラグイン開発には特に向いています！
⑱

まとめ：開発ワークフローへの組み込み

  • 爆速の初動実装：Gemini 3.5 Flashで1発出力。700行規模も一気に生成。
  • サブエージェント：並列でリサーチやテストを自動化。会話IDで文脈を整理。
  • Chrome拡張機能：Google製品ならではのネイティブ対応で高品質なプラグインを即座に生成。
  • コスト効率：他のAIとの使い分けで、トークン消費を抑えつつ本格開発へ。
簡単なものから本格的なアプリまで、Antigravity 2.0は開発の初手として非常に強力なツールです！
