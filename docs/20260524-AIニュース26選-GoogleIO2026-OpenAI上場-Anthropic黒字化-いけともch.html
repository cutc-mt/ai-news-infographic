Google I/O 2026 〜 使うAIから働くAIへ
①

Google I/O 2026：使うAIから働くAIへ

  • Google I/O 2026のテーマは「使うAIから働くAI」への転換
  • AIを電卓のような「使うツール」から、社員のように「一緒に働く存在」へと進化させる方向性が示された
  • 情報収集、買い物、仕事のあらゆる場面にGeminiが入り込むエージェント時代が到来
  • Googleの全サービスにGeminiが統合され、AIエージェントがユーザーの代わりに作業を実行する未来が現実に
情報収集、お買い物、メール管理⋯あらゆる場面でAIエージェントが活躍する時代が始まる！
②

Gemini 3.5 Flash｜爆速エージェントモデル

  • 爆速＆高性能を兼ね備えたエージェント時代の主力モデルとして昇格
  • これまでのFlashは「高速だけどちょっと物足りない」だったが、3.1 Proクラスの仕事ができるレベルに進化
  • 非常に速いレスポンスで、価格も安く、AIエージェント時代のメインモデルとしてリリース
  • Geminiアプリからすぐ使える。デザインも刷新され、より使いやすくなった
Gemini Chat（軽自動車）ではFlashのF1エンジン性能を活かしきれない。Antigravity 2のようなエージェントアプリ（F1マシン）でこそ真価を発揮！
③

Google Antigravity 2.0｜93体のAIがOSを作る

  • Claude CodeのGoogle版とも言えるエージェントアプリがバージョン2.0に進化
  • 公式デモでは93体のAIを同時に動かし、12時間で新しいOSを作成する実演を実施
  • 個人アカウントだけでなく、Google Cloud法人契約でも利用可能に。データ学習なしで法人展開しやすい
  • Claude CodeやCodexと比べて圧倒的なスピード。500行のページが一瞬で完成するレベル
Claude Code/Coworkと同じような操作感だが、スピードが段違い！精度は若干エラーもあるが、速さは圧倒的
④

Google AI Studio｜Workspace連携＋アプリ生成

  • Google Workspaceの情報（メール、資料など）と連携したアプリを簡単に作成可能に
  • 言葉だけでAndroidアプリをその場で作成し、Play Storeに申請もできる
  • Webアプリではなく、本当にスマホで使えるネイティブアプリとしてビルドされる
  • 裏側はGemini 3.5 Flashで動作し、精度が大幅に向上
ToDo管理アプリなどが数分で作れる！Galaxyを買って実際に試したくなるレベル
⑤

Gemini Omni｜物理法則を理解する動画生成

  • 文章、画像、音声、動画をすべて入力でき、動画アウトプットに変換する統合モデル
  • Geminiの世界知識（ワールドモデル）を活用し、重力や水の動きなどの物理法則を理解したリアルな動画を生成
  • 日本語対応！動画内に日本語の文字を自然に入れられるようになった
  • 生成した動画に対して「背景にフローチャートを追加して」など追加の修正指示も可能
Google Flowツールと組み合わせると、日本語の文字入り動画も作りやすい！
⑥

Google Flow｜専用ツールをバイブコーディングで自作

  • 動画作成ツールから上位クリエイティブツールへ進化。エフェクトや画像編集も可能
  • 独自の編集ツールを自分で作れる「ツールを作成」機能が追加
  • 作ったツールは他の人と共有でき、ベースにしてさらにアップデートも可能
  • 写真1枚から多角的なアングルを自動生成するアプリや、画像をプロンプトに分解するツールなどが即座に作成できる
バイブコーディングがクリエイティブ領域にも浸透！自分専用の処理ツールを即座に作れる
⑦

Managed Agents｜クラウド常時稼働＋AI Ultra値下げ

  • Managed Agents：クラウド上で常に動くAIエージェントを簡単に作れる法人向けサービス
  • ハーネス環境で安全にシステム連携しながらエージェントを構築。Claude、OpenAIも同様のサービスを提供中
  • AI Ultraプランが値下げ：新100ドルプラン登場。250ドル→200ドルに値下げ（日本円で約3万円）
  • Gemini Spark：24時間365日稼働する自律型AIエージェント。メール監視や自動通知を常時実行
Gemini Sparkは寝ている間も働いてくれるAI社員！日本展開はこれから順次提供予定
⑧

Universal Cart｜ECサイトを横断する新しい買い物

  • Amazonなどの単一ECサイトではなく、複数のECサイトを横断するUniversal CartをGoogleが提供
  • カートに登録しておくと、どこが一番安いか、クレジットカードの割引があるかを自動で通知
  • カート内商品の整合性チェック機能：スマホとUSBコネクターの互換性がないと警告してくれる
  • GoogleがECの一段上に位置し、ユーザーは商品を入れておくだけで「いつが買い時か」を自動判定
エージェンティックコマースの登場！買い物体験が根本的に変わるかも
⑨

検索の進化｜AIが専用UIをその場で生成

  • 検索ボックスに文章だけでなく、画像や資料、動画も入力して検索できるように進化
  • 検索結果画面でAIが専用のミニツールやアプリを自動生成してくれる新体験
  • 作成したツールは家族や友人と共有可能。Gemini 3.5 Flashの爆速があってこその体験
  • パーソナルインテリジェンス：Gmailやフォトを踏まえた深い背景情報に基づいて検索結果を返す
「週末の家族予定を計画して」と検索すると、専用プランナーツールがその場で作られる！
⑩

Information Agents + Ask YouTube｜自動リサーチAI

  • Information Agents：テーマを設定しておくと、AIが常時監視して新情報をレポート化するGoogle Alertの進化版
  • Ask YouTube：動画1本ずつ見るのではなく、複数動画を横断的に検索・要約してまとめサイトを作成
  • 「3歳の子供に自動車の乗り方を教えて」と聞くと、手順も整理されたまとめページが即座に生成される
  • AI Inbox：Gmailの内容を確認し、Google Driveの関連データも参照して集約・回答
YouTubeの動画を横断的に検索できるのは画期的！情報収集の効率が劇的に変わる
⑪

Docs Live + Google Pics｜音声ドキュメント＋画像編集

  • Docs Live：音声で話すだけでリアルタイムにドキュメントが生成される。修正指示も音声で対応
  • 「もう少しフォーマルにして」と話しかけると、内容を見た上で書き換えて新しいバージョンを作成
  • Google Pics：NanoBanana搭載の高度な画像編集ツール。CanvaやAdobe Expressの代替サービス
  • Docs LiveもGoogle Picsも、裏側はGemini 3.5 Flashの爆速が支えている
話すだけでドキュメントができあがる！音声インターフェースが仕事のスタイルを変える
⑫

ChatGPTアドイン＋Canva Magic Layers

  • ChatGPT PowerPointアドインがベータリリース。右側にChatGPTパネルが表示され、スライド作成を依頼可能
  • 画像を入れてスライドに変換する使い方も可能。ただしクオリティはまだ「まあまあ」レベル
  • Canva Magic Layers：AI生成画像のフォントや要素を分割し、編集可能な形式に変換
  • 完璧ではないが、サムネイル画像の文字修正などに実用的。編集チェックは必要
PowerPointもMagic Layersも「素案作成」には使えるが、完成度を高めるには手動編集が必須

Anthropic営業責任者のClaude活用術

  • 会議準備の自動化：カレンダーを確認し、お客さんごとに必要な情報を収集して資料を作成
  • 営業レポート自動生成：毎週3時間かけていた営業速報を自動化し、月曜会議前に自動共有
  • 4,000件の顧客スコアリングを一晩で完了。数百時間の作業をAIエージェントに任せた
  • Web情報、Salesforceデータ、実際の利用データを統合して自動評価。ダッシュボードも自動生成
エージェントは開発現場だけじゃない！営業現場での実践的な活用が急速に広がっている

OpenAI、1兆ドル規模の上場（IPO）準備を本格化

  • 2015年非営利団体としてスタートし、Microsoftと連携して資金調達後に営利企業（PBC）に組織再編
  • 直近評価額8,520億ドルで122億ドルを調達。上場は1兆ドル規模と予想
  • なぜ上場？→資金調達、投資家への出口（イグジット）、信頼感向上。2030年まで赤字前提でインフラ投資継続
  • イーロン・マスクの訴訟は「訴えた期間が遅すぎた」でクローズ済み
上場で財務情報が開示され、AI業界全体のスタンダードができる可能性。バブルか継続かの分岐点にも！
⑮

カーパシー氏がAnthropicへ電撃移籍

  • OpenAI共同創業者のアンドレイ・カーパシー氏がAnthropicに転職。「バイブコーディング」の生みの親
  • AnthropicではAIが自律的に自分を修正していくという研究に取り組む
  • Anthropicは安全性重視のスタンスで評価が高く、米政府の軍事利用要請も拒否して信頼を獲得
  • 最優秀タレントがAnthropicに集中する構造ができつつあり、好循環ループが発生中
優秀な人材＋AIエージェント＝最強のループ。Anthropicの成長加速が期待できる！

Anthropic初の営業黒字＋Stainless買収

  • 2026年第2四半期に初の営業黒字を達成。売上高は3ヶ月で109億ドル、約5億ドルの黒字を計上
  • OpenAIが大赤字の中、Anthropicは黒字化を実現。市場へのアピールも狙い
  • Stainlessを3億ドル超で買収。APIをSDKに自動変換するツールで、OpenAIやGoogleも利用していた
  • 買収後は外部提供を停止し、Anthropic独自機能に。競合の開発インフラを制限する強気の戦略
珍しいAnthropicの買収！インフラ的ポジションを確保し、競合にプレッシャーをかける一手

Apple新Siri + Alexa+ポッドキャスト + Spotify

  • Apple WWDCで新Siriを発表予定。プライバシー保護を強みに、会話を自動削除する機能も
  • AIを敬遠する層に向け、完全に独立したアプリで情報を保存しない安全性をアピール
  • Amazon Alexa+：テーマを入れると200種以上のニュース媒体から情報収集し、専用ラジオを自動生成
  • SpotifyもAI個人ポッドキャスト機能を追加。エンタメ寄りで、生成後に人間のポッドキャストへ誘導する方向
耳で情報収集する「オーディオインターフェース」の時代が到来！各社が競合中
⑱

Microsoft Work Trend Index｜組織活用のハードル

  • 人間の役割は「実行」から「判断と責任」へシフト。AIに丸投げではなく品質管理が重要
  • 65%が「AIを使わないと取り残される」と感じる一方、45%は「今の目標達成が優先」と回答
  • 組織の変化が個人の意識に追いついていない。企業の評価体制がAI活用に対応できていない
  • 組織カルチャーとマネージャーの支援が最も重要。個人のAIマインドセット以上に影響が大きいという結果
個人の頑張りより、組織の雰囲気と直属の上司の支援が鍵！成功・失敗の事例を組織に展開する仕組みが重要
