2026年5月18日〜5月24日

ChatGPT Personal Finance 〜 銀行口座をAIに連携する時代

①ChatGPTが銀行口座に連携: Personal Finance機能が登場

  • Plaid経由でアメリカ国内の12,000以上の金融機関と接続。チェース、フィデリティ、ロビンフッド、アメックスなどの大手にも対応
  • ダッシュボードでポートフォリオ、月別支出、サブスク一覧、支払い予定を一画面で確認可能に。デモでは月間4,035ドルの支出を自動集計
  • 口座データをもとにAIに質問すると、収入と支出を分析したうえで「飲食費を450ドル以内に抑えれば毎月150〜250ドル節約できる」など具体的な提案が返ってくる
  • 「財務メモリー」機能で住宅ローンや貯金目標を記憶。毎回状況を説明し直す手間がなくなる
RoiとHiro Financeの2社買収から半年かけて準備。フィンテック業界の超一流チームを引き抜いてからのリリースだった！

②月間2億人のプラットフォームが金融データにアクセス

  • 月間2億人が使うChatGPTに財務データが繋がると、世界最大規模のAI財務データベースが誕生する可能性
  • 誰がいつどこでいくら使ったか、どんなサブスクを持っているか、投資の傾向は何か。これらのデータが積み上がると、金融機関すら持てないレベルの個人プロファイリングが可能に
  • 現時点ではアメリカ在住のProプランユーザー限定。近々Plusユーザーへの拡大、最終的には全員が使える予定
  • Intuit（会計ソフト）との連携も計画中。実現すれば税務計算や専門家への相談までChatGPTで完結する時代が到来
読み取り専用だから安全と説明しているが、取引履歴と会話内容が組み合わさると個人の行動パターンは想像以上に丸裸になる

③発表前日にクラスアクション訴訟: プライバシーへの懸念

  • 5月14日、OpenAIに対して全米規模のクラスアクション訴訟が提起。ChatGPTのユーザー会話内容をMetaとGoogleに無断共有していたという疑惑
  • ChatGPTページ内にMetaとGoogleのトラッキングスクリプトが仕込まれていたと主張。連邦電子通信法とカリフォルニア州法の両方に違反するとされる
  • 1人あたり最大5,000ドルの損害賠償を求める集団訴訟。その翌日に「銀行口座をつないでください」という発表が来た
  • 専門家は「口座情報よりも取引履歴の方が危険」と指摘。食費、課金履歴、衝動買いパターンから精神状態や信用リスクまで丸裸になるとの懸念
トムズガイドは「まともな人間がOpenAIにここまでのアクセスを渡して安心できるのか」と批判。便利とリスクが表裏一体の機能
Netflix AIスタジオ 〜 アニメ産業を揺るがす動き

④NetflixがAIスタジオ「INKubator」を密かに設立

  • 2026年3月、公式発表なしに設立されたNetflix社内のAIアニメ制作スタジオ。キャリアサイトの求人情報から存在が発覚した
  • 生成AIを活用して映画レベルのアニメを目指す実験スタジオ。最初は短編アニメから始めて、いずれ長編コンテンツへ拡大する計画
  • 同月、ハリウッドのベン・アフレック氏が設立した映像後処理AI「InterPositive」を推定約900億円で買収済み
  • Netflixの2段階戦略: まず映像の後処理をAIにやらせる → 次に制作本体のAIにも踏み込む。同じ月に一気に仕込んだ布石
リーダーはドリームワークスやA24で活躍したセレナ・アゲルさん。採用はCGアーティスト、コンポジター、ソフトウェアエンジニアなど

⑤アニメーターの90%の仕事が消える可能性

  • ドリームワークス共同創設者のジェフリー・カッツェンバーグ氏が「AIによってアニメーターの仕事の90%が消える」と発言
  • 業界全体では3年以内に11万8,500人分のポジションが失われる可能性。特に3Dモデラー、コンポジター、エフェクトアーティスト、アニメーターに大きな影響
  • INKubator設立前にアニメーション組合が組織化を推進。Netflixアニメーションスタッフは44対13で組合加入を投票で決定
  • AIだけではなく海外への仕事の外注増加との二重苦で、アメリカのアニメ業界はすでにかなり厳しい状態に
アニメーション産業全体の29%が破壊的影響を受ける可能性があると試算されている

⑥ディズニーも参入: AIアニメが産業として動き出す

  • ディズニーはAnymateというスタートアップと提携し、これまで5ヶ月かかっていた短編アニメ制作を5週間に短縮することに成功
  • 仕組みはシンプル: アニメーターがキーポーズだけを描くと、駒と駒の間の動きをAIが自動生成。一番手間のかかる作業をAIに任せる
  • スタートアップのInvisible Universeは、数週間かかっていた短編コンテンツ制作を数時間に短縮することを実証済み
  • 最初に業界標準を作った会社がルールを決める流れが始まっている。Netflixの加入者数2億人超えのプラットフォームがAIアニメを本格投入する影響は測り知れない
著作権や肖像権の問題は業界全体で未解決。学習データの透明性がクリエイター側から懸念されている点は今後注目
Google I/O 2026 〜 「エージェント時代の本格到来」

⑦Gemini 3.5 Flash: GPT-5.5比で約4倍速、1/20のコスト

  • Google I/O 2026で発表された新モデル。GPT-5.5などの上位モデルとの比較で出力速度が約4倍とされる
  • 性能は従来モデルのGemini 3.1 Proを超えつつ、コストは約1/20。コーディングや複雑なタスクにも強くなっている
  • 本日より全世界でデフォルト利用が開始。Google検索のAIモード進化の土台を支えるモデル
  • 検索AIモードの刷新: 長くて複雑な質問に対応。スクリーンショット、PDF、写真、動画をそのまま検索に投げ込めるように。会話の文脈も保持
「ジェネラティブUI」機能で検索結果が3Dアニメーションで表示。惑星の軌道を3Dで見たり物理シミュレーションをその場で動かしたりできる

⑧Gemini Spark: PCを閉じても24時間稼働するAIエージェント

  • クラウド上で24時間365日稼働し続けるパーソナルAIエージェント。PCを閉じても動き続ける
  • Gmail、Docs、Sheets、Slidesだけではなく外部の様々なサービスとも連携。複数ツールを横断して自動処理
  • 来週からGoogle AI Ultraユーザー向けベータ開始。今年の夏に一般展開予定
  • 「デイリーブリーフ」機能も今日からロールアウト。Gmail、カレンダー、タスクをまとめて毎朝AIが整理してくれる
デモでは「パーティーを企画して」の一言で、招待状の集計、スプレッドシートの自動更新、スライド資料作成、メールの下書きまで全部自動完了！

⑨AIショッピングカートとAP2決済プロトコル

  • Search、Gemini、YouTube、Gmailを横断してAIが商品を自動でカートに追加。価格履歴確認、値下がり通知、在庫復活通知、レビューチェックまで対応
  • AP2プロトコルでAIがユーザーの代わりに決済まで実行可能。事前に設定した範囲内で動き、デジタル身分証で本人証明する安全設計
  • Googleウォレットのポイント考慮まで自動で対応。今年の夏にアメリカから展開予定
  • 気になった商品を事前に登録しておけば、値段が下がった瞬間に通知が来て、設定した予算内なら自動で購入してくれる
タイムセールに乗り遅れることがなくなる！買いそびれの不安が解消される新しい買い物体験

⑩Android XRオーディオグラスと開発者向け新機能

  • Android XRオーディオグラスが今秋アメリカで発売。サムスンとGeminiが組み、ジェントルモンスターとコラボしたデザイン
  • 耳に装着するタイプで、リアルタイム翻訳やナビゲーション通知の読み上げが可能。「Hey Google」でGeminiが起動しiPhoneでもAndroidでも両対応
  • マネージドエージェント「Gemini API」: 1回のAPIコールで完全なエージェント環境が即起動。隔離されたLinux環境でコード実行まで可能
  • Antigravity 2.0はスタンドアローンのデスクトップアプリとして登場。CLI版、セルフホスト版も同時リリース
注意: Gemini Code AssistのIDE拡張コンシューマー版は2026年6月18日以降に全プランで利用終了
Apple iOS 27 〜 iPhoneが仕事を完結させるデバイスへ

⑪キーボードAI統合: アプリ切り替え不要で文章支援

  • Write With SiriとHelp Me Writeの2つのオプションがキーボードに直接追加される
  • これまではAIを使うためにアプリを切り替えるかSiriを呼び出す必要があった。それがキーボードを開いたままその場でAIに文章を書いてもらえるように
  • メール作成中に「このメールを丁寧に書いて」と入力すれば、キーボードを閉じることなく完成文が出てくる
  • メール1本あたり5分以上の時間が浮く計算。iOS 26ではAI機能が「あるけど見つかりにくい」という課題があった
情報源はブルームバーグのマーク・ガーマン記者。Apple関連のリーク報道では業界ナンバーワンの信頼性

⑫Shortcuts自然言語自動化とAI文法チェッカー

  • Shortcutsの自然言語自動化: 「毎朝8時に天気予報を通知して」の一言で複雑なショートカットをAIが自動で組み上げる
  • これまではアプリ内でブロックをパズルのように組み合わせる必要があったものが、自然言語だけで設定完了に
  • AI文法チェッカーはGrammarlyのiPhone全体版のようなイメージ。文章の誤りを検知すると画面下からメニューがスライドアップ
  • Writing Toolsの進化版: リライトとテキスト生成の精度が大幅に向上。「もっとフォーマルに」「短くまとめて」などの指示が格段に賢く
文法チェッカーは全アプリ対応。メモでもメールでもSlackでもどこで文字を書いても反応してくれるのが強み

⑬iPhoneが仕事を完結させるデバイスへの変貌

  • AI壁紙生成: Image Playgroundが刷新され、テキスト入力だけでオリジナル壁紙が生成可能に
  • iOS 27は700以上のアプリと連携可能。情報の整理から共有までiPhoneだけで一連の仕事が完結する
  • 実践パターン: 急ぎのメール作成 → 報告書チェック → 毎朝の業務ルーティン自動化 → 会議メモの整理 → チームへの共有。全部iPhone完結
  • 正式発表はWWDC 2026（6月）。実際に使えるのは2026年秋以降。日本対応は数ヶ月遅れる可能性
Appleの狙いは「使いたい時にすぐ使えるAI」。iPhoneを見るだけのスマホから仕事を完結させるデバイスへ変えようとしている
ChatGPT × PowerPoint 〜 オフィス覇権争いの最前線

⑭PowerPointにChatGPTが直接統合: 作成・編集・分析

  • 作る: 会議メモやExcelデータを添付すれば0からスライドを生成
  • 更新する: 「A社向けにして」「経営陣向けに簡潔にして」と言えば既存スライドをそのまま書き換え
  • 分析する: 「このデッキのどこが弱いか教えて」と聞くと、ストーリーの穴や想定質問まで自動で洗い出し
  • スクリーンショットを貼り付けるだけで編集可能なスライドに変換。ホワイトボードの内容や手書きメモからデッキを起こせる
Gmail、Outlook、SharePointのデータをリアルタイムで引き込んでスライドに反映可能。コピペが丸ごと消える！

⑮OpenAI vs Microsoft: オフィス覇権争いの最前線

  • MicrosoftはOpenAIの最大スポンサーで100億ドル以上を投資してきたが、2024年に年次報告書で初めて競合他者として記載
  • 2026年にOpenAIがAmazon AWSと5兆円規模の提携を結んだことで、Microsoftが反発。Azureの既存契約の精神に反するとして問題に
  • OpenAIはWordやExcelに代わる独自オフィススイートを開発中。共同編集や会議の文字起こし機能付きでMicrosoftの事業を丸ごと奪いにいく構想
  • 3つ巴の構図に: Anthropicも2026年2月にClaudeのPowerPoint統合を発表。月額20ドルでほぼ同じ条件
ChatGPT Plusが月$20なのに対し、Microsoft 365 Premiumを$19.99に設定。完全に1円差の真剣勝負！
Musk vs OpenAI 〜 10年の因縁が90分で決着

⑯10年の因縁が90分で決着: 陪審員全員一致で全請求棄却

  • 2015年12月、マスクとアルトマンら11人が非営利団体としてOpenAIを共同設立。拠出金額は約10億ドル（約1,200億円）
  • 2017〜2018年にマスクが過半数株式・支配権・CEO就任を要求し拒否され、2018年にOpenAIから去る。2019年にOpenAIは営利会社に転換
  • 2026年5月18日、陪審員9人が審議開始からわずか90分で全会一致の全請求棄却を決定
  • 決め手は「時効」。カリフォルニア州法ではこの種の請求の時効が3年だが、マスクが2021年時点で営利転換を知っていたと認定。2024年2月の提訴は時効を超えていた
本案（非営利信託侵害の有無）は一切審理されず、手続き上の理由だけで却下。本質的な論争は決着していない

⑰法廷で暴かれたマスクの矛盾: xAIがOpenAIモデルで学習

  • マスク自身が法廷で「xAIはOpenAIのモデルを使って自社のGrokを訓練している」と認め、傍聴席がどよめいた
  • 非営利を盗まれたと訴えている本人が、その「盗まれた先のモデル」を使って営利事業を展開しているという矛盾
  • 衝撃的な証拠も次々と公開: マスクとザッカーバーグのOpenAI共同買収相談テキスト、ブロックマンの個人日記、エンジニア引き抜きの事実
  • アルトマンは「マスクに対して企業構造について何の約束もしていない」ときっぱり否定。ブロックマンも同様に否定した
xAIを吸収したSpaceXは6月12日にIPO上場最大規模の1.75兆円でナスダック上場予定。IPOを控える人がライバルを訴えていた構図

⑱OpenAIのIPOへの道がクリアに: 最後の障壁が消滅

  • 2025年10月にパブリックベネフィットコーポレーションへの移行を完了。非営利のOpenAI Foundationが1,300億ドル相当の株式を保有
  • 2026年3月に1,220億ドルの資金調達を完了。評価額は8,520億ドルに。ChatGPTの週間ユーザーは9億人に到達
  • 今回の判決で最後の法的不確実性が消え、2026年第4四半期のIPO申請に向けた道がクリアに。上場前に評価額1兆ドル到達の可能性も
  • マスクは控訴を表明したが、裁判所が覆す可能性は低いと見られている。「法廷で負けたなら市場と世論で勝つ」という構え
マスクが引けない3つの理由: SpaceXのIPO控え、自身の発言の重み、Grokのモデル独立性。勝てる戦いではなく止まれない戦いをしている状態
