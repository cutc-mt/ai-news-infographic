Antigravity 2.0 完全ガイド

第1章：概要
①

Googleが本格参戦：Antigravity 2.0

2026年5月20日、GoogleはAI開発プラットフォーム「Antigravity」のサービス全体を拡張。AIコーディングツールの競争に本格的に参戦しました。

  • OpenAIのCodexやAnthropicのClaude Codeと近い見た目・操作感
  • 以前のデスクトップアプリから画面構成がシンプルに進化
  • ターミナルで動くCLIも同時リリース
Googleがここまで本気を出してきたのは今回が初めて！これからのAIコーディング界隈がさらに盛り上がりそう〜

Antigravity シリーズ4つの構成要素

  • Antigravity IDE — 旧Antigravityの名称変更。統合開発環境。
  • Antigravity 2.0 — 新しいデスクトップアプリ。今回の主役
  • Antigravity CLI — ターミナルから起動できる軽量ツール
  • 利用条件 — Googleアカウントがあれば無料でお試し可能

さらに開発者向けのAntigravity SDKも公開されました。

IDE・2.0・CLI・SDKと4つのツールを揃えて、開発者から非エンジニアまで幅広くカバー！
③

Antigravity IDEとは

2025年11月にリリースされた旧Antigravityが、5月20日の拡張発表に合わせて「Antigravity IDE」に名称変更されました。

  • コードの生成や実行などプログラミングに必要な機能を1画面にまとめたツール
  • 既存ユーザーはIDEとしてそのまま継続利用可能
IDEは「統合開発環境」のこと。エディタ・ターミナル・デバッガが全部入ってるイメージだよ〜
④

Antigravity 2.0の特徴

AIエージェントを使いやすくするために作られた新しいデスクトップアプリ。複数のエージェントをまとめて動かす司令塔のような役割を持ちます。

  • 複数のエージェントを同時に稼働
  • 必要に応じてサブエージェントを自動で起動
  • スケジュールタスク機能 — 特定のタスクを特定のタイミングで自動実行
  • Google AI Studio・Firebase・Androidとスムーズに連携
Googleが「これを主力ツールとして押していく」と明言！今後3.0・4.0とバージョンアップが期待されるね〜
⑤

Antigravity CLIの登場

2.0のリリースと同時に、ターミナルで起動できるCLI（コマンドラインインターフェース）も登場しました。

  • 軽量かつ高速 — アイコンやボタンを使わずテキスト入力だけで操作
  • 英語や日本語などのテキスト入力に対応
  • 既存のJules CLIは6月18日にサービス停止予定
  • Antigravity CLIへの移行が推奨されている
Jules CLI使ってる人は要注意！6月18日までに移行を完了させよう。

第2章：インストール方法
⑦

Antigravity 2.0のインストール

公式サイトのダウンロードページからOS別のアプリをダウンロード。

  • macOS
  • Windows
  • Linux

Windows版では、インストール時にIDEの同時インストールも可能。チェックボックスをオンにするだけ！

初期設定ではテーマカラーの選択やデフォルト機能のインストールもできるよ〜
⑧

2.0インストール時の設定手順

インストール後の設定は以下の流れで進みます。

  • Explore Antigravity ボタンをクリック
  • Googleアカウントでログイン
  • データ共有の同意 — Googleに学習データとして使っていいか確認
  • テーマカラーを選択
  • 便利なデフォルト機能の選択
  • Finish ボタンで完了！
データ共有は任意だけど、プライベートなプロジェクトを扱う場合は慎重に選んでね。
⑨

Antigravity IDEのインストール

IDEも公式サイトから各OS用のアプリをダウンロードしてインストールできます。

  • 2.0インストール時に同時セットアップ可能
  • 他のコードエディターからの設定の引き継ぎに対応
  • テーマカラー・権限設定も変更可能
  • 後からでも設定変更ができるから安心
VS Codeなどからの移行もスムーズ！推奨設定を選ぶだけでOKだよ〜
⑩

Antigravity CLIのインストール

CLIはターミナル（黒い画面）を開いてコマンドを打つだけ。OSごとにコマンドが異なります。

  • Mac / Windows用に別々のインストールコマンドを用意
  • WindowsはPowerShellとコマンドプロンプトでコマンドが異なる
  • インストール後はagentyコマンドで起動
  • 初期設定画面でカラー・権限・データ収集の同意を設定
CLI導入前に2.0かIDE側でGoogleアカウント認証を済ませておく必要があるよ！
⑪

アップデート時の注意点

既に旧Antigravity IDEをインストールしていた場合は、アップデート前にいくつか注意が必要です。

  • 2.0へのアップデートで設定の一部がリセットされる現象が報告されている
  • アップデートで修正されつつあるが、アップデート前にネットで状況を確認推奨
  • 不安な場合は新規インストールが安全
念のため設定のバックアップを取ってからアップデートするのがおすすめ！
第3章：使い方と活用事例
⑫

基本的な使い方：チャットで開発

Antigravity 2.0はログイン後の画面にあるチャットフォームを使って、ウェブサイトやアプリのコンテンツを作成できます。

  • 左上のCreate Projectで作業フォルダーを指定
  • Gemini 2.5 Flash / Claude Sonnet 4 / GPT o3などの最新AIモデルを選択可能
  • 画像生成AI「Imagen」も使える — チャットで画像生成＆保存
「芝犬の画像を作ってプロジェクトフォルダーに納品して」って言うだけで画像が生成されるの、最高すぎる〜！
⑬

スケジュールタスク機能

画面左側のScheduled Tasksセクションでは、特定のタスクを特定のタイミングで自動実行してくれる便利な機能が使えます。

  • 定期的に実行したい作業を自動化
  • エージェントが決まった時間に勝手にタスクを処理
  • バッチ処理や定時ビルドなどにも活用可能
定期的なデータ収集やレポート生成などを自動化できたら、作業効率が跳ね上がりそう！
⑭

設定項目の解説

左下のSettingsから各種設定にアクセス可能。

  • Analytics — データ収集の同意設定
  • Permissions — プロジェクトフォルダーごとの読み書き権限
  • Project — エージェントの承認要求タイミングを設定
  • Models — モデルの使用状況を確認
  • Customizations — プラグインの管理
  • Rules — 全プロジェクト共通ルール（IDEが必要）
残念ながらまだ日本語化はできません。Rulesの変更にはIDEが必要です。
⑮

利用できるAIモデル一覧

Antigravity 2.0では複数社の最新AIモデルを切り替えて使えます。

  • Gemini 2.5 Flash — Googleの最新モデル
  • Claude Sonnet 4 — Anthropicの高性能モデル
  • GPT o3 — OpenAIの最新推論モデル
  • Imagen — 画像生成AI
マルチモデル対応がやばい！プロジェクトに合わせて最強のモデルを選べるね〜
⑯

実践：ウェブサイトを自動生成

2.0の機能を使ってウェブサイトを作る手順を解説！

  • プロジェクトフォルダーを読み込む
  • 設定で承認要求を減らし、エージェントに自由に行動させる
  • Stitch MCP（UIデザインツール）のプラグインをセット
  • Gemini 2.5 Flashをモデルに選択
  • 「コーヒー豆の販売サイトを作って」などとチャットで指示
Stitch MCPを使うと高品質なウェブサイトが自動生成される！非エンジニアでもサイト作れる時代が来た〜
⑰

修正もチャットで簡単

ウェブサイトが生成された後も、チャット経由で修正を依頼できます。

  • デザインの修正 — 「色を青系にして」「フォントを変更して」など
  • 機能の追加 — 「お問い合わせフォームを追加して」など
  • ファイルはプロジェクトフォルダーに自動保存される

対話形式で改善を重ねていくことができるため、プログラミング知識がなくてもプロ品質のサイトが作れます。

AIがコードを書いて、人間がデザインの方向性を決める。この分担が理想的だね！
