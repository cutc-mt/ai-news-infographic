DeepSWE ─ コーディングベンチマークの常識が覆る

⑦DeepSWEベンチマークの概要

  • スタートアップDatacurveが2026年5月26日に公開
  • 113のタスク・91のOSSリポジトリ・5言語を網羅
  • TypeScript, Go, Python, JavaScript, Rustに対応
  • 各タスクにプログラムベースの検証器を搭載
  • 参照解は平均668行・7ファイルに及ぶ大規模タスク
従来のSWE-Bench Proは平均120行/5ファイル。DeepSWEはその5.5倍の規模！

⑧DeepSWEランキング結果

  • GPT-5.5が70%で圧倒的1位
  • GPT-5.4が56%で2位に
  • Claude Opus 4.7が54%で3位
  • Claude Sonnet 4.6は32%、Gemini 3.5 Flashは28%
  • Claude Haiku 4.5はSWE-Benchで39%だがDeepSWEで0%に崩壊
Haikuの0%は衝撃！従来のベンチマークで「強い」と思われていたモデルが…

⑨SWE-Bench Proの検証器の問題点

  • 正解を24%の確率で誤って不合格にしていた
  • 不正解を8.5%の確率で誤って合格としていた
  • DeepSWEの検証器は誤否定0.3%・誤肯定1.1%と高精度
  • 創造的な解決策がテストスイートに阻害される事例を発見
  • 汚染問題: GitHub履歴から学習データに解答が混入するリスク
業界が頼ってきたベンチマークの誤り率が約1/3って…信じられない！
