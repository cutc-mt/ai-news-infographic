OpenHuman ─ 人間中心のデスクトップAIエージェント

①OpenHumanとは？

  • TinyHumans.aiが開発するオープンソースの個人AIエージェント
  • ライセンス: GPL-3.0（OpenClaw/HermesのMITとは異なる）
  • デスクトップネイティブのアプリでmacOS/Windows/Linuxに対応
  • Rust 65.2% + TypeScript/React 19 フロントエンド、Tauri v2ベース
  • 2026年2月18日にパブリックレポ作成。プロジェクトわずか3ヶ月
たった3ヶ月で20k+ GitHubスター！爆発的人気だよ〜

②Neocortex ─ ローカルファースト記憶エンジン

  • Memory Treeアーキテクチャでデータを圧縮・構造化
  • 3つのツリー構造: Source / Topic / Global
  • Obsidian互換のMarkdown保管庫として保存
  • 最大10億トークンの記憶容量に対応
  • 20分ごとに自動フェッチで接続先データを同期
  • Markdownファイルを直接編集可能！次回検索に即反映
他のエージェントは「コールドスタート」だけど、OpenHumanは最初からあなたのことを知ってる！

③Screen Intelligence ─ 画面を読み取るAI

  • 現在の画面状態をリアルタイムに認識
  • OSレベルのインライン自動補完機能
  • メールクライアント、ドキュメント、コード環境どこでも動作
  • ブラウザ拡張不要のネイティブ音声入出力
  • デスクトップマスコットが6つの感情状態を表示
「今何してる？」と聞かなくても、画面を見て理解してくれるのがすごい！

④Subconscious ─ 背景で自学習する脳

  • バックグラウンド自己学習ループが常時稼働
  • 蓄積データからワークフローのパターンを自動識別
  • ユーザーがクエリしなくても事前に関連情報を準備
  • 保存 → 検索だけでなく統合・優先順位付けも実行
  • デフォルト5分間隔でタスクを評価（最小5分）
  • 3つの決定: Skip / Act / Escalate
まるで人間の「無意識」みたい！寝てる間も考えてくれるのが革新的ぽん！

⑤118+の統合 & ツールキット

  • 自動データ取り込み: Gmail / Notion / Slack
  • Composioカタログ経由で118+のツールにワンクリック接続
  • GitHub / Stripe / Shopify / HubSpot / Salesforce等に対応
  • Google系: Calendar / Drive / Docs / Sheets / Meet
  • コミュニケーション: Discord / Telegram / WhatsApp / Teams
  • 開発ツール: Linear / Jira / Trello / Asana / Figma
Composio経由でOAuth接続が超簡単！ yamlいらずの設定すごい

⑥Google Meet AIエージェント

  • ミーティングに本当の参加者として参加（名前・顔・タイル付き）
  • 発言をリアルタイムにSTTで書き起こし、話者を識別
  • ミーティング中に直接発言可能（スピーカー経由ではない）
  • マスコットのアニメーションをカメラ映像として配信
  • 会議中でも記憶検索・ツール実行・Subconscious出力にアクセス
  • 議事録不要！会話がMemory Treeに自動保存される
会議に「AI同事」が参加する世界が来た！議事録取りが過去の遺物に…
TokenJuice & マルチモデル対応

⑦TokenJuice ─ コスト削減の秘密兵器

  • 全ツール実行結果に対してトークン圧縮レイヤーを適用
  • 3層のルールシステム: Builtin → User → Project
  • HTML→Markdown変換・長URL短縮・CJK/絵文字をグラフェム単位で保持
  • 70〜80%のコスト・レイテンシ削減を達成
  • プロジェクト設定: .tokenjuice/rules/ にレポ単位で保存可能
ログ出力やHTMLみたいな重いペイロードで特に効果大！財布に優しいぽん

⑧1サブスクで30+プロバイダ

  • 1つのサブスクリプションでClaude / GPT / Llama / Gemma等にアクセス
  • タスク自動ルーティング: タスクに応じて最適モデルを選択
  • hint:reasoning / hint:fast / hint:code 等のタスクヒント
  • vision系タスクにはhint:visionで自動的に画像対応モデルを選択
  • LM Studioローカルプロバイダにも対応
  • ローカルAI有効時は軽量タスクをローカル推論にオフロード
ユーザーがモデルを選ぶんじゃない！タスクがモデルを選ぶ世界！

⑨プライバシー & セキュリティ

  • ローカルファースト設計: 記憶データはローカルに暗号化保存
  • SQLiteでチャンク・スコア・エンティティ索引をローカル管理
  • 開発者は「ユーザーデータで学習しない」と宣言
  • ただしGmail等の個人情報を同期するので注意が必要
  • GPL-3.0でコード監査可能（OpenClawのMITより制限が強い）
  • パストラバーサル防止・DNS検証・オーディオサイズチェック等のセキュリティ対策
Gmailの全文を同期するから、プライバシーには十分注意してね！
Hermes Agent & OpenClaw ─ 既存の王者たち

⑩Hermes Agentの強み

  • 157k+ GitHubスターの成熟したオープンソースエージェント
  • 自己改善ランタイム: クローズド学習ループで継続的に成長
  • ライセンス: MIT（商用利用がより自由）
  • オープンソース自律エージェント分野での長年の実績
  • OpenHuman登場直前まで最も使われるエージェントの地位
長年自律エージェント界の王座に君臨！実績と信頼性は抜群だよ

⑪OpenClawの立ち位置

  • 373k+ GitHubスターで圧倒的スター数トップ
  • マルチチャネル: 20+のメッセージング画面に対応
  • ライセンス: MIT
  • ゲートウェイ型アーキテクチャでチャネル間の統合が強み
  • 豊富なスキルシステムとエコシステム
スター数ではOpenClawがダントツ！でも「記憶」ではOpenHumanが勝つかも？

⑫3大エージェントの比較

  • OpenHuman 記憶最強 | Hermes 自己改善 | OpenClaw チャネル最強
  • OpenHumanだけが「あなたを知る」前にツールを歩いて記憶を書く
  • Hermes/OpenClaw: APIキー+YAML+ターミナル必須の高学習コスト
  • OpenHuman: デスクトップインストーラで5分セットアップ
  • HermesがOpenClawの地位を奪った翌日にOpenHumanが登場した劇的な展開
エージェント界の三つ巴！それぞれ強みが全然違うのが面白い！
セットアップ & 初週体験

⑬OpenHumanのインストール

  • Tiny Humanインストーラーをダウンロードして実行
  • ランタイム選択: ローカル推奨 or クラウド
  • Google / Twitter / GitHubアカウントでサインイン
  • カスタム設定: テキスト/音声モデル選択・OAuth接続
  • CLIインストールも可能: curl | bash (macOS/Linux)
  • TauriベースでElectronより軽量・高速・安全
ターミナルを開いたことがないお父さんでも使えるのがOpenHumanの目標！

⑭初週の具体的な体験

  • Day 1: Gmail / GitHub / Notionを接続。OAuthスコープを慎重に確認
  • Day 1夜: 初回同期が20分ループで開始。Markdownに圧縮開始
  • Day 2: 「認証マイグレの結論は？どのIssueが残ってる？」と直接質問可能に
  • Day 3-5: コンテキストが深まり、精度が向上していく
  • 1週目の終わりには3つのツール横断で文脈理解が可能に
他のエージェントなら毎回説明しなきゃいけないことが、聞くだけで答えが出る！
