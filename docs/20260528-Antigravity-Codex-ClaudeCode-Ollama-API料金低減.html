動画の概要 ─ AI開発ワークフローの最適解

①今回の動画テーマ

  • Antigravity（アンチグラビティ）で爆速プロトタイピング
  • その後Claude CodeやCodexで本格的に拡張していく手法
  • Ollama（オラマ）のローカルAIでテスト費用をゼロにする方法
  • API料金を抑えつつ高品質な開発を行うハイブリッド戦略
  • チャンネル: AI共創イノベーション（佐藤氏）
初動はアンチグラビティでサクッと作って、後から拡張する！このムーブが超効率的だよ〜

②開発ワークフローの全体像

  • AntigravityでGemini 3.5 Flashを使って初期プロトタイプを爆速生成
  • Claude CodeやCodexで機能拡張・品質向上
  • OllamaでローカルLLMを動かしてテストコストを削減
  • APIモデルで本番の品質確認を行い仕上げる
  • ローカル ↔ APIのハイブリッド運用がポイント
4ステップで効率よく高品質なアプリが完成する！段階ごとに適切なツールを使い分けるのが鍵ぽん！

③Antigravityの強みと弱み

  • 強み: Gemini 3.5 Flashでめちゃくちゃ早い初期開発
  • プロトタイプ作成には最適なツール
  • 簡単なアプリならそのまま拡張しても十分対応可能
  • 弱み: 複雑なものにはまだ少し弱い
  • Gemini 3.5 Proの登場に期待中
初動だけなら最強！サクッと形にして、後から本格的に育てるのが正解だよね〜
Antigravityによる初期開発 ─ 爆速プロトタイピング

④Antigravityとは何か

  • Googleが開発したエージェントファーストの開発プラットフォーム
  • 内部でGemini 3.5 Flashを搭載し高速なコード生成を実現
  • Chrome拡張機能の作成も1発で可能
  • 実行プランの自動生成・思考プロセスの可視化機能付き
  • 画面操作の許可制で安全にブラウザ自動化ができる
プロンプトを入力すると「実行プラン」を組んでくれる！それがまた賢いんだよね〜

⑤NotebookEL ─ 検索アプリの開発

  • 前回アンチグラビティで作ったノート型アプリ
  • 検索機能を追加: 部分一致・完全一致・大文字小文字区別なし
  • 例: 「OpenAI」を入力すると関連スレッドだけを的確に抽出
  • スクロールボタンで上下にスムーズ移動
  • 検索機能は1発で作成できた！Antigravityの実力を発揮
「OpenAI」って入れたらピンポイントで出してくれる！検索がサクサク動くよ〜

⑥ダッシュボード形式への拡張

  • スクロール型からダッシュボード型にパワーアップ
  • 出店・日付など詳細情報を一覧表示
  • 項目をクリックすると画面が開くモーダル方式に変更
  • パネルが自由に移動できるドラッグ機能を追加
  • レスポンシブ対応でスマホでも動くように実装
Claude CodeとCodexで拡張した成果！初期開発の上にさらに価値を積み上げるポン！
Claude Code & Codexによる拡張 ─ 本格的な機能追加

⑦AIサイドバーの実装

  • AIサイドチャット機能を新規開発
  • ボタン一つでサイドバーが開閉するUI
  • チャットログが残るように改善（前回はログ消失）
  • 過去のやり取りや要約結果が履歴として表示される
  • クリアボタンで履歴をリセット可能
前回はチャットが消えちゃってたけど、今回はログが残る！超便利に進化したよ〜

⑧要約機能の実装

  • テキストを入力して「要約して」でAIが自動要約
  • 解析中はキラキラ光るアニメーションが表示される
  • 記事の主なポイントが箇条書きで出力される
  • テスト例: Anthropicの評価額に関するAI記事の要約
  • 停止ボタンで実行を途中で中断できる
「キラキラ光る」演出はClaude Code/Codexで追加した進化ポイントだよ！UX大事！

⑨DOM解析とセキュリティ対策

  • ページのDOM構造を内部で解析し構造を把握
  • 解析結果はコンテキストに含めない工夫で節約
  • 保護モード: APIキー等の安全を重視した安全動作モード
  • プロンプトインジェクション対策を実装
  • 自動モードオフ + 保護モードオンで最も安全
APIキーが入ってるから安全対策は必須！怖いなって思ったら保護モードで！
実践デモ ─ Amazonでの自動買い物

⑩Amazon自動操作のデモ

  • プロンプト: 「Amazonページに行って昆布1kgコスパいいものを検索してカートに入れて」
  • 自動で実行プランを生成してくれる
  • Amazonトップページにアクセス
  • 昆布1kgを検索
  • コスパの良い商品を選択してカートに追加
「昆布1kg」にしたの面白いw 実際に検索してカートに入れるところまでできた！

⑪実行プランの仕組み

  • プロンプトから自動でステップを計画（プランニング機能）
  • 思考プロセスがリアルタイムで可視化される
  • 1ステップごとに「許可」ボタンで確認しながら進行
  • 自動モードをONにすると全自動で一気に実行
  • サイドエージェントが自動でプロンプトを生成して入力
思考プロセスが見えるのすごくない？AIが何考えてるのか分かるのが良い！

⑫回数制限と安全対策

  • 自動制限の上限を8回に設定して無限ループを防止
  • 上限解除モードも今後追加予定
  • 制限に達したら「続行してください」で再開可能
  • 見てないと危ないかもしれないので注意喚起
  • 自動モードはプロンプトインジェクション等のリスクあり
無限ループ防止は安全対策として大事！放置すると危ないからね〜
Ollama活用 ─ ローカルAIでコストゼロ開発

⑬Ollamaとは

  • Ollama（オラマ）: オープンソースのローカルLLM実行ツール
  • インターネット不要でローカル環境で完結
  • Meta Llama・Google Gemmaなど多数のモデルに対応
  • API費用が完全にゼロで利用可能
  • 2026年5月にOpenAI Codexアプリにも対応
無料でLLMが使える！開発中のテスト費用がゼロになるのは超お得だよね！

⑭ローカルAI切替機能

  • ボタン1つでAPI ↔ ローカルAIを即座に切り替え
  • ローカルにはGemma 4Bモデルを導入
  • ローカル時は解析中のアニメーションも動作する
  • 画面がローカルモデルに切り替わる演出で分かりやすい
  • 「要約して」の機能もローカルモデルで実行可能
ポチッと切り替えるだけでローカルAIに！API代節約できるの最高！

⑮API費用低減の戦略

  • ステップ1: Ollamaのローカルモデルで低コストテスト
  • ステップ2: 動作確認できたらAPIモデルで本番品質確認
  • ステップ3: 品質OKなら本番公開
  • ローカルテストでは「動けばOK」で品質は問わない
  • 公開方法: APIのみ or ハイブリッドの2択
ローカルで動いたらAPIで仕上げる！この流れでコスト爆減ポン！
まとめ & ベストプラクティス

⑯画面操作の進化ポイント

  • 画面キャプチャ機能: 現在の画面をそのまま取り込んでAIに認識させる
  • 画像をクリックして詳細表示できる機能を追加
  • パネルのドラッグ＆ドロップ移動で自由なレイアウト
  • 小さくしてもレスポンシブに動作する実装
  • UIデザインのさらなる改善も今後の課題
前回のバージョンからだいぶ進化してる！キャプチャ機能が付いたのは大きい！

⑰ツール選びの判断基準

  • Antigravity → 初期プロトタイプ・サクッと形にしたい時
  • Claude Code → 本格的な機能拡張・コード品質を上げたい時
  • Codex → OpenAIエコシステムでの本格開発
  • Ollama → テスト段階・コスト削減・プライバシー重視
  • 小さなアプリならAntigravity単体でも十分な場面もある
用途に応じて使い分けるのがプロのやり方！全部混ぜるのが最強！

⑱まとめ ─ AI開発の最適解

  • 初動の速さはAntigravityに任せるのが最大のメリット
  • Ollamaでテスト費用をゼロにしつつ、APIで品質を担保
  • 保護モードと回数制限で安全な自動操作を実現
  • Claude Code / Codexで機能拡張して本格的なアプリへ
  • AIツールの組み合わせこそが開発効率の鍵！
Antigravityでサク作り → Codex/Claude Codeで拡張 → Ollamaでコスト削減！これが最強ワークフローぽん！🚀
