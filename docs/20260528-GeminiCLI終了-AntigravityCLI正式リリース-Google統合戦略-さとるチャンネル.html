🚀 Gemini CLI終了 → Antigravity CLI正式リリース

Googleの開発者向けツールがAntigravity 2.0に統合！6月18日にGemini CLIは消費者向け終了。速さ・非同期エージェント・完全移行ガイドを徹底解説

①今回の3つのテーマ

  • Gemini CLIとは何か？なぜ終了するのか？
  • Antigravity CLIで具体的に何が変わるのか？

Googleの開発者ツール戦略の大きな転換点！しっかり押さえておこう！

②そもそもGemini CLIって何？

  • Googleがリリースした開発者向けのターミナルAIツール
  • ChatGPTのように画面を開かなくてもコマンドでGeminiが動く
  • コードを書いたりファイルを操作したり、開発作業を丸ごとAIに任せられる
  • ターミナル環境だけでAIを活用できる画期的なツールだった
画面を開かなくてもOK！コマンド打つだけでAIが動くの、めちゃ便利だったんだよね

③なぜ終了するのか？ツールの散乱

  • これまでGoogleのAI開発ツールはバラバラだった
  • Gemini CLI（ターミナルツール）
  • Gemini Code Assist（ブラウザ拡張機能）
  • Antigravity 2.0（デスクトップアプリ）
  • これらを全部Antigravity 2.0を中心に1つに統合する戦略
別々に散らばってたツールを一つのプラットフォームにまとめるんだ！
スケジュールと統合の方向性

④移行スケジュール

5月19日 Antigravity CLIが正式リリース！
5月27日 今回のニュース報告日
6月18日 Gemini CLIとCode Assist拡張が消費者向けに完全終了
6月18日までに移行を完了させないとね！

⑤企業向けは継続利用可能

  • StandardやEnterpriseの企業契約ユーザーは引き続きGemini CLIを使える
  • 個人・消費者向けは6月18日で完全終了
  • 消費者はAntigravity CLIへの移行が必須
企業契約の人は安心してOK！でも個人開発者は要注意だよ！

⑥デスクトップ版と完全に同じ体験

  • Antigravity CLIはデスクトップ版と完全に同じエージェント基盤を使用
  • 同じモデル、同じ設定が共有できる
  • ターミナルでもデスクトップでも同じ体験ができる
  • シームレスな環境切り替えが可能に
ターミナル派もデスクトップ派も同じ機能が使えるのは嬉しい！
Antigravity CLIで具体的に何が変わるのか

⑦最大の変化は「速さ」

  • Gemini CLIはNode.jsベースだった
  • Antigravity CLIはRustでゼロから作り直し
  • ターミナルの起動速度が明らかに早くなった
  • コマンド入力後の反応速度も大幅に向上
Rust製だから爆速！体感で違いがわかるレベルの速さらしいよ！

⑧非同期エージェント機能

  • これまでは1つの作業を投げると終わるまでターミナルがロックされていた
  • Antigravity CLIはバックグラウンドで動かしながら別の作業ができる
  • 大規模なコード整理をエージェントに任せつつ別タスクを並行で進められる
  • 複数のエージェントを同時に実行できるのが革命的
ターミナルがロックされないのが地味にでかい！並行作業が捗る！

⑨デスクトップ版との連携

  • Antigravity 2.0のデスクトップ版との会話のインポートが可能
  • カスタマイズ性が強化されキーバインドやテーマを自由に変更できる
  • ツール許可はStrictとAlways Proceedで切り替え可能
  • 慎重に動かしたい時とガンガン進めたい時で使い分けができる
セキュリティ重視の人もスピード重視の人もOK！
機能の引き継ぎと互換性

⑩Gemini CLIの機能は引き継がれる

  • Agent Skills（エージェントスキル）→ 引き継がれる ✅
  • Hooks（フックス）→ 引き継がれる ✅
  • Sub-agent Extensions（サブエージェント拡張）→ 引き継がれる ✅
  • ただしWorkspace Skillsは手動移行が必要 ⚠️
大部分は自動で引き継がれるけど、Workspace Skillsだけ要注意！

⑪新機能の盛り沢山ぶり

  • Rust製で起動・応答が爆速
  • 非同期エージェントでバックグラウンド実行が可能
  • iPad + SSHなどモバイル環境でも快適に動作
  • テーマやキーバインドの自由なカスタマイズ
  • Strict / Always Proceedで許可モードの使い分け
ただのアップデートじゃない！ほぼ別物レベルの進化だよ！

⑫ツール統合の歴史と戦略

  • 以前: Gemini CLI + Code Assist + Antigravity 2.0 が別々に存在
  • 現在: Antigravity 2.0を中心に全部統合
  • Googleの開発者ツール戦略そのものの転換点となる
  • これは単なるバージョンアップではなくプラットフォームの統一
Googleの開発者戦略が大きく変わる重要な瞬間！歴史的な転換点かも！
開発者のリアルな声 ─ 好評のポイント

⑬好評: とにかく爆速！

  • 開発者からの一番多かった反応が「爆速」
  • Rust製の威力でGemini CLIの体感速度を大きく上回る
  • 起動・応答・実行すべてが体感でわかるレベルで高速化
「速さがヤバい」の一言！開発スピードが格段に上がるね！

⑭好評: 非同期エージェントが革命的

  • 二番目に好評だったのが非同期エージェント機能
  • 複数のエージェントを同時に動かせる
  • 「コード整理を任せつつ別機能を開発」という並行ワークが可能に
  • 待ち時間が減って開発効率が劇的に向上
複数エージェントの同時実行は本当に革命的らしい！みんな絶賛してる！

⑮意外な好評: モバイル環境対応

  • iPad + SSHからの接続が快適
  • タブレット環境でもストレスなく動作
  • Rust製の軽量さがリモート環境でも生きている
  • ソファでコーディングするモバイル開発者に嬉しい改善
iPadでカフェ開発できるのは想像以上に便利そう！
課題点と移行時の注意点

⑯報告されている課題点

  • レート制限の厳しさが一番多い指摘
  • プランモードとゴート（Goat）表示が今のところない
  • 拡張機能の移行で一部の機能が脱落するケースを報告
  • Workspace Skillsは手動対応が必要
レート制限には要注意！急いでても焦らず使おう！

⑰移行時の注意点

  • 自動移行だけでは完全に引き継げない
  • Workspace Skillsは手動で移動が必要
  • MCP設定ファイルの書き換えが必要
  • 設定ファイル名がsettings.json → mcp.jsonに変更
  • キー名もurl → serverUrlに変更されている
移行チェックリストを忘れずに！手動対応が必要な項目があるよ！

⑱まとめ: 戦略の転換点

  • 今回のAntigravity CLIはただのアップデートではない
  • Googleの開発者ツール戦略そのものの転換点
  • Rust製で爆速＋非同期エージェントで次世代へ
  • 6月18日までに移行を完了しよう！
Googleの開発者ツールが新時代に突入！早めに移行して乗り遅れないようにしよう！
