🧙 Hermes Agent 概要・使い方徹底解説

永続メモリ・自己進化スキル搭載の無料AIエージェント。使えば使うほど賢くなる自分専用AI秘書

Hermes Agent とは ─ 自分専用AIエージェントの革命

①Hermes Agent（ヘルメスエージェント）とは

  • 開発: Nous Research（米国のAI企業）
  • 2026年2月リリース、わずか3ヶ月でGitHubスター16万7,000件以上
  • 完全無料・オープンソースのAIエージェント
  • macOS・Windowsのターミナルで動作、Web UIも標準搭載
  • OpenRouter OpenAI xAI のサブスクリプションに連携可能
無料で自分専用AI秘書が手に入る！3ヶ月で16万スターはすごい…

②開発元：Nous Research（ニュースリサーチ）

  • 米国のAI企業。オープンソースと分散型のアプローチで挑むポジション
  • MetaのLlamaシリーズを微調整したLSシリーズを開発
  • OpenAIやAnthropicのような大手に対してオープンソース陣営の代表格
  • 誰でも自由に利用・改造できる体制でAI民主化を推進
オープンソースの雄が作った、誰でも使えるAIエージェント！

③最大の特徴：使えば使うほど賢くなる

  • ChatGPTやClaudeとの決定的な違い：作業手順をスキルとして保存
  • 自己進化スキル：複雑なタスクをこなすと、手順を自動で記憶
  • 次回似たタスクには保存したスキルを再利用→効率が上がる
  • 保存したスキルは後から手動で改善も可能
使うほど賢くなるAIなんて、まるで育てるみたい！
永続メモリ ─ AIが忘れない3つの記憶システム

④永続メモリ（Persistent Memory）

  • 3つのファイルにマークダウン形式で記憶を保存
  • セッションをまたいでも記憶が消えないのが最大の特徴
  • ユーザーが直接編集も可能で好みにカスタマイズできる
🧠 Soul.md ─ AIの人格・話し方
📝 Memory.md ─ エージェントのメモ
👤 User.md ─ ユーザーの好み・属性
3つのファイルでAIの記憶を完全管理。まるで脳の設計図！

⑤Soul.md ─ AIの人格を定義

  • AIエージェントの人格・性格を設定するファイル
  • 話し方のトーンやスタイルを自由にカスタマイズ
  • 「丁寧な口調」「フレンドリー」「専門的」など細かく指定可能
  • これにより自分好みのAI秘書に仕立てられる
Soul.md
├── 人格・性格の設定
├── 話し方のトーン
├── 応答スタイル
└── 役割の定義
AIに「丁寧な秘書風」にしてもらえば仕事がはかどる！

⑥Memory.md & User.md

  • Memory.md: エージェント自身が覚えておくメモ。作業の進捗や学習内容
  • User.md: ユーザーの好みや属性を記録。名前・趣味・嗜好など
  • 時間が経っても文脈を覚えているので何度も説明不要
  • 手動で直接編集すれば、即座にAIの振る舞いを変更可能
「前回の続きから」が自然にできる！毎回ゼロから説明しなくてOK
自己進化スキル ─ AIが自分で学ぶ仕組み

⑦自己進化スキルの仕組み

  • 複雑なタスクを完了すると、自動で作業手順を保存
  • 失敗ポイントや確認項目も記録して賢くなる
  • 保存形式はマークダウンのスキルファイル
  • 次回似たタスク時に保存済みのスキルを自動で再利用
失敗も記憶するから、同じミスを繰り返さない！

⑧スキルの活用と改善

  • スキルが蓄積されるほどタスクの処理効率が向上
  • 保存済みのスキルは手動で改善・編集もできる
  • 使えば使うほどパーソナライズされた賢さに育っていく
  • スキルをコミュニティで共有することも可能
自動保存される内容:
├── 作業手順（Step by Step）
├── 失敗ポイントと対策
├── 確認項目チェックリスト
└── 効率化のコツ
AIを育てる感覚！使うほど自分に合っていく

⑨機能拡張ツール

  • Web検索：最新情報の検索と収集
  • ブラウザ操作自動化：Webページの操作をAIに任せる
  • ターミナルコマンド実行：PC上の操作を自動化
  • ファイル編集：ドキュメントやコードの編集
  • 各ツールは有効/無効を個別に切り替え可能
  • MCP対応で外部ツールの接続にも対応
必要なツールだけONにしてカスタマイズ！
連携・拡張機能 ─ チャットアプリ・Cron・サブエージェント

⑩チャットアプリからのリモート操作

  • LINEからAIエージェントに指示を出せる
  • Telegramでのリモート操作に対応
  • Discord・Slackとの連携も可能
  • スマホから外出先でも自宅のPCのAIを操作できる
LINEで「明日の天気調べて」だけ送ればOK！

⑪Cron対応：定期実行

  • 指定した日時にタスクを自動実行する機能
  • 「毎朝8時にニュース要約」「毎週金曜にレポート作成」など
  • 寝ている間にAIが作業を自動で完了してくれる
  • 結果はLINEやDiscordに通知を受け取れる
寝てる間にAIが仕事を終わらせて朝には報告が届く…最高！

⑫サブエージェント機能

  • 独立した複数のエージェントを並列実行できる
  • タスクごとに専用のエージェントを割り当て可能
  • 複数のタスクを同時並行で処理→大幅な時間短縮
  • 複雑なプロジェクトをチーム分けして効率よく進める
AIチームを作って並列作業！大規模プロジェクトも楽々
セキュリティ ─ 安全に使うための仕組み

⑬安全機能と注意点

  • Telemetry: 怪しいコマンドを自動で停止する仕組み
  • パスワードやAPIキーの自動隠蔽機能（デフォルトON）
  • 機密情報がログに残らないように自動保護
  • 歴史が浅く安全性は高いが、ユーザー増加で攻撃リスクに注意
人気が上がれば攻撃者も増えるかも…セキュリティ設定はしっかり！

⑭インストール方法

  • Mac: ターミナルで1つのコマンドでインストール完了
  • Windows: PowerShellでインストール（β版、Macほどテストなし）
  • ワンクリックで自分専用のAI秘書が起動
# Macの場合（ターミナル）
curl -fsSL https://hermes.ai/install | bash

# Windowsの場合（PowerShell）
irm https://hermes.ai/install.ps1 | iex
コマンド1発でインストール！お手軽さが魅力

⑮AIモデルの選び方

  • OpenAI公式 Google xAI のサブスクリプションを利用可能
  • Codex認証でGPT-5.5等もHermes Agentで使用可能
  • 無料ならOpenRouter連携がおすすめ

無料モデルでも十分使える！まずはOpenRouterから始めよう
初期設定・Web UI ─ すぐに使い始めるための手順

⑯初期設定のステップ

  • GitHubのGetting Startedガイドに沿って設定
  • プロバイダーとAIモデルを追加
  • ツール・スキルの有効/無効を切り替え
  • メッセージアプリの設定（LINE・Telegramなど）
  • スラッシュコマンドにも対応している
GitHubのガイドが親切だから初心者でも安心！

⑰Web UIの活用

  • プロバイダー追加・AIモデル変更・ツールの有効/無効化
  • 日本語表示にも対応している
