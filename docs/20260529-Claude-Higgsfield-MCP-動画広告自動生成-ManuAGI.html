⑩ Claude CodeでのMCP設定方法

  • .claude/settings.json または claude mcp add コマンド
  • MCPサーバーのURL・認証情報を設定
  • Claude Codeが自動的にツールとして認識
  • 接続後はチャットで自然に指示できる
{
  "mcpServers": {
    "higgsfield": {
      "url": "https://higgsfield.ai/mcp",
      "token": "YOUR_TOKEN"
    }
  }
}
⚙️ 設定は一度だけ！
あとはClaudeが自動で使える

⑭ MCPのアーキテクチャ図解

  • ユーザー → Claude Codeで自然言語の指示を入力
  • Claude → MCP Client経由でリクエストを送信
  • MCP Server → 対象ツールにコマンドを実行
  • Higgsfield → 動画・画像・広告を生成して返す
  • 結果をClaudeがユーザーにフィードバック
ユーザー → Claude → MCP → Higgsfield → 生成結果 → Claude → ユーザー
🔄 一切のツール切り替え不要
ユーザーはClaudeと話すだけ！

⑮ セキュリティとアクセス制御

  • 認証情報の安全な管理 — Claudeがトークンを安全に保持
  • MCPサーバーごとの権限分離
  • ユーザーの明示的な許可なしではツール実行されない
  • JSON-RPCベースで標準化された通信
🔒 セキュリティはMCPの核心
