Claude Codeを自分専用に育てる！

対話履歴から興味や学習内容を抽出してパーソナライズする方法
📅 2026年5月30日
Claude Code × パーソナライズ × 興味プロファイル

①「Claude Codeを育てる」とは？

  • 単にコードを書かせるだけでなく、使えば使うほど自分を理解してくれる存在にする
  • YouTubeコンテンツ制作や調べ物にも活用している
  • 自分の興味に合った提案や情報収集が可能に
  • いわば自分だけのAIエージェントを育てるイメージ
にゃんたさんはコード以外にもYouTube作成・情報収集など幅広く活用中！

②仕組みの基本：対話ログの蓄積

  • claude codeと対話するとローカルに会話ログが自動蓄積される
  • 形式はJSONL（JSON Lines）でセッションごとに保存
  • このログを解析すれば、疑問・関心・興味が抽出できる
  • 結果をinterests.mdにまとめてプロファイル化
ローカルファイルにログが溜まるのがclaude codeの強み！

③Andrej Karpathyの知識ベースとの違い

  • Andrej氏はOpenAI創業者でAnthropicにジョイン
  • 氏のアプローチは広範な知識を整理するシステム
  • 知識の関係性や構造を理解する目的
  • にゃんたさんの手法は興味・関心に特化してより実用的
  • 調べ事や勉強のサポートに最適化
知識整理 vs 興味抽出、目的が違う！

④実演：興味プロファイルの活用

  • claude codeで「私の最近興味あることは？」と入力
  • interests.mdを参照して即座に回答
  • エージェントハーネスの評価やClaude Codeの育て方などが表示
  • 対話ログを反映した正確なプロファイルが生成される
自分の興味をclaude codeがちゃんと理解してる！

⑤興味に沿った情報収集

  • 「私の興味にありそうなことを調べて」
  • プロファイルを参照して関連情報を自動収集
  • YouTubeネタ探しにも活躍（にゃんたさんはこれをよく使う）
  • 以前動画の「言語モデルの出力が静かに壊れてる」の続編論文を提案
興味に合った提案をしてくれるから助かる！

⑥YouTube以外でも応用可能

  • 会社で使っている場合も同様の仕組みが使える
  • 会話ログから質問を抽出して自作問題集が作れる
  • メール作成の入力情報から得意分野がわかる
  • 本質は「対話ログを解析してパターンを抽出」する話
YouTubeやってない人も応用できる！

⑦Webアプリを使わなくてよくなる

  • パーソナライズが進むとClaudeのWebアプリが不要に
  • Webアプリの会話履歴エクスポート機能で移行も可能
  • claude codeなら直接ローカルにログが溜まるのが最大のメリット
  • 自分のデータを自分で管理できる安心感
ローカルにログがあるから一番便利！

⑧オープンソースコードの提供

  • にゃんたさんがGitHubに最小限のコードを公開中
  • 概要欄にリンクあり
  • 手元で試したい人は必見
  • claude codeのスキルとして組み込んで使える
GitHubからコードを取ってきてすぐ始められる！

⑨ノイズ除外の仕組み

  • 「こんにちは」などの雑談は興味分析から除外
  • スキル実行時の定型入力もノイズとして扱う
  • Claude Codeを使うとスキルの実行コマンドもログに残る
  • 意味のある対話だけを抽出するフィルターが重要
雑談を除外しないとプロファイルが汚れちゃう

⑩スコアリング：深掘り度の評価

  • 質問の深掘り度合いに応じてスコアを付与
  • 1回の質問で済んだ話題 → スコア低め
  • 何回も質問した話題 → 高いスコア＝強い関心
  • 「難しかった」か「すごく興味がある」かをスコアで判定
何回も聞くトピックは本気の興味！

⑪時間の重み付け

  • 興味は日々変わるので最新のものを優先
  • 直近の質問と1ヶ月前の質問では重みが異なる
  • 最近聞いたことほど調べ事に参照されやすい
  • 「スコア3」で時間による重み付けスコアを算出
昨日の興味と今日の興味は別物！

⑫interests.mdの生成と活用

  • 興味度が高い順に並べて出力
  • 定期的に更新することで常に最新の状態を維持
  • claude codeが「この人はこういう人だ」と理解する
  • 検索や調べ物の際に自動参照される
claude codeの「自分への理解」がここに凝縮される

⑬スキル化：/interest-profile sync

  • claude codeで/interest-profile syncを実行
  • 会話ログを参照して自動で興味をまとめる
  • 抽出されたシグナルを分析してinterests.mdに保存
  • スキルとして登録しておけば何度でも実行可能
/interest-profile sync
スラッシュコマンド一発で同期完了！

⑭進捗管理：どこまで分析したか

  • 全ログを毎回参照すると入力が膨大になる
  • 「まだ分析していないログ」だけを抽出
  • どこまで処理したかを記録して差分更新
  • コストも節約できる賢い設計
差分処理でトークン消費も節約！

⑮応用：学習問題集の自動生成

  • 会話ログから学習内容を抽出して問題集を作成
  • HTML形式で見やすいレポートとして出力
  • 間隔反復で記憶の定着に役立つ
  • 毎日復習する習慣が作れる
Claude Codeが自分専用の問題集を作ってくれる！

⑯Claude Desktopアプリの活用

  • claude codeの定期実行にはDesktopアプリが最適
  • ルーティン機能で好きな時間に自動実行
  • 「スタートアップ時に実行」をオンにすれば常駐OK
  • PCを開けば自動で定期処理が走る
CronよりDesktopアプリがおすすめ！

⑰Cron実行の注意点

  • Cronで実行するとClaudeの認証が期限切れになる問題が発生
  • にゃんたさんもCronではうまくいかなかった
  • Desktopアプリを使う方が安定して動作する
  • loopコマンドもあるがセッションの起動が手間
認証切れに要注意！Desktopアプリが安全

  • ChatGPTからコーディングエージェントへの関心シフト
  • OpenClawやLMS AgentなどパーソナライズAIが流行
  • 6月15日からサードパーティ製ツールがサブスクで使えるように
  • 重量課金を避けられてより手が届くようになる

自分専用AIエージェントの時代が来る！
