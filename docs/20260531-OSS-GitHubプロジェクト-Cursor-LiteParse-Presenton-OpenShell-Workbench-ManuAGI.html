①Cursor Plugins: 拡張可能なプラグインシステム

  • Cursorエディターを拡張するオープンソースのプラグイン集
  • 外部のツールやサービスとAI開発ワークフローを接続
  • コアのエディター動作を変更せずに新機能を追加可能
  • チームごとにAIアシスト開発体験をカスタマイズ
Cursorをもっと自分好みにカスタム！

②LiteParse: 軽量ドキュメント解析ライブラリ

  • Run LLMが開発したドキュメント解析ライブラリ
  • ドキュメントを構造化データに変換してAIパイプラインで活用
  • テキストと文書構造を抽出しつつ処理を軽量・高速に保つ
  • 検索・インデックス・言語モデルパイプラインに統合可能
軽さと実用性のバランスが良い！

③Stable World Model: 環境モデル学習

  • ワールドモデルを構築するオープンソースの研究フレームワーク
  • データから環境の表現とダイナミクスを学習
  • 未来の状態を予測し、複雑な相互作用をモデル化
  • シミュレーションや推論システムの研究に活用可能
AIが「世界を理解」する第一歩！

④ESM: タンパク質生物学の基盤モデル

  • Biohubが開発したタンパク質研究向けのMLフレームワーク
  • タンパク質配列から表現を学習し構造予測を支援
  • 機能解析や生物学研究のワークフローに対応
  • ローカルまたはクラウド環境でモデルを実行可能
AI × 生物学の最前線！
