セクション2: GeminiCLI終了 → Antigravity CLI

⑤ Gemini CLI の終了（6月18日）

  • Googleがオープンソース「Gemini CLI」のAPIアクセス終了を発表
  • 6月18日にAPI終了。対象は無料ユーザー・AI Pro/Ultraサブスク・個人Code Assist
  • コミュニティが6,000件以上のPRをマージしたOSSプロジェクト
  • 有料のCode Assist Standard/Enterpriseは継続利用可能
  • OSSコミュニティから強い批判が噴出。大炎上の状態に
6,000件のPRを貢献した
コミュニティに裏切り。
OSS界隈で大炎上中！

⑥ 後継: Antigravity CLI

  • Gemini CLIの後継としてクローズドソースの「Antigravity CLI」へ移行
  • Google自身「Gemini CLIと完全同等の機能はない」と認める
  • 機能ダウングレードで移行。OSSコミュニティの裏切りと受け取られる
  • 開発者からの反応は圧倒的にネガティブ。「なぜOSSにしたのか」の疑問
  • 企業利益を優先した判断だが、Googleの信頼を損なう結果に
「同等機能はない」と自ら告白。
有料版への強制移行と
見られているのが問題。

⑦ OSSコミュニティ vs 企業の軋轢

  • 企業がOSSを参入のトロイの木馬として使い、後から切り捨てるパターン
  • 6,000件のPR＝膨大な無償貢献。それを一方的に打ち切る判断
  • 開発者の不信感が高まる。「企業OSSは信用できない」という空気
  • ビジネスとOSSのバランスが今後の重要な議論に
  • GitHub Copilot・Cursorなど他のCLIツールへの乗り換えが加速しそう
「無料でOSSを作るから
使ってね」→「有料版に変えて」
このパターンに開発者は疲弊。
