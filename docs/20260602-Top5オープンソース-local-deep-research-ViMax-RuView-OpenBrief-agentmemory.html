プロジェクト1: Local Deep Research

複雑な質問に自動で深いリサーチを行うAIアシスタント
LearningCircuit/local-deep-research

① Local Deep Research とは

  • 複雑な質問を入力するだけで、AIが自律的に深いリサーチを実行
  • Web・学術論文・ローカル文書を横断検索
  • 検索結果をすべて引用付きレポートに統合して出力
  • LLMが自律的に検索戦略を決定するLangGraph Agent Strategy
  • pip installで簡単導入。Python環境があればすぐ始められる
「〇〇について調べて」
と言うだけでAIが
論文もWebも全部調べる！

② 20種類以上のリサーチ戦略

  • Quick Facts: 短時間で基本事実を素早く把握
  • Deep Analysis: テーマを多角的に深く掘り下げる分析
  • Academic Research: 論文を中心とした学術的な調査
  • 複数LLM対応・複数検索エンジン対応で柔軟に構成可能
  • ユーザーの目的に応じて最適な戦略を自動選択
戦略が20種類以上！
クイック調べから
学術リサーチまで
シーン別に使い分け。

③ 完全ローカルで高精度リサーチ

  • RTX 3090 + Qwen3.6-27Bで完全ローカル稼働可能
  • 外部API不要。プライバシーを完全に保護したまま利用
  • SimpleQA 95%（n=500）という驚異的な精度を実現
  • xbench-DeepSearch 77%（n=100）でも安定した高スコア
  • 初の完全ローカルでの高精度リサーチ実現という快挙
クラウドAPI一切不要！
RTX 3090があれば
自分のPCで95%精度の
リサーチができる。

④ LangGraphで自律的に意思決定

  • LangGraph Agent StrategyでLLMが自律的に検索内容を決定
  • 「何を」「どこを」「どう深く」調べるかをAIが判断
  • 単純な検索ではなく多段階の推論を経て精密な回答を生成
  • 各種LLM・検索エンジンをプラグイン的に組み合わせ可能
  • 研究者・開発者・ビジネスパーソンまで幅広い用途に対応
「どう調べるか」まで
AIが考えてくれる。
人間は質問するだけ！

プロジェクト2: ViMax

マルチエージェント動画生成フレームワーク
HKUDS/ViMax

⑤ ViMax とは

  • マルチエージェント動画生成のフレームワーク
  • アイデアを入力するだけで一貫性のあるマルチショット動画に自動変換
  • キャラクターとシーンの視覚的一貫性を自動保証
  • 単発クリップではなく本格的な動画作品を生成可能
  • 香港大学（HKU）の研究チームによる学術プロジェクト
「このストーリーを動画にして」
と言うだけでAIのチームが
本格的な動画を作ってくれる！

⑥ 4つのAI役割が全自動で動画制作

  • Director: 全体の構成・クリエイティブな方向性を決定
  • Screenwriter: ナラティブに沿った脚本を自動執筆
  • Producer: キャラクター設計・ストーリーボード作成を管理
  • Video Generator: 最終的な映像素材を自動生成
  • 脚本執筆 → ストーリーボード → キャラクター作成 → 最終動画生成をエンドツーエンドで完結
監督・脚本家・プロデューサー・
カメラマンが全部AI！
人間はアイデアだけ出せばOK。

⑦ ストーリーテリングに集中できる

  • 技術的な実装はAIに完全に任せて、クリエイターはストーリーに集中
  • ショット間のキャラクター一貫性が自動で維持される
  • シーンの視覚的連続性もAIが自動保証
  • 動画制作のハードルが劇的に低下。アイデアがあれば誰でもクリエイターに
  • コンテンツ制作者・マーケター・教育分野など幅広い応用が期待
映像制作が「アイデア出し」
だけで完結する世界。
技術はAIが全部やる。

プロジェクト3: RuView

WiFi信号をリアルタイム3Dビジョンに変換
ruvnet/RuView

⑧ RuView とは

  • WiFi信号をリアルタイム3Dビジョンに変換する革新的プロジェクト
  • 普通のWiFiルーターだけでカメラ不要で人の動きを検知
  • 人の動き・姿勢をWiFi信号の変化からリアルタイムに推定
  • DensePose技術で人体の精密な3Dマッピングを実現
  • 安価なESP32などのマイコンで動作可能
カメラなしで人の動きが
見える？！
WiFi信号だけで
3Dビジョンを実現。

⑨ Claude Code & Codex プラグイン同梱

  • Claude Code & Codex向けプラグインがプロジェクトに同梱
  • 9つのスキル + 7つのCLIコマンド + 3つのエージェントを搭載
  • ESP32セットアップ・モデル訓練・センシングを一貫サポート
  • mmWaveレーダーにも対応。WiFi以外の信号源でも利用可能
  • AIコーディングエージェントとシームレスに連携して開発を加速
Claude Codeでセットアップから
モデル訓練まで
AIが自動でやってくれる！

⑩ IoT・セキュリティ・ヘルスケアへの応用

  • スマートホームIoT: カメラなしのプライバシー保護型モニタリング
  • セキュリティ: 侵入検知・異常行動検知を非侵襲的に実行
  • ヘルスケア: 高齢者の見守り・転倒検知に活用可能
  • カメラを設置しなくても人の存在・動き・姿勢を把握できる
  • 「見る」のではなく「信号で感じる」という全く新しいパラダイム
プライバシーを守りながら
人の動きを見守る。
カメラ不要の未来が来る！

プロジェクト4: OpenBrief

動画・音声を要約付きブリーフィングに変換するデスクトップアプリ
tantara/openbrief

⑪ OpenBrief とは

  • 動画・音声を要約付きブリーフィングに変換するデスクトップアプリ
  • Tauri v2ベース。Electronより軽量・高速なネイティブ体験
  • 動画/音声のダウンロード → 文字起こし（Whisper）→ 要約生成
  • すべてローカルで処理。データが外部に送信されずプライバシー安全
  • 生成されたブリーフィングと対話可能なチャットUIを搭載
長い動画を見る時間がない…
AIが要約してくれる！
しかも全部ローカルで動く。

⑫ 既存AI要約ツールとの決定的な違い

  • 単発のプロンプトではなく、製品としての一貫した体験を提供
  • トランスクリプトを保持し、継続的にチャット可能
  • グラウンデッドサマリー: ソース（原文）に基づく確実な要約を生成
  • 「この部分をもっと詳しく」など対話的に深掘りできる
  • デスクトップアプリとしての使い勝手を重視した設計
ChatGPTに動画投げるのとは
根本的に違う。
トランスクリプトを保持して
何度でも深掘りできる。

⑬ 長尺動画のリサーチワークフロー化

  • 長時間の講演・チュートリアル動画を効率的にリサーチ
  • 全体の要約を見てから興味ある部分を深掘りする流れ
  • チャットUIで自然な対話しながら情報を吸収
  • Tauri v2で軽量・高速。Electronの重さはない
  • 情報収集の新しいワークフローとして活用できる
2時間の講座動画も
5分で要約確認して
気になるところを深掘り。
時間が劇的に節約できる！

⑭ 技術スタック: Tauri v2 + Whisper

  • Tauri v2: RustベースでElectron比メモリ使用量80%削減
  • Whisper: OpenAIの高精度音声認識をローカルで実行
  • ローカルLLMで要約生成。外部APIにデータを送信しない
  • Windows・macOS・Linuxにクロスプラットフォーム対応
  • オープンソースでコミュニティ主導の継続的な改善
Electronより軽い、
Whisperで高精度、
全部ローカル。
三拍子揃った構成！

プロジェクト5: AgentMemory

AIコーディングエージェント向けの永続的メモリソリューション
rohitg00/agentmemory

⑮ AgentMemory とは

  • AIコーディングエージェント向けの永続的メモリソリューション
  • ローカル実行。外部データベースサーバー不要で手軽に導入
  • エージェントの記憶をセッション間で保持
  • Claude Code / Codex等のコーディングエージェントと統合可能
  • 実際のベンチマークに基づく最適化で実用的な性能を実現
AIエージェントに記憶を
あげるプロジェクト！
セッションを超えて記憶が残る。

⑯ 解決する問題: エージェントの記憶喪失

  • Claude Codeがセッションを超えると記憶をすべて失う
  • CLAUDE.mdへの手動記録では情報量が不十分で遅れる
  • 「昨日どこまで調べた？」「どのファイルを編集した？」が消える
  • 同じ調査や作業を繰り返す無駄が毎セッション発生
  • これが開発者の最大のペインポイントになっていた
「昨日のセッションで
何やってたっけ…？」
これがもう起きない！

⑰ Claude Code / Codex と統合

  • Claude Codeのセッションに自動でメモリを紐付けて記憶を保持
  • Codexとも統合可能。複数のエージェントで記憶を共有
  • ローカルストレージに保存。外部サーバー不要でセキュア
  • インストール後ゼロコンフィグで使い始められる設計
  • 2026年5月にGitHubトレンドで急上昇。開発者からの需要爆発
Claude Codeを使っているなら
マストインストール。
昨日の記憶が
今日も残る世界に。
