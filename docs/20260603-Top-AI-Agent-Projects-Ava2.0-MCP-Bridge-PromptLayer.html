② Ava Studio: AIクリエイティブワークスペース

  • プロジェクト1 デジタル体験の制作に特化したAIプラットフォーム
  • コンテンツ生成、ビジュアル制作、コラボレーションツールを1つの環境に統合
  • アイデアからプロダクションまでをシームレスに移行できるストリームラインドなワークフロー
  • クリエイター、マーケター、ビジネスユーザー向け。複数のツールを切り替える必要なし
  • AIアシスタンスを制作プロセスに直接組み込むことで効率を大幅に向上
制作の現場でツール切り替えの手間がゼロになるのは大きな時短になりますね。

③ Monitor by Firecrawl: AI駆動のWeb監視

  • プロジェクト2 ウェブページの変更を自動追跡する監視システム
  • 選択したウェブサイトを継続的にチェックし、構造化されたアップデートを提供
  • 重要な更新が発生した時にユーザーに通知。手動でページを再訪問する必要なし
  • AIエージェントや自動化ワークフロー、ビジネスプロセスで活用可能なデータ形式で出力
  • 競合調査、自動データ収集、リサーチ業務での利用が想定される
AIエージェントに「ウェブ上の変化を監視して」を直接指示できるのがポイントです。

④ Artisan Ava 2.0: AI営業担当の決定版

  • プロジェクト3 セールスチーム向けのAIビジネス開発担当（BDR）
  • 見込み客の特定、アウトリーチ、初期段階のセールスワークフローを自動化
  • リード調査、パーソナライズされたコミュニケーション、パイプライン管理をAI駆動で実行
  • セールスチームが手作業を増やすことなくアウトバウンド活動を増やせる設計
  • プロスペクティングとエンゲージメントに特化したデジタルチームメイトとして機能
手作業ゼロでリードを追いかけるAI営業担当。SaaSのセールスチームには必見ですね。

⑤ MCP Bridge by AppFactor: エージェントとシステムを接続

  • プロジェクト4 Model Context Protocol（MCP）を活用した統合プラットフォーム
  • AIエージェントと外部ツール、サービス、ビジネスシステムを構造化されたブリッジで接続
  • エージェントのワークフローと運用ソフトウェアの間の相互運用性を簡素化
  • 開発者がAIアプリケーションを構築する際、ビジネスデータやアクションへのアクセスが必要な場合に最適
  • エージェントエコシステム全体にわたる相互運用性をシンプルに
MCP（Model Context Protocol）がエージェント界の共通言語になっていく流れを感じます。

⑥ Firecoach AI: AIコーチングアシスタント

  • プロジェクト5 対話型コーチングプラットフォームで自己成長をサポート
  • 習慣改善、生産性向上、個人的な成長をAIガイド付きのやり取りで実現
  • 構造化されたコーチング会話、アカウンタビリティサポート、進捗トラッキングを統合
  • 従来のコーチング契約なしに継続的なガイダンスを受けたい個人向け
  • 自己改善をインタラクティブなAI支援プロセスに変換するプラットフォーム
24時間いつでも話せるコーチがAIで手軽に手に入る時代に。

⑦ Integrio: API統合の発見と自動化

  • プロジェクト6 ソフトウェア製品間の統合を発見・自動化する開発者プラットフォーム
  • アプリケーションの動作とAPIを分析して、統合の機会とワークフローの接続を明らかに
  • 開発者や技術チームが統合の調査と実装をより高速に行えるよう設計
  • サービス間の接続とデータフローの自動化に必要な作業量を削減
「あのAPIとあのサービスを繋げる方法」を自動で見つけてくれるのは便利です。

⑧ WhisperFlow: 音声ファーストの生産性アシスタント

  • プロジェクト7 音声をテキスト、コマンド、ワークフローアクションに変換
  • デスクトップアプリケーションを横断して音声入力でメッセージ作成やメモ作成、タスク制御が可能
  • タイピングの代わりに自然な音声入力で操作。プロ向けのハンズフリー生産性を実現
  • 音声をアプリケーションワークフローに直接接続し、日常業務の摩擦を削減
キーボードに手を置かずに仕事ができる。デスクトップAIアシスタントの新形態です。

⑨ Base Dash: 組み込みダッシュボード

  • プロジェクト8 自社アプリにインタラクティブなダッシュボードを直接組み込める
  • 既存のデータソースに接続して、埋め込み型の分析体験を提供。基盤の構築は不要
  • ダッシュボードが基盤となる情報と同期を保つため、常に最新の状態を表示
  • カスタマーフェイシングや社内プロダクトを構築するチームに最適
ゼロから分析画面を作らなくていいのは、プロダクトチームにとって大きな時短になります。

⑩ Modev: AI搭載開発者ワークスペース

  • プロジェクト9 コーディング、プロジェクト管理、AIアシスタンスを1環境に統合
  • タスク管理、コード執筆、作業の連携をシームレスなインターフェースで実行
  • コンテキストスイッチを削減。複数の開発活動を1つの場所に集約
  • ソフトウェアチーム向けに設計。統合されたエンジニアリングワークフローを求める開発者に
IDE、プロジェクト管理、AIチャットを行き来する手間が消える未来。

⑪ Framer: AIでウェブサイトをプロダクション

  • プロジェクト10 ビジュアルエディティングとプロンプトベース生成を融合したウェブ構築プラットフォーム
  • デザインツール、コンテンツ管理、ホスティングをブラウザ上で統合
  • 創業者、マーケター、クリエイター向け。大規模なコーディングなしでプロフェッショナルサイトを構築
  • AIアシストのワークフローでウェブ制作を加速
「プロンプトでサイトを作ってそのまま公開」が当たり前の時代へ。

⑫ PromptLayer: AIアプリケーションの可観測性

  • プロジェクト11 プロンプト、モデルの相互作用、アプリケーションの動作を監視・評価・管理
  • ロギング、分析、実験機能、AIシステム全体にわたるワークフロー可視化を提供
  • プロダクションAIアプリケーションで信頼性と継続的な改善が必要な開発者に最適
  • AIワークフローが時間を通じてどのように機能しているかをチームで理解できる
プロンプトの管理は実運用で悩む人が多い分野。可観測性は必須です。

⑬ Drafted: AIプロフェッショナルライティング

  • プロジェクト12 プロフェッショナルなコンテンツやアウトリーチ素材をAIで効率的に作成
  • 対話型のワークフローでドラフト生成、メッセージの洗練、コンテンツ作成を支援
  • プロフェッショナル、セールスチーム、ビジネスユーザー向け。定期的な文書作成の負担を軽減
  • 洗練されたコンテンツの作成に必要な作業量を大幅に削減
営業メールのドラフト作成だけでも週に何時間も節約できそうですね。

⑭ Pancake: AIカスタマーサポート

  • プロジェクト13 デジタルチャネルを横断して顧客との会話を管理するAIプラットフォーム
  • カスタマーサービス、エンゲージメント、ワークフロー自動化をAIメッセージングツールで支援
  • コミュニケーション活動を一元化し、チームのレスポンス効率を向上
  • 顧客とのやり取りが大量のビジネスにとって、自動化は必須の戦略
カスタマーサポートの自動化は、対応品質の向上とコスト削減の両立に繋がります。

⑮ Pitch Agent: AIプレゼン生成アシスタント

  • プロジェクト14 対話型ワークフローでスライドデッキの作成・構成・洗練を支援
  • コンテンツの生成、アイデアの整理、プレゼンデザインをPitchプラットフォーム内で直接実行
  • 創業者、チーム、プロフェッショナル向け。短時間で高品質なプレゼンが必要な場合に
  • デッキの手作業での構築に費やす時間を大幅に削減
「10分で投資家向けデッキを作って」が現実になる未来。

⑯ Revolt: AIセールス＆顧客エンゲージメント

  • プロジェクト15 セールス会話、顧客エンゲージメント、コミュニケーションワークフローを自動化
  • リードとのやり取り、アウトリーチ活動、リレーションシップ管理をAI駆動で実行
  • 手作業を増やさずにスケーラブルな顧客エンゲージメントを実現したいチーム向け
  • コミュニケーションワークフローを簡素化しつつレスポンス性を維持

⑰ Memory（Memori）: AIアプリの記憶インフラ

  • プロジェクト17 AIアプリケーションとエージェントが会話やワークフローをまたいで文脈を保持
  • 関連情報を保存・整理・検索し、AIシステムが時間を通じて連続性を維持できるように
  • アシスタント、エージェント、長期間動作するAI体験を構築する開発者向け
  • セッション間のコンテキスト保持というAIの大きな課題にアプローチ
AIの記憶問題はエージェントの実用化を左右する重要課題。Memoriがその答えの一つです。

  • 情報アクセス、コラボレーション、AIアシスタンスを組み合わせてビジネスプロセスの摩擦を削減
  • チーム間の可視性と実行力を高める必要がある組織向け
  • 散在する運用知識を実行可能なワークフローに変換するプラットフォーム
企業の「暗黙知」をAIで構造化する。ナレッジマネジメントの進化形。
