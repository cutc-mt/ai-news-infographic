⑩VRAM要件: 量子化でさらに軽量化

  • 8ビット量子化で7GBのVRAMで動作可能
  • 16ビット（デフォルト）では24GBのVRAMが必要
  • 実稼働時はモデル重み＋KVキャッシュで追加メモリを消費
  • Macの統合メモリーなら32GBモデルで余裕で動く
量子化は量子コンピューターとは一切関係ないよ！小数点の桁数を減らすだけ！

⑪量子化とは: 小数点の桁数でサイズが変わる

  • 量子化＝モデルの重み（小数点）を丸めて容量を減らす技術
  • 16ビット → 8ビット → 4ビットと減らすほどサイズが小さくなる
  • ただし精度は下がる。サイズと精度のトレードオフが存在する
  • 4ビット量子化なら低スペックPCでも試せる程度に軽くなる
精度は少し落ちるけど、小さいVRAMでも動くから初心者にはおすすめ！

⑫KVキャッシュ: 文章生成の効率的な記憶

  • KVキャッシュは過去に生成したトークンを記憶する仕組み
  • キャッシュがなければ毎回最初から全トークンを再計算する必要がある
  • キャッシュを使うと非常に効率的に文章を生成できる
  • ただし会話が長くなるほどメモリを消費するので注意が必要
前の計算結果を覚えておくから、無駄な再計算がなくなる！超重要！

  • アテンションは「どの言葉に注目すべきか」を自動的に判断する仕組み
  • 言葉をベクトルに変換し、内積の計算で意味的な近さを測る
  • 何に注目すべきかは学習によって獲得された重みで決まる
  • 言葉同士の関係性が数値化され、最終的に次のトークンが決定される
アテンションを知らないで「LLMを知ってる」はちょっとまずいかも！詳しくは書籍で！
