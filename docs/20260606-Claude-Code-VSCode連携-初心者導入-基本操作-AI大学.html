ターミナル vs VS Code


ターミナル
⚫ 黒い画面
📝 テキストのみ
😰 初心者には不親切

VS Code
🎨 グラフィカル
📂 フォルダ構成可視化
😊 直感的に操作

利点①：ファイルの中身が見える

Claude Codeが生成したファイルの中身を視覚的に確認できる！コードもプレビューも一画面でOK👀
ターミナルだけだと `cat` コマンドでしか見られない…VS Codeならカラーシンタックスハイライト付きで表示

利点②：複数作業を並行できる

複数のClaude Codeを同時に立ち上げて、別々の作業を並行して進められる！⚡
VS Codeのタブや分割画面機能で、複数プロジェクトを同時に管理

利点③：フォルダ構成を確認

パソコン内のフォルダ構成をツリー表示で確認できる！どこに何があるか一目瞭然🌳
Claude Codeが作ったファイルがどこに保存されたかすぐ分かる

利点④：拡張機能でカスタマイズ

VS Codeの拡張機能で自分好みにカスタマイズできる！AI開発にも役立つ拡張がたくさん🔧
Claude Codeの設定ファイル（.claude/）もVS Code上で直接編集可能

① VS Codeのインストール

  1. 公式サイトにアクセス
    https://code.visualstudio.com/
  2. OSに合ったバージョンを選択
  3. ダウンロードしたファイルを実行
  4. インストールウィザードに従って完了
完全無料！ Microsoftアカウント不要で使える

② VS Codeを日本語化

  1. VS Codeを起動
  2. 左サイドバーの拡張機能アイコン（四角形）をクリック
  3. 検索窓に「Japanese」と入力
  4. 「Japanese Language Pack」をインストール
  5. VS Codeを再起動 → 日本語表示に！🎉

③ VS Codeの画面構成

画面は4つのエリアで構成されてるよ！覚えると操作がスムーズになる✨
① エクスプローラー — ファイル・フォルダ一覧
② エディター — ファイルを編集する場所
③ ターミナル — コマンドを実行
④ サイドバー — 検索・拡張機能など

④ VS CodeでClaude Codeを起動

  1. VS Codeの統合ターミナルを開く
    Ctrl + `（Mac: Cmd + `）
  2. ターミナルに claude と入力してEnter
  3. Claude Codeが起動！Claude Codeの操作はこのターミナル内で行う
  4. エディターでファイルの確認・編集も並行して可能！

Claude Code × VS Codeの活用術

Claude Codeに「コードを書いて」と指示した後、生成されたファイルをVS Codeで確認するのがおすすめ workflow！💡
Claude Code（ターミナル）で生成 → VS Code（エディター）で確認・修正 の往復作業が効率的

Claude Code 基本コマンド

claude — Claude Codeを起動
claude "プロンプト" — 直接プロンプトを実行
/exit — Claude Codeを終了
/help — ヘルプを表示
