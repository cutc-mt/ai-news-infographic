Gemma 4 12B + Hermes Agent

無料・ローカル・プライベート — AIエージェントがあなたのPCで自律稼働する時代が来た
2026年6月6日

① Gemma 4 12Bの衝撃 — ラップトップで動く最先端AI

  • Google DeepMindがApache 2.0で公開したオープンソースの多モーダルモデル。誰でも自由に利用・改変が可能
  • 従来の2倍サイズのモデルに迫る性能を持ちながら、メモリ消費は半分以下に抑えている
  • テキスト・画像・音声を1つの統合アーキテクチャで直接処理。重いエンコーダーなしでネイティブに理解
  • 16GBのVRAMまたは統合メモリがあれば動作可能。一般的なノートPCでも最先端のAI体験が手に入る
エンコーダーフリーの統合設計は、 モデルを軽くしつつ精度を保つ 革命的なアプローチです！

② エンコーダーフリー統合設計 — なぜこれが重要か

  • 従来は視覚エンコーダーと音声エンコーダーが別々に存在し、それぞれがLLMの本体と接続されていた
  • Gemma 4はLLM本体に直接入力する方式を採用。中間層を省くことで推論レイテンシが大幅に削減された
  • 音声の声の高さ・感情・話者などのニュアンスをテキスト変換なしでそのまま保持できるのが大きな利点
  • 構造がシンプルになるためモデルの総メモリフットプリントが小さくなり、エッジデバイスでの展開が現実に
音声を文字起こしせずに 直接理解できるのは、 ブレイクスルー的な設計です。

③ Hermes Agent — あなたと共に成長するAIエージェント

  • Nous Researchが開発したオープンソースの自律型AIアシスタント。2026年2月にリリース
  • 自己学習機能を搭載。実行したタスクからスキルを自動生成・精製して記憶し、使うほど賢くなる
  • 永続メモリで過去のやり取りや設定をセッションをまたいで保持。意図・トピック・履歴で検索可能
  • 20以上のプラットフォームに接続可能。TelegramやDiscord、Slackなど1つのエージェントで複数チャネル対応
「The Agent That Grows With You」 — 使うほど賢くなる、まさに パーソナルAIの理想形！

④ 無料・ローカル・完全プライベート — 3つの約束

  • 完全無料：Gemma 4はオープンソース、Hermes Agentもオープンソース。APIキー不要で月額料金ゼロ
  • ローカル実行：すべてのデータがユーザーのPC上で処理される。クラウドに送信されない
  • プライバシー保護：テレメトリや外部追跡がない。機密データをAIに渡しても安心
  • オフライン対応：インターネット接続なしでも動作する。外出先やフライト中でもAIが使える
API費用ゼロでGPT-4クラスの エージェント体験ができる — これがローカルAIの強み

⑤ Hermes Agentの実力 — チャットを超えた自律行動

  • ファイル管理：フォルダの作成・整理・検索を自律的に実行。PC上のファイルを勝手に整頓してくれる
  • ブラウザ操作：ウェブサイトの自動巡回、情報収集、フォーム入力などをエージェントに任せ可能
  • スケジューラ機能：cron形式の定期タスクを設定して24時間365日自律稼働させることができる
  • サブエージェント：複雑なタスクを自動的に小さなサブタスクに分割し、専門のサブエージェントに振り分ける
単に「お話しする」だけでなく、 「行動する」AI。 これがエージェントの本質です。

⑥ Hermesのマルチプラットフォーム接続

  • Telegram：ボットを作成してHermesと対話。BotFatherでトークンを取得して設定するだけ
  • Discord：サーバーにボットを追加してチーム全体でエージェントを共有可能
  • Slack：ワークスペース内でエージェントを活用。業務の自動化に役立つ
  • WhatsAppやEmailにも対応。1つのエージェントで全チャネル統一
どこからアクセスしても 同じエージェントが応答。 デバイスをまたいで一貫した体験が特徴です。

⑦ セットアップ手順 — 10分で始められる

  • Step 1：Hermes Agentをcurlコマンドでインストール。Linux、macOS、WSL2に対応
  • Step 2：Ollamaをインストール。ローカルLLMを管理するためのツール
  • Step 3：ollama pull gemma4:12bでGemma 4 12Bモデルをダウンロード
  • Step 4：hermes modelでOllamaのローカルAPIエンドポイントをカスタムプロバイダーとして登録
hermes model
# Provider: Custom (Ollama)
# URL: http://127.0.0.1:11434/v1
# Model: gemma4:12b
新しくWeb UIも追加され、 ブラウザから簡単に設定・管理が できるようになりました！

⑧ Ollama連携とモデル選択肢

  • Ollamaが提供するOpenAI互換APIにより、既存のツールや拡張機能がそのままGemma 4で動く
  • Gemma 4シリーズには12B / 26B / 31Bのサイズがある。ハードウェアに応じて選択可能
  • ローカルPCのスペックが足りない場合は、OpenRouterの無料APIで26B/31Bモデルにアクセスできる
  • メインモデルに強力なものを置き、Gemma 4をサブエージェントとして活用すればトークン消費を最適化可能
Hermes Agentは最低64Kトークンの コンテキストを必要とします。 モデル選びは重要です。
