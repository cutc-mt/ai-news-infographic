  • 24時間365日あなたのコンピューター上で稼働し、自律的にタスクを遂行する
  • OpenClawのように動くが、長期記憶・スキルの再利用・ユーザー理解がより優秀

③ 24/7継続稼働: 自律進化するAIシステム

  • ブラウザタブのように閉じられるものではなく、バックグラウンドで永続的にプロセスを実行
  • 稼働時間が長くなるほど自動的に能力が向上する自己改善の仕組みを搭載
  • 定期的なセルフナッジ（自己調整）と永続的なナレッジベースで継続的に賢くなる
寝ている間にも勝手に賢くなる。

  • ユーザーが再トレーニングしなくても、経験から学習して自律的に振る舞いを調整
  • 成功したタスクから再利用可能なスキルを自動生成・洗練させるスキルレイヤー
  • 「幅広さ」ではなく「深さ」と経験からの学習にフォーカスした設計思想
使えば使うほど賢くなる。

パフォーマンス向上につながります。

  • 永続的クロスセッションメモリで、セッションをまたいで情報を保持。毎回リセットされない

  • 時間をかけてユーザーのより深いモデルを構築し、パーソナライズされた応答を生成
  • 毎回の会話ではなく、継続的な関係性としてユーザーとの理解を深めていく設計
チャットボットとは違う。
あなたの「第二の脳」のように
記憶と理解を蓄積していきます。

⑨ インストール手順: GitHubから簡単セットアップ

  • 手順1 GitHubリポジトリのリリースページにアクセス
  • 手順2 OSに合ったインストーラーをダウンロード。macOSは.dmg、Windowsは.exe
  • 手順3 インストーラーを実行。ローカルに2GBの空き容量が必要
  • 手順4 「Get Started」をクリックしてHermesエージェントをローカルにインストール
# macOSの場合
ダウンロード → .dmgを開く → アプリにドラッグ

# Windowsの場合
ダウンロード → .exeを実行 → インストールウィザード
たった4ステップで完了。
プログラミング不要で始められます。

  • マルチエージェントオーケストレーションで、各エージェントが異なる役割を担当
  • 用途例: 1つはブログ執筆用、もう1つはコーディング用など専門分化が可能

⑮ ツールAPI統合: Firecrawl, Exa AI, Falなど

  • Fal API — 画像生成をエージェントに統合。クリエイティブタスクを自動化
  • Firecrawl — Webスクレイピングと検索。ウェブ上のデータを自律的に収集
  • Exa AI — 高精度なセマンティック検索で情報を効率的に発見
  • ツールAPIを早めに追加することが推奨。エージェントの能力が大幅に拡張される

⑯ スキル・ペルソナ・メモリ: エージェントをカスタマイズ

  • スキル — エージェントの能力とワークフローを拡張。タスクごとに最適なスキルを設定
  • ペルソナ — エージェントの応答スタイルを定義。トーンや指示をカスタマイズ可能
  • メモリ — エージェントのナレッジベースを管理。継続的な記憶をコントロール
  • ツール — Web検索、ブラウザ操作、ターミナル、ファイル操作など多数のツールを管理
エージェントの「性格」も
「能力」も自由に設計可能。

⑰ Cronジョブとゲートウェイ: スケジュール実行＆外部連携

  • Cronジョブでスケジュールタスクを設定。Claude CodeやCodexのように定期実行が可能
  • Telegram Discord iMessageへのゲートウェイ連携。スマホからも操作可能
