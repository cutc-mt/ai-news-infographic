⑦ 「掘り下げ」スキル: 不明点を自動解消

  • スキルとは、トリガーキーワードが入力された時に動きを覚えさせる仕組み
  • 「掘り下げ」は計画や設定を徹底的にインタビューで掘り下げるスキル
  • ぼんやりとしたイメージをClaude Codeが質問を繰り返して細かく具体化してくれる
  • 一度登録すれば毎回同じ動作をしてくれるので、毎回の指示が不要になる
プロンプトを正確に書くのは

代わりに質問してくれます！

⑧ スキル登録のステップ

  • 1カスタマイズ → スキルを選択
  • 2「＋」を押してスキルを作成
  • 3「スキルの指示を記述」を選択して内容を入力
  • 4名前を「掘り下げ」にして、ルールを記述して保存
  • 掘り下げのルール例: 1度に1つだけ質問 / 安易に完了しない / 送信前のチェック
  • スキルは使いこなすほど便利。会社の資料トーンや色指定も登録可能

⑩ 掘り下げ実践: 質問で精度を上げる

  • 「/掘り下げ」と入力してから「格納しているデータをもとにパワポ資料を作成したい」と送信

  • 選択するとさらに掘り下げて、「どの論点を意思決定パートに載せますか？」と続く
  • 自分では気づかなかった矛盾点や不明点もClaude Codeが見つけてくれる

⑪ プラン完成: 計画を確認して承認

  • 掘り下げ完了後、「全分岐点を解いてプランに落としました」と表示される
  • 右側にプランが表示される。「なぜ作るのか」「指定した方針」などが含まれている
  • 内容を確認してOKなら「承認」ボタンを押して自動モードに切り替え
  • 承認後はClaude Codeが自動で資料の作成に取りかかる。あとは待つだけ
