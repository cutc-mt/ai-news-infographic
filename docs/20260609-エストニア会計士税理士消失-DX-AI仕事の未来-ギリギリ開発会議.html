DXとAI時代の仕事の消滅と再定義

業界では「エストニアでは税理士・会計士が完全に消滅した」とよく言われている。日本のメディアでもIT化が駆逐した事例として引用される。

  • IT化が職業を駆逐した事例として日本メディアでも頻繁に引用
  • AI時代の「不要論」を裏付ける根拠として参照されることが増加

なぜ従来業務が消えたのか

④理由1: 税制を極めてシンプルにした

エストニアは税制を極端に単純化して、解釈の余地をほぼゼロにした。複雑な税制こそ税理士の市場だが、それを根本から変えた。

  • 節税アドバイスという仕事が制上で不要になった
  • 解釈論を圧縮し、税制を極めてシンプルに変更
  • 節税スキームの提案余地が完全に消滅
複雑な税制＝税理士の市場。シンプルな税制＝税理士不要。国の政策が職業を変えた！

⑤理由2: 国家データ連携基盤「X-Road」

エストニア政府が構築した国家データ連携基盤「X-Road」が申告書作成を不要にした。

  • 給与データ
  • 銀行口座情報
  • 保険・年金データ
  • 投資収益情報
  • 住宅・寄付データ

これら全てが一元化・連携されている。ログインして3分で完了する。

確定申告は「自動入力済みの内容を承認するだけ」の3分で終わる世界！

⑥理由3: SaaS会計プラットフォームの爆発的成長

企業が会計事務所ではなく、SaaS会計プラットフォームに移行。月1枚程度の料金で便利に使えるようになった。

  • 会計事務所からITサービスへの移行が加速
  • 月額制で低コストに。価格破壊が発生
  • 事務所に頼まなくても完結するようになった
SaaSの普及で会計事務所は不要に。価格破壊が従来のビジネスモデルを崩壊させた。
消えた3つの柱

⑦消えた仕事: 制度・技術・市場の3方向から同時に消失

従来の税理士の3大福（主要業務）が制度・技術・市場の3方向から同時に失われた。

  • 制度 シンプルな税制で節税アドバイスが不要に
  • 技術 X-Roadで申告書作成が3分のボタン押すだけに
  • 市場 SaaSプラットフォームが会計事務所を代替
消えた業務 個人申告・帳簿代行・仕訳・申告書作成・節税スキームの提案
残った価値 経営判断の助言・データ分析・IT導入支援

⑧99%がオンライン完結: 個人申告の革命

エストニアでは99%の個人所得税申告がオンラインで完結している。もはや税理士に頼る必要がない。

  • 99%の個人申告がオンライン完結
  • 申告は3分で終わる（ボタンを押すだけ）
  • 帳簿代行の需要が成立しなくなった
  • 節税スキーム提案の余地が消滅
日本ではまだ確定申告前に「申告書お願いします」と税理士にお金を払う文化があるけど、エストニアではもうそんな世界じゃない。

新しく生まれた仕事

⑩新需要1: 経営判断・投資判断のアドバイザー

M&Aや投資が活性化する中、経営者には数字を読み解けるアドバイザーのニーズが急増している。

  • M&Aによる事業買収の助言
  • 投資先の選定とリスク評価
  • 内部留保の再投資戦略
  • 新規事業参入や撤退の経営判断
ビジネスのドメイン知識はあっても、投資判断は専門家じゃないとわからない。その需要が爆増中。

⑪新需要2: IT導入支援アーキテクチャー

SaaSが便利になった一方で、プラットフォームだけでは解決できない領域が存在する。ここに新たな需要が生まれた。

  • 業務系システムとのAPI連携
  • データ可視化の相談事業
  • SaaSの適切な導入・設定方法
  • プラットフォームにないデータの活用方法
SaaSの「導入方法」や「API連携」が分からない企業が多く、ここに会計士の専門性が活きる！

⑫爆発的に増えた領域: ITコンサルへの進化

SaaS導入後のアーキテクチャー構築が爆発的に増加。実質的にITコンサルタントへと進化している。

  • クラウド会計の適切な設定
  • API連携の設計と実装支援
  • データ可視化・分析のコンサルティング
「これはもう会計士の仕事なのか？」と思うほど、ITコンサルに近い立ち位置になっている。
年収の劇的変化

⑬適応した人は年収3〜10倍: 1,400万〜2,300万円へ

変化に対応した税理士は年収が3倍から10倍に跳ね上がった。戦略アドバイザーやCFO的ポジションに就いている。

  • 戦略アドバイザーとして投資判断を支援
  • CFO的な立ち位置で経営に参画
  • 年収は1,400万〜2,300万円レベルに到達
Claudeの会計スキルを携えて各企業に入り込み、データを可視化して経営者と会議する。今まさに起きている光景！

⑭IT導入支援は年収1.5〜2倍: コンサル型への転身

IT導入支援に進んだ会計士は年収が1.5倍から2倍に増加。SaaSの相談役として新しい価値を提供している。

  • クラウド会計の設定・導入支援
  • API連携の設計とデータ活用
  • 企業のSaaS相談役として定着
実際には1.5〜2倍では済んでいない可能性も。AIツールを活用すればさらに上振れするはず。

⑮淘汰された人たち: 変化に対応できなかった末路

一方で、変化に適応できなかった人たちも多数存在する。定型業務がなくなり、道を閉ざされた。

  • 従来業務がゼロになり廃業した人
  • 細々と残る代行業務で月49万円程度で推移
  • 50代・60代のベテランほど対応が難しい
上と下に二極化。変化を受け入れた人は勝ち、受け入れられなかった人は淘汰された。
日本への示唆

⑯日本にも来る: ただし税制の複雑さが足かせに

日本でも同じ流れは来るが、日本の税制は非常に複雑なので急にはシンプルにならない。

  • エストニアは税制・IT化・DX基盤が一気に整った特殊ケース
  • 日本の税制は複雑で「旨みの節税」がまだ存在する
  • でも方向性は同じ。遅かれ早かれ変化は来る
エストニアは国ぐるみのDXが成功した稀有な例。日本は段階的に進むが、流れは同じ。

⑰日本の前例: 電子カルテ化で廃業が続出

日本でも電子カルテ化の際に病院や歯科医院の廃業がバッと出たという前例がある。

  • 電子カルテの導入でついていけない医療機関が廃業
  • 設備投資も必要で、補助金があっても厳しい
  • 「もういい年だし、無理だし」と撤退するベテランも
歯医者さんから直接聞いた話。「設備投資もいるし、もうこのタイミングかな」と廃業する人が続出。

⑱教訓: 変化を受け入れた者が勝つ時代

適応した専門職ほど報酬も役割も大きく伸びた。これは我々開発者にも他人事ではない。

  • AIによってコモディティ化された業務は消えていく
  • 上流のビジネス領域にシフトすることが鍵
  • 「AIに対してどうしたらいいか」を求められる時代に
  • 止まるつもりはない。変化に乗る人が勝つ
開発者も同じ。「開発」だけでなく、AI導入のコンサルやビジネス上流の支援が求められている。
