① 封印されていた神話 — Claude Mythosがついに一般公開へ

  • 2026年6月9日 AnthropicがClaude Mythosの一般向け版「Claude Fable 5」を正式リリース。招待制の「Mythos Preview」から約2ヶ月で解禁
  • 2026年4月7日にProject Glasswingとして発表。サイバー攻撃能力が強すぎて40社以上の限定パートナーのみに提供されていた
  • Mythosは27年間見過ごされていたOpenBSDのバグや、16年間放置のFFmpeg脆弱性を次々と発見した伝説のモデル
  • 主要OSと主要ブラウザすべてでゼロデイ脆弱性を実証。4つの脆弱性を連鎖させてサンドボックスを突破した実績を持つ

② Fable 5 と Mythos 5 — 同じ中身でセーフガードだけが違う

  • Claude Fable 5: セーフガードを組み込んだ一般向け製品版。誰でも利用可能
  • Claude Mythos 5: 一部のサイバーセキュリティ関連セーフガードを解除した限定版。審査を通ったパートナーのみに提供
  • 両モデルは全く同じ基盤モデル。性能差はセキュリティ制限の有無だけ
  • 名前の由来: Fable はラテン語の fabula（語られるもの）。Mythos はギリシャ語で「神話」。どちらも近い意味を持つ
Haiku（俳句）→ Sonnet（14行詩）→
Opus（大作）→Fable/Mythos（神話）

  • 2026年4月に「Claude Mythos Preview」が初登場。サイバー防御の専門家に限定提供されていた
  • Project GlasswingのパートナーにはAWS、Apple、Google、Microsoft、NVIDIA、CrowdStrikeなど40社以上

⑨ セーフガードの仕組み — リスク検出でOpus 4.8にフォールバック

  • 「リスクの高いリクエストを検出する分類器」を内蔵。対象はサイバーセキュリティ・生物学・化学の3領域
  • 検出された場合はClaude Opus 4.8にフォールバック。拒否ではなく安全なモデルに切り替える設計
  • フォールバックは平均でセッションの5%未満で発生。大部分のリクエストはFable 5がそのまま処理
  • 無害なリクエストが誤検知されるケースもあり。制限は厳しめに調整されている
