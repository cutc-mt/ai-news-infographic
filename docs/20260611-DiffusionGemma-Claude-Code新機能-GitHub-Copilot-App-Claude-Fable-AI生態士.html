  • 量子化すれば18GB VRAMに収まる。消費者GPUでもデプロイ可能

③ DiffusionGemmaの生成仕組みとは？

  • 従来のLLMは1トークンずつ順次生成（オートレグレッシブ）
  • DiffusionGemmaはブロック単位で並列処理。ノイズ除去を反復してテキストを洗練
  • 同等のオートレグレッシブモデル比で最大4倍高速の生成速度
  • インライン編集や素早い反復作業に特に適している
