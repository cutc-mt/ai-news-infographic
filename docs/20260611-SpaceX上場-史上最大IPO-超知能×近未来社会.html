SpaceX 上場 — 史上最大の IPO

Bioshok さんに聞く — 超知能×近未来社会 勉強会 第120回
2026年6月10日収録 / 6月12日IPO予定

① 史上最大級の IPO: SpaceX が6月12日に上場

  • 1株あたり約160ドル（約22,000円）。時価総額は1.75兆ドル（約300兆円）に達する見込み
  • 世界最高の時価総額の Apple（500〜600兆円）の半分近く。いきなり世界ベスト5クラスに躍り出る
  • 個人投資家向けに30%を割り当て（通常は20%）。需要の4倍超となっている
  • 個人投資家の割り当てが多いのは、熱狂による株価上昇を狙っているという見方も存在
株に興味がない素人が「申し込んだ」と投稿するレベルの熱狂。バブルの兆候か、それとも上昇の勢いか。

② 売上100倍の謎: イーロン・マスクプレミアム

  • 年間売上はたった186.7億ドル。時価総額が売上の約100倍という異常なバリュエーション
  • xAI を買収したことで約4兆円の累積赤字を抱える会社になっている
  • 通常の株価形成では説明がつかず、市場は「イーロン・マスクプレミアム」と呼んでいる
  • 評価の内訳：ロケット＋Starlink＋宇宙AIコンピュート＋マスクプレミアム
売上100倍は異常すぎる。AI 系のインフレ期待が根底にあるのかもしれません。

③ バブルか上昇続行か: 機関投資家の見方

  • 「靴磨きの少年」が株式を語り始めたらバブル崩壊の兆候——そのレベルに近づいている
  • 楽観派：素人が熱狂している＝注目度が高い＝まだまだ上がるというロジック
  • アメリカの機関投資家の多数派は上昇継続を予想。大衆の熱狂に乗る方が得という判断
  • 悲観派：素直にバブル崩壊の直前だと見る考え方も根強く存在している
Bioshok さんも「最初はバブるだろう」と言いつつ少し申し込んだそう。お祭り感覚での参加が多い。

④ 宇宙データセンター構想: 次世代ハイパースケーラー

  • 人工衛星にソーラーパネルを取り付けデータセンターを構築。太陽光で24時間発電可能
  • 宇宙空間に熱を放出すれば冷却OK。地球上の冷却問題が一切ない
  • 5年以内のビジネス化は困難だが、2030年代には地上データセンターと競争力を持つ可能性
  • AGI 登場で計算資源需要が爆発すれば、SpaceX は宇宙時代のハイパースケーラーになるかも
CoreWeave のようなハイパースケーラーが宇宙でSpaceX になるという構図。

⑤ 宇宙経済圏は実在するか: スケールで話は別次元

  • Bioshok さん：「実在する」。ただし規模によって段階が全く違う
  • 短期的：Starlink や宇宙データセンターなど、現実的なビジネス
  • 長期的：地球経済の何百倍という宇宙文明レベル。200年以上先の話
  • 普通の投資家は5〜10年で元を取りたい。200年先の宇宙経済圏を考えて買う人はほとんどいない
Starlink の話と何百倍の宇宙経済圏の話は全く別次元の議論です！

⑥ 産業爆発: AGI が前提を変える超指数関数的成長

  • 産業爆発：数百年先の経済規模が、AGI 後10〜20年で実現するという概念
  • 工場が工場を作るフィードバックループが回れば、地球経済はゴミのようなレベルに縮小
  • 火星に植民地、ラグランジュ点にマクドナルド・シリンダー級のスペースコロニーが何千個も
  • カルダシェフスケール文明（タイプ2）への急速な移行が現実的な議論として成立しつつある
マスクがよく言っている「タイプ2文明」の話は夢物語ではないという見方。

⑦ Anthropic が証明した指数関数的成長の現実

  • Anthropic の売上：2023年ゼロ → 2024年数億円 → 2025年5000億円 → 2026年4兆円の予測
  • Mythos リリース時の売上予測グラフが指数関数の曲線を描いていた
  • ダリオ・アモが売上ゼロの時点で「数年後に1兆円」と予測し、それが現実になりつつある
  • 経済史上初めて指数関数的成長が実証された。これがAGI後の宇宙経済圏のロジックに繋がる
この成長が証明されたタイミングでSpaceX も「自分も実現する」を主張するストーリー。

⑧ エネルギーを制する者が宇宙を制す

  • 太陽系で最大のエネルギー源は太陽。地球の核の何万倍ものエネルギーを持つ天然の核融合炉
  • Bioshok さん：「エネルギーを握ったものが全てを握る」。宇宙の覇者になるための鍵はエネルギー
  • SpaceX は太陽エネルギーの活用において1〜2歩リード。24時間発電できる宇宙空間の有利さ
  • ただし宇宙の覇者になるにはAGI が必要。SpaceX 単体では勝てないという条件付き
太陽エネルギー＋AGI＝宇宙文明の覇者。SpaceX はその条件の一部。

⑨ Fast Takeoff vs Slow Takeoff: SpaceXの命運

  • Fast Takeoff：AGI が急速に自己改善し、3年で宇宙開発の知見を追い抜く世界
  • Slow Takeoff：AGI の進化が10〜20年のスパン。SpaceX の20年のノウハウが生きる
  • Fast Takeoff では超知能が第1原理計算で全て解決。20年の試行錯誤が無意味化する恐れ
  • Slow ならノウハウが10年以上有効。Fast なら3年でアドバンテージ消失
Fast Takeoff を信じる人はSpaceX の株を買わない方がいい。Slow なら微妙に買ってもOK。

⑩ Fast Takeoff で覇権を失うシナリオ

  • 超知能が人間の20年間の打ち上げノウハウを3日で再現する可能性。蓄積が一瞬で無意味に
  • SpaceX が宇宙覇者である可能性は徐々に下がっていく。シェアがゼロにはならないが差は縮まる
  • 3番手4番手の大手サプライヤーに落ちる可能性。Anthropic、OpenAI、Google も宇宙覇者に
  • AGI企業が計算資源の調達も自律的に解決すれば、外部ローンチの優位性は薄れる
xAI が遅れている以上、AI 面での出遅れが致命的な弱点に。

⑪ マスクの戦略敗北: xAI 合体の失敗と Compute 貸与

  • 当初の構想：宇宙のパイオニア SpaceX＋AI のトップ xAIが合体して宇宙とAIを制覇
  • 現実：SpaceX がGoogle に計算リソースを貸すインフラになっている。つまり余っている
  • xAI から創設者が何人か抜け、人材流出も発生。AI 競争で半分負けを認めている状態
  • Bioshok さん：Fast Takeoff シナリオなら負けを認めざるを得ない。Slow なら巻き返せるかも
宇宙＋AI の夢の構想が霧散しつつある。しかしマスクは事業家として交渉の余地も残している。

⑫ Cursor 買収権: データでAI競争に差をつける狙い

  • SpaceX が約1兆円を支払い、Cursor（旧 Anysphere）の企業買収権を取得。4〜5兆円で完全買収可能
  • Bioshok さん：Cursor を経由して何百万人が開発。良質なコーディングデータが大量に蓄積
  • AI 競争で重要なのはデータ＋人材＋学習環境を構築できる人材
  • AI研究はソフトウェア開発に似ており、スーパーコーダーの開発が次のAI研究の自動化に繋がる
Claude Code や Codex の後継になる可能性。コーディング環境のデータはAI開発の競争力そのもの。

⑬ マスクビジョンの空転: 株価が下がらない理由

  • xAI が性能面で勝てていない。宇宙＋AI の両面リーディングの物語が消化しつつある
  • それでも株価が高いのは「イーロン・マスクプレミアム」の存在。Bioshok さんも大いに存在すると認識
  • 多くの投資家は Starlink や宇宙データセンターにワクワク感を抱いて買いに来ている
  • 期待に乗って参加する人が集まり、「お祭りに参加しよう」というムードも拡大中
産業爆発を知っている人は少ない。株価を支えているのはStarlink への期待が大半。

⑭ 協力シナリオ: SpaceX × Anthropic × Google

  • Fast Takeoff でも SpaceX が完全に終わるわけではない。マスクが Anthropic や Google と協力する可能性も
  • 事業家としてのマスクが良い形で交渉をすれば、株価が大きく上昇するシナリオもあり得る
  • Bioshok さん：Anthropic や OpenAI、Google もいずれ宇宙に行く。彼らの株を買うことは宇宙文明のインフラを買うのと同じ
  • ただし短期的に見ればAnthropic や Google を買っておいた方が確実というのが Bioshok さんの見方
SpaceX の株価は短期で爆発的に上がる可能性もある。協力シナリオ次第。

⑮ Anthropic vs SpaceX: 投資順序が鍵

  • AI企業は世界GDP（100兆ドル）の0.1%の1000億ドルしかまだ稼げていない。まずは地球経済を制覇する段階
  • 地球経済を2倍にするだけで十分な投資余地がある。まだ宇宙経済圏の話をする段階ではない
  • 理想的なポートフォリオ：Anthropic 30%＋OpenAI 30%＋Google 30%＋SpaceX 5〜10%
  • Anthropic や OpenAI が100兆ドル企業になってから、改めて SpaceX のシェアを増やせばいい
投資には順序が大事。「今上がる株」と「将来上がる株」は別です。

⑯ Claude Fable の実力: AGI まだ遠いが衝撃的

  • かさん：Fable は明らかに Claude 4.7や4.8より正確。コーデックスよりも全然上という評価
  • 指示を与えれば何日も勝手に動き続ける。30分→1時間→3時間と自律的に稼働
  • Bioshok さん：Fable は Mythos とほぼ同じ能力だが、「まだ全然 AGI ではない」
  • 厳密なAGI の定義：アインシュタインレベルの科学的発見をなし、自己フィードバックループで人間が介入不要
Fable がAGI ではないと言うのはAGI があまりにもすごすぎるから。Fable 自体は相当すごい。

⑰ Bioshok さんの AGI 定義: カルダシェフスケール以上

  • AGI とは地球上の宇宙科学を全て発展させ、宇宙文明への移行を担えるレベルの能力
  • 単にコーディングできるだけでは不十分。新しい科学的発見やパラダイムの発見が必要
  • 無限に動ける必要がある。この瞬間に銀河系を制覇できない＝AGI ではない
  • カルダシェフスケールのタイプ1〜2文明への移行まで担えるなら初めて AGI と呼べる
Bioshok さんのAGI 定義は一般的な認識より遥かに高い。だからFable がすごくても「まだAGIではない」と言う。

⑱ まとめ: 超知能の観点から見た SpaceX 上場

  • SpaceX の上場は宇宙ビジネスの本格参入の歴史的イベントだが、バリュエーションは割高感あり
  • Fast Takeoff では SpaceX の20年のノウハウが3日で無意味化する可能性。AI競争の出遅れが致命傷
  • 短期的に確実なのはAnthropic や OpenAI。長期なら SpaceX も宇宙覇者になる可能性
  • 株式投資は自己判断で。この番組は投資助言ではありません
宇宙×AI×超知能の交差点。
歴史的IPOだが、冷静な判断が求められています。
