② CodeBurn: トークンのお金はどこへ行った？

  • CodeBurnはAIコーディングのトークン消費とコストを可視化するローカルCLIツール
  • 18種類のAIコーディングアシスタントに対応。Claude Code、Codex、Cursor、OpenCodeなど幅広くサポート
  • クラウドにデータを送信しないローカルファースト設計。プライバシーに配慮
  • Ink製のTUIダッシュボードでターミナル上に美しく表示。13カテゴリのタスク分類機能付き
「私のAIコーディング代、
月にいくらかかってる？」
これで分かる！

③ CodeBurnの多機能さを解剖

  • 13カテゴリのタスク自動分類機能。コーディング・デバッグ・リファクタリングなどを自動で仕分け
  • LiteLLMの価格エンジンで正確なコスト計算。通貨変換にも対応
  • CSV/JSON形式でのデータエクスポート対応。期間比較も可能でコスト推移を追跡
  • モデル比較エンジン搭載。どのモデルがどのタスクにコスト効率がいいか分析できる
npm install -g codeburn
codeburn daily    # 日次レポート
codeburn compare   # モデル比較
macOSメニューバーアプリもあって
常にコストを意識できる仕組み

④ CodeBurnの対応ツール一覧

  • Claude Code — JSONLファイルからメッセージIDで重複排除して正確に集計
  • Codex CLI — rolloutファイルを読み取り累積トークン数で交差検証
  • Cursor — SQLiteデータベースからtoken countを抽出。better-sqlite3が必要
  • OpenCode / Pi / OMP にも対応。サブタスクは二重カウントを回避

⑧ Agent Skills: AIにスキルを教える仕組み

  • SKILL.mdというMarkdownファイルでAIエージェントにツールの使い方を教えるシステム
  • OpenClaw、Claude Code、Codex CLIなど複数のエージェントで共通利用可能
  • セキュリティレビュー済みのキュレーション済みスキルをマーケットプレイスからインストール
  • 500以上のスキルが登録済み。10カテゴリ・2000件以上のユーザーレビューあり
「AIエージェントのスキル」
= AIの能力を拡張する
プラグインのようなもの

⑨ Agent Skillsの活用例

  • Web Search Plus — 6つの検索エンジンを自動選択するスマート検索ルーティング
  • Browser Use — ブラウザ操作の自動化。スクリーンショットやデータ抽出も
  • GitHub Manager — PR管理・Issue追跡・コードレビューを自動化
  • Terminal Commander — 複数コマンドの orchestration とエラー回復を自動化

⑪ Turbovec: 超高速ベクトル量子化

  • TurbovecはRust製の高速ベクトル量子化ライブラリ。Pythonバインディング付き
  • Google ResearchのTurboQuant（ICLR 2026採択）を実装した実用的なツール
  • 16倍の圧縮を実現。1536次元のFP32ベクトルを6,144バイト→384バイトに
  • データ非依存のアプローチ。コードブックの再訓練が不要でインフラがシンプル
RAGやセマンティック検索で
メモリとコストを大幅削減！

⑫ Turbovecの技術的アプローチ

  • ベクトルを超球面上の方向として扱い、単位長に正規化して処理
  • ランダムな直交回転を適用して予測可能な座標分布（ベータ分布→ガウス）を実現
  • Lloyd-Maxスカラ量子化で最適バケッティング。SIMD組み込み関数で高速スコア計算
  • 新しいベクトルを動的に追加可能。FAISS PQのように再訓練が不要

Recall (d=3072, 2-bit)
FAISS 0.903
一部条件でFAISSを上回る

⑬ Turbovecのベンチマーク結果

  • ARM M3 MaxではFAISSの2〜25%以内の速度。実用レベルの性能
  • x86 Sapphire RapidsではFAISSより1.4〜3.7倍遅いが、最適化が進行中
  • Lightning AIとPredibaseの共同創業者が専門家として推薦
  • MITライセンスでオープンソース。5,600以上のGitHubスター
cargo build --release
pip install maturin
maturin build --release
x86の最適化が進めば

⑭ CCUsage: AIコーディングCLIの使用量分析

  • CCUsageはコーディングエージェントCLIのトークン使用量とコストを分析する高速ローカルツール
  • 15種類のソースに対応。Claude Code、Codex、Amp、Droid、Goose、OpenClaw、Kimiなど
  • 日次・週次・月次・セッション単位でのレポート出力に対応
  • オフラインモード対応。ネットワークなしでもキャッシュ済みの価格データで動作
CodeBurnと似ているが、
対応ソース数で勝る
のがCCUsageの強み

⑮ CCUsageの便利な機能群

  • フォーカスビュー — 全ソースを一覧後に特定のCLIに絞り込んで分析可能
  • キャッシュトークンの作成と読み取りを別々に追跡。正確なコスト計算
  • JSON出力 — プログラム的な利用に向けた構造化データエクスポート
  • Raycast拡張も公開。macOSのRaycastから手軽に使用量を確認
bunx ccusage daily      # 日次レポート
bunx ccusage weekly     # 週次レポート
bunx ccusage claude     # Claude Codeのみ
bunx ccusage --offline  # オフラインモード
