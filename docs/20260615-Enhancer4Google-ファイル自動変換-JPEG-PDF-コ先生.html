  • Googleの各種サービスをより快適に使えるChrome拡張機能
  • 「共通機能」の中に「ダウンロード時自動変換」を搭載

③画像変換機能の概要

  • 扱いづらい画像フォーマットをJPEGに自動変換してダウンロード
  • 対応形式: HEIC HEIF TIFF WebP BMP AVIF
  • 変換先フォーマットはJPEGを指定。各形式を個別にオン/オフ可能
  • ダウンロード時に裏側で自動変換が走るので、意識する必要なし
例えばiPhoneのHEIC写真も、自動でJPEGになって落ちてくる！

④新対応！AVIF形式とは？

  • AVIFは2019年頃から登場した新しい画像フォーマット
  • 圧縮方式が優秀で、高品質かつ軽量な画像を実現
  • Web上での画像公開には重宝される形式として普及中
  • しかしローカルに保存すると扱いづらいという課題があった
  • ユーザーからの要望を受けて、バージョン1.3.8で即座に対応を実現
「AVIFをJPEGに変換できない？」という声に即対応！

⑤画像変換の設定方法

  • 「エンハンサーfor Google」のアイコンをクリックして設定画面を開く
  • 「共通機能」の中にある「ダウンロード時自動変換」を選択
  • 変換したい形式をオンにして、変換先フォーマットとしてJPEGを選択
  • 不要な形式はオフにすれば、そのまま元の形式でダウンロードされる
オン・オフは形式ごとに個別設定できるから、自分の用途に合わせて！

⑥変換の実際の動作例

  • AVIF画像をダウンロードしようとすると、自動的にJPEGに変換される
  • 変換前のフォーマットがオフの状態なら、そのまま元の形式で保存される
  • 例: WebPをオフ → WebPのまま保存。AVIFをオフ → AVIFのまま保存
  • 変換対象を明確にオンにしておけば、常にJPEGで手に入る
Windowsで「表示できない！」と困った時にこの機能が大活躍！

⑦ドキュメント変換機能: 2つの方式

  • Word・Excel・PowerPointのPDF変換には2つの方式が存在
  • 方式①: Googleドライブ内のファイル → 純正エンジンでPDF化
  • 方式②: ドライブ外のファイル → GAS（Apps Script）でPDF化
  • なぜ2つに分かれているのか？それはファイルIDの有無が鍵だから
2つのPDF変換がある理由、実は重要なんです！

⑧方式①: Googleドライブ純正エンジン

  • Googleドライブに格納されたファイルにはファイルIDが振られている
  • このファイルIDを使ってGoogle純正のPDFエクスポートが利用可能
  • ダウンロードボタンを押すだけで自動的にPDF変換が完了
  • ユーザー側で特別な設定は不要。デフォルトで動作する
ドライブ内のファイルなら何もしなくてもOK！シームレスに変換される

⑨方式②: GASを使ったPDF変換

  • Google検索で見つけたWordファイルや、Gmailの添付ファイルなど
  • これらはファイルIDが存在しないので純正エンジンは使えない
  • そこでGAS（Google Apps Script）をデプロイして裏側で変換を実行
  • デプロイしたWebアプリのURLを拡張機能の設定に貼り付ける方式
ドライブ外のファイルでも、GASがあればPDF変換ができる！

⑪GASデプロイ: 手順①プロジェクト作成

  • Googleドライブ → 新規 → その他 → Google Apps Script を開く
  • 初期状態では「無題のプロジェクト」になっている
  • プロジェクト名は管理しやすい名前に変更（リポジトリ名を使うのがおすすめ）
  • エディタ上の code.gs にコードをコピー＆ペースト
  • リポジトリの「Copy raw file」ボタンで簡単にコードをコピー可能
「Copy raw file」を押すだけでコードがコピーされる！超簡単

⑫GASデプロイ: 手順②サービス追加

  • コードを貼り付けたら必ず保存ボタンをクリック
  • 左サイドパネルの「サービス」を追加をクリック
  • 一覧から Drive API を選択する
  • バージョンは必ずV2を選ぶこと！V3だと一部の関数でエラーが発生
  • IDは変更不要、そのまま「追加」をクリック
V3ではなくV2！ここ間違えやすいから注意！

⑬GASデプロイ: 手順③デプロイ実行

  • 右上の「デプロイ」ボタン → 「新しいデプロイ」を選択
  • 歯車マークをクリックして ウェブアプリ を選択
  • 「次のユーザーとして実行」は「自分」に必ず設定
  • 「アクセスできるユーザー」は「全員」を選択（必須!）
「全員」は社内の全員ではなく、「全員（匿名含む）」を選ぶこと！

⑭デプロイ時の注意点

  • デプロイ時に「アクセスを承認」のダイアログが表示される
  • ダイアログの一番下の「続行（Continue）」ボタンを押す
  • Google Workspace環境の場合、組織で「全員」を選べない制限がある場合も
  • その場合はGAS機能のみ動作しないので注意（純正エンジン部分はOK）
  • 構築自体は慣れれば数分で完了する
丁寧にやれば10分、慣れたら数分！時間はかからないよ

⑮デプロイURLの登録

  • デプロイが完了するとデプロイURLが発行される
  • 発行されたURLを「コピー」ボタンでコピー
  • エンハンサーfor Googleの設定画面 → ドキュメント変換設定
  • 「GAS Web App URL」の入力欄にURLを貼り付ける
  • これでドライブ外のファイルもPDF変換が可能になる！
URLを貼り付けるだけで設定完了！あとは自動で動くよ

⑯実際の変換動作デモ

  • Google検索で見つけた .docx ファイルを右クリック
  • 「ドキュメントをPDFに変換中...」というメッセージが表示される
  • 裏側ではGASサーバーにドキュメントが送信され、サーバー上でPDF変換が実行される
  • 完了すると自動的にPDFファイルがダウンロードされる
  • Googleドライブの場所に関係なく、どこであってもPDFが手に入る
ドライブの外にあるファイルでも、ポチッとするだけでPDFに！
