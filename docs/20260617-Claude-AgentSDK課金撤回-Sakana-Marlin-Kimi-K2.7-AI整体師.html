① Claude Agent SDK：従量課金制度の撤回

  • Anthropicは2026年6月15日から、Claude Agent SDKの利用をサブスクリプション枠とは別の月次クレジット制に移行する予定だった
  • クレジット超過分はAPIレートでの従量課金される計画だった
  • しかし施行日当天に突然一時停止。「何も変更されていない」と発表
  • 当面の間は既存のサブスクリプション枠から引き続き消費される
開発者は移行準備を進めていたのに
当日になって突然撤回とは…
コミュニティは困惑中

② 課金変更の内容とは

  • 対象はClaude Agent SDK、claude -pコマンド、サードパーティ製アプリ
  • 従来：サブスクリプションの利用枠で消費 → 変更案：別枠の月次クレジット制
  • Zedエディタなどサードパーティツールの利用コストが跳ね上がる懸念があった
  • Anthropicは「プランを更新するために取り組んでいる」と継続協議を表明
月額$200払ってるのに
追加課金が発生するのは
開発者にとって痛い

③ Claude Managed Agentsのベータ版公開

  • AnthropicはClaude Managed Agentsの公開ベータ版を同時にリリース
  • エージェントをスケジュール実行できるようになった
  • 環境変数を安全に保管する機能が追加された
  • 本格的なエージェント運用に向けた一歩前進
課金は撤回したけど
Managed Agentsは進化中。
エージェント運用はこれからが本番

  • Fable 5には高度な安全機能、Mythos 5はサイバーセキュリティ能力が高い
  • しかし米国政府の輸出管理指令により6月12日にアクセスが停止

⑤ Sakana AIがMarlinを提供開始

  • Sakana AI（東京発のAIスタートアップ）が初の商用プロダクトを発表
  • リリース日は2026年6月15日
  • 製品名はSakana Marlin（サカナ・マーリン）
  • B2B向けの自律型リサーチエージェント
日本のAIスタートアップが
ついに商用プロダクトを届けた！
Sakana AIの大きな一歩

⑥ Marlinの特徴：深掘りリサーチ

  • 一般的なチャットボットのような即時応答ではない
  • 最大8時間にわたり自律的に推論を実行
  • 60〜100ページ規模の戦略レポートや役員向けスライドを生成
  • 適切な情報源とともに出力される
8時間も自律で動き続けて
100ページのレポートを出す。
Deep Researchの進化系

⑦ Marlinの自律リサーチの仕組み

  • ユーザーがリサーチテーマを与えるだけで起動
  • 仮説立案→ウェブからの情報収集→矛盾点の検証を自律的に実行
  • 推論プロセスを繰り返して精度の高い分析に到達
  • ビジネスの戦略的意思決定を支援する設計
テーマを渡すだけ。
あとはAIが自律で回す。

⑧ Marlinの料金体系

  • 1回の実行につき100クレジット（1クレジット=98円）
  • 1回あたり約9,800円の従量課金からスタート
  • 月額プランも提供されている
  • 企業のリサーチ部門向けに設計された価格帯

  • 256Kトークンのコンテキストウィンドウをサポート
  • MoonViTビジョンエンコーダでテキスト・画像・動画の入力に対応
