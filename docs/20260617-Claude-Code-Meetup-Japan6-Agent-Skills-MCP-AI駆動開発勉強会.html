④ Agent Skillsの5つの構成要素

  • ルール エージェントの振る舞いを定義。一番最初に設定すべき基盤
  • ツール Claude Codeに読み取り・書き込み・実行などの権限を付与
  • スキル 再利用可能な手順書。特定タスクの定型パターン
  • MCP 外部サービスとの統合インターフェース
  • サブエージェント 複雑タスクを分割して並列・直列処理
ルール→ツール→スキル→MCPの順で足すのがベストプラクティス！
