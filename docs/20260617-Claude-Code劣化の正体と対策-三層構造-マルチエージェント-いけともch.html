⑩ 対策①：モデルを別のバージョンに切り替える

  • めちゃくちゃ簡単で対策も不要。すぐに実行できる最も手軽な対策
  • Opus 4.8がおかしくて信用できないなら、Opus 4.7に切り替えるだけ
  • 4.7はマイナーチェンジだったので大きな性能差はない
  • 背景にはFible 5の提供停止があり、その負荷が4.8に集中した可能性も
