GenLatteの魔法: Flutter × Firebase × Gemini

①GenLatteとは: AIでラテアートを生成する体験

  • GenLatteはGoogle Cloud Next '26で披露されたAI搭載カフェのデモだ。
  • ユーザーがGeminiが生成したオリジナル・ラテアートを選び、実際に飲めるラテとして提供される。
  • 目でも舌でも楽しめる体験型ブースとして大好評を博した。
「自分だけのラテアート」をAIで生成して飲めるなんて、最高に新鮮！

②技術スタック: Flutter + Firebase + Gemini

  • Flutter: 注文画面のUIを構築。クロスプラットフォームで高速開発。
  • Firebase: バックエンドサービスを提供。リアルタイム通信とデータ管理。
  • Gemini: AI画像生成エンジン。特に「Nano Banana」モデルを使用。
フロントもバックエンドもAIもGoogleの技術で一貫構成！

③GenUI: ジェネレーティブUIのパラダイム

  • Generative UI (GenUI): AIが動的にUI要素を生成・組み立てる新しい手法。
  • A2UI (Agent-to-UI): エージェントがUIそのものを駆動し、柔軟な体験を提供。
  • 従来の静的インターフェースを超え、リアルタイムにパーソナライズされる。
テキストチャットボットの次は、AIがUI自体を作る時代！

④注文の流れ: ユーザー体験のステップ

  • ラテの種類を選択: ラテ、ノンファット、モカなど。
  • デザインのテーマを質問に答えて説明: 「雪山」や「動物」など。
  • Geminiが4つのデザイン案を生成。
  • お気に入りを選んでSubmit。
  • バリスタがフォームに画像を印刷してラテを提供。
「雪山」をテーマにしたら本当に綺麗なデザインが出来たよ！

⑤Gemini Nano Banana: エッジAI画像生成

  • Nano Banana: Geminiの小型モデルで、高速に画像を生成。
  • ラテアートのデザイン候補を数秒で4パターン生成する。
  • ユーザーが選ぶことで、Human-in-the-Loopの検証を実現。
  • エッジデバイスでの高速推論が可能な軽量モデル。
人間が確認して選ぶ → AIの品質担保の新しい形！

⑥Human-in-the-Loop: 人間による検証の重要性

  • AIが生成した結果を人間が確認・選択する仕組み。
  • コーディングエージェントでもテストと検証パラダイムが重要。
  • AIの出力スピード向上だけでなく、保守コストの削減を目指す。
  • 人間とAIの協調こそが実用的なアプリ開発の鍵。
AIに任せっきりではなく、人間の目で最終チェックする設計が大事

  • エージェントが自動でコードを生成し、開発スピードが劇的に向上。
