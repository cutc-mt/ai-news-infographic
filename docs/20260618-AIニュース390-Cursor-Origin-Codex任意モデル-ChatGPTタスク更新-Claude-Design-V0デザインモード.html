AIニュースまとめ #390

2026年6月18日 - Cursor Origin発表・Codex任意モデル対応・ChatGPTタスク機能・Claude Design・V0デザインモード

① Cursor Origin: GitHubに対抗

  • CursorがOriginという名前で、コードストレージとGitホスティングサービスを秋にリリース予定。
  • GitHubの対抗馬になると話題になっている。
  • エージェントが管理しやすい設計で、AIファーストなGit管理を目指す模様。
  • ウェイトリストに登録可能だが、有料か無料かはまだ不明。秋リリースまで待つ必要あり。
GitHub依存から脱却？Cursorの世界観が広がりそう！

② ローカルエージェントをクラウドへ

  • Cursorのローカルエージェント作業をクラウドに移動させることがより簡単になった。
  • PCを閉じても作業を継続できるのが大きなメリット。
  • 現状はGitHubとの連携が必要だが、Originが登場すればCursor独自のGit管理でクラウド連携が可能に。
  • コミットをクラウドにプッシュすれば、別端末からも作業を引き継げる。

③ Cursor iOSアプリが登場予定

  • CursorのiOSアプリがリリースされる計画がある。
  • ②のクラウド移行機能と連携することで、スマホからもコーディング操作が可能に。
  • ChatGPTアプリ経由でCodexが操作できるように、Cursorもモバイルで使えるようになるかも。
スマホでコード操作！どこでも開発できる未来

④ Codex CLI: 任意のモデルに対応

  • CodexアプリのCLI・SDKがOpenAIモデル以外にも対応。
  • config.tomlファイルを更新するだけで、GLM-5.2やOllama、LM Studioのローカルモデルが使える。
  • タスクごとに得意なモデルを使い分けができるのが強み。
  • ローカルモデルならデータを海外サーバーに送らないセキュリティ上のメリットも。
中華系モデルも試せる！コスパ重視で使い分けよう

⑤ ChatGPTタスク機能がアップデート

  • ウェブ版ChatGPTのタスク機能がバージョンアップ。より高速で信頼性が向上。
  • 新しいスケジュールページから管理が簡単になった。
  • 以前のタスク機能の強化版という位置づけ。より安定した動作が期待できる。

⑥ Pulse機能がまもなく終了

  • プロ契約向けのPulse（パルス）機能がもうすぐ終了する。
  • Pulseはユーザーの好みに合わせてニュースを自動収集・整理してくれる機能だった。
  • 代わりにタスク機能に全員移行する方針。Pulseの代替としてスケジュール機能を使う流れに。
Pulseを使ってなかった人には影響なし。タスク機能の活用法を模索中

⑦ Claude Designがアップデート

  • Claude Codeと同期できるようになり、開発ワークフローとデザインが連携。
  • キャンバス上で直接編集ができるようになり、使い勝手が大幅改善。
  • ただしクレジット消費が激しく、本格的なデザイン制作には工夫が必要。
  • Anthropicが手放さずにアップデートを続けている＝重要プロダクトとして注力中。

⑧ V0にデザインモードが登場

  • VercelのV0にデザインモードが追加。直接デザイン画面にプロンプトを打ち込んで修正できる。
  • CodexデスクトップやClaude Designと同様の対話型デザイン編集が可能に。
  • フロントエンド開発とデザインの境界がさらに曖昧になっていく流れ。
デザインツールがコーディングツールに統合されていく！

⑨ Cursor×SpaceXの買収の噂

  • CursorがSpaceXと合併・買収の話が出たのとほぼ同時にOriginの発表があった。
  • これは噂レベルの話で、実際の買収が進行中かは不明。
  • ただAIコーディング業界のM&A加速を示唆する動きとして注目されている。
火星に行くコードを書く日が来るかも！？

⑩ AIコーディングツール競争激化

  • Cursor・Codex・Claude Code・V0と、主要プレイヤーが全員アップデートを連発。
  • コーディングだけでなく、デザイン・Git管理・モバイル操作へ領域を拡大中。

  • 中華系モデルなど、様々なモデルをローカルで試せる環境が整いつつある。
