Claude Code 最新実践ガイド

2026年6月18日 - アプリ・Webサイト・スライド・動画まで全部作る方法

① Claude Codeとは

  • Anthropicが開発したAIエージェントツール。開発チームの一員のようにプロジェクトに参加する。
  • プログラムを書いたり、ファイルの中身を読んで間違いを直したり、新しい機能を追加したりしてくれる。
  • ただコードを書くだけのAIではなく、「ここを直して」「この機能を追加して」と日本語で指示できる。

② 4つの利用形態

  • CLI版: ターミナルの黒い画面で操作。Claude Codeのほぼ全機能が使える一番人気の形態。
  • デスクトップアプリ: 2026年4月に全面強化され、さらに使いやすく進化。
  • Webブラウザ版: ブラウザ上から直接利用できる。
  • VS Code拡張機能: 無料のコードエディター内で動作。フォルダ構成を見ながら開発できる。

③ カスタマイズ機能

  • Claude.md: プロジェクトの書き方やルールをファイルに書いておくと、毎回それを読んで作業してくれる。
  • メモリ機能: よく使うコマンドや過去に直したエラーの内容を覚えてくれるので、同じ説明を繰り返さなくてよい。
  • スキル: よく使う作業手順をまとめて登録。再利用可能なワークフローを作れる。
  • フック: ファイルを直した後に自動でコードを整形したり、間違いがないかチェックしたりできる。
  • MCP: Claude Codeと外部サービスをつなげる仕組み。
自分専用・チーム専用の開発アシスタントに！

④ 利用料金と前提条件

  • Claude Codeは基本的にClaudeの有料サブスクリプションユーザー向けに提供されている。
  • 利用にはあらかじめパソコンにClaude Codeをインストールしておく必要がある。
  • 非エンジニアのユーザーにもじわじわと浸透し始めており、プログラミング知識がなくても利用可能。

⑤ Webサイトの作り方: 準備

  • まずパソコン内にプロジェクトフォルダーを用意する（例: デスクトップに「web」フォルダを作成）。
  • コマンドプロンプト（ターミナル）を立ち上げて、cdコマンドでプロジェクトフォルダーに移動する。
  • そのフォルダーの中でClaude Codeを立ち上げると、自動的にプロジェクトフォルダーがセットされる。

⑥ Webサイトの作り方: 設定

  • /modelコマンド: Claude Codeで使うAIモデルを選択。今回はClaude Opus 4.8を選択。
  • /effortコマンド: AIにどれだけ深く考えさせるか、思考の強さを選べる。中間の「ハイ」がおすすめ。
上位モデル＋高思考で高品質な出力を！

⑦ Webサイトの作り方: 生成

  • テキスト入力欄に作りたいWebサイトの内容と、ファイルをプロジェクトフォルダーに納品してほしい旨を伝えて送信。
  • Claude Codeがブレインストーミング系のスキルを自動で使い、デザインや構成について会話しながら決めていく。
  • Anthropic公式の品質の高いスキルでコーディングがスタートし、HTMLファイル等を納品してくれる。

⑧ パーミッションモード

  • デフォルトではClaude Codeは作業のたびにユーザーの承認を求めてくる。
  • acceptEditsモードをセットすると、ファイルの編集や一般的なファイル操作を確認なしで実行してくれる。
  • Shift + Tabキーでチャット開始前にパーミッションモードを切り替え可能。
自動モードは誤操作のリスクがあるので注意！

⑨ VS Codeでのアプリ開発

  • VS Codeは無料のコードエディター。Claude Codeの拡張機能で連携できる。
  • プロジェクトフォルダーの構成を把握しながら開発を進められる。
  • 内蔵のWebブラウザーでHTMLを確認できるため、別ブラウザを立ち上げる必要がない。
  • VS Codeのターミナルを開くと、自動的にプロジェクトフォルダーがセットされる。

⑩ プランモードで計画立案

  • /planコマンドでプランモードに移行。いきなりコードを書かずに、まずは実装の流れを整理する。
  • Claude Codeがアプリの種類や機能、技術構成などを聞いてくるので、それに答えると詳細な実装計画を立ててくれる。
  • 計画を確認し、修正したい項目は修正。問題なければ実行を指示するとコーディングが開始される。
計画→確認→実行の流れで品質アップ！

⑪ ビジュアルコンパニオン

  • 「ビジュアルコンパニオン機能を使ってください」と入力すると、Webブラウザ上で試作品を表示できる。
  • ゲームやアプリのビジュアルスタイルを数パターン用意してくれる。
  • コンテンツのUIを視覚的に把握しながら制作を進められるため、イメージと違うものができるのを防げる。

⑫ デスクトップアプリの進化

  • 2026年4月に全面強化されてさらに使いやすくなった。
  • スキル: Claude Codeに特定の専門知識や新たな機能を付与できる。
  • コネクタ: Canva、Notion、Gmailなど色々なアプリと接続して機能を使える。
  • プラグイン: スキルやコネクタをひとまとめにして共有できる仕組み。

⑬ Remotionスキルで動画制作

  • RemotionはReactを活用して動画を作れるAIツール。日本語の指示でアニメーション動画を生成できる。
  • GitHubからZIPファイルをダウンロードし、Remotionフォルダを再度ZIPに圧縮してデスクトップアプリにドラッグ＆ドロップでインストール。
  • プロンプトを入力すると動画の内容・スタイル・表示言語などを聞かれ、会話しながらストーリーボードを作ってくれる。

⑭ 動画のレンダリングと出力

  • ストーリーボードや実装計画を確認し、問題がなければ続行を依頼すると自動でタスクを実行して動画を生成。
  • Webブラウザで動く専用プレイヤーを使うこともできる。
  • コマンドでプレイヤーを立ち上げ、Render Videoボタンを押せばMP4フォーマットでダウンロード可能。
動画編集の知識がなくても高品質な解説動画が作れる！

⑮ サブエージェント機能

  • サブエージェントはメインのAIエージェントとは別に、複数のエージェントを並列で動かせる仕組み。
  • 1体のAIだけで動かすよりも高速に作業を実行できる。
  • コンテンツを作る際にサブエージェントを使うかどうか聞かれることがある。

⑯ スライド資料の自動生成

  • Claude Codeには高品質なスライド資料を作るためのスキルが標準で実装されている。
  • スライド資料の生成を依頼すると、自動でスライド生成用スキルを使って品質の高い資料を作ってくれる。
  • デスクトップアプリのコネクタ機能でCanvaとも連携可能。
プレゼンの構成から本文まで自動で生成！

⑰ ルーティン機能で作業自動化

  • ルーティン: 定期的に実行するタスクを自動で実行してくれるスケジュール機能。
  • CLI版の/loopコマンドとは違い、パソコンを閉じていてもクラウド上で自動で進めてくれる。
  • 左サイドバーの「ルーティン」から、作業の名前・指示・トリガーを設定して作成。
  • トリガーには日時指定のほか、GitHubのファイル変更履歴との連携やAPI呼び出しにも対応。
  • 接続しているアプリのコネクターもトリガーに利用できる。
日々の作業を自動化して習慣化しよう！

⑱ 今後の展望とまとめ

  • Claude CodeはAnthropicが特に力を入れているツールの1つで、開発スピードもトップクラス。
  • 半年後にはさらに機能が強化され、今とは全く違う開発スタイルが当たり前になっている可能性がある。
  • AIツールに興味のある方は、ぜひ自分の手で試してみてください。
