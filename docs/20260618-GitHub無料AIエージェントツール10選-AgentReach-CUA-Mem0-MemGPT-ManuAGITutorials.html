①全体概要: エージェントを次のレベルへ

  • AIエージェントが賢いテキストボックスから本物のオペレーターに進化する10のOSSプロジェクトを紹介。
  • インターネットアクセス、デスクトップ操作、長期記憶、セキュリティスキャンまで網羅。
  • ランダムなテストに何週間も費やす代わりに、今すぐ使えるプロジェクトのスタックを提供。
すべて無料＆オープンソース！

②AgentReach: エージェントに目と耳を

  • ウェブページの読み込みと検索エンジンへのアクセスをエージェントに提供。
  • YouTubeの字幕取得やRSSフィードへの対応。
  • GitHub、X、Reddit、LinkedIn、V2EX、小紅書など多数のプラットフォーム接続。
  • クッキーはローカル保存でプライバシー安全。
  • マルチバックエンドルーティングで、壊れた連携も裏で切り替え可能。
doctorコマンドで
診断もできるよ 🩺

③CUA: コンピュータ操作エージェント

  • 1つのAPIでLinux、macOS、Windows、Androidをクロスプラットフォーム対応。
  • クラウドサンドボックスからローカル環境まで対応。
  • スクリーンショットの取得、シェルコマンドの実行、マウスクリック、キーボード入力が可能。
  • モバイルのジェスチャー操作にも対応。
「読める」から「動ける」へ！
OSWorldやWindowsArenaでベンチマーク評価も 📊

④Agent Memory: 持続的な記憶レイヤー

  • Cloud Code、GitHub Copilot CLI、Cursor、Gemini CLI、OpenClawなど主要なMCPクライアントに対応。
  • ツール使用イベントをキャプチャし、秘密情報をフィルタリングして保存。
  • 観測データを構造化ファクトに圧縮して索引付け。
  • BM25＋ベクトル検索で関連情報を未来の会話に注入。
  • コンテキスト全ダンプではなく関連項目のみ取得でトークンコスト大幅削減。
毎回同じプロジェクトの説明を
し直さなくていい！ 🧠

⑤Lazy Codex: Codexの構造化ハーネス

  • Lazy Vim for Codexのような構造化アプローチ。大規模・複雑なコードベース向け。
  • init /deep → 階層的なプロジェクト記憶をエージェントに作成。
  • solve /ballash /plan → コード変更前に計画を書く。
  • startWork → 計画を実行し、solve /loop → 検証までループ。
  • Explore、Librarian、Plan、Reviewerなどサブエージェントの役割を選択可能。
エージェントコーディング最大の
隠れコスト「カオス」を解決 🛡️

⑥Hermes Agent: 成長するエージェント

  • CLI使用、パーソナリティ、セッション、永続メモリ、MCP統合を搭載。
  • Cronスケジューリングで定期タスクの自動実行が可能。
  • メッセージゲートウェイでTelegram、Discord、Slack、WhatsApp、Signalに接続。
  • Home Assistantへの統合でスマートホーム連携も。
  • 6つのバックエンド対応: Local、Docker、SSH、Singularity、Modal、Daytona。
Noose Portalでモデル＋検索＋
画像生成＋TTSを1つのサブスクリプションで 💎

⑦Agent Skills: エンジニアリングを体系化

  • Addy Osmaniによるプロダクション級のエンジニアリングスキルパック。
  • 7つのスラッシュコマンドで開発ライフサイクルをカバー。
  • /spec → /plan → /build → /test → /review → /simplify → /ship。
  • 仕様、アトミックプランニング、テスト、レビューなど強力なワークフローを自動適用。
  • Clawed Code、Cursor、Gemini CLI、Windsurf、OpenCode等に対応。プレーンMarkdown形式。
「速く」ではなく「より良く」
コードを書かせる秘訣 🔧

⑧Skill Scanner: スキルの安全性を検査

  • NVIDIAが開発するAIエージェントスキル向けセキュリティスキャナー。
  • 入力形式: Gitリポジトリ、URL、ZIPファイル、ディレクトリ、単一ファイル。
  • 64種類の脆弱性パターンを16カテゴリで検出。
  • プロンプトインジェクション、データ漏洩、権限昇格、サプライチェーンリスクなどをカバー。
  • 静的チェック＋LLM意味解析のハイブリッド評価。0-100のリスクスコアで出力。
エージェントのマジックを
盲目的にインストールしないで！ 💀

⑨PM Skills: プロダクト思考のAI化

  • AIオペレーティングシステムとして68個のPMスキルと42のワークフローを搭載。
  • 9つのプラグインでディスカバリー、戦略、実行、成長、リリースを網羅。
  • 汎用AIはテキストを返すが、PM Skillsは構造化されたフレームワークを提供。
  • /discoverコマンドでブレインストーミング→仮説→優先順位付け→実験設計をチェーン実行。
  • Claude Code/Claude CodeWork向け設計。Codexにもプラグインとしてインストール可能。
AIが「書く」から「プロダクトチームのように
考える」へと進化 🎯

⑩Rooter: トークンコストを最適化

  • ローカルのスマートルーターでモデルプロバイダー間の自動フォールバックを実現。
  • RTK Token Saverでツール出力を圧縮し、入力トークンを20-40%削減。
  • 3層フォールバック: サブスクリプション → 安価 → 無料プロバイダー。
  • リアルタイムクォータ追跡と使用量分析を提供。
  • Claude Code、Codex、Cursor、Kiro、Copilot、Antigravity等に対応。
トークン消費を気にせず
実験し続けよう！ 🐷

⑪AgentPet: デスクトップの相棒

  • macOSとWindowsに対応するAIコーディングエージェント監視ペット。
    • Cloud Code、Codex、Gemini CLI、Cursor、OpenCode、Windsurf等のHooksに対応。
    • メニューバーでエージェントの状態（実行中 / 完了 / 入力待ち）をリアルタイム表示。
    • エージェントが完了または注意を要する時にアクティブ通知を送信。
    • ゲーム化要素: XP、レベル、ステージ、統計HUD、ウェブプロフィール、リーダーボード。
    エージェントを見守る可愛い
    デスクトップペット！ 🐾
