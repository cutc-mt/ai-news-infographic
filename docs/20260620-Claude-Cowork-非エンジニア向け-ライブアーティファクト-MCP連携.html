Claude Cowork 完全解説

非エンジニア向け！ライブアーティファクトで仕事革命
2026年6月20日

① Claude Coworkとは: 非エンジニアのためのAIアプリ

  • Anthropicが提供するデスクトップアプリ内の機能。パソコンに入れてそのまま使える
  • 月額20ドル以上の有料プランで利用可能。Claude Pro以上が必要
  • 非エンジニアでもClaude Codeの力を活用できる。プログラミング知識は一切不要
  • 裏側ではClaude Codeが動いている。コードの複雑さをAIが丸吸いしてくれる仕組み
「プログラミング不要」が最大の売り。
エンジニア以外のビジネス層に
AI活用の門戸を開いた划期的機能です。

② 3つのモード: チャット・コワーク・コード

  • チャットモード — 通常のClaudeと同じ。文章の質問や要約などに使える基本モード
  • コワークモード — 非エンジニア向け。フォルダ連携やダッシュボード作成など実務で活躍
  • コードモード — エンジニア向け。Claude Codeのフル機能が使える本格開発モード
  • コワークとコードは裏側でClaude Codeが動いている。違いは表示の簡さと機能の露出度
モード切替はアプリ内でワンタップ。
現場の目的に合わせて柔軟に切り替えができます。

③ 利用量2倍キャンペーン: 7月5日まで

  • 7月5日まで限定で利用量が2倍に。普段使いなら十分すぎる量になる
  • 1日5時間の制限が10時間に拡大される。週間制限はそのまま変わらない点に注意
  • 月額20ドルのClaude Proで参加可能。追加料金なしで2倍使えるお得な期間
  • 試してみたい人はこの期間がチャンス。コワークの便利さを体験するのに最適
週間制限は変わらないので
集中的に使う日の計画を

④ ローカルフォルダ連携: PC内の資料に直接アクセス

  • パソコン上のフォルダに直接アクセス可能。仕事の資料をAIが読み込める
  • 議事録、集法、タスク管理など用途ごとのフォルダを設定できる
  • 例えば議事録フォルダを読み取らせて、タスク表と集法を自動で作成させる
  • 過去のフォーマットをAIが学習して同じ形式で新しい資料を生成してくれる
「このフォーマットで書いて」
と一度教えれば、次からは
自動で同じ形式で出力！

⑤ フォーマット学習機能: 過去の形式を継承

  • フォルダ内の既存ドキュメントを読み込んで書き方のルールを学習する
  • エクセルのタスク表なら同じ列構成、議事録なら同じ見出し形式で出力
  • 一度設定すれば毎回プロンプトで指定する手間が省ける。時間の大幅節約に
  • フォーマットを変えたい時は新しいテンプレートを教えるだけでAIが適応する
チームの共有フォーマットを

⑥ プロジェクト機能: 用途別の設定管理

  • プロジェクトごとにプロンプト・メモリー・ファイルを個別に設定できる
  • WebアプリのClaudeで作ったプロジェクトからインポートも可能
  • 複数プロジェクトを切り替えて使える。仕事と趣味など用途の区別が明確に
  • プロジェクト単位でルールを管理できるので複数業務の並行運用に便利

⑦ スケジューリング機能: 定期的にAIを自動起動

  • Coworkを定期的に自動起動させる機能。毎日決まった時間にAIが作業を開始
  • 例えば毎朝8時にAIニュースを検索してダッシュボードを自動作成させる設定が可能
  • Googleドライブからの定期データ取得にも対応。最新のドキュメントを自動で反映
  • 週次レポートの自動生成や、朝の情報収集の自動化などルーチンワークをAIに丸投げできる

⑧ ディスパッチ機能: スマホから遠隔操作

  • スマホからCoworkを起動・操作できる。パソコンの前にいなくてもAIに仕事を頼める
  • 出先から「このファイルを要約して」と指示を出すと、帰宅前には結果ができている
  • スケジューリングと組み合わせれば、AIが作業を終えたら通知を受け取ることも可能
  • チームメンバーが各自のスマホから同じCowork環境にアクセスすることもできる

  • Slack — チャンネルのメッセージを読み取ってタスクを抽出したり通知を送ったりできる
  • Gmail — メールの振り分けや要約、重要度判定をAIに任せられる
  • MCP（Model Context Protocol）を通じて多彩な外部サービスに拡張可能
