Claude Designが進化！
完璧なデザインソフトになった件を検証

ワイヤーフレーム・プロトタイプ・デザインシステムまで
Claude Designの全機能を徹底解説
📅 2026年6月20日 | チャンネル: ゲト

①Claude Designとは

  • Claudeに内蔵されたデザイン専用機能で、テキストからUIを作れる
  • Web版：Claude画面の左側にある「デザイン」ボタンからアクセス
  • デスクトップ版：左下の「デザイン」からアクセス可能
  • UIもめちゃくちゃ綺麗になって、以前よりずっと見やすくなった
Claude自体がデザインツールに進化した！
テキストの指示だけで画面が作れる

②ワイヤーフレーム機能とは

  • ウェブサイトを作る際の大枠の構成を先に組み立てる手法
  • デザインをコテコテに作るのではなく、要素の配置をまず決める
  • 白黒のシンプルな見た目で情報の構成を視覚化する
  • 要素だけを並べて、全体のバランスを確認できる
建物で言えば設計図！
デザインの前に構造を固める

③ワイヤーフレームの使い方

  • まずは指示文を投げる（例：ECサイトを作って）
  • 「ワイヤーフレーム」を選択して送信
  • いくつかの質問に答えると、自動でワイヤーフレームが生成される
  • 質疑応答のやり取りだけで、自然な形で構成が固まる
指示→質問→自動生成！
誰でも直感的に使える

④スマホ版も同時に作成

  • PC版だけでなく、スマホ版のレイアウトも自動で作ってくれる
  • レスポンシブ対応のサイトを一発で構築できる
  • PCとスマホ両方の画面を確認しながら進められる
  • 手動で2つのレイアウトを作る手間が大幅に削減
1回の指示でPC＋スマホ両方搞定！
これすごく便利じゃん...

⑤PDFでエクスポートも可能

  • 完成したワイヤーフレームをPDF形式で書き出しできる
  • チームメンバーと構成案を簡単に共有できる
  • 印刷してミーティングで使うことも可能
  • ファイルとして保存して、後から確認もできる
議事録や要件定義にも使える！
クライアント向けの提案書にも

⑥情報整理を先に終わらせる

  • 情報整理・デザイン・コーディングを同時にやるとぐちゃぐちゃになる
  • ステップ1：ワイヤーフレームで情報整理を完結させる
  • ステップ2：構成が固まったらデザインに進む
  • ステップ3：最後にコーディングで実装する
分けてやるのがコツ！
ワイヤーフレーム＝設計段階

⑦プロトタイプ機能とは

  • 実際に動作するウェブサイトを作れる機能
  • 要素をクリックすると、本物のサイトみたいに画面遷移する
  • 静的なデザインではなく、インタラクティブな体験を作れる
  • 完成前のイメージを実際に触って確認できる
ワイヤーフレームの次の段階！
動くサイトならイメージが伝わる

⑧ECサイトのプロトタイプ例

  • 例えばスキルマーケットのECサイトを作らせた
  • スキル詳細ページから商品の情報を確認できる
  • カートに追加の操作が実際にできる
  • 支払い方法の選択画面まで再現される
指示するだけでECサイトが動く！
プロトタイプの完成度がすごい

⑨一連の流れを再現できる

  • スキル詳細 → カート追加 → 支払い
  • 購入完了 → セットアップ方法の画面まで繋がる
  • ユーザーの導線全体を一つのプロトタイプとして作成できる
  • UX設計の段階で実際の操作感を体験できる
商品ページから購入完了まで
全部の流れが体験できる！

⑩進化した編集機能

  • 右上の「エディット」を押すと、全ての要素が選択可能になる
  • テキストの内容も直接変更できる
  • ドラッグ＆ドロップで位置の変更が可能
  • Deleteキーで不要な要素を削除できる
Figmaのような直感的な操作！
テキスト生成＆GUI編集の両立

⑪レイヤーパネルで細かく調整

  • 左側にレイヤーパネルがあり、要素の重なり順を管理できる
  • ピンポイントで特定の背景のグラデーションを消したり変更できる
  • フォントサイズや文字のスタイルを細かく調整できる
  • 部分的な修正が簡単で、全体をやり直す必要がない
一部だけ直したい時に最高！
デザイナー的な微調整ができる

⑫変数を使った文字列処理

  • 変数を使った文字列の動的処理が可能
  • プログラミングとデザインソフトの中間のような存在
  • 同じ要素を変数で一括変更できる
  • コンポーネント化して再利用できるスマートな設計
デザインとコードの架け橋！
エンジニアともデザイナーとも
話が通じるツール

⑬シェアボタンで連携

  • シェアボタンから様々なツールにデザインを送信できる
  • Claude Codeに送信してコード実装に進める
  • Vercel、Wix、Canvas、Adobe Creative等にも対応
  • デザインからデプロイまで一気に進められる
デザイン→実装→デプロイ
すべてワンストップで完結！

⑭Claude Codeで即実装

  • Web版ではOpen Claude Code Webで即実装を開始できる
  • デザインをそのままClaude Codeに渡してコードを生成
  • デザインと実装の切り替えがシームレス
  • 画像からコードへの変換が自動で行われる
デザインをコードに変換！
エンジニアの作業を大幅削減

⑮Vercelに一気にデプロイ

  • デザインからVercelへのデプロイまで一気に進められる
  • 完成したサイトを世界中に公開できる
  • デザインから公開までの時間が劇的に短縮される
  • URLが即座に発行されて、すぐに共有可能になる
デザインしたらすぐ公開！
理想のスピード感で制作できる

⑯デザインシンクで同期

  • Claude Codeのターミナルで/design loginで接続する
  • /design syncでClaude CodeとClaude Designを同期できる
  • ストーリーブック形式でローカルにデザインシステムが生成される
  • ローカルのデザインシステムをClaude Designに同期してUIで微調整できる
CodeとDesignの同期！
コード側の変更がすぐUIに反映

⑰コンポーネントの作成と同期

  • Claude Codeで新しいコンポーネント（グラフやナビゲーション等）を作れる
  • 作ったコンポーネントをデザインシステムに保存できる
  • 保存したものをClaude Designに同期してUIで確認
  • コンポーネントの再利用で制作の効率が上がる
コード側で部品を作って
Design側で見た目を調整！

⑱デザインシステムの構築

  • フォント、アイコン、ロゴ、カラー（ブルーの5色など）のルールを自動設定
  • デザインシステムを作ればLPや動画、アプリで統一感を保てる
  • 自分のブランドセットを使いながら制作できるようになる
  • 初期設定はかなり難しいので、Claude Codeと対話しながら進めるのがおすすめ
