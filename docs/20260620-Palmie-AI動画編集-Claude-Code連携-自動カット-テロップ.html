Palmie — Claude Codeで動かすAI動画編集

オープンソース・無料のAIファースト動画編集ソフトが登場
2026年6月20日

① Palmie（パルミ）とは: AIファーストな動画編集ソフト

  • オープンソース 無料 のAIファースト動画編集ソフト
  • 現時点ではMacのみ対応（Windows版は未定）
  • Premiere ProやFinal Cut Proライクな操作感で、Claude Codeから直接指示を出してAI駆動で編集ができる
  • Claude Codeとネイティブ連携可能。Claude CodeやCodexからPalmie本体に直接コマンドを送れる
動画編集ソフトにAIを直接組み込む発想が新しい。クリエイターの作業を根本から変える可能性があります。

② 動画編集の「めんどくさい」を解決したい

  • 「またええって言っちゃった」→ フィラーワード（えー、あー）の削除が手作業で大変
  • 無駄な余白や沈黙をカットするのが地味に時間を食う
  • テロップ（字幕）を手動で入れるのは想像以上に面倒
  • Claude Code単体では動画編集を完璧にやりきれないという課題があった
  • Palmieはその「Claude Codeでは足りない部分」を補完する存在
YouTuberあるある。「えーっと」「あのー」を消す作業だけでも数時間かかるんですよね。

③ 文字起こしとフィラーワードの自動検出

  • 動画をインポートすると自動で文字起こしを実行
  • AIが発話タイミングを解析してフィラーワードを自動で検出
  • 「えーっと」「あのー」「じゃあ」などの不要な言葉を特定
  • 音声波形から無音部分も並行して検出する仕組み
文字起こしとフィラーワード検出がワンストップで完了するのは大きいです。

④ ジェットカット: 0.2秒の余白も許さない

  • 超精密な自動カットがPalmieの目玉機能
  • 無音部分だけでなく0.2秒の短い余白も見逃さずに削除
  • 「ジェットカット」と呼ばれる高速カットモードを搭載
  • 数十分の動画でも一瞬で不要部分を刈り取れる
0.2秒の余白も許さない厳しさ！視聴者には見えないレベルの快適さを追求してます。

⑤ テロップ自動挿入: 字幕もAIにお任せ

  • 文字起こしデータからテロップ（字幕）を自動で生成
  • 標準スタイル: 画面中央下に白文字黒縁取り
  • 1シーン最大10文字・単語の途中で区切らないなど、読みやすさに配慮したルール設定が可能
  • Claude Codeからの指示で表示条件を細かくカスタマイズできる
テロップ1文字ずつ手作業でタイミングを合わせる地味な作業が全部自動化されます。

⑥ Claude Code/Codexとの連携の仕組み

  • Claude CodeのターミナルからPalmieのコマンドを直接実行可能
  • Claude Code ↔ Palmie 間のAPI連携でシームレスなワークフロー
  • 音声入力でもClaude Codeに追加指示を出せる。キーボードを打たずに編集が進む
  • Codex（OpenAIのコーディングモデル）とも連携可能で、モデルを問わない柔軟性
Claude Code → Palmie CLI → 自動編集実行
# 例: 「フィラーワード全カットして」
AIコーディングエージェントと動画編集ソフトが直結しているのが最大のポイント！

⑦ 実演ステップ1: 動画素材のインポート

  • 編集したい動画素材をPalmieにインポートするだけ
  • Premiere Pro風のタイムラインUIで直感的に操作できる
  • 複数の動画トラック・オーディオトラックにも対応
  • インポート直後にプレビューで内容確認が可能
Palmie本体のUIは直感的。Macのネイティブアプリらしいサクサク感が特徴です。

⑧ 実演ステップ2: Claude Codeで指示を出す

  • Claude Codeに自然言語で編集指示を入力するだけ
「フィラーワードを全カット、
無音部分もカット、
0.2秒の余白も許さず」
  • Palmie連携が自動起動して文字起こし→カット処理の流れが開始
  • 裏側でAIがタイムラインを解析して不要部分を特定し続ける
「0.2秒の余白も許さず」って指示、めちゃくちゃクリエイターの心情分かってる。

⑨ 音声入力で追加指示: 手を離さず編集

  • 音声でClaude Codeにリアルタイムに追加指示を送れる
  • 例: 「切れてないのにカットしてる箇所を修正して」
  • 修正指示を出すと、Palmieが即座に該当箇所を再調整
  • キーボード・マウスを使わずに音声だけで編集ワークフローを完結可能
ヘッドセット装着して声で指示するだけで、動画がどんどん仕上がっていく感覚。未来感があります。

⑩ マルチエージェントで品質を改善する

  • サブエージェントを使ってカット結果の検証を自動実行
  • 修正用エージェントがブラッシュアップを担当。初回カットの問題点を洗練させる
  • 検証→修正→再検証のループで品質を段階的に向上
  • 初回は50点程度のカット精度が、ループを経て実用レベルに到達
1発完璧じゃなくていい。修正ループを自動化すれば実用レベルに到達する。これが大事。

⑪ テロップ追加の指示例: 細かい条件もOK

  • Claude Codeで自然言語でテロップのルールを指定できる
「画面中央下に日本語テロップ、
1シーン最大10文字、
単語の途中で区切らない」
  • AIが文字起こしのタイミング情報を基に、条件に合致するテロップを自動生成
  • 白文字・黒縁取りの標準スタイルで即座に適用される
「単語の途中で区切らない」って。日本語テロップあるあるをAIがちゃんと考慮してくれます。

⑫ 再生して品質確認: 出来栄えをチェック

  • 自動編集が完了したらPalmie上で再生して品質を確認
  • カット位置の違和感やテロップのタイミングを視覚的にチェック
  • 気になる箇所があれば、再度Claude Codeに音声で指示して微調整
  • 満足したら書き出して完成。エクスポート形式も複数対応
AIがやった部分と自分で直す部分のバランスを自在にコントロールできます。

⑬ 品質評価: 初回50点→ループで向上

  • 初回カット は50点程度の精度。カット位置のズレや過剰カットが発生する
  • サブエージェント検証で問題箇所を特定してスコアアップ
  • 修正エージェントでブラッシュアップしてさらに精度改善
  • 修正ループを繰り返すことで実用レベルに到達する
完璧じゃなくていい。最初は荒削りで、AIループで磨いていく。このアプローチが新しい。

⑭ スキル化で再現性を高める

  • 良好な編集パターンをスキル（テンプレート）として保存可能
  • 次回から同じ品質のカットを再現性高く再実行できる
  • 「Vlog用カット」「インタビュー用」など用途別スキルの構築が期待される
  • スキルを共有すれば、コミュニティで編集ワークフローが横展開
Claude Codeの「スキル」機能と相性が抜群。一度作ったワークフローが資産になります。

⑮ 将来展望: BGM・効果音もAI自動化へ

  • BGMの自動選曲・自動配置までAIで自動化するワークフロー構築の可能性
  • 効果音の自動挿入: 転換ポイントにシンセサイザー音やトランジション効果を自動付与
  • 高画質化や色味の自動補正など、映像品質のAI向上も視野に入っている
  • カット以外の全工程をAIで自動化するエンドツーエンドのワークフローを目指す
今はカットとテロップだけど、将来的には動画制作の全工程がAIで回る時代が来るかも！

⑯ Claude Code連携が鍵の理由

  • Claude Codeはターミナル上で動く強力なAIコーディングエージェント
  • PalmieのCLIと組み合わせることで、動画編集をコードとして記述・実行できる
  • 「AIに編集を指示する」のではなく、「AIが編集プログラムを書いて実行する」
  • Codexなど他のAIコーディングモデルにも対応するモデル非依存な設計
AIが「編集手順をプログラミングして実行する」というアプローチが面白い。人間は監督、AIが編集者です。

⑰ 利用条件と注意点

  • 完全無料 オープンソースで誰でも利用可能
  • 現時点ではMacのみ対応。Windows版のリリースは未定
  • Claude Code自体の利用にはAnthropicのサブスクリプションが必要
  • Palmie本体はMacネイティブアプリとして直感的に操作できる
  • CLI操作に慣れていると、Claude Codeとの連携がよりスムーズ
オープンソースなのでコミュニティの貢献次第で機能がどんどん拡張されそう！

⑱ まとめ: AI駆動編集の新時代

  • Palmie + Claude Codeで動画編集の地味な作業をAIに丸投げできる
  • フィラーワード削除・無音カット・テロップ挿入を自動化
  • マルチエージェントの修正ループで品質を自動的に向上させる仕組み
  • 1発完璧でなくても、ループを回せば実用レベルに到達する
  • 今後はBGM・効果音・画質向上までAI自動化のワークフローが広がる可能性
クリエイターは「作ること」に集中できて、
「切ること・整えること」をAIに任せる。
