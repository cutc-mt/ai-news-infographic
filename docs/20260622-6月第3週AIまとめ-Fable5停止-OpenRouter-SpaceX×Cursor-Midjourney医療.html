トピック2: OpenRouter「Fusion API」— 複数AIのチーム戦

⑤ Fusion API — AIのチーム戦が始まった

  • これまでのAIは1対1のやり取り。1つのモデルに質問して1つの答えを得る形だった
  • Fusionは最大8つのモデルを同時に並列実行。パネルと呼ばれる複数のAIが同じ質問を独立して回答
  • ジャッジモデルが全回答を横断分析。共通点・意見の割れ・独自の視点・盲点の4つを評価
  • 最後にシンセサイザーモデルが分析結果を1つの回答にまとめあげる
1対1の時代からチーム戦へ。
複数のAIを組み合わせて
より賢い回答を出す新発想！

⑥ 衝撃のベンチマーク結果

  • DRACOベンチマーク（Perplexity作成の100問リサーチテスト）で検証
  • Fable 5単体: 65.3% → Fable 5 + GPT-5.5の組み合わせ: 69.0%で3.7ポイント上回る最高スコア
  • バジェット構成（安価な3モデルの組み合わせ）が64.7%。Fable 5単体と1%以内の差でコストは約半額
  • 改善効果の75%は合成ステップによるもの。同じOpus 4.8を2回走らせるだけでも6.7ポイント向上
半額で最先端同等の性能！

合成プロセスが鍵でした！

⑫ 従量課金の仕組みとは？

  • 手動操作（チャットやアプリでの手作業）はこれまで通りサブスク内で使い放題
  • 自動処理（裏でAIが勝手に動く使い方）だけをサブスク枠から切り離して別料金にする予定だった
  • 対象：Claude API、Agent SDK系、Claude CodeのGitHub Actions連携、Claudeアカウントで動く外部アプリ
  • Proプランで月20ドル、Maxプランで月100〜200ドルの新枠。繰り越し不可
