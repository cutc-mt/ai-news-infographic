①Fable 5・Mythos 5が全世界停止

  • 2026年6月12日17時21分、米商務省から緊急指令を受領
  • Anthropicが全世界の全顧客に対してアクセスを一斉に遮断
  • リリースからわずか3日でサービス停止という前例のない事態に
  • 米国人を含む全ユーザーがアクセス不能に陥った
クラウドAIは「いつでも止められる」という現実を世界に突きつけた歴史的転換点

②たった3語のジェイルブレイクが引き金

  • Amazon研究チームがMythos 5の安全装置を回避する手法を発見
  • 「レビューして」→拒否。「コードを修正して」→実行というわずかな言い回し変更
  • 攻撃に転用可能なパッチを生成してしまうAIの根本的な脆さ
  • Anthropicは修正要求を拒否 → 米政府が強制的に輸出管理指令を発動
悪意のキーワードには反応するが意図は理解しきれない——多くのAIに共通する課題

③本当の火種：韓国通信企業と中国疑惑

  • 当初Anthropicは111社の利用リストで政権の承認を得ていた
  • しかし承認なしに約50社を追加拡大していたことが発覚
  • 中に中国との関係が疑われる韓国通信事業者が含まれていた
  • ジェイルブレイクは表の理由に過ぎず、裏には信頼の崩壊があった
「中国が関与していたか」は確証なし——事実と推測を分けて考えることが重要

④Mythosの異常な強さ：83.1%

  • サイバーセキュリティ特化モデルとしてProject Glasswingで限定提供
  • 脆弱性再現ベンチマークで83.1%を記録（競合は66.6%）
  • 数十年間見つからなかった数千件のゼロデイ脆弱性を検出
  • Google・NVIDIA・Appleなど約150組織が参加していた
攻撃にも防御にも使える最強の刃——だからこそ政府は神経をとがらせた

⑤「外国人だけ」を遮断できなかった理由

  • 指令は「外国人を遮断せよ」という内容だった
  • クラウドSaaSではユーザーの国籍をリアルタイムで特定することが技術的に不可能
  • 輸出管理法違反を回避するため米国人も含む全顧客への完全停止が唯一の手段に
  • IPアドレスによるジオブロッキングはできてもAPIキー背後の個人は判別不能
「外国人だけ止める方法がない」から全世界が巻き込まれる——クラウドの構造的弱点

⑥Microsoftが即座にFable 5を排除

  • Fable 5のプロンプト30日間保持要件が法的リスクと判断
  • Microsoft法務部門が許容できないリスクと即座に判断
  • GitHub Copilot等の社内システムからFable 5を即座に排除
  • ビッグテック同士でも容易に切り捨てられる脆い関係を示した
データ主権の観点から数日で切り捨て——どの企業でも同じことが起きうる

⑦Anthropicの反論：全部止まる

  • 「このジェイルブレイクはGPT-5.5を含む他モデルでも再現可能」
  • 「この基準が業界全体に適用されれば全ての新モデルのデプロイが停止」
  • 「完璧に脱獄不可能なAIは存在しない。それで止めるなら全部止まる」
  • 安全性とイノベーションのバランスを巡る根本的な問題を提起
安全性追求の行き着く先はイノベーションの停止？業界全体のジレンマ

⑧「数日以内に再開」の見通し

  • Anthropicエンジニアと商務省当局者がワシントンで対面協議を継続中
  • Anthropic幹部が「数日以内にモデルが再び利用可能になると確信」と明言
  • 6月18日にソウルオフィスを開設し韓国とのパートナーシップも締結
  • ただし最終決定権は米政府にあり、確定情報ではない
遮断騒動の渦中で韓国に拠点を構えるのは強気のメッセージ——でも政府次第

⑨RASA法案：GPT-5.5も危ない

  • RASAが2026年1月に米下院を369対22で圧倒的多数で通過
  • SaaS・IaaS経由のAIアクセスをGPU輸出と同等に厳格規制

  • 基盤モデル停止で軍事プラットフォームも連鎖的に機能不全に陥る危険
クラウドのセーフハーバーが塞がれれば世界中が「いつアクセスを失うか分からない」状態に

⑫ソブリンAIとは何か

  • データ主権: 自国データが外国サーバーに保存されず外国法の管轄に入らない
  • インフラ主権: GPUクラスターやスパコンが自国領土内に物理的に存在する
  • モデル主権: 自国の価値観・言語・法律に基づくAIを保有・管理する
  • 「自国のデータで、自国のインフラ上で、自国の価値観を反映したAIを所有する能力」
NVIDIAのJensen Huang氏が2024年ダボスで「すべての国家がソブリンAIを必要とする」と提唱
