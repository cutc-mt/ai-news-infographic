② Fuguの仕組み: GPT・Claude・Geminiを束ねる

  • ユーザーがFuguに指示を出すと、裏側でGPT・Claude・Geminiの3つが動く仕組み
  • Fuguが「Conductor（指揮者）」として役割分担を自動決定。構成・実行・検証を各モデルに振り分ける
  • ICLR 2026採択の「TRINITY」フレームワークが技術の中核。Thinker・Worker・Verifierの3役を動的に割り当て
  • 7Bパラメータの小型モデルが強化学習でコーディネーション戦略を学習。再帰的に自分自身も呼び出し可能
例えばClaudeが構成を作って、
Codexが実行役になり、
Geminiがビジョン機能を使う —
いい感じに混ざって回答を出す

③ Thinker / Worker / Verifier の3役分担

  • Thinker（思考担当）: タスク全体の方針を立て、問題を構造化する。難問では複数モデルを並列起用
  • Worker（実行担当）: Thinkerが定義したサブタスクを実際に解く。コード生成・計算・データ抽出などを担当
  • Verifier（検証担当）: Workerの出力を確認し、誤りや欠落を指摘。別モデルが担当することで幻覚を抑制
  • 人が設計したワークフローではなく、学習したコーディネーターが動的に決めるのがTRINITYの肝

⑤ 料金体系: サブスク3段 + 従量課金

  • Standard $20/月: 軽量な日常使用向け
  • Pro $100/月: Standardの10倍の利用枠
  • Max $200/月: Standardの20倍、長尺・連続ワークロード向け
  • 従量課金API: Fugu Ultraは入力$5/出力$30（272K超で$10/$45に段階制）
  • キャンペーン中: 7月末まで登録で2ヶ月間無料。単一エージェントなら基盤モデルの標準レートのみ

⑥ 使い方: Codex CLIからFuguを起動

  • console.sakana.aiでGoogleアカウントまたはメールでサインアップ
  • クレジットカードを登録してAPIキーを取得
  • ドキュメントの「CodexでFuguを使う方法」に記載のコマンドをターミナルで実行
  • APIキーを入力（見た目は変わらないがしっかり入力される）、インストール完了
  • codex-fugu コマンドで起動。Codex上のスキルも全て使える
npx @openai/codex@latest --provider openai-compatible \
  --base-url https://api.sakana.ai/v1 \
  codex-fugu "指示内容"

⑦ 最大の強み: 日本語がめちゃくちゃ上手い

  • ケト氏の実感：「Claudeが日本語が上手いと言われてきたけど、そんなの日の目にならないぐらい上手い」
  • 資料作成やスライド資料で違和感のない自然な日本語を生成。海外製AI特有の不自然な言い回しが減る
  • ヘッドラインのタイトル付けや手書きルールのような不自然な日本語表現が少ない印象
  • 理由はGPT・Claude・Geminiに加えてSakana AI独自の国産LLMも混ぜてまとめてくれるため
OpenRouterは国産モデルなし。
日本語タスクならFugu有利。

⑧ データプライバシーとエンタープライズ向け設定

  • 学習データ利用のオプトアウトが簡単に設定可能。エンタープライズ導入に好印象
  • 標準版Fuguではプロバイダー・モデルの除外設定が可能。「OpenAIモデルは使わない」等の構成ができる
  • Fugu Ultraは品質維持のためエージェントプールが固定。除外設定はできない仕様
  • 注意点: EU/EEAはGDPR対応中で未提供。欧州案件では採用不可

⑨ ウェブサイト制作: Fuguハイ vs Fugu Ultra

  • 「最高におしゃれでインパクトの強いウェブサイトを作って」のプロンプトで検証
  • Fuguハイ: 謎のオブジェクトが右に配置されるが、テキストレイアウトは微妙。30分程度で完成
  • Fugu Ultra: 背景アニメーション、変なカーソル追従、コンテンツ整理が優秀。HTMLのテンプレートとして違和感なし
  • モデルを変えるだけで全然印象が変わる。Fugu Ultraを使うのが正解の印象
Fuguは謎の3Dオブジェクトが

⑩ ゲーム制作検証: バグ修正は苦手

  • レースゲームは3DアセットAIで車がリアルだが、フィールドは微妙でバグ多数
  • Fableなら「玉を入れる」ことを理解するが、Fuguは見た目の修正やバグ修正に弱い印象

⑮ 記事執筆検証: 日本語のリライトで強み発揮

  • YouTube動画のURLを渡して文字起こし→記事の原稿を依頼する検証を実施
  • 日本語の使い方が自然で、海外製AI特有の不自然な表現が少ないと評価
  • 記事の原稿作成にはFuguを使いたくなる水準。ただし料金が高すぎるため日常使いは躊躇
