  • 複雑な制約条件もより確実に扱えるようになった。頼み事がより正確に反映される

有料ユーザーに即日ロールアウト、
無料ユーザーにも翌日提供。
一番使われているモデルだからこそ影響大。

② OpenAI初の自社AIチップ「Jalapeño」誕生

  • OpenAIがゼロから設計した初のAIチップを発表。名称は「Jalapeño」
  • Broadcomと協力して量産化。半導体業界の大手と連携したスピード感のある開発
  • ChatGPT、Codex、API、今後のエージェント製品を動かすLLM特化型
  • 製品→モデル→インフラまで全スタックを自社で揃える戦略の重要な一歩
チップはAI経済の土台。
自社製によりインテリジェンスの
スケールとアクセス拡大を目指す。

③ Jalapeñoがもたらす業界への影響

  • 発表から600万回以上閲覧される大反響。X（Twitter）でトレンドに
  • NVIDIAへの依存を減らす狙い。インフラコストの大幅削減が期待される
  • 「Jalapeño」の名前はハバネロに次ぐ辛さ — OpenAIらしいユーモアのある命名
  • GoogleのTPU、AmazonのTrainiumなど、各社の自社チップ競争に新たなプレイヤー参戦

垂直統合で競争力を高める

④ メルカリがChatGPT公式アプリで参戦

  • Apps in ChatGPTでメルカリの公式アプリ提供を6月23日より開始
  • ChatGPTの会話の中で商品を自然に検索できる。アプリ切り替えが不要に
  • 出品する商品の説明文の下書きもAIが自動生成。出品の手間を大幅削減
  • 日本企業がChatGPTのエコシステムに公式参加した先駆け的事例として注目
「AIプラットフォーム」に
企業アプリが集まる未来の
さきがけと言える取り組み。

⑤ Notionから直接Cursorにタスクを委任できるように

  • Notion上の仕様書（Spec）をそのままCursorに送ってタスク実行が可能に
  • Cursor SDK上に構築されているため、クラウドエージェントはCursorと同じモデル・ランタイムで動作
  • タスクをCursorに割り当てるとPRを作成してチーム全体でレビュー可能に
  • 企画・設計ドキュメントと実装のギャップを埋める革新的なワークフロー
ドキュメント書いたら
そのままコードになる世界。
非エンジニアでも実装に関与可能！

⑥ Notion→Cursorが変える開発現場のリアル

  • これまで企画書を書いてからチケットを作り、エンジニアに割り振るという手順が必要だった
  • Notionの仕様書に@CursorとメンションするだけでAIがコードを生成
  • PMD（プロダクトマネージャー）とエンジニアのコミュニケーションが劇的に効率化
  • 27万回以上閲覧されるなど開発者コミュニティで大きな話題に
「仕様書→コード」の直結化。
AIコーディングツールと
既存ツールの統合が加速中。

⑦ Cursorでオープンモデル「GLM 5.2」が使えるように

  • 中国のZhipu AIが開発したGLM 5.2がCursorで利用可能に
  • Fireworks AIとのパートナーシップで提供。オープンモデルの活用範囲が拡大
  • Cursor社の自社ベンチマークで良好な結果を記録。実用的な性能を確認
  • ClaudeやGPTだけでなく、オープンモデルの選択肢も増えて開発者の自由度が向上
Claude Miller（Cursor CEO）が
自ら発表。オープンモデル推しの
姿勢を明確に示しています。

⑧ Genspark Design登場 — Claude Opus 4.7搭載のデザインAI

  • Claude Opus 4.7を搭載した次世代のデザイン・創作AI
  • デザインの背景がなくても、ラフなアイデアからプロ品質のデザインを生成
  • UIプロトタイプ、動画、HTMLアニメーション、ポスターなど1つのツールで全て対応
  • Figmaファイルのアップロードに対応。デザインをコードに変換する機能も搭載
「モックアップするだけでなく
出荷できる」がコンセプト。
デザイン→コードの一体化。

⑨ Genspark Designの注目機能を深掘り

  • 独自のデザインシステムを構築可能。Figmaファイルを読み込んで再利用できる
  • チームでデザインを共有して、プロジェクト横断で一貫したブランドイメージを維持
  • Genspark Code上に構築されているため、アプリやウェブデザインをワンクリックでコード化
  • 「本番環境にデプロイできるレベル」まで出力品質を追求している点が従来ツールと一線を画す
Canva + Figma + コード生成
を1つにまとめたような
強力なツールです。

⑩ Gemini 3.5 FlashにComputer Useサポート導入

  • ブラウザ、モバイル、デスクトップなど複数のプラットフォームを横断して推論・実行

⑪ Qwen-AgentWorld — 7つの環境をシミュレートする革新的モデル

  • AlibabaのQwen-AgentWorldは単一言語モデルで7つのエージェント環境をシミュレート
  • 対応環境：MCP・Search・Terminal・SWE・Web・OS・Android
  • 環境モデリングが最初の学習目標。事後の適応ではなく、最初から環境を理解して学習
  • 論文はarXivに公開済み。オープンソースでGitHubとHuggingFaceから入手可能
LLMは環境で動くように訓練するのが
普通だけど、環境そのものをモデリング
する発想が斬新です。

⑫ Qwen-AgentWorldの驚異的なベンチマーク結果

  • AgentWorldBenchでClaude Opus 4.8とGPT-5.4を上回る性能を達成
  • 環境を予測する学習（LWM）がエージェントの能力を飛躍的に向上させることを実証
  • エージェント専用の追加訓練なしでも、予測知識がそのまま転移する驚きの結果
  • AIエージェントを仮想環境で強化学習できるSim RLが実環境訓練を上回る可能性を示唆
「環境を理解する」ことで
「環境で行動する」能力も上がる。
