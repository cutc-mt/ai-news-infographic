  • 2026年は自律型AIエージェントが爆発的に普及した年

5つのプロジェクトはすべて異なる領域（ホスティング・協調・デザイン・開発・コーディング）をカバーしています。

  • 異なるツール・サービス間の連携がシームレスに

Agent 37 Cloud

⑤ Agent 37 Cloud: 永続型AIエージェントのプラットフォーム

  • 「常に稼働する」AIエージェントをクラウド上にホスティングするサービス
  • エージェントのファイル、メモリ、接続されたツールがセッション間で保持される
  • 明示的に削除するまで消えない、真の「永続性」を実現
  • 1000以上のアプリケーションとの統合、カスタムスキル、シェルコマンド実行に対応
従来の「使うたびに初期化される」AIとは違い、ずっと覚えていてくれるのです。まるでデジタルアシスタントがずっと待機しているような感覚！

⑥ Agent 37 Cloudの技術的特徴

  • カーネルレベルの分離 セキュリティを強固に保つ隔離技術を採用
  • ホワイトラベル対応 ブランド名を入れたカスタムデプロイが可能
  • ゼロコンフィグネットワーキング 面倒な設定なしでネットワーク接続が完了
  • API駆動 API経由で各顧客に専用のエージェントインスタンスを提供
  • HermesやOpenClawなど様々なAIエージェントに対応
インフラの複雑さをすべて抽象化してくれるので、開発者はエージェントの構築に集中できます。

⑦ Agent 37 Cloudのユースケース

  • 破損したドキュメントの自動修復
  • プロジェクト管理の自動化。タスクの割り当てから進捗追跡まで
  • YouTubeコンテンツの自動生成。企画からスクリプト、編集まで
  • 企業向けのカスタムAIソリューション構築
文書の修復、プロジェクト管理、動画制作まで。AIエージェントがまるで従業員のように働いてくれます。
AgentX

⑧ AgentX: マルチエージェントAI協調フレームワーク

  • 複数のAIエージェントが協調して作業するためのフレームワーク
  • UC Berkeleyの研究機関RDIが開発。学術的バックグラウンドが信頼の証
  • 高度な思考、リサーチ、ツール使用の能力を持つ洗練されたAIエージェントを構築可能
  • 開発者向けプラットフォームとして、エージェントの構築・評価・デプロイを統合支援
Berkeleyの研究者が本気で取り組むプロジェクト。学術的な正しさと実用性の両方を追求しています。

⑨ AgentX AgentBeatsコンペティション

  • Agentic AI Summit @ UC Berkeleyで開催される大型コンペティション
  • フェーズ1 エージェントAI用のベンチマークを構築・改善
  • フェーズ2 そのベンチマークで最高のAIエージェントを開発
  • 単一ドメインではなく汎用的な能力を評価する画期的なコンペ
  • モデル主権と移植性も評価の重要な要素
単一の課題ではなく「色んなことができるAI」を競い合う。まさにAGIに近づくための大会ですね！

⑩ AgentXの技術的アプローチ

  • 設定可能な割引率 交渉パラメータを柔軟に調整可能
  • 交渉ラウンド制御 最大交渉回数やゲーム回数を設定
  • Maris プライバシーを強化したエージェント協調開発パラダイム
  • FieldWorkArena エージェントAIの厳密な評価ベンチマーク
  • A2Aインターフェースに対応。ツール使用・メモリ・タスク遂行を統合
複数のAIエージェントが交渉や協調を行う研究は、今後の社会実装の基盤になりそうです。
Alai 2.0

⑪ Alai 2.0: AIデザインパートナー

  • テキストプロンプト1つでブランドに合ったプレゼンやデザインが自動生成
  • プレゼン資料、ソーシャルメディア投稿、広告、インフォグラフィックなど多彩なフォーマットに対応
  • CanvaやGammaなどの既存のデザインツールの強力な対抗馬
  • ブランドのカラー・フォント・スタイルを記憶して一貫性を維持
デザイナーでなくても、プロ品質のビジュアルコンテンツが作れるようになります。クリエイティブの民主化ですね！

⑫ Alai 2.0の主要機能

  • ブランドシステム 色やテーマ、タイポグラフィを自動キャプチャ
  • 複数ブランド管理 プロジェクトごとに異なるブランド体系を切り替え可能
  • 1つのスライドに対して複数のデザイン案をAIが生成。選んで洗練させるフロー
  • .pptxエクスポート。Google SlidesやPowerPointでそのまま編集可能
  • プレゼン、SNS投稿、広告、カルーセルなどマルチキャンバスサイズに対応
「このテーマでスライドを作って」の1行で数十枚の資料が完成。その後の微調整も直感的です。
MeshPilot

⑬ MeshPilot: 開発者向けAI生産性プラットフォーム

  • 自律型CLIエージェントと協働できる開発環境
  • AIエージェントが永続的なターミナルセッションでコマンド実行・コード記述・ファイル管理
  • ビジュアルカンバス カンバンボード形式のタスク管理
  • 開発者がエージェントの作業をリアルタイムで監視し、必要に応じて介入可能
複数のAIコーディングエージェントを同時並行で実行できるのが最大の魅力です。

⑭ MeshPilotの独自機能

  • MeshUtility Dictation 音声入力と「MeshPrompt書き換え」機能を統合
  • ローカルとクラウドのプライバシー制御可能な文字起こしに対応
  • プッシュトークまたはトグルモードでハンズフリー入力が可能
  • MeshMemory セマンティックメモリ層でコードベースの文脈を自動保存
  • Windows、macOS、Linuxに対応。ローカルファーストでデータプライバシーを確保
音声でコードを指示して、AIが実行してくれる。未来の開発フローがここにあります。
Backgrind

⑮ Backgrind: AIコーディング用の常時最前面オーバーレイ

  • 常に最前面に表示されるオーバーレイウィンドウ
  • 開発者がAIにタスクを送り、入力が必要な時だけ通知を受け取る設計
  • エディタ、ブラウザ、フルスクリーンゲームの上でも動作
  • 音声対話に対応。話しかけるだけでAIとやり取り可能
ゲームをしながらAIにコーディングさせることも可能！DiscordやOBSのオーバーレイと同じ仕組みです。

⑯ Backgrindのマルチエージェント対応

  • Claude CodeやCursor CLIなど既存のCLIエージェントをそのまま統合
  • 同じログイン認証・履歴・フックを引き継いで利用可能
