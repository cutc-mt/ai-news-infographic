① Obsidianとは？：自分専用のセカンドブレインを作れるアプリ

  • 世界中でじわじわと注目を集めている無料の知識管理アプリ。ノートを書きためていくだけで自分専用のWikipediaのようなデータベースを構築できる
  • 単なるメモ帳ではなく、セカンドブレインを作れるツールとして多くの人に利用されている
  • Windows、Mac、Linuxに対応。iOSやAndroidのモバイルアプリも用意されている
  • 最近はAIとの連携機能も充実してきており、Claudianプラグインを使うとClaude CodeやCodexが利用可能に
知識管理とアイデア整理に強いアプリ。

③ ローカル保存：データを自分の手元に置けるのが最大の強み

  • ノートはすべてマークダウンファイルとしてローカルに保存される
  • 研究や執筆、日記などのデータをクラウド任せにせず、自分の手元に持っておける
  • ネットが切れても使えるのが大きなメリット。クラウドサービスの障害も影響しない
  • 有料プランで複数デバイス同期。無料でもGitHubやGoogleドライブで同期可能
データの所有権は自分にある。
この安心感がObsidianの人気の秘密。

④ Obsidianの主要機能：ノート作成からプロジェクト管理まで

  • マークダウンエディター — 文字を打つだけで自動で綺麗に見た目が整うノートを作成可能
  • 内部リンク — ノート同士をつないでアイデアの網の目を作れる
  • キャンバス機能 — メモや画像、PDF、動画、音声、ウェブページを並べて企画ボード風に整理
  • データベース — プロジェクト管理向けのデータベースを作成できる
プラグインでさらに機能を追加。

⑤ インストール方法：公式サイトから手軽に導入

  • obsidian.mdの公式サイトにアクセス。日本語でない場合は地球儀ボタンから切り替え可能
  • 「Obsidianを入手」ボタンを押してインストーラーをダウンロード。立ち上げてインストールを完了させる
  • 初回起動時に保管庫（Vault）を作成する。クイックスタートで自動生成、または名前と保存場所を指定して作成
  • 保管庫の中に「ようこそ」ノートが生成される。このフォルダにノートや画像、動画が保存されていく
保管庫は複数作成可能。

⑥ 画面構成：3つの要素でシンプルに organized

  • リボン — 画面左端にアイコンが並んだ細いバー。クイックスイッチャーやグラフビューなどを起動可能
  • 左サイドバー — 新規ノートを立ち上げる場所。ここからエディター画面にノートを表示
  • エディター画面 — ノートを書き込む場所。複数ノートの表示や、右/下への分割も自由自在
  • 左下の歯車アイコンでエディター設定や外観の変更ができる。テーマも自由にカスタマイズ可能

⑦ 基本操作：ノート作成からプロパティ設定まで

  • 左サイドバーの上部の新規ノートボタンを押すとエディター画面に新しいノートが開く
  • ノートの最上部にタイトルを入力。その下にプロパティ（YAMLフロントマター）を設定可能
  • プロパティにはテキストやリンク、日付、チェックボックスなど多彩な項目を追加できる
  • マークダウン記法で見出しや内部リンク、外部URLリンクを活用。公式ヘルプに詳細な書式ガイドあり

⑧ グラフビューとキャンバス：ノートの繋がりを可視化

  • グラフビュー — 保管庫のノート間の関係を視覚化。内部リンクの繋がりを一目で確認可能
  • キャンバス機能 — 新規キャンバスを作成して、メモや画像を自由に配置してアイデアを整理
  • キャンバスはコアプラグインとして最初からインストール済み。設定画面のコアプラグインでオン・オフ可能
  • その他にも多数のコアプラグインが用意されており、表やカレンダー、タスク管理などの機能を追加できる

⑨ Claudianとは？：Obsidian内でAIエージェントを動かすプラグイン

  • Obsidianにはコミュニティプラグインというサードパーティ製の拡張機能が存在。6月下旬時点で5000を超えるプラグインが利用可能
  • その中で最も人気のAIプラグインが「Claudian」。ベータ版を経て5月にコミュニティプラグインとして正式公開
  • Claude CodeやCodexなどのAIエージェントをObsidian内で立ち上げて、チャット感覚で操作できる
  • 対応AIエージェント：Claude Code、Codexのほか、OpenCodeとPI（π）も利用可能（6月下旬時点）

⑪ Claudianのインストール：設定画面から簡単に導入

  • Obsidian画面の左下にある歯車ボタンを押し、コミュニティプラグインのカテゴリーへ移動
  • 「コミュニティプラグインを有効化」ボタンを押し、閲覧ボタンから検索画面を開く
  • 検索欄に「Claudian」と入力し、説明部分を押してからインストールボタンを押す
  • 有効化ボタンを押すと設定画面にClaudianが表示される。GeneralタブのLanguageを日本語に変更可能
リボンの左側にアイコンが表示。
押すと右サイドバーが立ち上がります。

⑫ Claude Codeの設定：モデル・深度・セーフモードの使い分け

  • 右サイドバーのClaudianで、フォームの左下からモデルと考える深さを設定可能
  • セーフモード — 作業のたびにユーザーに許可を求める安全なモード
  • Yoloモード — Claude Codeが自分の判断で動き、ほぼ承認不要。ただし意図しない行動のリスクあり
  • Yoloモードを使う際は注意が必要。Claudianがユーザーの意図と違う行動を取る可能性がある
Yoloモードは便利ですが
リスクを理解した上で使いましょう。
安心なのはセーフモードです。

⑬ Codexの有効化：トグルをオンにするだけでGPT系モデルが使える

  • 設定画面のCodexタブにある「Enable Codex Provider」トグルをオンにするだけ
  • Claudianのモデル選択部分でGPT系のモデルが選べるようになる
  • Claude CodeもCodexも各エージェントのサブスクリプション範囲内で利用可能。追加料金なし
  • エラーが出る場合は設定画面でCLIのパスを入力すると解決することが多い

⑭ テキスト処理：リライト・要約・翻訳をノート内で完結

  • 特定のノートを立ち上げた状態でClaudianを開くと、そのノートが自動的にClaudianにセットされる
  • ノート内の文章を選択してリライトや要約、翻訳を依頼。チャットに処理結果が表示される
  • チャットで指示してリライト内容をノートに反映させたり、リライト前に戻したりできる
  • ホットキーを登録しておけば、ノート内のショートカットキーでClaudianを直接立ち上げ可能

⑮ タイトル一括変更：大量のノート名をAIが自動で整理

  • 大量にノートがある場合、ノートの内容を元にしてすべてのノートのタイトルを一括で変更できる
  • AIが各ノートの内容を読み取って、内容に合ったタイトルを自動的に提案してくれる
  • 手作業で1つずつ変更する手間を大幅に削減。数十個のノートでも一瞬で処理
  • 変更内容を確認してから反映できるので、誤った変更の心配もない

⑯ スキル機能：専門知識をAIエージェントに付与する

  • スキルとはAIエージェントに特定の専門知識や新たな機能を付与する仕組み
  • 「ClaudianでObsidian Skillsを導入してください」と入力して送信するだけで公式スキルがインストールされる
  • チャット欄にスラッシュ（/）を入力してスキルを呼び出す。例えば「/obsidian-skills」と打つ
  • Obsidian Skillsを使うと、保管庫内の全ノートを参考にしたデータベースの自動生成が可能

⑰ データベース自動生成：全ノートの情報を一覧表示

  • Claudianのチャットで「保管庫内の全ノートを参考にデータベースを作ってください」と依頼するだけ
  • AIが全ノートの情報を読み取って、タイトルや作成日、更新日、サイズなどが表示されるデータベースを自動生成
  • 生成されたデータベースはテキストの指示で表示項目の追加や削除ができる。必要な情報だけを絞り込める
  • 大量のノートを一覧で把握できるため、情報の整理や活用が劇的にやりやすくなる
