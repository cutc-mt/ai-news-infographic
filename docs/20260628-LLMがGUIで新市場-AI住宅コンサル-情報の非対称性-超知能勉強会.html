2026年6月24日、超知能＆近未来社会勉強会第122回が開催。「リアルかフェイクか」という番組形式で、AI時代ならではの新ビジネスアイデアを参加者が判定した。

  • 開催日: 2026年6月24日（水）
  • 出演者: うんさん、池田さん、かさん
  • 形式: 「リアルかフェイクか」判定

出演者がリアルかフェイクかを判定！番組パクリのスタイルでお馴染み👀

②テーマ1の核心: LLMがGUIになると何が変わる

パソコンがCUI（コマンドライン）からGUI（マウス操作）に進化したように、今LLMがチャット画面中心からGUIへと変わりつつある。これがビジネスをどう変えるのか。

  • CUI時代: テキスト入力で操作
  • GUI時代: マウスと画面で直感的操作
  • 現在のLLM: チャット画面が中心
  • 未来のLLM: GUIで直感的に操作
  • ロボットにはGUIが必須
CUI→GUIの大革命を、もう一度AIが起こそうとしている！🔄

③AI住宅ライブコンサルティング: 夢の注文住宅体験

図面に「ウッドデッキを広くしたい」「窓を大きくしたい」と会話しながらリアルタイムにビフォーアフターが変わり、見積もりも自動計算されるという構想。

  • 図面に直接書き込んで指示
  • ビフォーアフターを即座に表示
  • 見積もりも自動で更新
  • 会話しながら設計を同時進行
  • 注文住宅のあり方がガラッと変化
「ここ広げたい」→ 図面が変わって見積もりも出る！夢のような体験🏠

④DraftAI: YC出身の住宅デザインAI

アメリカのY Combinator出身スタートアップDraftAIが、1ヶ月で12万人のユーザーが32万件の住宅デザインを生成。図面をボタン操作で変更し、3Dビューで確認できる。

  • Y Combinator出身のスタートアップ
  • 1ヶ月で12万人が利用
  • 32万件のデザインが生成された
  • 図面の操作はボタン一つ
  • 3Dビューで設計をリアルに確認
12万人が1ヶ月で使ってる！需要は確実にあるよね⭐

⑤Claude Code Designの登場: テキストで即修正

Claude Code Designはテキストでチャット形式のUI修正ツール。「この文字を大きくして」と打ち込むと、即座に画面のテキストが変更される。

  • チャット形式で修正を指示
  • 「リアルって字を大きくして」→ 即変更
  • 小さな修正窓で操作
  • 発表: 3日ほど前の最新機能
  • でもチャット形式なのでGUIが欲しい声も
チャットで文字変更できるのはすごいけど…GUIでやりたいよね？✨

⑥Google Magic Pointer: ポインタで囲んでAI修正

Googleが発表したMagic Pointerは、マウスで写真やデザインの一部を囲むだけでAIに修正を指示できる機能。GUIならではの直感的な操作を実現している。

  • マウスで範囲を囲んで指示
  • 写真の一部を簡単に変更可能
  • デザイン修正をGUIで直感的に
  • Claude Code Designに似た概念
  • Google公式が既に実現済み
囲むだけでAIが修正！Googleがもう実現してるってことね🎯

⑦Alun: ウェブサイトの修正指示ツール

日本発のツールAlunは、ウェブサイトの画像を取り込んで修正箇所を言葉で指示して共有するサービス。ソースコード不要でデザイナーとクライアントの間の意思疎通を改善する。

  • ウェブサイトの画像を取り込む
  • 修正箇所を言葉で記入
  • デザイナーに修正内容を共有
  • ソースコードは不要
  • 月額2,000〜3,000円で利用
  • ユーザー1万〜2万人
「ここの色を変えて」を共有できるだけでも便利！📝

⑧会議中のリアルタイム修正: 変わるビジネス会議

クライアントと会議しながら「ここをこう変えて」と言えばAIが即座に修正を反映する未来。Web制作・建設業界などで、顧客対応のあり方が大きく変わる。

  • 会議中にリアルタイム修正
  • Web制作会社で即活用可能
  • 建設・建築業界にも応用
  • 2〜3秒で反映が条件
  • プロジェクターで画面を共有
「ここ変えて」→ ピン！出た！会議が劇的に速くなる⚡

⑨GUI+音声の威力: 喋りながら指さすのが最強

池田さんが指摘したように、音声とポインティングの組み合わせが最強の入力インターフェースになる。「ここをこうして」と言いながら指さすのが一番自然で効率的。

  • 音声で指示を入力
  • マウスで対象をポインティング
  • キーボード入力はもはや不要に
  • 口頭筆記と同じ感覚
  • 30日分の資料修正も音声で一気に
「手が検索になった」もうキーボードいらない時代が来る！🔥

⑩落とし穴: 「永遠に決まらない」会議の危険性

リアルタイム修正ができると、「まだ変えられるから」と永遠に決定が先延ばしになるリスクがある。建設業界の納期遅れや、意思決定の属人化が問題に。

  • 「ここも変えたい」が永遠に続く
  • 決定が先延ばしになる
  • 建設業界はフィジカルギアなので遅延致命的
  • 営業のスキルで「締める」必要がある
  • 納期が守れなくなるリスク
変更できすぎると逆に困る…人間の「決める力」が重要！😤

⑪決定の心理: 「悩みたい」をショートカットできない

人間には「過程を経て自分で決めた」という納得感が必要。AIが最短ルートで答えを出せても、人はあえて悩みながら決める。この心理はAIでは代替できない。

  • 人間は過程を経て納得したい生き物
  • 会議の最初55分はダラダラ、最後5分で決まる
  • 締め切りまでダラダラしたい心理
  • この心理をショートカットできない
  • クライアントの言葉を集約する難しさ
「最初から答えわかってるのに悩みたい」これ人間の本質だね🧠

⑬テーマ2の核心: 情報の非対称性が崩れる

第2テーマは「AIで情報の非対称性が崩れると市場チャンスがある」。企業側が有利な情報格差がAIによって解消されると、既存のビジネスモデルが根底から変わるという主張。

  • 現状: 情報の非対称性で企業側が有利
  • AIが情報の透明化を促進
  • 2段否定のロジック: 「崩れるとチャンス」
  • 代表例: 保険業界
  • 不動産業界も非対称の典型
企業だけが知ってた情報がAIで全部バレる！それがビジネスチャンス⚖️

⑭生命保険の不透明性: 「なぜこの金額？」

生命保険は個人側がなぜその金額なのかを知る術がない。保険会社側がリードしてパターンを提示するが、計算根拠やマージンはほぼ開示されない。極めて不透明な市場。

  • 「なぜ月3,820円なの？」と聞いても答えない
  • 保険会社側がプランをリード
  • 計算根拠はほぼ不明
  • マージンも非公開
  • 説明はある程度あるが曖昧
保険料の内訳、誰も知らないよね…AIが計算できるようになったら大変！😖

⑮AIによる保険料の透明化: マージンがバレる

「1000万円の保障で月3,820円」と聞けば、AIが即座に計算してコストとマージンを明示できる。アクチュアリーの計算は確率に基づくもので、基本は同計算額に乗っているだけ。

  • AIが保険料の妥当性を即計算
  • コストとマージンの内訳が明示される
  • アクチュアリーの計算は確率ベース
  • 再保険市場の発展（イギリス等）
  • つまびらかにすれば分かりやすいだけ
「実はコストはこれだけでマージンがこれだけ」AIが暴露！👀

⑯逆指値保険: 顧客主導の入札マーケット

AIに相談して「入院1日5,000円、死亡1000万円保障」を提示すると、保険会社に入札させる仕組みが成立する可能性。大口契約では既にオーダーメイド入札が行われている。

  • 顧客が希望条件を提示
  • 保険会社が入札で競争
  • 大口契約では既に実践済み
  • 大企業や行政向けにオーダーメイド対応
  • 間に入るプラットフォームサービスが必要
保険会社が競り合ってくれる！プラットフォーム作ればチャンスかも🏪

⑰ポリマーケットの台頭: 恋愛も入札する時代

アメリカで人気のポリマーケットでは「大学合格したら100万円払う、不合格なら50万円もらえる」「告白成功で1万円払う、振られれば10万円もらえる」のような予測市場が成立している。

  • 受験結果の予測市場
  • 恋愛の告白成功を賭ける
  • Polymarket（アメリカ）が人気
  • 情報透明化トレンドの最先端
  • 予測市場が新たな投資ジャンルに
振られたら10万円もらえる…リスクヘッジじゃん！😂
