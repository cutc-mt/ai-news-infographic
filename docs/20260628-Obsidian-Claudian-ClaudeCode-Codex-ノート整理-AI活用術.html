第1章：Obsidianの概要

①Obsidianとは： 自分専用のセカンドブレインを作れる知識管理アプリ

  • 世界中で人気を集める知識管理・アイデア整理アプリ
  • ノートを書きためるだけで自分専用のWikipediaのような知識データベースを構築
  • 基本機能は無料で提供されている
  • Windows・Mac・Linuxに対応したデスクトップアプリ
  • iOSとAndroidのモバイルアプリも用意
メモ帳ではなく、思考の拠点になるアプリだよ！

②データは全部ローカル保存： クラウドに頼らない安心設計

  • ノートはすべてマークダウン形式のローカルファイル
  • 研究や執筆、日記などのデータを自分の手元に保存
  • インターネットが切れても使える
  • 有料プランでデバイス間同期が可能
  • 無料でもGitHubやGoogle Driveで同期できる
データの所有権が自分にあるのが大きな強み！

③Obsidianの主要機能： ノート・リンク・キャンバス・DB

  • エディター：文字を打つだけで見た目が整うノート作成
  • 内部リンク：ノート同士をつなげてアイデアの網の目を作成
  • キャンバス：メモ・画像・PDF・動画を並べて整理
  • DB機能：プロジェクト管理向けのデータベース構築
  • プラグインで表・カレンダー・タスク管理などを追加可能
ノート同士をリンクで繋ぐのがObsidianの真骨頂！

④ブラウザ拡張機能： ウェブ記事をObsidianにクリップ

  • Chromeなどのブラウザ拡張機能が用意されている
  • ウェブサイトの記事やコンテンツを簡単にクリップして保存
  • クリップした内容は自動的にマークダウン形式で保存される
  • 気になった記事をその場でノート化できる
気になる記事を見つけたら一瞬でノートに！

第2章：Obsidianの基本的な使い方

⑤インストール方法： 公式サイトからダウンロード

  • 公式サイトにアクセスしてダウンロードボタンを押す
  • 地球儀ボタンで日本語表示に変更可能
  • インストーラーを立ち上げてセットアップ完了
  • 初回起動時に保管庫（Vault）を作成する
  • クイックスタートボタンで自動生成も可能
保管庫＝ノートや画像を保存するフォルダのこと

⑥画面構成： リボン・サイドバー・エディターの3要素

  • リボン（左端）：クイックスイッチャーやグラフビューを起動
  • 左サイドバー：新規ノートの立ち上げとファイル一覧
  • エディター画面：ノートの作成と編集を行うメインエリア
  • グラフビューでノート間のつながりを視覚化
  • エディターは分割表示で複数ノートを同時閲覧可能
グラフビューで自分の知識の広がりを実感できる！

⑦ノートの基本操作： マークダウン記法で書く

  • #シャープで見出しを作成
  • ノート同士をつなぐ内部リンク記法を利用可能
  • ハイフン3本でプロパティ（メタデータ）を追加
  • プロパティにテキスト・リンク・日付・チェックボックスを設定
  • 外部のマークダウンファイルをドラッグ＆ドロップで追加
記法の詳細は公式ヘルプの「編集と書式設定」を参照

⑧設定とカスタマイズ： エディタ・外観・プラグイン

  • 左下の歯車アイコンで各種設定を変更
  • エディター設定でノートの入力環境を構築
  • 外観設定でObsidianのテーマを変更
  • コアプラグインは最初からインストール済み
  • 設定画面のコアプラグインでオン・オフを切替可能
自分好みの環境にカスタマイズして使いやすく！

第3章：ClaudianプラグインでAIを活用

⑨Claudianとは： Obsidian内でAIエージェントを使える超人気プラグイン

  • コミュニティプラグインの中で最も人気のAI系プラグイン
  • ベータ版を経て5月に公式リリース
  • AIチャットでテキストの指示を送るだけのシンプル操作
  • コミュニティプラグインは6月時点で5000以上存在
待望の正式リリースで一気に普及中！

⑩対応AIエージェント： Claude Code・Codex・OpenCodeなど

  • Claude Code：Anthropic社のAIエージェント
  • Codex：OpenAI社のAIエージェント
  • OpenCode・Piにも対応
  • どのエージェントでも同じ操作感で利用可能
  • 各エージェントのサブスクリプション範囲内で利用
Claude Pro（月額22ドル）でClaude Codeが使える！

⑪事前準備： Claude CodeとCodexのCLI版をインストール

  • CLI（コマンドライン）版のインストールが必要
  • Claude Code：Claude Pro（月額22ドル）に登録
  • Codex：ChatGPT Plus（月額3000円）に登録
  • ターミナルで1つのコマンドを打てばインストール完了
  • インストール方法は別動画で詳しく解説
どちらも有料プランが必要だけど、機能は充実してる！

⑫Claudianのインストール： 設定→コミュニティプラグインから検索

  • 歯車ボタン→コミュニティプラグインを有効化
  • 閲覧ボタンを押して「Claudian」を検索
  • インストールボタン→有効化ボタンの順に押す
  • ジェネラルタブの言語設定を日本語に変更可能
  • リボンの下部にアイコンが表示される
数クリックでインストール完了！簡単だよ

⑬セーフモードと養老モード： AIの自律性をコントロール

  • セーフモード：作業ごとにユーザーに許可を求める
  • 養老モード：AIが自分の判断で動く（ほぼ承認不要）
  • 養老モードは意図しない動作をするリスクがある
  • フォーム左下でモデルや思考の深さを設定可能
  • Codexは設定画面のトグルをオンで有効化
養老モードは便利だけど、使い方には注意が必要！

⑭リライト・要約・翻訳： 選択した文章をAIで加工

  • ノート内の特定の文章を選択してClaudianを開く
  • リライト：文章のトーンや表現を変更
  • 要約：長い文章を短くまとめる
  • 翻訳：日本語と英語などを相互に変換
  • チャットで指示してノートに反映や元に戻すことも可能
選ぶだけ！ChatGPTみたいにチャットで指示を出せる

⑮一括ファイル名変更： 大量ノートのタイトルを一気に整理

  • ノートが大量にある場合に一括でタイトル変更が可能
  • 保管庫内の全ノートの内容を読み取って最適なタイトルを提案
  • ユーザーが確認して承認すれば一括反映
  • 手動でやると何時間もかかる作業が数分で完了
ノートが数百枚あってもAIが一気に整理してくれる！

⑯スキル機能： AIに専門知識を追加できる拡張システム

  • スキル＝AIに特定の専門知識や機能を追加する仕組み
  • Obsidian公式スキルも用意されている
  • 「ClaudianでObsidianスキルズを導入してください」と入力でインストール
  • スラッシュコマンドでスキルを呼び出せる
スキルを追加するだけでAIの能力が大幅にアップ！

⑰データベース自動構築： 全ノート情報をDB化

  • Obsidian Skillsを呼び出してDB作成スキルを使用
  • 保管庫内の全ノートを読み取ってデータベースを自動生成
  • タイトル・作成日時・更新日時・サイズなどを一覧表示
  • テキストの指示で表示項目の追加・削除も自由自在
  • ノートの全体像を把握するのに最適
「全ノートのDBを作って」の一言で完成！魔法みたい

⑱インライン編集とトラブルシューティング： ショートカットで素早くAI活用

  • ホットキー設定でショートカットキーを登録
  • ノート内でショートカットを押してClaudianを直接起動
  • 変更後の文章を確認してAcceptで反映
  • 動かない場合はCLIのパス設定を確認する
  • パスはターミナルで`which`コマンド等で確認可能
エラーが出たらCLIパスを設定画面に入力してみてね！
