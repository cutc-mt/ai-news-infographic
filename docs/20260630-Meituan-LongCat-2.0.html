⑨ エージェントベンチマークで強い理由

  • エージェントタスクとはファイル編集・コマンド実行・エラー修正のループを回すタスク
  • ワンショットの回答ではなく、複数ステップを繰り返しながら自律的に問題を解く形式
  • Claude CodeやOpenClawなどのコーディングハーネス上で実際にタスクを実行して評価
  • LongCat 2.0はこの形式で一貫して高スコア。エージェント用途に最適化されている可能性

⑯ エージェント vs ワンショット: モデルの性質

  • ワンショット: 1回の質問に対して1回の回答を生成する形式。チャットアプリなどに最適
  • エージェント: ファイルを読み、コードを書き、エラーを修正し、再試行する自律的なループ形式
  • LongCat 2.0は明らかにエージェント寄りの設計。ワンショットでは弱く見えるが、それは設計意図かもしれない
  • 今後は「ワンショットスコア」だけでモデルを評価するのは不十分になる可能性
