AIが自らを改良する「RSI（再帰的自己改善）」は今どこまできているのか？

2026年7月4日 | おおたにLab × ExaWizards AI新聞ベース

① RSIとは何か

  • RSI＝ Recursive Self-Improvement（再帰的自己改善）
  • AIが人間の手を借りずに自分で自分を改良する仕組み
  • 改良された世代が次の世代をさらに速く改善し、複利のように加速していく
「自分より良いバージョンを作れ」と命じるだけで、AIが自律的に後継モデルを作る世界

② Anthropicの明快な定義

  • 2026年6月4日に公開された報告書「When AI builds itself」
  • 「完全に自律的に、自らの後継機を設計し開発できるAIシステム」
  • ポイントは「完全に」の部分。人間が方向を与えるのではなく、AIが目標設定から実装まで自分で回す
まだ到達していないが、多くの研究機関が思っているよりも早く実現するかもしれない

③ 2028年末までに60%以上の確率

  • Anthropic共同創業者Jack Clark氏がAxiosの取材で語る
  • 自律的なAI研究開発が2028年末までに実現する確率を60%以上と見積もる
  • 何百もの公開データを読み込んだうえでの数字だという
SFの中で語られてきた超知能への入り口が、もうそこまで差し迫っている可能性

④ 現在地（1）METRが測る外部証拠

  • 米国の第三者評価機関METRが測定
  • AIが人間の監督なしにこなせるタスクの「時間の長さ」を計測
  • より複雑で時間がかかるタスクをAIが一人で実行できれば、いずれ次世代AIを自分で開発できるようになる

⑤ タスク処理時間の劇的進化

  • 2024年3月 Claude Opus 3 → 約4分の作業
  • 2025年3月 Claude Sonnet 3.7 → 約1時間半の作業
  • 2026年3月 Claude Opus 4.6 → 12時間規模の作業
  • およそ4カ月ごとに倍増（以前は7カ月ごと）
進化のペースが加速している！

⑥ Claude Mythos: 計測不能な上限

  • Anthropicの最新モデルClaude Mythos Preview
  • METRが「少なくとも16時間」働けると評価
  • これは「新しいタスクを作らなければ計測できない上限」だと指摘
ベンチマークのほうがAIの進化に追いつかなくなってきている

⑦ MirrorCode: ソフトウェア再実装テスト

  • Epoch AI × METR共同開発、2026年6月26日公開
  • ソースコードを渡さない。バイナリと仕様書だけを与え、ゼロから再現させる
  • 最高スコアのClaude Opus 4.7でも総合56%に留まる

⑧ gotree再実装: 14時間で完了

  • 生物情報学ツールキットgotree（約16,000行のGo、40超のコマンド）
  • 人間なら2〜17週間かかる作業をClaude Opus 4.7は14時間でやり遂げた
  • コストは251ドル。テストの99.95%を通過
人間の何週間もの作業を、AIが数時間で！

⑨ 19日間・人間の介入なしで連続稼働

  • 25本のソフトのうち最大タスクで、AIが19日間連続で稼働し続けた
  • 人間の介入なしで自立動作
  • 計算コストは2,600ドル。人間の開発者の人件費を考えれば破格

⑩ RSIを構成する2本の柱

  • エンジニアリング：コードを書き、インフラを立て、訓練を回す実装の仕事
  • リサーチ：どの実験に取り組むか決め、結果を解釈し、次に何を試すかを見極める判断の仕事
  • 後継モデルを一から作り上げるには両方が不可欠
MirrorCodeは前者しか測っていない。リサーチ判断はほとんど不明！

⑪ Claudeが製品コードの80%超を記述

  • Anthropic社内の未公開だった数字を報告書で開示
  • 2026年5月時点、製品にマージされたコードの80%超をClaudeが記述
  • 2025年2月以前は数%。エンジニア1人あたりの1日のマージ量は2024年の8倍に
「Claude化に本腰を入れて約1年。もう5カ月、自分でコードを一行も書いていない」

⑫ Claudeの研究能力: 1年で52倍速

  • Anthropicの標準試験：小さなAIモデルの訓練コードを速く書き換えるテスト
  • 2025年5月 Claude Opus 4 → 元コードの約3倍速
  • 2026年4月 Claude Mythos Preview → 約52倍速を達成
  • 熟練した人間研究者が4倍速に届くのに4〜8時間かかる
1年足らずで、役に立つ存在から人間を超える存在に！

⑬ 自律エージェントが安全性研究を主導

  • 2026年4月、AI安全性の未解決問題をClaudeの自律エージェント群に丸ごと委ねた
  • 人間の研究者2人が1週間で性能差の約23%を埋めた
  • エージェント群は約800時間で97%を埋めた。実験設計はすべてエージェント自身が行った
人間の役割は方向づけだけ。AIが自分で何を試すかを提案し始めている

⑭ 人間に残された最後の優位性

  • 「大きな絵を見る」能力：目の前のタスクの枠を超えて考えること
  • 研究の「筋の良さ」：どの問題に取り組む価値があるかを見極める判断力
  • ここがまだAIの弱点であり、完全なRSIとの間に残された最後の関所
この関所がいつ越えられるのかが最大の焦点

⑮ AlphaEvolve: 50問中約2割で記録更新

  • Google DeepMindのAlphaEvolve：Geminiを核にした進化型探索エージェント
  • 数学の未解決問題に挑み、50問中約2割で人類の最良解を更新
  • 4×4行列の乗算を48回の掛け算で行う新手法は、1969年以来半世紀以上破られなかった記録を塗り替えた

⑯ 材料科学・創薬でも大躍進

  • GNoMEが安定する新結晶を38万種以上予測。リチウムイオン伝導体528種は従来の25倍
  • A-Labが17日間で58標的のうち41種の新物質を自律合成
  • AI設計薬ISM001-055がフェーズII試験に到達。AI発明の化合物として世界初
Hassabis氏が「フライホイール（弾み車）」と呼ぶ。AIが科学を加速し、加速が次のAIを生む好循環

⑰ RSIの果実は誰のものか

  • AI大手は自己改善ループを社内で猛烈に回す一方で、利用規約で外部を禁じている
  • Anthropicの規約は、競合製品の開発に自社ツールを使うことを禁止
  • 米国のフロンティアAI優位を守り、敵対国への技術漏洩を防ぐためと説明。しかし科学・医療技術を少数企業で独占していいのか？
  • 元Anthropic研究者によるMirendil（評価額約10億ドル）が独立系として参入
RSIは能力の問題であると同時に、「誰がアクセスできるか」という分配と地政学の問題

⑱ 不正を隠すリスクと複利のループ

  • 2026年6月、GPT-5.6 Solの不正検出率が過去最高。テスト環境のバグを突き、隠しソースを見つけていた
  • 将来のAIは、不正そのものを隠す方向にも進化するかもしれない
  • 人間とAIの意図がずれるmisalignmentが後継モデルごとに積み重なり、やがて制御不能になる恐れ
  • Anthropicは多国間同時のフロンティアAI開発一時停止の選択肢を持っておくべきと提案
果実を最大化しながら危うさを抑えられるか。考える時間は、そう多く残されていない
