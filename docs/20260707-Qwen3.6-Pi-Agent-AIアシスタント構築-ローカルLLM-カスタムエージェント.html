🤖 Qwen 3.6 + Pi Agent: 自分だけのAIアシスタントを構築

Bitforge
①

Pi Agentとは

  • pie.dev はオープンソースのエージェントハーネス
  • OpenClawエージェントやHermes Agentを動かす基盤技術
  • LLMを中心にツール・スキル・コネクタをラップする仕組み
  • 軽量で必要な機能だけを追加できるモジュラー設計
Piはオールインワンじゃなくて、自分で組み立てるのが魅力！🧩
②

Pi Agentの基本機能

  • ファイルの読み書きとコマンド実行が可能
  • ブラウザ制御でWeb操作を自動化
  • カスタムスキルをアップロードして定型作業を登録
  • プラグイン・コネクタで外部システムと連携
コンピュータの操作を全部AIに任せられる！🖥️
③

持続的メモリ機能

  • 永続的な記憶を作成してコンテキストを保持
  • セッションをまたいで過去の情報を記憶
  • 長期的なタスクでも一貫性のある対応が可能
  • 企業の知識ベースとしての活用も想定
AIに記憶ができるって画期的！🧠
④

今回のゴール設定

  • AI従業員を構築して受信箱を管理
  • Gmail・Outlook・Zendesk に対応
  • ローカルAIモデルで完全自前運用
  • 24時間365日自動応答を目標に
自分だけのAI従業員を作るぞ！🎯
⑤

LM StudioでローカルLLM

  • LM Studio はローカルモデル実行の定番ツール
  • API URLを提供して外部ツールからアクセス可能
  • LM Link で複数マシンのモデルにも接続
  • データは全てローカルでプライバシー安全
ローカルならAPIキー不要で安全！🔒
⑥

Qwen 3.6 35Bモデル

  • 今回はQwen 3.6 35BをMac Studioで実行
  • ローカルでも高速で快適な応答を実現
  • エージェント構築・コード生成にも十分な性能
  • 40以上のモデルをLM Studio で管理可能
Qwen 3.6はローカルAIの最適解の一つ！⚡
⑦

Step1: Piのインストール

  • インストールコマンドをターミナルに貼り付けて実行
  • 空のプロジェクトフォルダにPiを導入
  • pi と入力するだけで起動可能
  • 初期状態ではモデル未接続なのでパッケージが必要
数分でセットアップ完了！簡単！📥
⑧

LM Studio拡張の追加

  • pi-lm-studio パッケージをインストール
  • デバイス上の全モデルに自動アクセス
  • エラーが消えてすぐに会話可能に
  • 複数マシン間のLM Link接続にも対応
一発でローカルモデルと接続完了！✅
⑨

Piの軽量設計思想

  • インストール直後は最小限のハーネスのみ
  • 巨大なシステムプロンプトや事前スキルなし
  • 不要な機能を含まず用途に合わせて拡張
  • OpenClawのような多機能ツールとの対比
余計なものが一切ないのが最高！🪶
⑩

自己構築能力

  • Piは自身の内部ドキュメントを読んで理解
  • フォルダ構造・スキル・拡張機能を自動生成
  • 会話ベースでエージェントを構築できる
  • Qwen 3.6がすべてのコードを書いた
AIが自分で自分を作る！メタです！🔨
⑪

カスタマーサポートエージェント

  • Zendesk のチケットを自動で読み取り
  • チケットの文脈を理解して適切な応答を生成
  • 生成した応答をZendeskに自動送信
  • Bitforge社のサポートチームとして機能
サポートの自動化はビジネスの即効性がある！🎧
⑫

自動生成されたフォルダ構造

  • extensions/zendesk.ts — API連携の拡張機能
  • prompts/ — 応答のトーン・品質・構造を定義
  • skills/ — 応答品質と構造のスキル
  • チケットのタグ分類と対応ツールも自動設定
ローカルLLMが全部自動生成！すごすぎ！📂

環境変数とAPI認証

  • Zendesk管理画面でAPIトークンを発行
  • 認証情報をコードに直接書かないのが鉄則
  • Piが拡張機能の修正も自動で行った
.envファイルで安全に認証管理！🔑

cron jobで24/7自動化

  • 5秒ごとにZendesk受信箱をポーリング
  • ステータスが「新規」のチケットを自動検出
  • 検出したチケットをPiのPrint Modeに送信

  • 出力結果をチケットに貼り付け可能
対話なしで応答を生成できるのが鍵！🖨️
