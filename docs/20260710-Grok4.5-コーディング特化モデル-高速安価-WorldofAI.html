Grok 4.5 徹底レビュー

Opus & GPT レベルのコーディング力、さらに高速・安価・スマート
2026年7月10日

① Grok 4.5 の発表

  • SpaceX AI から正式リリース
  • コーディングエージェントとソフトウェア工学に特化した初のモデル
  • Cursor と共に共同訓練を実施
  • コーディング・科学・工学・数学の大規模データセットで学習
ベンチマークのスコアだけでなく、実際のエンジニアリング問題を解決できるモデルを目指して開発されました。

② ベンチマーク成績

  • TerminalBench: 83.3% — GPT 5.5 とほぼ同等
  • SWE-bench Pro: 64.7% — GPT 5.5 を上回る
  • ただし Fable 5 の 80.4% には及ばず
  • Deep SWE 1.0: 62% — 全フロンティアモデル中第3位
  • World of AI ランキングでも第4位にランクイン
Artificial Analysis でも第4位。最先端モデルの仲間入りを果たしました。

③ 圧倒的な速度

  • 80 TPS（トークン/秒）で高速出力
  • 「高速モデル」相当のスピードを搭載
  • Opus 4.8 max よりも速く結果を返す
  • 日常的なコーディング作業で待ち時間ゼロに近い体験
フロンティアモデルのスピードとは思えない爆速レスポンス！開発体験が劇的に向上します。

④ トークン効率

  • SWE-bench Pro で Opus 4.8 max より4.2倍少ない出力トークン
  • 最新の最先端モデル比で2倍のトークン効率
  • GPT 5.5 と同等かそれ以上のトークン効率
  • 長時間のコーディングタスクで特に威力を発揮
トークン効率が良い＝コストが安いだけでなく、レスポンスも速くなるというダブルのメリット。

⑤ 破格の価格設定

  • 入力: $2 / 100万トークン
  • 出力: $6 / 100万トークン
  • 他の最先端モデルと比べて圧倒的に安価
  • 高精度・高速・安価の三拍子揃ったコスパ最強モデル
この価格でこの性能なら、日常の開発作業で使わない手はありません。

⑥ コンテキストウィンドウ

  • 現在は500K トークンのコンテキストに対応
  • 間もなく100万トークンにアップグレード予定
  • Elon Musk が次週にも拡張を示唆
  • 大規模なコードベースも一度に処理可能に
100万トークン実現で、さらに大規模プロジェクトに対応できるようになります。

⑦ macOS クローン生成

  • トップバーは全モデル中最も優秀な仕上がり
  • 壁紙の変更やアプリのドラッグ＆ドロップに対応
  • カレンダーアクセスなど本物そっくりの機能を実装
  • FPS シューティングゲームまで単発生成
  • Launchpad に多数のSVG アイコンを配置
アプリの機能の深さでは Opus や Fable に一歩譲るものの、見た目のクオリティは最高クラス。

⑧ SaaS ランディングページ

  • React 18 をアーキテクチャに採用
  • GSAP + ScrollTrigger でアニメーションを実装
  • デザインセンスが非常に優れていると評価
  • パッケージ・コンポーネント・タイポグラフィの使い分けが巧み
速度とコストの面でも有利なので、プロトタイプ作成やスキャフォールド生成に最適です。

⑨ Minecraft クローン

  • 全モデル中第3位の生成品質
  • クラフティングシステムを実装
  • ブロックの配置やツールの独自のテクスチャを生成
  • 松明やモブの動きなど本物に近い挙動を再現
  • 1位は Fable 5、2位は GPT 5.5x high output
ツールのテクスチャ生成は他のモデルでは見たことがないレベル。驚きのクオリティです。

⑩ 3D F1 ドリフト

  • F1 ドーナツプロンプトでレーシングカーを横向きに傾ける演出
  • 環境やライティング・シェーダーは良好
  • ドリフトの煙エフェクトをリアルに表現
  • 物理演算の精度と建物の窓の生成に課題
見た目は良いものの、ドリフトの物理が不正確で 6/10 点。オープンモデルの中にはもっと良いものもあり。

⑪ 3D 太陽系の課題

  • テクスチャやコントラストが不足して地味な仕上がり
  • 各惑星は単色のベーシックな色のみ
  • 小惑星帯は生成できたものの、全体的に味気ない印象
  • 3D 分野は追加の訓練が必要と指摘
モデル自体が悪いわけではありません。フロントエンドでは優秀ですが、3D はまだ成長の余地あり。

⑫ SVG 生成の強さ

  • 蝶の SVG: 対称性・形状バリエーション・色彩が秀逸
  • SVG 絵画プロンプトで創造的な作品を生成
  • 独自のタッチを加えた要素でポップ感を演出
  • アニメーション付きラバランプの CSS 実装も良好
色相や照明・ブロブの物理表現も自然。SVG 領域ではフロンティア級の実力を発揮。

⑬ レイトレーシング レンダラー

  • WebGL とシェーダーパッケージで構築
  • 大気フォグや手続き型スカイを表現
  • アニメーション付きで複数の視覚効果を披露
  • WebGL を活用したレンダリング品質は高評価
単一ショットでこれだけ複雑なレンダラーを生成できるのは、Grok 4.5 の強みの一つです。

⑭ ゼルダ風 3D シーン

  • 「Breath of the Wild」風のローポリ 3D 環境を生成
  • 丘の起伏や照明パッケージが非常に自然
  • 桜の木や環境音まで完備
  • 構図やミニマリスティックな美しさで最高評価
Grok 4.5 の中で最も印象的なビジュアルクリエイティビティ！オープンワールドの雰囲気が素晴らしい。

⑮ エージェントオーケストレーター

  • 複数モデルを組み合わせるワークフローに最適
  • Sonnet 5 の代わりに Grok 4.5 を差し替え可能
  • 優秀な作業馬（ワークホース）としての役割
  • 強力なコーディング性能＋トークン効率を両立
Fable 5 と組み合わせて、最難関タスクは Fable に・日常開発は Grok 4.5 に、という使い分けが推奨されます。

⑯ 他モデルとの比較

  • Fable 5: 最強の性能だがコストが高い
  • GPT 5.5: 高性能だがトークン効率で Grok に譲る場面も
  • Grok 4.5: 速度・コスト・品質の最良のバランス
  • Opus 4.7: Grok 4.5 の下位にランクイン
  • 「最強である必要はない」のが Grok 4.5 の強み
総合性能では Fable 5 や GPT 5.5 に一歩譲るものの、実用的なバランスでは最強の候補です。

⑰ 利用可能なプラットフォーム

  • Grok Build で直接アクセス可能
  • API を通じた利用に対応
  • Cursor プラン内で利用可能
  • World of AI ベンチマークツールでもテスト可
  • EU 地域は未対応（7月中旬に順次ローンチ予定）
EU ユーザーはもう少し待つ必要がありますが、他の地域では今すぐ使えます。

⑱ 総評: 最高のデイリードライバー

  • 高速・安価・強力の三拍子を揃えた実用モデル
  • デバッグ・日常コーディング・エージェントワークフローに最適
  • フロントエンドや SVG でトップクラスの生成品質
  • 3D 分野は今後の改善に期待
  • SpaceX AI が本気で戻ってきたことを証明するモデル
「長い間で見た中で最高のモデルの一つ」— Fable 5 を最難関タスク用に残しつつ、Grok 4.5 を日常使いにするのが賢い戦略です。
