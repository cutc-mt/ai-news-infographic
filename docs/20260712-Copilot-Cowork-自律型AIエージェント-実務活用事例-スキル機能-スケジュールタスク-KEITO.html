Copilot Coworkとは？
M365の最新AIエージェント

  • Fortune 500企業の過半数がすでに導入済み

Microsoftのエコシステムの中で動く自律型AIエージェント！想像以上に優秀🪄
②

Copilot Chat との違い
助手 vs 作業する同僚

  • Copilot Chat＝「下書きを返す助手」文章の作成や壁打ち・相談
  • Copilot Cowork＝「成果物を返す同僚」ファイルを作り込んで仕事をする
  • PowerPoint・Word・ExcelなどMicrosoft製アプリに強い
  • Outlookメールの確認から返信下書きまで自動作成
  • 画像生成でバナー制作も可能
  • スケジュールタスクで毎日自動実行も設定できる
Chatは相談相手、Coworkは仕事をしてくれる同僚！この違いが超重要🤝
③

メール整理デモ
受信トレイを自動で整理

  • 自然言語で「48時間分の受信トレイを整理して」と指示
  • 総数30件のメールを分類し、緊急度で並べてくれる
  • 当日締め切りのメールを優先的に抽出
  • 各メールの返信下書きを自動で作成
  • 「この内容で送信」ボタンからそのまま返信可能
  • AIが書いた返信文を確認後、承認制で安全に送信
メール返信が承認作業になるだけ！毎朝のメール処理が劇的に楽になる📧

Excel見積もり作成
スキルで商品マスター参照

  • 「冷房終了しました。2台です。見積もりを作成」自然な言葉でOK
  • あらかじめ覚えさせた商品マスターを自動参照

パソコン上の全ファイルは触れないけどOneDrive経由ならOK！注意⚠️

ファイル生成の仕組み
OneDrive連携が鍵

  • 生成されるファイルはOneDrive内に直接作成される
  • ダウンロード不要でPowerPointやWordのデータとして完成
  • パソコンのローカルフォルダーにOneDriveを接続していれば同期される
  • 注意：パソコン上の全ファイル操作は一切不可
  • OneDrive上のフォルダー操作だけが許可範囲
ダウンロードの手間ゼロ！OneDriveに直接ポンと出てくるのが最高📁

スキル機能とは？
ワークフローを記憶させる

  • あらかじめCoworkにタスクのワークフローを覚え込ませる
  • スキルを呼び出すだけで覚えた手順で仕事をこなす
  • スラッシュコマンド（/）でスキルを即座に起動
  • 例：/pptx で毎回同じデザインのスライド資料を生成
  • 例：/myevent で毎回同じバナー画像を生成
  • 再現性の高さが実務活用のポイント
スキルこそCoworkの核心！覚え込ませれば何度でも再現できる🧠

スキルの作成方法
自然言語で登録するだけ

  • カスタマイズ→スキル→追加→新規作成
  • デフォルトの文章が表示されるのでやりたいことを伝える
  • 自然言語で指示：「テーマについて検索してExcelに構成案を出力」
  • 送信するとAIが自動でスキルを完成させる
  • 完成後は即座に使える状態で利用可能
自然言語で「これ作っといて」だけでスキルが完成！超簡単✨

スケジュール済みタスク
毎朝自動でメール確認

  • 「毎日朝9時にOutlookのメールを確認」自然言語で設定
  • 優先度の高いメールを自動で抽出
  • 返信案を下書きとして保存してくれる
  • 実際のOutlook下書きフォルダーに保存される
  • 「遠信不要・既読済み」と判断したものはスキップ

毎朝メール返信の下書きが出来てる！今すぐやるべき活用法⏰
⑮

Copilot Credits
従量課金の料金体系

  • Copilot Credits従量課金モデル
  • 1クレジット = $0.01（PayGo方式）
  • Lightタスク：100〜300クレジット
  • Mediumタスク：400〜700クレジット
  • Heavyタスク：700クレジット以上
  • 低コスト設計がCoworkの5つの設計要素の一つ
1クレジット1円！軽いタスクなら数百円で済むコスト感💰
⑯

コスト管理
テナント別の上限設定

  • テナント・グループ・ユーザー別に上限を設定可能
  • 利用アラートで予算超過を未然に防止
  • 管理者が課金グループを作成してユーザーを管理
  • どのユーザーがどれくらい消費したかを可視化
  • 同じMicrosoftクレジットカードで従量課金を設定
上限設定とアラートで請求書ショックを防ぐ！管理者の必須設定📊
⑰

管理者の3ステップ
課金→エージェント→承認

  • 課金設定：Copilot Creditsの従量課金情報を登録する
  • エージェント追加：管理センター → エージェントタブ → Coworkを検索してオンにする
  • ユーザー承認：各ユーザーにCopilotアカウントでCoworkを承認
  • 従量課金のグループを作成しCowork利用ユーザーを含める
  • この3つの要素が揃えばユーザー画面にCoworkタブが表示
管理者がやらないと使えない！IT管理者のいる会社が導入できる⚙️
