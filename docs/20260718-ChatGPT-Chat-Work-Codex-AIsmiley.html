ChatGPT「Chat」「Work」「Codex」の違いと使い分け方

3つのモードが並ぶ新UIを徹底解説！どっちを使えばいいのか迷わなくなる

2026年7月18日

① 2026年7月9日の大発表

  • ChatGPTに新モード「Work」が登場
  • 開発者向けだった「Codex」アプリがデスクトップアプリに統合
  • これまでのアプリは「ChatGPT Classic」に改名
  • 画面にChat・Work・Codexの3つのモードが並ぶように
新機能追加＋アプリ改名が同時に起きたから
当然混乱するよね！専門家も迷ってたよ

② 専門家も混乱していた事実

  • ウォートン校のイーサン・モリック教授：「Workが何なのか分からない」と投稿
  • 著名AIエンジニアのサイモン・ウィリソン氏：「本当に混乱している！」と白旗
  • Hacker Newsでは「WorkとCodexを切り替えても何も変わらないように見える」との声
専門家が迷うんだから、
私たちが迷うのは当然。
この記事で地図を渡すよ！

③ 3秒で決まる使い分け

  • Chat → 答えが欲しい（質問・相談・アイデア出し）
  • Work → 成果物が欲しい（スライド・表・ドキュメント）
  • Codex → PCの中を触らせたい（ファイル整理・作業代行）
ポイントは機能で覚えないこと！
「最後に受け取りたいものは何？」
これだけで3秒で選べる

④ Chatモードの役割

  • 質問・検索・ブレインストーミングに対応
  • 使える場所：Web・スマホ・デスクトップすべて
  • 消費は一番軽い、財布に優しい
  • これまでのChatGPTと同じ使い勝手

例：「この企画、どう思う？」→ Chatで十分！

普段の会話はChatのまま。
無理に新しいモードを使う必要なし

⑤ Workモードの役割

  • トピックの調査と情報の分析を担当
  • スプレッドシート・スライド・ドキュメント・レポートを完成形で作成
  • 使える場所：Web・スマホ（有料）・デスクトップ
  • 消費は重い（Codexとプール共有）

例：「この3つの資料から、会議用のスライドを作って」→ Work！

「答える」で終わらず
「仕上げる」まで進むのがWorkの強み

⑥ Codexモードの役割

  • コードの作成・デバッグ・テスト・レビュー
  • ローカルフォルダ・リポジトリ・ターミナルを操作
  • 使える場所：デスクトップのみ（Web・スマホでは不可）
  • 消費は重い（Workとプール共有）

例：「ダウンロードフォルダの中で、消していいファイルの候補を出して」→ Codex！

「コーデックス＝コード専用」
という名前の罠！
実はファイル片付けも得意

⑦ Codexはコード専用じゃない

  • 週間500万人以上が利用中
  • うち100万人以上はソフトウェア開発以外で活用
  • PCの片付けやファイル整理にも大活躍
  • 「15分で9GB空いた」という実例も
本質は「手元のPCを直接触る係」
コードを書かない人でも使える！

⑧ 初心者はChatとWorkから始める

  • まずはChatとWorkの2つで回す
  • ファイル整理などの「手を動かす系」が必要になったらCodexを足す
  • 3つ全部を理解してから始めようとすると挫折しやすい
  • この順番がいちばん挫折しない
日々の仕事の大半は
「資料や答えが欲しい」だから
Workが先！Codexは後でOK

⑨ 知らないと損するお金の話

  • WorkとCodexは使用量（クレジット）を同じプールで共有
  • ChatGPT for Excel・Workspace Agentsも同じプールから消費
  • WorkもCodexも通常のChatよりずっと消費が重い
  • 「普通のチャットのつもりでWorkで会話していたらクレジットを食われた」と悲鳴
「答えが欲しいだけならChat」
これは財布を守るルールでもある！
人間、お金がかかると学習が早いね

⑩ 新しいデスクトップアプリの構成

  • 左上のモード切り替えでChat・Work・Codexを切替
  • 左ナビの「クイックチャット」で新しい会話を開始
  • 従来のCodexアプリはアップデートで新アプリへ移行
  • ChatGPT Classicは新機能が来ない（更新・修正のみ継続）
WorkとCodexが使えるのが新アプリ
使えないのがClassic
見分け方は簡単だよ

⑪ 3つのモード対応環境まとめ

  • Chat: Web・スマホ・デスクトップ → 消費は軽い
  • Work: Web（有料）・スマホ（有料）・デスクトップ（全プラン）→ 消費は重い
  • Codex: デスクトップのみ → 消費は重い

CodexはWeb・スマホではモードとして選べない。モバイルアプリの「リモート」タブからデスクトップ版タスクにアクセス可能。

外出先でCodexを使いたい？
モバイルの「リモート」タブ経由なら
デスクトップのタスクを確認できるよ

⑫ Workでできる6つの主要機能

  • 成果物生成: スプレッドシート・スライド・ドキュメントを完成形で作成
  • Sites: 共有可能なWebサイト・Webアプリを作成
  • プラグイン連携: 1,400超の外部ツールと接続
  • Scheduled Tasks: 定期実行・変化の監視を自動化
  • 実行前計画: 作業方針を整理し、実行前に確認
  • ローカル操作: デスクトップ限定で内蔵ブラウザ＋PC直接操作
これ1つで調査から成果物まで
一気通貫で任せられるのがWork！

⑬ Workの成果物生成の具体例

  • 月末の予算差異分析: Excelの集計を自動化
  • マーケティングブリーフ: 顧客調査から販促素材まで一連対応
  • 営業ミーティング準備: 競合比較表付きの提案資料を自動生成
  • 複数ステップのワークフロー全体を文脈を引き継ぎながら処理
テンプレートや参照ファイルを渡せば
その形式に沿って作ってくれる
数週間の分析が数時間に短縮！

⑭ Scheduled Tasksの自動化

  • 繰り返しの業務をWorkに常駐させることが可能
  • 時刻指定の定期実行に対応
  • 情報の変化をトリガーにした起動も可能
  • SlackやTeamsの新着メッセージを基にドキュメントを自動更新
  • 重要な変更をチームへ共有するところまで任せられる
毎週・毎月の定型業務こそ
AIに任せるのが正解！
人は確認するだけ

⑮ 対応プランと料金モデル

  • デスクトップ: Freeを含む全プランでChat・Work・Codex利用可能
  • Web・モバイル: Pro・Enterprise・Eduから先行提供
  • Free・GoはGPT-5.6 Terraのみ
  • Plus以上はSol・Terra・Lunaから選択可能
  • Pro・Enterprise限定でultra（複数エージェント並列）あり
無料ユーザーでもデスクトップなら
Workが試せるのは大きい！
まずはFreeで感触を掴もう

⑯ 先行導入企業の事例

  • Zapier: リード分析システムを構築。月7桁ドルのパイプラインを特定
  • Virgin Atlantic: 競合調査データセットを作成。数週間→数時間に短縮
  • NVIDIA: イベント準備のExcel作業を自動化。準備時間の約40%を削減
  • RingCentral: 散在する情報を集約。パイロット6社→約80社に拡大を支援

OpenAI社内でも財務・営業を含むほぼ100%のチームがWorkとCodexを活用中。

定型的な集計・整理業務ほど
導入効果が検証しやすい
まずは身近な業務から！

⑰ セキュリティとデータの注意点

  • Auto-review: 重要操作を上位モデルが実行前に審査
  • レッドチーム演習で保護データの抜き取りを100%ブロック
  • 法人向けプランではモデル学習に使用されない
  • 個人向けプランは学習設定をオフにする必要あり
  • 権限は最小限から始める運用が安全
エージェントは広くアクセスするから
導入前の確認が欠かせない
「権限は最小限」が鉄則

⑱ 明日から始める3ステップ

  • 普段の質問はChatのまま、無理に新しいモードを使わない
  • 今週1回だけ、「自分がよく知っている仕事」をWorkに渡してみる
  • 出来の良し悪しを自分で判断できる業務から始めるのが確実
地図は1枚で足ります。
答え→Chat、成果物→Work、
PCを触らせたい→Codex。
