Claude MCP × Higgsfieldで1つの動画から広告・Shorts・バズコンテンツを自動生成

①AIクリエイティブの限界: プロンプト1回では不十分

  • ほとんどのAIワークフローはプロンプトから始まり1枚の画像で終わる
  • 便利だが、真のブランドワークフローとは言えない
  • ブランドにはビジュアルシステムと一貫性が必要
画像を1枚作るだけではブランドは築けない

②ブランドに必要な要素: ビジュアルシステムの全体像

  • 製品の世界観を定義する一貫性のある色彩感覚
  • パッケージデザインやキャンペーン画像
  • ソーシャルメディア素材と統一感のある動き
すべてが一体となって感じられる動きが鍵

③Higgsfield MCPの導入: Claudeをクリエイティブツールに接続

  • Claudeの設定 → コネクタ → カスタムコネクタを選択
  • Higgsfield MCPのエンドポイントを入力して追加
  • 接続が確立されると、クリエイティブツールにアクセス可能に
数ステップでAIと画像生成を連携

④架空ブランド「Luma Botanica」: ボタニカルスキンケア

  • 穏やかで思慮深い毎日の習慣を求める人向けのスキンケアブランド
  • 主力商品はガラス製フェイシャルセラムボトル
  • 温かみがあり高級感のある、自然でエディトリアルな印象
無機質すぎず、素朴すぎない絶妙なバランス

⑤Claudeをクリエイティブディレクターに任命する

  • ビジネス目標を与え、Claudeに意思決定を委ねる
  • ターゲット層の定義、カラーパレットの提案、写真スタイルの選択
  • 繰り返し使う素材の定義や初期ブランドアセットの推奨
人間は方向性だけ決めて、細部はAIに任せる

⑥ClaudeとHiggsfieldの役割分担: 思考と生成

  • Claudeは計画と調整のレイヤーを担当する
  • Higgsfieldは実際のメディア生成を担当する
  • 手作業でプロンプトを翻訳する必要なし。Claudeが変換する
思考するAIと生成するAIの完璧な分業

⑦ヒーロー商品画像の生成: ブランドの基盤を築く

  • 最初はヒーロープロダクト画像を作成。他の全ての基盤となる
  • ボトル、ラベル、素材の質、照明、キャンペーンのトーンを統一
  • 見出し配置用の十分な余白を確保した構図に
ウェブサイトのヒーロー画像兼広告素材に

⑧バリエーション生成: 世界観を保ったまま展開

  • 同じビジュアルシステム内でトップダウン型の儀式的な画像を作成
  • セラム、リネン布、陶器のトレイ、植物の葉などの小物を配置
  • 新しいスタイルではなく、同じシステムの新しい楽曲を制作する感覚
アルバムの異なる曲のように一貫性を保つ

⑨サポート画像の作成: テクスチャ・棚・使用例

  • 美容液のテクスチャークローズアップで詳細なアセットを提供
  • バスルームの棚風景で生活感のあるブランド体験を表現
  • 柔らかな光の中で製品を塗布する手の画像で使用感を伝える
異なる役割でも同じ光・色彩・抑制を共有

⑩ソーシャルキャンペーン: チャネルごとの構図最適化

  • 画面を見損なわない3つの構図をClaudeがHiggsfieldに依頼
  • 中央配置の正方形・縦長ポートレート・幅広バナー
  • 視覚的アイデンティティは一定で、フレーミングだけが変わる
ランダムな画像生成ではなくブランドシステムの構築

⑪画像レビュー: 一貫性の徹底チェック

  • 生成された画像をまとめて色調・製品形状・照明・背景素材を比較
  • 主役画像・補助画像・詳細画像としての適切な役割を推奨
  • 人間が最優秀なアセットを承認し、劣った部分だけ修正を指示
「より良くする」ではなく具体的に一貫性を検証する

⑫精密修正: 承認済みアセットのピンポイント調整

  • ボトル形状がずれたら修正版の商品画像を依頼
  • 緑が強すぎたらアイボリーとセージグリーンの色合いに戻す
  • 照明が強すぎたら柔らかな朝の光に変更を指示
全体を作り直すのではなく必要な部分だけ微調整

⑬動画制作への移行: 静止画が先、動画が後

  • 静止画システムが承認されて初めて動画制作に移行する
  • 動画は既存の視覚システムに制御された動きを追加するもの
  • 新しい視覚言語の発明ではなく、承認済みアセットに動きを付与
順番が重要：先に絵を決めてから動かす

⑭3種類のプロダクト動画: ズーム・滴・フォーカス

  • ①カメラをゆっくりズームイン。ガラス面のハイライトと植物の影が変化
  • ②フラットレイ構図でセラムの滴がフレーム内に入る演出
  • ③手の自然な動きからボトルへ焦点がソフトに移行
動きはシンプルで、ブランドのトーンを損なわない

⑮クロップ最適化: 用途に応じたアスペクト比

  • 縦長: Short・ストーリーズ向け。焦点要素を失わない配置
  • 正方形: ソーシャルフィード向けの簡潔な商品構成
  • 横長: ウェブサイトやプレゼンテーション向けに余白を確保
同じビジュアルシステムの異なる応用例

⑯決定報告書: Claudeの画像比較と推奨

  • どの画像が最も印象的か？ 製品の一貫性に問題はないか？
  • どの画像が最も優れたネガティブスペース（余白）を持っているか？
  • ランディングページ・メール・SNS投稿のどれに最適かを推奨
次に取るべき行動が明確になる比較レポート

⑰小規模チームのワークフロー: 役割の分業

  • デザイナーは方向性を承認する
  • マーケティング担当者はキャンペーンのバリエーションをリクエスト
  • エージェントが同じ視覚ルールを生成ステップ全体に適用
一人の記憶に依存せず、AIが一貫性を維持する

⑱結論: 1つの会話から実用的なブランドシステムを構築

  • AIに画像を作らせるのではなく、ブランドシステムを構築する
  • 初期の意思決定が一貫していれば、後の修正時間は最小限に
  • Claude MCP × Higgsfieldでクリエイティブワークフローの常識が変わる
