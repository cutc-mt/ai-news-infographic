⑥基本的な使い方: ソース入力からノート生成

  • 「ノートを新規作成」ボタンを押して開始。
  • Web検索するか、ファイルをアップロード、またはURLを入力。
  • ソースをもとにAIが1冊の仮想ノートを自動生成。
ソースを入れるだけでノートが完成！

⑦ノートの3大カテゴリ: Sources・Chat・Studio

  • Sources — ソースの追加・管理。
  • Chat — ソースについてAIチャットで質問。
  • Studio — ポッドキャスト、動画、レポート、インフォグラフィックを生成。
3つのカテゴリで使い分け

⑧コンテンツ生成: 音声・動画・スライド・クイズ

  • 音声付きのポッドキャスト風解説が自動生成。
  • 解説動画や縦長ショート動画の作成も可能。
  • レポート、インフォグラフィック、クイズなど多彩な形式。
生成前に内容を細かく調整できる

⑨最近の追加機能: マインドマップ・フラッシュカード等

  • インフォグラフィック生成時に参照ソースを確認可能。
