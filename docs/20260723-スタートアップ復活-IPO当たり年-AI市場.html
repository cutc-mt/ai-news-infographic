世界のスタートアップ冬は終わった — AI時代の幕開けと米中の激しい攻防

① 世界のスタートアップ冬は明けた

  • 長らく続いた冬の時代がついに終わりを迎えた。少なくともアメリカは明確に冬が明けたという状況
  • 中国もだいぶ復活しており、インドも勢いを取り戻しつつある
  • スタートアップ大国の順位はアメリカ中国イギリスインドの4カ国がトップを占める
  • 米中の比率は今1.3対1程度。アメリカが依然として最大だが、中国も急速に追い上げている
海原さん（リブライトパートナーズ代表）がついに宣言。「冬が明けた」と自信たっぷりに語る状況です。

② グローバル資金調達が回復軌道に

  • クォーターごとの伸びを見ると、金額ベースは去年から明確に戻ってきている
  • 件数はまだ戻っていないように見えるが、第4四半期の数字は年明けのレポートで後から追加される
  • 年明けにフォローアップすると、去年より約10%増加というデータが出てくるのが通例
  • グローバル全体で金額も件数も回復しつつあり、この流れは当面続くと予測される
黄色い点は年明けに追加分が入ってくる未確定データ。統計の読み方には要注意ですね。

③ AIが資金調達回復の最大の原動力

  • アメリカでは資金調達の金額ベースで全体の65%、件数ベースでも40%がAI関連のディール
  • サンフランシスコでは多分99%がAI関連という極端な状況。現地の投資家も「全部AI」と証言
  • Yコンビネーター出身の福山太郎さんも「出てくる話の80%以上がAI」と明言
  • このAIブームによってスタートアップ冬の時代が打ち破られたという状況
冬の時代は約4年続いた。

④ AIがついにSaaSを抜いて首位に

  • 米国VC協会の統計で、2024年にAIがSaaSを抜いて資金調達件数でトップに。それまで10年ずっとSaaSが1位だった
  • SaaS企業もデッキ（プレゼン資料）には「私たちはAIカンパニー」と書くケースが急増
  • AIとSaaSの上位2セクターを合わせると約1万件。そのほとんどが実質的にAI企業
  • SaaSの時代は終わり、AIの時代が本格的に到来したという明確なシフト

⑤ 巨人の独占からトリクルダウンへ

  • 2024年は資金調達額の全体の68%がトップ10企業が占める極端な集中状態だった
  • しかし2026年にはその割合が45%にまで低下。トップ企業への依存が半分弱まで減った
  • 約1万件のAI関連企業が登場し、AIエージェントやエンドユース領域まで資金が降りてきた
  • 90年代のインターネットの土管のように、AIがインフラ化して無数のスタートアップが誕生

⑥ Exit Is Back — 出口が再開した

  • スタートアップ資金調達の回復の最大の要因はエグジット（IPO・M&A）が良くなったから
  • Exit is back — IPOとM&Aを合わせたエグジットが金額・件数ともに完全に過去最高水準に回復
  • アメリカでは件数ベースで73%がM&A、5%がIPO。日本は逆にIPOに依存しすぎる問題が指摘されている
  • 金額ベースではIPOとM&Aがほぼ均衡。ビリオンダラー以上の企業価値でIPOするのが一般的
出口が悪いと入り口も閉まる。

⑦ 2026年はIPO当たり年

  • 2026年の上半期のIPO資金調達額が、バブル期だった2021年の通期とほぼ匹敵するレベルに到達
  • SpaceXが1社で1,000億ドル（約15兆円）を調達。OpenAIのIPO延期報道も出たがAntropicは年内予定
  • Databricks、Stripeなど大型IPOが控えており、市場最高のIPO当たり年になる確率が高い
  • S&P500、ナスダック、ダウがオールタイムハイを更新中。テック株に対して極めて良好な市場環境

⑧ 2020-21のバブルと今の違い

  • 2020-21年は完全にバブル。ゼロ金利政策と大量のマネタリーベース拡大（4倍増）が原因
  • SPAC（空っぽの会社を上場させる裏口上場）が乱用され、ファンダメンタルズの無い企業も次々と上場
  • 今回の復活はファンダメンタルズが非常に良い。株価収益倍率も90年代の100倍・何百倍のような異常値ではない
  • インフレ抑制のために金利が上昇した冬の時代を経て、選別された企業だけが生き残ったことが信頼を生んでいる
業界関係者は全員

⑨ 金融政策がスタートアップを左右する

  • 金利はコロナ時にゼロに張り付き、マネタリーベースはそれまでの4倍に急増。実質マイナス金利の状態でバブルが発生
  • インフレによって金利が急上昇し、マネタリーベースも低下。これがスタートアップ冬の時代の直接的な原因
  • 現在は金利が再び下がり、マネタリーベースも回復。金融環境が好転したことで資金調達も再び活発化
  • 過去のサイクル（リーマン→ユニコーン時代→冬→復活）は約7〜8年周期。2030年までは続きそう
金融政策がスタートアップの
命運を握る。

⑩ ソフトウェア時代からVDE時代へ

  • トップラインからボトムラインへ。売上ではなく利益をKPIとしてしっかり見るようになった
  • 資金調達のほとんどがAIまたはディープテックにシフト。ソフトウェア中心からハードテック中心への転換
  • 「骨太のスタートアップブーム」— 安易なソフトウェアから、時間もお金もかかる本物のテクノロジーへ

⑪ VDE（バーティカル・ドメイン・エキスパート）

  • VDE（Vertical Domain Expert） — 各産業・バーティカルに深く入り込み、AIを深く実装するビジネスモデル
  • 代表例はPalantir。産業ごとにAIを深く組み込み、産業効率を革命的に向上させる
  • ベンダーロックイン効果が極めて強い。一度導入すると他社への切り替えが困難になり、AI企業は大きな収益を得る
  • SaaSに似ているが、高度なAIを活用する点が異なる。世界各地のバーティカルごとにウィナーが誕生すると予測
Palantirのアレクサンダー・カープCEOは

国家と提携せよと訴えています。

⑫ AI導入は雇用を増やす — VDEの真実

  • 最新の統計でAIアダプションをしている会社ほど雇用が増えているという結果が発表された
  • コーディングの作業はAIで大幅に自動化されるが、現場でのフィードバックループやハイタッチな作業は人間が必要
  • インドのゴールドマンサックスやアクセンチュアでは何万人単位のプログラマーが一著しく減る見込み
  • ただしドメインのノウハウを蓄積できるVDEは、内製化よりも断然優れた結果を出せる。バーティカルで勝てば独占的な優位性が得られる
