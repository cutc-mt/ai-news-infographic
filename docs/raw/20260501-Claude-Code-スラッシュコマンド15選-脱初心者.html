セクション1：基本コントロール

/model — モデル切替

/model
  • 使用するAIモデルを切り替えられる
  • 選択可能：HaikuSonnetOpus
  • 思考量（Effort）も調整可能：ロー → ミディアム → ハイ
  • Haikuは安い・早い、Opusは高い・賢い
簡単な作業はHaiku、複雑な実装はOpus！コストと品質のバランスを意識しよう ✨

/clear — セッションクリア

/clear
  • 会話履歴をリセットしてクリーンな状態に
  • 文字数制限（トークン上限）に近づいた時に活用
  • 会話が長いと回答が遅くなる・品質が落ちる
  • 誤った方向に進んだ時のやり直しにも使える
頻繁に使うコマンド！毎回クリアせず、状況に応じて使おう 🧹

/compact — 会話圧縮

/compact
  • 会話履歴をギュッと圧縮して節約
  • 文脈を維持したまま容量を削減
  • /clearで全部消すのは惜しい時に最適
  • 消すならclear、要約して残すならcompact
/clearで消しちゃうと記憶喪失…/compactなら記憶を残したままスッキリ！📖

/cost — コスト確認

/cost
  • セッションでどのぐらいコストを使ったかを確認
  • 上限の何%まで使っているかチェック可能
  • コスト管理に必須のコマンド
100万トークンくらい上限があるみたい。無駄遣いしないようにこまめにチェック！💰
セクション2：自動化・スケジュール

/loop — 定期ループ実行

/loop
  • 指定した間隔で繰り返し実行
  • 最大50回の実行設定が可能
  • 活用例：ルーティンとして登録して毎朝9時にニュース収集
  • CLI版限定（デスクトップでは使えない）
24時間ループも設定できる！例：「24時間に1回おはよう！」的に登録も可能 🌅

/schedule — スケジュール実行

/schedule
  • 特定の日時にタスクを実行
  • 複数タスクを順番に並列実行も可能
  • 完了通知をスマホに送る設定もできる
  • loop=決められた感覚で繰り返し / schedule=指定日時に実行
onlineタスクとlocalタスクで分けてスケジュール管理すると効率的！📋
セクション3：情報管理

/memory — 記憶管理

/memory
  • Claude Codeの記憶（CLAUDE.md）を管理
  • ルールを覚えさせると以降のセッションに引き継がれる
  • CLAUDE.mdが膨れると読み込みが遅くなる → 必要な情報だけ書く
  • プロジェクトごとのルールや好みを記憶させよう
Claude Codeを育てる感覚！ポケモンのようにレベルアップさせるんだ 🐾

/mcp — MCP接続管理

/mcp
  • 接続中のMCPサーバー一覧を表示
  • 新しいMCPの追加や設定も可能
  • 外部ツール（WordPress、X等）との連携
  • 設定はclaude_desktop_config.jsonで管理
MCPを使えばClaude Codeの能力が爆増！各種ツールを接続して活用範囲を広げよう 🔧

/skills — スキル一覧

/skills
  • 登録済みのスキル一覧を表示
  • 各スキルに設定されたエージェントを確認
  • 新しいスキルを追加・ダウンロード可能
  • マーケットからプラグインもインストールできる
スキルを呼び出すだけで一連の作業フローが全部自動で動くんだよ！✨
セクション4：ワークフロー強化

/hooks — フック設定

/hooks
  • 特定の操作をトリガーにして自動実行を設定
  • 例：ファイル削除前に確認を必須にする
  • フォルダ整理ルールをフックに登録も可能
  • 編集前後・コマンド実行前後でツールを動かせる
「ここぞ」という時に自動で安全確認や処理を挟めるんだ！⚓

/use — ツール使用

/use
  • 特定のツールをClaude Codeに使用させる
  • どのスキルにどのエージェントを使うかマッピング
  • 作業の流れを最適化して効率的に進められる
何の作業にどのエージェントを使うか明確にしておくと効率が上がるよ！🗺️

/plan — プランモード

/plan
  • すぐに実行せず、まず計画を立てるモード
  • Claudeがプランを提案 → 承認後に実行に移行
  • 誤った実行を防ぐための安全策
  • 複雑なタスクや大規模変更前に使うのがおすすめ
とりあえず相談してみて、できそうなら実行！慎重に進めよう 🧙
セクション5：高度な機能

/resume — セッション再開

/resume
  • 過去のセッションを再開・復活
  • 前の会話を圧縮して記憶を引き継ぐ
  • Web版 → デスクトップ版へのセッション移行にも
  • 完全な引き継ぎではなく要約ベース
ターミナル消しちゃった！？/resumeで前の作業に復帰できるよ 🔄

/remote — リモート操作

/remote
  • スマホから遠隔操作でClaude Codeを動かせる
  • 外出先で事務所のPCを操作できる
  • QRコードをスマホで読み込むだけで接続完了
  • Web版限定（デスクトップアプリにはない）
外出中に作業させたい時にスマホで遠隔操作できるのは超便利！📱

/plugins — プラグイン管理

/plugins
  • プラグインマーケットから追加・管理
  • 興味あるプラグインを簡単にインストール
  • 一覧表示・設定確認が手軽にできる
  • /reloadでプラグインの即座に反映も可能
Claude公式プラグインもここから管理！新機能をサクッと試せるよ 🧩
