01

Codex × GPT-5.5 が拓く AI スーパーアプリ

OpenAI の Codex アプリは GPT-5.5 を搭載し、データ分析からアプリ開発、ゲーム構築、ブラウザ操作、PC 制御まで幅広いタスクをこなす。まさに「何でもできる AI アシスタント」の実現形。

概要
02

OS World ベンチマークで 78.7% を達成

OS World Verified という、AI が実際の PC 環境を自律操作できるかを測るベンチマークで、GPT-5.5 は 78.7% のスコアを記録。従来モデルを大幅に上回り、トークン効率も大幅に改善された。

ベンチマーク
03

Browser Use プラグインの登場

OpenAI は Codex 内に Browser Use プラグインを新設。これにより、フロントエンドを構築したあとに GPT-5.5 が実際のブラウザでテストを走らせる「ビルドと検証のサイクル」が完全に閉じるようになった。

新機能
04

ビルド → テスト → 修正が自動で完結

Codex に「フロントエンドを作って」と指示すると、GPT-5.5 が実際のユーザーのようにブラウザを操作してアプリをクリックし、コンソールやネットワークログを同時に監視してバグを自動修正する。

開発フロー
05

視覚能力でユーザーと同じ画面を見る

GPT-5.5 のビジョン機能により、Codex はユーザーが見ているものをそのまま認識できる。スクリーンショットの撮影、ページ遷移の確認、UI 要素の識別などをリアルタイムで行いながらタスクを進める。

技術
06

最小限の人間介入で高品質な成果物を

自律型コーディングエージェントとして、人間が介入することなく高品質でテスト済みのソフトウェア変更を届けることができる。開発プロセス全体を自動化する大きなステップフォワード。

自動化
07

PC操作速度が 42% 向上

OpenAI は Computer Use のパフォーマンスを大幅に改善し、最新の Codex アプデートで 42% の高速化を実現。大規模言語モデルが人間と同じスピードで GUI を操作できるのは今回が初めて。

パフォーマンス
08

ダッシュボードと /act コマンド

Codex を起動するとメインダッシュボードが表示される。＋ボタンからプラグインメニューを開き、Browser Use を確認・インストールできる。/act コマンドを使えばチャットパネルから直接プラグインを操作可能。

操作方法
09

権限はデフォルト設定が推奨

Browser Use や Computer Use を使う場合、権限設定はデフォルトのままにしておくのがおすすめ。細かいタスクでなければ知能レベルを「低」に設定すると、レート制限に達しにくくなる実用的なTips。

Tips
10

プロジェクトは新規から始めるのが正解

作業を隔離するため、新規プロジェクトからスタートすることを推奨。これにより、既存ファイルへの意図しない変更を防ぎ、Browser Use や Computer Use のテスト環境をきれいに保てる。

ベストプラクティス
11

ブラウザ操作の基本 — なんでも頼める

メールの送信、ローカルホストでのチェックアウト フローのテスト、Web ページの開きなど、ブラウザ経由のほぼすべての作業を Codex に任せられる。Chrome を開いて YouTube チャンネルを表示させるだけでも動作確認になる。

ユースケース
12

定期実行の自動化も設定可能

Codex 内で自動化ワークフローを作成し、Browser Use プラグインを指定時間（例: 毎朝9時）に実行するようスケジュール設定できる。一度設定すれば、繰り返しの定型作業を全自動で回せる。

自動化
13

実例: AIニュース収集 → PDF 生成

「毎日新しいAIトピックを収集してPDFにまとめて」と指示するだけで、Browser Use が Web を巡回・収集し、情報源を整理した完全な PDF を短時間で生成。深い分析と定期的なタスク実行の両方をこなす。

実例
14

自作アプリのユーザーテストを自動化

メモアプリを GPT-5.5 に作らせたあと、「このアプリのユーザーフローをテストして」と指示。左パネルでタスクが次々と実行され、新規メモの追加や動作確認がリアルユーザーと同じように行われる。

テスト
15

チェスゲームの自動プレイで機能検証

チェスゲームを作らせたあと「チェスをプレイして」と頼むだけで、Browser Use がゲームを実際にプレイして各機能が正しく動くかを検証。バグを検出すると、コンソールログと視覚情報から原因を特定して自動修正する。

検証
16

デスクトップのファイル整理もお任せ

Computer Use を使えば、デスクトップ上のバラバラなファイル（サムネイル画像15枚など）を「整理して」と一声かけるだけで、自動的に番号順にリネーム＆フォーマット揃えまで完了。手作業ゼロで片付けが終わる。

日常タスク
17

iPhone ミラーリングでスマホ操作も可能

Codex と Apple の iPhone ミラーリングアプリを組み合わせると、Mac 経由で iPhone 上のアプリを操作できる。モバイルアプリのUXテスト、SNS自動投稿、メッセージ管理、iOS 向けQAテストなど多彩な活用が期待される。

拡張性
18

まとめ — ブラウザ・PC・スマホを横断するAIエージェントの時代へ

Codex は Windows と macOS の両方で無料で使える。Claude Code と比べて OpenAI の有料枠の使い勝手が良く、Browser Use ＋ Computer Use により、ブラウザ・デスクトップ・スマホをまたいで自律的に動く AI エージェントの時代が幕を開けた。
