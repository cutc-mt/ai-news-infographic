🎬 AI動画生成ワークフロー

Claude Code + Remotion + Archon でつくる次世代動画制作

📅 2026年5月3日

①

Archonとは？

  • オープンソースのワークフローエンジン
  • 動画生成の全体を14ステップで自動管理
  • 「プロジェクトマネージャー」役として機能
  • GitHubで公開中のレポジトリあり
Archonが全体の流れを制御して、各AIツールに指示を出すよ！
②

Claude Codeの役割

  • 「AI映画監督」として全体をオーケストレーション
  • シーンプランニングを自動で設計
  • ナレーション・音声の自動生成
  • BGM・効果音の自動作曲
  • TypeScriptコンポジションを直接記述
Claude Codeが企画からコードまで全部やってくれるのがすごい！
③

Remotionの役割

  • プログラマティックな動画生成ツール
  • TypeScriptで動画をコードとして記述
  • 39ルールのベストプラクティススキルを注入
  • リアルなトランジション・動的尺対応
「AI slop」を避けてプロ品質に仕上げる秘訣がココ！
④

14ステップの流れ

  • URLからコンテンツ取得
  • シーン構成の自動プランニング
  • ナレーション・音楽・効果音の生成
  • TypeScriptコンポジションの記述
  • 品質チェック → 自動修正 → MP4レンダリング
全部自動で回るから、放っておいても完成する！
⑤

コンテンツの自動取得

  • URLを入力するだけ！
  • 記事・ブログ・商品ページに対応
  • ウェブスクレイピングで内容を抽出
  • 抽出テキストをシーン構成に変換
記事のURLを渡すだけで数分で動画完成！やばみ！
⑥

シーンプランニング

  • 内容を複数シーンに分割
  • 各シーンの尺を動的に計算
  • ビジュアルイメージの自動設計
  • シーン間の自然なつながりを設計
Claude Codeが映像監督の頭脳として働く！
⑦

ナレーション生成

  • AIによる自然な音声合成
  • 内容に合わせた話し方を自動調整
  • 尺に合わせたテンポ制御
  • 感情豊かな読み上げに対応
人間が読んでるみたいな自然さが目標！
⑧

音楽・効果音の自動生成

  • シーンに合ったBGMを自動作曲
  • 転換点での効果音を自動追加
  • 音量バランスの自動調整
  • 動画と音声の完璧な同期
音楽までAIが作る！著作権フリーで安心！
⑨

TypeScriptコンポジション

  • Remotion用のTSコードを自動生成
  • 各シーンをコンポーネントとして記述
  • アニメーション・レイアウトの自動設計
  • 39ルールに基づく品質基準を適用
動画制作＝ソフトウェア開発という発想の転換！
⑩

39ルールの品質基準

  • リアルなトランジションを必ず使用
  • 動的な尺で単調さを排除
  • フック（引っかかり）を冒頭に配置
  • 「AIっぽい」動画の典型パターンを回避
これが「AI slop」じゃないプロ品質の秘密！
⑪

品質チェック＆自動修正

  • レンダリング前に自動で品質検証
  • クリティカルな問題を自動検出＆修正
  • 問題があれば再生成を自動で実行
  • 品質基準を満たすまで反復
人間が確認しなくても一定品質を保証！
⑫

最終レンダリング

  • 完成したMP4動画を自動出力
  • プロダクショングレードの画質
  • 数分〜数十分で完成
  • そのままSNS投稿可能
URL→MP4まで完全自動！神ワークフロー！

「使い切り」じゃなくて自分のシステムとして構築！
